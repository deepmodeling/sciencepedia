## 引言
在[演化生物学](@entry_id:145480)中，构建物种演化历史的“[生命之树](@entry_id:139693)”是一个核心目标。然而，当我们从宏观的物种关系深入到微观的基因组时，常常发现单个基因的演化路径（[基因树](@entry_id:143427)）与物种的整体分化历史（[物种树](@entry_id:147678)）并不一致。这种[基因树](@entry_id:143427)与物种树之间的冲突，并不仅仅是数据错误或分析方法的局限，其背后往往隐藏着一个深刻而普遍的生物学过程——不完全谱系分选（Incomplete Lineage Sorting, ILS）。理解这一现象是准确解读海量基因组数据、重建生命历史的关键所在。

本文将系统地引导你攻克这一重要概念。在“原理与机制”一章中，我们将回归基本定义，运用[溯祖理论](@entry_id:155051)解释不完全谱系分选发生的根本原因及其概率受到哪些因素的调控。接下来，在“应用与跨学科联系”一章中，我们将展示ILS如何从一个[系统发育分析](@entry_id:172534)的“干扰项”转变为一个充满洞察力的“信息源”，探讨其在[物种界定](@entry_id:176819)、[性状演化](@entry_id:165250)分析以及区分[基因渗入](@entry_id:174858)等方面的广泛应用。最后，通过“动手实践”部分，你将有机会通过计算和案例分析，将理论知识转化为解决实际问题的能力。让我们一同启程，揭开隐藏在[基因树](@entry_id:143427)冲突背后的演化奥秘。

## 原理与机制

在[演化生物学](@entry_id:145480)中，理解物种的演化历史是核心任务之一。我们通常通过构建“物种树”（species tree）来描绘物종间的分化与[亲缘关系](@entry_id:172505)。然而，当我们深入到基因组层面，考察单个基因的演化历史——即“[基因树](@entry_id:143427)”（gene tree）——时，常常会发现[基因树](@entry_id:143427)的拓扑结构与[物种树](@entry_id:147678)并不一致。这种[基因树](@entry_id:143427)与物种树之间的拓扑结构冲突被称为“[基因树](@entry_id:143427)-[物种树不一致性](@entry_id:168924)”（gene tree-species tree discordance）。这种不一致性并非总是由数据错误或分析方法不当引起，它往往源于一个深刻的生物学过程，即**不完全谱系分选**（Incomplete Lineage Sorting, ILS）。本章将深入探讨不完全谱系分选的基本原理、核心机制及其在演化分析中的重要意义。

### 基因树、[物种树](@entry_id:147678)与不一致性

首先，我们必须精确区分两个核心概念：**[物种树](@entry_id:147678)**与**基因树**。[@problem_id:2726218]

**[物种树](@entry_id:147678)**（species tree）描绘的是物种或种群谱系随时间分化的历史。树上的每一个分枝点代表一个[物种形成](@entry_id:147004)事件，即一个祖先[种群分化](@entry_id:188346)为两个或多个后代种群。因此，物种树追踪的是种群层面的演化关系。

**基因树**（gene tree）则描绘了在特定[基因座](@entry_id:177958)（locus）上，从不同个体或物种中取样的等位基因（alleles）的演化历史。基因树的分枝点代表这些等位基因的**溯祖共祖**（coalescence）事件，即它们追溯到的[共同祖先](@entry_id:175919)。

基因的演化历史镶嵌在物种的演化历史之中，但两者并不等同。想象一个简单的物种树 `((A,B),C)`，它表示物种 A 和物种 B 的亲缘关系比它们各自与物种 C 的关系更近。与此[物种树](@entry_id:147678)**一致**（concordant）的基因树拓扑结构也将显示 A 和 B 的等位基因首先聚合，然后才与 C 的等位基因聚合，即 `((A,B),C)`。然而，另外两种可能的基因树拓扑结构——`((A,C),B)` 和 `((B,C),A)`——则与[物种树](@entry_id:147678)**不一致**（discordant）。不完全谱系分选是导致这种不一致性的主要内源性生物学原因。

### 溯祖过程与不完全谱系分选的起源

不完全谱系分选的根本原因在于**[祖先多态性](@entry_id:172529)**（ancestral polymorphism）的存在以及[基因谱系](@entry_id:172451)在物种分化过程中的随机分选。[@problem_id:1940282]

要理解这一点，我们需要引入**[溯祖理论](@entry_id:155051)**（Coalescent Theory）的视角。该理论不从现在向前看，而是从当前样本中的[基因谱系](@entry_id:172451)（lineages）向后追溯它们的[共同祖先](@entry_id:175919)。在一个种群中，任意两个[基因谱系](@entry_id:172451)都有一定的概率在上一代找到[共同祖先](@entry_id:175919)，这个过程被称为**溯祖共祖**。

ILS 的发生必须满足一个基本前提：在[物种形成](@entry_id:147004)事件发生时，祖先种群在该[基因座](@entry_id:177958)上是多态的，即存在至少两种不同的等位基因。[@problem_id:1940282] 如果祖先种群是单态的（即所有个体都携带相同的等位基因），那么所有后代物种必然继承同一个等位[基因谱系](@entry_id:172451)，基因树将不可避免地与物种树一致。

让我们通过一个具体的例子来阐明ILS的机制。[@problem_id:1940277] 假设有三个物种 X、Y、Z，其真实的物种树为 `((X,Y),Z)`。在所有这三个物种的共同祖先种群中，一个名为 *Florin* 的[基因座](@entry_id:177958)是多态的，同时存在 `F1` 和 `F2` 两种等位基因。

1.  **第一次物种分化**：祖先[种群分化](@entry_id:188346)为 Z 的谱系和 (X,Y) 的[共同祖先](@entry_id:175919)谱系。此时，`F1` 和 `F2` 两种等位基因的[多态性](@entry_id:159475)可能在这两个新谱系中都得以保留。
2.  **谱系分选（一）**：在通往物种 Z 的谱系中，由于遗传漂变，`F2` 等位基因可能丢失，使得物种 Z 最终只保留了 `F1`。
3.  **关键阶段**：在 (X,Y) 的[共同祖先](@entry_id:175919)种群中，`F1` 和 `F2` 的多态性持续存在。这至关重要，因为这意味着[多态性](@entry_id:159475)跨越了第一次物种分化事件。
4.  **第二次物种分化**：(X,Y) 的共同祖先[种群分化](@entry_id:188346)为物种 X 和物种 Y。
5.  **谱系分选（二）**：在物种 X 的谱系中，`F2` 等位基因可能因随机漂变而丢失，导致物种 X 固定了 `F1`。同时，在物种 Y 的谱系中，`F1` [等位基因丢失](@entry_id:175439)，导致物种 Y 固定了 `F2`。

最终，物种 X 的 *Florin* 基因为 `F1`，物种 Y 为 `F2`，而物种 Z 为 `F1`。当我们基于这些基因序列构建[基因树](@entry_id:143427)时，由于物种 X 和 Z 共享相同的 `F1` 等位基因，[系统发育分析](@entry_id:172534)会将它们聚为一类，从而得到一个 `((X,Z),Y)` 的[基因树](@entry_id:143427)。这个基因树与真实的 `((X,Y),Z)` [物种树](@entry_id:147678)发生了冲突。这个冲突的根源在于，(X,Y) 共同祖先种群中的祖传多态性未能在 X 和 Y 分化之前“分选完毕”（即固定为一个等位基因），这就是**不完全谱系分选**的本质。

### 影响不完全谱系分选概率的因素

ILS 并非在所有情况下都以相同频率发生。其概率主要受两个相互关联的因素控制：**物种分化之间的时间间隔**和祖先种群的**有效种群大小**。

为了量化这些影响，演化生物学家使用**溯祖单位**（coalescent units）来衡量时间。对于一个二倍体物种，1 个溯祖单位时间等于 $2N_e$ 个世代，其中 $N_e$ 是**[有效种群大小](@entry_id:146802)**（effective population size）。$N_e$ 反映了种群中参与繁殖的个体数量，代表了遗传漂变的强度。溯祖单位的本质是将[绝对时间](@entry_id:265046)（世代数）根据种群大小进行了缩放，它衡量的是发生溯祖事件的相对机会。

对于一个 `((A,B),C)` 的三物种系统，ILS 的概率与连接 A-B 分化点和 (A,B)-C 分化点的**内部支路**（internal branch）的长度密切相关。设该内部支路持续的时间为 $\tau$ 个世代，此期间祖先种群的有效种群大小为 $N_e$。那么，该内部支路的长度（以溯祖单位计）为 $t = \frac{\tau}{2N_e}$。

从物种 A 和 B 中抽取的两个[基因谱系](@entry_id:172451)在该内部支路中未能发生溯祖的概率为 $\exp(-t)$。如果它们未能溯祖，它们将与来自物种 C 的谱系一起进入更深层的共同祖先种群。此时，三个谱系中任意一对发生溯祖的概率是均等的。因此，产生与物种树不一致的[基因树](@entry_id:143427)（即 `((A,C),B)` 或 `((B,C),A)`）的总概率为：

$$P_{\text{discordant}} = \frac{2}{3} \exp\left(-\frac{\tau}{2N_e}\right)$$

这个公式清晰地揭示了影响 ILS 概率的两个关键因素：

1.  **物种分化间隔时间（$\tau$）**：两次连续的物种分化事件之间的时间间隔越短，内部支路 $\tau$ 就越小。这为谱系在其中发生溯祖提供的“时间窗口”就越窄，从而导致 ILS 的概率显著增加。[@problem_id:1940306] 例如，一个物种树中观察到一个极短的内部支路，是预测该区域可能存在高度 ILS 的一个强烈信号。一项研究发现，如果分化间隔从 $\tau_{\text{long}} = 3N_e$ 缩短到 $\tau_{\text{short}} = 0.5N_e$，不一致基因树的发生概率会增加 $\exp(1.25)$ 倍，即大约 3.5 倍，这凸显了快速辐射演化对[系统发育重建](@entry_id:185306)的挑战。[@problem_id:1940312]

2.  **[有效种群大小](@entry_id:146802)（$N_e$）**：祖先种群的[有效种群大小](@entry_id:146802)越大，ILS 的概率也越高。这是因为在较大的种群中，任意两个个体是近亲的概率较低，[基因谱系](@entry_id:172451)需要追溯更长的时间才能找到[共同祖先](@entry_id:175919)。因此，对于一个固定的时间间隔 $\tau$，更大的 $N_e$ 会使溯祖单位时间 $t$ 变短，从而增加谱系未能溯祖的概率。例如，对于两个演化历史时间完全相同的甲虫[类群](@entry_id:182524)，如果一个类群的祖先[有效种群大小](@entry_id:146802)为 $N_{e,B} = 500,000$，而另一个为 $N_{e,A} = 10,000$，前者发生 ILS 的概率会比后者高出约 $1.80 \times 10^{4}$ 倍。[@problem_id:1940294]

这个简单的[溯祖模型](@entry_id:202220)还有一个重要的推论：在没有其他演化力量干扰的情况下，两种不一致的基因树拓扑结构 `((A,C),B)` 和 `((B,C),A)` 的预期频率是**相等的**。[@problem_id:1940273] 这是因为一旦三个谱系（A、B 和 C）都进入了深层的[共同祖先](@entry_id:175919)种群，它们之间谁先发生溯祖完全是一个随机对称的过程。A 与 C 先溯祖的概率，和 B 与 C 先溯祖的概率是完全一样的，均为 $\frac{1}{3}$。因此，$P(((A,C),B)) = P(((B,C),A)) = \frac{1}{3} \exp(-t)$。

### 自然选择对不完全谱系分选的影响

上述讨论主要基于[中性演化](@entry_id:172700)模型，即遗传漂变是驱动等位基因频率变化的唯一力量。然而，自然选择可以深刻地改变基因的溯祖模式，从而影响 ILS 的概率。

#### [平衡选择](@entry_id:150481)：ILS的助推器

**[平衡选择](@entry_id:150481)**（Balancing selection）是一种维持多个等位基因在种群中长期共存的[自然选择模式](@entry_id:136310)，例如[杂合子优势](@entry_id:143056)或[频率依赖性选择](@entry_id:155870)。在主要组织相容性复合体（MHC）基因中，这种选择机制尤为常见。当一个[基因座](@entry_id:177958)长期处于[平衡选择](@entry_id:150481)之下，它会极大地延长等位[基因谱系](@entry_id:172451)的溯祖时间。[@problem_id:1940323]

如果[平衡选择](@entry_id:150481)在一个物种分化之前就已经在祖先种群中作用了很长时间，它会主动维持[祖先多态性](@entry_id:172529)。这使得等位[基因谱系](@entry_id:172451)可以轻易地跨越多个物种形成事件而不发生溯祖。例如，在人类、黑猩猩和大猩猩的演化中，某些受[平衡选择](@entry_id:150481)的基因（如[MHC基因](@entry_id:187974)）的等位[基因谱系](@entry_id:172451)可能比这三个物种自身的分化历史还要古老。这种现象被称为**[跨物种多态性](@entry_id:196940)**（trans-species polymorphism）。因此，与[中性演化](@entry_id:172700)的基因相比，受长期[平衡选择](@entry_id:150481)作用的基因更有可能表现出与[物种树](@entry_id:147678)不一致的基因树，即具有更高的 ILS 概率。[@problem_id:1940323]

#### [选择性清除](@entry_id:169307)：ILS的抑制剂

与[平衡选择](@entry_id:150481)相反，**选择性清除**（Selective sweep）会显著降低 ILS 的概率。[@problem_id:1940317] 当一个携带有利突变的新等位基因出现并迅速在种群中[扩散](@entry_id:141445)至固定时，它会通过“遗传搭便车”（genetic hitchhiking）效应，顺带清除与之连锁的其他[基因座](@entry_id:177958)上的遗传变异。

这个过程极大地压缩了该[基因座](@entry_id:177958)的溯祖时间。所有个体在该[基因座](@entry_id:177958)上的等位基因都将追溯到那个携带有利突变的、非常晚近的共同祖先。如果一次强烈的[选择性清除](@entry_id:169307)发生在物种 A 和 B 的共同祖先种群中，就在它们分化为两个物种之前，那么该[基因座](@entry_id:177958)的[祖先多态性](@entry_id:172529)将被彻底清除。A 和 B 将继承来自同一个近期祖先的等位基因。因此，它们的[基因谱系](@entry_id:172451)必然会在其[共同祖先](@entry_id:175919)种群中发生溯祖，从而产生与物种树 `((A,B),C)` 一致的基因树。在这种情况下，发生 ILS 的可能性被大大降低甚至消除了。[@problem_id:1940317]

### 区分不完全谱系分选与[基因渗入](@entry_id:174858)

除了 ILS，**[基因渗入](@entry_id:174858)**（Introgression），即物种形成后通过杂交发生的基因流动，也是导致[基因树](@entry_id:143427)-物种树不一致的重要原因。在分析[系统发育](@entry_id:137790)基因组数据时，区分这两种现象至关重要。

一个关键的诊断特征在于不一致[基因树](@entry_id:143427)拓扑结构的**[频率分布](@entry_id:176998)**。如前所述，纯粹的 ILS 模型预测两种不一致的拓扑结构 `((A,C),B)` 和 `((B,C),A)` 应该以相等的频率出现。[@problem_id:1940300]

然而，如果观察到这两种不[一致拓扑](@entry_id:158991)结构的频率存在显著的**不对称性**，那么这强烈暗示了[基因渗入](@entry_id:174858)的发生。例如，对于物种树 `((A,B),C)`，如果分析大量的[基因座](@entry_id:177958)后发现，`((B,C),A)` 拓扑的频率远高于 `((A,C),B)`，最合理的解释是物种 B 和物种 C 之间在分化后发生了[基因流](@entry_id:140922)动。来自 C 的基因流入 B 的基因库，会系统性地增加 B 和 C 在[基因树](@entry_id:143427)上[聚类](@entry_id:266727)的机会，从而抬高 `((B,C),A)` 的频率，而 `((A,C),B)` 的频率则不受影响或仅由背景水平的 ILS 产生。[@problem_id:1940300]

因此，通过统计[全基因组](@entry_id:195052)范围内成千上万个独立基因树的拓扑结构[分布](@entry_id:182848)，演化生物学家可以利用这种对称性或不对称性的信号，来推断物种间复杂的演化历史，区分古代的 ILS 事件和更近期的[基因渗入](@entry_id:174858)事件。

总之，不完全谱系分选是理解物种演化和[系统发育](@entry_id:137790)基因组学的一个基石概念。它源于[祖先多态性](@entry_id:172529)在物种快速分化或有效种群巨大的背景下的随机遗传过程。通过理解其背后的原理和影响因素，并结合对自然选择和[基因渗入](@entry_id:174858)等其他演化力量的考量，我们能够更准确地解读基因组数据中蕴含的复杂演化历史。
