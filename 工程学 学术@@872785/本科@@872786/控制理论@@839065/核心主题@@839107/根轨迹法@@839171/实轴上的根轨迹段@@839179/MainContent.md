## 引言
在[反馈控制系统](@entry_id:274717)的分析与设计中，[根轨迹法](@entry_id:273543)是一种强大的图形化工具，它直观地揭示了系统动态特性随增益变化的趋势。然而，要绘制一幅完整且准确的[根轨迹图](@entry_id:264447)，第一步也是最关键的一步，是确定轨迹在[s平面](@entry_id:271584)[实轴](@entry_id:148276)上的[分布](@entry_id:182848)。许多初学者在此遇到困难，不清楚如何系统性地判断哪些实轴线段是有效的[根轨迹](@entry_id:272958)。本文旨在填补这一知识空白，提供一个关于根轨迹[实轴](@entry_id:148276)段的全面指南。在接下来的内容中，我们将分三步深入探讨。首先，在“原理与机制”一章，我们将阐明确定[实轴](@entry_id:148276)段的根本条件，并介绍一个快速判定的实用捷径——“奇偶性法则”，同时讲解如何计算极点在[实轴](@entry_id:148276)上的[分离点](@entry_id:265082)与汇合点。接着，在“应用与[交叉](@entry_id:147634)学科联系”一章，我们将展示这些原理如何应用于[系统稳定性](@entry_id:273248)分析、[控制器设计](@entry_id:274982)，并揭示其与[数字控制](@entry_id:275588)、现代控制等高级领域的深刻联系。最后，通过“动手实践”部分的精选习题，你将有机会亲手应用所学知识，巩固理解并提升解决实际问题的能力。

## 原理与机制

在分析[反馈控制系统](@entry_id:274717)的动态行为时，[根轨迹图](@entry_id:264447)是一种不可或缺的图形工具。它描绘了当系统的一个参数（通常是[比例增益](@entry_id:272008) $K$）从零变化到无穷大时，闭环系统极点的所有可能位置。[根轨迹](@entry_id:272958)的构建遵循一系列明确的规则，这些规则源于系统的特征方程。本章将深入探讨确定[根轨迹](@entry_id:272958)[实轴](@entry_id:148276)段的根本原理与机制，这是绘制完整[根轨迹图](@entry_id:264447)的第一步，也是至关重要的一步。

### 实轴段的基本条件

一个标准的负反馈系统，其[特征方程](@entry_id:265849)为 $1 + L(s) = 0$，其中 $L(s)$ 是[开环传递函数](@entry_id:276280)，通常可以表示为 $L(s) = K G(s)$。因此，特征方程可以写成：

$$1 + K G(s) = 0$$

其中 $K$ 是可调增益。对于 $s$ 平面中的任何一点要成为[根轨迹](@entry_id:272958)的一部分，它必须对于某个实数增益 $K$ 满足这个方程。重新整理该方程，我们得到：

$$G(s) = -\frac{1}{K}$$

现在，我们考虑位于[实轴](@entry_id:148276)上的点，即 $s = \sigma$，其中 $\sigma$ 是一个实数。对于一个[开环传递函数](@entry_id:276280) $G(s)$，其系数均为实数，当输入 $s = \sigma$ 为实数时，其输出 $G(\sigma)$ 也必然是实数。

在最常见的情况下，我们关心的是**标准[根轨迹](@entry_id:272958)**，即增益 $K$ 为正值（$K > 0$）时的轨迹。在这种情况下，$ -1/K $ 是一个负实数。因此，一个[实轴](@entry_id:148276)上的点 $\sigma$ 位于标准根轨迹上的**充分必要条件**是 $G(\sigma)$ 的值为负实数。

$$G(\sigma)  0$$

这一条件等价于幅角条件，即 $\angle G(\sigma) = (2\ell+1)\pi$，其中 $\ell$ 是任意整数。

为了具体说明，我们来看一个磁悬浮列车的简化模型，其[开环传递函数](@entry_id:276280)为 [@problem_id:1603766]：

$$L(s) = \frac{K(s+5)}{(s+2)(s-2)}$$

这里，$G(s) = \frac{s+5}{(s+2)(s-2)}$。为了判断一个[实轴](@entry_id:148276)上的测试点（例如 $s=0$）是否在[根轨迹](@entry_id:272958)上，我们只需计算 $G(0)$ 的符号：

$$G(0) = \frac{0+5}{(0+2)(0-2)} = \frac{5}{-4}  0$$

因为 $G(0)$ 是负数，所以我们可以找到一个正增益 $K = -1/G(0) = 4/5  0$，使得 $s=0$ 是闭环[特征方程](@entry_id:265849)的一个根。因此，$s=0$ 点在根轨迹上。相反，如果我们测试 $s=4$：

$$G(4) = \frac{4+5}{(4+2)(4-2)} = \frac{9}{12}  0$$

由于 $G(4)$ 是正数，我们无法找到一个正的 $K$ 满足 $G(4) = -1/K$。因此，$s=4$ 不在标准根轨迹上。

### 奇偶性法则：一个实用的捷径

虽然直接计算 $G(\sigma)$ 的符号是可行的，但在绘制[根轨迹](@entry_id:272958)时，有一个更快捷、更直观的法则，我们称之为**奇偶性法则**。这个法则源于对 $\angle G(\sigma)$ 的系统性分析。

对于[实轴](@entry_id:148276)上的一个测试点 $\sigma$，[开环传递函数](@entry_id:276280) $G(s)$ 的每个实数极点或零点对其[相角](@entry_id:274491)的贡献非常简单。一个位于 $\sigma$ 右侧的实数零点 $z_i$（即 $z_i  \sigma$）贡献的[相角](@entry_id:274491)为 $\angle(\sigma - z_i) = \pi$。同样，一个位于 $\sigma$ 右侧的实数极点 $p_j$（即 $p_j  \sigma$）贡献的[相角](@entry_id:274491)为 $\angle(\sigma - p_j) = \pi$。而任何位于 $\sigma$ 左侧的实数极点或零点，其贡献的[相角](@entry_id:274491)均为 $0$。

因此，$G(\sigma)$ 的总[相角](@entry_id:274491)可以表示为：

$$\angle G(\sigma) = \sum \angle(\sigma - z_i) - \sum \angle(\sigma - p_j) = (N_{z, \text{right}} - N_{p, \text{right}})\pi$$

其中 $N_{z, \text{right}}$ 和 $N_{p, \text{right}}$ 分别是位于 $\sigma$ 点右侧的实数[零点和极点](@entry_id:177073)的数量。[根轨迹](@entry_id:272958)存在的条件是 $\angle G(\sigma)$ 是 $\pi$ 的奇数倍，这意味着 $(N_{z, \text{right}} - N_{p, \text{right}})$ 必须是一个奇数。由于一个整数加上或减去一个偶数不改变其奇偶性，这个条件等价于 $(N_{z, \text{right}} + N_{p, \text{right}})$ 是一个奇数。

一个特别需要注意的要点是，**复[共轭极点](@entry_id:166341)和零点对[实轴](@entry_id:148276)段的确定没有影响** [@problem_id:1603738]。一对位于 $s = a \pm jb$ 的复[共轭极点](@entry_id:166341)对[实轴](@entry_id:148276)上一点 $\sigma$ 的总[相角](@entry_id:274491)贡献为：
$$-\left( \angle(\sigma - (a+jb)) + \angle(\sigma - (a-jb)) \right)$$
这两个[相角](@entry_id:274491)互为[相反数](@entry_id:151709)，它们的和总是 $0$ 或 $2\pi$ 的整数倍，因此对奇偶性没有影响。

综上，我们得到确定标准[根轨迹](@entry_id:272958)（$K0$）实轴段的**奇偶性法则**：

**实轴上的一个线段属于[根轨迹](@entry_id:272958)，当且仅当该线段右侧的开环实数极点和实数零点的总数为奇数。**

这个法则极大地简化了根轨迹的绘制。例如，对于一个开环系统，其实数极点位于 $s=0, -2, -4$，没有实数零点 [@problem_id:1603737]。
- 在区间 $(-2, 0)$ 内任取一点，其右侧有一个实数极点（在 $s=0$），数量为1（奇数），所以该区间是[根轨迹](@entry_id:272958)的一部分。
- 在区间 $(-4, -2)$ 内任取一点，其右侧有两个实数极点（在 $s=0, -2$），数量为2（偶数），所以该区间不是根轨迹的一部分。
- 在区间 $(-\infty, -4)$ 内任取一点，其右侧有三个实数极点（在 $s=0, -2, -4$），数量为3（奇数），所以该区间是[根轨迹](@entry_id:272958)的一部分。

这个法则还揭示了一个深刻的结构特性：如果一个系统所有[开环极点和零点](@entry_id:276317)都位于[实轴](@entry_id:148276)上，且其实轴[根轨迹](@entry_id:272958)仅包含一个有限闭合区间，例如 $[-4, -2]$，那么该系统实数[极点和零点](@entry_id:262457)的总数必须是偶数 [@problem_id:1603745]。这是因为在区间 $(-\infty, -4)$ 的右侧，包含了所有的实数[极点和零点](@entry_id:262457)。既然该区间不属于[根轨迹](@entry_id:272958)，那么其实数[极点和零点](@entry_id:262457)的总数必须是偶数。反之，如果一个系统实数极点和零点的总数是奇数，那么其实轴根轨迹必定包含一个延伸至 $s = -\infty$ 的无限长线段 [@problem_id:1603716]。

### [实轴上的极点](@entry_id:191960)运动与分离/汇合点

确定了哪些线段属于根轨迹后，下一个问题是[闭环极点](@entry_id:274094)如何在这些线段上运动。[根轨迹](@entry_id:272958)的起点（$K=0$）是[开环极点](@entry_id:272301)，终点（$K \to \infty$）是开环零点或无穷远处。

#### 极点-零点间的运动
如果一个实轴根轨迹段位于一个[开环极点](@entry_id:272301) $p$ 和一个开环零点 $z$ 之间，那么随着增益 $K$ 从 $0$ 增加，[闭环极点](@entry_id:274094)将从[开环极点](@entry_id:272301) $p$ 开始，沿着实轴移动到开环零点 $z$ [@problem_id:1603724]。我们可以通过一个简单的例子来精确描述这个过程。考虑一个系统，其[开环传递函数](@entry_id:276280)为 $L(s) = K \frac{s-z}{s-p}$。其特征方程为 $1+K\frac{s-z}{s-p}=0$，解得[闭环极点](@entry_id:274094)的位置为：
$$s(K) = \frac{p+Kz}{1+K} = \frac{1}{1+K}p + \frac{K}{1+K}z$$
这个表达式是 $p$ 和 $z$ 的一个凸组合。当 $K=0$ 时，$s(0)=p$。当 $K \to \infty$ 时，$s(K) \to z$。因此，[闭环极点](@entry_id:274094)单调地从[极点移动](@entry_id:269721)到零点。

#### 极点-极点间的运动
如果一个[实轴](@entry_id:148276)根轨迹段位于两个相邻的[开环极点](@entry_id:272301) $p_1$ 和 $p_2$ 之间（假设 $p_2 > p_1$），那么当增益 $K$ 从 $0$ 增加时，源于 $p_1$ 的[闭环极点](@entry_id:274094)将向左移动，而源于 $p_2$ 的[闭环极点](@entry_id:274094)将向右移动，彼此相向而行 [@problem_id:1603753]。最终，它们将在该线段上的某一点相遇，然后“分离”进入复平面。这个相遇点被称为**[分离点](@entry_id:265082)（breakaway point）**。

#### 分离点与汇合点的计算
**分离点**和**汇合点（break-in point）**是[根轨迹](@entry_id:272958)上的多重根（即两个或多个[闭环极点](@entry_id:274094)重合的点）。在这些点上，增益 $K$ 作为 $s$ 的函数达到[局部极值](@entry_id:144991)。因此，寻找这些点的条件是：
$$\frac{dK}{ds} = 0$$

计算步骤如下：
1.  从[特征方程](@entry_id:265849) $1+KL(s)=0$ 中，将 $K$ 表示为 $s$ 的函数：$K(s) = -1/L(s)$。
2.  对 $K(s)$ 求关于 $s$ 的导数，并令其等于零：$\frac{dK}{ds} = 0$。
3.  解出该方程的根。这些根是[分离点](@entry_id:265082)或[汇合](@entry_id:148680)点的候选点。
4.  检验每个候选点：一个有效的候选点必须位于根轨迹上，并且对应的增益 $K = K(s)$ 必须是实数且为正。

例如，对于具有[开环极点](@entry_id:272301) $s=0, -2, -4$ 的系统 [@problem_id:1603737]，我们有 $L(s) = \frac{K}{s(s+2)(s+4)}$。
1.  $K(s) = -s(s+2)(s+4) = -(s^3 + 6s^2 + 8s)$。
2.  $\frac{dK}{ds} = -(3s^2 + 12s + 8) = 0$。
3.  解得候选点为 $s = -2 \pm \frac{2}{\sqrt{3}} \approx -0.845, -3.155$。
4.  我们已经知道[根轨迹](@entry_id:272958)存在于 $(-2, 0)$ 和 $(-\infty, -4)$。候选点 $s \approx -0.845$ 位于 $(-2, 0)$ 区间，是有效的[分离点](@entry_id:265082)。而 $s \approx -3.155$ 位于 $(-4, -2)$ 区间，该区间不属于根轨迹，因此它不是一个有效的分离点。

[汇合](@entry_id:148680)点是复[共轭[极](@entry_id:166341)点移动](@entry_id:269721)到[实轴](@entry_id:148276)上相遇的点，通常发生在两个实数开环零点之间。其计算方法与[分离点](@entry_id:265082)完全相同 [@problem_id:1603762]。在某些情况下，我们可能需要利用[分离点](@entry_id:265082)或[汇合](@entry_id:148680)点的位置来反推系统参数 [@problem_id:1603761]。

### 补[根轨迹](@entry_id:272958)（$K  0$）

到目前为止，我们一直假设增益 $K$ 是正的。然而，在某些应用中，考虑负增益（$K  0$）也是有意义的。由负增益产生的[根轨迹](@entry_id:272958)被称为**补[根轨迹](@entry_id:272958)（complementary root locus）**。

对于 $K  0$，我们可以令 $K = -|K|$，其中 $|K|0$。特征方程变为 $1 - |K|G(s) = 0$，即 $G(s) = 1/|K|$。对于实轴上的点 $\sigma$，这意味着 $G(\sigma)$ 必须是一个正实数。

$$G(\sigma)  0$$

这对应的[相角条件](@entry_id:264793)是 $\angle G(\sigma) = 2\ell\pi$。应用与之前相同的逻辑，我们可以推导出补[根轨迹](@entry_id:272958)的奇偶性法则：

**[实轴](@entry_id:148276)上的一个线段属于补根轨迹（$K  0$），当且仅当该线段右侧的开环实数极点和实数零点的总数为偶数（包括零）。**

这意味着，对于任何一个给定的系统，[实轴](@entry_id:148276)上的任何一段要么属于标准[根轨迹](@entry_id:272958)，要么属于补[根轨迹](@entry_id:272958)，二者互补且覆盖整个[实轴](@entry_id:148276)（除了[开环极点和零点](@entry_id:276317)本身）[@problem_id:1603752]。例如，考虑一个[开环传递函数](@entry_id:276280) $G(s) = \frac{(s+2)(s+6)}{(s+1)(s+4)(s+8)}$。
- 在区间 $(-4, -2)$，右侧有两个实数[奇点](@entry_id:137764)（在 $-2$ 的零点和在 $-1$ 的极点），总数为2（偶数）。因此，该段属于补[根轨迹](@entry_id:272958)（$K  0$）。
- 在区间 $(-2, -1)$，右侧有一个实数[奇点](@entry_id:137764)（在 $-1$ 的极点），总数为1（奇数）。因此，该段属于标准根轨迹（$K0$）。

通过掌握这些原理和机制，我们可以精确、高效地确定任何[线性时不变系统](@entry_id:276591)根轨迹的实轴部分，为进一步的[系统分析](@entry_id:263805)和设计奠定坚实的基础。
