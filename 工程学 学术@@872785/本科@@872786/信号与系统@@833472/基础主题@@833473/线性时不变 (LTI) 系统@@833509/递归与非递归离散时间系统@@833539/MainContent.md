## 引言
在[离散时间信号](@entry_id:272771)处理中，系统通过[差分方程](@entry_id:262177)来描述。这些方程的结构——即当前输出是否依赖于过去的输出——将系统清晰地划分为两大阵营：[递归系统](@entry_id:274740)和[非递归系统](@entry_id:272577)。这个看似简单的分类，实际上是理解和设计数字系统的基石。然而，许多学习者常常困惑于这两种结构在系统行为、稳定性及实际应用中究竟意味着什么。本文旨在填补这一知识鸿沟，揭示“递归”与“非递归”背后深刻的理论内涵与工程实践意义。

本文将分三步深入探讨这一主题。在“原理与机制”一章中，我们将建立递归与[非递归系统](@entry_id:272577)的严格定义，并将其与有限冲激响应（FIR）和[无限冲激响应](@entry_id:180862)（IIR）的概念联系起来。接着，在“应用与跨学科联系”一章，我们将展示这些理论如何在数字信号处理、金融建模和控制论等不同领域中发挥作用。最后，通过“动手实践”部分，你将有机会通过具体计算来巩固所学知识。让我们首先从定义和基本原理开始，探索这两类系统的核心差异。

## 原理与机制

在[离散时间信号](@entry_id:272771)处理领域，我们通过数学模型来描述和分析系统。这些模型的核心是差分方程，它精确地定义了输入信号 $x[n]$ 如何被转换为输出信号 $y[n]$。根据[差分方程](@entry_id:262177)的结构，我们可以将[离散时间系统](@entry_id:263935)分为两大基本类别：**递归 (recursive)** 系统和**非递归 (non-recursive)** 系统。这一分类不仅是理论上的划分，更深刻地决定了系统的内在属性、行为特点以及在实际应用中的设计与实现考量。

### 递归与[非递归系统](@entry_id:272577)的定义

一个离散时间系统的分类取决于其当前输出 $y[n]$ 的计算方式。

**[非递归系统](@entry_id:272577) (Non-recursive System)** 的输出 $y[n]$ 仅依赖于当前和/或过去的**输入**值（即 $x[n], x[n-1], x[n-2], \dots$）。其通用[差分方程](@entry_id:262177)形式为：

$$y[n] = \sum_{k=0}^{M} b_k x[n-k]$$

其中 $M$ 是一个有限的非负整数，代表系统“记忆”的最远输入。由于输出不依赖于任何先前的输出值，这种结构也被称为**前馈 (feedforward)** 结构。

**[递归系统](@entry_id:274740) (Recursive System)** 的输出 $y[n]$ 不仅依赖于当前和/或过去的输入值，还依赖于一个或多个**过去**的**输出**值（即 $y[n-1], y[n-2], \dots$）。其通用的[线性常系数差分方程](@entry_id:260895)形式为：

$$\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]$$

为了计算 $y[n]$（假设 $a_0 \neq 0$），我们通常将其整理为：

$$y[n] = -\frac{1}{a_0} \sum_{k=1}^{N} a_k y[n-k] + \frac{1}{a_0} \sum_{k=0}^{M} b_k x[n-k]$$

方程右侧出现的 $y[n-k]$ (其中 $k>0$) 项是[递归系统](@entry_id:274740)的标志。这些项构成了**反馈 (feedback)** 环路，即系统的过去输出被“反馈”回来，参与当前输出的计算。

在判断一个系统是递归还是非递归时，必须仔细检查其最终的、经过代数简化的差分方程 [@problem_id:1747676]。例如，一个系统可能表面上看起来是递归的，如：

$y[n] = 3(x[n] + y[n-1]) - 3y[n-1]$

然而，通过简单的代数展开和化简，我们得到：

$y[n] = 3x[n] + 3y[n-1] - 3y[n-1] = 3x[n]$

最终的表达式表明，$y[n]$ 仅依赖于当前输入 $x[n]$。因此，该系统实际上是一个[非递归系统](@entry_id:272577)。相反，像 $y[n] = x[n] + 0.8y[n-1]$ 或 $y[n] = y[n-3] + x[n-1]$ 这样的系统，由于其对过去输出的依赖无法消除，所以是典型的[递归系统](@entry_id:274740) [@problem_id:1747676] [@problem_id:1747711]。

### 结构与行为：有限冲激响应(FIR)与[无限冲激响应](@entry_id:180862)(IIR)

系统的递归或非递归结构直接决定了其对最基本信号——[单位冲激](@entry_id:272155)信号 $\delta[n]$——的响应行为。一个系统的**冲激响应 (impulse response)** $h[n]$ 是指当输入 $x[n]=\delta[n]$ 时，系统在初始静止状态下的输出。冲激响应的持续时间是区分这两类系统行为的关键。

**[非递归系统](@entry_id:272577)** 总是具有**有限冲激响应 (Finite Impulse Response, FIR)**。这是因为在一个没有反馈的结构中，单个冲激输入的影响只能在系统中“传播”有限的时间。这个时间由差分方程中输入项的最大延迟决定。例如，考虑一个3点[移动平均滤波器](@entry_id:271058) [@problem_id:1747700]：

$$y[n] = \frac{1}{3} (x[n] + x[n-1] + x[n-2])$$

当输入为 $x[n] = \delta[n]$ 时，输出（即冲激响应）为：

$$h[n] = \frac{1}{3} (\delta[n] + \delta[n-1] + \delta[n-2])$$

这个响应仅在 $n=0, 1, 2$ 这三个时刻非零，在所有其他时刻均为零。它的持续时间是有限的。因此，这类系统被称为 **[FIR滤波器](@entry_id:262292)**。任何形如 $y[n] = \sum_{k=0}^{M} d_k x[n-k]$ 的系统都是[FIR滤波器](@entry_id:262292) [@problem_id:1747711] [@problem_id:1747717]。

**[递归系统](@entry_id:274740)** 通常具有**[无限冲激响应](@entry_id:180862) (Infinite Impulse Response, IIR)**。[反馈环](@entry_id:273536)路的存在使得单个冲激的能量可以在系统中持续循环，理论上可以产生无限长的输出。考虑一个简单的一阶[递归系统](@entry_id:274740) [@problem_id:1747700]：

$$y[n] = 0.5 y[n-1] + 0.5 x[n]$$

当输入为 $x[n] = \delta[n]$ 且系统初始静止时 ($y[-1]=0$)，我们可以迭代计算其冲激响应：
$h[0] = 0.5 y[-1] + 0.5 x[0] = 0.5(0) + 0.5(1) = 0.5$
$h[1] = 0.5 h[0] + 0.5 x[1] = 0.5(0.5) + 0.5(0) = 0.25$
$h[2] = 0.5 h[1] + 0.5 x[2] = 0.5(0.25) + 0 = 0.125$
...
$h[n] = 0.5^{n+1} u[n]$

显然，对于所有 $n \ge 0$，该响应都是非零的，并会无限延续下去，尽管其值会趋近于零。因此，这类系统被称为 **[IIR滤波器](@entry_id:273934)**。一个模拟音频回声效果的系统 $y[n] = x[n] + \alpha y[n-D]$ 也是一个典型的IIR系统，其冲激响应由一系列衰减的回声脉冲组成，理论上无限延伸 [@problem_id:1747699]。

对于[线性时不变 (LTI) 系统](@entry_id:178866)，"非递归"与"FIR"、"递归"与"IIR"这两对术语常常可以互换使用。

### 关键属性与实践考量

系统的结构不仅影响其冲激响应，还深刻地决定了其稳定性、实现复杂度和对初始条件的依赖等关键工程属性。

#### 稳定性

**有界输入有界输出 (Bounded-Input, Bounded-Output, BIBO) 稳定性**是衡量一个系统是否实用的首要标准。一个BIBO稳定的系统，对于任何有界的输入信号，其输出信号也必须是有界的。对于[LTI系统](@entry_id:271946)，[BIBO稳定性](@entry_id:157773)的充要条件是其冲激响应 $h[n]$ 绝对可和，即：

$$\sum_{n=-\infty}^{\infty} |h[n]| \lt \infty$$

基于这个条件，我们可以得出关于两类[系统稳定性](@entry_id:273248)的重要结论：

1.  **所有FIR系统都是[无条件稳定](@entry_id:146281)的**。由于FIR系统的冲激响应 $h[n]$ 只有有限个非零项，其[绝对值](@entry_id:147688)之和必然是一个有限数。因此，非递归结构本身就保证了稳定性 [@problem_id:1746815]。

2.  **IIR系统不保证稳定**。由于其冲激响应是无限的，其[绝对值](@entry_id:147688)之和可能收敛也可能发散。IIR系统的稳定性取决于[反馈系数](@entry_id:275731)的大小。例如，对于系统 $y[n] = \alpha y[n-1] + x[n]$，其冲激响应为 $h[n] = \alpha^n u[n]$。其[绝对值](@entry_id:147688)之和为 $\sum_{n=0}^{\infty} |\alpha|^n$。这是一个[几何级数](@entry_id:158490)，只有当 $|\alpha| < 1$ 时它才收敛。如果 $|\alpha| \ge 1$，系统就是不稳定的。在[Z变换](@entry_id:157804)域中，这对应于系统的极点必须严格位于单位圆内。

这种对系数精度的敏感性是IIR系统在实际实现中的一个关键挑战。在[数字信号处理器 (DSP)](@entry_id:748428) 中，系数值必须以有限的位数（例如，定点数）来表示，这会导致**[量化误差](@entry_id:196306)**。考虑一个理想情况下稳定的系统，其系数 $\alpha = 1 - 2^{-10}$ 非常接近1。如果在硬件实现中，用于表示小数部分的位数 $B$ 不足，例如 $B10$，量化后的系数 $\alpha_q$ 可能会被舍入为1。这将导致[系统的极点](@entry_id:261618)恰好位于单位圆上，从而使原本稳定的系统在实际部署中变得不稳定 [@problem_id:1747723]。这个问题在FIR系统中则不存在，因为它们的稳定性不依赖于系数的精确值。

#### [初始条件](@entry_id:152863)与系统“记忆”

FIR系统具有**[有限记忆](@entry_id:136984)**。其输出 $y[n]$ 仅依赖于最近的 $M+1$ 个输入样本。计算 $y[n]$ 只需要知道 $x[n], x[n-1], \dots, x[n-M]$。如果系统从静止状态开始，我们只需假定 $n0$ 时的所有输入均为零。

相比之下，IIR系统具有**无限记忆**。理论上，很久以前的输入仍会通过[反馈环](@entry_id:273536)路对当前输出产生影响（尽管影响可能很小）。在计算上，这意味着为了求得 $y[n]$，我们需要知道过去的输出值 $y[n-1], y[n-2], \dots$。这就引出了**[初始条件](@entry_id:152863)**的重要性。要启动一个递归过程，必须提供一组初始的输出值（例如 $y[-1], y[-2], \dots$）。这些[初始条件](@entry_id:152863)代表了系统在计算开始之前的状态 [@problem_id:1747689]。例如，在计算由 $y_B[n] = 0.5 y_B[n-1] + x[n]$ 描述的系统在 $n=0$ 时的输出时，我们必须知道 $y_B[-1]$ 的值。而对于[非递归系统](@entry_id:272577) $y_A[n] = 0.5 x[n] - 0.25 x[n-1]$，计算 $y_A[0]$ 只需要 $x[0]$ 和 $x[-1]$ 的值。

### 深入探讨：系统组合与[非线性](@entry_id:637147)

尽管基本定义清晰，但在更复杂的场景下，对系统进行分类需要更深入的分析。

#### 系统级联的效应

当多个子系统级联时，整个系统的性质由总的输入-输出关系决定。如果级联中包含至少一个递归子系统，通常整个系统也是递归的。例如，将一个递归的回声产生器与一个非递归的滤波器级联，最终得到的系统[差分方程](@entry_id:262177)仍会包含过去的输出项，因此是递归的 [@problem_id:1747720]。

然而，存在一种特殊但重要的例外情况：**极点-零点相消 (pole-zero cancellation)**。如果一个非递归子系统被精心设计，使其Z变换的零点恰好能抵消掉递归子[系统的极点](@entry_id:261618)，那么整个级联系统可能会变为非递归的。考虑一个由以下两个子系统构成的级联系统 [@problem_id:1747710]：
子系统1 (FIR): $w[n] = x[n] - \alpha x[n-1]$
子系统2 (IIR): $y[n] = \alpha y[n-1] + w[n]$

将第一个方程代入第二个方程，得到：
$y[n] = \alpha y[n-1] + (x[n] - \alpha x[n-1])$
整理后得到：
$y[n] - \alpha y[n-1] = x[n] - \alpha x[n-1]$
从这个方程可以看出，系统的输入-输出关系可以简化为 $y[n] = x[n]$（假设[初始条件](@entry_id:152863)一致）。这是一个最简单的[非递归系统](@entry_id:272577)。尽管其中一个组件是递归的，但其动态特性被前级完全抵消了。这提醒我们，最终分类必须基于从端到端（end-to-end）的整体系统行为。

#### 递归结构与线性无关

最后，必须强调“递归”这一结构属性与“线性”这一系统属性是相互独立的。虽然我们经常在[线性常系数差分方程](@entry_id:260895)的背景下讨论递归，但递归结构本身并不保证系统是线性的。

例如，考虑一个由以下方程描述的控制器 [@problem_id:1747658]：
$$y[n] = 0.8 y[n-1] + \text{sgn}(x[n])$$

其中 $\text{sgn}(\cdot)$ 是[符号函数](@entry_id:167507)。由于 $y[n]$ 的计算依赖于 $y[n-1]$，该系统毫无疑问是递归的。然而，由于[符号函数](@entry_id:167507)的存在，该系统不满足叠加原理（例如，$\text{sgn}(x_1+x_2) \neq \text{sgn}(x_1) + \text{sgn}(x_2)$），因此它是**[非线性](@entry_id:637147)**的。这个例子清晰地表明，一个系统可以是递归的，但同时也是[非线性](@entry_id:637147)的。在分析系统时，必须独立地检验其各项基本属性（线性、[时不变性](@entry_id:198838)、因果性、稳定性）。

总之，递归与非递归的划分是理解离散时间系统的基石。它不仅关系到系统的数学表示，更直接关联到系统的冲激响应长度、稳定性保障、实现复杂度和计算要求等一系列核心的理论与实践问题。
