## 引言
对称性是自然界与数学中最深刻、最普适的指导原则之一。从晶体的完美结构到基本粒子的相互作用，对称性的存在往往意味着更深层次的秩序与和谐。在[微分方程](@entry_id:264184)的领域，对称性同样扮演着至关重要的角色。一个[微分方程](@entry_id:264184)若拥有对称性，便意味着其解的集合在某种变换下保持不变。这种[不变性](@entry_id:140168)不仅揭示了方程的内在结构，更常常是求解方程的关键所在。然而，一个根本性的问题摆在面前：我们如何才能系统地、而非依靠直觉或猜测，找出给定[微分方程](@entry_id:264184)的所有对称性并加以利用呢？

本文旨在深入探讨由挪威数学家 [Sophus Lie](@entry_id:196493) 开创的革命性方法，该方法完美地解答了上述问题。它将[微分方程的对称性](@entry_id:198228)与连续变换群（即[李群](@entry_id:137659)）的理论紧密联系起来，提供了一套强大的算法工具。通过学习这套方法，读者将不再将对称性视为一种偶然的幸运，而是能够主动地去发现、分析和利用它来攻克复杂的[微分方程](@entry_id:264184)。

为实现这一目标，本文将分为三个核心章节：
*   在**原理与机制**一章中，我们将奠定理论基础，详细介绍如何用无穷小生成元来描述对称性，如何通过“延长”算子处理导数的变换，并最终推导出用于寻找对称性的“决定方程”。
*   在**应用与跨学科联系**一章中，我们将展示这些理论在实践中的威力，通过实例说明如何利用对称性进行降阶、寻找[不变解](@entry_id:175378)、建立与物理守恒律的联系，并探讨其在物理、工程等领域的广泛应用。
*   最后，在**动手实践**部分，你将有机会通过解决具体问题来巩固所学知识，亲身体验从一个给定的[微分方程](@entry_id:264184)出发，一步步揭示其对称性并加以利用的全过程。

现在，让我们一同踏上这段旅程，去探索[微分方程](@entry_id:264184)背后隐藏的优雅对称世界。

## 原理与机制

在上一章引言中，我们初步探讨了对称性在理解和[求解微分方程](@entry_id:137471)中的重要作用。本章将深入探讨这些对称性的数学原理和核心机制。我们将学习如何用精确的数学语言描述对称性，如何从给定的[微分方程](@entry_id:264184)中系统地找出其所有对称性，以及这些对称性所蕴含的深刻[代数结构](@entry_id:137052)。这些内容构成了[李群](@entry_id:137659)方法分析[微分方程](@entry_id:264184)的理论基石。

### 对称性与无穷小变换

一个[常微分方程](@entry_id:147024)（ODE）的**对称性**是一种变换，它作用于方程的[自变量](@entry_id:267118)和因变量，但保持方程的[解集](@entry_id:154326)不变。换言之，如果 $y(x)$ 是一个解，那么经过对称变换后得到的函数 $\bar{y}(\bar{x})$ 仍然是原方程（在新的[坐标系](@entry_id:156346)下）的一个解。

考虑一个作用于 $(x,y)$ 平面上的单参数**李群变换**：
$$
\bar{x} = \phi(x, y; \epsilon), \quad \bar{y} = \psi(x, y; \epsilon)
$$
其中 $\epsilon$ 是连续的群参数，$\epsilon=0$ 对应于[恒等变换](@entry_id:264671)，即 $\phi(x, y; 0) = x$ 且 $\psi(x, y; 0) = y$。

对于[微分方程](@entry_id:264184)的研究，直接处理具有任意参数 $\epsilon$ 的有限变换通常很复杂。[李群](@entry_id:137659)理论的精髓在于，我们可以通过分析在[恒等变换](@entry_id:264671)附近（即 $\epsilon \to 0$ 时）的**无穷小变换**来研究整个群的性质。将上述变换在 $\epsilon=0$ 附近作泰勒展开，我们得到：
$$
\bar{x} \approx x + \epsilon \left. \frac{\partial\phi}{\partial\epsilon} \right|_{\epsilon=0} = x + \epsilon \xi(x,y)
$$
$$
\bar{y} \approx y + \epsilon \left. \frac{\partial\psi}{\partial\epsilon} \right|_{\epsilon=0} = y + \epsilon \eta(x,y)
$$
这里的函数 $\xi(x,y)$ 和 $\eta(x,y)$ 定义了无穷小变换的[主部](@entry_id:168896)。它们构成了一个向量场
$$
X = \xi(x,y) \frac{\partial}{\partial x} + \eta(x,y) \frac{\partial}{\partial y}
$$
这个向量场被称为该变换群的**无穷小生成元**。它完全捕捉了群在单位元附近的局部行为，并且通[过积分](@entry_id:753033)该向量场（求解其流），可以重构出整个单参数群。因此，研究一个[微分方程](@entry_id:264184)的连续对称性，等价于研究其[无穷小生成元](@entry_id:270424)。

### [不变性条件](@entry_id:171412)与延长

一个[微分方程](@entry_id:264184)不仅包含变量 $x$ 和 $y$，还包含 $y$ 的各阶导数，如 $y', y'', \dots$。一个对称变换作用在 $(x,y)$ 平面上，必然会诱导出对这些导数的相应变换。为了建立对称性的精确判据，我们必须考虑生成元 $X$ 的作用如何“提升”或**延长（prolong）**到包含导数的空间中。

这个包含自变量、因变量及其各阶导数的空间被称为**射流空间（jet space）**。例如，二阶射流空间的坐标为 $(x, y, y', y'')$。[无穷小生成元](@entry_id:270424) $X$ 在 $n$ 阶射流空间上的延长记为 $X^{(n)}$。

对于一阶ODE $y' = F(x,y)$，我们需要知道 $y'$ 如何在无穷小变换下改变。根据[链式法则](@entry_id:190743)，变换后的导数为 $\bar{y}' = \frac{d\bar{y}}{d\bar{x}}$。利用无穷小展开，我们有：
$$
\frac{d\bar{y}}{d\bar{x}} = \frac{d(y + \epsilon \eta)}{d(x + \epsilon \xi)} = \frac{y' + \epsilon \frac{d\eta}{dx}}{1 + \epsilon \frac{d\xi}{dx}} \approx (y' + \epsilon D_x(\eta))(1 - \epsilon D_x(\xi)) \approx y' + \epsilon [D_x(\eta) - y'D_x(\xi)]
$$
这里，$D_x = \partial_x + y'\partial_y + y''\partial_{y'} + \dots$ 是关于 $x$ 的**[全导数](@entry_id:137587)算子**。它将 $y$ 及其各阶导数都视为 $x$ 的函数。

因此，我们看到 $y'$ 的无穷小变为 $y' + \epsilon \eta^{(1)}$，其中
$$
\eta^{(1)}(x,y,y') = D_x(\eta) - y'D_x(\xi)
$$
$\eta^{(1)}$ 是**一阶延长**的系数。一阶[延长算子](@entry_id:144790)为 $X^{(1)} = X + \eta^{(1)} \partial_{y'}$。ODE $y'-F(x,y)=0$ 的**[不变性条件](@entry_id:171412)**就是其在一阶[延长算子](@entry_id:144790)作用下为零（在方程的解[流形](@entry_id:153038)上）：
$$
X^{(1)}(y' - F(x,y)) \Big|_{y'=F(x,y)} = \eta^{(1)} - \xi F_x - \eta F_y \Big|_{y'=F(x,y)} = 0
$$

这个过程可以递归地进行下去。二阶[延长算子](@entry_id:144790) $X^{(2)}$ 的形式为：
$$
X^{(2)} = X^{(1)} + \eta^{(2)} \partial_{y''} = \xi\partial_x + \eta\partial_y + \eta^{(1)}\partial_{y'} + \eta^{(2)}\partial_{y''}
$$
其中，二阶延长系数 $\eta^{(2)}$ 定义为：
$$
\eta^{(2)}(x,y,y',y'') = D_x(\eta^{(1)}) - y''D_x(\xi)
$$
一个 $n$ 阶ODE $\Delta(x, y, y', \dots, y^{(n)}) = 0$ 的无穷小对称性判据是：
$$
X^{(n)}(\Delta) \Big|_{\Delta=0} = 0
$$
这意味着 $n$ 阶[延长算子](@entry_id:144790)作用于该[微分方程](@entry_id:264184)表达式后，在方程的解所满足的约束下，结果恒等于零。

作为一个具体的计算范例，我们来计算射影生成元 $X = x^2\partial_x + xy\partial_y$ 的二阶延长系数 $\eta^{(xx)}$（这里我们使用 $\eta^{xx}$ 记法代替 $\eta^{(2)}$ 以示清晰）。在此例中，$\xi = x^2$ 且 $\eta = xy$。
首先计算[全导数](@entry_id:137587)：
$D_x(\xi) = D_x(x^2) = \partial_x(x^2) = 2x$
$D_x(\eta) = D_x(xy) = \partial_x(xy) + y'\partial_y(xy) = y + xy'$
接着计算一阶延长系数 $\eta^x$：
$\eta^x = D_x(\eta) - y'D_x(\xi) = (y+xy') - y'(2x) = y - xy'$
最后，我们计算二阶延长系数 $\eta^{xx}$：
$\eta^{xx} = D_x(\eta^x) - y''D_x(\xi) = D_x(y-xy') - y''(2x)$
$D_x(y-xy') = \partial_x(y-xy') + y'\partial_y(y-xy') + y''\partial_{y'}(y-xy') = -y' + y' + y''(-x) = -xy''$
因此，$\eta^{xx} = -xy'' - 2xy'' = -3xy''$。这个结果对于验证该生成元是否为 $y''=0$ 的对称性至关重要 [@problem_id:1101464]。

现在，让我们利用这个机制来完整地验证一个对称性。考虑线性齐次ODE $y'' + 4y = 0$。我们来验证向量场 $X = \cos(2x)\partial_y$ 是否是其对称性生成元 [@problem_id:1101408]。
这里，$\xi(x,y) = 0$ 且 $\eta(x,y) = \cos(2x)$。
1.  计算 $D_x(\xi)$ 和 $D_x(\eta)$：
    $D_x(\xi) = D_x(0) = 0$
    $D_x(\eta) = D_x(\cos(2x)) = \partial_x(\cos(2x)) = -2\sin(2x)$
2.  计算延长系数 $\eta^{(1)}$ 和 $\eta^{(2)}$：
    $\eta^{(1)} = D_x(\eta) - y'D_x(\xi) = -2\sin(2x) - y'(0) = -2\sin(2x)$
    $\eta^{(2)} = D_x(\eta^{(1)}) - y''D_x(\xi) = D_x(-2\sin(2x)) - y''(0) = \partial_x(-2\sin(2x)) = -4\cos(2x)$
3.  构建二阶[延长算子](@entry_id:144790) $X^{(2)}$：
    $X^{(2)} = \xi\partial_x + \eta\partial_y + \eta^{(1)}\partial_{y'} + \eta^{(2)}\partial_{y''} = \cos(2x)\partial_y - 2\sin(2x)\partial_{y'} - 4\cos(2x)\partial_{y''}$
4.  将 $X^{(2)}$ 作用于方程表达式 $\Delta = y'' + 4y$：
    $X^{(2)}(y''+4y) = (\cos(2x)\partial_y - 2\sin(2x)\partial_{y'} - 4\cos(2x)\partial_{y''})(y''+4y)$
    在射流空间中，$x, y, y', y''$ 被视为独立坐标，所以我们分别对各项求偏导：
    - $\cos(2x)\partial_y(y''+4y) = 4\cos(2x)$
    - $-2\sin(2x)\partial_{y'}(y''+4y) = 0$
    - $-4\cos(2x)\partial_{y''}(y''+4y) = -4\cos(2x)(1) = -4\cos(2x)$
    将各项结果相加，得到 $4\cos(2x) + 0 - 4\cos(2x) = 0$。
由于结果为0，[不变性条件](@entry_id:171412) $X^{(2)}(\Delta)|_{\Delta=0}=0$ 得到满足。因此，$X=\cos(2x)\partial_y$ 确实是 $y''+4y=0$ 的一个对称性生成元。值得注意的是，$\cos(2x)$ 本身是该方程的一个解，这揭示了一个深刻的联系：对于线性齐次ODE，任何一个解 $y_s(x)$ 都能生成一个对称性 $X = y_s(x)\partial_y$。

### 决定方程：从头寻找对称性

到目前为止，我们只讨论了如何验证一个*给定*的向量场是否为对称性。然而，[李群](@entry_id:137659)方法的真正威力在于它提供了一种算法，可以*找出*一个给定ODE的所有点对称性。这个过程的核心是求解一组称为**决定方程（determining equations）**的[偏微分方程](@entry_id:141332)。

其基本思想是：[不变性条件](@entry_id:171412) $X^{(n)}(\Delta)|_{\Delta=0}=0$ 必须对ODE的*所有*解都成立。由于我们事先并不知道解的具体形式，我们必须将此条件视为一个关于 $x, y, y', \dots$ 的恒等式。我们将方程中的最[高阶导数](@entry_id:140882)（例如 $y^{(n)}$）用方程本身替换掉，然后将结果视为一个关于低阶导数（例如 $y', \dots, y^{(n-1)}$）的多项式。由于这个多项式必须恒等于零，因此各项导数幂的系数必须分别等于零。

这个过程将[不变性条件](@entry_id:171412)分解为一组关于未知函数 $\xi(x,y)$ 和 $\eta(x,y)$ 的[线性偏微分方程](@entry_id:172517)。这就是决定[方程组](@entry_id:193238)。求解这个[方程组](@entry_id:193238)，就可以得到该ODE所拥有的所有点对称性的无穷小生成元。

以著名的**[贝塞尔方程](@entry_id:164013)**为例 [@problem_id:1101428]：
$$
x^2 y'' + x y' + (x^2 - \nu^2) y = 0 \quad (\nu \neq 0)
$$
我们寻找形如 $X = \xi(x)\partial_x + \alpha(x) y \partial_y$ 的对称性（这是线性方程对称性的一般形式，忽略了来自线性叠加的平凡对称性）。通过建立并分析其决定方程，可以发现，为了使[不变性条件](@entry_id:171412)成立，必须有 $\xi(x)=0$。进一步的分析表明 $\alpha(x)$ 必须是一个常数。这意味着[贝塞尔方程](@entry_id:164013)（对于 $\nu \neq 0$）的非平凡点对称性代数是由单个生成元 $X = y\partial_y$（乘以任意常数）张成的一维[李代数](@entry_id:137954)。这个对称性对应于方程在 $y \to cy$ 变换下的尺度不变性，这与方程是线性和齐次的这一事实是一致的。

### 对称性的李[代数结构](@entry_id:137052)

一个给定的[微分方程](@entry_id:264184)通常不止一个对称性。例如，方程 $y''=0$ 的解是直线 $y=c_1x+c_2$，它在平移、旋转、缩放等多种变换下保持不变。所有这些对称性的无穷小生成元集合构成一个[向量空间](@entry_id:151108)。更有趣的是，这个空间具有一种称为**[李代数](@entry_id:137954)**的[代数结构](@entry_id:137052)。

[李代数](@entry_id:137954)的核心运算是**[李括号](@entry_id:636461)（Lie bracket）**。对于两个向量场 $X_1 = \xi_1\partial_x + \eta_1\partial_y$ 和 $X_2 = \xi_2\partial_x + \eta_2\partial_y$，它们的李括号定义为：
$$
[X_1, X_2] = (X_1(\xi_2) - X_2(\xi_1))\partial_x + (X_1(\eta_2) - X_2(\eta_1))\partial_y
$$
其中 $X_1(\xi_2)$ 表示用算子 $X_1$ 作用于函数 $\xi_2$。[李括号](@entry_id:636461)的几何意义是两个向量场生成的流的交换子在无穷小层面上的体现。

一个根本性的定理指出：如果 $X_1$ 和 $X_2$ 都是同一个[微分方程的对称性](@entry_id:198228)生成元，那么它们的[李括号](@entry_id:636461) $[X_1, X_2]$ 也必然是该方程的对称性生成元。这意味着对称性生成元的集合在[李括号](@entry_id:636461)运算下是封闭的，从而构成一个[李代数](@entry_id:137954)。

让我们通过一个例子来验证这个[闭包性质](@entry_id:136899)。考虑[自由粒子](@entry_id:148748)方程 $y''=0$ 的两个对称性生成元 [@problem_id:1101373]：
$V_1 = y \partial_x$ (伽利略变换)
$V_2 = x^2 \partial_x + xy \partial_y$ ([射影变换](@entry_id:163230))
我们计算它们的李括号 $V_3 = [V_1, V_2]$。
$V_1$ 的分量为 $\xi_1=y, \eta_1=0$。
$V_2$ 的分量为 $\xi_2=x^2, \eta_2=xy$。
计算 $V_3$ 的 $\partial_x$ 分量 $\xi_3 = V_1(\xi_2) - V_2(\xi_1)$：
$V_1(\xi_2) = y \partial_x(x^2) = 2xy$
$V_2(\xi_1) = (x^2 \partial_x + xy \partial_y)(y) = x^2(0) + xy(1) = xy$
所以 $\xi_3 = 2xy - xy = xy$。
类似地可以计算 $\eta_3 = y^2$。于是我们得到新的生成元 $V_3 = xy \partial_x + y^2 \partial_y$，它也是 $y''=0$ 的一个对称性。

认识到对称性构成李代数是非常重要的。例如，许多重要的物理系统，如[谐振子](@entry_id:155622)，其对称性代数是著名的 $\mathfrak{sl}(2,\mathbb{R})$。当这个系统被[非线性](@entry_id:637147)项扰动时，比如在**Ermakov-Pinney方程** $y'' + \omega_0^2 y = c y^{-3}$ 中，我们有时会发现它继承了原[线性系统](@entry_id:147850)的部分或全部对称性结构。事实上，这个方程就拥有一个与[谐振子](@entry_id:155622)相同的 $\mathfrak{sl}(2,\mathbb{R})$ [对称代数](@entry_id:194266)，这为求解该[非线性方程](@entry_id:145852)提供了关键线索 [@problem_id:1101295]。

### 对称性的应用

[李群](@entry_id:137659)方法的最终目标是利用对称性来简化和求解微分方程。以下是一些关键应用。

#### 构造不变方程

对称性理论可以反向使用：给定一个李群或李代数，我们可以构造出所有在该变换下保持不变的[微分方程](@entry_id:264184)。这提供了一种根据第一性原理（即所期望的对称性）来构建物理模型的方法。

例如，我们想知道什么样的[二阶ODE](@entry_id:204212) $y''=F(y')$ 在欧几里得平面上的刚体运动（平移和旋转）下保持不变。这个运动群 $SE(2)$ 由三个生成元张成：$x$方向平移 $V_1 = \partial_x$，$y$方向平移 $V_2 = \partial_y$，以及原点旋转 $V_3 = -y\partial_x + x\partial_y$ [@problem_id:1101318]。
- $V_1$ 和 $V_2$ 的[不变性](@entry_id:140168)要求 $F$ 不能显含 $x$ 和 $y$，这在 $y''=F(y')$ 的形式中已经满足。
- [旋转不变性](@entry_id:137644) $V_3$ 提供了最强的约束。通过计算 $V_3$ 的二阶延长并应用[不变性条件](@entry_id:171412)，我们最终会得到一个关于函数 $F$ 的一阶ODE：
  $$
  (1+y'^2)F'(y') - 3y'F(y')=0
  $$
  求解这个方程得到 $F(y') = C(1+y'^2)^{3/2}$，其中 $C$ 是任意常数。
这个结果有着深刻的几何意义。表达式 $\frac{y''}{(1+y'^2)^{3/2}}$ 正是[平面曲线的曲率](@entry_id:171642) $\kappa$。因此，要求方程在 $SE(2)$ 下不变，等价于要求其解曲线具有恒定的曲率。这些曲线正是直线（$C=0$）和圆（$C \neq 0$）。

同样，对于一阶方程 $y'=F(x,y)$，若要求它在“[双曲旋转](@entry_id:271877)”生成元 $X = x\partial_x - y\partial_y$ 下不变，通过[不变性条件](@entry_id:171412)可以推导出 $F(x,y)$ 必须满足一个[偏微分方程](@entry_id:141332) $xF_x - yF_y = -2F$。这个方程的通解形式为 $F(x,y)=x^{-2}\phi(xy)$，其中 $\phi$ 是任意单变量函数 [@problem_id:1101261]。

#### 整流与典范坐标

对称性的一个核心应用是简化方程。**整流定理（Rectification Theorem）**表明，对于任何非奇异的向量场 $X$，总能找到一个局部坐标系 $(t, s)$，使得在该[坐标系](@entry_id:156346)下 $X$ 的形式变为最简单的平移 $\tilde{X} = \partial_t$。这个新的[坐标系](@entry_id:156346)被称为**典范坐标**。

典范坐标的寻找过程本身就是求解偏微分方程：
- 新的时间坐标 $t(x,y)$ 满足 $X(t) = (\xi\partial_x + \eta\partial_y)t = 1$。
- 新的不变坐标 $s(x,y)$ 满足 $X(s) = (\xi\partial_x + \eta\partial_y)s = 0$。

一旦找到典范坐标，原[微分方程](@entry_id:264184)在 $(t,s)$ [坐标系](@entry_id:156346)下将只显含 $s$ 和它的导数，因为方程在 $t$ 方向是平移不变的。这通常能将方程的阶数降低一阶。

让我们考虑如何[整流](@entry_id:197363)二维尺度缩放生成元 $X = x\partial_x + y\partial_y$ [@problem_id:1101283]。我们寻找一个典范坐标 $t(x,y)$ 满足 $X(t)=1$。同时，我们注意到该生成元与[旋转生成元](@entry_id:154292) $R = -y\partial_x + x\partial_y$ 的[李括号](@entry_id:636461)为零，$[X,R]=0$，这意味着尺度缩放和旋转是相容的对称性。因此，我们可以寻找一个同时满足[旋转不变性](@entry_id:137644)的典范坐标，即 $R(t)=0$。
- 条件 $R(t) = (-y\partial_x + x\partial_y)t = 0$ 意味着 $t$ 必须是[旋转不变量](@entry_id:170459)，即 $t$ 必须是半径 $r = \sqrt{x^2+y^2}$ 的函数，可设 $t = f(r)$。
- 将此形式代入条件 $X(t)=1$ 中，我们得到 $X(f(r)) = (x\partial_x+y\partial_y)f(r) = r f'(r) = 1$。
- 求解这个简单的ODE得到 $f(r) = \ln(r) + C$。
- 因此，$t(x,y) = \ln(\sqrt{x^2+y^2}) + C$。如果再给定一个[归一化条件](@entry_id:156486)，比如 $t(1, \sqrt{3})=1$，我们就可以确定常数 $C$，得到唯一的典范坐标 $t(x,y) = \ln(\frac{\sqrt{x^2+y^2}}{2}) + 1$。

#### [一阶常微分方程组](@entry_id:635184)的对称性

[李群](@entry_id:137659)方法同样适用于一阶ODE系统。考虑一个[自治系统](@entry_id:173841) $\dot{\mathbf{x}} = \mathbf{F}(\mathbf{x})$，例如：
$$
\begin{cases}
\dot{x} = F(x,y) \\
\dot{y} = G(x,y)
\end{cases}
$$
这个系统本身可以看作是由向量场 $V = F(x,y)\partial_x + G(x,y)\partial_y$ 生成的相流。一个[无穷小生成元](@entry_id:270424) $X = \xi(x,y)\partial_x + \eta(x,y)\partial_y$ 是该系统的对称性，当且仅当它生成的变换将系统的[轨道](@entry_id:137151)映为系统的[轨道](@entry_id:137151)。这在无穷小层面上的条件是 $X$ 生成的流与 $V$ 生成的流相交换，即它们的李括号为零：
$$
[X, V] = 0
$$
将这个[李括号](@entry_id:636461)展开，就得到关于 $\xi, \eta$ 和 $F, G$ 的一组[偏微分方程](@entry_id:141332)，这便是系统情况下的[不变性条件](@entry_id:171412)。

以简谐振子系统为例 [@problem_id:1101276]：
$$
\begin{cases}
\dot{x} = y \\
\dot{y} = -\omega^2 x
\end{cases}
$$
这里系统的向量场是 $V = y\partial_x - \omega^2x\partial_y$。假设我们有一个待定的对称性生成元 $X = \frac{y}{\omega} \partial_x + \alpha \omega x \partial_y$，其中 $\alpha$ 是一个未知常数。通过计算李括号 $[X, V]$ 并使其等于零，我们可以解出 $\alpha$ 的值。计算结果表明，只有当 $\alpha=-1$ 时，李括号才为零。这说明 $X = \frac{y}{\omega}\partial_x - \omega x\partial_y$ 是简谐振子系统的一个对称性生成元。这个对称性在物理上与[能量守恒](@entry_id:140514)相关。

本章我们详细阐述了[李群分析](@entry_id:174076)方法的核心原理与机制，从无穷小生成元、延长理论、[不变性条件](@entry_id:171412)，到决定方程和李[代数结构](@entry_id:137052)。我们还通过一系列例子展示了这些原理在验证对称性、寻找对称性、构造不变方程以及简化方程等方面的具体应用。掌握了这些工具，我们就能更深刻地洞察[微分方程](@entry_id:264184)的内在结构，并为求解它们开辟新的道路。
