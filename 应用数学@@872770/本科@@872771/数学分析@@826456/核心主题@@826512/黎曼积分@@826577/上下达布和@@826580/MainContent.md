## 引言
计算曲线下方所围成的面积是微积分诞生之初的核心问题之一。虽然直观上我们可以通过不断切分区域并用矩形逼近来获得答案，但如何将这一朴素思想转化为严谨的数学语言，构成了数学分析发展史上的一个关键挑战。法国数学家 Jean-Gaston Darboux 提出的上、下[达布和](@entry_id:137075)，正是为了解决这一问题而设计的精妙工具。它通过系统地“包夹”真实面积，为积分的定义提供了坚实的基础。

本文旨在带领读者深入理解[达布和](@entry_id:137075)的理论及其应用。我们将从最基本的概念出发，解决从直观逼近到形式化定义的知识鸿沟。通过本文的学习，你将掌握：
- **原理与机制**：我们将首先详细介绍[区间划分](@entry_id:264619)、上[下确界](@entry_id:140118)的概念，并在此基础上严格定义[上达布和](@entry_id:142308)与下[达布和](@entry_id:137075)。随后，我们将探讨其核心性质，并最终引出判断函数是否可积的著名判据。
- **应用与跨学科联系**：在理论基础之上，我们将展示[达布和](@entry_id:137075)如何被用作强大的分析工具，以证明各[类函数](@entry_id:146970)（如单调函数和[连续函数](@entry_id:137361)）的[可积性](@entry_id:142415)，并探讨[可积函数](@entry_id:191199)类在代数运算下的封闭性。同时，我们也将审视[黎曼积分的局限性](@entry_id:186923)，并一窥其与数值分析、[测度论](@entry_id:139744)等领域的深刻联系。
- **实践练习**：理论的学习需要通过实践来巩固。最后，我们将提供一系列精心设计的练习题，引导你亲手计算[达布和](@entry_id:137075)，并运用[达布判据](@entry_id:159776)解决具体问题，从而将抽象知识内化为你的分析技能。

让我们一同开启这段从基本原理到深刻应用的数学之旅，真正领会[达布和](@entry_id:137075)在构建现代积分理论中的基石作用。

## 原理与机制

在上一章引言中，我们探讨了计算[曲线下面积](@entry_id:169174)这一古老问题的直观思想。为了将这一思想形式化，数学家们发展了多种积分理论，其中最基础和最直观的便是由法国数学家 Jean-Gaston Darboux 提出的[达布积分](@entry_id:142929)。该理论的核心工具是[上达布和](@entry_id:142308)与下[达布和](@entry_id:137075)，它们通过一系列矩形面积来“包夹”真实的面积。本章将深入探讨[达布和](@entry_id:137075)的定义、基本性质及其在判断函数可积性中的核心作用。

### [达布和](@entry_id:137075)的定义

为了精确地定义曲线下的面积，我们首先需要一种系统地分割区间的方法。

考虑一个闭区间 $[a, b]$，其中 $a  b$。区间 $[a, b]$ 的一个**划分 (partition)** $P$ 是一个有限的点集 $\{x_0, x_1, \dots, x_n\}$，满足 $a = x_0  x_1  \dots  x_n = b$。这些点将区间 $[a, b]$ 分割成 $n$ 个子区间 $[x_{i-1}, x_i]$，其中 $i=1, 2, \dots, n$。第 $i$ 个子区间的长度记为 $\Delta x_i = x_i - x_{i-1}$。所有子区间长度的总和等于整个区间的长度：$\sum_{i=1}^n \Delta x_i = x_n - x_0 = b-a$。

通常，我们要求划分中的点是严格递增的。但我们也可以考虑一种更广义的划分，其中点只需满足 $a = x_0 \le x_1 \le \dots \le x_n = b$。在这种情况下，如果某个 $x_i = x_{i-1}$，那么对应的子区间 $[x_{i-1}, x_i]$ 的长度 $\Delta x_i$ 为零。正如我们将看到的，这种长度为零的子区间对[达布和](@entry_id:137075)的贡献也为零，因此在计算中不会产生影响 [@problem_id:2334074]。

现在，考虑一个定义在 $[a, b]$ 上的**[有界函数](@entry_id:176803) (bounded function)** $f$。有界性保证了函数在任何子区间上的值不会趋于无穷。对于由划分 $P$ 产生的每一个子区间 $I_i = [x_{i-1}, x_i]$，我们定义：

-   $m_i = \inf\{f(x) \mid x \in [x_{i-1}, x_i]\}$，即 $f$ 在该子区间上的**[下确界](@entry_id:140118) (infimum)**。
-   $M_i = \sup\{f(x) \mid x \in [x_{i-1}, x_i]\}$，即 $f$ 在该子区间上的**[上确界](@entry_id:140512) (supremum)**。

[下确界](@entry_id:140118) $m_i$ 代表了可以构造的、完全位于曲线下方的矩形的最大可能高度，而上确界 $M_i$ 则代表了完全覆盖该段曲线的矩形的最小可能高度。基于此，我们定义：

-   函数 $f$ 关于划分 $P$ 的**下[达布和](@entry_id:137075) (Lower Darboux Sum)** 为：
    $$L(f, P) = \sum_{i=1}^{n} m_i \Delta x_i$$

-   函数 $f$ 关于划分 $P$ 的**[上达布和](@entry_id:142308) (Upper Darboux Sum)** 为：
    $$U(f, P) = \sum_{i=1}^{n} M_i \Delta x_i$$

直观上，$L(f, P)$ 是所有内接矩形面积之和，它给出了[曲线下面积](@entry_id:169174)的一个下界估计。相反，$U(f, P)$ 是所有外接矩形面积之和，给出了面积的一个[上界](@entry_id:274738)估计。

让我们从最简单的情形开始。考虑一个常值函数 $f(x)=c$ 定义在区间 $[a,b]$ 上。对于任意一个划分 $P=\{x_0, \dots, x_n\}$，在任何子区间 $[x_{i-1}, x_i]$ 上，函数的[上确界](@entry_id:140512)和下确界都是 $c$ 本身，即 $M_i = m_i = c$。因此，[上达布和](@entry_id:142308)与下[达布和](@entry_id:137075)分别为：
$$U(f, P) = \sum_{i=1}^n c \Delta x_i = c \sum_{i=1}^n \Delta x_i = c(b-a)$$
$$L(f, P) = \sum_{i=1}^n c \Delta x_i = c \sum_{i=1}^n \Delta x_i = c(b-a)$$
这表明，对于常值函数，无论我们如何划分区间，其[上达布和](@entry_id:142308)与下[达布和](@entry_id:137075)总是相等的，并且等于我们直观预期的矩形面积 [@problem_id:1344397]。

### [达布和](@entry_id:137075)的基本性质

[达布和](@entry_id:137075)具有一系列至关重要的性质，这些性质是建立积分理论的基石。

#### 基本不等式

对于任意一个[有界函数](@entry_id:176803) $f$ 和任意一个划分 $P$，显然在每个子区间上都有 $m_i \le M_i$。由于 $\Delta x_i \ge 0$，这意味着 $m_i \Delta x_i \le M_i \Delta x_i$，将所有子区间加总可得：
$$L(f, P) \le U(f, P)$$

此外，设 $m = \inf_{x \in [a,b]} f(x)$ 和 $M = \sup_{x \in [a,b]} f(x)$ 分别是 $f$ 在整个区间 $[a,b]$ 上的全局[下确界](@entry_id:140118)和上确界。那么对于任何子区间 $I_i$，我们都有 $m \le m_i \le M_i \le M$。因此，对于任意划分 $P$：
$$L(f,P) = \sum m_i \Delta x_i \ge \sum m \Delta x_i = m \sum \Delta x_i = m(b-a)$$
$$U(f,P) = \sum M_i \Delta x_i \le \sum M \Delta x_i = M \sum \Delta x_i = M(b-a)$$
综上，我们得到了第一个核心关系：对于任意划分 $P$，总有
$$m(b-a) \le L(f, P) \le U(f, P) \le M(b-a)$$
这个性质为所有的[达布和](@entry_id:137075)提供了一个统一的界限 [@problem_id:1344394]。

#### [划分的加细](@entry_id:144047)

一个更有力的性质涉及到改变划分。如果一个划分 $P^*$ 包含了划分 $P$ 的所有点，那么我们称 $P^*$ 是 $P$ 的一个**加细 (refinement)**。例如，在划分 $P$ 的某个子区间 $[x_{k-1}, x_k]$ 中增加一个新点 $c$，即 $x_{k-1}  c  x_k$，就得到了一个加细划分 $P^*$。

加细划分会如何影响[达布和](@entry_id:137075)？直观地看，增加分割点会让矩形更好地拟合曲线，从而下和应该增大，上和应该减小。

**定理：** 若 $P^*$ 是 $P$ 的一个加细，则 $L(f, P) \le L(f, P^*)$ 且 $U(f, P^*) \le U(f, P)$。

**证明思路：** 我们只需考虑增加一个点的情形。假设 $P^* = P \cup \{c\}$，其中 $x_{k-1}  c  x_k$。划分 $P^*$ 与 $P$ 的区别仅在于子区间 $[x_{k-1}, x_k]$ 被分成了两个更小的子区间 $[x_{k-1}, c]$ 和 $[c, x_k]$。设 $f$ 在这两个新子区间上的[上确界](@entry_id:140512)分别为 $M'_1$ 和 $M'_2$。由于这两个区间都是原区间 $[x_{k-1}, x_k]$ 的[子集](@entry_id:261956)，它们的上确界必然不会超过原区间的上确界 $M_k$，即 $M'_1 \le M_k$ 且 $M'_2 \le M_k$。因此，对上和的贡献从 $M_k(x_k - x_{k-1})$ 变成了 $M'_1(c - x_{k-1}) + M'_2(x_k - c)$。
$$M'_1(c - x_{k-1}) + M'_2(x_k - c) \le M_k(c - x_{k-1}) + M_k(x_k - c) = M_k(x_k - x_{k-1})$$
由于其他子区间的贡献不变，所以 $U(f, P^*) \le U(f, P)$。对下和的证明是类似的。

例如，对于函数 $f(x)=x^2$ 在 $[0, 2]$ 上，考虑划分 $P=\{0, 1, 2\}$ 和其加细 $P^*=\{0, 1/2, 1, 2\}$。我们可以精确计算出 $U(f,P)=5$ 和 $U(f,P^*) = 37/8 = 4.625$，清晰地看到上和减小了 [@problem_id:2334060]。通过对 $f(x) = \sin(\pi x/2)$ 在 $[0,1]$ 上的计算，我们也可以验证 $L(f,P) \le L(f,Q)$ 和 $U(f,Q) \le U(f,P)$，其中 $Q$ 是 $P$ 的加细 [@problem_id:2334104]。

加细的性质引出了一个至关重要的推论。对于任意两个划分 $P_1$ 和 $P_2$，我们可以构造它们的**[公共加细](@entry_id:146567) (common refinement)** $P_{12} = P_1 \cup P_2$。根据加细定理，我们有：
$$L(f, P_1) \le L(f, P_{12}) \le U(f, P_{12}) \le U(f, P_2)$$
这表明，**任意一个下和总是不大于任意一个上和**，无论这两个和是基于哪个划分计算的。这个性质是定义[达布积分](@entry_id:142929)的前提，因为它确保了所有下和构成的集合与所有上和构成的集合是分离的 [@problem_id:1314885]。

#### [达布和](@entry_id:137075)的代数性质

[达布和](@entry_id:137075)在函数的代数运算下表现出一些有用的（虽然不完全是线性的）性质。

1.  **函数的加法：** 对于两个[有界函数](@entry_id:176803) $f$ 和 $g$，以及任意划分 $P$，我们有：
    $$L(f, P) + L(g, P) \le L(f+g, P) \quad \text{以及} \quad U(f+g, P) \le U(f, P) + U(g, P)$$
    这是因为在任何子区间上，两个函数[下确界](@entry_id:140118)之和总是不大于它们和的[下确界](@entry_id:140118)，即 $\inf(f) + \inf(g) \le \inf(f+g)$。需要注意的是，这里通常是严格小于号。例如，在特定区间上，一个函数的最小值点可能与另一个函数的最小值点不同，导致和函数在任何一点的值都大于两个函数在各自最小值点的取值之和 [@problem_id:2334069]。

2.  **函数的数乘：** 设 $g(x) = c f(x)$，其中 $c$ 为常数。
    -   如果 $c \ge 0$，则 $\sup(cf) = c \sup(f)$ 且 $\inf(cf) = c \inf(f)$。因此，$U(cf, P) = c U(f, P)$ 且 $L(cf, P) = c L(f, P)$。
    -   如果 $c  0$，情况则发生逆转。乘以一个负数会使[上确界](@entry_id:140512)变为[下确界](@entry_id:140118)，反之亦然。即 $\sup(cf) = c \inf(f)$ 且 $\inf(cf) = c \sup(f)$。因此，$U(cf, P) = c L(f, P)$ 且 $L(cf, P) = c U(f, P)$ [@problem_id:2334096] [@problem_id:1344414]。

3.  **函数的平移：**
    -   **垂直平移：** 若 $g(x) = f(x) + C$，则每个子区间的上确界和[下确界](@entry_id:140118)都会增加 $C$。因此，$U(g, P) = U(f, P) + C(b-a)$ 且 $L(g, P) = L(f, P) + C(b-a)$ [@problem_id:2334088]。
    -   **水平平移：** 若 $g(x) = f(x-c)$ 定义在区间 $[a+c, b+c]$ 上，并取一个相应的平移划分 $P' = \{x_i+c\}$，则每个子区间的长度不变，且 $g$ 在 $[x_{i-1}+c, x_i+c]$ 上的值域与 $f$ 在 $[x_{i-1}, x_i]$ 上的值域完全相同。因此，$L(g, P') = L(f, P)$ 且 $U(g, P') = U(f, P)$。这体现了面积的平移不变性 [@problem_id:2334043]。

4.  **函数的[序关系](@entry_id:138937)：** 如果对所有 $x \in [a,b]$ 都有 $f(x) \le g(x)$，那么在任何子区间上都有 $\sup(f) \le \sup(g)$ 和 $\inf(f) \le \inf(g)$。因此，对于任意划分 $P$，我们有 $L(f,P) \le L(g,P)$ 和 $U(f,P) \le U(g,P)$ [@problem_id:1344428]。

### 上积分、下积分与可积性

我们已经看到，所有下和的集合 $\{L(f, P) \mid P \text{ 是划分}\}$ 有一个[上界](@entry_id:274738)（例如 $M(b-a)$），而所有上和的集合 $\{U(f, P) \mid P \text{ 是划分}\}$ 有一个下界（例如 $m(b-a)$）。根据[实数的完备性](@entry_id:143849)公理，这两个集合分别存在上确界和下确界。

我们定义：
-   函数 $f$ 在 $[a,b]$ 上的**下[达布积分](@entry_id:142929) (Lower Darboux Integral)** 为：
    $$\underline{\int_a^b} f(x) \,dx = \sup_{P} \{L(f, P)\}$$
-   函数 $f$ 在 $[a,b]$ 上的**上[达布积分](@entry_id:142929) (Upper Darboux Integral)** 为：
    $$\overline{\int_a^b} f(x) \,dx = \inf_{P} \{U(f, P)\}$$

下[达布积分](@entry_id:142929)可以被看作是所有内接矩形面积和能达到的最大值，而上[达布积分](@entry_id:142929)则是所有外接矩形面积和能达到的最小值。我们已经证明 $L(f,P_1) \le U(f,P_2)$，这意味着下积分永远不会超过上积分：
$$\underline{\int_a^b} f(x) \,dx \le \overline{\int_a^b} f(x) \,dx$$

当这两个值相等时，我们说函数 $f$ 在 $[a,b]$ 上是**达布可积的 (Darboux integrable)**（这与黎曼可积是等价的）。这个公共的值就被定义为 $f$ 在 $[a,b]$ 上的**[达布积分](@entry_id:142929) (Darboux Integral)**，记为 $\int_a^b f(x) \,dx$。

这个定义引出了一个判断函数是否可积的强大判据。

**[黎曼可积性判据](@entry_id:161334) (Riemann's Criterion for Integrability):** 一个[有界函数](@entry_id:176803) $f$ 在 $[a, b]$ 上可积的充分必要条件是：对于任意给定的 $\epsilon > 0$，都存在一个划分 $P$，使得
$$U(f, P) - L(f, P)  \epsilon$$

这个差值 $U(f, P) - L(f, P) = \sum_{i=1}^n (M_i - m_i) \Delta x_i$ 有着重要的几何意义。$M_i - m_i$ 是函数在第 $i$ 个子区间上的**振幅 (oscillation)**，记为 $\omega_i$。因此，这个差值是所有“不确定区域”矩形条的面积之和。[可积性](@entry_id:142415)意味着我们可以通过选择足够精细的划分，使得这些不确定区域的总面积变得任意小。我们也可以定义一个分段常数函数，即**划分-振幅函数 (partition-oscillation function)** $\Omega_{f,P}(x)$，它的值在每个子区间 $[x_{k-1}, x_k)$ 上等于 $\omega_k$。那么 $U(f,P) - L(f,P)$ 就等于 $\Omega_{f,P}(x)$ 在划分 $P$ 下的上和 [@problem_id:2334054]。

### 可积性的条件与实例

哪些函数是可积的？[黎曼判据](@entry_id:142731)为我们提供了判断的工具。

#### 单调与[连续函数](@entry_id:137361)

两类非常重要的[可积函数](@entry_id:191199)是单调函数和[连续函数](@entry_id:137361)。

对于一个在 $[a,b]$ 上的**单调递增函数**，在任何子区间 $[x_{i-1}, x_i]$ 上，其最小值在左端点 $x_{i-1}$ 处取得，最大值在右端点 $x_i$ 处取得。即 $m_i = f(x_{i-1})$ 且 $M_i = f(x_i)$。如果我们采用一个将区间 $n$ 等分的均匀划分，则每个子区间长度为 $\Delta x = (b-a)/n$。此时，[上和与下和](@entry_id:146229)之差为：
$$U(f, P_n) - L(f, P_n) = \sum_{i=1}^n (f(x_i) - f(x_{i-1})) \Delta x = \frac{b-a}{n} \sum_{i=1}^n (f(x_i) - f(x_{i-1}))$$
这是一个伸缩求和（telescoping sum），其结果为 $\frac{b-a}{n} (f(x_n) - f(x_0)) = \frac{b-a}{n}(f(b)-f(a))$ [@problem_id:2334108]。当 $n \to \infty$ 时，这个差值趋向于0。因此，任何单调函数都是可积的。

对于**[连续函数](@entry_id:137361)**，其[可积性](@entry_id:142415)源于在[闭区间](@entry_id:136474)上的**[一致连续性](@entry_id:140948) (uniform continuity)**。一致连续性保证了只要子区间足够短，函数在其中的振幅就可以被控制得任意小。具体来说，对于任意 $\epsilon > 0$，我们可以选择一个划分 $P$，使其最长的子区间长度（称为划分的**网格尺寸 (mesh)**，记为 $\|P\|$）足够小，使得每个子区间上的振幅 $M_i - m_i  \frac{\epsilon}{b-a}$。这样一来，$U(f, P) - L(f, P)  \sum \frac{\epsilon}{b-a} \Delta x_i = \epsilon$。因此，所有在闭区间上的[连续函数](@entry_id:137361)都是可积的。

[连续函数的可积性](@entry_id:141937)有一个深刻的推论：如果一个非负[连续函数](@entry_id:137361) $f$ 的下[达布和](@entry_id:137075)对于**所有**划分都为0，那么这个函数必然是零函数 $f(x)=0$。因为如果存在某点 $x_0$ 使得 $f(x_0)>0$，那么根据连续性，在 $x_0$ 附近的一个小区间上 $f(x)$ 恒大于一个正数，这会导致在该区间上计算的下和项大于零，与假设矛盾 [@problem_id:2334048]。

#### [函数光滑性](@entry_id:161935)与收敛速度

对于[可积函数](@entry_id:191199)，一个自然的问题是，当划分越来越细时，$U(f,P)-L(f,P)$ 收敛到0的速度有多快？这个速度与函数的“光滑”程度密切相关。

-   如果函数 $f$ 满足**Lipschitz连续**条件，即存在常数 $K$ 使得 $|f(x)-f(y)| \le K|x-y|$ 对所有 $x,y$ 成立，那么在任何子区间上 $M_i - m_i \le K \Delta x_i$。这可以导出 $U(f, P) - L(f, P) \le K(b-a) \|P\|$ [@problem_id:1344405]。[收敛速度](@entry_id:636873)与网格尺寸成正比。

-   如果函数满足更弱的 **Hölder连续**条件，即 $|f(x)-f(y)| \le C|x-y|^\alpha$（其中 $0  \alpha  1$），则可以证明 $U(f, P) - L(f, P) \le C(b-a) \|P\|^\alpha$ [@problem_id:2334089]。

这些结果表明，函数越光滑（$\alpha=1$ 对应Lipschitz，比 $\alpha1$ 的Hölder更光滑），[达布和](@entry_id:137075)之差向零收敛的速度就越快。

#### [不连续函数](@entry_id:143848)的世界

函数的积分理论中最有趣的部分在于处理[不连续函数](@entry_id:143848)。

一个典型的**不[可积函数](@entry_id:191199)**是**[狄利克雷函数](@entry_id:140770) (Dirichlet function)**，它在有理数点取值为一个常数 $k_1$，在无理数点取值为另一个常数 $k_2$（假设 $k_1 > k_2$）。由于[有理数和无理数](@entry_id:173349)在[实数轴](@entry_id:147286)上都是稠密的，所以在任何子区间 $[x_{i-1}, x_i]$ 内，都既有有理数点又有无理数点。因此，对于任意划分 $P$，$m_i=k_2$ 且 $M_i=k_1$。这导致：
$$L(f, P) = k_2(b-a) \quad \text{和} \quad U(f, P) = k_1(b-a)$$
由于这对于所有划分都成立，我们得到下积分为 $k_2(b-a)$，上积分为 $k_1(b-a)$。因为 $k_1 \ne k_2$，上下积分不相等，所以[狄利克雷函数](@entry_id:140770)不可积 [@problem_id:2334046]。一个更复杂的变体，如在[有理点](@entry_id:195164)取 $\sin(\pi x/2)$，在无理点取 $\cos(\pi x/2)$，也可以通过分析其上下[包络函数](@entry_id:749028) $\max\{\sin, \cos\}$ 和 $\min\{\sin, \cos\}$ 的积分来确定其上下积分，并证明其不[可积性](@entry_id:142415) [@problem_id:2334053]。

然而，并非所有[不连续函数](@entry_id:143848)都不可积。

-   **有限个不连续点：** 如果一个函数仅在有限个点上不连续，那么它是可积的。我们可以通过在这些不连续点周围建立非常窄的子区间来“隔离”它们的影响。这些窄区间的总长度可以任意小，因此它们对 $U-L$ 的贡献可以被控制得任意小。

-   **可数个不连续点：** 事实上，即使不连续点是可数无限的，函数也可能是可积的。一个著名的例子是**托马函数 (Thomae's function)**，它在无理点取0，在有理数 $p/q$（最简形式）处取 $1/q$。对于此函数，任何包含无理数的子区间的[下确界](@entry_id:140118)都是0，因此 $L(f,P)=0$ 对所有划分成立。可以证明，上和也可以被做得任意小，因此该函数可积且积分为0。

-   **一个不可积的变体：** 考虑一个类似托马函数的构造，但规定在无理数点函数值为 $1/2$ [@problem_id:2334063]。在这种情况下，任何子区间的[下确界](@entry_id:140118)仍然为0（因为可以找到分母任意大的有理数），所以下积分为0。然而，任何子区间都包含无理数，所以[上确界](@entry_id:140512)至少为 $1/2$。这导致上积分至少为 $1/2(b-a)$，函数不可积。

-   **一个不可数不连续点的[可积函数](@entry_id:191199)：** [康托集](@entry_id:141903) $C$ 是一个结构奇特的集合，它是不可数的，但其“长度”（[勒贝格测度](@entry_id:139781)）为0。考虑其特征函数 $f(x)$，当 $x \in C$ 时为1，否则为0。
    -   下积分：[康托集](@entry_id:141903)是无处稠密的，意味着任何子区间都包含不属于[康托集](@entry_id:141903)的点。因此，任何子区间的下确界 $m_i$ 都是0，导致 $L(f,P)=0$ 对所有划分成立。所以下积分为0。
    -   上积分：康托集可以被有限个总长度任意小的区间覆盖。例如，在[康托集构造](@entry_id:152967)的第 $k$ 步，集合 $C_k$ 的总长度为 $(2/3)^k$。我们可以构造一个划分 $P_k$，其子区间要么完全在 $C_k$ 内，要么与 $C_k$ 不交。对于这个划分，$U(f,P_k)$ 就等于 $C_k$ 的总长度 $(2/3)^k$。当 $k \to \infty$ 时，这个值趋于0。因此上积分也是0 [@problem_id:2334111]。
    由于上下积分相等且都为0，所以[康托集](@entry_id:141903)的特征函数是可积的，其积分为0。

这个例子揭示了一个深刻的定理（[勒贝格可积性](@entry_id:195349)定理）：一个[有界函数](@entry_id:176803)在闭区间上黎曼可积的充要条件是其[不连续点集](@entry_id:160308)是一个[零测集](@entry_id:157694)。

最后，即使一个函数处处可微，其可积性的分析也可能非常微妙。函数 $f(x)=x^2\sin(1/x)$（并定义$f(0)=0$）在 $x=0$ 处可微，但其导数在 $x=0$ 附近无界[振荡](@entry_id:267781)。分析[达布和](@entry_id:137075)在包含原点的子区间上的行为，可以深入了解这种[振荡](@entry_id:267781)如何影响[积分的收敛](@entry_id:187300)性 [@problem_id:2334042]。

总之，[达布和](@entry_id:137075)为我们提供了一个强大而直观的框架，不仅用以定义积分，更用以探索函数的[可积性](@entry_id:142415)与函数本身的分析性质（如连续性、[单调性](@entry_id:143760)、光滑度及不连续点的结构）之间的深刻联系。
