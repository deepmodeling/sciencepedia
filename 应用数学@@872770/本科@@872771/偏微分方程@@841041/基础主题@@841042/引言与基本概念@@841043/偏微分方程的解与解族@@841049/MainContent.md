## 原理与机制

在介绍性章节中，我们了解了[偏微分方程](@entry_id:141332) (PDE) 是描述物理世界中各种现象的基本数学语言。现在，我们将深入探讨这些方程的核心：它们的解。本章的目标是系统地阐述什么是[偏微分方程](@entry_id:141332)的解，如何对这些方程进行分类，以及这些分类如何决定其解的结构和性质。我们将从最基本的概念出发，逐步建立一个分析框架，使我们能够理解从波动、热传导到静电场的各种现象背后的数学原理。

### 什么是[偏微分方程](@entry_id:141332)的解？

一个函数被称为某个[偏微分方程](@entry_id:141332)的**解 (solution)**，前提是该函数及其所有出现在方程中的偏导数都在其定义域内存在，并且当将该函数及其导数代入方程时，方程恒成立。从本质上讲，解是一个满足特定[微分](@entry_id:158718)约束的函数。

验证一个给定的函数是否为解，是最直接理解这一概念的方式。这个过程通常涉及两步：计算函数的相关偏导数，然后将它们代入原方程，检验等式是否成立。

让我们通过一个在物理学中至关重要的例子来阐明这一点：一维[热传导方程](@entry_id:194763) $u_t = k u_{xx}$，其中 $u(t,x)$ 代表在时间 $t$ 和位置 $x$ 的温度，$k$ 是正常数，称为[热扩散](@entry_id:148740)系数。考虑一个[高斯函数](@entry_id:261394)形式：

$u(t,x) = \frac{A}{\sqrt{t}} \exp\left(-\frac{x^2}{Bt}\right)$

其中 $A$ 和 $B$ 是常数。这种形式的函数在描述[扩散](@entry_id:141445)现象中起着基础性的作用。我们的任务是，如果这个函数是[热方程](@entry_id:144435)的一个解，那么常数 $B$ 必须与 $k$ 有什么关系？[@problem_id:2134067]

为了验证这一点，我们需要计算 $u$ 对 $t$ 的一阶偏导数 $u_t$ 和对 $x$ 的[二阶偏导数](@entry_id:635213) $u_{xx}$。

首先，计算时间导数 $u_t$：
运用乘法法则和链式法则，我们得到：
$\frac{\partial u}{\partial t} = A \left( -\frac{1}{2} t^{-3/2} \exp\left(-\frac{x^2}{Bt}\right) + t^{-1/2} \exp\left(-\frac{x^2}{Bt}\right) \cdot \frac{x^2}{Bt^2} \right) = u(t,x) \left( -\frac{1}{2t} + \frac{x^2}{Bt^2} \right)$

接下来，计算空间导数 $u_{xx}$：
$u_x = \frac{A}{\sqrt{t}} \exp\left(-\frac{x^2}{Bt}\right) \cdot \left(-\frac{2x}{Bt}\right) = -\frac{2x}{Bt} u(t,x)$
$u_{xx} = -\frac{2}{Bt} u(t,x) - \frac{2x}{Bt} u_x = -\frac{2}{Bt} u(t,x) - \frac{2x}{Bt} \left(-\frac{2x}{Bt} u(t,x)\right) = u(t,x) \left( -\frac{2}{Bt} + \frac{4x^2}{B^2t^2} \right)$

现在，我们将这些导数代入[热方程](@entry_id:144435) $u_t = k u_{xx}$：
$u(t,x) \left( -\frac{1}{2t} + \frac{x^2}{Bt^2} \right) = k \cdot u(t,x) \left( -\frac{2}{Bt} + \frac{4x^2}{B^2t^2} \right)$

由于 $u(t,x)$ 不恒为零，我们可以消去这一项：
$-\frac{1}{2t} + \frac{x^2}{Bt^2} = -\frac{2k}{Bt} + \frac{4kx^2}{B^2t^2}$

为了使这个等式对所有的 $x$ 和 $t$ 都成立，各项的系数必须相等。比较 $x^2$ 的系数和常数项（相对于$x$）的系数，我们得到两个方程：
$\frac{1}{Bt^2} = \frac{4k}{B^2t^2} \implies \frac{1}{B} = \frac{4k}{B^2}$
$-\frac{1}{2t} = -\frac{2k}{Bt} \implies \frac{1}{2} = \frac{2k}{B}$

解这两个方程中的任意一个，我们都得到 $B = 4k$。这表明，函数 $u(t,x) = \frac{A}{\sqrt{4kt}} \exp\left(-\frac{x^2}{4kt}\right)$ 是热方程的一个精确解。这个函数被称为[热方程](@entry_id:144435)的**[基本解](@entry_id:184782) (fundamental solution)**，它描述了在 $t=0$ 时刻于 $x=0$ 点处的一个点热源随时间的演化。

另一个经典的例子是[二维拉普拉斯](@entry_id:746156)方程 $\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$，它描述了[稳态](@entry_id:182458)（不随时间变化）的物理现象，如静电势或[稳态温度分布](@entry_id:176266)。考虑函数 $u(x,y) = \arctan\left(\frac{y}{x}\right)$，它在极坐标中代表角度。让我们验证它是否是拉普拉斯方程的解 [@problem_id:2134063]。

我们计算其[二阶偏导数](@entry_id:635213)：
$u_x = \frac{1}{1 + (y/x)^2} \cdot \left(-\frac{y}{x^2}\right) = \frac{x^2}{x^2+y^2} \cdot \left(-\frac{y}{x^2}\right) = -\frac{y}{x^2+y^2}$
$u_y = \frac{1}{1 + (y/x)^2} \cdot \left(\frac{1}{x}\right) = \frac{x^2}{x^2+y^2} \cdot \left(\frac{1}{x}\right) = \frac{x}{x^2+y^2}$

$u_{xx} = \frac{\partial}{\partial x} \left(-\frac{y}{x^2+y^2}\right) = -y \cdot \left(-\frac{2x}{(x^2+y^2)^2}\right) = \frac{2xy}{(x^2+y^2)^2}$
$u_{yy} = \frac{\partial}{\partial y} \left(\frac{x}{x^2+y^2}\right) = x \cdot \left(-\frac{2y}{(x^2+y^2)^2}\right) = -\frac{2xy}{(x^2+y^2)^2}$

将它们相加：
$u_{xx} + u_{yy} = \frac{2xy}{(x^2+y^2)^2} - \frac{2xy}{(x^2+y^2)^2} = 0$

等式成立，因此 $u(x,y) = \arctan(y/x)$ 确实是[拉普拉斯方程](@entry_id:143689)的一个解（在 $x \neq 0$ 的区域）。这类满足[拉普拉斯方程](@entry_id:143689)的函数被称为**调和函数 (harmonic functions)**。

### [偏微分方程](@entry_id:141332)的分类：一个分析框架

为了系统地研究[偏微分方程](@entry_id:141332)，我们需要一个分类体系。方程的类型深刻地影响其解的行为以及用于求解的数学方法。主要的分类标准包括阶数、线性性和（对于二阶线性方程）其类型（双曲型、抛物型或椭圆型）。

#### 阶数、线性与齐次性

*   **阶数 (Order)**：一个[偏微分方程的阶](@entry_id:169120)数由方程中出现的最[高阶导数](@entry_id:140882)决定。例如，$u_t = u_{xx}$ 是二阶的，而 $u_t + c u_x = 0$ 是一阶的。

*   **线性 (Linearity)**：这是最重要的分类标准。如果未知函数 $u$ 及其所有[偏导数](@entry_id:146280)都以一次方的形式出现，并且它们的系数最多只依赖于[自变量](@entry_id:267118)（例如 $x, y, t$），那么该方程是**线性的 (linear)**。否则，方程是**[非线性](@entry_id:637147)的 (nonlinear)**。例如，$u_{tt} - c^2 u_{xx} = \sin(x)$ 是线性的，而[伯格斯方程](@entry_id:177995) $u_t + u u_x = 0$ 是[非线性](@entry_id:637147)的，因为存在 $u$ 与其导数 $u_x$ 相乘的项。同样，$u_{tt} - u_{xx} = u^2$ 也是[非线性](@entry_id:637147)的。

    对于一个[线性方程](@entry_id:151487)，如果将所有包含 $u$ 及其导数的项移到等号左边后，右边恒等于零，则称该方程是**齐次的 (homogeneous)**。如果右边是一个不恒为零的函数（只依赖于[自变量](@entry_id:267118)），则称该方程是**非齐次的 (inhomogeneous)** 或**非均匀的**。这个非零函数通常被称为**源项 (source term)**。

我们可以将一个[线性偏微分方程](@entry_id:172517)写成算子形式 $L[u] = g$，其中 $L$ 是一个[线性微分算子](@entry_id:174781)，$g$ 是[源项](@entry_id:269111)。线性的关键属性是 $L[c_1 u_1 + c_2 u_2] = c_1 L[u_1] + c_2 L[u_2]$，其中 $c_1$ 和 $c_2$ 是常数。这个性质，即**[叠加原理](@entry_id:144649) (superposition principle)**，是线性理论的基石。

让我们通过一个综合性例子来练习这些定义 [@problem_id:2134062]。考虑方程：
$t u_{tt} - k u_{xx} + \exp(u) = f(x,t)$

*   **阶数**：方程中最高阶的导数是 $u_{tt}$ 和 $u_{xx}$，它们都是二阶的。因此，这是一个**二阶**[偏微分方程](@entry_id:141332)。
*   **线性**：方程中出现了 $\exp(u)$ 这一项。这是一个关于 $u$ 的[非线性](@entry_id:637147)函数（它的[泰勒展开](@entry_id:145057)包含 $u$ 的高次幂）。因此，这个方程是**[非线性](@entry_id:637147)的**。
*   **齐次性**：由于方程右侧存在一个不恒为零的函数 $f(x,t)$，该方程是**非齐次的**。即使我们将它写成 $t u_{tt} - k u_{xx} + \exp(u) - f(x,t) = 0$ 的形式，源项的存在性并未改变。

因此，该方程是一个二阶、[非线性](@entry_id:637147)、非齐次的[偏微分方程](@entry_id:141332)。

#### 二阶[线性方程](@entry_id:151487)的分类

对于[二阶线性偏微分方程](@entry_id:195856)，存在一个更精细的分类，它与方程解的性质密切相关。一个通用的二阶线性方程可以写成：
$A u_{xx} + B u_{xy} + C u_{yy} + D u_x + E u_y + F u = G$
（这里我们使用 $(x,y)$ 作为[自变量](@entry_id:267118)，但同样适用于 $(x,t)$）。

方程的类型由一个称为**[判别式](@entry_id:174614) (discriminant)** 的量决定，$\Delta = B^2 - 4AC$。
*   如果 $\Delta > 0$，方程是**双曲型的 (hyperbolic)**。典型的例子是[波动方程](@entry_id:139839) $u_{tt} - c^2 u_{xx} = 0$。[双曲型方程](@entry_id:145657)描述的现象（如[波的传播](@entry_id:144063)）具有有限的传播速度和清晰的波前。
*   如果 $\Delta = 0$，方程是**抛物型的 (parabolic)**。典型的例子是热传导方程 $u_t - k u_{xx} = 0$。[抛物型方程](@entry_id:144670)描述的现象（如[扩散](@entry_id:141445)）具有无限的[传播速度](@entry_id:189384)，初始的扰动会瞬间影响到整个区域。
*   如果 $\Delta  0$，方程是**椭圆型的 (elliptic)**。典型的例子是拉普拉斯方程 $u_{xx} + u_{yy} = 0$。椭圆型方程通常描述[平衡态](@entry_id:168134)或[稳态](@entry_id:182458)系统，其解非常光滑，并且在一个点的值受到整个边界的影响。

值得注意的是，如果系数 $A, B, C$ 不是常数，而是自变量的函数，那么方程的类型可能会在定义域的不同区域发生变化。考虑方程 $t u_{xx} - u_{tt} = 0$ [@problem_id:2134046]。

与[标准形式](@entry_id:153058) $A u_{xx} + B u_{xt} + C u_{tt} + \dots = 0$ 比较，我们有 $A=t$, $B=0$, $C=-1$。判别式为：
$\Delta = B^2 - 4AC = 0^2 - 4(t)(-1) = 4t$

因此，方程的类型依赖于 $t$ 的值：
*   当 $t > 0$ 时，$\Delta = 4t > 0$，方程是**双曲型**的。
*   当 $t = 0$ 时，$\Delta = 4t = 0$，方程是**抛物型**的。
*   当 $t  0$ 时，$\Delta = 4t  0$，方程是**椭圆型**的。

这个方程，被称为 Tricomi 方程的一个变种，在[跨音速流](@entry_id:160423)[体力](@entry_id:174230)学中出现，其中流体的行为在亚音速（椭圆型）和超音速（双曲型）区域之间转换。

### [线性偏微分方程](@entry_id:172517)解的结构

线性是[偏微分方程理论](@entry_id:189232)中最强大的一个概念。它使得我们可以将复杂的解分解为更简单的部分，并从已知的解构造出新的解。

#### [叠加原理](@entry_id:144649)及其失效

正如前面提到的，对于一个[线性齐次方程](@entry_id:167132) $L[u] = 0$，如果 $u_1$ 和 $u_2$ 都是解，那么它们的任意[线性组合](@entry_id:154743) $c_1 u_1 + c_2 u_2$ 也必然是解。这是因为 $L[c_1 u_1 + c_2 u_2] = c_1 L[u_1] + c_2 L[u_2] = c_1 \cdot 0 + c_2 \cdot 0 = 0$。这个强大的**[叠加原理](@entry_id:144649) (superposition principle)** 意味着我们可以通过叠加[基本解](@entry_id:184782)（如[平面波](@entry_id:189798)或傅里叶模式）来构建更复杂的解。

然而，对于[非线性方程](@entry_id:145852)，叠加原理**完全失效**。这是线性和[非线性](@entry_id:637147)世界之间最根本的区别。让我们通过两个例子来明确这一点。

考虑非线性方程 $u_{tt} - u_{xx} = u^2$ [@problem_id:2134057]。给定两个解 $u_1(x,t) = -6/x^2$ 和 $u_2(x,t) = 6/t^2$。它们的和是 $u_S = u_1 + u_2 = 6/t^2 - 6/x^2$。如果[叠加原理](@entry_id:144649)成立，$u_S$ 也应该是该方程的解。让我们来检验一下。
左边是 $L[u_S] = \frac{\partial^2 u_S}{\partial t^2} - \frac{\partial^2 u_S}{\partial x^2}$。我们有：
$\frac{\partial^2 u_S}{\partial t^2} = \frac{36}{t^4}$，$\frac{\partial^2 u_S}{\partial x^2} = -\frac{36}{x^4}$。所以 $L[u_S] = \frac{36}{t^4} + \frac{36}{x^4}$。
右边是 $(u_S)^2 = (6/t^2 - 6/x^2)^2 = \frac{36}{t^4} - \frac{72}{x^2 t^2} + \frac{36}{x^4}$。
显然，$L[u_S] \neq (u_S)^2$。它们之间的差值为 $L[u_S] - (u_S)^2 = \frac{72}{x^2t^2} \neq 0$。因此，$u_1+u_2$ 不是解。

再看一个[流体力学](@entry_id:136788)中的基本模型——[无粘性伯格斯方程](@entry_id:168659) $u_t + u u_x = 0$ [@problem_id:2134054]。这是一个典型的一阶[非线性方程](@entry_id:145852)。已知 $u_1(x,t) = c$（常数）和 $u_2(x,t) = x/(t+1)$ 都是它的解。它们的和是 $u(x,t) = c + x/(t+1)$。将 $u$ 代入方程：
$u_t = -\frac{x}{(t+1)^2}$
$u_x = \frac{1}{t+1}$
$u_t + u u_x = -\frac{x}{(t+1)^2} + \left(c + \frac{x}{t+1}\right) \frac{1}{t+1} = -\frac{x}{(t+1)^2} + \frac{c}{t+1} + \frac{x}{(t+1)^2} = \frac{c}{t+1}$
结果不为零（除非 $c=0$），所以 $u_1+u_2$ 不是解。这两个例子清楚地表明，非线性方程的解不能简单地相加。

#### 通解

[偏微分方程](@entry_id:141332)的解通常不是唯一的函数，而是一个庞大的函数族。这个函数族被称为**通解 (general solution)**。通解中通常包含一个或多个任意函数。

对于一阶[线性齐次方程](@entry_id:167132) $\alpha u_x - \beta u_y = 0$，其中 $\alpha, \beta$ 是非零常数，我们可以通过一种称为**[特征线法](@entry_id:177800) (method of characteristics)** 的技术找到其通解。其基本思想是寻找一些曲线，沿着这些曲线，[偏微分方程](@entry_id:141332)可以简化为一个常微分方程。
对于这个方程，特征线由 $\frac{dx}{\alpha} = \frac{dy}{-\beta}$ 给出，积分得到 $\beta x + \alpha y = C$，其中 $C$ 是常数。这意味着解 $u$ 在这些称为特征线的直线上是常数。因此，解的值只依赖于它在哪条特征线上，也就是说，它只依赖于 $\beta x + \alpha y$ 的值。所以，通解的形式是：
$u(x,y) = F(\beta x + \alpha y)$
其中 $F$ 是任意一个[可微函数](@entry_id:144590)。

我们可以通过变量替换来更严格地证明这一点 [@problem_id:2134058]。令新坐标为 $\xi = \beta x + \alpha y$ 和 $\eta = \alpha x - \beta y$。根据链式法则：
$u_x = u_\xi \xi_x + u_\eta \eta_x = \beta u_\xi + \alpha u_\eta$
$u_y = u_\xi \xi_y + u_\eta \eta_y = \alpha u_\xi - \beta u_\eta$
代入原方程 $\alpha u_x - \beta u_y = 0$：
$\alpha(\beta u_\xi + \alpha u_\eta) - \beta(\alpha u_\xi - \beta u_\eta) = (\alpha^2 + \beta^2) u_\eta = 0$
由于 $\alpha, \beta$ 非零，$\alpha^2 + \beta^2 \neq 0$，因此我们必须有 $u_\eta = 0$。这表明 $u$ 不依赖于 $\eta$，只依赖于 $\xi$。所以 $u(\xi, \eta) = F(\xi)$，换回原坐标就是 $u(x,y) = F(\beta x + \alpha y)$。

例如，对于方程 $4u_x - u_y = 0$ [@problem_id:2134059]，其通解是 $F(x+4y)$。这意味着像 $\sin(x+4y)$、$\exp(-(x+4y))$ 甚至常数 $15$ 这样的函数，都是该方程的解，它们只是对应于不同的函数 $F$。

对于[二阶波动方程](@entry_id:754606) $u_{tt} = c^2 u_{xx}$，通解由**达朗贝尔 (d'Alembert)** 给出：
$u(x,t) = f(x-ct) + g(x+ct)$
其中 $f$ 和 $g$ 是任意的二阶[可微函数](@entry_id:144590)。这个解有非常直观的物理解释：$f(x-ct)$ 代表一个以速度 $c$ 向右传播的波，其形状由函数 $f$ 决定；而 $g(x+ct)$ 代表一个以速度 $c$ 向左传播的波，其形状由函数 $g$ 决定。波动方程的任何解都可以表示为这两个方向相反的行波的叠加。

#### 非齐次方程的解结构

对于一个线性非[齐次方程](@entry_id:163650) $L[u] = g$，其解的结构也非常优美。其通解可以表示为：
$u(x,t) = u_h(x,t) + u_p(x,t)$
这里，$u_h$ 是对应的齐次方程 $L[u]=0$ 的通解，而 $u_p$ 是原始非齐次方程 $L[u]=g$ 的**任何一个特解 (particular solution)**。
这个结论很容易证明。将 $u = u_h + u_p$ 代入算子 $L$：
$L[u] = L[u_h + u_p] = L[u_h] + L[u_p]$ (由于 $L$ 是线性的)
我们知道 $L[u_h] = 0$ 且 $L[u_p] = g$，所以 $L[u] = 0 + g = g$。这表明 $u_h + u_p$ 确实是原方程的解。

这个原理意味着，求解一个非齐次问题的任务可以分解为两部分：(1) 找到齐次方程的通解；(2) 找到非齐次方程的任何一个[特解](@entry_id:149080)。

例如，考虑[非齐次波动方程](@entry_id:176877) $u_{tt} - 4u_{xx} = -\cos(t)$ [@problem_id:2134053]。
对应的[齐次方程](@entry_id:163650)是 $u_{tt} - 4u_{xx} = 0$，这是一个[波动方程](@entry_id:139839)，波速 $c=2$。其通解是 $u_h(x,t) = F(x-2t) + G(x+2t)$。
我们被告知一个[特解](@entry_id:149080)是 $u_p(x,t) = \cos(t)$。（我们可以验证：$u_{p,tt} = -\cos(t)$，$u_{p,xx}=0$，所以 $u_{p,tt}-4u_{p,xx}=-\cos(t)$）。
因此，该非[齐次方程](@entry_id:163650)的通解就是：
$u(x,t) = F(x-2t) + G(x+2t) + \cos(t)$
其中 $F$ 和 $G$是任意的二阶[可微函数](@entry_id:144590)。

### 从通解到特解：初始与边界条件的角色

通解包含任意函数，代表了满足[偏微分方程](@entry_id:141332)的所有可能解的集合。在物理问题中，我们通常寻找一个唯一的解，这个解不仅要满足方程本身，还要满足一组特定的**辅助条件 (auxiliary conditions)**。这些条件通常在区域的边界（**边界条件 (boundary conditions, BCs)**）或在某个初始时刻（**初始条件 (initial conditions, ICs)**）给出。

让我们回到[一维波动方程](@entry_id:164824) $u_{tt} = c^2 u_{xx}$，其通解为 $u(x,t) = f(x-ct) + g(x+ct)$。假设我们研究一根无限长的弦，并给出它在 $t=0$ 时的初始位移和初始速度：
$u(x,0) = u_0(x)$
$u_t(x,0) = v_0(x)$

我们的任务是利用这两个初始条件来确定通解中的任意函数 $f$ 和 $g$。
将通解代入初始条件：
$u(x,0) = f(x) + g(x) = u_0(x)$ (1)
$u_t(x,t) = -cf'(x-ct) + cg'(x+ct)$
$u_t(x,0) = -cf'(x) + cg'(x) = v_0(x)$ (2)

对(1)式求导，得到 $f'(x) + g'(x) = u_0'(x)$。
将(2)式除以 $c$，得到 $-f'(x) + g'(x) = v_0(x)/c$。
联立这两个关于 $f'$ 和 $g'$ 的方程，可以解出：
$g'(x) = \frac{1}{2}\left(u_0'(x) + \frac{v_0(x)}{c}\right)$
$f'(x) = \frac{1}{2}\left(u_0'(x) - \frac{v_0(x)}{c}\right)$

对这两个式子积分，我们就可以确定 $f$ 和 $g$ 的形式（相差一个常数，但这些常数会在 $f+g$ 中抵消）。将它们代回通解，经过一番推导，可以得到一个只用[初始条件](@entry_id:152863) $u_0$ 和 $v_0$ 表示的解，这就是著名的**[达朗贝尔公式](@entry_id:175303) ([d'](@entry_id:189153)Alembert's formula)**：
$$u(x,t) = \frac{1}{2}[u_0(x-ct) + u_0(x+ct)] + \frac{1}{2c}\int_{x-ct}^{x+ct} v_0(s) ds$$

这个公式优雅地展示了初始条件如何决定未来的状态。解在 $(x,t)$ 点的值，由初始位移在 $x-ct$ 和 $x+ct$ 两点的值的平均，加上初始速度在区间 $[x-ct, x+ct]$ 上的积分贡献共同决定。

让我们应用这个公式来解决一个具体问题 [@problem_id:2134039]。考虑[波动方程](@entry_id:139839) $u_{tt} = c^2 u_{xx}$，初始条件为：
$u(x,0) = Ax^2$
$u_t(x,0) = B\cos(kx)$

根据[达朗贝尔公式](@entry_id:175303)，我们有 $u_0(x) = Ax^2$ 和 $v_0(x) = B\cos(kx)$。
第一部分是：
$\frac{1}{2}[u_0(x-ct) + u_0(x+ct)] = \frac{1}{2}[A(x-ct)^2 + A(x+ct)^2] = \frac{A}{2}[x^2 - 2xct + c^2t^2 + x^2 + 2xct + c^2t^2] = A(x^2 + c^2t^2)$

第二部分是：
$\frac{1}{2c}\int_{x-ct}^{x+ct} v_0(s) ds = \frac{1}{2c}\int_{x-ct}^{x+ct} B\cos(ks) ds = \frac{B}{2ck} [\sin(ks)]_{x-ct}^{x+ct} = \frac{B}{2ck} [\sin(k(x+ct)) - \sin(k(x-ct))]$

利用和差化积公式 $\sin\alpha - \sin\beta = 2\cos\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}$，我们得到：
$\frac{B}{2ck} [2\cos(kx)\sin(kct)] = \frac{B}{ck}\cos(kx)\sin(kct)$

将两部分加起来，我们就得到了该[初值问题](@entry_id:144620)的唯一解：
$u(x,t) = Ax^2 + Ac^2t^2 + \frac{B}{ck}\cos(kx)\sin(kct)$

这个例子完美地展示了本章所学的核心思想：从一个包含任意函数的通解出发，通过施加物理上合理的[初始条件](@entry_id:152863)，我们最终可以确定一个唯一的、描述系统特定演化的特解。理解解的结构和性质，是掌握和应用[偏微分方程解](@entry_id:166250)决实际问题的关键所在。
