## 应用与跨学科联系

在前面的章节中，我们已经详细探讨了[量子计算](@entry_id:142712)的基本原理和核心机制，包括[量子比特](@entry_id:137928)、叠加、纠缠和量子门。这些构成了[量子信息处理](@entry_id:158111)的理论基石。然而，理论的价值最终体现在其应用之中。本章的目的是展示这些基本原理如何转化为解决现实世界问题的强大工具，并揭示[量子计算](@entry_id:142712)与物理学、计算机科学、化学和金融等多个学科之间深刻而广泛的联系。

我们将不再重复介绍核心概念，而是将重点放在展示这些概念在不同情境下的应用、扩展和整合。通过一系列精心设计的问题情境，我们将探索量子算法的威力、[量子通信](@entry_id:138989)的独特能力、应对现实世界噪声的策略，以及[量子计算](@entry_id:142712)在推动其他科学前沿方面的巨大潜力。本章旨在为您架起一座从抽象原理到具体应用的桥梁，使您能够欣赏到[量子计算](@entry_id:142712)作为一门交叉学科的广度与深度。

### 核心量子算法：[量子优势](@entry_id:137414)的基石

量子算法是[量子计算](@entry_id:142712)威力的集中体现。它们利用量子力学独有的特性，例如叠加和纠缠，以在特定问题上实现超越[经典计算](@entry_id:136968)机的计算速度。下面我们探讨几种奠基性的[量子算法](@entry_id:147346)。

#### Deutsch 算法与[量子并行性](@entry_id:137267)

Deutsch 算法是第一个清晰展示[量子计算](@entry_id:142712)潜力的范例。尽管其解决的问题本身意义不大，但它绝妙地演示了“[量子并行性](@entry_id:137267)”的概念：通过一次计算，[量子计算](@entry_id:142712)机可以确定一个函数的全局属性，而[经典计算](@entry_id:136968)机则需要多次计算。该算法的核心机制是**相位反冲（phase kickback）**。当一个作用于两个[量子比特](@entry_id:137928)的受控门，其目标比特处于特定叠加态（如 $|-\rangle = \frac{1}{\sqrt{2}}(|0\rangle - |1\rangle)$）时，目标比特的状态保持不变，但一个相位因子会被“反冲”到控制比特上。

考虑一个函数 $f: \{0, 1\} \rightarrow \{0, 1\}$，其量子预言机（oracle）定义为 $U_f |x, y\rangle = |x, y \oplus f(x)\rangle$。在 Deutsch 算法中，当辅助比特处于 $|-\rangle$ 态时，预言机的作用等效于在第一个比特上施加一个单比特门 $U'_f$，其作用为 $U'_f|x\rangle = (-1)^{f(x)}|x\rangle$。这意味着函数的值 $f(x)$ 被编码成了第一个[量子比特](@entry_id:137928)的相位。

例如，如果我们发现这个等效的单比特门恰好是泡利-Z 门，即 $U'_f = Z$，其中 $Z|0\rangle = |0\rangle$ 且 $Z|1\rangle = -|1\rangle$，我们可以立即推断出 $(-1)^{f(0)} = 1$ 和 $(-1)^{f(1)} = -1$。这直接表明 $f(0)=0$ 和 $f(1)=1$。因此，该函数是**平衡**的（balanced），即其输出值中 0 和 1 的数量相等。通过完整的 Deutsch 算法流程——对输入态 $|01\rangle$ 分别施加[阿达玛门](@entry_id:146898)、应用预言机、再对第一个比特施加[阿达玛门](@entry_id:146898)——最终对第一个比特的测量将以 100% 的概率得到结果‘1’，这也同样证实了函数是平衡的。这个简单的例子揭示了量子算法如何通过巧妙地操纵相位来获取经典方法难以企及的信息 [@problem_id:2098726]。

#### Grover 搜索算法与[振幅放大](@entry_id:147663)

在处理非结构化数据库[搜索问题](@entry_id:270436)时，经典算法平均需要检查 $N/2$ 个条目才能找到目标，最坏情况下需要检查 $N$ 个。Grover 算法提供了一种二次加速，仅需约 $O(\sqrt{N})$ 次操作。它并非利用相位反冲，而是通过一种称为“[振幅放大](@entry_id:147663)”的迭代过程来实现。

Grover 算法的每一次迭代包含两个关键步骤：
1.  **预言机（Oracle）**：这个操作会识别出“标记”的目标状态 $|w\rangle$ 并将其相位翻转，即 $|w\rangle \rightarrow -|w\rangle$，而所有其他状态保持不变。
2.  **[扩散](@entry_id:141445)算符（Diffusion Operator）**：该操作 $U_s = 2|s\rangle\langle s| - I$（其中 $|s\rangle$ 是所有计算[基态](@entry_id:150928)的均匀叠加）可以被直观地理解为“关于平均值的翻转”。它的作用是放大刚刚被预言机标记为负相位的状态的振幅，同时减小其他状态的振幅。

几何上，每次迭代都将[状态向量](@entry_id:154607)向目标状态 $|w\rangle$ 旋转一小个角度。经过约 $\frac{\pi}{4}\sqrt{N}$ 次迭代后，[状态向量](@entry_id:154607)将非常接近目标状态，此时进行测量，就能以很高的概率得到正确答案。例如，在一个 2-[量子比特](@entry_id:137928)系统（$N=4$）中搜索标记状态 $|10\rangle$ 时，从均匀叠加态 $|s\rangle = \frac{1}{2}(|00\rangle + |01\rangle + |10\rangle + |11\rangle)$ 出发，仅需一次 Grover 迭代，系统状态就会确定性地演化到目标态 $|10\rangle$。这清晰地展示了[振幅放大](@entry_id:147663)机制的惊人效果 [@problem_id:2098754]。

#### [量子相位估计](@entry_id:136538)

[量子相位估计](@entry_id:136538)（Quantum Phase Estimation, QPE）是许多复杂[量子算法](@entry_id:147346)（包括 Shor 分解算法）的核心子程序。其目标是估计一个酉算符 $U$ 的某个[本征态](@entry_id:149904) $|u\rangle$ 所对应的[本征值](@entry_id:154894) $e^{2\pi i\phi}$ 中的相位 $\phi$。

QPE 算法使用两组[量子比特](@entry_id:137928)：一组是用于存储相位信息的“计数寄存器”，另一组是处于待测[本征态](@entry_id:149904) $|u\rangle$ 的“目标寄存器”。算法的核心步骤是通过一系列受控 $U^{2^k}$ 门，将相位信息 $\phi$ 以[傅里叶变换](@entry_id:142120)的形式编码到计数寄存器中。最后，通过对计数寄存器应用[逆量子傅里叶变换](@entry_id:139405)（QFT⁻¹），就可以将相位信息从傅里叶空间转换回计算基，从而通过测量读出 $\phi$ 的一个二[进制](@entry_id:634389)近似值。

如果相位 $\phi$ 能够用计数寄存器的有限位数精确表示，例如 $\phi = y/2^n$（其中 $n$ 是计数寄存器的比特数，$y$ 是一个整数），那么 QPE 算法的测量结果将是确定性的，直接给出整数 $y$ 的二[进制](@entry_id:634389)表示。例如，使用一个 3-比特的计数寄存器来估计 T 门（$T|1\rangle = e^{i\pi/4}|1\rangle$）作用于其本征态 $|1\rangle$ 的相位 $\phi = \pi/4 / (2\pi) = 1/8$。由于 $1/8$ 可以精确地表示为 3-比特二进制小数 $0.001$，即对应整数 $y=1$，QPE 算法在测量计数寄存器时将以 100% 的概率得到结果 '001' [@problem_id:2098738]。

### [量子通信](@entry_id:138989)与网络

纠缠不仅是[量子算法](@entry_id:147346)的资源，也是[量子通信](@entry_id:138989)协议的基石，它使得信息的传输和处理能够以超越经典物理限制的方式进行。

#### [量子隐形传态](@entry_id:144485)

[量子隐形传态](@entry_id:144485)（Quantum Teleportation）是一种利用事先共享的[纠缠对](@entry_id:160576)和经典通信信道，将一个任意的未知[量子态](@entry_id:146142)从一处传输到另一处的技术。值得强调的是，这个过程传输的是[量子信息](@entry_id:137721)本身，而非物质。

协议的标准流程如下：Alice 希望将[量子比特](@entry_id:137928) C 的状态 $|\psi\rangle_C$ 传送给 Bob。他们事先共享一个[纠缠对](@entry_id:160576)（A 和 B）。Alice 对她持有的两个[量子比特](@entry_id:137928)（C 和 A）进行一次联合的贝尔基测量，这会使她的两个比特随机地投影到四个贝尔态之一。然后，Alice 通过经典信道告诉 Bob 她的测量结果（2个经典比特）。根据收到的信息，Bob 对他持有的[量子比特](@entry_id:137928) B 施加一个相应的本地修正门（I, X, Z 或 XZ），从而将 B 的状态恢复成原始的 $|\psi\rangle_C$。例如，如果 Alice 的测量结果是 $|\Psi^+\rangle_{CA} = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)$，Bob 就需要对他的[量子比特](@entry_id:137928)施加一个泡利-X 门来完成状态的重构 [@problem_id:2098732]。

#### [超密编码](@entry_id:137220)

[超密编码](@entry_id:137220)（Superdense Coding）可以看作是[量子隐形传态](@entry_id:144485)的“逆”过程。它利用一个共享的纠缠对，仅通过发送一个[量子比特](@entry_id:137928)，就能传输两个经典比特的信息。这突破了[经典信息论](@entry_id:142021)中一个[量子比特](@entry_id:137928)最多携带一个比特信息的限制（Holevo 界限）。

在协议开始时，Alice 和 Bob 共享一个贝尔态，例如 $|\Phi^+\rangle$。Alice 根据她想发送的两位经典比特信息（`00`, `01`, `10`, `11`）对她手中的[量子比特](@entry_id:137928)施加四种不同的本地操作之一（分别是 $I$, $X$, $Z$, $ZX$）。这个操作会将共享的纠缠对转换为四个标准[贝尔态](@entry_id:140749)中的一个。然后，Alice 将她的[量子比特](@entry_id:137928)发送给 Bob。Bob 收到后，同时拥有了这对[量子比特](@entry_id:137928)，他通过一个固定的解码电路（通常是一个 CNOT 门后跟一个[阿达玛门](@entry_id:146898)）对这两个比特进行操作，最后通过测量即可确定性地读出 Alice 发送的两个经典比特。例如，如果 Alice 想发送 `10`，她会对她的比特应用 $Z$ 门，Bob 在收到并执行解码操作后，测量的结果将确定性地是 $|10\rangle$ [@problem_id:2098735]。

#### [纠缠交换](@entry_id:137925)

要构建远距离的[量子网络](@entry_id:144522)，一个关键的挑战是如何在从未直接相互作用的两个遥远节点之间建立纠缠。[纠缠交换](@entry_id:137925)（Entanglement Swapping）协议优雅地解决了这个问题。

设想 Alice 和 Charlie 是两个遥远的节点，他们希望共享一个纠缠对。他们可以借助一个中间节点 Bob。首先，建立一个在 Alice 和 Bob 之间的纠缠对（[量子比特](@entry_id:137928)1和2），再建立一个在 Bob 和 Charlie 之间的纠缠对（[量子比特](@entry_id:137928)3和4）。此时，Bob 拥有了两个分别来自不同[纠缠对](@entry_id:160576)的[量子比特](@entry_id:137928)（2和3）。如果 Bob 对他手中的这两个[量子比特](@entry_id:137928)进行一次贝尔基测量，这个测量操作会将纠缠关系“交换”出去。测量的结果是，Alice 的[量子比特](@entry_id:137928)1和 Charlie 的[量子比特](@entry_id:137928)4会瞬间变得纠缠，尽管它们从未有过直接的物理接触。Bob 的测量结果决定了 A 和 C 之间最终形成的具体贝尔态。例如，如果初始的两个[纠缠对](@entry_id:160576)分别是 $|\Psi_{12}^-\rangle$ 和 $|\Phi_{34}^+\rangle$，而 Bob 对2和3的测量结果是 $|\psi_{23}^+\rangle$，那么 Alice 和 Charlie 的比特将塌缩到[纠缠态](@entry_id:152310) $\frac{1}{\sqrt{2}}(|00\rangle - |11\rangle)$ 上 [@problem_id:2098719]。这一技术是构建[量子中继器](@entry_id:197735)和大规模[量子通信](@entry_id:138989)网络的基础。

### 理论与现实的桥梁：[纠错](@entry_id:273762)、噪声与模拟

[量子态](@entry_id:146142)极其脆弱，容易受到环境噪声的干扰而发生退相干，这是实现可扩展[量子计算](@entry_id:142712)的最大障碍。因此，理解噪声、开发能够抵御噪声的[量子纠错码](@entry_id:266787)，以及利用[量子计算](@entry_id:142712)机模拟其他量子系统，是连接理论与现实的关键环节。

#### [量子纠错](@entry_id:139596)

经典纠错通过信息冗余（如将 `0` 编码为 `000`）来对抗错误。[量子纠错码](@entry_id:266787)（QEC）也采用类似的思想，但更为精妙，因为它必须在不违反“[不可克隆定理](@entry_id:146200)”和不因测量而破坏[量子态](@entry_id:146142)的前提下检测和纠正错误。

QEC 的核心思想是将一个“[逻辑量子比特](@entry_id:142662)”的信息编码到多个“[物理量子比特](@entry_id:137570)”构成的[纠缠态](@entry_id:152310)中。[错误检测](@entry_id:275069)通过所谓的“综合征测量”（syndrome measurement）来完成。这些测量被设计成只揭示发生了何种错误（如哪个比特发生了翻转），而不泄露任何关于[逻辑量子比特](@entry_id:142662)本身的信息（即 $\alpha$ 和 $\beta$ 的值）。

一个最简单的例子是[三量子比特比特翻转码](@entry_id:141854)，它将逻辑态 $|0_L\rangle$ 和 $|1_L\rangle$ 分别编码为 $|000\rangle$ 和 $|111\rangle$。通过测量两个[稳定子算符](@entry_id:141669)（如 $S_1 = Z_1Z_2$ 和 $S_2 = Z_2Z_3$）的[本征值](@entry_id:154894)，我们可以构建一个综合征。例如，如果没有错误发生，两个测量值都是 $+1$。如果第二个[量子比特](@entry_id:137928)发生了一个比特翻转（X 错误），那么两个测量值都会变为 $-1$。根据不同的综合征值 `(+1, +1)`, `(+1, -1)`, `(-1, +1)`, `(-1, -1)`，我们可以唯一地确定是无错误、第三、第一还是第二个比特发生了错误，并施加相应的 X 门来纠正它 [@problem_id:2098717]。

更强大的[纠错码](@entry_id:153794)，如 Shor 九[量子比特](@entry_id:137928)码，能够同时纠正任意单比特的比特翻转（X 错误）、相位翻转（Z 错误）以及它们的组合（Y 错误）。通过巧妙的嵌套编码结构，这类[纠错码](@entry_id:153794)能够满足[量子纠错](@entry_id:139596)的条件，即不同错误作用在逻辑态上产生的状态是相互正交的，从而可以被无[歧义](@entry_id:276744)地识别和纠正 [@problem_id:2098721]。

#### 理解量子噪声

为了设计有效的纠错码，我们必须首先理解噪声如何影响量子系统。[量子信道](@entry_id:145403)理论为此提供了数学框架。一个常见的[噪声模型](@entry_id:752540)是退相干信道（dephasing channel），它不会改变计算[基态](@entry_id:150928) $|0\rangle$ 和 $|1\rangle$ 的布居数，但会[随机化](@entry_id:198186)它们之间的[相对相位](@entry_id:148120)。

我们可以通过分析噪声对纠缠的影响来量化其破坏性。例如，考虑一个处于最大纠缠贝尔态 $|\Phi^+\rangle$ 的[双量子比特系统](@entry_id:203437)，其中一个比特以概率 $p$ 经历一次相位翻转（Z 错误）。系统的最终状态会变成一个混合态。通过计算这个混合态的“并发度”（concurrence，一种[纠缠度量](@entry_id:139894)），可以发现纠缠度的大小为 $|1 - 2p|$。当 $p=0$（无错误）或 $p=1$（确定性 Z 错误）时，并发度为1，系统仍处于最大[纠缠态](@entry_id:152310)（分别为 $|\Phi^+\rangle$ 和 $|\Phi^-\rangle$）。然而，当 $p=0.5$ 时，相位信息完全随机化，并发度降为0，纠缠完全消失。这个例子清晰地展示了环境噪声是如何逐步侵蚀[量子计算](@entry_id:142712)中最宝贵的资源——纠缠的 [@problem_id:2098714]。

#### 模拟量子系统

[Richard Feynman](@entry_id:155876) 最初设想[量子计算](@entry_id:142712)机时，其主要动机就是模拟其他量子系统。因为用[经典计算](@entry_id:136968)机模拟一个包含 $n$ 个相互作用的[量子比特](@entry_id:137928)的系统，需要处理一个维度为 $2^n$ 的[希尔伯特空间](@entry_id:261193)，其资源消耗随 $n$ [指数增长](@entry_id:141869)。而一个 $n$ [量子比特](@entry_id:137928)的[量子计算](@entry_id:142712)机，其本身就是一个可以被精确调控的 $2^n$ 维系统，因此是模拟其他量子系统的天然平台。

[量子模拟](@entry_id:145469)的关键在于将待模拟系统的[哈密顿量](@entry_id:172864) $H$ 的[时间演化算符](@entry_id:196774) $U(t) = \exp(-iHt/\hbar)$ 分解为一系列[量子计算](@entry_id:142712)机可以执行的基本量子门。例如，考虑一个描述两个自旋相互作用的简化[伊辛模型](@entry_id:139066)[哈密顿量](@entry_id:172864) $H = J (\sigma_z^{(1)} \otimes \sigma_z^{(2)})$。其[演化算符](@entry_id:182628)为 $U(\gamma) = \exp(-i\gamma (\sigma_z^{(1)} \otimes \sigma_z^{(2)}))$，其中 $\gamma$ 是与[相互作用强度](@entry_id:192243)和时间相关的参数。这个双比特纠缠门不能直接在大多数量子硬件上实现，但它可以被精确地分解为一个由两个 CNOT 门和一个单比特 Z-旋转门组成的序列：一个 CNOT 门，接着在目标比特上施加一个 $R_z(2\gamma)$ 旋转，最后再施加一个 CNOT 门。这个例子展示了如何将复杂的物理相互作用“数字化”为量子电路，为模拟[凝聚态物质](@entry_id:747660)、[高能物理](@entry_id:181260)和[量子化学](@entry_id:140193)中的复杂现象铺平了道路 [@problem_id:2098710]。

### 跨学科前沿

[量子计算](@entry_id:142712)的影响力远远超出了物理学和计算机科学的范畴，它正在与化学、金融、[材料科学](@entry_id:152226)等领域深度融合，有望解决这些领域中一些最棘手的计算难题。

#### [量子化学](@entry_id:140193)

精确计算分子的[电子结构](@entry_id:145158)是[量子化学](@entry_id:140193)的核心挑战，对于[药物设计](@entry_id:140420)和新[材料发现](@entry_id:159066)至关重要。[经典计算](@entry_id:136968)机在处理包含复杂电子关联的分子时力不从心。[量子计算](@entry_id:142712)机，特别是通过[变分量子本征求解器](@entry_id:150318)（VQE）等算法，为解决这些问题提供了新的途径。

然而，将经典的计算化学方法直接移植到[量子计算](@entry_id:142712)机上并非易事。以广泛应用的[耦合簇理论](@entry_id:141746)（CCSD）为例，其核心是使用一个非酉的指数算符 $e^{\hat{T}}$ 来构造[波函数](@entry_id:147440)。[量子计算](@entry_id:142712)机的原生操作是[酉演化](@entry_id:145020)，因此无法直接实现这种非[酉变换](@entry_id:152599)。这揭示了一个深刻的观点：[量子计算](@entry_id:142712)不仅是提供更强的算力，它还要求我们重新思考算法的设计[范式](@entry_id:161181)，发展“量子原生”的算法，例如使用[幺正耦合簇](@entry_id:204066)（Unitary Coupled Cluster）作为 VQE 的[拟设](@entry_id:184384)[波函数](@entry_id:147440)。这种方法的转变正是跨学科研究的魅力所在，它促使不同领域的专家共同创造新的解决方案 [@problem_id:2453718]。

#### 计算金融与优化

金融领域中的许多问题，如[衍生品定价](@entry_id:144008)、投资[组合优化](@entry_id:264983)和[风险分析](@entry_id:140624)，都深受“维度灾难”的困扰——即问题规模随变量数量（如资产数量）呈指数增长。[量子计算](@entry_id:142712)有望通过其处理高维[向量空间](@entry_id:151108)的能力来缓解这一问题。

[量子振幅估计](@entry_id:139353)（QAE）是蒙特卡洛模拟的量子对应物，它能将达到目标精度 $\epsilon$ 所需的查询次数从经典方法的 $O(1/\epsilon^2)$ 降低到 $O(1/\epsilon)$，实现二次加速。这对于需要高精度定价的金融模型尤其有价值。然而，这并不意味着[量子计算](@entry_id:142712)能完全消除维度灾难。一个完整的量子[金融算法](@entry_id:142919)的成本不仅取决于 $\epsilon$，还依赖于制备编码金融数据的[量子态](@entry_id:146142)以及执行定价函数的预言机的成本，这些成本通常仍会随维度 $d$ [多项式增长](@entry_id:177086)。此外，像 HHL 算法（用于求解线性方程组）等其他工具，也为解决某些可被转化为大规模[线性系统](@entry_id:147850)的金融模型提供了潜在的指数级加速。

一个关键的领悟是，量子算法的优势主要体现在计算高维[分布](@entry_id:182848)的**宏观统计量**（如[期望值](@entry_id:153208)、风险价值）上，而非提取[分布](@entry_id:182848)的全部细节。试图从一个[量子态](@entry_id:146142)中读出所有信息将需要指数级的测量次数，从而抵消所有[量子优势](@entry_id:137414)。因此，[量子计算](@entry_id:142712)在金融中的应用，是关于以一种全新的方式提出问题和提取答案 [@problem_id:2439670] [@problem_id:2453718]。

#### 实验实现与表征

理论和算法的最终检验来自于实验。在实验室中，一个核心任务是验证和表征所制备的[量子态](@entry_id:146142)和所执行的[量子操作](@entry_id:145906)是否符合预期。[量子态](@entry_id:146142)层析（Quantum State Tomography）就是这样一种技术。

由于我们无法直接“看到”一个[量子态](@entry_id:146142)的[波函数](@entry_id:147440)，我们必须通过在大量相同副本上进行不同基下的测量来间接地重构它。对于一个单[量子比特](@entry_id:137928)，其状态可以由密度矩阵 $\rho = \frac{1}{2}(I + r_x \sigma_x + r_y \sigma_y + r_z \sigma_z)$ 描述，其中 $(r_x, r_y, r_z)$ 是布洛赫向量的三个实数分量。通过分别在 $\sigma_x$, $\sigma_y$, $\sigma_z$ 的本征基下进行大量测量，我们可以从测量结果的统计频率中估算出这三个分量的[期望值](@entry_id:153208) $\langle\sigma_i\rangle = r_i$。例如，$r_z$ 可以通过在 Z 基下测量得到 $+1$ 和 $-1$ 结果的次数差来估计：$r_z \approx (N_{z,+} - N_{z,-})/N_{total}$。这个过程将抽象的密度矩阵与可测量的实验数据联系起来，是任何[量子计算](@entry_id:142712)实验中不可或缺的一环 [@problem_id:2098762]。

#### 计算机科学与物理学的基础

最后，[量子计算](@entry_id:142712)的发展也促使我们重新审视计算和物理现实的基本边界。

**CHSH 游戏与[贝尔不等式](@entry_id:156237)**：CHSH 游戏是一个涉及两方（Alice 和 Bob）协作的游戏，它生动地展示了量子非定域性的力量。在这个游戏中，利用事先共享的[纠缠对](@entry_id:160576)，Alice 和 Bob 可以采用一种量子策略，使得他们的平均获胜概率达到约 85% ($\cos^2(\pi/8)$)。而任何基于经典物理（即只允许局域实在性和经典通信）的策略，其获胜概率都无法超过 75%。量子策略之所以能够胜出，是因为纠缠使得 Alice 和 Bob 的测量结果之间存在一种超越经典物理所允许的强关联。这不仅是[贝尔不等式](@entry_id:156237)被违背的一个实验证明，也从计算和信息的角度揭示了量子世界与经典世界的深刻差异 [@problem_id:2098742]。

**[丘奇-图灵论题](@entry_id:138213)**：一个常见的误解是，像 Shor 算法这样的指数级加速是否会“推翻”计算机科学的基石——[丘奇-图灵论题](@entry_id:138213)。该论题断言，任何能被算法解决的问题都能被图灵机解决。答案是否定的。关键在于区分“可计算性”（computability）和“计算复杂性”（complexity）。[丘奇-图灵论题](@entry_id:138213)关注的是前者：什么问题原则上是可解的。而[量子计算](@entry_id:142712)的优势在于后者：对于某些可解问题，它能以多快的速度解决。任何[量子计算](@entry_id:142712)机的计算过程，虽然可能极其缓慢，但都可以被一台经典[图灵机模拟](@entry_id:152131)。因此，[量子计算](@entry_id:142712)机并没有扩展“可计算”问题的边界，而是极大地改变了“高效可计算”问题的版图。它没有推翻[丘奇-图灵论题](@entry_id:138213)，而是提出了一个强化的版本，即“量子[丘奇-图灵论题](@entry_id:138213)”，它推测没有物理上可实现的计算设备能比[量子计算](@entry_id:142712)机更强大 [@problem_id:1450187]。
