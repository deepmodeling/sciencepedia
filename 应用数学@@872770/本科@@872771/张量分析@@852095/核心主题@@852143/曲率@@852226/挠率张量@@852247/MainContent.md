## 引言
在[微分几何](@entry_id:145818)的探索中，[仿射联络](@entry_id:160152)是理解弯曲空间中向量如何变化的核心工具。然而，联络本身不仅仅是一个[微分算子](@entry_id:140145)，其内在结构蕴含着深刻的几何信息。一个自然而然的问题是：联络的结构本身告诉我们关于空间的什么信息？[挠率张量](@entry_id:204137)正是解答这个问题的关键，它衡量了空间在无穷小尺度下的“扭曲”或“缠绕”程度，解决了仅凭[协变导数](@entry_id:152476)无法完全刻画几何的问题。

本文将系统地引导您深入理解[挠率张量](@entry_id:204137)。在“原则与机制”一章中，我们将从挠率的严格定义出发，揭示其与[联络系数](@entry_id:157618)对称性的关系，并探讨其作为真正几何对象的核心属性。接着，在“应用与跨学科联系”一章中，我们将跨出纯数学的范畴，展示挠率如何在固体物理学中描述材料缺陷，以及如何在广义相对论的替代理论中扮演[引力](@entry_id:175476)的核心角色。最后，通过“动手实践”部分提供的一系列计算练习，您将有机会亲手应用所学知识，将抽象理论转化为具体的计算能力。让我们一同开启探索空间扭曲奥秘的旅程。

## 原则与机制

在引入了作为微分几何核心工具的[仿射联络](@entry_id:160152)之后，我们自然会探究其更深层次的性质。一个联络 $\nabla$ 不仅定义了向量场沿另一个向量场方向的变化率，其内在结构本身也蕴含着丰富的几何信息。本章将深入探讨一个由联络自身定义的关键几何对象——**[挠率张量](@entry_id:204137)**（Torsion Tensor）。[挠率张量](@entry_id:204137)衡量了联络在何种程度上偏离了某种“对称性”，并揭示了空间在无穷小尺度下的扭曲特性。

### [挠率张量](@entry_id:204137)的定义与基本性质

让我们从最根本的定义出发。给定一个光滑流形 $M$ 上的[仿射联络](@entry_id:160152) $\nabla$，以及其上的任意两个光滑向量场 $X$ 和 $Y$，与它们相关的**[挠率张量](@entry_id:204137)** $T(X, Y)$ 是一个通过以下公式定义的新的向量场：

$$T(X, Y) = \nabla_X Y - \nabla_Y X - [X, Y]$$

为了完全理解这个定义，我们需要回顾其中的每一个组成部分。$\nabla_X Y$ 是向量场 $Y$ 沿向量场 $X$ 方向的协变导数，它描述了 $Y$ 在 $X$ 方向上的变化。$[X, Y]$ 则是 $X$ 和 $Y$ 的**李括号**（Lie Bracket），其定义为 $[X, Y] = XY - YX$，衡量了这两个向量场作为微分算子时的非对易性。从某种意义上说，[李括号](@entry_id:636461)代表了沿着一个向量场再沿着另一个向量场进行[无穷小位移](@entry_id:202209)所产生的内在“流动差异”。因此，[挠率张量](@entry_id:204137) $T(X, Y)$ 精确地量化了由联络定义的“平行移动”的非对易性（$\nabla_X Y - \nabla_Y X$）与由向量场本身的流动定义的非对易性（$[X, Y]$）之间的差异。

从定义式出发，我们可以立即观察到[挠率张量](@entry_id:204137)的一个核心代数性质。交换 $X$ 和 $Y$ 的位置，我们得到：

$$T(Y, X) = \nabla_Y X - \nabla_X Y - [Y, X]$$

由于李括号是反对称的，即 $[Y, X] = -[X, Y]$，我们可以将上式重写为：

$$T(Y, X) = -(\nabla_X Y - \nabla_Y X) - (-[X, Y]) = -(\nabla_X Y - \nabla_Y X - [X, Y]) = -T(X, Y)$$

这表明[挠率张量](@entry_id:204137)在其两个向量参数上是**反对称的**（antisymmetric）。这个性质，即 $T(X, Y) = -T(Y, X)$，具有深刻的几何与代数意涵。它意味着对于任意向量场 $X$，都有 $T(X, X) = 0$。

这个看似简单的反对称性质，能立刻导出一个重要的结论。考虑一个一维光滑流形。在其上任意一点 $p$，切空间 $T_p M$ 是一维的。这意味着在点 $p$ 的任何两个[切向量](@entry_id:265494) $v, w \in T_p M$ 都是线性相关的，即存在一个标量 $\lambda$ 使得 $w = \lambda v$。由于[挠率张量](@entry_id:204137)在每一点都表现为一个[双线性映射](@entry_id:186502) $T_p: T_p M \times T_p M \to T_p M$，我们可以利用其[双线性](@entry_id:146819)和[反对称性](@entry_id:261893)来计算 $T_p(v, w)$：

$$T_p(v, w) = T_p(v, \lambda v) = \lambda T_p(v, v) = \lambda \cdot 0 = 0$$

由于这在[流形](@entry_id:153038)的每一点对任意切向量都成立，我们得出结论：**在任何[一维流](@entry_id:269448)形上，任意仿射[联络的[挠](@entry_id:192913)率张量](@entry_id:204137)恒为零**。这个结论并非源于联络的任何特殊选择，而是由一维空间的内在限制和[挠率张量](@entry_id:204137)的[反对称性](@entry_id:261893)共同决定的 [@problem_id:1685028]。

### [局部坐标](@entry_id:181200)下的[挠率张量](@entry_id:204137)

为了在实际中进行计算，我们需要将[挠率张量](@entry_id:204137)的抽象定义转化为在局部坐标系下的分量表达式。这是连接几何直观与具体计算的桥梁。

令 $(x^1, \dots, x^n)$ 为[流形](@entry_id:153038)上的一个[局部坐标系](@entry_id:751394)，其对应的[坐标基](@entry_id:270149)底为 $\{\partial_i = \frac{\partial}{\partial x^i}\}$。我们回顾，[联络系数](@entry_id:157618)（或称 Christoffel 符号）$\Gamma^k_{ij}$ 正是由联络作用于基底向量所定义的：

$$\nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k$$

这里我们采用了爱因斯坦求和约定，即对重复出现的上下指标自动求和。同时，[坐标基](@entry_id:270149)底向量的一个关键性质是它们的李括号为零，因为偏导数的求导次序可以交换：

$$[\partial_i, \partial_j]f = \partial_i(\partial_j f) - \partial_j(\partial_i f) = 0$$

对于任意光滑函数 $f$ 成立，因此 $[\partial_i, \partial_j] = 0$。

现在，我们将[挠率张量](@entry_id:204137)的定义应用于[坐标基](@entry_id:270149)底向量 $\partial_i$ 和 $\partial_j$ [@problem_id:1543298]：

$$T(\partial_i, \partial_j) = \nabla_{\partial_i} \partial_j - \nabla_{\partial_j} \partial_i - [\partial_i, \partial_j]$$

将上述关系代入，得到：

$$T(\partial_i, \partial_j) = \Gamma^k_{ij} \partial_k - \Gamma^k_{ji} \partial_k - 0 = (\Gamma^k_{ij} - \Gamma^k_{ji}) \partial_k$$

另一方面，[挠率张量](@entry_id:204137) $T$ 是一个 $(1,2)$ 型张量，其分量 $T^k_{ij}$ 的定义就是 $T(\partial_i, \partial_j) = T^k_{ij} \partial_k$。比较两式，我们立即得到了[挠率张量](@entry_id:204137)分量的核心计算公式：

$$T^k_{ij} = \Gamma^k_{ij} - \Gamma^k_{ji}$$

这个公式提供了一个非常直观的理解：**[挠率张量](@entry_id:204137)的分量恰好是[联络系数](@entry_id:157618)（Christoffel 符号）在其下方两个指标上的反对称部分**。如果一个联络的所有 Christoffel 符号在其低指标上都是对称的，即 $\Gamma^k_{ij} = \Gamma^k_{ji}$，那么它的[挠率张量](@entry_id:204137)就恒为零。这样的联络被称为**[无挠联络](@entry_id:181337)**（torsion-free connection）。广义相对论中至关重要的 **Levi-Civita 联络**就是[无挠联络](@entry_id:181337)的一个典型范例。

例如，考虑一个[二维流形](@entry_id:188198)，其坐标为 $(x^1, x^2)$，某个联络的非零系数为 $\Gamma^1_{12} = 4x^2$ 和 $\Gamma^1_{21} = -x^1$。那么，在点 $(x^1, x^2)=(2, 3)$ 处，[挠率张量](@entry_id:204137)作用在基底向量上的结果 $T(\partial_1, \partial_2)$ 的分量可以计算如下 [@problem_id:1558716]：
$T^1_{12} = \Gamma^1_{12} - \Gamma^1_{21} = 4x^2 - (-x^1) = 4(3) + 2 = 14$。
$T^2_{12} = \Gamma^2_{12} - \Gamma^2_{21} = 0 - 0 = 0$。
因此，$T(\partial_1, \partial_2)$ 在该点是一个分量为 $\begin{pmatrix} 14  0 \end{pmatrix}$ 的向量。

### 挠率的张量属性

一个初学者可能会问：既然 Christoffel 符号 $\Gamma^k_{ij}$ 本身在坐标变换下不表现为张量行为，为什么它们的差 $T^k_{ij}$ 就可以构成一个张量的分量呢？这是一个深刻且关键的问题。答案在于，在[坐标变换](@entry_id:172727)下，Christoffel 符号的非张量性变换项恰好在其低指标 $i,j$ 上是对称的，因此在作差时被完美地消去了。

我们在此不详细[推导 Christoffel 符号](@entry_id:263591)的完整变换法则，而是通过一个实例来验证 $T^k_{ij}$ 的确遵循 $(1,2)$ 型张量的变换法则。一个 $(1,2)$ 型张量的分量在从[坐标系](@entry_id:156346) $\{x^i\}$ 变换到 $\{\bar{x}^a\}$ 时，其分量变换规律为：

$$\bar{T}^a_{bc} = \frac{\partial \bar{x}^a}{\partial x^k} \frac{\partial x^i}{\partial \bar{x}^b} \frac{\partial x^j}{\partial \bar{x}^c} T^k_{ij}$$

考虑一个[二维流形](@entry_id:188198)，在 $(x,y)$ [坐标系](@entry_id:156346)下，非零[联络系数](@entry_id:157618)为 $\Gamma^1_{12} = \beta y$ 和 $\Gamma^2_{21} = \delta x$。其挠率分量为 $T^1_{12} = \beta y$, $T^2_{12} = -\delta x$, $T^1_{21} = -\beta y$, $T^2_{21} = \delta x$。现在引入新[坐标系](@entry_id:156346) $(u,v)$，关系为 $x = u$ 和 $y = v(1 + \alpha u)$。我们来计算新[坐标系](@entry_id:156346)下的挠率分量 $\bar{T}^2_{12}$。根据[张量变换法则](@entry_id:185176)，并代入具体的偏导数和原[坐标系](@entry_id:156346)下的挠率分量，经过一番计算，可以得到 [@problem_id:1561593]：

$$\bar{T}^2_{12} = -\alpha \beta v^2 (1 + \alpha u) - \delta u$$

这个计算过程虽然繁琐，但它有力地证明了 $T^k_{ij}$ 确实是一个[张量场](@entry_id:190170)的分量。这意味着挠率是一个内蕴的、不依赖于观察者（[坐标系](@entry_id:156346)）的几何对象。它的存在与否，以及它在某一点的大小和方向，都是空间的客观属性。

### 计算实例与应用

掌握了挠率的定义和分量形式后，我们可以通过一系列计算来加深理解。

**1. 对一般向量场的计算**

我们不仅可以计算基底向量的挠率，也可以计算任意向量场的挠率。例如，在一个 $\mathbb{R}^3$ [流形](@entry_id:153038)上，给定联络 $\Gamma^3_{12} = x^1$ 和 $\Gamma^3_{21} = x^2$（其余为零），以及向量场 $X = x^1 \frac{\partial}{\partial x^1}$ 和 $Y = x^2 \frac{\partial}{\partial x^2}$。要计算 $T(X,Y)$，我们需要分别计算 $\nabla_X Y$, $\nabla_Y X$ 和 $[X,Y]$ [@problem_id:1685048]。利用[协变导数](@entry_id:152476)和[李括号](@entry_id:636461)的坐标表达式，我们发现 $[X,Y]=0$，而：

$(\nabla_X Y)^3 = \Gamma^3_{ij} X^i Y^j = \Gamma^3_{12} X^1 Y^2 = x^1 \cdot x^1 \cdot x^2 = (x^1)^2 x^2$
$(\nabla_Y X)^3 = \Gamma^3_{ij} Y^i X^j = \Gamma^3_{21} Y^2 X^1 = x^2 \cdot x^2 \cdot x^1 = x^1 (x^2)^2$

因此，$T(X,Y) = (\nabla_X Y - \nabla_Y X)$ 的唯一非零分量是第3分量，其值为 $(x^1)^2 x^2 - x^1 (x^2)^2 = x^1 x^2 (x^1 - x^2)$。这展示了如何将分量公式应用于一般的向量场。

**2. 挠率分量的性质**

从 $T^k_{ij} = \Gamma^k_{ij} - \Gamma^k_{ji}$ 可以直接看出，当 $i=j$ 时（无求和），$T^k_{ii} = \Gamma^k_{ii} - \Gamma^k_{ii} = 0$。这意味着[挠率张量](@entry_id:204137)对角分量（下指标相同）恒为零。这个性质在处理包含挠率的复杂表达式时非常有用。例如，在一个包含 $\sum_{l} T^l_{lj}$ 这样的项的计算中，我们只需要考虑 $l \neq j$ 的项即可 [@problem_id:1685045]。

**3. 从[无挠联络](@entry_id:181337)构造有挠联络**

挠率是联络的属性，而非[流形](@entry_id:153038)或度规的必然属性。我们可以从一个给定的[无挠联络](@entry_id:181337)（如 Levi-Civita 联络）出发，通过修改其[联络系数](@entry_id:157618)来“引入”挠率。例如，在极坐标 $(r, \theta)$ 下，欧氏度规 $ds^2 = dr^2 + r^2 d\theta^2$ 对应的 Levi-Civita 联络是[无挠的](@entry_id:161664)，其非零 Christoffel 符号为 $\Gamma^1_{22}=-r$ 和 $\Gamma^2_{12}=\Gamma^2_{21}=1/r$。现在，我们定义一个新联络 $\tilde{\nabla}$，使其大部分系数与 Levi-Civita 联络相同，但修改其中一个，例如令 $\tilde{\Gamma}^2_{12} = 3/r$。那么新[联络的挠率](@entry_id:192913)分量 $\tilde{T}^2_{12}$ 将不再为零 [@problem_id:1558737]：

$$\tilde{T}^2_{12} = \tilde{\Gamma}^2_{12} - \tilde{\Gamma}^2_{21} = \frac{3}{r} - \Gamma^2_{21} = \frac{3}{r} - \frac{1}{r} = \frac{2}{r}$$

这表明通过改变联络的定义，我们可以在平直的欧氏空间上引入非零的挠率。

**4. [挠率张量](@entry_id:204137)的模长**

既然挠率是一个张量，如果[流形](@entry_id:153038)上还定义了一个度规张量 $g_{ij}$，我们就可以定义[挠率张量](@entry_id:204137)的**模长平方**，这是一个[标量场](@entry_id:151443)，用于衡量空间中“扭曲”的剧烈程度。其定义为：

$$||T||^2 = T^k_{ij} g_{kl} g^{im} g^{jn} T^l_{mn}$$

这是一个完全的缩并，结果是一个标量。例如，在一个具有非对角度规和非[对称联络](@entry_id:187741)的假想空间中，我们可以通过计算所有非零的 $T^k_{ij}$ 分量，然后利用度规 $g_{ij}$ 及其逆 $g^{ij}$ 进行指标的上升、下降和缩并，最终得到一个依赖于坐标和常数的标量函数，它在每一点给出了挠率的强度 [@problem_id:1084467]。

### [无挠联络](@entry_id:181337)与基底的选择

我们已经知道，在[坐标基](@entry_id:270149)底 $\{\partial_i\}$ 中，联络[无挠的](@entry_id:161664)条件等价于 Christoffel 符号的对称性 $\Gamma^k_{ij} = \Gamma^k_{ji}$。一个非常重要且微妙的问题是：这个结论对于任意基底都成立吗？答案是**否定**的。

考虑一个任意的（非坐标）基底 $\{e_i\}$，也称为**[非完整标架](@entry_id:635857)**（anholonomic frame）。[联络系数](@entry_id:157618) $\omega^k_{ij}$ 在此基底下定义为 $\nabla_{e_i} e_j = \omega^k_{ij} e_k$。与[坐标基](@entry_id:270149)底不同，[非坐标基](@entry_id:160990)底向量之间的[李括号](@entry_id:636461) $[e_i, e_j]$ 通常不为零。其[李括号](@entry_id:636461)可以再次用基底展开，定义了所谓的**[结构常数](@entry_id:157960)** $C^k_{ij}$：$[e_i, e_j] = C^k_{ij} e_k$。

在这种一般情况下，[挠率张量](@entry_id:204137)的分量为：

$$T(e_i, e_j) = \nabla_{e_i} e_j - \nabla_{e_j} e_i - [e_i, e_j] = (\omega^k_{ij} - \omega^k_{ji} - C^k_{ij}) e_k$$

因此，挠率分量为 $T^k_{ij} = \omega^k_{ij} - \omega^k_{ji} - C^k_{ij}$。
此时，联络[无挠的](@entry_id:161664)条件 ($T^k_{ij} = 0$) 变为：

$$\omega^k_{ij} - \omega^k_{ji} = C^k_{ij}$$

这表明，在一个[非坐标基](@entry_id:160990)底中，**即使联络是[无挠的](@entry_id:161664)，其[联络系数](@entry_id:157618)也未必是对称的**。它们的反对称部分恰好等于基底向量的李括号（[结构常数](@entry_id:157960)）。

一个清晰的例子可以说明这一点 [@problem_id:1558721]。在标准 $\mathbb{R}^2$ 平面上，我们使用标准的平直联络（所有 $\Gamma^k_{ij}=0$，显然无挠）。现在我们不使用[坐标基](@entry_id:270149)底 $\{\partial_x, \partial_y\}$，而是选择一个新基底 $e_1 = \partial_x$, $e_2 = x \partial_y$。在这个新基底中计算[联络系数](@entry_id:157618)，我们发现：

$\nabla_{e_1} e_2 = \nabla_{\partial_x} (x \partial_y) = (\partial_x x) \partial_y + x (\nabla_{\partial_x} \partial_y) = \partial_y = \frac{1}{x} e_2$
$\nabla_{e_2} e_1 = \nabla_{x \partial_y} (\partial_x) = x (\nabla_{\partial_y} \partial_x) = 0$

因此，我们得到[联络系数](@entry_id:157618) $\omega^2_{12} = 1/x$ 和 $\omega^2_{21} = 0$。显然，$\omega^2_{12} \neq \omega^2_{21}$。[联络系数](@entry_id:157618)是**不对称的**，尽管我们从一个[无挠联络](@entry_id:181337)开始。原因在于基底的[李括号](@entry_id:636461)非零：$[e_1, e_2] = [\partial_x, x\partial_y] = (\partial_x x) \partial_y = \partial_y = \frac{1}{x} e_2$，所以[结构常数](@entry_id:157960) $C^2_{12} = 1/x$。我们可以验证无挠条件：$\omega^2_{12} - \omega^2_{21} = 1/x - 0 = 1/x = C^2_{12}$。这完美地印证了我们的理论。

### 联络的分解与几何诠释

[挠率张量](@entry_id:204137)不仅是一个独立的几何对象，它还揭示了任何[仿射联络](@entry_id:160152)的内在结构。一个重要的定理指出，**任何一个[仿射联络](@entry_id:160152) $\nabla$ 都可以唯一地分解为一个[无挠联络](@entry_id:181337) $\mathring{\nabla}$ 和一个 $(1,2)$ 型张量场 $A$ 的和**：

$$\nabla_X Y = \mathring{\nabla}_X Y + A(X, Y)$$

我们可以利用挠率来显式地构造这个分解。在[坐标系](@entry_id:156346)中，上述关系写为 $\Gamma^k_{ij} = \mathring{\Gamma}^k_{ij} + A^k_{ij}$。我们要求 $\mathring{\nabla}$ 是[无挠的](@entry_id:161664)，即其[联络系数](@entry_id:157618) $\mathring{\Gamma}^k_{ij}$ 是对称的。一个任意的量 $\Gamma^k_{ij}$ 可以被分解为其对称和反对称部分：

$$\Gamma^k_{ij} = \frac{1}{2}(\Gamma^k_{ij} + \Gamma^k_{ji}) + \frac{1}{2}(\Gamma^k_{ij} - \Gamma^k_{ji})$$

右边的第一项关于 $i,j$ 对称，第二项反对称。第二项正是 $\frac{1}{2} T^k_{ij}$。因此，我们可以立即识别出分解的各个部分 [@problem_id:1685031]：

- [无挠联络](@entry_id:181337) $\mathring{\nabla}$ 的系数为 $\Gamma^k_{ij}$ 的对称部分：$\mathring{\Gamma}^k_{ij} = \frac{1}{2}(\Gamma^k_{ij} + \Gamma^k_{ji})$
- [张量场](@entry_id:190170) $A$ 的分量为：$A^k_{ij} = \frac{1}{2}(\Gamma^k_{ij} - \Gamma^k_{ji}) = \frac{1}{2} T^k_{ij}$

从这个分解，我们还可以得到无挠部分 Christoffel 符号的另一种等价表达：

$$\mathring{\Gamma}^k_{ij} = \Gamma^k_{ij} - A^k_{ij} = \Gamma^k_{ij} - \frac{1}{2} T^k_{ij}$$

这个分解定理非常强大，它告诉我们任何联络都可以看作是一个“标准”的[对称联络](@entry_id:187741)（无挠部分）加上一个由[挠率张量](@entry_id:204137)决定的“扭曲”项。

最后，[挠率张量](@entry_id:204137)有一个优美的几何诠释。它描述了**无穷小平行四边形未能闭合的程度**。想象在[流形](@entry_id:153038)上一点 $p$ 出发，沿向量 $X$ 的方向移动一个无穷小距离 $\epsilon$，然后沿向量 $Y$ 的方向移动一个无穷小距离 $\delta$，到达点 $p_1$。再从 $p$ 点出发，先沿 $Y$ 移动 $\delta$，再沿 $X$ 移动 $\epsilon$，到达点 $p_2$。在平坦的欧氏空间中，$p_1$ 和 $p_2$ 会重合。但在一个有挠率的空间中，这两点一般会错开，它们之间的位移向量近似为 $\epsilon \delta T(X, Y)$。因此，挠率正是这种无穷小回路“不闭合”的度量，它揭示了空间的一种内在“扭曲”。这种思想在晶体[位错理论](@entry_id:160051)等物理学领域中有直接的应用。

通过本章的学习，我们建立了对[挠率张量](@entry_id:204137)的系统理解。它从一个看似抽象的定义出发，通过分量表达式与 Christoffel 符号的对称性建立了联系，并被证明是一个真正的张量。我们不仅学会了如何计算它，更重要的是理解了它在不同基底下的表现，以及它如何刻画联络的内在结构和空间的几何特性。
