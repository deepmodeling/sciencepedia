## 引言
分子对称性是自然界的一种基本属性，它深刻地影响着分子的电子结构、[光谱](@entry_id:185632)性质和[化学反应性](@entry_id:141717)。然而，如何将这种直观的几何概念转化为一个严谨且可操作的数学工具，以解决复杂的[量子化学](@entry_id:140193)问题，是理论化学中的一个核心挑战。群的表示理论，特别是大[正交定理](@entry_id:141650)，正是为应对这一挑战而生的强大框架。它允许我们将抽象的对称操作与具体的[矩阵代数](@entry_id:153824)联系起来，从而揭示隐藏在[哈密顿量](@entry_id:172864)背后的深刻结构，并系统性地简化计算。

本文旨在系统地介绍[群表示论](@entry_id:141930)的核心原理及其在化学与相关学科中的广泛应用。读者将通过三个章节的学习，逐步掌握这一强大的理论工具。在“原理与机制”一章中，我们将从基本定义出发，学习如何构建表示矩阵，理解可约与[不可约表示](@entry_id:263310)的本质区别，并深入剖析作为理论基石的大[正交定理](@entry_id:141650)。随后，在“应用与[交叉](@entry_id:147634)学科联系”一章中，我们将看到这些抽象原理如何转化为具体的化学洞见，例如构建分子[轨道](@entry_id:137151)、预测[光谱选择定则](@entry_id:139860)以及解释[晶体场](@entry_id:147193)中的[能级分裂](@entry_id:193178)。最后，“动手实践”部分将提供练习，巩固所学知识。通过本文，你将学会如何运用对称性的语言来解读和预测分子的行为。

## 原理与机制

继前一章对分子对称性及其与群论的联系进行了初步介绍之后，本章将深入探讨[群表示](@entry_id:156757)理论的核心原理与机制。我们将阐明如何将抽象的对称操作转化为具体的矩阵，并引出[可约表示与不可约表示](@entry_id:197263)的概念。本章的中心是宏伟而深刻的**大[正交定理](@entry_id:141650) (Great Orthogonality Theorem)**，我们将展示该定理如何成为一个强大的分析工具，不仅能够系统地分解表示，还能揭示量子力学[哈密顿量](@entry_id:172864)的内在结构。最后，我们将探讨一些进阶主题，如表示的忠实性、自旋的引入如何催生[双值群](@entry_id:187359)的概念，以及如何根据表示的“实在性”对其进行分类。

### 群的表示：从[对称操作](@entry_id:143398)到矩阵

在[量子化学](@entry_id:140193)中，我们关心的是对称操作如何变换描述分子状态的函数，例如[原子轨道](@entry_id:140819)或分子[轨道](@entry_id:137151)。[群表示](@entry_id:156757)理论为此提供了一套严谨的数学语言。一个群 $G$ 在矢量空间 $V$ 上的**表示 (representation)** $\Gamma$ 是一个映射（或称同态），它将群中的每一个元素 $g \in G$ 对应于 $V$ 上的一个可逆线性算符 $\Gamma(g)$，并且保持群的乘法结构。也就是说，对于任意 $g_1, g_2 \in G$，必须满足：

$$
\Gamma(g_1 g_2) = \Gamma(g_1) \Gamma(g_2)
$$

这个同态性质是表示理论的基石。它确保了矩阵的乘法忠实地反映了对称操作的复合。由此可以直接导出两个重要属性：单位元 $e \in G$ 被映射为恒等算符 $I$，即 $\Gamma(e) = I$；元素的逆 $g^{-1}$ 被映射为算符的逆，即 $\Gamma(g^{-1}) = \Gamma(g)^{-1}$。[@problem_id:2920271]

为了使这一抽象定义具体化，我们需要为矢量空间 $V$ 选取一组[基矢](@entry_id:199546) $\{ |v_1\rangle, |v_2\rangle, \dots, |v_n\rangle \}$。此时，每个线性算符 $\Gamma(g)$ 都可以用一个 $n \times n$ 的矩阵来表示。矩阵的元素 $[\Gamma(g)]_{ij}$ 通过考察[对称操作](@entry_id:143398) $g$ 对[基矢](@entry_id:199546)的作用来确定：

$$
\hat{g} |v_j\rangle = \sum_{i=1}^{n} |v_i\rangle [\Gamma(g)]_{ij}
$$

这里，$\hat{g}$ 表示作用在函数上的对称操作算符。第 $j$ 个[基矢](@entry_id:199546)经变换后的结果，其在第 $i$ 个[基矢](@entry_id:199546)上的分量，就是表示矩阵的 $(i,j)$ 元。

让我们通过一个实例来构造表示矩阵。考虑 $C_{2v}$ [点群](@entry_id:142456)，其[对称操作](@entry_id:143398)为 $\{E, C_2(z), \sigma_v(xz), \sigma_v'(yz)\}$。我们选择一个以分子中心为原点的三维[笛卡尔坐标](@entry_id:167698)[基矢](@entry_id:199546) $\{|x\rangle, |y\rangle, |z\rangle\}$。这些[基矢](@entry_id:199546)如同 $p_x, p_y, p_z$ [原子轨道](@entry_id:140819)一样变换。操作 $g$ 对点 $(x,y,z)$ 的作用规则如下：
- $E(x,y,z) \to (x,y,z)$
- $C_2(z)(x,y,z) \to (-x,-y,z)$
- $\sigma_v(xz)(x,y,z) \to (x,-y,z)$
- $\sigma_v'(yz)(x,y,z) \to (-x,y,z)$

根据这些变换规则，我们可以逐一写出表示矩阵 [@problem_id:2920306]：
- $\hat{E}$ 使所有[基矢](@entry_id:199546)不变：$\hat{E}|x\rangle = |x\rangle, \hat{E}|y\rangle = |y\rangle, \hat{E}|z\rangle = |z\rangle$。
- $\hat{C}_2(z)$ 使 $|x\rangle \to -|x\rangle, |y\rangle \to -|y\rangle, |z\rangle \to |z\rangle$。
- $\hat{\sigma}_v(xz)$ 使 $|x\rangle \to |x\rangle, |y\rangle \to -|y\rangle, |z\rangle \to |z\rangle$。
- $\hat{\sigma}_v'(yz)$ 使 $|x\rangle \to -|x\rangle, |y\rangle \to |y\rangle, |z\rangle \to |z\rangle$。

将这些变换写成矩阵形式，我们得到一个三维表示 $\Gamma_{xyz}$：
$$
\Gamma(E) = \begin{pmatrix} 1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix}, \quad
\Gamma(C_2) = \begin{pmatrix} -1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix}
$$
$$
\Gamma(\sigma_{xz}) = \begin{pmatrix} 1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix}, \quad
\Gamma(\sigma_{yz}) = \begin{pmatrix} -1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix}
$$

我们可以验证其同态性质。例如，$C_{2v}$ 群的[群乘法表](@entry_id:149069)告诉我们 $\sigma_{v}(xz) \sigma_{v}'(yz) = C_2(z)$。让我们看看对应的[矩阵乘法](@entry_id:156035)是否也成立 [@problem_id:2920306]：
$$
\Gamma(\sigma_{xz}) \Gamma(\sigma_{yz}) = \begin{pmatrix} 1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix} \begin{pmatrix} -1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix} = \begin{pmatrix} -1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix} = \Gamma(C_2)
$$
矩阵的乘法完美地再现了群操作的复合关系。

并非所有表示矩阵都是对角矩阵。例如，考虑一个由三个等价的 $1s$ [轨道](@entry_id:137151) $\{|\phi_A\rangle, |\phi_B\rangle, |\phi_C\rangle\}$ 构成的体系，它们位于一个等边三角形的顶点。该体系的[对称群](@entry_id:146083)是 $S_3$，即三个顶点的置换群。对于[置换](@entry_id:136432)操作 $(ABC)$（即 $A \to B \to C \to A$），[基矢](@entry_id:199546)的变换为：
- $(ABC)|\phi_A\rangle = |\phi_B\rangle$
- $(ABC)|\phi_B\rangle = |\phi_C\rangle$
- $(ABC)|\phi_C\rangle = |\phi_A\rangle$
相应的表示矩阵就不是对角的 [@problem_id:2920304]：
$$
\Gamma((ABC)) = \begin{pmatrix} 0  0  1 \\ 1  0  0 \\ 0  1  0 \end{pmatrix}
$$

### [可约表示与不可约表示](@entry_id:197263)

在上述 $C_{2v}$ 的例子中，我们注意到所有表示矩阵都是对角矩阵。这意味着每个[基矢](@entry_id:199546) $|x\rangle$, $|y\rangle$, $|z\rangle$在所有[对称操作](@entry_id:143398)下都只变换为自身的倍数。换言之，由 $|x\rangle$ 生成的一维[子空间](@entry_id:150286)，由 $|y\rangle$ 生成的一维[子空间](@entry_id:150286)，以及由 $|z\rangle$ 生成的一维[子空间](@entry_id:150286)，都是在 $C_{2v}$ 群操作下封闭的。这样的[子空间](@entry_id:150286)被称为 **[不变子空间](@entry_id:152829) (invariant subspace)**。[@problem_id:2920275]

一个表示如果存在一个非平凡的（即既非零空间也非全空间）[不变子空间](@entry_id:152829)，则称该表示为 **[可约表示](@entry_id:137110) (reducible representation)**。反之，如果唯一的不变子空间只有[零空间](@entry_id:171336)和其自身，那么该表示就是 **[不可约表示](@entry_id:263310) (irreducible representation, irrep)**。[不可约表示](@entry_id:263310)是[群表示](@entry_id:156757)的“基本构件”，如同化学中的元素。

对于有限群，任何[可约表示](@entry_id:137110)都可以被“完全约化”，即表示空间 $\mathcal{H}$ 可以分解为一系列相互正交的不可约[子空间](@entry_id:150286)的[直和](@entry_id:156782)：$\mathcal{H} = \bigoplus_i \mathcal{W}_i$。这意味着通过一个合适的[基变换](@entry_id:189626)（幺正变换），所有的表示矩阵可以被同时化为**块[对角形式](@entry_id:264850)**。每个对角块本身就是一个[不可约表示](@entry_id:263310)。[@problem_id:2920275]

$$
\Gamma(g) \rightarrow \Gamma'(g) = U^{-1} \Gamma(g) U =
\begin{pmatrix}
\Gamma^{(1)}(g)  0  \cdots \\
0  \Gamma^{(2)}(g)  \\
\vdots   \ddots
\end{pmatrix}
$$

在 $C_{2v}$ 的例子中，三维表示 $\Gamma_{xyz}$ 显然是可约的，因为它已经呈现为三个一维[表示的[直](@entry_id:138310)和](@entry_id:156782)。然而，如何系统地识别和构造不变子空间呢？对于 $S_3$ [群作用](@entry_id:268812)于三个 $1s$ [轨道](@entry_id:137151)的例子，我们可以尝试构造一个在所有置換操作下都保持不变的矢量。一个自然的选择是所有[基矢](@entry_id:199546)的对称组合 [@problem_id:2920304]：
$$
|\psi_{sym}\rangle = \frac{1}{\sqrt{3}} (|\phi_A\rangle + |\phi_B\rangle + |\phi_C\rangle)
$$
不难验证，任何 $S_3$ 中的[置换](@entry_id:136432)操作作用于 $|\psi_{sym}\rangle$ 都只会得到它自身。因此，由 $|\psi_{sym}\rangle$ 张成的一维[子空间](@entry_id:150286)是一个[不变子空间](@entry_id:152829)。这直接证明了该三维表示是可约的。

### [表示的特征标](@entry_id:198072)

处理高维矩阵是繁琐的。幸运的是，表示的许多重要性质都蕴含在一个更简单的量中：**特征标 (character)**。表示 $\Gamma$ 中元素 $g$ 的特征标 $\chi(g)$ 定义为其表示矩阵的迹 (trace)：
$$
\chi(g) = \mathrm{Tr}[\Gamma(g)] = \sum_i [\Gamma(g)]_{ii}
$$
迹的一个重要性质是它在相似变换下保持不变：$\mathrm{Tr}(U^{-1}AU) = \mathrm{Tr}(A)$。这意味着一个[表示的特征标](@entry_id:198072)不依赖于基的选择。此外，同一**[共轭类](@entry_id:143916) (conjugacy class)** 中的所有元素都具有相同的特征标。

例如，对于我们之前构造的 $C_{2v}$ 表示 $\Gamma_{xyz}$，其特征标为 [@problem_id:2_920306]：
- $\chi(E) = \mathrm{Tr}(\Gamma(E)) = 1+1+1 = 3$
- $\chi(C_2) = \mathrm{Tr}(\Gamma(C_2)) = -1-1+1 = -1$
- $\chi(\sigma_{xz}) = \mathrm{Tr}(\Gamma(\sigma_{xz})) = 1-1+1 = 1$
- $\chi(\sigma_{yz}) = \mathrm{Tr}(\Gamma(\sigma_{yz})) = -1+1+1 = 1$
所以，该[表示的特征标](@entry_id:198072)向量为 $(3, -1, 1, 1)$。

对于 $S_3$ 的[置换表示](@entry_id:142960)，特征标可以通过计算对角线元素之和得到。一个更快捷的方法是，置换[表示的特征标](@entry_id:198072)等于在该操作下保持不变的[基矢](@entry_id:199546)的数量 [@problem_id:2920304]。
- $e$：3个[轨道](@entry_id:137151)都不变，$\chi(e)=3$。
- $(AB)$：$|\phi_C\rangle$ 不变，$\chi((AB))=1$。
- $(ABC)$：没有[轨道](@entry_id:137151)保持不变，$\chi((ABC))=0$。

特征标构成了从复杂的[矩阵表示](@entry_id:146025)到简洁的数值向量的桥梁，并成为应用群论的核心工具。

### 大[正交定理](@entry_id:141650)及其推论

[群表示](@entry_id:156757)理论的威力集中体现在**大[正交定理](@entry_id:141650) (Great Orthogonality Theorem, GOT)** 中。这个定理描述了不可约表示[矩阵元](@entry_id:186505)之间深刻的[正交关系](@entry_id:145540)。对于一个阶为 $|G|$ 的[有限群](@entry_id:139710) $G$，其任意两个酉（unitary）[不可约表示](@entry_id:263310) $\Gamma^{(\alpha)}$ 和 $\Gamma^{(\beta)}$（维度分别为 $l_\alpha$ 和 $l_\beta$），其矩阵元满足如下关系 [@problem_id:2920303]：

$$
\sum_{R \in G} [\Gamma^{(\alpha)}_{ij}(R)]^* [\Gamma^{(\beta)}_{kl}(R)] = \frac{|G|}{l_\alpha} \delta_{\alpha\beta} \delta_{ik} \delta_{jl}
$$

这里的星号表示[复共轭](@entry_id:174690)，$\delta$ 是克罗内克符号。该公式看起来可能令人生畏，但其物理和数学含义是清晰的。它表明，如果我们将不同[不可约表示](@entry_id:263310)的任何两个矩阵元（或者同一不可约表示中不同位置的[矩阵元](@entry_id:186505)）视为分量，构成 $|G|$ 维空间中的矢量，那么这些矢量是相互正交的。

这个定理的根源在于**[舒尔引理](@entry_id:136779) (Schur's Lemma)**。[舒尔引理](@entry_id:136779)主要处理所谓的**纠缠算符 (intertwining operator)**，即满足 $M\Gamma^{(\alpha)}(g) = \Gamma^{(\beta)}(g)M$ 的矩阵 $M$。引理指出：如果 $\Gamma^{(\alpha)}$ 和 $\Gamma^{(\beta)}$ 是不等价的不可约表示，则 $M$ 必须为零矩阵；如果它们是等价的[不可约表示](@entry_id:263310)，则 $M$ 必须是[单位矩阵](@entry_id:156724)的标量倍。通过构造一个对所有群元素求平均的纠缠算符，并应用[舒尔引理](@entry_id:136779)，就可以推导出大[正交定理](@entry_id:141650)的完整形式。[@problem_id:2920255]

尽管大[正交定理](@entry_id:141650)本身非常强大，但在化学应用中，其更直接有用的是关于特征标的推论。通过对矩阵元的指标求迹，我们可以得到**特征标的[正交定理](@entry_id:141650)** [@problem_id:2920261]：

$$
\sum_{R \in G} [\chi^{(\alpha)}(R)]^* \chi^{(\beta)}(R) = |G| \delta_{\alpha\beta}
$$

将群元素按[共轭类](@entry_id:143916) $C_k$（其中包含 $n_k$ 个元素）重新组织求和，此公式变为：
$$
\sum_k n_k [\chi^{(\alpha)}(C_k)]^* \chi^{(\beta)}(C_k) = |G| \delta_{\alpha\beta}
$$
这表明，不同不可约[表示的特征标](@entry_id:198072)向量是正交的。这个简单的关系是构建和验证群的**特征标表 (character table)** 的基础。

[特征标正交性](@entry_id:188239)的最重要应用之一是**约化公式 (reduction formula)**。任何一个[可约表示](@entry_id:137110) $\Gamma_{red}$ 的特征标 $\chi_{red}$ 都可以写成其包含的不可约表示特征标的线性组合：$\chi_{red} = \sum_i a_i \chi^{(i)}$。其中，系数 $a_i$ 是[不可约表示](@entry_id:263310) $\Gamma^{(i)}$在 $\Gamma_{red}$ 中出现的次数。利用正交性，我们可以像在[傅里叶分析](@entry_id:137640)中提取系数一样，轻松地“投影”出每个系数 $a_i$ [@problem_id:2920271]：

$$
a_i = \frac{1}{|G|} \sum_{R \in G} [\chi^{(i)}(R)]^* \chi_{red}(R)
$$

让我们将此公式应用于之前的例子。对于 $C_{2v}$ 的 $\Gamma_{xyz}$ 表示，其特征标为 $(3, -1, 1, 1)$。使用 $C_{2v}$ 的标准[特征标表](@entry_id:146676)，我们可以计算出它包含的[不可约表示](@entry_id:263310)：
- $a_{A_1} = \frac{1}{4}[1(3) + 1(-1) + 1(1) + 1(1)] = 1$
- $a_{A_2} = \frac{1}{4}[1(3) + 1(-1) + (-1)(1) + (-1)(1)] = 0$
- $a_{B_1} = \frac{1}{4}[1(3) + (-1)(-1) + 1(1) + (-1)(1)] = 1$
- $a_{B_2} = \frac{1}{4}[1(3) + (-1)(-1) + (-1)(1) + 1(1)] = 1$
因此，$\Gamma_{xyz} = A_1 \oplus B_1 \oplus B_2$。这意味着 $\{|x\rangle, |y\rangle, |z\rangle\}$ [基矢](@entry_id:199546)分别属于（或者说，变换行为如同）$B_1, B_2, A_1$ 这三个不同的[不可约表示](@entry_id:263310)。

同样，对于 $S_3$ 的[置换表示](@entry_id:142960)，其特征标为 $(3, 1, 0)$（分别对应 $E$, $3 \times C_2$, $2 \times C_3$ 三个共轭类）。使用 $S_3$ 的特征标表，可以算出 $a_{A_1}=1, a_{A_2}=0, a_E=1$。因此，该三维[表示分解](@entry_id:139061)为 $\Gamma = A_1 \oplus E$。[@problem_id:2920304]

[特征标正交关系](@entry_id:143950)还可用于确定未知的特征标。例如，如果我们知道一个群的部分特征标表，可以利用正交性约束来解出剩余的未知值，这在处理复杂的群时非常有用。[@problem_id:2920261]

### 群论与量子力学：对称性匹配

[群表示](@entry_id:156757)理论与量子力学的深刻联系源于一个中心事实：分子的[哈密顿量](@entry_id:172864) $H$ 与其对称群 $G$ 的所有操作算符 $\hat{g}$ 对易，即 $[\hat{H}, \hat{g}] = 0$。

这个[对易关系](@entry_id:136780)的最重要推论是，[哈密顿量](@entry_id:172864)不会混合属于不同不可约表示的[量子态](@entry_id:146142)。也就是说，如果态 $|\psi_i\rangle$ 属于不可约表示 $\Gamma^{(i)}$，态 $|\psi_j\rangle$ 属于不等价的[不可约表示](@entry_id:263310) $\Gamma^{(j)}$ ($i \neq j$)，那么它们之间的[哈密顿量](@entry_id:172864)矩阵元必定为零：
$$
\langle \psi_i | \hat{H} | \psi_j \rangle = 0 \quad (\text{if } i \neq j)
$$
这个结论是[舒尔引理](@entry_id:136779)在量子力学中的直接体现，它构成了[光谱学](@entry_id:141940)中**选择定则 (selection rules)** 的基础。我们可以通过构造**投影算符 (projection operator)** 来严格证明这一点。对于每个不可约表示 $\Gamma^{(r)}$，可以定义一个[投影算符](@entry_id:154142) $P^{(r)}$ [@problem_id:2920275] [@problem_id:2920307]：
$$
P^{(r)} = \frac{l_r}{|G|} \sum_{g \in G} [\chi^{(r)}(g)]^* \Gamma(g)
$$
该算符能将任意矢量投影到属于 $\Gamma^{(r)}$ 表示的[子空间](@entry_id:150286)中。这些[投影算符](@entry_id:154142)是正交的（$P^{(r)}P^{(s)} = \delta_{rs} P^{(s)}$）。由于 $H$ 与所有 $\Gamma(g)$ 对易，它也必然与 $P^{(r)}$ 对易。因此：
$$
\langle \psi_r | \hat{H} | \psi_s \rangle = \langle \psi_r | \hat{H} P^{(s)} | \psi_s \rangle = \langle \psi_r | P^{(s)} \hat{H} | \psi_s \rangle = \langle (P^{(s)}\psi_r) | \hat{H} | \psi_s \rangle
$$
因为 $|\psi_r\rangle$ 属于 $\Gamma^{(r)}$ [子空间](@entry_id:150286)，当 $r \neq s$ 时，$P^{(s)}|\psi_r\rangle = 0$，所以矩阵元为零。[@problem_id:2920307]

这意味着，如果我们选择的[基矢](@entry_id:199546)是**对称匹配[基矢](@entry_id:199546) (symmetry-adapted basis)**，即每个[基矢](@entry_id:199546)都变换为某个特定的不可约表示，那么[哈密顿量](@entry_id:172864)矩阵将自动呈现[块对角结构](@entry_id:746869)。每个对角块对应一个不可约表示。这极大地简化了[量子化学](@entry_id:140193)计算，因为它允许我们将一个大的[久期方程](@entry_id:200202)问题分解为几个互不相干的小问题。[@problem_id:2920275]

值得注意的是，群论还告诉我们更多。在单个[不可约表示](@entry_id:263310) $\Gamma^{(i)}$ 所张成的[子空间](@entry_id:150286)内，[哈密顿量](@entry_id:172864)必须是单位矩阵的常数倍。这意味着，所有属于同一个不可约表示的（对称匹配）[基函数](@entry_id:170178)所对应的能级是**简并 (degenerate)** 的。然而，如果一个表示空间中包含同一个不可约表示的多个副本（例如，在 $\Gamma = 2A_1 \oplus \dots$ 的情况中），[哈密顿量](@entry_id:172864)在这些副本之间可以存在非零矩阵元，即对称性允许它们之间发生耦合。[@problem_id:2920275]

### 高级主题

#### 忠实与非[忠实表示](@entry_id:144577)

一个表示 $\Gamma$ 的**核 (kernel)** 是指群中所有被映射为恒等矩阵 $I$ 的元素的集合：$\ker\Gamma = \{g \in G \mid \Gamma(g)=I\}$。如果一个[表示的核](@entry_id:202190)只包含单位元 $e$，即 $\ker\Gamma = \{e\}$，则称该表示为**[忠实表示](@entry_id:144577) (faithful representation)**。这意味着不同的群元素被映射为不同的矩阵。从物理上看，一个表示是忠实的，意味着所选的[基函数](@entry_id:170178)集合能够“感知”到群中的每一个非平凡对称操作；没有任何[对称操作](@entry_id:143398)能让所有[基函数](@entry_id:170178)都保持不变。

反之，如果一个[表示的核](@entry_id:202190)包含非单位元，则称其为**非[忠实表示](@entry_id:144577) (unfaithful representation)**。最极端的例子是**全对称表示**（通常记为 $A_1$, $A_g$ 等），它将所有群元素都映射为 $+1$。其核是整个群 $G$。这代表了一个对所有[对称操作](@entry_id:143398)都保持不变的[基函数](@entry_id:170178)，例如水分子的 $s$ [轨道](@entry_id:137151)。[@problem_id:2920271]

一个有趣的事实是，一个忠实的表示可以由非忠实的不可约表示组合而成。例如，在 $C_{2v}$ 群中，[不可约表示](@entry_id:263310) $B_1$ 和 $B_2$ 各自都是非忠实的（它们的核分别包含 $\{E, \sigma_{xz}\}$ 和 $\{E, \sigma_{yz}\}$）。然而，它们的[直和表示](@entry_id:140467) $\Gamma = B_1 \oplus B_2$（其[基矢](@entry_id:199546)可以是 $\{p_x, p_y\}$）的核是两个子核的交集，即 $\ker(B_1) \cap \ker(B_2) = \{E\}$。因此，这个二维表示是忠实的。[@problem_id:2920271]

#### [双值群](@entry_id:187359)与自旋

当我们在[量子化学](@entry_id:140193)中考虑电子的**自旋 (spin)** 时，标准的[分子点群](@entry_id:153797)理论需要扩展。自旋是半整数角动量，描述[自旋态](@entry_id:149436)的[波函数](@entry_id:147440)（[旋量](@entry_id:158054)）在空间旋转 $2\pi$ 后，其符号会反转，即乘以 $-1$。这与旋转 $0$ 度（恒等操作）得到的结果不同。这种“双值”行为意味着旋量实际上构成了普通[点群](@entry_id:142456) $G$ 的**投影表示 (projective representation)**，其[乘法规则](@entry_id:197368)为 $\Gamma(R_1)\Gamma(R_2) = \pm\Gamma(R_1R_2)$。

为了恢复严格的[线性表示](@entry_id:139970)，我们引入了**[双值群](@entry_id:187359) (double group)** $2G$ 的概念。[双值群](@entry_id:187359)在原有群 $G$ 的基础上，增加了一个新的、与单位元 $E$ 不同的抽象元素 $\bar{E}$，它代表旋转 $2\pi$。[群的阶](@entry_id:137115)数因此加倍。在[双值群](@entry_id:187359)中，原来的投影表示变成了真正的[线性表示](@entry_id:139970)。对于整数自旋（如轨道角动量）的单值表示，$\bar{E}$ 的表示矩阵与 $E$ 相同，其特征标等于表示的维度 $d$。而对于[半整数自旋](@entry_id:148826)的双值表示，$\bar{E}$ 的表示矩阵是 $-I$，其特征标等于 $-d$。这个特征标的符号是区分单值和双值表示的关键。因此，当考虑自旋-轨道耦合等效应时，必须使用[双值群](@entry_id:187359)及其特征标表。[@problem_id:2920290]

#### 不可约表示的分类

不可约表示自身可以根据它们与复共轭的关系分为三类。这个分类可以通过**弗罗贝尼乌斯-舒尔指示量 (Frobenius-Schur indicator)** $I_\Gamma$ 来完成：
$$
I_\Gamma = \frac{1}{|G|} \sum_{g \in G} \chi(g^2)
$$
该指示量的值只能是 $1$, $0$ 或 $-1$。

- **$I_\Gamma = 1$：[实表示](@entry_id:146117) (real type)**。表示可以被变换为所有[矩阵元](@entry_id:186505)均为实数的形式。在物理上，这意味着属于该表示的[波函数](@entry_id:147440)[基矢](@entry_id:199546)可以选择为纯实函数。例如，$C_{2v}$ 群的所有不可约表示都是[实表示](@entry_id:146117)。

- **$I_\Gamma = 0$：[复表示](@entry_id:144331) (complex type)**。表示与其复[共轭表示](@entry_id:139136)不等价。这意味着无法通过[基变换](@entry_id:189626)将表示矩阵全部变为实数。属于此表示的[本征函数](@entry_id:154705)必然是复函数，且其[时间反演](@entry_id:182076)伙伴（大致为其复共轭）属于一个不等价的表示。这两者通常是简并的。例如，$C_3$群的两个一维[复表示](@entry_id:144331)就是这种情况。

- **$I_\Gamma = -1$：[伪实表示](@entry_id:191119) (pseudoreal or quaternionic type)**。表示与其复共轭等价，但无法变换为纯[实形式](@entry_id:193866)。这类表示在处理有自旋的体系时尤为重要。对于具有[半整数自旋](@entry_id:148826)的体系，时间反演对称性（若存在）要求每个能级至少是双重简并的，即**[克拉默斯简并](@entry_id:146198) (Kramers' degeneracy)**。[伪实表示](@entry_id:191119)描述的正是这种不可被对称性操作（除[时间反演](@entry_id:182076)外）解除的内禀简并。例如，[四元数群](@entry_id:147721) $Q_8$ 的二维不可约表示就是伪实的。[@problem_id:2920297]

通过这些原理与机制，群论为我们提供了一套系统而优雅的框架，用以理解和利用[分子对称性](@entry_id:202199)，从而深刻地洞察分子的[电子结构](@entry_id:145158)、[光谱](@entry_id:185632)和化学性质。
