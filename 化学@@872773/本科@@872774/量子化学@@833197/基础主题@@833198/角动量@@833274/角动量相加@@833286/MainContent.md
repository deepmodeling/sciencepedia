## 引言
在量子力学中，角动量是描述粒子状态的一个基本物理量，它包含轨道运动产生的轨道角动量和粒子内禀的自旋角动量。当一个系统包含多个粒子，或一个粒子同时拥有这两种角动量时，它们会发生复杂的相互作用或“耦合”。如何描述耦合后的总角动量及其对系统能级和性质的影响，是理解从原子光谱到材料磁性等众多[物理化学](@entry_id:145220)现象的关键。本文旨在系统地解决这一问题，为读者提供一个关于角动量加和理论的全面框架。

本文将分为三个核心部分。在“原理和机制”一章中，我们将深入探讨角动量耦合的基本数学规则，引入直观的矢量模型，并详细对比适用于不同物理情境的[LS耦合](@entry_id:137266)与[jj耦合](@entry_id:140838)方案。接下来，在“应用与跨学科联系”一章中，我们将展示这些理论如何应用于解释原子光谱的精细结构、洪特规则、[塞曼效应](@entry_id:154144)，并进一步延伸至分子磁性、核物理乃至粒子物理学等前沿领域。最后，“动手实践”部分将通过一系列精选问题，帮助读者巩固所学知识，将理论应用于解决实际问题。通过学习，您将掌握量子力学中这一强大而普适的工具。

## 原理和机制

在量子力学中，角动量是一个核心概念，它不仅包括了经典的[轨道角动量](@entry_id:191303)，还引入了粒子固有的[内禀角动量](@entry_id:189727)——自旋。当一个系统由多个粒子组成，或一个粒子同时具有[轨道](@entry_id:137151)和[自旋角动量](@entry_id:149719)时，这些不同的角动量会相互作用和耦合。理解[角动量耦合](@entry_id:145967)的原理和机制对于解释原子和分子的[电子结构](@entry_id:145158)、[光谱](@entry_id:185632)的[精细结构](@entry_id:140861)以及磁性等多种物理化学现象至关重要。本章将系统地阐述角动量加和的基本规则、物理图像、及其在[多电子原子](@entry_id:157716)中的具体应用。

### 角动量耦合的基本规则

当两个或多个角动量在一个量子系统中相互作用时，它们的矢量和——[总角动量](@entry_id:155748)——是守恒的。假设我们有两个角动量，由算符 $\vec{J}_1$ 和 $\vec{J}_2$ 表示，它们对应的量子数分别为 $j_1$ 和 $j_2$。[总角动量算符](@entry_id:149439)定义为它们的矢量和：$\vec{J} = \vec{J}_1 + \vec{J}_2$。

根据量子力学理论，[总角动量量子数](@entry_id:164948) $J$ 并非任意值，而是遵循一套严格的加和规则。$J$ 的可能取值范围是从 $|j_1 - j_2|$ 到 $j_1 + j_2$，并且以整数为步长。这可以表示为：

$J = |j_1 - j_2|, |j_1 - j_2| + 1, \dots, j_1 + j_2$

这个规则被称为 **Clebsch-Gordan 级数** 或三角不等法则。

一个典型的例子是[原子光谱](@entry_id:143136)中解释[谱线](@entry_id:193408)[精细结构](@entry_id:140861)的 **[自旋-轨道耦合](@entry_id:143520)** 现象。考虑一个原子中处于 d [轨道](@entry_id:137151)的电子 [@problem_id:1351466]。对于 d 电子，其[轨道角动量量子数](@entry_id:167573) $l=2$。作为电子，其固有的自旋角动量[量子数](@entry_id:145558) $s=1/2$。当这两个[角动量耦合](@entry_id:145967)时，总的电子[角动量量子数](@entry_id:172069) $j$ 的可[能值](@entry_id:187992)为：

$j = |l - s|, \dots, l + s = |2 - \frac{1}{2}|, \dots, 2 + \frac{1}{2}$

即 $j$ 可以取 $\frac{3}{2}, \frac{5}{2}$。这意味着，原本单一的 d [轨道能级](@entry_id:151753)，在考虑[自旋-轨道耦合](@entry_id:143520)后，会分裂成两个能量非常接近的能级，分别对应 $j=\frac{3}{2}$ 和 $j=\frac{5}{2}$ 的状态。

一个重要的基本原理是 **态数守恒**。在耦合之前，系统可以用所谓的 **非[耦合表象](@entry_id:136812)** $|j_1, m_1; j_2, m_2\rangle$ 来描述，其中 $m_1$ 和 $m_2$ 是各自角动量在 z 轴上的投影量子数。总的状态数（或简并度）是各个子系统状态数的乘积，即 $(2j_1+1)(2j_2+1)$。耦合之后，系统用 **[耦合表象](@entry_id:136812)** $|J, M; j_1, j_2\rangle$ 描述，其中 $M$ 是[总角动量](@entry_id:155748)在 z 轴上的投影[量子数](@entry_id:145558)。对于每一个可能的 $J$ 值，都有 $2J+1$ 个 $M$ 的取值（从 $-J$ 到 $J$）。所有可能的耦合态的总数是 $\sum_J (2J+1)$。态数守恒原理表明：

$(2j_1+1)(2j_2+1) = \sum_{J=|j_1-j_2|}^{j_1+j_2} (2J+1)$

在我们刚才的 $l=2, s=1/2$ 的例子中，非[耦合表象](@entry_id:136812)的态数为 $(2 \cdot 2 + 1)(2 \cdot \frac{1}{2} + 1) = 5 \times 2 = 10$。在[耦合表象](@entry_id:136812)中，我们有两个可能的 $J$ 值：$\frac{3}{2}$ 和 $\frac{5}{2}$。对应的态数为 $(2 \cdot \frac{3}{2} + 1) + (2 \cdot \frac{5}{2} + 1) = 4 + 6 = 10$。两者完全相等，验证了态数守恒原理。

### 矢量模型：一种半经典图像

为了更直观地理解角动量耦合，我们可以借助 **矢量模型**。在这个半经典的图像中，一个角动量量子数为 $j$ 的角动量被描绘成一个长度为 $\hbar\sqrt{j(j+1)}$ 的矢量。由于不确定性原理，这个矢量并非静止不动，而是在空间中绕着某个特定的轴（通常选为 z 轴）**进动**（precession），其在 z 轴上的投影是确定且量子化的，大小为 $m\hbar$，其中 $m = -j, -j+1, \dots, j$。

当两个角动量 $\vec{L}$ 和 $\vec{S}$ 耦合形成总角动量 $\vec{J} = \vec{L} + \vec{S}$ 时，矢量模型提供了一个清晰的物理图像。在没有外场的情况下，由于 $\vec{L}$ 和 $\vec{S}$ 之间的相互作用（例如[自旋-轨道耦合](@entry_id:143520)），它们各自不再是[守恒量](@entry_id:150267)。相反，$\vec{L}$ 和 $\vec{S}$ 会围绕着它们合成的总角动量矢量 $\vec{J}$ 快速进动。而总角动量 $\vec{J}$ 本身，作为一个守恒量，则会较为缓慢地绕着空间的 z 轴进动。

这个模型解释了为什么在存在耦合时，$m_l$ 和 $m_s$ 不再是“[好量子数](@entry_id:262514)”，因为 $L_z$ 和 $S_z$ 的值在快速变化。然而，$J_z$ 的值是守恒的，因此 $m_j$ 是一个[好量子数](@entry_id:262514)。

矢量模型还允许我们计算角动量矢量之间的夹角。考虑矢量关系 $\vec{J} = \vec{L} + \vec{S}$，我们可以得到 $\vec{J} \cdot \vec{J} = (\vec{L} + \vec{S}) \cdot (\vec{L} + \vec{S}) = \vec{L}^2 + \vec{S}^2 + 2\vec{L} \cdot \vec{S}$。因此，$\vec{L}$ 和 $\vec{S}$ 的[点积](@entry_id:149019)可以表示为：

$\vec{L} \cdot \vec{S} = \frac{1}{2} (\vec{J}^2 - \vec{L}^2 - \vec{S}^2)$

将[角动量算符](@entry_id:153013)的[本征值](@entry_id:154894)代入，即 $J^2 = \hbar^2 j(j+1)$, $L^2 = \hbar^2 l(l+1)$, $S^2 = \hbar^2 s(s+1)$，我们得到：

$\langle \vec{L} \cdot \vec{S} \rangle = \frac{\hbar^2}{2} [j(j+1) - l(l+1) - s(s+1)]$

另一方面，根据[点积](@entry_id:149019)的定义，$\vec{L} \cdot \vec{S} = |\vec{L}| |\vec{S}| \cos\theta$，其中 $\theta$ 是 $\vec{L}$ 和 $\vec{S}$ 之间的夹角。结合上述关系，我们可以解出 $\cos\theta$：

$\cos\theta = \frac{j(j+1) - l(l+1) - s(s+1)}{2\sqrt{l(l+1)s(s+1)}}$

例如，对于一个处于 $3d$ 态的单价离子电子（如 $Be^+$ 的[激发态](@entry_id:261453)），$l=2, s=1/2$ [@problem_id:1978401]。在总角动量最大的状态下，$j = l+s = 5/2$。代入公式计算可得：

$\cos\theta = \frac{\frac{5}{2}(\frac{5}{2}+1) - 2(2+1) - \frac{1}{2}(\frac{1}{2}+1)}{2\sqrt{2(2+1)\frac{1}{2}(\frac{1}{2}+1)}} = \frac{\frac{35}{4} - 6 - \frac{3}{4}}{2\sqrt{6 \cdot \frac{3}{4}}} = \frac{2}{2\sqrt{\frac{9}{2}}} = \frac{1}{\frac{3}{\sqrt{2}}} = \frac{\sqrt{2}}{3}$

由此得到的夹角 $\theta = \arccos(\frac{\sqrt{2}}{3}) \approx 61.87^\circ$。这个具体的角度形象地展示了在特定的耦合状态下，[轨道](@entry_id:137151)和[自旋角动量](@entry_id:149719)之间的相对取向。

### 多个角动量的耦合

当系统中存在三个或更多角动量时，我们可以通过 **逐步耦合** 的方法来确定所有可能的[总角动量](@entry_id:155748)。具体做法是先耦合其中两个角动量，得到一组中间角动量，然后将第三个角动量与这组中间角动量的每一个可能值进行耦合，以此类推。

一个重要的结论是，角动量耦合满足 **结合律**。这意味着，耦合的顺序不影响最终得到的[总角动量量子数](@entry_id:164948) $J$ 的集合。例如，对于三个角动量 $j_1, j_2, j_3$，采用 $(j_1+j_2)+j_3$ 的耦合方案与采用 $j_1+(j_2+j_3)$ 的方案，将得到完全相同的 $J$ 值集合 [@problem_id:1351472]。

让我们来看一个包含三个电子的体系，每个电子的[自旋量子数](@entry_id:142550) $s=1/2$ [@problem_id:1351454]。我们来计算可能的总自旋[量子数](@entry_id:145558) $S$。

**步骤 1：耦合前两个电子的自旋**
$s_1=1/2, s_2=1/2$。中间总自旋 $S_{12}$ 的可能值为：
$S_{12} = |s_1 - s_2|, \dots, s_1+s_2 = |\frac{1}{2}-\frac{1}{2}|, \dots, \frac{1}{2}+\frac{1}{2} = 0, 1$

**步骤 2：将第三个电子的自旋与中间[自旋耦合](@entry_id:180500)**
现在，我们将 $s_3=1/2$ 与 $S_{12}$ 的每一个可[能值](@entry_id:187992)进行耦合。

- 当 $S_{12}=0$ 时，耦合 $s_3=1/2$：
  $S = |0 - \frac{1}{2}|, \dots, 0+\frac{1}{2} = \frac{1}{2}$

- 当 $S_{12}=1$ 时，耦合 $s_3=1/2$：
  $S = |1 - \frac{1}{2}|, \dots, 1+\frac{1}{2} = \frac{1}{2}, \frac{3}{2}$

**步骤 3：合并所有可能的 S 值**
将上述所有情况得到的 $S$ 值合并，得到唯一的集合为 $\{1/2, 3/2\}$。这意味着一个三电子体系可以处于总自旋为 $1/2$ 的 **双重态** (doublet) 或[总自旋](@entry_id:153335)为 $3/2$ 的 **四重态** (quartet)。

### 在[多电子原子](@entry_id:157716)中的应用：[光谱项](@entry_id:175979)

[角动量耦合](@entry_id:145967)理论在[原子光谱学](@entry_id:155968)中有着核心的应用，它用于推导和标记[多电子原子](@entry_id:157716)的电子态，即 **[光谱项](@entry_id:175979)**。对于较轻的原子（通常原子序数 $Z \lt 40$），电子间的静电排斥作用远大于[自旋-轨道耦合](@entry_id:143520)作用，此时使用 **Russell-Saunders (LS) 耦合** 方案是很好的近似。

LS 耦合的步骤如下：
1.  将所有电子的[轨道角动量](@entry_id:191303)矢量相加，得到总轨道角动量 $\vec{L} = \sum_i \vec{l}_i$。其[量子数](@entry_id:145558) $L$ 由逐步耦合得到。
2.  将所有电子的[自旋角动量](@entry_id:149719)矢量相加，得到[总自旋角动量](@entry_id:175552) $\vec{S} = \sum_i \vec{s}_i$。其量子数 $S$ 由逐步耦合得到。
3.  最后，将[总轨道角动量](@entry_id:265302) $\vec{L}$ 和[总自旋角动量](@entry_id:175552) $\vec{S}$ 耦合，得到原子的总[电子角动量](@entry_id:198934) $\vec{J} = \vec{L} + \vec{S}$。

一个原子态由一组 $(L, S)$ 值定义，称为一个 **[光谱项](@entry_id:175979)**，记作 $^{2S+1}L$。其中 $2S+1$ 称为 **[自旋多重度](@entry_id:263865)**。$L$ 的值通常用字母表示：$L=0, 1, 2, 3, 4, \dots$ 分别对应 S, P, D, F, G, ...。由于自旋-轨道耦合，每个[光谱项](@entry_id:175979)会进一步分裂成多个 **能级**，每个能级由一个特定的 $J$ 值表征，记作 $^{2S+1}L_J$。

#### 非等价电子

当原子中的价电子处于不同的亚层时（即它们的[量子数](@entry_id:145558) $n$ 或 $l$ 不同），它们被称为 **非等价电子**。在这种情况下，所有通过角动量加和规则得到的 LS 组合都是允许的。

例如，考虑一个具有 $p^1d^1$ 电子组态的原子，即一个电子在 p 亚层 ($l_1=1$)，另一个在 d 亚层 ($l_2=2$) [@problem_id:1351456]。
- **[总轨道角动量](@entry_id:265302) L**：$L = |l_1-l_2|, \dots, l_1+l_2 = |1-2|, \dots, 1+2 = 1, 2, 3$。这对应于 P, D, F 项。
- **[总自旋角动量](@entry_id:175552) S**：两个电子 ($s_1=1/2, s_2=1/2$) 的[总自旋](@entry_id:153335)为 $S = 0, 1$。这对应于[自旋多重度](@entry_id:263865)为 1 ([单重态](@entry_id:154728)) 和 3 ([三重态](@entry_id:156705))。

由于电子是非等价的，我们可以将每个 $L$ 值与每个 $S$ 值组合，得到以下[光谱项](@entry_id:175979)：
$^1P, ^3P, ^1D, ^3D, ^1F, ^3F$

接下来，我们可以为每个项找到所有可能的总角动量 $J$ 值 [@problem_id:1351482]：
- $^1P (L=1, S=0): J=1 \rightarrow ^1P_1$
- $^3P (L=1, S=1): J=0, 1, 2 \rightarrow ^3P_0, ^3P_1, ^3P_2$
- $^1D (L=2, S=0): J=2 \rightarrow ^1D_2$
- $^3D (L=2, S=1): J=1, 2, 3 \rightarrow ^3D_1, ^3D_2, ^3D_3$
- $^1F (L=3, S=0): J=3 \rightarrow ^1F_3$
- $^3F (L=3, S=1): J=2, 3, 4 \rightarrow ^3F_2, ^3F_3, ^3F_4$

将所有唯一的 $J$ 值收集起来，我们得到集合 $\{0, 1, 2, 3, 4\}$。

#### 等价电子与泡利原理

当两个或多个电子处于同一亚层时（即它们具有相同的 $n$ 和 $l$ 量子数），它们被称为 **等价电子**。此时，**[泡利不相容原理](@entry_id:141850)** 会施加额外的限制。该原理指出，交换任意两个[费米子](@entry_id:146235)（如电子）的全部坐标（空间坐标和自旋坐标），体系的总[波函数](@entry_id:147440)必须是反对称的。

总[波函数](@entry_id:147440) $\Psi$ 可以近似写成空间部分 $\psi_{space}$ 和自旋部分 $\chi_{spin}$ 的乘积。为了使 $\Psi$ 是反对称的，必须满足：
- 如果空间部分 $\psi_{space}$ 对称，则自旋部分 $\chi_{spin}$ 必须反对称。
- 如果空间部分 $\psi_{space}$ 反对称，则自旋部分 $\chi_{spin}$ 必须对称。

最简单的例子是[氦原子](@entry_id:150244)的[基态](@entry_id:150928) $1s^2$ [@problem_id:1978412]。两个电子都在 $1s$ [轨道](@entry_id:137151)，因此它们的空间[波函数](@entry_id:147440) $\psi_{space} = \psi_{1s}(\vec{r}_1)\psi_{1s}(\vec{r}_2)$ 是对称的。为了满足泡利原理，自旋部分必须是反对称的。对于两个自旋 $1/2$ 的粒子，唯一的反对称自旋态是总自旋 $S=0$ 的单重态。因此，氦原子的[基态](@entry_id:150928)必须是 $^1S$ 项。

对于更复杂的等价电子组态，如 $np^2$ [@problem_id:1978407]，我们可以应用一个更普适的规则。对于两个等价电子，可以证明，当总[轨道角动量[量子](@entry_id:167573)数](@entry_id:145558)为 $L$ 时，空间[波函数](@entry_id:147440)的[交换对称性](@entry_id:151892)为 $(-1)^L$。而总[自旋[量子](@entry_id:142550)数](@entry_id:145558)为 $S$ 的双电子[自旋[波函](@entry_id:190161)数](@entry_id:147440)的[交换对称性](@entry_id:151892)为 $(-1)^{S+1}$（$S=0$ 对称，$S=1$ 反对称）。为了使总[波函数](@entry_id:147440)反对称，两者的对称性乘积必须为 $-1$：
$(-1)^L \cdot (-1)^{S} = -1$
这等价于 $(-1)^{L+S} = -1$，即 **$L+S$ 必须是奇数**。

对于 $np^2$ 组态 ($l_1=l_2=1$)，非等价电子会产生 $L=0,1,2$ 和 $S=0,1$ 的所有组合。但现在我们必须应用 $L+S$ 为偶数的选择定则：
- $L=0$ (S项): $S$ 必须为 $0$ ($0+0=0$，偶数)。允许 $^1S$ 项。
- $L=1$ (P项): $S$ 必须为 $1$ ($1+1=2$，偶数)。允许 $^3P$ 项。
- $L=2$ (D项): $S$ 必须为 $0$ ($2+0=2$，偶数)。允许 $^1D$ 项。

因此，对于 $np^2$ 组态，只允许存在 $^1S$, $^3P$, 和 $^1D$ 这三个[光谱项](@entry_id:175979)，远少于非等价 $p^2$ 组态的情况。

### 耦合态与量子测量

[角动量耦合](@entry_id:145967)的讨论引出了一个关键的量子力学概念：基的选取。我们可以用[非耦合基](@entry_id:156676)态 $|l, m_l; s, m_s\rangle$ 或耦合[基态](@entry_id:150928) $|j, m_j; l, s\rangle$ 来描述一个有自旋-轨道相互作用的系统。在存在这种相互作用时，[哈密顿量](@entry_id:172864)的[本征态](@entry_id:149904)是耦合态 $|j, m_j\rangle$。这意味着 $j$ 和 $m_j$ 是“[好量子数](@entry_id:262514)”，它们对应于守恒的物理量。相反，$m_l$ 和 $m_s$ 不再是[好量子数](@entry_id:262514)，因为 $\hat{L}_z$ 和 $\hat{S}_z$ 算符与包含自旋-[轨道](@entry_id:137151)项的[哈密顿量](@entry_id:172864)不对易。

这在[量子测量](@entry_id:272490)中具有深刻的含义 [@problem_id:1978435]。假设一个系统被制备在某个特定的精细结构能级上，例如一个 p 电子 ($l=1, s=1/2$) 处于 $j=3/2, m_j=1/2$ 的状态。如果我们此时去测量该电子的轨道角动量 z 分量 ($L_z$)，结果会是什么？

由于系统状态不是 $L_z$ 的[本征态](@entry_id:149904)，测量结果将是概率性的。为了计算不同结果的概率，我们需要将耦合态 $|j, m_j\rangle$ 展开为非耦合态 $|m_l, m_s\rangle$ 的[线性组合](@entry_id:154743)。这种展开的系数被称为 **Clebsch-Gordan 系数**。对于这个例子，展开式为：

$|j=\frac{3}{2}, m_j=\frac{1}{2}\rangle = \sqrt{\frac{2}{3}}|l=1, m_l=0\rangle|s=\frac{1}{2}, m_s=\frac{1}{2}\rangle + \sqrt{\frac{1}{3}}|l=1, m_l=1\rangle|s=\frac{1}{2}, m_s=-\frac{1}{2}\rangle$

根据[量子测量](@entry_id:272490)的基本假设，测量 $L_z$ 的可能结果是其[本征值](@entry_id:154894)，即 $m_l\hbar$。得到某个特定结果的概率是其对应[基态](@entry_id:150928)在展开式中系数的模方。因此：
- 测量到 $L_z=0\hbar$ (即 $m_l=0$) 的概率是 $(\sqrt{2/3})^2 = 2/3$。
- 测量到 $L_z=1\hbar$ (即 $m_l=1$) 的概率是 $(\sqrt{1/3})^2 = 1/3$。
- 测量到 $m_l=-1$ 的概率是 $0$，因为该项没有出现在展开式中。

这个例子清楚地表明，即使系统处于一个[总角动量](@entry_id:155748)完全确定的状态，其子角动量的分量也可能是不确定的，只能通过概率来描述。

### 高级主题：LS 耦合与 jj 耦合方案

我们已经详细讨论了 LS 耦合，它适用于轻原子。然而，随着[原子序数](@entry_id:139400) $Z$ 的增加，[原子核](@entry_id:167902)对电子的库仑作用增强，导致每个电子的自旋-轨道相互作用急剧增强（其强度约与 $Z^4$ 成正比）[@problem_id:2760475]。对于重原子，自旋-[轨道](@entry_id:137151)作用可能强于电子间的静电排斥作用。在这种情况下，LS 耦合不再是好的近似，需要采用另一种称为 **jj 耦合** 的方案。

两种耦合方案的物理基础和数学流程对比如下 [@problem_id:2760452]：

- **LS 耦合 (适用于轻原子)**：
  - **物理图像**：电子间静电作用 $\gg$ 自旋-[轨道](@entry_id:137151)作用。
  - **流程**：$(\sum_i \vec{l}_i) + (\sum_i \vec{s}_i) \rightarrow \vec{L} + \vec{S} \rightarrow \vec{J}$。
  - **[好量子数](@entry_id:262514) (近似)**：$L, S, J, M_J$。能级首先按 $(L, S)$ 聚集成[光谱项](@entry_id:175979)，然后自旋-[轨道](@entry_id:137151)作用将每个项分裂成 $J$ 能级。

- **jj 耦合 (适用于重原子)**：
  - **物理图像**：自旋-[轨道](@entry_id:137151)作用 $\gg$ 电子间静电作用。
  - **流程**：对于每个电子，先耦合其自身的[轨道](@entry_id:137151)和[自旋角动量](@entry_id:149719) $\vec{j}_i = \vec{l}_i + \vec{s}_i$。然后将所有单个电子的总角动量 $\vec{j}_i$ 耦合起来，得到原子的[总角动量](@entry_id:155748) $\vec{J} = \sum_i \vec{j}_i$。
  - **[好量子数](@entry_id:262514) (近似)**：各个 $j_i$, $J, M_J$。能级首先根据每个电子的 $j_i$ 值分裂，然后较弱的电子间相互作用再将这些[能级分裂](@entry_id:193178)成不同的总 $J$ 值。

值得强调的是，LS 耦合和 jj 耦合只是描述同一个量子系统的两种不同的 **极限近似** 和 **[基组](@entry_id:160309)**。对于任何给定的原子组态，无论采用哪种方案，最终推导出的可能总角动量 $J$ 的集合是完全相同的，并且总的[量子态](@entry_id:146142)数目也必须守恒。例如，对于之前讨论的 $p^1d^1$ 组态，在 jj 耦合方案下，我们首先计算单个电子的 $j$ 值：
- p 电子 ($l=1, s=1/2$): $j_1 = 1/2, 3/2$。
- d 电子 ($l=2, s=1/2$): $j_2 = 3/2, 5/2$。
然后将这些 $(j_1, j_2)$ 对进行耦合：
- $(j_1,j_2) = (\frac{1}{2}, \frac{3}{2}) \rightarrow J = 1, 2$
- $(j_1,j_2) = (\frac{1}{2}, \frac{5}{2}) \rightarrow J = 2, 3$
- $(j_1,j_2) = (\frac{3}{2}, \frac{3}{2}) \rightarrow J = 0, 1, 2, 3$
- $(j_1,j_2) = (\frac{3}{2}, \frac{5}{2}) \rightarrow J = 1, 2, 3, 4$
将所有这些 $J$ 值收集起来，我们得到的唯一值集合是 $\{0, 1, 2, 3, 4\}$，与 LS 耦合得到的结果完全一致。

在现实中，许多原子处于这两种极限情况之间，称为 **[中间耦合](@entry_id:167774)**。在这种情况下，$L, S$ 和各个 $j_i$ 都不是[好量子数](@entry_id:262514)，只有 $J$ 才是严格的[好量子数](@entry_id:262514)。真实的能级结构需要通过对包含所有相互作用的[哈密顿量](@entry_id:172864)矩阵进行数值对角化来获得，而 LS 耦合态或 jj 耦合态则可以作为进行这种计算的方便的[基组](@entry_id:160309)。因此，对这两种耦合方案的透彻理解，为我们分析和预测从轻原子到重原子的整个[元素周期表](@entry_id:190860)中的[原子结构](@entry_id:137190)和[光谱](@entry_id:185632)提供了坚实的基础。[@problem_id:2760475]
