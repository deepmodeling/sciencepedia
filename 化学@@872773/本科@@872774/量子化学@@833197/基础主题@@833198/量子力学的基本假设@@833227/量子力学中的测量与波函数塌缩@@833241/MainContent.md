## 引言
量子世界中的“观察”行为与我们的日常经验截然不同。在经典物理中，测量是一个被动的过程，可以无限精确地记录系统既有的属性而不干扰它。然而，在量子力学中，测量不仅是主动的、介入性的，其结果更是根本上概率性的。这种奇异的性质构成了[量子理论](@entry_id:145435)最深刻、最反直觉的核心之一，即“[测量问题](@entry_id:189139)”。理解量子测量不仅是理论物理学家的追求，也是化学家、[材料科学](@entry_id:152226)家和工程师驾驭微观世界以开发新技术的关键。本文旨在系统地揭开量子测量的神秘面纱，阐明其基本原理，并展示其在不同科学和工程领域中的广泛应用。

在接下来的内容中，我们将分三步深入探讨这一主题。首先，在“原理与机制”一章中，我们将详细阐述波恩法则、[波函数坍缩](@entry_id:152132)、投影算符以及测量不[相容可观测量](@entry_id:151766)和纠缠系统时的独特后果，为整个理论奠定坚实的数学和物理基础。接着，在“应用与[交叉](@entry_id:147634)学科联系”一章，我们将探索这些原理如何应用于现实世界，从解释分子[光谱](@entry_id:185632)和[化学反应动力学](@entry_id:274455)，到驱动[量子计算](@entry_id:142712)和纳米技术等前沿科技，并触及[退相干](@entry_id:145157)和现实本质等深刻问题。最后，“动手实践”部分将提供一系列精心设计的问题，帮助您将抽象的理论转化为具体的计算和物理直觉。

## 原理与机制

在量子力学中，测量是一个核心且独特的概念，它不仅是从系统中提取信息的过程，更是深刻改变系统状态的物理作用。与经典物理学中观察者可以被动地记录系统属性不同，[量子测量](@entry_id:272490)是主动的、介入性的，并且其结果本质上是概率性的。本章将系统地阐述[量子测量](@entry_id:272490)的基本原理和核心机制，包括波恩法则、[波函数坍缩](@entry_id:152132)、[投影算符](@entry_id:154142)、不[相容可观测量](@entry_id:151766)以及对纠缠系统和[广义测量](@entry_id:154280)的讨论。

### 测量假设：从[概率幅](@entry_id:150609)到概率

[量子态](@entry_id:146142)由[希尔伯特空间](@entry_id:261193)中的一个态矢量 $|\psi\rangle$ 描述。对于任何一个[物理可观测量](@entry_id:154692)（如能量、位置、动量），都有一个相应的厄米算符 $\hat{A}$。该算符拥有一组完备的正交归一的[本征态](@entry_id:149904) $\{|\phi_k\rangle\}$，其对应的[本征值](@entry_id:154894)为 $\{a_k\}$。根据[叠加原理](@entry_id:144649)，系统的任意态矢量 $|\psi\rangle$ 都可以展开为这些[本征态](@entry_id:149904)的[线性组合](@entry_id:154743)：

$$|\psi\rangle = \sum_{k} c_k |\phi_k\rangle$$

这里的系数 $c_k$ 是复数，称为**概率幅 (probability amplitude)**。它通过[内积](@entry_id:158127)计算得出：$c_k = \langle \phi_k | \psi \rangle$。

概率幅本身不是一个可以直接测量的物理量，但它包含了关于测量结果的全部概率信息。这由量子力学的基本假设之一——**波恩法则 (Born rule)** 所规定：当对处于 $|\psi\rangle$ 态的系统测量[可观测量](@entry_id:267133) $\hat{A}$ 时，得到结果为特定[本征值](@entry_id:154894) $a_k$ 的概率 $P_k$ 等于相应概率幅模的平方：

$$P_k = |c_k|^2 = |\langle \phi_k | \psi \rangle|^2$$

由于 $|\psi\rangle$ 和 $|\phi_k\rangle$ 通常是归一化的，所有可能结果的概率之和为1，即 $\sum_k P_k = \sum_k |c_k|^2 = 1$。

值得注意的是，概率幅 $c_k$ 是一个复数，可以写成极坐标形式 $c_k = |c_k| \exp(i\theta_k)$。波恩法则只涉及其模 $|c_k|$，那么相位 $\theta_k$ 的物理意义是什么呢？单个[概率幅](@entry_id:150609)的绝对相位是不可测量的。例如，将整个态矢量乘以一个[全局相位](@entry_id:147947)因子 $\exp(i\alpha)$，即 $|\psi'\rangle = \exp(i\alpha)|\psi\rangle$，所有的[概率幅](@entry_id:150609)都会变为 $c'_k = \exp(i\alpha)c_k$。然而，测量概率 $P_k = |c'_k|^2 = |\exp(i\alpha)c_k|^2 = |c_k|^2$ 保持不变。因此，[全局相位](@entry_id:147947)没有物理意义。

然而，不同[概率幅](@entry_id:150609)之间的**相对相位 (relative phases)** 具有至关重要的物理意义。它们决定了量子**干涉 (interference)** 现象的结果 [@problem_id:1380376]。考虑一个简单的双能级系统，其状态为 $|\psi\rangle = c_0|0\rangle + c_1|1\rangle$。如果我们测量一个在 $\{|0\rangle, |1\rangle\}$ 基下的可观测量，得到结果0或1的概率分别为 $|c_0|^2$ 和 $|c_1|^2$，这与 $c_0$ 和 $c_1$ 之间的[相对相位](@entry_id:148120)无关。但是，如果我们测量另一个可观测量，其本征态是 $\{|+\rangle, |-\rangle\}$，其中 $|\pm\rangle = \frac{1}{\sqrt{2}}(|0\rangle \pm |1\rangle)$，情况就不同了。测量得到 "+" 结果的[概率幅](@entry_id:150609)为：

$$a_+ = \langle + | \psi \rangle = \frac{1}{\sqrt{2}}(\langle 0| + \langle 1|)(c_0|0\rangle + c_1|1\rangle) = \frac{1}{\sqrt{2}}(c_0 + c_1)$$

相应的概率为：

$$P_+ = |a_+|^2 = \frac{1}{2}|c_0 + c_1|^2 = \frac{1}{2}(c_0 + c_1)(c_0^* + c_1^*) = \frac{1}{2}(|c_0|^2 + |c_1|^2 + c_0c_1^* + c_1c_0^*)$$

令 $c_0 = |c_0|\exp(i\theta_0)$ 和 $c_1 = |c_1|\exp(i\theta_1)$，上式中的[交叉](@entry_id:147634)项变为 $2\text{Re}(c_0c_1^*) = 2|c_0||c_1|\cos(\theta_0 - \theta_1)$。可见，测量概率 $P_+$ 明确地依赖于相对相位 $\delta = \theta_0 - \theta_1$。这正是量子干涉的核心：不同“路径”（在这里指 $|0\rangle$ 和 $|1\rangle$ 分量）的[概率幅](@entry_id:150609)可以相长或相消，从而影响最终的测量概率。

### [波函数坍缩](@entry_id:152132)：投影假设

波恩法则描述了测量得到某个结果的概率，但它没有说明测量行为本身对系统状态的影响。量子力学的另一个核心假设，即**投影假设 (projection postulate)** 或**[波函数坍缩](@entry_id:152132) (wavefunction collapse)**，填补了这一空白。它规定：

**如果对处于 $|\psi\rangle$ 态的系统测量可观测量 $\hat{A}$ 并得到了非简并的[本征值](@entry_id:154894) $a_k$，那么在测量完成的瞬间，系统的状态会从原始的叠加态 $|\psi\rangle$ 立即变为与该测量结果对应的本征态 $|\phi_k\rangle$。**

这个过程被称为“坍缩”，因为它将一个包含多种可能性的叠加态“压缩”到一个确定的状态。

让我们通过一个具体例子来理解这一过程 [@problem_id:1380398]。考虑一个被限制在[一维无限深势阱](@entry_id:271157)中的粒子，其初始状态是一个归一化的叠加态：
$$|\Psi(0)\rangle = \frac{2}{\sqrt{5}}|\phi_1\rangle + \frac{1}{\sqrt{5}}|\phi_2\rangle$$
其中 $|\phi_1\rangle$ 和 $|\phi_2\rangle$ 分别是能量为 $E_1$ 和 $E_2$ 的[基态](@entry_id:150928)和第一[激发态](@entry_id:261453)。根据波恩法则，测量能量得到 $E_1$ 的概率是 $|\frac{2}{\sqrt{5}}|^2 = \frac{4}{5}$，得到 $E_2$ 的概率是 $|\frac{1}{\sqrt{5}}|^2 = \frac{1}{5}$。

假设一次能量测量实际得到的结果是[基态能量](@entry_id:263704) $E_1$。根据投影假设，在测量之后，系统的状态立即从 $|\Psi(0)\rangle$ 坍缩为对应的能量本征态 $|\phi_1\rangle$。此时，系统不再处于叠加态。如果我们紧接着计算任何其他物理量的[期望值](@entry_id:153208)，都必须在这个新的、坍缩后的状态 $|\phi_1\rangle$ 上进行。例如，位置平方的[期望值](@entry_id:153208) $\langle x^2 \rangle$ 在测量后变为：

$$\langle x^2 \rangle_{post} = \langle \phi_1 | \hat{x}^2 | \phi_1 \rangle = \int_0^L \phi_1^*(x) x^2 \phi_1(x) dx$$

这个计算结果将是一个确定的值（对于一维[势阱](@entry_id:151413)，结果为 $L^2(\frac{1}{3} - \frac{1}{2\pi^2})$），它不同于在初始叠加态 $|\Psi(0)\rangle$ 下计算得到的[期望值](@entry_id:153208)。这清晰地表明，测量行为彻底改变了系统的状态，进而改变了其未来的演化和性质。

### [投影算符](@entry_id:154142)的形式化

[波函数坍缩](@entry_id:152132)的概念可以通过**[投影算符](@entry_id:154142) (projection operators)** 进行更普适和严格的数学表述。对于一个由[可观测量](@entry_id:267133) $\hat{A}$ 的归一化本征态 $|\phi_k\rangle$ 张成的完备空间，我们可以定义一系列投影算符。

对于一个非简并的[本征值](@entry_id:154894) $a_k$，其对应的[投影算符](@entry_id:154142) $\hat{P}_k$ 定义为：

$$\hat{P}_k = |\phi_k\rangle\langle\phi_k|$$

这是一个外积。当 $\hat{P}_k$ 作用于任意态矢量 $|\psi\rangle$ 时，它会将其投影到由 $|\phi_k\rangle$ 定义的方向上：$\hat{P}_k |\psi\rangle = |\phi_k\rangle\langle\phi_k|\psi\rangle = c_k |\phi_k\rangle$。投影算符是厄米的 ($\hat{P}_k^\dagger = \hat{P}_k$) 且幂等的 ($\hat{P}_k^2 = \hat{P}_k$) 。

利用投影算符，我们可以更优雅地表述测量过程：
1.  **概率**：测量[可观测量](@entry_id:267133) $\hat{A}$ 得到结果 $a_k$ 的概率是 $P_k = \langle \psi | \hat{P}_k | \psi \rangle = \| \hat{P}_k |\psi\rangle \|^2$。
2.  **坍缩**：如果测量结果是 $a_k$，测量后的状态 $|\psi_{post}\rangle$ 是初始状态 $|\psi_{pre}\rangle$ 在 $|\phi_k\rangle$ 上的投影，并重新归一化：

    $$|\psi_{post}\rangle = \frac{\hat{P}_k |\psi_{pre}\rangle}{\sqrt{\langle\psi_{pre}|\hat{P}_k|\psi_{pre}\rangle}} = \frac{\hat{P}_k |\psi_{pre}\rangle}{\| \hat{P}_k |\psi_{pre}\rangle \|}$$

这个形式化的优点在于它可以自然地推广到**简并 (degenerate)** 的情况。如果一个[本征值](@entry_id:154894) $a$ 对应于一个由多个正交归一的本征态 $\{|\phi_{a,i}\rangle\}_{i=1}^d$ 张成的 $d$ 维[子空间](@entry_id:150286)，那么测量得到结果 $a$ 意味着系统坍缩到了这个[子空间](@entry_id:150286)。相应的投影算符是投影到该[子空间](@entry_id:150286)中所有[基矢](@entry_id:199546)上的投影算符之和 [@problem_id:1380385]：

$$\hat{P}_a = \sum_{i=1}^d |\phi_{a,i}\rangle\langle\phi_{a,i}|$$

例如，要构建一个算符来判断氢原子中的电子是否处于[主量子数](@entry_id:143678) $n=2$ 的能层，我们需要投影到所有 $n=2$ 态张成的[子空间](@entry_id:150286)。对于 $n=2$，[角量子数](@entry_id:164193) $l$ 可以是 0 或 1。当 $l=0$ 时，$m_l=0$；当 $l=1$ 时，$m_l = -1, 0, 1$。因此，$n=2$ 的[能层](@entry_id:160747)是四重简并的，由态 $|2,0,0\rangle, |2,1,-1\rangle, |2,1,0\rangle, |2,1,1\rangle$ 张成。对应的投影算符即为：

$$\hat{P}_{n=2} = |2,0,0\rangle\langle 2,0,0| + |2,1,-1\rangle\langle 2,1,-1| + |2,1,0\rangle\langle 2,1,0| + |2,1,1\rangle\langle 2,1,1| = \sum_{l=0}^{1} \sum_{m_l=-l}^{l} |2,l,m_l\rangle\langle 2,l,m_l|$$

如果一个初始态 $|\psi\rangle$ 经过测量后确认电子处于 $n=2$ [能层](@entry_id:160747)，其末态将是 $|\psi_{post}\rangle = \frac{\hat{P}_{n=2}|\psi\rangle}{\|\hat{P}_{n=2}|\psi\rangle\|}$，这是一个 $n=2$ 各态的叠加态。

[投影算符](@entry_id:154142)在处理对称性测量时也同样强大 [@problem_id:1380351]。考虑一个处于叠加态 $\Psi(x,0) = \frac{1}{\sqrt{6}}(\psi_1 + 2\psi_2 + \psi_3)$ 的粒子在一维[势阱](@entry_id:151413)中。定义一个关于中心 $x=L/2$ 的反射算符 $\hat{S}$，其作用为 $\hat{S}f(x) = f(L-x)$。[势阱](@entry_id:151413)的能量本征态 $\psi_n(x) = \sqrt{\frac{2}{L}}\sin(\frac{n\pi x}{L})$ 是 $\hat{S}$ 的本征态，其[本征值](@entry_id:154894)为 $(-1)^{n+1}$。因此，$n$ 为奇数的态（如 $\psi_1, \psi_3$）是 $\hat{S}$ 的[本征值](@entry_id:154894)为 $+1$ 的[本征态](@entry_id:149904)（对称），$n$ 为偶数的态（如 $\psi_2$）是[本征值](@entry_id:154894)为 $-1$ 的[本征态](@entry_id:149904)（反对称）。如果对系统测量 $\hat{S}$ 并得到结果 $+1$，则意味着状态坍缩到了由所有[本征值](@entry_id:154894)为 $+1$ 的本征态张成的[子空间](@entry_id:150286)。初始态中，只有 $\psi_1$ 和 $\psi_3$ 分量被保留。测量后的未归一化态为 $\frac{1}{\sqrt{6}}(\psi_1 + \psi_3)$，归一化后为 $\frac{1}{\sqrt{2}}(\psi_1 + \psi_3)$。系统从一个混合对称性的态坍缩成了一个纯对称的态。

### 测量与不[相容可观测量](@entry_id:151766)

两个物理量是否可以同时被精确测量，取决于它们对应算符的[对易关系](@entry_id:136780)。如果两个算符 $\hat{A}$ 和 $\hat{B}$ 对易，即 $[\hat{A}, \hat{B}] = \hat{A}\hat{B} - \hat{B}\hat{A} = 0$，则它们被称为**[相容可观测量](@entry_id:151766) (compatible observables)**。[相容可观测量](@entry_id:151766)拥有一组共同的本征态。因此，如果系统处于它们的一个共同[本征态](@entry_id:149904)，同时测量 $\hat{A}$ 和 $\hat{B}$ 将得到确定的值，并且测量过程不会相互干扰。

一个重要的推论是，如果系统已经处于[哈密顿算符](@entry_id:144286) $\hat{H}$ 的一个非简并本征态 $|E_k\rangle$，此时测量任何与 $\hat{H}$ 对易的[可观测量](@entry_id:267133) $\hat{A}$，将不会改变系统的状态 [@problem_id:1380346]。因为 $[\hat{H}, \hat{A}]=0$ 保证了 $|E_k\rangle$ 同时也是 $\hat{A}$ 的一个[本征态](@entry_id:149904)。测量只是确认了 $\hat{A}$ 的[本征值](@entry_id:154894)，而系统早已处于该本征态，因此无需坍缩。

反之，如果两个算符不对易，$[\hat{A}, \hat{B}] \neq 0$，它们就是**不[相容可观测量](@entry_id:151766) (incompatible observables)**。它们不共享一套完备的共同[本征态](@entry_id:149904)。这意味着，一个算符的[本征态](@entry_id:149904)通常是另一个算符的多个本征态的叠加。根据海森堡不确定性原理，我们无法同时精确地知道这两个物理量的值。对其中一个量的测量，必然会干扰另一个量。

具体来说，如果系统处于 $\hat{A}$ 的本征态 $|\phi_k\rangle$，此时测量 $\hat{B}$，系统将会坍缩到 $\hat{B}$ 的某个[本征态](@entry_id:149904) $|\chi_j\rangle$。由于 $|\phi_k\rangle$ 不是 $\hat{B}$ 的[本征态](@entry_id:149904)，它可以展开为 $|\phi_k\rangle = \sum_j d_j |\chi_j\rangle$。因此，测量后系统状态发生改变，并且关于物理量 $\hat{A}$ 的信息变得不确定（系统不再处于 $\hat{A}$ 的本征态）。

让我们看两个典型例子：
1.  **位置与能量**：在[一维无限深势阱](@entry_id:271157)中，位置算符 $\hat{x}$ 和[哈密顿算符](@entry_id:144286) $\hat{H}$ 不对易。假设粒子初始处于能量[基态](@entry_id:150928) $|\psi_1\rangle$。现在我们进行一次位置测量，发现粒子位于区间的左三分之一，即 $0  x  L/3$ [@problem_id:1380360]。这次测量将[波函数](@entry_id:147440)“截断”，使其只在 $0  x  L/3$ 区间内非零。这个新的、坍缩后的[波函数](@entry_id:147440)显然不再是原来的正弦函数 $\psi_1(x)$，因此它不再是能量的本征态。它变成了一个各种[能量本征态](@entry_id:152154)的叠加。因此，如果紧接着测量能量，我们有一定概率得到除[基态能量](@entry_id:263704) $E_1$ 之外的其他能量值。测量的顺序决定了系统的[状态和](@entry_id:193625)性质。

2.  **动量与能量**：同样在一维[势阱](@entry_id:151413)中，[动量算符](@entry_id:151743) $\hat{p} = -i\hbar \frac{d}{dx}$ 与[哈密顿算符](@entry_id:144286) $\hat{H}$ 也不对易。粒子初始处于[基态](@entry_id:150928) $|\psi_1\rangle$。[基态](@entry_id:150928)[波函数](@entry_id:147440)是驻波，可以看作是两个方向相反的行进波（动量本征态）的叠加，因此其动量不确定。如果我们进行一次精确的动量测量，并得到结果 $p = +\frac{\pi\hbar}{L}$ [@problem_id:1380394]，根据投影假设，系统状态将坍缩为对应的动量[本征态](@entry_id:149904)，其[波函数](@entry_id:147440)形式为 $\phi_p(x) = \frac{1}{\sqrt{L}}\exp(ipx/\hbar)$。这个[平面波](@entry_id:189798)状态显然不是[势阱](@entry_id:151413)的能量本征态（它不满足边界条件）。为了确定其能量[分布](@entry_id:182848)，我们需要将其在能量本征基 $\{|\psi_n\rangle\}$ 下展开。计算展开系数 $a_n = \langle \psi_n | \phi_p \rangle$，我们就可以得到测量到任意能量 $E_n$ 的概率 $P_n = |a_n|^2$。这再次说明，对一个不[相容可观测量](@entry_id:151766)的测量会“摧毁”系统原有的确定状态，并将其置于一个新的叠加态中。

### 纠缠体系的测量

当测量作用于由多个粒子组成的**纠缠系统 (entangled system)** 时，其后果更显奇特。纠缠是指一个[多粒子系统](@entry_id:192694)的状态无法表示为各个子系统状态的乘积。这些粒子构成一个不可分割的整体，即使它们在空间上相距遥远。

最著名的例子是[自旋单重态](@entry_id:153133) (spin-singlet state) 的双电子系统 [@problem_id:1380375]。其态矢量为：
$$|\Psi\rangle = \frac{1}{\sqrt{2}} \left( |\alpha(1)\beta(2)\rangle - |\beta(1)\alpha(2)\rangle \right)$$
其中 $|\alpha\rangle$ 代表自旋向上（$S_z$ [本征值](@entry_id:154894)为 $+\hbar/2$），$|\beta\rangle$ 代表自旋向下（$S_z$ [本征值](@entry_id:154894)为 $-\hbar/2$）。这个状态的特点是：我们无法单独描述电子1或电子2的自旋状态；我们只知道它们的[总自旋](@entry_id:153335)为零，且自旋方向总是相反的。

假设实验者Alice对她手中的电子1进行 $S_{z,1}$ 的测量，并得到结果 $+\hbar/2$。这个测量对应于将系统投影到电子1自旋向上的[子空间](@entry_id:150286)。该[投影算符](@entry_id:154142)为 $\hat{P}_{\alpha(1)} = (|\alpha(1)\rangle\langle\alpha(1)|) \otimes I_2$，其中 $I_2$ 是作用于电子2的恒等算符。将此算符作用于初始的纠缠态：

$$ \hat{P}_{\alpha(1)}|\Psi\rangle = \frac{1}{\sqrt{2}} \left( \hat{P}_{\alpha(1)}|\alpha(1)\beta(2)\rangle - \hat{P}_{\alpha(1)}|\beta(1)\alpha(2)\rangle \right) $$

由于 $\langle\alpha(1)|\alpha(1)\rangle = 1$ 和 $\langle\alpha(1)|\beta(1)\rangle = 0$，上式变为：

$$ \hat{P}_{\alpha(1)}|\Psi\rangle = \frac{1}{\sqrt{2}} (|\alpha(1)\beta(2)\rangle - 0) = \frac{1}{\sqrt{2}}|\alpha(1)\beta(2)\rangle $$

将这个坍缩后的态矢量重新归一化（除以其模 $\frac{1}{\sqrt{2}}$），我们得到测量后的最终状态：

$$ |\Psi_{post}\rangle = |\alpha(1)\beta(2)\rangle $$

这个结果揭示了纠缠测量的惊人之处：Alice对电子1的测量，瞬间决定了远方电子2的状态。在测量之前，电子2的自旋方向是不确定的；在Alice测得电子1自旋向上之后，电子2的状态立即坍缩为确定的自旋向下态 $|\beta(2)\rangle$。这种超越空间的瞬时关联，被爱因斯坦称为“[鬼魅般的超距作用](@entry_id:143486)”，是量子力学非局域性的深刻体现。

### [广义测量](@entry_id:154280)与[弱测量](@entry_id:139653)

前面讨论的投影测量是一种理想化的“强测量”，它要么给出确切的[本征值](@entry_id:154894)并将系统完全投影到相应的本征态，要么给出“否”的回答。然而，现实中的测量过程更加多样和复杂。**[广义测量](@entry_id:154280) (generalized measurement)** 理论为描述这些非理想情况提供了框架。

一种常见的情形是**粗糙测量 (coarse measurement)**，即测量仪器的分辨率有限，只能确定物理量位于某个范围内。例如，我们不精确测量粒子的位置，而只是判断它位于[势阱](@entry_id:151413)的左半边（$0 \le x \le L/2$）还是右半边 [@problem_id:1380359]。如果初始状态是[基态](@entry_id:150928) $\psi_1(x)$，测量发现粒子在左半边，那么[波函数坍缩](@entry_id:152132)后的状态是原始[波函数](@entry_id:147440)被“截断”到该区域，然后重新归一化：

$$ \psi_{post}(x) = \begin{cases} N \psi_1(x)  \text{ if } 0 \le x \le L/2 \\ 0  \text{ otherwise} \end{cases} $$

其中 $N$ 是归一化常数，由 $N^2 \int_0^{L/2} |\psi_1(x)|^2 dx = 1$ 决定。这个新的状态 $\psi_{post}(x)$ 描述了我们关于粒子位置的新知识。如果我们在此基础上进行更精细的测量，比如寻找粒子在 $0 \le x \le L/4$ 的概率，我们只需计算 $|\psi_{post}(x)|^2$ 在该区间的积分。这本质上是一个[条件概率](@entry_id:151013)问题。

更进一步，**[弱测量](@entry_id:139653) (weak measurement)** 是一种从系统中提取少量信息，同时对系统状态的扰动也尽可能小的测量方式。与强测量中突兀的投影不同，[弱测量](@entry_id:139653)可以被建模为测量仪器与系统之间的微弱耦合。例如，一个不精确的位置测量可以被描述为一个高斯形式的算符作用于[波函数](@entry_id:147440)上 [@problem_id:1380341]：

$$ |\psi_{post}\rangle \propto M(x)|\psi_{pre}\rangle, \quad \text{其中 } M(x) = \exp\left(-\frac{(x-x_m)^2}{4\sigma^2}\right) $$

这里，$x_m$ 是测量中心，$\sigma$ 代表测量的不确定度。当 $\sigma$很大时，[高斯函数](@entry_id:261394)非常平缓，对[波函数](@entry_id:147440)的改变极小，我们几乎得不到位置信息，但状态也几乎不受干扰。当 $\sigma$很小时，高斯函数变得非常尖锐，近似于一个狄拉克 $\delta$ 函数，此时测量接近于一个强的位置投影。对于中等大小的 $\sigma$，测量提供了关于粒子在 $x_m$ 附近概率大小的部分信息，同时将[波函数](@entry_id:147440)“过滤”成一个新的形状，而不是将其完全坍缩到一个[本征态](@entry_id:149904)。这个新的、被扰动后的状态仍然是一个叠加态，可以继续用于后续的计算和预测。弱[测量理论](@entry_id:153616)为理解[量子到经典的过渡](@entry_id:153498)以及探索量子测量的本质提供了深刻的见解。
