## 引言
在代数拓扑领域，同调群是用于区分不同拓扑空间的核心[不变量](@entry_id:148850)，但从其基本定义（如[奇异同调](@entry_id:158380)）出发进行计算往往异常繁琐。对于一大类被称为[CW复形](@entry_id:150589)的空间，存在一种更为强大和直观的计算工具——[胞腔同调](@entry_id:264549)。n维球面作为拓扑学中最基本的对象之一，是展示[胞腔同调](@entry_id:264549)威力的完美范例。本文旨在系统性地解决如何利用空间的几何分解来高效计算其同调群这一问题。

本文将引导读者深入掌握[胞腔同调](@entry_id:264549)的计算精髓。在第一章“原则与机制”中，我们将从n维球面最简洁的几何表示出发，揭示[胞腔链复形](@entry_id:160435)、[边界映射](@entry_id:151165)以及粘附[映射的度](@entry_id:158493)是如何协同作用，从而确定最终的同调群。第二章“应用与跨学科联系”将视野拓宽，探讨如何运用这些原理来分析由简单空间构造出的更复杂结构（如[积空间](@entry_id:151693)、商空间和[射影空间](@entry_id:157963)），并揭示其与[流形拓扑学](@entry_id:270831)、[Morse理论](@entry_id:151962)等领域的深刻联系。最后，在“动手实践”部分，我们将通过一系列精选习题，巩固并应用所学知识。通过这一系列学习，读者将能够将抽象的拓扑概念转化为具体的代数计算。

## 原则与机制

在本章中，我们将深入探讨计算球面的同调群的核心方法——[胞腔同调](@entry_id:264549)。相较于[奇异同调](@entry_id:158380)的定义，[胞腔同调](@entry_id:264549)提供了一种更为具体和计算性更强的途径，特别是对于那些具有良好几何分解（即[CW复形](@entry_id:150589)结构）的空间。我们将从最基本的球面分解开始，逐步揭示粘附[映射的度](@entry_id:158493)在决定同调群结构中的关键作用，并通过一系列例子展示，无论我们选择多么复杂的几何分解，最终得到的同调群都是不变的，这深刻地体现了同调的拓扑不变量本质。

### n-维球面的最小CW结构

理解任何拓扑空间同调群计算的第一步，往往是从其最简洁的几何表示入手。对于$n$-维球面 $S^n$ 而言，其最经济的CW结构（或称胞腔分解）源于一个非常直观的几何构造。

我们可以将 $S^n$ 想象成一个 $n$-维[闭圆盘](@entry_id:148403) $D^n = \{x \in \mathbb{R}^n : \|x\| \le 1\}$，将其边界 $S^{n-1} = \{x \in \mathbb{R}^n : \|x\| = 1\}$ 上所有的点捏合成一个单点。这个过程被称为[商拓扑](@entry_id:150384)构造，记作 $D^n/S^{n-1}$。这个新空间在拓扑上与 $S^n$ 是等价的，即两者[同胚](@entry_id:146933)。例如，将二维圆盘（一个饼）的边界（圆周）捏合到一点，我们就得到了一个[二维球面](@entry_id:269890)。

这个构造天然地为我们提供了一个计算 $S^n$ 同调的CW结构[@problem_id:1635602]。这个结构仅包含两个胞腔：
1.  一个 **0-维胞腔** $e^0$：它就是那个被边界所有点粘合于其上的单点。
2.  一个 **$n$-维胞腔** $e^n$：它对应于 $D^n$ 的内部，其边界通过粘附过程被“贴”到了0-维胞腔上。

根据[胞腔同调](@entry_id:264549)的定义，一个空间的 $k$-维胞腔链群 $C_k(X)$ 是由该空间所有 $k$-维胞腔生成的自由[阿贝尔群](@entry_id:150284)。对于 $S^n$ 的这个最小CW结构，其胞腔链群异常简单：
- $C_n(S^n) \cong \mathbb{Z}$，由 $n$-维胞腔 $e^n$ 生成。
- $C_0(S^n) \cong \mathbb{Z}$，由 0-维胞腔 $e^0$ 生成。
- 对于所有其他维度 $k$（即 $k \neq 0, n$），由于没有对应维度的胞腔，所以 $C_k(S^n) = 0$。

接下来我们构建[链复形](@entry_id:150246)，并确定其中的[边界映射](@entry_id:151165) $d_k: C_k \to C_{k-1}$。
$$
\dots \to C_{n+1} \xrightarrow{d_{n+1}} C_n \xrightarrow{d_n} C_{n-1} \xrightarrow{d_{n-2}} \dots \xrightarrow{d_1} C_0 \xrightarrow{d_0} 0
$$
在此[链复形](@entry_id:150246)中，所有[边界映射](@entry_id:151165)均为零。这是因为，对于 $n>1$，除 $C_n$ 和 $C_0$ 外所有链群均为零，因此任何[边界映射](@entry_id:151165)的定义域或靶群必为零。对于 $n=1$ 的情况，[边界映射](@entry_id:151165) $d_1$ 也为零，因为1-胞腔的两个端点在同一个0-胞腔上代数抵消。

既然所有[边界映射](@entry_id:151165)都是零映射，[计算同调](@entry_id:161051)群 $H_k(S^n) = \ker(d_k) / \text{im}(d_{k+1})$ 就变得非常直接：
- **$H_n(S^n)$**：$H_n(S^n) = \ker(d_n) / \text{im}(d_{n+1})$。因为 $d_n=0$，所以 $\ker(d_n) = C_n \cong \mathbb{Z}$。又因为 $C_{n+1}=0$，所以 $\text{im}(d_{n+1})=0$。因此，$H_n(S^n) \cong \mathbb{Z}/0 \cong \mathbb{Z}$。
- **$H_0(S^n)$**：$H_0(S^n) = \ker(d_0) / \text{im}(d_1)$。根据定义，$d_0$ 总是零映射，所以 $\ker(d_0) = C_0 \cong \mathbb{Z}$。我们已知 $d_1=0$，所以 $\text{im}(d_1)=0$。因此，$H_0(S^n) \cong \mathbb{Z}/0 \cong \mathbb{Z}$。这反映了 $S^n$ 是路径连通的。
- **$H_k(S^n)$ 对于 $0  k  n$**：$H_k(S^n) = \ker(d_k) / \text{im}(d_{k+1})$。在此范围内，$C_k=0$ 和 $C_{k+1}=0$，导致 $d_k$ 和 $d_{k+1}$ 都是零映射，因此 $\ker(d_k)=0$ 且 $\text{im}(d_{k+1})=0$。所以，$H_k(S^n) = 0/0 = 0$。

综上所述，我们得到了 $n$-维球面 ($n \ge 1$) 的标准同调群：
$$
H_k(S^n; \mathbb{Z}) \cong \begin{cases} \mathbb{Z}  \text{若 } k=0 \text{ 或 } k=n, \\ 0  \text{其它情况}. \end{cases}
$$
由于同调群是拓扑不变量，任何与 $S^n$ [同胚](@entry_id:146933)的空间都具有相同的同调群。例如，$(n+1)$-维单纯形 $\Delta^{n+1}$ 的边界 $\partial\Delta^{n+1}$ 是一个与 $S^n$ [同胚](@entry_id:146933)的几何对象。尽管我们可以用其所有的面、棱和顶点构造一个复杂的CW结构，但我们无需进行繁琐的计算，只需利用[同胚](@entry_id:146933)[不变性](@entry_id:140168)，就可以断定其同调群与 $S^n$ 的同调群一致[@problem_id:1635619]。

### 核心机制：粘附映射与度

在上一节中，[边界映射](@entry_id:151165)恰好都为零，这使得计算异常简单。但在更一般的情况下，[边界映射](@entry_id:151165) $d_k$ 是非零的，它携带了胞腔之间如何“粘合”的关键信息。这个信息被编码在 **粘附[映射的度](@entry_id:158493) (degree)** 中。

[胞腔边界公式](@entry_id:271213)的核心思想是：$k$-维链 $e_\alpha^k$ 在[边界映射](@entry_id:151165) $d_k$ 下的像，是其边界 $\partial e_\alpha^k$ 粘附到 $(k-1)$-骨架 $X_{k-1}$ 上时，“缠绕”在各个 $(k-1)$-维胞腔 $e_\beta^{k-1}$ 上的代数次数。具体来说，$d_k(e_\alpha^k)$ 中 $e_\beta^{k-1}$ 的系数，由复合映射 $\partial e_\alpha^k \cong S^{k-1} \xrightarrow{\phi_\alpha} X_{k-1} \to X_{k-1}/(X_{k-1} \setminus e_\beta^{k-1}) \cong S^{k-1}$ 的度决定。

让我们通过一个基本例子来阐明这个原理。考虑通过将一个2-维胞腔 $e^2$ 粘附到一个圆 $S^1$ 上来构造空间 $X_k$。$S^1$ 本身可以看作由一个0-维胞腔和一个1-维胞腔构成。粘附映射 $\phi_k: \partial e^2 \to S^1$ 是一个从圆到圆的映射，其[拓扑度](@entry_id:264252)为整数 $k$。

这个空间 $X_k$ 的CW结构包含一个0-胞腔 $e^0$，一个1-胞腔 $e^1$ 和一个2-胞腔 $e^2$。其[胞腔链复形](@entry_id:160435)为：
$$
0 \longrightarrow C_2(X_k) \xrightarrow{d_2} C_1(X_k) \xrightarrow{d_1} C_0(X_k) \longrightarrow 0
$$
其中 $C_2 \cong \mathbb{Z}$, $C_1 \cong \mathbb{Z}$, $C_0 \cong \mathbb{Z}$。
- 映射 $d_1: C_1 \to C_0$ 的计算与 $S^1$ 的情况相同。1-胞腔 $e^1$ 的两个端点都粘附到同一个0-胞腔 $e^0$ 上，它们的贡献（一个+1，一个-1）正好抵消，所以 $d_1 = 0$。
- 映射 $d_2: C_2 \to C_1$ 的系数由粘附映射 $\phi_k$ 的度决定。根据[胞腔边界公式](@entry_id:271213)，$d_2$ 在链[群的生成元](@entry_id:137215)上表现为乘以度 $k$。因此，这个[链复形](@entry_id:150246)是 $0 \to \mathbb{Z} \xrightarrow{\times k} \mathbb{Z} \xrightarrow{0} \mathbb{Z} \to 0$。

现在我们可以计算 $X_k$ 的同调群：
- $H_2(X_k) = \ker(d_2)$。如果 $k \neq 0$，乘以 $k$ 是一个单射，核为0。如果 $k=0$，核为 $\mathbb{Z}$。
- $H_1(X_k) = \ker(d_1) / \text{im}(d_2) = \mathbb{Z} / k\mathbb{Z}$。这个群是 $\mathbb{Z}$（当 $k=0$），$\mathbb{Z}_{|k|}$（当 $|k|  1$），或 0（当 $|k|=1$）。
- $H_0(X_k) = \ker(d_0) / \text{im}(d_1) = \mathbb{Z} / 0 \cong \mathbb{Z}$。

这个简单的例子揭示了粘附[映射的度](@entry_id:158493)如何直接塑造同调群。我们可以分析几种特殊情况：
- **当 $k=0$ 时**：粘附映射是常值映射（[零伦的](@entry_id:148739)）。这意味着 $e^2$ 的边界被压缩到一点。得到的空间 $X_0$ 是 $S^1$ 和 $S^2$ 的[楔和](@entry_id:270607)，记为 $S^1 \vee S^2$。其同调群为 $H_2 \cong \mathbb{Z}, H_1 \cong \mathbb{Z}, H_0 \cong \mathbb{Z}$。这显然与 $S^2$ 的同调群不同，因为其一维同调群非零[@problem_id:1635603]。
- **当 $|k|=1$ 时**：$H_1(X_k) = \mathbb{Z}/ \mathbb{Z} = 0$ 且 $H_2(X_k) = 0$。此时空间的所有高维同调群均为零，只有 $H_0 \cong \mathbb{Z}$。这正是单点空间的同调特征。因此，当粘附[映射的度](@entry_id:158493)为 $\pm 1$ 时，所构造的空间是可缩的，即与单点具有相同的[同伦](@entry_id:139266)和同调性质[@problem_id:1635598]。
- **当 $|k|1$ 时**：$H_1(X_k) = \mathbb{Z}_{|k|}$ 是一个有限阶的[循环群](@entry_id:138668)，称为 **挠率[子群](@entry_id:146164)**。这表明，通过选择合适的度，我们可以在同调群中制造出挠率现象。

### 推广：处理多个胞腔

当我们在一个骨架上粘附多个胞腔时，上述原理可以自然推广。此时，[边界映射](@entry_id:151165) $d_k$ 将是一个从高维自由阿贝尔群到低维自由[阿贝尔群](@entry_id:150284)的同态，其矩阵表示的元素由各个粘附[映射的度](@entry_id:158493)决定。

设想我们从一个 $(n-1)$-维球面 $S^{n-1}$ 出发，粘附两个 $n$-维胞腔 $e^n_A$ 和 $e^n_B$。粘附映射 $\phi_A$ 的度为12，$\phi_B$ 的度为-30 ($n \ge 2$) [@problem_id:1635601]。
该空间 $X$ 的CW结构包含一个 $(n-1)$-胞腔（来自 $S^{n-1}$）和两个 $n$-胞腔。因此，相关的链群为 $C_n(X) \cong \mathbb{Z} \oplus \mathbb{Z}$（由 $e^n_A, e^n_B$ 生成）和 $C_{n-1}(X) \cong \mathbb{Z}$（由 $S^{n-1}$ 的胞腔生成）。

[边界映射](@entry_id:151165) $d_n: \mathbb{Z}^2 \to \mathbb{Z}$ 将[基向量](@entry_id:199546) $(1,0)$（代表 $e^n_A$）映为 $12$，将 $(0,1)$（代表 $e^n_B$）映为 $-30$。因此，对任意链 $(a,b) \in \mathbb{Z}^2$，有 $d_n(a,b) = 12a - 30b$。
$d_n$ 的像 $\text{im}(d_n)$ 是 $\mathbb{Z}$ 中由 $12$ 和 $-30$ 生成的[子群](@entry_id:146164)，即由它们的最大公约数生成的[子群](@entry_id:146164)：$\text{im}(d_n) = \gcd(12, 30)\mathbb{Z} = 6\mathbb{Z}$。
$(n-1)$-维同调群为 $H_{n-1}(X) = \ker(d_{n-1}) / \text{im}(d_n)$。由于 $(n-2)$-骨架是空的，$d_{n-1}=0$，所以 $\ker(d_{n-1}) = C_{n-1} \cong \mathbb{Z}$。
因此，$H_{n-1}(X) \cong \mathbb{Z} / 6\mathbb{Z}$。这是一个6阶循环群。

这个方法非常普适。我们可以进一步探究，在什么条件下，通过将两个 $n$-胞腔（度分别为 $k$ 和 $m$）粘附到 $S^{n-1}$ 上所得到的空间 $X$ 会成为一个 **同调n-球面**（即其同调群与 $S^n$ 相同）[@problem_id:1635629]。
- $H_{n-1}(X) \cong \mathbb{Z} / \gcd(|k|,|m|)\mathbb{Z}$。要使 $H_{n-1}(X)=0$（这是 $S^n$ 的特征），我们必须有 $\gcd(|k|,|m|) = 1$。
- $H_n(X) = \ker(d_n)$。这里 $d_n: \mathbb{Z}^2 \to \mathbb{Z}$ 是映射 $(a,b) \mapsto ka+mb$。只要 $k,m$ 不全为零（$\gcd(|k|,|m|)=1$ 保证了这一点），这个映射的核就是一个秩为1的自由[阿贝尔群](@entry_id:150284)，即 $\ker(d_n) \cong \mathbb{Z}$。这与 $H_n(S^n) \cong \mathbb{Z}$ 相符。
- 其他维度的同调群 $H_p(X)$ ($0  p  n-1$) 显然为零，而 $H_0(X) \cong \mathbb{Z}$ 因为空间是连通的。
因此，当且仅当 $\gcd(|k|,|m|)=1$ 时，所构造的空间 $X$ 是一个同调$n$-球面。这完美地展示了如何利用[胞腔同调](@entry_id:264549)的机制来设计具有特定拓扑性质的空间。

### 同调的不变性：不同胞腔分解的比较

到目前为止，我们使用的CW结构都是根据空间的构造方式“量身定做”的。然而，同调论的一个基石是其 **[拓扑不变性](@entry_id:181048)**：只要两个空间同胚，无论我们为它们选择多么不同的CW分解，计算出的同调群必然是相同的。

让我们以 $S^2$ 为例，验证这一点。我们已经知道其最小CW结构给出的同调群为 $H_2 \cong \mathbb{Z}, H_1=0, H_0 \cong \mathbb{Z}$。现在考虑其它更复杂的分解。

- **四面体表面**：一个四面体的表面同胚于 $S^2$。我们可以将其4个顶点作为0-胞腔，6条棱作为1-胞腔，4个面作为2-胞腔[@problem_id:1635626]。
  - 链群为 $C_2 \cong \mathbb{Z}^4$, $C_1 \cong \mathbb{Z}^6$, $C_0 \cong \mathbb{Z}^4$。
  - 通过计算[边界映射](@entry_id:151165)矩阵的秩和零化度，可以发现：
    - $\text{rank}(\text{im}(d_1)) = 3$，因此 $H_0 = C_0/\text{im}(d_1)$ 的秩为 $4-3=1$，即 $H_0 \cong \mathbb{Z}$。
    - $\text{rank}(\ker(d_1)) = \dim(C_1) - \text{rank}(\text{im}(d_1)) = 6-3=3$。
    - $\text{rank}(\text{im}(d_2)) = \dim(C_2) - \text{rank}(\ker(d_2)) = 4-1=3$。
    - 由于 $\ker(d_1)$ 和 $\text{im}(d_2)$ 都是 $C_1$ 中秩为3的[子群](@entry_id:146164)，并且我们知道 $\text{im}(d_2) \subseteq \ker(d_1)$，所以它们必然相等。因此 $H_1 = \ker(d_1)/\text{im}(d_2) = 0$。
    - $H_2 = \ker(d_2)$ 是一个秩为1的群，即 $H_2 \cong \mathbb{Z}$。
  - 结果与最小结构完全一致。

- **立方体表面**：类似地，一个立方体的表面也同胚于 $S^2$。它提供了另一个CW分解：8个顶点（0-胞腔），12条棱（1-胞腔），6个面（2-胞腔）[@problem_id:1635594]。尽管链群的维数变得更大（$C_2 \cong \mathbb{Z}^6, C_1 \cong \mathbb{Z}^{12}, C_0 \cong \mathbb{Z}^8$），但通过类似的秩-零化度分析，我们同样得到 $H_2 \cong \mathbb{Z}, H_1=0, H_0 \cong \mathbb{Z}$。

最后，我们来看一个更具[一般性](@entry_id:161765)的例子：$S^n$ 的 **赤道结构**。这个结构在每个维度 $k$ ($0 \le k \le n$) 上都有两个胞腔，可以看作是北半球和南半球的分解 [@problem_id:1635625]。其链群为 $C_k(S^n) \cong \mathbb{Z}^2$。一个典型的[边界映射](@entry_id:151165)定义为 $d_k(e^k_+) = e^{k-1}_+ + e^{k-1}_-$ 和 $d_k(e^k_-) = -(e^{k-1}_+ + e^{k-1}_-)$。
对于这个[链复形](@entry_id:150246)，我们可以精确地看到中间维度的同调群是如何消失的。对于 $1 \le k  n$：
- $k$-维循环群 $\ker(d_k)$ 是由 $e^k_+ + e^k_-$ 生成的[子群](@entry_id:146164)，同构于 $\mathbb{Z}$。
- $(k-1)$-维边界群 $\text{im}(d_k)$ 是由 $e^{k-1}_+ + e^{k-1}_-$ 生成的[子群](@entry_id:146164)，也同构于 $\mathbb{Z}$。

现在考虑同调群 $H_p(S^n) = \ker(d_p) / \text{im}(d_{p+1})$，其中 $1 \le p \le n-1$。
- $p$-维[循环群](@entry_id:138668) $Z_p = \ker(d_p)$ 由 $e^p_+ + e^p_-$ 生成。
- $p$-维边界群 $B_p = \text{im}(d_{p+1})$ 由 $d_{p+1}(e^{p+1}_+)$ 和 $d_{p+1}(e^{p+1}_-)$ 生成，也就是由 $e^p_+ + e^p_-$ 生成。
因此，我们发现 $Z_p = B_p$ [@problem_id:1635613]。这意味着每个 $p$-维循环同时也是一个 $(p+1)$-维链的边界。因此，它们的[商群](@entry_id:145113) $H_p(S^n)$ 是平凡群0。
而在[链复形](@entry_id:150246)的两端， $H_n = \ker(d_n)/0 \cong \mathbb{Z}$ 且 $H_0 = C_0/\text{im}(d_1) \cong \mathbb{Z}^2/\mathbb{Z} \cong \mathbb{Z}$。最终结果再次与我们用最小结构得到的结果吻合。

这个例子生动地揭示了在一个非平凡的[链复形](@entry_id:150246)中，高维胞腔的边界如何精确地“填满”了低维胞腔形成的“洞”，从而导致中间维度的同调群为零。这也再次证明了[胞腔同调](@entry_id:264549)作为一种强大的计算工具，其结果不依赖于我们选择的几何分解方式，忠实地反映了空间内在的、不变的拓扑属性。
