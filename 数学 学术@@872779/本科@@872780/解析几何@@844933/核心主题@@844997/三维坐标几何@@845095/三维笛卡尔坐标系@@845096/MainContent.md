## 引言
从二维平面到三维空间的跨越，是数学和科学领域一次深刻的认知飞跃。三维[笛卡尔坐标系](@entry_id:169789)正是实现这一飞跃的关键工具，它为我们提供了一个通用框架，用以量化、分析和操控我们所处的物理世界。无论是设计复杂的工程结构、导航航天器，还是创建逼真的虚拟现实，其背后都离不开对三维空间的精确数学描述。

然而，仅仅直观地感知三维空间是不够的。为了解决实际问题，我们必须掌握将几何直觉转化为严谨代数语言的能力。本文旨在系统性地填补这一认知鸿沟，将抽象的空间概念与具体的代数运算联系起来。

通过本文的学习，你将建立起对[三维坐标系](@entry_id:163946)的全面理解。在第一章 **原理与机制** 中，我们将从基本定义出发，构建三维框架，并深入探讨[距离公式](@entry_id:164913)、投影以及用方程描述球面等基本几何体的方法。接下来的第二章 **应用与跨学科联系** 将展示这些原理如何赋能于工程设计、物理建模和计算机图形学等多个领域，揭示其作为现代科技基石的广泛影响力。最后，在第三章 **动手实践** 中，你将通过解决具体问题，将理论知识应用于实践，巩固并深化所学概念。

## 原理与机制

本章在前一章介绍的基础上，旨在建立并阐释三维[空间分析](@entry_id:183208)的坚实框架。我们将系统地探讨定义三维[笛卡尔坐标系](@entry_id:169789)的基本原理，并深入研究其核心机制，包括距离的计算、几何体的方程表示以及[坐标变换](@entry_id:172727)等，为利用代数方法解决[三维几何](@entry_id:176328)问题奠定基础。

### 三维笛卡尔框架

#### 坐标、坐标轴与坐标平面

从二维平面到三维空间的拓展，需要引入第三个维度。三维[笛卡尔坐标系](@entry_id:169789)统由三个相互垂直的坐标轴构成，通常标记为 $x$ 轴、$y$ 轴和 $z$ 轴。这三个坐标轴交于一点，称为**原点**，记作 $O(0,0,0)$。为了确保系统的一致性，我们通常采用**[右手定则](@entry_id:156766)**来确定坐标轴的方向：如果您将右手的手指从正 $x$ 轴卷向正 $y$ 轴，那么您的大拇指将指向正 $z$ 轴的方向。

空间中的任何一点 $P$ 的位置都可以通过一个唯一的有序三元组 $(x, y, z)$ 来确定。这三个数值分别代表了从原点出发，沿着 $x$ 轴、 $y$ 轴和 $z$ 轴方向需要移动的距离。例如，点 $P(a,b,c)$ 表示从原点开始，沿 $x$ 轴方向移动 $a$ 个单位，接着平行于 $y$ 轴方向移动 $b$ 个单位，最后平行于 $z$ 轴方向移动 $c$ 个单位到达的位置。

这三个坐标轴两两组合，可以定义三个**坐标平面**：$xy$ 平面（所有 $z=0$ 的点的集合）、$yz$ 平面（所有 $x=0$ 的点的集合）和 $xz$ 平面（所有 $y=0$ 的点的集合）。这些平面是分析[三维几何](@entry_id:176328)的基础。

#### 卦限

正如二维平面中的坐标轴将平面分为四个象限一样，三维空间中的三个坐标平面将整个空间分割成八个区域，我们称之为**卦限**。每个卦限由点的坐标 $(x, y, z)$ 的正负号组合来定义。

卦限的编号有一个标准约定。首先，$z > 0$ 的区域（即 $xy$ 平面上方的空间）包含第1到第4卦限，其编号对应于 $xy$ 平面上的象限：
- **第1卦限**: $x > 0, y > 0, z > 0$
- **第2卦限**: $x < 0, y > 0, z > 0$
- **第3卦限**: $x < 0, y < 0, z > 0$
- **第4卦限**: $x > 0, y < 0, z > 0$

然后，$z < 0$ 的区域（即 $xy$ 平面下方的空间）包含第5到第8卦限。每个卦限都位于其编号减4的卦限的正下方。例如，第5卦限在第1卦限下方，第6卦限在第2卦限下方，以此类推：
- **第5卦限**: $x > 0, y > 0, z < 0$
- **第6卦限**: $x < 0, y > 0, z < 0$
- **第7卦限**: $x < 0, y < 0, z < 0$
- **第8卦限**: $x > 0, y < 0, z < 0$

理解卦限对于确定空间中点和向量的位置至关重要。例如，考虑一个从原点出发的位置向量 $\vec{u}$，其终点位于第7卦限。根据定义，其分量 $(u_x, u_y, u_z)$ 必须满足 $u_x < 0$, $u_y < 0$ 和 $u_z < 0$。现在，如果我们用一个负实数（记为 $-k$，其中 $k > 0$）来缩放这个向量，得到新向量 $\vec{v} = (-k)\vec{u}$，那么 $\vec{v}$ 的分量将是 $((-k)u_x, (-k)u_y, (-k)u_z)$。由于 $k>0$ 且 $u_x, u_y, u_z$ 均为负，新向量的每个分量都将变为正数。因此，向量 $\vec{v}$ 的终点将位于 $x>0, y>0, z>0$ 的区域，即第1卦限 [@problem_id:2145477]。

### 距离、投影与基本几何形式

#### [距离公式](@entry_id:164913)

三维空间中分析几何的核心工具之一是计算两点之间的距离。[假设空间](@entry_id:635539)中有两点 $P_1(x_1, y_1, z_1)$ 和 $P_2(x_2, y_2, z_2)$。我们可以通过两次应用毕达哥拉斯定理（[勾股定理](@entry_id:264352)）来推导它们之间的[距离公式](@entry_id:164913)。

首先，考虑这两点在 $xy$ 平面上的投影，它们的坐标分别是 $(x_1, y_1, 0)$ 和 $(x_2, y_2, 0)$。它们之间的水平距离的平方是 $(x_2 - x_1)^2 + (y_2 - y_1)^2$。现在，我们可以想象一个以这个水平距离为底，以两点的高度差 $|z_2 - z_1|$ 为高的直角三角形。该三角形的斜边就是连接 $P_1$ 和 $P_2$ 的线段。根据毕达哥拉斯定理，斜边长度的平方等于两直角边长度的平方和。

因此，点 $P_1$ 和 $P_2$ 之间的距离 $d$ 的平方为：
$d^2 = ((x_2 - x_1)^2 + (y_2 - y_1)^2) + (z_2 - z_1)^2$

由此得到三维空间中的**[距离公式](@entry_id:164913)**：
$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}$

#### 点到坐标轴与坐标平面的距离

利用[距离公式](@entry_id:164913)，我们可以推导出计算一个点到坐标轴和坐标平面最短距离的表达式。点 $P(x_0, y_0, z_0)$ 到一个平面或一条直线的最短距离，是指 $P$ 点与该平面或直线上所有点之间距离的最小值。

- **到坐标平面的距离**：点 $P(x_0, y_0, z_0)$ 到 $xy$ 平面（$z=0$）的最短距离，是它与它在该平面上的投影点 $(x_0, y_0, 0)$ 之间的距离，即 $\sqrt{(x_0-x_0)^2 + (y_0-y_0)^2 + (z_0-0)^2} = \sqrt{z_0^2} = |z_0|$。同理，它到 $yz$ 平面和 $xz$ 平面的距离分别是 $|x_0|$ 和 $|y_0|$。

- **到坐标轴的距离**：点 $P(x_0, y_0, z_0)$ 到 $x$ 轴的最短距离稍微复杂一些。$x$ 轴上的任意一点可以表示为 $(t, 0, 0)$。$P$ 点到该点的距离的平方为 $D^2(t) = (x_0-t)^2 + (y_0-0)^2 + (z_0-0)^2 = (x_0-t)^2 + y_0^2 + z_0^2$。要找到最短距离，我们需要最小化这个表达式。显然，当 $t=x_0$ 时，该表达式取最小值。此时，距离的平方为 $y_0^2 + z_0^2$。因此，点 $P$ 到 $x$ 轴的最短距离为 $d_x = \sqrt{y_0^2 + z_0^2}$。这个距离恰好等于点 $P$ 与其在 $yz$ 平面上的投影 $(0, y_0, z_0)$ 到原点的距离。

通过对称性，我们可以得到点 $P(x_0, y_0, z_0)$ 到三个坐标轴的最短[距离公式](@entry_id:164913)：
- 到 $x$ 轴的距离：$d_x = \sqrt{y_0^2 + z_0^2}$
- 到 $y$ 轴的距离：$d_y = \sqrt{x_0^2 + z_0^2}$
- 到 $z$ 轴的距离：$d_z = \sqrt{x_0^2 + y_0^2}$

例如，在航空管制中，一个位于 $(-8, 15, 20)$ 的无人机，其到 $x$ 轴、 $y$ 轴和 $z$ 轴的距离可以计算如下：
$d_x = \sqrt{15^2 + 20^2} = \sqrt{225 + 400} = \sqrt{625} = 25$ 米。
$d_y = \sqrt{(-8)^2 + 20^2} = \sqrt{64 + 400} = \sqrt{464} = 4\sqrt{29}$ 米。
$d_z = \sqrt{(-8)^2 + 15^2} = \sqrt{64 + 225} = \sqrt{289} = 17$ 米 [@problem_id:2162226]。

#### 正交投影

点到坐标轴或坐标平面的最短距离问题，内在地关联着**[正交投影](@entry_id:144168)**的概念。一个点 $P(x,y,z)$ 在一个平面或一条直线上的[正交投影](@entry_id:144168)，是从该点向该平面或直线作垂线所得到的垂足。

- 在 $xy$ 平面上的投影为 $P'(x,y,0)$。
- 在 $yz$ 平面上的投影为 $P'(0,y,z)$。
- 在 $xz$ 平面上的投影为 $P'(x,0,z)$。
- 在 $x$ 轴上的投影为 $P'(x,0,0)$。

投影操作在几何上可以直观地理解为“投射阴影”。假设太阳光线平行于 $x$ 轴，一个物体在 $yz$ 平面（可以想象成一块竖直的太阳能板）上形成的阴影就是该物体所有点的[正交投影](@entry_id:144168)的集合。

考虑一架无人机从点 $A(10, 2, 5)$ 直线飞行到点 $B(3, -4, 8)$。其在 $yz$ 平面上的投影轨迹，就是连接点 $A$ 和点 $B$ 在 $yz$ 平面上投影点的线段。$A$ 的投影是 $S_A(0, 2, 5)$，$B$ 的投影是 $S_B(0, -4, 8)$。这个投影轨迹（即无人机在太阳能板上的“影子”路径）的长度就是 $S_A$ 和 $S_B$ 之间的距离：
$L = \sqrt{(0-0)^2 + (-4-2)^2 + (8-5)^2} = \sqrt{(-6)^2 + 3^2} = \sqrt{36+9} = \sqrt{45} = 3\sqrt{5}$ 米 [@problem_id:2162243]。

### 用方程描述几何轨迹

#### 轨迹的概念

分析几何的核心思想是用[代数方程](@entry_id:272665)来描述几何形状。一个关于变量 $x, y, z$ 的方程，例如 $F(x,y,z)=0$，定义了一个点的集合，这些点 $(x,y,z)$ 的坐标都满足该方程。这个点的集合被称为一个**轨迹**（locus），在三维空间中通常表现为一个[曲面](@entry_id:267450)。

#### 球面

最基本和最完美的轨迹是球面。**球面**被定义为空间中所有到一个[固定点](@entry_id:156394)（**球心**）的距离等于一个正常数（**半径**）的点的集合。

设球心为 $C(h,k,l)$，半径为 $R$。根据[距离公式](@entry_id:164913)，球面上任意一点 $P(x,y,z)$ 必须满足条件 $PC = R$。将其用坐标表示出来，就是：
$\sqrt{(x-h)^2 + (y-k)^2 + (z-l)^2} = R$

对两边进行平方，我们得到球面的**标准方程**：
$(x-h)^2 + (y-k)^2 + (z-l)^2 = R^2$

展开这个方程，我们会得到一个形如 $x^2 + y^2 + z^2 + Ax + By + Cz + D = 0$ 的方程，其中 $A, B, C, D$ 是常数。这个形式被称为球面的**一般方程**。反过来，如果给定一个一般方程，只要 $x^2, y^2, z^2$ 的系数相同（并且可以化为1），我们就可以通过**[配方法](@entry_id:265480)**来确定它的几何性质。

例如，一个地下洞穴的表面被模型化为方程 $x^2 + y^2 + z^2 - 12x + 8y - 10z + k = 0$ [@problem_id:2162189]。为了找出其球心和半径，我们对每个变量进行配方：
$(x^2 - 12x) + (y^2 + 8y) + (z^2 - 10z) + k = 0$
$((x-6)^2 - 36) + ((y+4)^2 - 16) + ((z-5)^2 - 25) + k = 0$
$(x-6)^2 + (y+4)^2 + (z-5)^2 = 77 - k$

由此可知，该[球面方程](@entry_id:177405)的球心是 $(6, -4, 5)$，半径的平方是 $R^2 = 77-k$。如果已知该球面经过某一点，例如 $P(2,1,3)$，我们就可以代入该点坐标来确定常数 $k$，进而求出半径和体积。

#### 由距离关系定义的[曲面](@entry_id:267450)

除了球面，许多其他复杂的[曲面](@entry_id:267450)也可以通过点到其他点或线的距离关系来定义。

- **圆柱面和圆锥面**：一个简单的例子是“所有到一个固定直线（轴）的距离为常数的点的轨迹”。如果轴是 $z$ 轴，常数为 $R$，那么这个条件可以写成 $\sqrt{x^2+y^2} = R$，即 $x^2+y^2=R^2$。这就是一个以 $z$ 轴为中心轴、半径为 $R$ 的无限长的**圆柱面**。

如果条件变为“点到 $z$ 轴的距离与它到 $xy$ 平面的距离成正比”，例如，距离 $z$ 轴的距离是距离 $xy$ 平面距离的两倍，我们可以得到方程 $\sqrt{x^2+y^2} = 2|z|$。平方后得到 $x^2+y^2 = 4z^2$。这个方程描述了一个以 $z$ 轴为[对称轴](@entry_id:177299)的**圆锥面**，其顶点在原点 [@problem_id:2162248]。这类由方程定义出的轨迹在物理学和工程学中有广泛应用，例如在分析[粒子加速器](@entry_id:148838)中[带电粒子](@entry_id:160311)的运动轨迹时。

- **[椭球体](@entry_id:165811)**：椭圆在三维空间的推广是[椭球体](@entry_id:165811)。一个**长球面**（或称旋转椭球体）可以被定义为空间中所有到两个[固定点](@entry_id:156394)（**[焦点](@entry_id:174388)**）的距离之和为常数的点的轨迹。假设[焦点](@entry_id:174388)为 $F_1(-d,0,0)$ 和 $F_2(d,0,0)$，距离之和为常数 $L$（且 $L>2d$）。那么对于轨迹上的任意点 $P(x,y,z)$，我们有：
$\sqrt{(x+d)^2 + y^2 + z^2} + \sqrt{(x-d)^2 + y^2 + z^2} = L$

经过复杂的代数化简，这个方程可以写成标准形式 $\frac{x^2}{a^2} + \frac{y^2+z^2}{b^2} = 1$，其中[半长轴](@entry_id:164167) $a = L/2$，半短轴 $b = \sqrt{a^2 - d^2}$。这个[曲面](@entry_id:267450)是一个绕 $x$ 轴旋转而成的[椭球体](@entry_id:165811)。在声学研究中，如果两个声源同时发出声音，那么所有在同一时刻听到声音叠加的点的集合就构成了这样一个椭球体 [@problem_id:2162235]。

- **阿波罗尼斯球**：另一个有趣且不那么直观的轨迹是，空间中所有到两个[固定点](@entry_id:156394)的距离之*比*为常数 $k$（$k>0, k \neq 1$）的点的集合。这个轨迹被称为**阿波罗尼斯球**。

考虑两个信标分别位于 $A(-3d,0,0)$ 和 $B(5d,0,0)$。假设一个探测器 $P(x,y,z)$ 接收到的信号强度与距离的平方成反比。如果来自 $A$ 的信号固有功率是来自 $B$ 的4倍，那么当探测器接收到的两个信号强度相等时，我们有 $\frac{4}{PA^2} = \frac{1}{PB^2}$，即 $PA = 2 \cdot PB$ [@problem_id:2162214]。
令 $PA^2 = 4 \cdot PB^2$，代入坐标：
$(x+3d)^2 + y^2 + z^2 = 4\left[(x-5d)^2 + y^2 + z^2\right]$

展开并整理此方程：
$x^2 + 6dx + 9d^2 + y^2 + z^2 = 4(x^2 - 10dx + 25d^2 + y^2 + z^2)$
$x^2 + 6dx + 9d^2 + y^2 + z^2 = 4x^2 - 40dx + 100d^2 + 4y^2 + 4z^2$
$3x^2 - 46dx + 91d^2 + 3y^2 + 3z^2 = 0$
$x^2 - \frac{46}{3}dx + \frac{91}{3}d^2 + y^2 + z^2 = 0$

通过对 $x$ 进行配方，我们得到：
$(x - \frac{23}{3}d)^2 - (\frac{23}{3}d)^2 + \frac{91}{3}d^2 + y^2 + z^2 = 0$
$(x - \frac{23}{3}d)^2 + y^2 + z^2 = (\frac{529}{9} - \frac{273}{9})d^2 = \frac{256}{9}d^2$

这是一个球心在 $(\frac{23}{3}d, 0, 0)$，半径为 $R = \sqrt{\frac{256}{9}d^2} = \frac{16}{3}d$ 的[球面方程](@entry_id:177405)。这个例子有力地证明了，一个看似复杂的几何条件可以通过代数方法化简为一个我们熟知的基本几何形状。

### [坐标变换](@entry_id:172727)

#### 坐标轴的平移

在许多应用中，我们需要在不同的[坐标系](@entry_id:156346)之间切换。最简单的[坐标变换](@entry_id:172727)是**平移**，即保持坐标轴方向不变，只移动原点的位置。

假设我们有一个原始[坐标系](@entry_id:156346) $S$，其原点为 $O(0,0,0)$。我们建立一个新的[坐标系](@entry_id:156346) $S'$，其坐标轴分别与 $S$ 的坐标轴平行，但其原点 $O'$ 在 $S$ 系中的坐标为 $(h,k,l)$。

对于空间中的任意一点 $P$，它在 $S$ 系中的坐标为 $(x,y,z)$，在 $S'$ 系中的坐标为 $(x',y',z')$。从向量的角度看，点 $P$ 相对于原点 $O$ 的位置向量 $\vec{OP}$ 等于原点 $O'$ 相对于 $O$ 的位置向量 $\vec{OO'}$ 与点 $P$ 相对于 $O'$ 的位置向量 $\vec{O'P}$ 之和。
$\vec{OP} = \vec{OO'} + \vec{O'P}$

用坐标表示：
$(x,y,z) = (h,k,l) + (x',y',z')$

由此，我们得到坐标平移的变换公式：
$x' = x - h$
$y' = y - k$
$z' = z - l$

这个简单的关系在实际问题中非常有用。例如，一个卫星在以小行星[质心](@entry_id:265015)为原点的 $S$ 系中工作，而一个着陆器在小行星表面建立了自己的局部坐标系 $S'$ [@problem_id:2162229]。要将在卫星[坐标系](@entry_id:156346)中测量的某个地质特征的坐标转换为着陆器[坐标系](@entry_id:156346)中的坐标，只需将该特征的原始坐标减去着陆器在原始[坐标系](@entry_id:156346)中的坐标即可。

#### 反射

**反射**是另一种基本的几何变换。三维空间中的[反射变换](@entry_id:175518)可以关于一个点、一条直线或一个平面进行。

- **关于坐标平面的反射**：
  - 关于 $yz$ 平面 ($x=0$) 的反射，将点 $(x,y,z)$ 变为 $(-x,y,z)$。
  - 关于 $xz$ 平面 ($y=0$) 的反射，将点 $(x,y,z)$ 变为 $(x,-y,z)$。
  - 关于 $xy$ 平面 ($z=0$) 的反射，将点 $(x,y,z)$ 变为 $(x,y,-z)$。

- **关于原点的反射**：
  - 关于原点 $(0,0,0)$ 的反射，将点 $(x,y,z)$ 变为 $(-x,-y,-z)$。这等价于连续进行三次关于坐标平面的反射。

- **关于任意平面的反射**：
  - 对于平行于坐标平面的平面，例如 $z=k$，反射规则也相对简单。点 $P(x,y,z)$ 关于平面 $z=k$ 的反射点 $P'(x',y',z')$，其 $x,y$ 坐标不变，即 $x'=x, y'=y$。$z'$ 坐标满足 $P$ 和 $P'$ 的中点在平面 $z=k$ 上，即 $\frac{z+z'}{2}=k$，因此 $z' = 2k-z$。

我们可以通过组合这些基本变换来执行更复杂的操作。考虑一个粒子初始位置为 $P_0(a,b,c)$，经历以下序列变换 [@problem_id:2162211]：
1. 关于 $yz$ 平面反射得到 $P_1(-a,b,c)$。
2. 将 $P_1$ 关于原点反射得到 $P_2(a,-b,-c)$。
3. 将 $P_2$ 关于平面 $z=k$ 反射得到最终位置 $P_f(a, -b, 2k-(-c)) = (a, -b, 2k+c)$。

初始位置 $P_0(a,b,c)$ 与最终位置 $P_f(a,-b,2k+c)$ 之间的距离为：
$d = \sqrt{(a-a)^2 + (-b-b)^2 + (2k+c-c)^2} = \sqrt{0^2 + (-2b)^2 + (2k)^2} = \sqrt{4b^2+4k^2} = 2\sqrt{b^2+k^2}$。
这个结果揭示了一个有趣的性质：最终的距离与粒子的初始 $x$ 和 $z$ 坐标无关。

### 对[斜坐标系](@entry_id:164861)的推广

到目前为止，我们讨论的都是基于标准正交笛卡尔坐标系的几何学，其中坐标轴相互垂直，且[基向量](@entry_id:199546)是单位向量。然而，在某些领域，如[晶体学](@entry_id:140656)中，自然描述一个系统的坐标轴可能不是相互垂直的。这就引出了**[斜坐标系](@entry_id:164861)**的概念。

一个[斜坐标系](@entry_id:164861) $S'$ 由一个原点 $O'$ 和一组[线性无关](@entry_id:148207)但不必正交的[基向量](@entry_id:199546) $\{\mathbf{u}_1, \mathbf{u}_2, \mathbf{u}_3\}$ 定义。空间中任意一点 $P$ 在这个新[坐标系](@entry_id:156346)中的坐标 $(c_1, c_2, c_3)$ 是指，从新原点 $O'$ 到点 $P$ 的位置向量 $\vec{O'P}$ 可以被唯一地表示为[基向量](@entry_id:199546)的线性组合：
$\vec{O'P} = c_1\mathbf{u}_1 + c_2\mathbf{u}_2 + c_3\mathbf{u}_3$

假设我们知道 $P$ 和 $O'$ 在标准笛卡尔坐标系 $S$ 中的位置向量，分别为 $\mathbf{r}_P$ 和 $\mathbf{r}_{O'}$，并且知道 $S'$ 的[基向量](@entry_id:199546)在 $S$ 系中的表示。那么 $\vec{O'P} = \mathbf{r}_P - \mathbf{r}_{O'}$。因此，我们有：
$\mathbf{r}_P - \mathbf{r}_{O'} = c_1\mathbf{u}_1 + c_2\mathbf{u}_2 + c_3\mathbf{u}_3$

这是一个向量方程。通过将所有向量用它们在标准笛卡尔系 $S$ 中的分量表示，我们可以将其转化为一个包含三个未知数 $(c_1, c_2, c_3)$ 的三元一次[线性方程组](@entry_id:148943)。

例如，在研究晶体中的杂质时，可能需要在实验室的标准[笛卡尔坐标系](@entry_id:169789)和晶体自身的[斜坐标系](@entry_id:164861)之间进行转换 [@problem_id:2162220]。设杂[质点](@entry_id:186768) $P$ 在 $S$ 系中坐标为 $(5,-3,2)$，[斜坐标系](@entry_id:164861) $S'$ 的原点 $O'$ 在 $S$ 系中为 $(1,1,-1)$，[基向量](@entry_id:199546)为 $\mathbf{u}_1=(2,1,0)$, $\mathbf{u}_2=(1,2,1)$, $\mathbf{u}_3=(-1,0,3)$。
首先计算相对位置向量 $\mathbf{v} = \mathbf{r}_P - \mathbf{r}_{O'} = (5-1, -3-1, 2-(-1)) = (4, -4, 3)$。
然后求解方程 $\mathbf{v} = c_1\mathbf{u}_1 + c_2\mathbf{u}_2 + c_3\mathbf{u}_3$：
$\begin{pmatrix} 4 \\ -4 \\ 3 \end{pmatrix} = c_1 \begin{pmatrix} 2 \\ 1 \\ 0 \end{pmatrix} + c_2 \begin{pmatrix} 1 \\ 2 \\ 1 \end{pmatrix} + c_3 \begin{pmatrix} -1 \\ 0 \\ 3 \end{pmatrix}$

这可以写成矩阵形式 $M\mathbf{c} = \mathbf{v}$，其中 $M$ 是以 $\mathbf{u}_1, \mathbf{u}_2, \mathbf{u}_3$ 为列向量的矩阵，$\mathbf{c}$ 是坐标 $(c_1, c_2, c_3)$ 组成的列向量。
$\begin{pmatrix} 2  1  -1 \\ 1  2  0 \\ 0  1  3 \end{pmatrix} \begin{pmatrix} c_1 \\ c_2 \\ c_3 \end{pmatrix} = \begin{pmatrix} 4 \\ -4 \\ 3 \end{pmatrix}$

通过求解这个线性方程组（例如，通过求 $M$ 的[逆矩阵](@entry_id:140380)），就可以得到点 $P$ 在[斜坐标系](@entry_id:164861) $S'$ 中的坐标 $\mathbf{c} = M^{-1}\mathbf{v}$。这种方法将几何问题转化为线性代数的计算，展示了向量和矩阵方法在处理复杂[坐标系统](@entry_id:156346)时的强大威力。
