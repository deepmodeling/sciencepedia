## 引言
功能性[磁共振成像](@entry_id:153995)（fMRI）是探索人脑功能的强大工具，而通用线性模型（GLM）是分析fMRI数据的核心框架。通过GLM，我们能估计出与实验任务相关的神经活动幅度（β系数）。然而，科学问题往往比“某个区域是否有激活”更为复杂，我们更关心不同条件间的关系、认知过程的动态变化，或脑区间的功能整合。直接解释单个β系数在回答这些复杂问题时存在局限，尤其当模型中的回归量相互关联时，其解释会变得模糊且不稳定。这在神经科学家将抽象的理论假设转化为可验证的统计陈述时，造成了巨大的知识鸿沟。

本文旨在填补这一鸿沟，全面阐述fMRI GLM中用于假设检验的“对比”（contrast）技术。我们将系统地引导您：在“原理与机制”一章中，您将掌握对比的数学定义、构建原则以及[t检验](@entry_id:272234)与[F检验](@entry_id:274297)的统计基础，理解其为何是严谨[假设检验](@entry_id:142556)的基石。接着，在“应用与跨学科连接”一章中，我们将通过认知、临床等领域的真实案例，展示如何运用对比检验[交互作用](@entry_id:164533)、学习效应和[脑网络](@entry_id:912843)动态等复杂现象。最后，“动手实践”部分将提供练习，助您将理论知识转化为实践技能。通过本文的学习，您将能够自信地运用对比，将精巧的[实验设计](@entry_id:142447)转化为精确而有意义的神经科学发现。

## 原理与机制

在功能性[磁共振成像](@entry_id:153995)（fMRI）的通用线性模型（GLM）框架下，我们构建一个[设计矩阵](@entry_id:165826) $\mathbf{X}$ 来模拟所有预期的血氧水平依赖（BOLD）信号变化。通过[最小二乘法拟合](@entry_id:1127151)模型 $\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$，我们得到参数估计值 $\boldsymbol{\hat{\beta}}$。这些参数代表了设计矩阵中每个回归量（regressor）对观测数据 $\mathbf{y}$ 的贡献幅度。然而，科学研究所关心的通常不是孤立的参数值，而是关于实验条件之间关系的具体假设。例如，我们可能想知道“A条件下的神经活动是否强于B条件？”或者“药物干预是否调节了任务诱发的活动？”。直接检验单个 $\beta$ 系数（例如，检验 $H_0: \beta_A = 0$）往往不足以回答这些复杂的问题，有时甚至会产生误导。

本章将深入探讨“对比”（contrast）的原理与机制。对比是将科学假设转化为可在GLM框架下进行检验的数学语言的强大工具。我们将从基本定义出发，系统阐述如何构建、检验和解释对比，并探讨在处理相关回归量、[多重比较](@entry_id:173510)和软件实现差异等高级问题时所需遵循的核心原则。

### 对比在通用线性模型中的核心作用

在GLM中，设计矩阵 $\mathbf{X}$ 的作用是构建一个全面的模型，不仅包括我们感兴趣的实验任务（effects of interest），还包括我们不感兴趣但必须控制的效应，如头部运动、生理噪声和信号漂移，这些统称为“无关效应”（effects of no interest）或“干扰回归量”（nuisance regressors）。相应地，参数向量 $\boldsymbol{\beta}$ 包含了所有这些效应的估计幅度 。

一个核心挑战在于，单个参数 $\beta_i$ 的解释是“在保持所有其他回归量不变的情况下，第 $i$ 个回归量增加一个单位所引起的信号变化”。当回归量之间存在相关性时——这在fMRI[实验设计](@entry_id:142447)中非常普遍，例如，两个认知过程在时间上高度重叠——这种“保持其他不变”的解释就变得很脆弱。高度相关的回归量共享方差，使得模型难以唯一地将信号变化归因于其中任何一个。这导致 $\beta$ 系数的估计值[方差膨胀](@entry_id:756433)，且对模型中包含哪些其他回归量高度敏感。

更根本的是，描述信号的回归量集合（即[设计矩阵](@entry_id:165826)的列）并非唯一。我们可以对回归量进行任意可逆的线性变换（称为“基底变换”或“重[参数化](@entry_id:265163)”），而模型的整体[拟合优度](@entry_id:176037)以及对数据的预测保持不变 。例如，我们可以用回归量 $\{x_A, x_B\}$，也可以用它们的和与差 $\{x_A+x_B, x_A-x_B\}$ 来构建模型。这两种[参数化](@entry_id:265163)方式下的 $\beta$ 系数值截然不同，但它们描述的是同一个[信号子空间](@entry_id:185227)。

科学假设不应依赖于我们描述模型的任意数学选择。例如，“A过程引发了神经响应”这一论断的真伪，不应因为我们是使用 $x_A$ 还是 $x_A$ 的某个[正交化](@entry_id:149208)版本而改变。因此，我们需要一种方法来表述独立于特定[参数化](@entry_id:265163)方式的、具有稳定科学内涵的假设。

这就是**对比**的用武之地。一个对比被定义为参数 $\boldsymbol{\beta}$ 的一个[线性组合](@entry_id:154743)，形式为 $c^\top\boldsymbol{\beta}$，其中 $c$ 是一个权重向量，称为**对比向量**。这个标量值 $c^\top\boldsymbol{\beta}$ 在模型的重[参数化](@entry_id:265163)下具有[不变性](@entry_id:140168)，从而能够代表一个可解释的、稳定的科学假设 。具体而言，如果模型从 $\mathbf{X}$ 重[参数化](@entry_id:265163)为 $\mathbf{X}^* = \mathbf{XA}$（其中 $\mathbf{A}$ 为[可逆矩阵](@entry_id:171829)），则新的参数向量为 $\boldsymbol{\beta}^* = \mathbf{A}^{-1}\boldsymbol{\beta}$。原假设对应的量 $c^\top\boldsymbol{\beta}$ 在新坐标系下表示为 $(A^\top c)^\top \boldsymbol{\beta}^*$。可见，假设的本质（一个关于[信号子空间](@entry_id:185227)的陈述）得以保留，只是其在不同坐标系下的表达形式（对比向量）发生了相应改变。因此，检验形如 $H_0: c^\top\boldsymbol{\beta} = 0$ 的假设，提供了一种严谨且可解释的方式来探究fMRI数据背后的神经机制。

### 使用对比向量构建假设

构建一个有效的对比向量，[实质](@entry_id:149406)上是将一个用自然语言描述的科学问题，精确地翻译成一个施加于模型参数 $\boldsymbol{\beta}$ 上的[线性约束](@entry_id:636966)。对比向量 $c$ 的维度必须与参数向量 $\boldsymbol{\beta}$ 的维度 $p$ 相同，它的每个元素都是一个权重，对应于 $\boldsymbol{\beta}$ 中的一个参数。

**基本构建原则：**
1.  **赋予正权重（+1）：** 对于假设中我们预期会“增加”的效应所对应的参数。
2.  **赋予负权重（-1）：** 对于假设中我们预期会“减少”或作为比较基线的效应所对应的参数。
3.  **赋予零权重（0）：** 对于与当前假设无关的所有参数。这至关重要，因为将权重设为零，意味着我们是在“控制”了这些参数所解释的方差之后，检验剩余效应。

让我们通过几个例子来具体说明这个过程。

**示例1：单个条件的激活**
假设我们想检验“条件A是否引起了显著的BOLD响应？”。这等同于检验条件A的回归量所对应的参数 $\beta_A$ 是否显著大于零。假设 $\boldsymbol{\beta} = [\beta_A, \beta_B, \beta_{\text{intercept}}]^\top$，那么我们的假设是 $H_0: \beta_A = 0$。为了表达这个假设，我们构建的对比向量为：
$c = [1, 0, 0]^\top$
这样，$c^\top\boldsymbol{\beta} = (1)\beta_A + (0)\beta_B + (0)\beta_{\text{intercept}} = \beta_A$。

**示例2：两个条件的直接比较**
一个更常见的假设是比较两个条件，例如“条件A的响应幅度是否大于条件B？”。这对应于检验 $H_0: \beta_A - \beta_B = 0$（双边检验）或 $H_1: \beta_A - \beta_B > 0$（[单边检验](@entry_id:170263)）。在一个包含条件A、B及其时间导数、头部运动参数和截距项的复杂模型中，参数向量可能形如：
$\boldsymbol{\beta} = [\beta_{\text{A,can}}, \beta_{\text{A,deriv}}, \beta_{\text{B,can}}, \beta_{\text{B,deriv}}, \dots, \beta_{\text{motion}}, \beta_{\text{int}}]^\top$
其中，$\beta_{\text{A,can}}$ 和 $\beta_{\text{B,can}}$ 分别是A和B条件的主[响应函数](@entry_id:142629)（canonical HRF）幅度。我们的假设只关心这两个主响应幅度的差异。因此，我们为 $\beta_{\text{A,can}}$ 分配权重+1，为 $\beta_{\text{B,can}}$ 分配权重-1，而所有其他参数（包括时间导数、运动参数和截距项）的权重都设为0。对应的对比向量为：
$c = [1, 0, -1, 0, 0, \dots, 0]^\top$
这个“0”权重的使用体现了一个关键原则，我们将在下一节深入讨论。

**示例3：更复杂的比较**
我们还可以构建更复杂的假设，例如检验“条件A的响应是否大于条件B和C的平均响应？” 。这个假设可以写成 $H_0: \beta_A - \frac{\beta_B + \beta_C}{2} = 0$。为了保持对比向量中的权重为整数或简单分数以便于解释，我们通常将其乘以一个常数，这并不会改变检验的本质（我们稍后会证明）。因此，该假设等价于 $H_0: 2\beta_A - \beta_B - \beta_C = 0$。在一个包含9个回归量（截距、A/B/C的主响应、A的[参数调制](@entry_id:1129338)项、A/B/C的时间导数、运动干扰项）的模型中，对应的参数向量为：
$\boldsymbol{\beta} = [\beta_{\text{int}}, \beta_{A}, \beta_{B}, \beta_{C}, \beta_{\text{A,mod}}, \beta_{\text{A,der}}, \beta_{\text{B,der}}, \beta_{\text{C,der}}, \beta_{\text{motion}}]^\top$
要检验上述假设，对比向量应为：
$c = [0, 1, -1/2, -1/2, 0, 0, 0, 0, 0]^\top$
这里，截距项、[参数调制](@entry_id:1129338)项、所有时间导数项和运动项的权重都为0，因为它们不是我们直接比较的对象。

### 干扰参数不变性原则

在构建对比向量时，一个必须严格遵守的原则是：**所有与当前检验的科学假设无直接关系的参数（即干扰参数）所对应的权重必须为零** 。这些干扰参数通常包括截距项、头部运动参数、生理噪声回归量、扫描漂移模型（如[离散余弦变换](@entry_id:748496)基函数）以及不直接属于假设一部分的任务相关回归量（如时间导数或[参数调制](@entry_id:1129338)器）。

将干扰参数的权重设为零，其统计学意义是“在控制了这些干扰效应所解释的方差之后，检验我们感兴趣的效应”。这确保了我们的统计推断是针对特定假设的，而不会被无关的变异来源所混淆。

这一原则的深层理论依据是**假设的可解释性**与**对干扰参数重[参数化](@entry_id:265163)的[不变性](@entry_id:140168)**。在实践中，干扰回归量的具体形式往往是任意选择的。例如，我们可以用6个运动参数，也可以用它们的主成分；我们可以用多项式基函数来模拟漂移，也可以用离散余弦基函数。这些不同的选择构成了对干扰效应子空间的重[参数化](@entry_id:265163)。一个有意义的科学假设，不应该因为我们选择了何种方式来对无关噪声建模而改变其结论。

我们可以严格证明，要使假设 $c^\top\boldsymbol{\beta}=0$ 的解释不随干扰参数的任意[可逆线性变换](@entry_id:149915)而改变，其**充分必要条件**就是对比向量 $c$ 中与所有干扰参数对应的权重（记作 $c_n$）必须为[零向量](@entry_id:156189)，即 $c_n = 0$ 。如果一个对比赋予了干扰参数非零权重（例如，检验 $H_0: \beta_A - 0.5 \beta_{\text{motion1}} = 0$），那么这个假设的意义就变得模糊不清。它混合了任务效应和伪迹，并且其检验结果会随着我们如何缩放或表示运动参数而改变，从而丧失了科学上的可解释性。

### [假设检验](@entry_id:142556)：[t检验](@entry_id:272234)与[F检验](@entry_id:274297)

构建了对比向量（或矩阵）后，下一步就是进行统计检验。根据假设的维度，我们选择[t检验](@entry_id:272234)或[F检验](@entry_id:274297)。

#### [t检验](@entry_id:272234)：检验单一约束

当我们的假设只涉及一个[线性组合](@entry_id:154743)时，我们使用**[t检验](@entry_id:272234)**。这个假设由一个**对比向量** $c \in \mathbb{R}^p$ 定义，其[原假设](@entry_id:265441)为 $H_0: c^\top\boldsymbol{\beta} = 0$ 。

[t统计量](@entry_id:177481)的构造思想是，比较我们估计的效应 $c^\top\boldsymbol{\hat{\beta}}$ 与其在[原假设](@entry_id:265441)下的[期望值](@entry_id:150961)（0）的差异，并用该效应的[标准误](@entry_id:635378)（standard error）进行标准化。其通用形式为：
$t = \frac{c^\top\boldsymbol{\hat{\beta}}}{\sqrt{\widehat{\text{Var}}(c^\top\boldsymbol{\hat{\beta}})}} = \frac{c^\top\boldsymbol{\hat{\beta}}}{\hat{\sigma}\sqrt{c^\top(\mathbf{X}^\top\mathbf{X})^{-1}c}}$
其中，$\hat{\sigma}^2$ 是模型残差的[方差估计](@entry_id:268607)。在原假设成立的条件下，该统计量服从自由度为 $df = T - \text{rank}(\mathbf{X})$ 的[t分布](@entry_id:267063)，其中 $T$ 是时间点数，$p=\text{rank}(\mathbf{X})$ 是模型中独立回归量的数量。

值得注意的是，[t统计量](@entry_id:177481)的值不受对比向量 $c$ 自身缩放的影响。例如，对比向量 $[1, -1, 0]$ 和 $[0.5, -0.5, 0]$ 会产生完全相同的t值（和p值），因为分子和分母会按相同比例缩放。这再次印证了对比检验的是效应的相对关系，而非其绝对大小。

#### [F检验](@entry_id:274297)：检验联合假设

当我们需要同时检验多个[线性约束](@entry_id:636966)时，我们使用**[F检验](@entry_id:274297)**。这种联合假设由一个**对比矩阵** $C \in \mathbb{R}^{m \times p}$ 定义，其中矩阵的每一行都是一个对比向量。原假设为 $H_0: C\boldsymbol{\beta} = \mathbf{0}$，这里 $\mathbf{0}$ 是一个 $m \times 1$ 的[零向量](@entry_id:156189) 。这等同于检验 $C$ 的所有行向量定义的假设是否“同时”为零。

一个典型的例子是**综合检验（omnibus test）**，即检验一组任务条件是否“有任何”激活。例如，对于包含三个任务条件（$\beta_1, \beta_2, \beta_3$）的模型，我们可能想检验 $H_0: \beta_1=0 \text{ and } \beta_2=0 \text{ and } \beta_3=0$。这个联合假设无法通过单个[t检验](@entry_id:272234)完成，而需要一个[F检验](@entry_id:274297)。对应的对比矩阵为：
$C = \begin{pmatrix} 0  & 1 & 0 & 0 & \dots \\ 0 & 0 & 1 & 0 & \dots \\ 0 & 0 & 0 & 1 & \dots \end{pmatrix}$
（假设第一列为截距项）。

[F统计量](@entry_id:148252)衡量的是包含我们感兴趣的回归量所能解释的方差，与模型残差方差的比值。它服从一个[F分布](@entry_id:261265)，该分布由两个自由度参数定义：
- **[分子自由度](@entry_id:175192)** $df_1 = \text{rank}(C)$，即我们同时检验的[线性独立](@entry_id:153759)约束的数量。
- **分母自由度** $df_2 = T - \text{rank}(\mathbf{X})$，即模型的残差自由度。

#### [t检验](@entry_id:272234)与[F检验](@entry_id:274297)的关系

当[F检验](@entry_id:274297)只包含一个约束时（即 $m=1$ 且 $\text{rank}(C)=1$），它与双边[t检验](@entry_id:272234)是等价的。具体来说，一个服从 $F(1, df_2)$ 分布的[F统计量](@entry_id:148252)，其值等于一个服从 $t(df_2)$ 分布的[t统计量](@entry_id:177481)的平方：
$F_{1, df_2} = (t_{df_2})^2$
这意味着对于单个假设，例如 $H_0: \beta_A - \beta_B = 0$，我们可以用[t检验](@entry_id:272234)来获得效应的方向性信息（即 $A>B$ 还是 $B>A$），或者用[F检验](@entry_id:274297)来检验是否存在“任何”差异（不区分方向）。[F检验](@entry_id:274297)因其统计量总是非负，天然地适用于非方向性的假设。

### 高级主题与实践考量

#### 交互作用对比

在[因子设计](@entry_id:921332)（factorial design）实验中，我们常常对不同因素之间的**[交互作用](@entry_id:164533)（interaction）**感兴趣。[交互作用](@entry_id:164533)指的是一个因素的效应依赖于另一个因素的水平。例如，在一个 $2 \times 2$ 的[因子设计](@entry_id:921332)中，因素A有两个水平（A1, A2），因素B有两个水平（B1, B2），我们可以检验A和B之间是否存在交互作用。

假设模型中有四个回归量，分别对应四个实验单元格 $(A_1B_1), (A_1B_2), (A_2B_1), (A_2B_2)$，其参数为 $\beta_{11}, \beta_{12}, \beta_{21}, \beta_{22}$。交互作用的检验可以通过一个巧妙的对比向量 $c = [1, -1, -1, 1]^\top$ 来实现 。这个对比所检验的量 $c^\top\boldsymbol{\beta} = \beta_{11} - \beta_{12} - \beta_{21} + \beta_{22}$ 可以从两个等价的角度来理解，即“差异的差异”（difference-of-differences）：

1.  **因素B的简单效应在因素A的不同水平上是否存在差异？**
    $c^\top\boldsymbol{\beta} = (\beta_{11} - \beta_{12}) - (\beta_{21} - \beta_{22})$
    这里，$(\beta_{11} - \beta_{12})$ 是在A1水平下，B1与B2的效应差异（因素B的简单效应）。$(\beta_{21} - \beta_{22})$ 是在A2水平下，B1与B2的效应差异。整个表达式检验的是这两个“差异”本身是否存在差异。

2.  **因素A的简单效应在因素B的不同水平上是否存在差异？**
    $c^\top\boldsymbol{\beta} = (\beta_{11} - \beta_{21}) - (\beta_{12} - \beta_{22})$
    同理，这检验的是因素A的简单效应是否随因素B的水平变化而变化。

这两个角度是代数等价的，都准确地捕捉了交互作用的本质。若 $c^\top\boldsymbol{\beta} = 0$，则意味着两个因素是可加的，不存在[交互作用](@entry_id:164533)。

#### [共线性](@entry_id:270224)、可估计性与对比方差

**[共线性](@entry_id:270224)（Collinearity）**，即设计矩阵中回归量之间的高度相关，是[fMRI分析](@entry_id:1125162)中的一个普遍问题。它对参数估计和[假设检验](@entry_id:142556)有着深远的影响。当两个回归量 $x_1$ 和 $x_2$ 高度相关时（相关系数 $\rho$ 接近1），模型很难区分它们各自对信号的独特贡献 。

这首先体现在单个参数估计的方差上。对于一个包含两个标准化回归量的简单模型，$\beta_1$ 估计的方差为 $\text{Var}(\hat{\beta}_1) = \frac{\sigma^2}{n(1-\rho^2)}$。随着 $\rho$ 趋近于1，分母趋近于0，方差会急剧膨胀。这意味着我们对 $\beta_1$ 的真实值越来越不确定，检验其是否为零的统计功效（power）也随之降低 。

然而，[共线性](@entry_id:270224)对不同对比的影响并非一概而论。考虑两个对比：差异对比 $c_\Delta = [1, -1]^\top$ 和总和对比 $c_\Sigma = [1, 1]^\top$。它们的方差分别为：
- $\text{Var}(c_\Delta^\top\boldsymbol{\hat{\beta}}) = \text{Var}(\hat{\beta}_1 - \hat{\beta}_2) = \frac{2\sigma^2}{n(1-\rho)}$
- $\text{Var}(c_\Sigma^\top\boldsymbol{\hat{\beta}}) = \text{Var}(\hat{\beta}_1 + \hat{\beta}_2) = \frac{2\sigma^2}{n(1+\rho)}$

当 $\rho \to 1$ 时，差异对比的方差趋于无穷，这符合我们的直觉：当两个事物几乎无法分辨时，估计它们的差异变得极其困难。但与此同时，总和对比的方差却趋于一个有限的最小值 $\frac{\sigma^2}{n}$。这说明，即使我们无法确定每个回归量的独立贡献，我们或许仍能精确地估计它们的共同效应。这个现象凸显了通过设计恰当的对比来提出在[共线性](@entry_id:270224)条件下仍能稳健回答的科学问题的重要性。

当[共线性](@entry_id:270224)达到极致（$\rho=1$），设计矩阵便会**[秩亏](@entry_id:754065)（rank-deficient）**。此时，模型参数不再是唯一可识别的，某些参数或其线性组合变得**不可估计（non-estimable）**。一个对比 $c^\top\boldsymbol{\beta}$ 是可估计的，当且仅当对比向量 $c$ 位于设计矩阵 $\mathbf{X}$ 的[行空间](@entry_id:148831)中 。直观上，这意味着该假设所关注的效应组合能够被数据所唯一确定。如果一个对比是不可估计的，强行进行检验是没有意义的。在这种情况下，分析软件通常会通过采用[广义逆](@entry_id:140762)（generalized inverse）来找到一个解，但这实际上检验的是用户指定的对比向量在可估计空间上的一个**投影**，其结果可能并非研究者最初的意图。因此，在设计实验和分析模型时，避免或识别完全[共线性](@entry_id:270224)至关重要。

#### 体素内[多重比较](@entry_id:173510)

研究者在分析一个体素时，往往会检验多个不同的对比，以探索数据的多个方面。例如，对于一个三条件的实验，我们可能检验三个主效应和三个两两比较，总共6个[t检验](@entry_id:272234) 。这种做法引入了**体素内（within-voxel）的[多重比较问题](@entry_id:263680)**。

如果我们为每个检验都使用 $\alpha = 0.05$ 的显著性水平，那么在所有[原假设](@entry_id:265441)都为真的情况下（即该体素没有任何真实效应），至少犯一次I类错误（[假阳性](@entry_id:197064)）的概率——即**族系误差率（Family-Wise Error Rate, FWER）**——会显著膨胀。假设这6个检验是独立的，FWER会是 $1 - (1-0.05)^6 \approx 0.265$，远高于预期的5%。

一种控制体素内FWER的有效策略是采用分层检验，通常以一个综合[F检验](@entry_id:274297)作为“守门员”（gatekeeper）。首先，我们进行一个单一的综合[F检验](@entry_id:274297)，检验所有感兴趣的效应是否联合为零（例如，检验 $H_0: \beta_1=\beta_2=\beta_3=0$），并将此[F检验](@entry_id:274297)的显著性水平设为 $\alpha$。只有当这个综合检验的结果是显著的，我们才被“允许”继续进行后续的各个[t检验](@entry_id:272234)。这种方法（Fisher's Protected LSD）能够在$\alpha$水平上提供对FWER的强控制，因为它确保了只有在有证据表明存在“某种”效应时，我们才去寻找具体的效应，从而控制了在纯噪声情况下犯错的概率 。

#### 软件实现与对比缩放

不同的[fMRI分析](@entry_id:1125162)软件包（如SPM, FSL, AFNI）在内部处理设计矩阵时可能有不同的**缩放约定（scaling conventions）**。例如，某个软件可能会自动将所有回归量归一化到相同的高度或方差。这意味着，即使对于完全相同的[实验设计](@entry_id:142447)，不同软件内部的 design matrix $\mathbf{X}_i$ 和 parameter estimates $\boldsymbol{\hat{\beta}}_i$ 也可能不同 。

如果研究者将在一个“物理”尺度上定义的对比向量 $c_{\text{phys}}$（例如，$[1, -1, 0]$）不加修改地输入到所有软件中，由于各软件的 $\boldsymbol{\beta}_i$ 尺度不同，这个固定的数值向量实际上是在检验不同的假设，从而导致t值或[F值](@entry_id:178445)失去可比性。

为了确保结果的可比性，必须根据每个软件的[参数化](@entry_id:265163)方式来相应地转换对比向量。如果某软件的内部[设计矩阵](@entry_id:165826) $\mathbf{X}_i$ 是通过对一个基准设计 $\mathbf{X}$ 进行列缩放得到的，即 $\mathbf{X}_i = \mathbf{X}D_i$（其中 $D_i$ 是[对角缩放](@entry_id:748382)矩阵），那么基准模型中的参数 $\boldsymbol{\beta}$ 和该软件模型中的参数 $\boldsymbol{\beta}_i$ 的关系是 $\boldsymbol{\beta} = D_i \boldsymbol{\beta}_i$。因此，一个在基准参数下定义的假设 $c_{\text{phys}}^\top\boldsymbol{\beta} = 0$，在软件i的参数下必须表示为 $(D_i^\top c_{\text{phys}})^\top\boldsymbol{\beta}_i = 0$。正确的做法是使用转换后的对比向量 $c_i = D_i^\top c_{\text{phys}}$ 。虽然这在实践中可能难以操作，但理解这一原理对于正确解释和比较来自不同分析流程的结果至关重要。

总之，对比是[fMRI数据分析](@entry_id:1125164)中连接理论模型与科学假设的桥梁。深刻理解其构建原则、统计特性以及在实践中可能遇到的陷阱，是进行严谨、可重复神经影像研究的基石。
