## 引言
在探索生命演化宏伟画卷的征程中，准确地组织和命名生物是理解其历史关系的基础。然而，并非所有对生物的分类都等同地反映了真实的演化路径。许多看似合理的生物组合，实际上可能掩盖或歪曲了它们深刻的谱系联系。为了解决这一根本性问题，现代[演化生物学](@entry_id:145480)和系统发育学引入了三个关键概念——[单系群](@entry_id:142386)、[并系群](@entry_id:147411)和[多系群](@entry_id:168427)——作为区分自然类群与人为分组的精确工具。本文旨在系统性地阐明这三者之间的区别及其深远意义。在“原理与机制”一章中，我们将深入探讨它们的定义、识别标准以及背后的演化过程。接着，在“应用与跨学科联系”一章中，我们将展示这些概念如何被用于修正我们对[生命之树](@entry_id:139693)的理解，并解决从[基因组学](@entry_id:138123)到[古生物学](@entry_id:151688)等不同领域的问题。最后，通过“动手实践”部分，您将有机会运用所学知识解决具体的系统发育难题。让我们首先进入第一章，揭示这些分[类群](@entry_id:182524)体的核心原理与机制。

## 原理与机制

在系统发育学的框架内，理解[类群](@entry_id:182524)（taxon）的组织方式是重建[生命之树](@entry_id:139693)的核心。并非所有看似合理的生物分组都具有同等的演化意义。现代[分类学](@entry_id:172984)，或称[支序学](@entry_id:143946)（cladistics），坚持认为，有效的[生物分类](@entry_id:162997)单元必须精确地反映其演化历史。为了实现这一目标，系统发育学家使用了三个关键术语来描述[类群](@entry_id:182524)与其[共同祖先](@entry_id:175919)之间的关系：**[单系群](@entry_id:142386)（monophyletic）**、**[并系群](@entry_id:147411)（paraphyletic）**和**[多系群](@entry_id:168427)（polyphyletic）**。本章将深入探讨这些概念的定义、潜在的[演化机制](@entry_id:196221)以及它们在[系统发育推断](@entry_id:182186)中的应用与意义。

### [单系群](@entry_id:142386)：演化的完整单元

在现代[系统发育分析](@entry_id:172534)中，**[单系群](@entry_id:142386)**（monophyletic group）是唯一被认为是自然且有效的分类单元。一个[单系群](@entry_id:142386)，也被称为**[支系](@entry_id:171685)**（clade），由一个共同祖先及其**所有**后代组成。这个定义看似简单，却蕴含着深刻的演化逻辑：一个[单系群](@entry_id:142386)代表了生命之树上的一个完整分支，包含了从一个祖先节点分化出来的全部历史。

为了准确识别[单系群](@entry_id:142386)，我们必须精确地理解物种间的[亲缘关系](@entry_id:172505)。例如，假设我们通过基因测序确定了六个新发现的物种（A, B, C, D, E, F）之间的[演化关系](@entry_id:175708)。物种 F 是所有其他物种的外群；在内群中，一支包含物种 A、B、C，另一支包含物种 D、E；在第一支中，物种 A 和 B 的[亲缘关系](@entry_id:172505)最近，共同构成物种 C 的姐妹群。基于这样的[系统发育树](@entry_id:140506)，我们可以评估不同的物种组合是否构成[单系群](@entry_id:142386)。

考虑一个包含物种 A、B、C 的类群 $ \{A, B, C\} $。该类群的[最近共同祖先](@entry_id:136722)（Most Recent Common Ancestor, MRCA）是物种 C 与 (A, B) 分支的共同祖先。该祖先的所有后代恰好就是物种 A、B 和 C。因此，类群 $ \{A, B, C\} $ 是一个[单系群](@entry_id:142386)。相比之下，考虑[类群](@entry_id:182524) $ \{A, B, C, D\} $。这个类群的 MRCA 是包含 (A, B, C) 和 (D, E) 的更大分支的祖先。该祖先的后代包含了物种 E，但物种 E 并没有被包含在 $ \{A, B, C, D\} $ 这个组合中。由于该组合包含了一个祖先但未能包含其所有后代，因此它不是一个[单系群](@entry_id:142386) [@problem_id:1948228]。

那么，我们如何在实践中识别[单系群](@entry_id:142386)呢？[单系群](@entry_id:142386)通常由**共源衍征**（synapomorphy）来定义。共源衍征是指在一个类群的[最近共同祖先](@entry_id:136722)中首次出现、并传递给其所有后代的衍生特征（derived character）。这些特征是演化创新的证据，将一个[支系](@entry_id:171685)与其亲属区分开来。例如，在一个假想的[系外行星](@entry_id:183034)生命研究中，科学家发现物种 A、B、C 共同拥有一个独特的、复杂的三叶晶状体光感受器。进一步的研究表明，这一结构在其[最近共同祖先](@entry_id:136722)中首次演化出来，并且不存在于更疏远的物种 D 和 E 中。因此，这个三叶晶状体[光感受器](@entry_id:151500)就是一个共源衍征，它明确地定义了包含物种 A、B、C 的[单系群](@entry_id:142386) ‘Crystallopsida’ [@problem_id:1948230]。哺乳动物的毛发和泌乳能力也是定义哺乳动物这个[单系群](@entry_id:142386)的经典共源衍征。

### [并系群](@entry_id:147411)：不完整的历史片段

与[单系群](@entry_id:142386)反映完整演化历史不同，**[并系群](@entry_id:147411)**（paraphyletic group）描绘的是一幅不完整的图景。一个[并系群](@entry_id:147411)包含其[最近共同祖先](@entry_id:136722)，但**并非全部**后代。这种分组通常源于对演化模式的传统解读，即倾向于将演化出显著不同特征的某一支后代从其祖先[类群](@entry_id:182524)中“移除”。

一个典型的例子是，假设一个远古祖先物种 *Protofundus antiquus* 分化出两个谱系。一个谱系演化为 *Thermococcus vulcanus* 和 *Pyrococcus abyssi*，另一个演化为 *Aciduliprofundum cryophilum* 和 *Geoglobus infernus*。如果一个研究者提议建立一个名为“Vulcanaceae”的科，包含祖先 *P. antiquus* 以及 *T. vulcanus* 和 *P. abyssi*，但明确排除 *A. cryophilum* 和 *G. infernus*，那么“Vulcanaceae”就是一个[并系群](@entry_id:147411)。因为它包含了共同祖先，但故意遗漏了该祖先的一部分后代 [@problem_id:1948204]。

[并系群](@entry_id:147411)的形成往往与分类学者对[性状演化](@entry_id:165250)的关注点有关。一种常见情况是基于**[祖征共享](@entry_id:169776)**（symplesiomorphy）来定义[类群](@entry_id:182524)。[祖征共享](@entry_id:169776)是指一个或多个物种共同保留了其远祖的原始特征状态。如果仅仅因为物种 A、B、C 保留了某个祖征（如发光器官）就将它们归为一类，而忽略了它们的姐妹群 D 和 E（它们是同一祖先的后代，但丢失了该特征），那么所形成的类群就是[并系群](@entry_id:147411) [@problem_id:1948254]。

这种基于祖征的分类方法在历史上非常普遍，导致了许多我们熟知的但并非[单系群](@entry_id:142386)的分类单元。最经典的例子是“爬行类”（Reptilia）。传统的爬行类包括龟、蜥蜴、蛇和鳄鱼，但不包括鸟类。然而，[系统发育分析](@entry_id:172534)明确指出，鳄鱼与鸟类的[亲缘关系](@entry_id:172505)比其与蜥蜴或龟的关系更近。这意味着鸟类是从爬行动物祖先演化而来的一个分支。因此，一个包含鳄鱼但不包含鸟类的“爬行类”定义，是一个典型的[并系群](@entry_id:147411)。

在研究过程中，形态学数据和分子数据之间的冲突有时也能揭示[并系群](@entry_id:147411)的存在。例如，对一组名为“petramorphs”的生物进行形态学分析，发现物种 $\alpha, \beta, \gamma$ 共享一种复杂的三叶壳结构，而物种 $\delta$ 和外群则拥有简单的单叶壳。这似乎表明 $ \{\alpha, \beta, \gamma\} $ 是一个[单系群](@entry_id:142386)。然而，DNA 序列分析却得出了一个截然不同的[系统发育树](@entry_id:140506)： `(外群, ($\alpha$, ($\beta$, ($\gamma$, $\delta$))))`。在这个树上，$ \{\alpha, \beta, \gamma\} $ 包含了其共同祖先，但排除了后代 $\delta$。因此，基于形态的这个分组实际上是[并系群](@entry_id:147411)。对这种冲突最简约的演化解释是，三叶壳结构是在所有 petramorphs 的[共同祖先](@entry_id:175919)中演化出来的（即它是一个共源衍征），但随后在演化向物种 $\delta$ 的谱系中**二次丢失**了 [@problem_id:1948246]。这一过程被称为**性状逆转**（character reversal），是导致基于形态的分类与真实演化史不符的重要原因之一。

### [多系群](@entry_id:168427)：趋同的幻象

第三种类型的分组是**[多系群](@entry_id:168427)**（polyphyletic group）。一个[多系群](@entry_id:168427)是由两个或多个谱系的成员组成，但**不包含**这些谱系的[最近共同祖先](@entry_id:136722)。这种分组通常是基于**同塑性**（homoplasy）特征而错误建立的，即性状上的相似性并非源于共同祖先，而是由**[趋同演化](@entry_id:263490)**（convergent evolution）、**平行演化**（parallel evolution）或性状逆转独立产生。

一个宏观且直观的例子是，有人可能提议将现代大白鲨、海豚和已灭绝的鱼龙归为一个名为“海洋捕食者”的[类群](@entry_id:182524)，理由是它们都拥有相似的纺锤形（鱼雷状）身体和用于游动的鳍。然而，系统发育学证据表明，鲨鱼是软骨鱼，海豚是哺乳动物（其祖先是陆生的），而鱼龙是独特的海洋爬行动物谱系。它们的[最近共同祖先](@entry_id:136722)是一种早期的有颌脊椎动物，完全不具备这种高度特化的体型。它们相似的身体形态是为了适应高速水生捕食生活方式而独立演化出来的，是[趋同演化](@entry_id:263490)的经典案例。因此，“海洋捕食者”是一个典型的[多系群](@entry_id:168427) [@problem_id:1948247]。

[趋同演化](@entry_id:263490)不仅发生在宏观形态上，在分子水平也同样普遍。例如，研究人员在一种萤火鱿、一种灯笼鱼和一种甲藻中发现了一种名为 `Lux-Variant-Z` 的高度相似的[荧光素酶](@entry_id:155832)。如果仅凭共享这一复杂酶就将它们归为一类，便会形成一个[多系群](@entry_id:168427)。深入的[系统发育分析](@entry_id:172534)表明，这三个物种的[最近共同祖先](@entry_id:136722)是一种简单的、不发光的单细胞真核生物。`Lux-Variant-Z` 酶是在鱼类和[鱿鱼](@entry_id:139582)的谱系中独立演化形成的，而甲藻则是通过与不相关细菌的**水平基因转移**（Horizontal Gene Transfer, HGT）获得了该基因 [@problem_id:1948219]。

在微生物世界中，水平基因转移是造成[多系群](@entry_id:168427)混淆的一个主要因素。[质粒](@entry_id:263777)（plasmid）等[可移动遗传元件](@entry_id:153658)可以在亲缘关系很远的物种间（甚至跨越域，如细菌和[古菌](@entry_id:147706)）传递基因。如果一个分[类群](@entry_id:182524)，如“Luminosa”，被定义为包含所有拥有特定基因（如 `luxG`）的生物，而这个基因是通过 HGT 广泛传播的，那么“Luminosa”必然是一个[多系群](@entry_id:168427)，因为它汇集了不同演化分支的成员，而它们的[共同祖先](@entry_id:175919)并不拥有该基因 [@problem_id:1948206]。

### 深入考量：[系统发育推断](@entry_id:182186)中的复杂性

准确地区分这三[类群](@entry_id:182524)体不仅需要清晰的定义，还需要对[系统发育树](@entry_id:140506)的构建和解释有更深入的理解。

首先，单系、并系和多系的概念都内含了“祖先”与“后代”的方向性。这种[方向性](@entry_id:266095)仅在**[有根树](@entry_id:266860)**（rooted tree）上才存在。一个**[无根树](@entry_id:199885)**（unrooted tree）只展示了物种间的相对[亲缘关系](@entry_id:172505)（拓扑结构），但没有指定演化的时间方向，即没有指定谁是祖先。因此，仅凭一棵[无根树](@entry_id:199885)，我们无法确定性地对一个[类群](@entry_id:182524)进行分类。例如，对于五个物种 P, Q, R, S, T，如果[无根树](@entry_id:199885)的拓扑结构显示 (P,Q) 和 (S,T) 分别是姐妹群，而 R 连接在连接这两对的中心枝上，那么对于类群 $ \{R, S, T\} $ 的分类将完全取决于根（root）的位置。如果根位于将 $ \{P, Q\} $ 与 $ \{R, S, T\} $ 分开的枝上，则 $ \{R, S, T\} $ 是[单系群](@entry_id:142386)。但如果根位于其他位置，例如将 $ \{S, T\} $ 与 $ \{P, Q, R\} $ 分开的枝上，则 $ \{R, S, T\} $ 会变为[并系群](@entry_id:147411)。这凸显了确定树根对于正确解释演化关系至关重要 [@problem_id:1948244]。

其次，即使我们使用了复杂的算法来构建[系统发育树](@entry_id:140506)，所使用的方法和模型也可能引入系统性偏差，从而产生错误的[类群](@entry_id:182524)。一个著名的例子是**[长枝吸引](@entry_id:141763)**（Long-Branch Attraction, LBA）。当两个或多个谱系演化速率非常快时（在树上表现为很长的枝），它们会独立地积累大量随机的基因突变。如果使用的演化模型过于简单（例如，假设所有位点的突变速率都相同），算法可能会错误地将这些谱系归为一类，因为它们共享了大量由趋同演化产生的“假”信号（同塑性），而不是真正的共源衍征。

例如，假设对五个物种 A, B, C, D, E（E 为外群）进行[系统发育分析](@entry_id:172534)，其中物种 C 和 D 的[演化速率](@entry_id:202008)远高于 A 和 B。使用一个简单的、未考虑[位点间速率异质性](@entry_id:177947)（Among-Site Rate Variation, ASRV）的模型进行分析，可能会以极高的置信度得出一个将 C 和 D 聚在一起的“Rapidis”[类群](@entry_id:182524)。然而，当我们换用一个更复杂、更符合生物学现实的（例如，包含伽马[分布](@entry_id:182848)来模拟 ASRV）模型时，可能会得到一个完全不同的、支持度也很高的树，其中 C 与 A 聚类，D 与 B [聚类](@entry_id:266727)。在这种情况下，后一个结果通常被认为是更可靠的。第一个分析中得到的“Rapidis”[类群](@entry_id:182524)，实际上是一个由[长枝吸引](@entry_id:141763)造成的**[多系群](@entry_id:168427)假象**（polyphyletic artifact） [@problem_id:1948252]。

综上所述，对单系、并系和[多系群](@entry_id:168427)的理解是现代演化生物学的基础。它要求我们将分类实践建立在对演化历史的真实重建之上，并警惕那些由[趋同演化](@entry_id:263490)、性状丢失或分析方法局限性所造成的误导性分组。只有[单系群](@entry_id:142386)——反映了完整演化谱系的分支——才能作为我们理解生命多样性宏伟蓝图的坚实基石。
