## Principles and Mechanisms

In our exploration of mathematical analysis, the concept of a function transitions from a mere computational rule to a more abstract and powerful idea: a **mapping** between sets. This chapter delves into the fundamental principles and mechanisms governing these mappings. We will move beyond the simple evaluation of functions at points to analyze their global properties, their behavior across entire sets of inputs, and the structural consequences of their composition.

### The Foundation: Well-Defined Mappings

At its core, a function $f$ from a set $X$ to a set $Y$, denoted $f: X \to Y$, is a rule that assigns to **each** element $x \in X$ a **single, uniquely determined** element $y \in Y$. The set $X$ is the **domain** of the function, and $Y$ is its **[codomain](@entry_id:139336)**. This requirement of a unique assignment is paramount and gives rise to the concept of a function being **well-defined**.

The necessity of a well-defined rule becomes most apparent when the elements of the domain can be represented in multiple ways. Consider the set of rational numbers, $\mathbb{Q}$. Any rational number can be written as a fraction $p/q$ in infinitely many ways; for example, $1/2 = 2/4 = -3/-6$. If a rule for a function with domain $\mathbb{Q}$ depends on the specific choice of numerator $p$ and denominator $q$, rather than the value of the rational number itself, it fails to be a [well-defined function](@entry_id:146846).

For instance, consider a rule $f_C: \mathbb{Q} \to \mathbb{Z}$ proposed as $f_C(p/q) = p + 2q$. If we apply this rule to the number one-half using the representation $1/2$, we get $f_C(1/2) = 1 + 2(2) = 5$. However, using the equivalent representation $2/4$, we get $f_C(2/4) = 2 + 2(4) = 10$. Since a single element in the domain ($1/2$) is mapped to multiple, distinct elements in the codomain ($5$ and $10$), this rule does not constitute a function. The same failure occurs for rules like $f_A(p/q) = p-q$ [@problem_id:1797406].

In contrast, a rule like $f_D: \mathbb{Q} \to \mathbb{Q}$ defined by $f_D(p/q) = \frac{3p}{q}$ is well-defined. This is because the expression can be rewritten as $3(\frac{p}{q})$, which depends only on the value of the rational number, not its representation. If we take $p/q = kp/kq$ for any non-zero integer $k$, the rule gives $\frac{3(kp)}{kq} = \frac{3p}{q}$, yielding a consistent, unique output. This demonstrates that for a rule to define a function, its output must be invariant under different representations of the same input element [@problem_id:1797406].

### Image and Preimage: Tracking Sets Through Functions

While we can evaluate a function at a single point, its true character is often revealed by how it transforms entire subsets of its domain and [codomain](@entry_id:139336). This leads us to the crucial concepts of [image and preimage](@entry_id:148315).

The **image** of a subset $A \subseteq X$ under a function $f: X \to Y$, denoted $f(A)$, is the set of all outputs in $Y$ that are generated by elements from $A$. Formally, $f(A) = \{f(x) \mid x \in A\}$. The image of the entire domain, $f(X)$, is often called the **range** of the function. The range is a subset of the [codomain](@entry_id:139336), but not necessarily equal to it.

Calculating the [image of a function](@entry_id:262157), especially a composite function, requires tracing the set of values through each transformation. Consider a composite function $h = g \circ f$, where $f(x) = \frac{3\pi}{2} + \frac{5\pi}{4}\cos(x)$ and $g(y) = \lfloor y/\pi \rfloor$. To find the image of $h$, we first determine the range of $f$. Since the cosine function oscillates between $-1$ and $1$, the values of $f(x)$ will lie in the closed interval $[\frac{3\pi}{2} - \frac{5\pi}{4}, \frac{3\pi}{2} + \frac{5\pi}{4}]$, which simplifies to $[\frac{\pi}{4}, \frac{11\pi}{4}]$. This interval is the domain for the next step of the composition. The argument to the [floor function](@entry_id:265373) in $g$ is $y/\pi$, which now varies continuously over the interval $[\frac{1}{4}, \frac{11}{4}]$. The [floor function](@entry_id:265373), $\lfloor \cdot \rfloor$, then maps this continuous interval to a discrete set of integers. The integers produced are all integers $k$ such that $k \le t$ for some $t \in [\frac{1}{4}, \frac{11}{4}]$. This yields the set $\{\lfloor 1/4 \rfloor, \dots, \lfloor 11/4 \rfloor\} = \{0, 1, 2\}$. Thus, the image of the [composite function](@entry_id:151451) $h$ is the set $\{0, 1, 2\}$ [@problem_id:1300283].

Conversely, the **preimage** (or [inverse image](@entry_id:154161)) of a subset $C \subseteq Y$, denoted $f^{-1}(C)$, is the set of all elements in the domain $X$ that map into $C$. Formally, $f^{-1}(C) = \{x \in X \mid f(x) \in C\}$. It is crucial to understand that the notation $f^{-1}$ here does not imply that an inverse function exists; it is simply notation for the set of inputs that land in a target set of outputs.

As an example, let $\mathcal{B}$ be the set of all finite [binary strings](@entry_id:262113) and consider the length function $L: \mathcal{B} \to \mathbb{N}_0$, where $\mathbb{N}_0 = \{0, 1, 2, \dots\}$. Let's find the [preimage](@entry_id:150899) of the set $A = \{4, 6\}$. By definition, $L^{-1}(A)$ is the set of all [binary strings](@entry_id:262113) $s \in \mathcal{B}$ such that their length $L(s)$ is an element of $A$. This means we are looking for all strings whose length is either 4 or 6. This set is infinite and contains strings like "0000", "1010", "111111", etc. It is not the set $\{4, 6\}$ itself, nor is it a count of the elements; it is a subset of the domain $\mathcal{B}$ [@problem_id:1300290].

The [preimage](@entry_id:150899) has a very clean interaction with [set operations](@entry_id:143311). For instance, for any subsets $C, D$ of the [codomain](@entry_id:139336), the identity $f^{-1}(C \cup D) = f^{-1}(C) \cup f^{-1}(D)$ always holds. This means we can find the [preimage](@entry_id:150899) of a union by finding the preimages of the individual sets and then taking their union. This property can be useful in calculations. For the function $f(x) = x^2 \pmod{12}$ mapping from $\{1, \dots, 100\}$ to $\{0, \dots, 11\}$, finding the number of elements in the [preimage](@entry_id:150899) of $\{0,2,3,4,6,8,9,10\}$ involves identifying which input residues modulo 12 produce these output values when squared, and then counting how many integers from 1 to 100 fall into those [residue classes](@entry_id:185226) [@problem_id:1797383].

### Classifying Mappings: Injectivity, Surjectivity, and Bijectivity

Beyond the basic definition, functions can be classified by their mapping behavior. The three most fundamental properties are injectivity, [surjectivity](@entry_id:148931), and bijectivity.

A function $f: X \to Y$ is **injective** (or **one-to-one**) if distinct elements in the domain always map to distinct elements in the [codomain](@entry_id:139336). Formally, for all $x_1, x_2 \in X$, if $x_1 \neq x_2$, then $f(x_1) \neq f(x_2)$. The logically equivalent contrapositive is often easier to use in proofs: if $f(x_1) = f(x_2)$, then $x_1 = x_2$. An [injective function](@entry_id:141653) never maps two different inputs to the same output.

A function $f: X \to Y$ is **surjective** (or **onto**) if its image is equal to its entire [codomain](@entry_id:139336). That is, for every element $y \in Y$, there exists at least one element $x \in X$ such that $f(x) = y$. A [surjective function](@entry_id:147405) "hits" every possible target in the [codomain](@entry_id:139336).

A function is **bijective** if it is both injective and surjective. A [bijection](@entry_id:138092) establishes a perfect [one-to-one correspondence](@entry_id:143935) between the elements of the domain and the codomain.

To illustrate these concepts, consider a set of three files, $F = \{\text{file}_1, \text{file}_2, \text{file}_3\}$. The set of all possible access privilege sets is the power set of $F$, denoted $\mathcal{P}(F)$, which has $2^3=8$ elements. Let's define a function $L: \mathcal{P}(F) \to \{0, 1, 2, 3\}$ that maps a privilege set $A \in \mathcal{P}(F)$ to its [cardinality](@entry_id:137773), $L(A) = |A|$ [@problem_id:2299536].

*   **Is $L$ injective?** No. There are multiple distinct sets with the same [cardinality](@entry_id:137773). For example, the sets $\{\text{file}_1\}$ and $\{\text{file}_2\}$ are different, but $L(\{\text{file}_1\}) = 1$ and $L(\{\text{file}_2\}) = 1$. Since different inputs map to the same output, the function is not injective.
*   **Is $L$ surjective?** Yes. For every integer $y$ in the [codomain](@entry_id:139336) $\{0, 1, 2, 3\}$, we can find a set $A$ that maps to it. For $y=0$, we can use the [empty set](@entry_id:261946) $\emptyset$. For $y=1$, we can use $\{\text{file}_1\}$. For $y=2$, we can use $\{\text{file}_1, \text{file}_2\}$. For $y=3$, we can use the full set $F$. Since every element in the codomain is the image of at least one element in the domain, the function is surjective.

Since $L$ is surjective but not injective, it is not bijective. This example highlights that [injectivity and surjectivity](@entry_id:262885) are independent properties.

### Deeper Connections: Composition, Symmetry, and Set-Theoretic Behavior

The properties of [injectivity and surjectivity](@entry_id:262885) have profound implications, creating connections between a function's structure, its interaction with [set operations](@entry_id:143311), and its behavior under composition.

#### Properties of Composite Functions

The [composition of functions](@entry_id:148459), $g \circ f$, inherits properties from its constituent functions $f$ and $g$ in a precise way.

First, consider [injectivity](@entry_id:147722). If the composite function $g \circ f: A \to C$ is injective, what does this imply about $f: A \to B$ and $g: B \to C$? Let's assume $f(x_1) = f(x_2)$ for some $x_1, x_2 \in A$. Applying the function $g$ to both sides gives $g(f(x_1)) = g(f(x_2))$, which is equivalent to $(g \circ f)(x_1) = (g \circ f)(x_2)$. Since we are given that $g \circ f$ is injective, this implies $x_1 = x_2$. Therefore, $f$ must be injective. It is important to note that $g$ does not need to be injective. For example, if $A=\{1\}, B=\{1, 2\}, C=\{1\}$, let $f(1)=1$ and $g(1)=g(2)=1$. The composite function $g \circ f$ is injective (mapping $1 \to 1$), but $g$ is not [@problem_id:1300250].

**Theorem:** If $g \circ f$ is injective, then $f$ is injective.

Next, consider [surjectivity](@entry_id:148931). If $g \circ f: A \to C$ is surjective, what does this tell us? By definition, for any $c \in C$, there must be an $a \in A$ such that $(g \circ f)(a) = c$, or $g(f(a)) = c$. Let $b = f(a)$. Since $f$ maps from $A$ to $B$, this $b$ is an element of $B$. We have just shown that for any $c \in C$, we can find a $b \in B$ (namely, $f(a)$) such that $g(b) = c$. This is precisely the definition of $g$ being surjective. In this case, $f$ does not need to be surjective. In the previous [counterexample](@entry_id:148660), $f$ was not surjective onto $B$ if we define $B=\{1,2\}$, yet $g \circ f$ was surjective onto $C$ [@problem_id:1300282].

**Theorem:** If $g \circ f$ is surjective, then $g$ is surjective.

#### Symmetry, Periodicity, and Injectivity

A function's structural properties, such as symmetry or [periodicity](@entry_id:152486), place strong constraints on its mapping behavior. A function $f: \mathbb{R} \to \mathbb{R}$ is **periodic** if there exists a constant $T > 0$ (a period) such that $f(x+T) = f(x)$ for all $x \in \mathbb{R}$. By definition, we have two distinct inputs, $x$ and $x+T$, that map to the same output value. This is a direct violation of the condition for injectivity. Therefore, any periodic function on $\mathbb{R}$ cannot be injective [@problem_id:2299521].

Similarly, consider an **[even function](@entry_id:164802)**, which satisfies $f(-x) = f(x)$ for all $x \in \mathbb{R}$. For any non-zero number $x_0$, we have $x_0 \neq -x_0$, yet $f(x_0) = f(-x_0)$. Once again, we have found two distinct inputs that map to the same output. This means that any [even function](@entry_id:164802) defined on $\mathbb{R}$ cannot be injective [@problem_id:2299543]. Common examples like $f(x) = x^2$ or $f(x) = \cos(x)$ clearly illustrate this principle. Interestingly, a function can be both even and odd, but only if it is the zero function, $f(x)=0$ for all $x$, which is also not injective on $\mathbb{R}$ [@problem_id:2299521].

#### A Set-Theoretic Characterization of Injectivity

Finally, we can establish a deeper characterization of injectivity based on how a function interacts with set intersection. For any function $f: X \to Y$ and any subsets $A, B \subseteq X$, it is always true that $f(A \cap B) \subseteq f(A) \cap f(B)$. An element $y$ in $f(A \cap B)$ comes from some $x \in A \cap B$. This $x$ is in both $A$ and $B$, so its image $y$ must be in both $f(A)$ and $f(B)$, and thus in their intersection.

The more restrictive condition of equality, $f(A \cap B) = f(A) \cap f(B)$ for **all** subsets $A, B \subseteq X$, turns out to be equivalent to the function being injective.

To prove this, first assume $f$ is injective. We need to show the reverse inclusion, $f(A) \cap f(B) \subseteq f(A \cap B)$. Let $y \in f(A) \cap f(B)$. This means there exists an $a \in A$ with $f(a) = y$ and a $b \in B$ with $f(b) = y$. Since $f(a) = f(b)$ and $f$ is injective, we must have $a=b$. This element is therefore in $A$ and in $B$, meaning it is in their intersection, $A \cap B$. Its image, $y$, is therefore in $f(A \cap B)$.

Conversely, suppose the equality holds for all subsets. To show $f$ is injective, assume for contradiction that it is not. Then there exist $x_1, x_2 \in X$ with $x_1 \neq x_2$ but $f(x_1) = f(x_2)$. Let's choose our subsets cleverly: let $A = \{x_1\}$ and $B = \{x_2\}$. Because $x_1 \neq x_2$, their intersection is empty: $A \cap B = \emptyset$. The image of the empty set is the [empty set](@entry_id:261946), so $f(A \cap B) = \emptyset$. However, $f(A) = \{f(x_1)\}$ and $f(B) = \{f(x_2)\}$. Since $f(x_1)=f(x_2)$, the intersection is $f(A) \cap f(B) = \{f(x_1)\}$, which is not empty. This contradicts our assumption that $f(A \cap B) = f(A) \cap f(B)$. Therefore, the function $f$ must be injective [@problem_id:1300256]. This remarkable result shows that the microscopic property of injectivity (concerning pairs of points) is equivalent to a macroscopic property of how the function behaves on all possible subsets of its domain.
