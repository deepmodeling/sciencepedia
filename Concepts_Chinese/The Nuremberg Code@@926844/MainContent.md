## 引言
Nuremberg Code 诞生于历史上最黑暗的篇章之一的灰烬中，它不仅是一份历史文件，更是现代医学和研究伦理的基石。在此之前，医学界主要遵循家长式作风（paternalism）的原则，即医生的判断至高无上。二战后医生审判（Doctors’ Trial）中揭露的骇人实验，暴露了这种模式的灾难性不足，表明迫切需要一种以不可侵犯的人权为中心的新伦理。本文旨在探讨这场伦理革命的深远影响。首先，我们将审视 Nuremberg Code 的核心**原则与机制**，详细阐述其向患者自主性的根本性转变以及为执行这些原则而建立的体系。随后，我们将探讨其广泛的**应用与跨学科联系**，展示它如何提供一个视角来重新评估历史、构建当今的研究，并应对未来的伦理困境。

## 原则与机制

要理解医学伦理的世界，不能仅仅背诵一堆规则。我们必须像在任何科学中一样，回归到第一性原理。我们必须问：我们试图解决的根本问题是什么？在 1940 年代末之前，医学的指导原则继承自 Hippocrates 时代，是一种职业家长式作风。医生的职责是为患者的最大利益行事（**行善原则**）并避免造成伤害（**不伤害原则**）。这是一种建立在信任之上的关系，医生凭借其专业知识成为最终的决策者。患者的角色主要是顺从，相信医生最了解情况。

然而，这种模式无论对临床护理的初衷有多好，都在二战后的 Nuremberg 医生审判中被揭露的恐怖行径所粉碎。世界直面了纳粹医生对未经同意的囚犯进行怪诞且致命的实验的现实。他们的辩护常常援引一种扭曲的功利主义逻辑——少数人的痛苦是为了多数人的利益而正当的。这场灾难性的道德失败表明，旧的家长式模式是灾难性地不足。需要一个新的基础，这个基础不建立在医生的自由裁量权之上，而是建立在人类受试者不可侵犯的权利之上。这个基础就是 Nuremberg Code。

### 哥白尼式的革命：“绝对必要”原则

Nuremberg Code 代表了医学伦理学上的一场哥白尼式革命。它将道德宇宙的中心从医生转移到了个体身上。这一点体现在其第一条，也是最著名、最有力的一条中：**“人类受试者的自愿同意是绝对必要的。”** [@problem_id:4867469]

这短短一句话在今天看来或许显而易见，但在当时却是一个激进的突破。它宣称，人的身体不是科学或社会可以支配的资源。它将个人确立为主权者，在被纳入任何实验之前，必须征得其自由给予的许可。该法典接着定义了“自愿同意”的真正含义，建立了一个清晰度和严格性都令人惊叹的标准 [@problem_id:4858986]：

*   **能力与自由：** 当事人必须具有作出同意的法律能力，并且其所处环境必须能够使其行使自由选择权，“不受任何暴力、欺诈、欺骗、胁迫、越权或任何其他[隐蔽](@entry_id:196364)形式的约束或强迫”。这为抵御任何形式的压力竖起了一道墙。

*   **知识与理解：** 受试者必须“[对相关](@entry_id:203353)主题的各项要素有充分的知识和理解，从而能够做出明智和开明的决定。”这包括全面披露实验的性质、目的、持续时间、方法以及所有可预见的风险和不适。你不能对自己不理解的事情表示同意。

*   **不受约束的退出权：** 当事人必须有随时以任何理由终止参与的自由，且不受惩罚。这项权利是持续且无条件的，确保同意不是一次性的交易，而是一种持续的协议状态。[@problem_id:4951002]

其核心在于，这项原则是一个深刻的哲学声明。它断言，人本身就是目的，而不仅仅是达到目的的手段。个人不能被工具化，即使是为了推进科学或改善公共卫生这样的崇高目标。这个原则起着哲学家所称的**边际约束**（side constraint）的作用；它在计算总体利益时具有**词典式优先**（lexical priority）的地位。这意味着，在开始计算一项研究（我们称之为 $U$）的潜在社会效益之前，你必须首先满足个人自愿同意的绝对要求。你不能用一个人的自主权来换取更大的利益，无论这个利益看起来有多大 [@problem_id:4865185]。

### 安全的支架：十项要点

虽然自愿同意是基石，但 Nuremberg Code 用其十项要点围绕它构建了一个完整的伦理支架。它认识到仅有同意是不够的；研究本身必须在伦理上是合理的。其他原则协同作用以保护参与者：

*   **科学价值与必要性：** 实验必须是必要的，旨在为“社会利益产生成果”，并且无法通过其他方法获得。这可以防止轻率或浪费的实验。

*   **坚实的科学基础：** 研究必须基于先前的动物实验结果，以及对所研究疾病或问题的自然史的透彻了解。

*   **禁止极端风险：** 在其最严格的条款之一中，该法典规定：“不应进行任何有先验理由相信会发生死亡或致残伤害的实验。”这为可允许的风险设定了一个近乎绝对的上限，这是对纳粹时代致命实验的直接回应 [@problem_id:4858986]。

*   **有利的风险-收益比：** 所承担的风险程度绝不能超过由待解决问题的人道主义重要性所决定的程度。

这些原则综合起来，为负责任的研究创建了一个蓝图，要求研究不仅要获得同意，而且要在科学上有效、必要且安全。

### 从理想到制度：执行机制

一套美好的原则如果没有[约束力](@entry_id:170052)，就毫无用处。Nuremberg Code 是法院判决的一部分，而不是国际条约或立法机构通过的法律。那么，它如何成为现代研究伦理的基石呢？答案在于其原则被巧妙地逐步编织到科学的法律、专业和监管体系的结构中——我们可以称之为**伦理-法律接口** [@problem_id:4487840]。

这个过程是分层进行的。首先，该法典建立了一个**注意标准（standard of care）**。在法庭上，未能获得适当同意的研究人员不再仅仅是不道德的；他们可能要为**殴击罪（battery）**——一种未经授权的接触——的侵权行为承担责任。一个进行设计拙劣、风险过高的研究的研究人员不仅仅是一个糟糕的科学家；他们可能因为违反了对参与者的注意义务而承担**过失（negligence）**责任 [@problem_id:4487840]。

但是，在伤害发生后依赖诉讼来保护人们是一种糟糕的方式。该法典的真正力量在其被主动制度化的过程中得以释放。这是通过一系列关键的历史发展实现的，每一次发展都建立在上一次的基础上，并且常常由新伦理失范事件的揭露所推动 [@problem_id:4865213] [@problem_id:4763914]：

1.  **Declaration of Helsinki (1964):** 全球医学界通过世界医学协会（World Medical Association）采纳了 Nuremberg 的原则。Helsinki 是一个关键步骤，因为它是由医生为医生制定的。它引入了一个革命性的思想，即每个研究方案都必须在研究开始*之前*由一个**独立的伦理委员会**进行审查。这将保护从反应性的、事后的机制转变为主动的、预防性的机制。[@problem_id:4865217]

2.  **Belmont Report 与美国 Common Rule:** Nuremberg 来之不易的教训不幸地被国内的丑闻所强化，其中最臭名昭著的是 Tuskegee Syphilis Study，在该研究中，贫困的非裔美国男性被欺骗性地招募，并被故意数十年不予治疗。这种发生在美国本土的骇人听闻的虐待行为证明，伦理暴行并非仅限于某个邪恶政权。美国政府的回应是委托制定了 **Belmont Report**（1979年）。这份里程碑式的文件不仅仅是罗列规则；它将核心伦理思想提炼为三个优美简洁、强有力的原则：**尊重个人（Respect for Persons）**（Nuremberg 的同意原则）、**行善（Beneficence）**（风险-收益分析），以及关键的第三个原则——**公正（Justice）**。公正原则要求公平分配研究的负担和利益，这是对在 Tuskegee 事件中看到的剥削行为的直接回应，当时一个弱势群体承担了所有风险，却被剥夺了应得的利益 [@problem_id:4763914]。

这些 Belmont 原则随后成为美国联邦法律（即 **Common Rule**）的直接伦理基础，该法律强制要求各机构设立**机构审查委员会（Institutional Review Boards, IRBs）**来审查和批准研究 [@problem_id:4503060]。

这个多层次的体系——法律责任、专业自律和具有[约束力](@entry_id:170052)的政府监督——是将 Nuremberg Code 从一项历史判决转变为一个活生生的、可执行的现实的机制。

### 活的法典：自愿性原则的演变

这个伦理框架的最终美妙之处在于，它不是一个僵化、脆弱的教条。它是一套活的原则，随着我们遇到新的、更复杂的挑战而不断演变。例如，Nuremberg Code 的绝对主义既是其优点，也是其局限。它要求有能力的个人亲自同意，这为针对无同意能力者（如紧急情况下的昏迷患者或患有严重痴呆症的人）的关键研究设置了障碍 [@problem_id:4858986]。

Declaration of Helsinki 和 Belmont Report 提供了解决这一问题的思想工具。它们发展了**代理同意（surrogate consent）**的概念，允许法定代理人代表无行为能力者做决定，并为这些**弱势群体**引入了严格的保障措施。

同样，Belmont Report 的框架提供了对**自愿性（voluntariness）**更为精妙的理解。它仔细区分了**胁迫（coercion）**（一种威胁，如果拒绝，会让某人的处境变得更糟）和**不当影响（undue influence）**（一种可能扭曲判断的过度或不当奖励）。这使我们能够审慎地分析复杂情况，例如在紧急情况下必须放弃同意的重症监护试验，或向低收入社区参与者提供报酬的研究。其目标不仅仅是遵守规则，而是要确保即使在困难的情况下，个人的决定从根本上仍然是自己的 [@problem_id:4867514]。

从历史上最黑暗时刻之一的灰烬中，Nuremberg Code 播下了一颗新伦理的种子。几十年来，在医学界的培育和法律的编纂下，它已成长为一个强大而优雅的结构，在保护个人尊严的同时，仍然允许至关重要的科学工作照亮通往全人类更健康未来的道路。它证明了这样一个理念：即使是对知识最强大的追求，也必须屈服于人权这一简单而深刻的真理。
