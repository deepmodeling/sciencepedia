## 引言
我们脚下的大地是活的，并且在呼吸。这种集体的呼出过程被称为[陆地呼吸作用](@article_id:310041)，是地球[碳循环](@article_id:301597)的关键组成部分，但其复杂的运作机制在很大程度上仍然是不可见的。理解这一过程——区分活体[根系](@article_id:377746)的呼吸与分解微生物的呼吸——是现代生态学中最重大的挑战之一。本文全面概述了[陆地呼吸作用](@article_id:310041)，为理解地球的碳经济提供了指南。第一章**“原理与机制”**将剖析基本概念，解释不同类型的呼吸作用以及用于平衡生态系统碳收支的核算框架。随后的**“应用与跨学科联系”**一章将展示这些知识如何应用，从使用高科技通量塔诊断森林健康，到预测生态系统将如何应对气候变暖。

## 原理与机制

想象一下雨后漫步在森林中。空气中弥漫着潮湿泥土浓郁的芳香。从某种意义上说，你所闻到的正是生态系统的呼吸。它是无数生物的集体呼气，是地球[碳循环](@article_id:301597)核心的一个过程。但究竟是什么在呼吸？作为科学家，我们又该如何理解这个巨大而无形的过程？要回答这些问题，我们必须成为碳核算师，细致地追踪生命本身的收入与支出。

### 大地之息：两种呼吸作用

我们的第一反应可能是将那种泥土的气味完全归因于分解作用——细菌和真菌分解落叶和枯木的工作。这确实是故事的重要部分。生物体消耗死亡有机物以获取能量的这个过程，被称为**[异养](@article_id:327299)呼吸作用**（$R_h$）。这些分解者，即**[异养生物](@article_id:374508)**（源自希腊语，意为“取食他者”），是自然界最终的回收者。

但它们并非唯一在呼吸的生物。如果你能窥视土壤表面之下，你会发现一个由高耸树木和植物的根系主导的、熙熙攘攘的隐秘世界。这些[根系](@article_id:377746)是活的，和任何活组织一样，它们必须通过呼吸来为其自身的生长和维持提供能量。这就是**[自养呼吸](@article_id:367197)作用**（$R_a$）。这些植物，即**[自养生物](@article_id:373971)**（源自希腊语，意为“自我供养”），通过光合作用制造自己的食物，然后通过呼吸“消耗”部分食物来维持生命。

因此，从森林地面升起的二氧化碳（$CO_2$）总通量，一个被称为**土壤总呼吸**的量，并不仅仅是腐烂的气息。它是一首二重奏，是来自[异养](@article_id:327299)[分解者](@article_id:365774)的$CO_2$和来自活的[自养](@article_id:325769)[根系](@article_id:377746)的$CO_2$的混合物[@problem_id:1838083]。理解这两种基本呼吸流之间的平衡，是现代生态学的一大挑战与成就。

### 碳核算师：平衡生态系统的账目

要理解呼吸作用的角色，我们必须将其置于生态系统整个碳收支的大背景下。把一个生态系统想象成一家以碳为货币进行交易的企业。

这家企业的总收入是**总[初级生产力](@article_id:311694)（GPP）**。这是植物从大气中吸收并通过光合作用转化为有机化合物的总碳量。它是整个系统的总收入，是其中所有生命的燃料。

然而，没有哪家企业是无成本运营的。植物有运营成本。为了维持细胞、生长新组织以及运输水分和养分，它们必须燃烧一部分刚刚固定的碳。这项支出就是我们已经介绍过的**[自养呼吸](@article_id:367197)（$R_a$）**。

植物支付完自身代谢账单后剩余的碳，就是**净初级生产力（NPP）**。这是植物的净利润，是分配给实际生长（新叶、木材和根）以及储存的碳。用公式表示，这种关系简单而优雅：

$$NPP = GPP - R_a$$

NPP是陆地上几乎所有[食物网](@article_id:379922)的基础。它是食草动物的食物来源，当植物死亡时，它又成为[分解者](@article_id:365774)——[异养生物](@article_id:374508)的食物来源。

当这些[异养生物](@article_id:374508)消耗由NPP提供的有机物时，它们也会呼吸，释放$CO_2$。这就是**[异养](@article_id:327299)呼吸（$R_h$）**。

现在，我们可以提出关于整个生态系统的终极问题：它是在增加碳还是在减少碳？它是大气$CO_2$的净汇，还是净源？这个底线被称为**[净生态系统生产力](@article_id:363499)（NEP）**。它是总输入碳（GPP）与*所有*生物呼吸输出的总碳（[自养](@article_id:325769)和[异养](@article_id:327299)呼吸之和，称为**生态系统呼吸**，$R_{eco}$）之间的差值。

$$NEP = GPP - R_{eco} = GPP - (R_a + R_h)$$

我们还可以看到一个与NPP的美妙联系。既然$NPP = GPP - R_a$，那么可以推导出：

$$NEP = NPP - R_h$$

正的NEP意味着生态系统正在积累碳，充当[碳汇](@article_id:381098)。负的NEP意味着它正在失去碳，充当碳源。值得注意的是，NEP只核算光合作用和呼吸作用的生物通量。它不包括火灾、采伐或碳淋溶到河流中等事件造成的碳损失，这些损失被计入一个更全面的预算，称为净生态系统碳平衡（NECB）[@problem_id:2508865]。对我们而言，GPP、$R_a$和$R_h$之间的互动是主要内容。

### 一日之况：从通量塔数据到[碳通量](@article_id:373068)

这些定义可能看起来很抽象，但它们对应着真实可测的量。想象一座高塔矗立在温带草原上，上面布满了灵敏的仪器。这是一个**[涡度](@article_id:303185)相关**塔，一种能够测量生态系统与上方大气之间$CO_2$净流动的奇妙设备[@problem_id:2483748]。我们称这个测量到的流量为**净生态系统交换量（NEE）**。按照惯例，当生态系统吸收$CO_2$时（如晴天），通量为负值。当它释放$CO_2$时（如夜晚），通量为正值。这意味着$NEP = -NEE$。

让我们来看一个典型夏日的数据[@problem_id:2483760]：
- 在12小时的白天里，通量塔测量到一个强烈的净吸收：$NEE_{day} = -4.8$单位的碳。
- 在12小时的夜晚里，它测量到一个稳定的释放：$NEE_{night} = +2.0$单位的碳。

夜间测量的美妙之处在于其简单性。夜晚没有阳光，所以GPP为零。生态系统只在进行呼吸作用。因此，测得的夜间通量直接给出了生态系统总呼吸速率：$R_{eco} = NEE_{night} = +2.0$单位。等等，这是12小时的呼吸量。我们暂时假设呼吸速率在24小时内是恒定的（我们稍后会完善这一点）。那么，每日生态系统呼吸量是$R_{eco} = 2 \times 2.0 = 4.0$单位。让我们退一步，`2483760`号问题暗示我们需要使用更仔细的计算。24小时的总NEE是$-4.8 + 2.0 = -2.8$单位。这意味着生态系统净增加了$2.8$单位的碳，所以$NEP = 2.8$单位。现在，总GPP是多少？使用我们的主方程$NEP = GPP - R_{eco}$。哦，我们还不能解出来。我们需要一整天的$R_{eco}$。

让我们重读`2483760`号问题。它提供了独立的箱式测量法得出的每日总量：$R_a = 1.5$单位和$R_h = 0.7$单位。啊哈！所以每日生态系统总呼吸是$R_{eco} = R_a + R_h = 1.5 + 0.7 = 2.2$单位。（[涡度](@article_id:303185)相关塔的夜间通量2.0只是夜间时段的，而呼吸速率通常会因昼夜温差而变化）。

现在我们可以解出所有量了！
- 我们知道每日的净结果：$NEP = -NEE = -(-2.8) = 2.8$单位。
- 我们知道总呼吸成本：$R_{eco} = 2.2$单位。
- 我们现在可以计算每日总收入，即GPP：$GPP = NEP + R_{eco} = 2.8 + 2.2 = 5.0$单位。
- 并且我们可以计算植物的净利润，即NPP：$NPP = GPP - R_a = 5.0 - 1.5 = 3.5$单位。

就这样，通过几个关键测量，我们剖析了这片草原一整天的碳经济。我们看到了总光合吸收量（5.0单位），植物自身消耗了多少（1.5单位），剩下用于生长和食物网的量（3.5单位），[分解者](@article_id:365774)消耗了多少（0.7单位），以及整个生态系统的最终净增益（2.8单位）。

### 生态学家的工具箱：如何窥探一个呼吸的森林

那个例子用了一个捷径——我们被*给予*了$R_a$和$R_h$的分解值。但科学家实际上是如何将它们分开的呢？这需要一些巧妙的侦查工作。

最直接（尽管有些粗暴）的方法之一是**[根系](@article_id:377746)排除法**[@problem_id:2479601]。生态学家会在一块样地周围挖一条足够深的沟，切断所有与周围树木相连的活根，并安装一个屏障。然后，放置在被壕沟围绕的样地内土壤表面的箱式测量仪将测得一个理论上纯粹是[异养](@article_id:327299)呼吸（$R_h$）的$CO_2$通量，因为[自养](@article_id:325769)根系的组分已被排除。通过与邻近完整样地（测量的是$R_s = R_h + R_{a,root}$）的通量进行比较，就可以通过相减求解出根系呼吸。

当然，自然界从不那么简单。挖沟这个行为本身就会改变土壤环境，比如改变其温度或湿度，而这又会影响微生物的活动速率。一个好的科学家必须意识到这些潜在的人为影响并进行校正。例如，如果挖沟的样地变得稍冷，测得的$R_h$将低于在更温暖、完整的样地中实际发生的值。通过测量这个温差，并知道微生物呼吸通常如何响应温度（通常用一个称为$Q_{10}$的因子来描述），就可以应用校正以获得更准确的估计[@problem_id:2505172] [@problem_id:2479601]。

另一个非常聪明的工具是使用**[稳定同位素](@article_id:343922)**。大气中的碳主要以两种稳定形式存在：常见的碳-12（$^{12}C$）和稍重的碳-13（$^{13}C$）。植物在光合作用期间，对较轻的$^{12}C$表现出轻微的“偏好”。这意味着新产生的植物糖在同位素上比大气“更轻”。已经在土壤中存在多年的古老有机物则具有不同的同位[素特征](@article_id:316387)。通过向森林“脉冲”注入人工富含$^{13}C$的$CO_2$，然后追踪从土壤中呼出的$CO_2$的同位[素特征](@article_id:316387)，科学家可以区分由“近期”糖分驱动的呼吸（[自养](@article_id:325769)根系呼吸）和由“古老”碳驱动的呼吸（[异养](@article_id:327299)呼吸）。这就像给生态系统的午餐打上标签，然后观察谁在何时吃了它[@problem_id:2479601]。

### 生命的代价：植物的生长与维持预算

让我们再次聚焦于植物自身的成本，即其[自养呼吸](@article_id:367197)（$R_a$）。为什么这个成本如此之高，常常占到GPP总量的近一半？因为生命是昂贵的。我们可以将这个成本分为两大类，很像一个企业的预算[@problem_id:2794566]。

首先是**维持性呼吸（$R_m$）**。这是仅仅为了维持生命所需的能量。它是修复[细胞器](@article_id:314982)、替换降解蛋白质以及维持细胞膜两侧适当化学梯度的成本。它相当于一个工厂即使不生产任何东西，也要保持灯火通明、暖气运行和安保系统在线的代谢成本。

其次是**生长呼吸（$R_g$）**。这是与构建新组织直接相关的能量。它是将简单的糖转化为复杂的分子（如[纤维素](@article_id:305338)、木质素和蛋白质）以形成新叶、茎和根的成本。这是花费在工厂流水线本身的能量。

这个建造过程的效率是可以量化的。成功并入新生物量的底物碳的比例称为**建成产量（$Y_g$）**。例如，如果$Y_g = 0.75$，这意味着每4克用于生长的碳底物中，有3克最终进入最终结构，而有1克作为$CO_2$通过生长呼吸“损失”掉。这为新生物量生产速率（NPP）与该生产的呼吸成本（$R_g$）之间提供了一个强大的、机理性的联系[@problem_id:2794566]。这种更深入的理解揭示了，植物的碳预算是一种动态的分配策略，平衡了生存的即时成本与生长的长期投资。

### 行星的节律：基林曲线的年度叹息

现在，让我们完成从森林地面到整个地球的最后一次、令人惊叹的飞跃。自20世纪50年代以来，夏威夷莫纳罗亚的持续监测追踪了大气中$CO_2$的浓度。这个记录，即著名的**基林曲线**，显示出两个显著的特征：一个由人类活动导致的无情、长期的上升，以及一个叠加在该趋势上的、规则的年度“锯齿状”摆动。

是什么导致了这种年度节律，这种行星的呼吸？这正是我们刚刚讨论的所有生态系统过程的集[体效应](@article_id:325186)[@problem_id:1862261]。地球的陆地并非[均匀分布](@article_id:325445)；北半球拥有远多于南半球的陆地，因此也拥有更多的陆地植被。

在北半球的春季和夏季，广阔的森林和草原苏醒过来。光合作用（GPP）以巨大的规模加速，远远超过生态系统总呼吸（$R_{eco}$）。北方大陆开始从大气中大量吸收$CO_2$。地球，实际上是在吸气。这导致全球大气$CO_2$浓度下降，在九月左右达到最低点。

然后，随着秋冬的到来，树叶凋落，光合作用急剧下降，但呼吸作用仍在继续。[异养生物](@article_id:374508)享用着一年中丰富的落叶，所有生物都继续进行维持性呼吸以度过寒冬。现在，$R_{eco}$超过了GPP。大陆从$CO_2$的净汇转变为净源。地球呼气，大气$CO_2$水平上升，在五月左右达到峰值，恰好在下一个春天的新绿开始新一轮循环之前。

这个美丽的、行星尺度的[振荡](@article_id:331484)是我们故事的伟[大统一](@article_id:320777)。它展示了土壤微生物中酶促反应和单个植物根中[细胞呼吸](@article_id:306727)的微观过程，当跨越大陆和季节进行倍增时，如何累加成一个我们可以从太平洋中部的山顶上测量到的全球现象。这是一个深刻的提醒，提醒我们生命的相互联系，以及定义我们这个生命世界的优雅、复杂和永不停息的能量与物质流动。
