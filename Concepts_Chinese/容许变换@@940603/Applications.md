## 应用与跨学科联系

现在我们已经探讨了测量尺度及其“容许变换”的原理，你可能会想把这些当作抽象的数学分类知识存档。但这样做就完全错过了重点！这些思想不是来自陈旧教科书的枯燥规则；它们是定量科学的基石。它们是沉默的守护者，确保我们的统计工具不会误导我们，确保我们的结论反映的是世界的本来面目，而不仅仅是我们碰巧使用的任意标签或单位。不变性原则是我们把原始数据转化为可靠知识的指南。让我们沿着科学流程走一遭，看看这个原则在每一步中是如何发挥作用的。

### 科学家的初步观察：描述和可视化世界

科学家拿到一个新数据集后做的第一件事，就是试图感受它——总结它，可视化它。“典型”值是什么？数据有多分散？即使是这些基本问题，也受到[测量理论](@entry_id:153616)的制约。

假设你是一名医学研究者，刚从一项临床试验中获得了一份新数据集[@problem_id:4838920]。你手头有患者的基因型（名义变量）、他们在数字评定量表上的自我报告疼痛（序数）、以[摄氏度](@entry_id:141511)为单位的体温（区间），以及他们血液中某种蛋白质的浓度（比率）。每种变量的“平均值”是什么？不变性原则为我们提供了一个优美而层次分明的答案。

对于名义的基因型，其中的标签只是名称，唯一能在任何重新标记后存活下来的操作是计数。我们可以找到最常见的类别——**众数**——但计算一个“平均基因型”是荒谬的。当我们转向[序数](@entry_id:150084)的疼痛评分时，我们的工具箱扩展了。因为顺序现在有了意义，我们不仅可以找到众数，还可以把所有观察值排列起来，找到中间的那个：**中位数**。中位数在任何保序变换下都是不变的。如果你把疼痛评分从$\{1, 2, 3\}$重新标记为$\{10, 20, 100\}$，中间的病人仍然是中间的病人。但我们仍然不能使用算术平均值，因为那将假定从疼痛评分2到3的跳跃与从8到9的跳跃是相同的，而我们无法证明这个假设。

只有当我们达到区间尺度，比如摄氏温度，我们才能有意义地谈论**[算术平均值](@entry_id:165355)**和**方差**。为什么？因为平均值是基于差值的和，而差值正是在区间尺度上被保留的东西。如果我们从[摄氏度](@entry_id:141511)切换到华氏度，一个形式为$y = ax+b$的容许仿射变换，平均值也只是以同样的方式变换。科学结论保持稳定。最后，对于比率尺度，比如具有真零点的[蛋白质浓度](@entry_id:191958)，我们解锁了完整的工具箱。我们现在可以使用依赖于比率的统计量，比如**[几何平均数](@entry_id:275527)**或[变异系数](@entry_id:272423)，它们在适用于比率数据的重新[标度变换](@entry_id:166413)（$y=ax$）下是不变的[@problem_id:4838874]。这其中有一种绝妙的优雅：随着我们[数据结构](@entry_id:262134)的日益丰富，我们可以提出的有意义问题的集合——以及我们可以使用的统计工具——也随之增长。

同样的逻辑也决定了我们如何可视化数据。条形图非常适合名义数据，因为它只显示每个类别的计数；打乱条形的顺序（一种重新标记）并不会改变故事。但对名义数据使用[直方图](@entry_id:178776)将是一个错误，因为它暗示了一种不存在的顺序和邻接关系。同样，对序数疼痛量表使用箱[线图](@entry_id:264599)可能会产生严重的误导。它会诱使我们的眼睛去比较箱体（[四分位距](@entry_id:169909)）和须的长度，把它们当作有意义的距离。但对于序数尺度来说，这些距离只是我们任意选择数值标签所产生的幻觉！一个更好的显示离散程度的方法是直接报告对应于第一和第三[四分位数](@entry_id:167370)的类别，比如['轻度', '重度']，或者，更好的方式是用一个精心排序的条形图来展示完整的分布[@problem_id:4993204] [@problem_id:4838806]。不变性原则不仅告诉我们该计算什么，还告诉我们该如何观察。

### 构建和检验模型：从数据到发现

当我们从描述数据转向构建模型和检验假设时，[测量理论](@entry_id:153616)的作用变得更加关键。这些是科学发现的引擎，如果我们给它们加了不合适的燃料，它们就可能熄火失灵。

考虑一个现代数据科学中普遍存在的问题：为机器学习模型预处理数据[@problem_id:5194300]。如果我们有一个像血型（`A`, `B`, `AB`, `O`）这样的名义变量，将其编码为`1, 2, 3, 4`是一个灾难性的错误。这强加了错误的顺序和距离，如果我们只是简单地置换标签，我们模型的预测就可能发生巨大变化！不变性原则要求一种将每个类别视为独特但无序的编码方式。这正是**[独热编码](@entry_id:170007)**（或[虚拟变量](@entry_id:138900)创建）所做的事情。它创造了一个每个类别都是其自身维度的空间，从而尊重了数据的名义性质[@problem_id:4783287]。

那么[缺失数据](@entry_id:271026)呢？你可能想通过取观测分数的平均值来填补缺失的序数疼痛评分。这看起来很合理，但这是一个微妙的陷阱。让我们看看为什么。想象一下，我们把疼痛级别编码为`1, 2, 3, 4, 5`。平均值可能是`3.0`。现在，如果我们使用另一个完全有效的[序数](@entry_id:150084)编码`1, 4, 9, 16, 25`（因为对于正数，$y=x^2$是保序的）会怎样？这些新编码值的平均值将*不*是$3^2=9$。它会是某个其他的数，当我们把它映射回去时，会得到一个不同的插补值。用一些示例[数字计算](@entry_id:186530)表明，在第一种编码下的插补值可能是`3.0`，但在第二种编码下它可能是`3.19`！[@problem_id:4838882]。我们的结果依赖于一个任意的选择，这意味着这个过程是无意义的。这就是为什么对于[序数数据](@entry_id:163976)，用**[中位数](@entry_id:264877)**进行[插补](@entry_id:270805)或使用像**热卡填充法**（使用一个实际观测到的值）这样的方法是站得住脚的——它们的结果对于我们如何标记类别是不变的。

假设检验的选择也取决于这些原则。我们常被告知，在参数检验（如**[配对t检验](@entry_id:169070)**）和[非参数检验](@entry_id:176711)（如**[Wilcoxon符号秩检验](@entry_id:168040)**）之间的选择是关于[正态性假设](@entry_id:170614)的。这没错，但有更深层的原因。t检验依赖于计算平均差。这个操作只对区间或比率数据有意义。另一方面，[Wilcoxon检验](@entry_id:172291)是基于差值的秩。事实证明，这个排序过程在区间尺度的仿射变换（$y=ax+b$）下是不变的。所以，如果我们有一个真正的区间测量，比如一个经过心理计量模型仔细验证以确保具有相等区间的疼痛评分，那么[Wilcoxon检验](@entry_id:172291)是一个完全有效且稳健的选择[@problem_id:4838786]。理论引导我们为工作选择正确的工具。

### 最高追求：量化因果效应

[测量理论](@entry_id:153616)最深刻的应用或许出现在我们提出终极科学问题时：我们的干预是否*导致*了某个效应？“因果效应”的定义本身就取决于我们结果的测量尺度。

考虑一个药物试验，其结果是在一个区间尺度上测量的，比如一个具有任意零点的症状评分。我们应该如何衡量效应？我们可以看均值之差，$E[Y(1) - Y(0)]$，或者均值之比，$E[Y(1)] / E[Y(0)]$。让我们检查一下不变性。如果我们移动尺度的原点（一个容许的变换，$y=x+b$），均值之差保持不变：$(E[Y(1)]+b) - (E[Y(0)]+b) = E[Y(1)] - E[Y(0)]$。结论是稳定的。但均值之比改变了：$(E[Y(1)]+b) / (E[Y(0)]+b) \neq E[Y(1)] / E[Y(0)]$。一种药物在一个尺度上看可能是有益的，而在另一个尺度上看可能是有害的！因此，对于区间数据，**加性因果效应**（差值）是有意义的，而乘性效应（比率）则没有。

现在，考虑一个比率尺度的结果，比如血液中生物标志物的浓度，它有一个真零点。如果我们改变单位（一个容许的变换，$y=ax$），均值之差会被乘以$a$，但均值之比是完全不变的：$(a E[Y(1)]) / (a E[Y(0)]) = E[Y(1)] / E[Y(0)]$。在这里，**乘性因果效应**是那个稳健、有意义的量[@problem_id:4838912]。这种美丽的二元性无处不在。对于事件发生时间数据（一种比率尺度），生存分析中使用的**风险比**之所以强大，正是因为它对于时间单位的变化（例如，从天到周）是不变的[@problem_id:4838920]。

### 一个普适原则

从医生的办公室到数据科学家的电脑，容许变换原则是一个普适的指南。它告诉我们，来自MRI机器的高科技测量值，如表观扩散系数（$ADC$）或容积转移常数（$K^{trans}$），几乎总是比率尺度变量，因为它们基于具有绝对零点——无扩散、无转移——的物理过程。这立即告诉我们，计算这些生物标志物的比率和百分比变化是一种有意义的科学行为[@problem_id:4566412]。

最终，这个听起来简单的想法——我们的结论不应依赖于我们的标签——是物理学中最深刻真理之一的回响。自然的基本定律不依赖于我们用来描述它们的任意坐标系。同样，我们从数据中得出的结论必须超越我们设计的任意测量系统。寻找在容许变换下不变的统计量、模型和可视化方法，其本身就是一种对更客观真理的探寻。这不仅仅是为了避免错误；这是为了努力通过更清晰的镜头看世界。
