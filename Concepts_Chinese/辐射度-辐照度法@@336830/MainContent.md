## 引言
房间里的物体不接触是如何交换热量的？这场由[热辐射](@article_id:305527)承载的无形对话，是一场复杂的能量之舞。精确预测封闭腔内每个表面的净换热量是热工领域的一项基本挑战。其难点在于需要考虑无数次的反射和吸收，这些过程将每个表面的命运与其所有相邻表面耦合在一起。本文通过介绍[辐射度](@article_id:316940)-[辐照度](@article_id:355434)法来揭示这一过程的奥秘，这是一个功能强大的框架，它为看似混乱的问题带来了优雅的简洁性。

在接下来的章节中，我们将从头开始构建这个框架。在“原理与机制”一章中，我们将定义[辐射度](@article_id:316940)和[辐照度](@article_id:355434)的核心概念，推导控制方程，并揭示著名的[电网络类比](@article_id:336914)法，该方法将问题转化为直观的[电路分析](@article_id:335949)。随后，在“应用与跨学科联系”一章中，我们将看到该模型的实际应用，探索工程师如何利用它来设计从航天器绝热到工业熔炉的各种系统，以及它如何与[热力学](@article_id:359663)基本定律深度联系。

*图1：一个三表面封闭腔的[电网络类比](@article_id:336914)。每个表面的温度设定了一个源电压 $E_b$。表面自身的属性产生一个表面[热阻](@article_id:304530)，将源连接到主[辐射度](@article_id:316940)节点 $J$。封闭腔的几何形状产生了连接各[辐射度](@article_id:316940)节点的空间[热阻](@article_id:304530)。从一个表面传出的净热量 $Q$ 就是流经其表面热阻的电流。*

## 原理与机制

想象一下，你身处一个墙壁被涂成不同颜色、有些温暖有些寒冷的房间里。即使没有触摸，你也能感觉到热炉辐射出的温暖，也能感觉到冷窗玻璃带来的寒意。这场由光承载的无形对话，就是辐射换热的主题。我们的目标是精通这种光的语言，预测每个表面在这场持续的交换中获得或失去多少能量。为此，我们需要理解我们故事中两个主角的角色：**[辐照度](@article_id:355434) (Irradiation)** 和 **[辐射度](@article_id:316940) (Radiosity)**。

### 辐射能量的两个方面

想一下房间里的任何一个表面。它同时受到来自周围环境的辐射轰击，也在发出自身的辐射。我们需要一种方法来量化这两个过程。

首先，让我们考虑入射的能量。我们将投射到单位面积表面上的总[辐射功率](@article_id:330890)称为**[辐照度](@article_id:355434)**，用符号 $G$ 表示。它是表面*接收*到的所有能量的度量。如果你能站在表面的一小块区域上，向外望向周围的半球空间，那么[辐照度](@article_id:355434)就是从所有可能方向到达的光能的总和 [@problem_id:2519539]。

接下来，我们需要描述表面*发出*的能量。这就是它的**[辐射度](@article_id:316940)**，符号为 $J$。这部分出射的能量有两个组成部分。第一部分是表面自身的热发光，即其**发射**。第二部分是其反射的入射[辐照度](@article_id:355434)的一部分。因此，[辐射度](@article_id:316940)是发射和反射的总和。它是表面*返还*给封闭腔的所有能量。

一个表面失去的净能量就是它发出的能量与接收的能量之差。因此，离开面积为 $A$ 的表面的净热流率 $Q$ 由一个优美简洁的关系式给出：

$$Q = A (J - G)$$

我们的巨大挑战是找出封闭腔内每个表面的 $J$ 和 $G$ 值。

### 控制方程：关于表面和几何的故事

为了求解我们的未知数，我们需要建立游戏规则。这些规则以两个连接[辐射度](@article_id:316940)和[辐照度](@article_id:355434)的基本方程形式出现。

#### 表面方程：一个表面的身份

让我们更仔细地研究[辐射度](@article_id:316940) $J$。正如我们所说，它是发射和反射的总和。

$$J = (\text{Emitted Flux}) + (\text{Reflected Flux})$$

发射部分取决于表面的温度 $T$ 和一个称为**[发射率](@article_id:303723)**的属性 $\varepsilon$。[发射率](@article_id:303723)是一个介于0和1之间的数字，它告诉我们与理想的理论辐射体——**黑体**相比，该表面的辐射能力如何。一个理想黑体的发射率为 $\varepsilon=1$，而一个理想反射体的[发射率](@article_id:303723)为 $\varepsilon=0$。黑体的总发射功率由著名的 Stefan-Boltzmann 定律给出，$E_b = \sigma T^4$。因此，一个真实表面的发射通量为 $\varepsilon E_b$。

反射部分是入射[辐照度](@article_id:355434) $G$ 中被反射的比例。这个比例就是**[反射率](@article_id:323293)** $\rho$。因此，反射通量为 $\rho G$。

综上所述，一个表面的[辐射度](@article_id:316940)为：

$$J = \varepsilon E_b + \rho G$$

现在，我们引入一个至关重要的简化：**灰体表面假设**。我们假设辐射属性，如发射率和[反射率](@article_id:323293)，在所有波长（颜色）的光中都是恒定的。对于一个不透明的灰体表面，对入射辐射的简单[能量平衡](@article_id:311249)告诉我们，被吸收的部分（$\alpha$，[吸收率](@article_id:304948)）和被反射的部分必须总和为一：$\alpha + \rho = 1$。此外，Kirchhoff [热辐射](@article_id:305527)定律告诉我们，对于一个灰体表面，[吸收率](@article_id:304948)等于发射率，$\alpha = \varepsilon$ [@problem_id:2519577]。这导出了一个非常简洁的关系：$\rho = 1 - \varepsilon$。

将此代入我们的[辐射度](@article_id:316940)方程，得到我们的第一个基本关系式，即表面方程 [@problem_id:2498972]：

$$J_i = \varepsilon_i E_{b,i} + (1 - \varepsilon_i) G_i$$

这个方程是关于表面 $i$ 的身份的陈述。它将其出射[辐射度](@article_id:316940) $J_i$ 与其固有属性（$\varepsilon_i$）、其温度（$E_{b,i}$）以及投射到其上的辐射（$G_i$）联系起来。

#### 封闭腔方程：几何的对话

现在来看谜题的第二部分。[辐照度](@article_id:355434) $G_i$ 从何而来？它来自封闭腔中所有表面的[辐射度](@article_id:316940)，包括表面 $i$ 本身！离开任何表面 $j$ 的辐射向外传播，其中一部分会落在表面 $i$ 上。

这就是几何学登场的时刻。我们定义一个纯粹的几何量，称为**[角系数](@article_id:310017)**，$F_{ij}$。它代表离开表面 $i$ 的总辐射中直接投射到表面 $j$ 的部分。它仅取决于两个表面的大小、形状、距离和方向。这就像在问：“从表面 $i$ 的角度看，表面 $j$ 占据了其视野的多大比例？”

在所有表面都是**漫射的**——意味着它们向所有方向均匀地辐射和反射光线，像一张哑光纸而不是镜子——的假设下，我们可以写出一个简单而强大的表达式。表面 $i$ 上的[辐照度](@article_id:355434)是封闭腔中所有表面贡献的总和。来自表面 $j$ 的贡献是其[辐射度](@article_id:316940) $J_j$ 乘以从 $i$ 到 $j$ 的[角系数](@article_id:310017) $F_{ij}$。对所有 $N$ 个表面求和，得到我们的第二个基本关系式 [@problem_id:2519539]：

$$G_i = \sum_{j=1}^{N} F_{ij} J_j$$

这就是封闭腔方程。它表明，每个表面的辐射状态都通过几何[角系数](@article_id:310017)与其他每个表面耦合在一起。不了解所有相邻表面的情况，就无法确定一个表面的命运。

### 神来之笔：[辐射网络类比法](@article_id:316824)

我们现在有了一个针对所有[辐射度](@article_id:316940)的[线性方程组](@article_id:309362)。对于一个简单的封闭腔，我们可以用代数方法求解它们。但随着表面数量的增加，这将成为一项艰巨的任务。在这里，物理学家和工程师们像变戏法一样，揭示了两个看似无关的领域——传热学和电学——之间深刻而优美的统一性。他们意识到，控制[辐射交换](@article_id:310940)的方程与[直流电路](@article_id:324934)的方程形式完全相同 [@problem_id:2517077] [@problem_id:2519541]。

这种**网络类比法**使我们能够将辐射问题“绘制”成电路图，并使用所有简单、直观的[电路分析](@article_id:335949)工具来解决它。对应关系如下：

*   **黑体发射功率** $E_{b,i} = \sigma T_i^4$ 完全由表面温度决定，其作用类似于**电池或电源的电压**。它是辐射的驱动势。
*   **[辐射度](@article_id:316940)** $J_i$ 的作用类似于电路中**主节点的电势（电压）**。
*   离开表面的**净辐射热流率** $Q_i$ 相当于流入网络的、从该表面流出的**电流**。

通过这些对应关系，两个基本方程转变为对电路元件——电阻——的描述。

#### 表面[热阻](@article_id:304530)

我们通过将 $J_i = \varepsilon_i E_{b,i} + (1 - \varepsilon_i) G_i$ 与 $Q_i = A_i(J_i - G_i)$ 结合来重新整理我们的表面方程。经过一番代数运算，我们发现：

$$Q_i = \frac{E_{b,i} - J_i}{(1 - \varepsilon_i) / (\varepsilon_i A_i)}$$

这看起来和欧姆定律 $I = \Delta V / R$ 完全一样！电流 $Q_i$ 由“源电压”$E_{b,i}$ 和“[节点电压](@article_id:639058)”$J_i$ 之间的电势差驱动。分母中的项是一个电阻，我们称之为**表面热阻**：

$$R_{s,i} = \frac{1 - \varepsilon_i}{\varepsilon_i A_i}$$

这个电阻具有奇妙的物理意义。它代表了表面将其内部热能以辐射形式释放出去所受到的阻力。请注意，如果一个表面是理想黑体（$\varepsilon_i = 1$），其表面热阻为零 [@problem_id:2519528]。这意味着节点电势 $J_i$ 直接连接到源电势 $E_{b,i}$，因此 $J_i = E_{b,i}$。这完全合乎情理：黑体的[能量流](@article_id:303208)出仅取决于其自身温度，而不取决于任何反射的能量（因为它不反射任何能量）。如果一个表面的发射能力很差（$\varepsilon_i$ 很小），其表面[热阻](@article_id:304530)会非常大，表示热量离开表面有很大的“阻抗”。

#### 空间[热阻](@article_id:304530)

现在考虑两个表面 $i$ 和 $j$ 之间的净能量交换。它是离开 $i$ 并击中 $j$ 的能量减去离开 $j$ 并击中 $i$ 的能量。使用[角系数](@article_id:310017)，这个净交换率 $Q_{i \leftrightarrow j}$ 可以表示为：

$$Q_{i \leftrightarrow j} = A_i F_{ij} (J_i - J_j) = \frac{J_i - J_j}{1 / (A_i F_{ij})}$$

这又一次是欧姆定律！它描述了在电势为 $J_i$ 和 $J_j$ 的两个节点 $i$ 和 $j$ 之间流动的电流。连接它们的电阻就是**空间热阻**：

$$R_{ij} = \frac{1}{A_i F_{ij}}$$

这个电阻代表了[辐射交换](@article_id:310940)的几何障碍。如果表面很大（$A_i$ 大）并且它们彼此有很好的视角（$F_{ij}$ 大），则空间[热阻](@article_id:304530)很小，它们可以轻松地[交换能](@article_id:297520)量。如果它们相距很远或被[遮挡](@article_id:370461)（$F_{ij}$ 小），则电阻很大。
