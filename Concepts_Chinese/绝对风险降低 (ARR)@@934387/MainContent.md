## 引言
当一种新疗法声称能“将风险降低50%”时，这对患者来说到底意味着什么？医学统计的世界常常充满了听起来令人印象深刻的百分比，但这可能会掩盖干预措施在现实世界中的真实影响。这种模糊性为希望做出明智决策的临床医生和患者创造了一个关键的知识鸿沟。为了驾驭这一领域，我们需要一种更清晰、更真实的风险语言。

本文介绍了绝对风险降低（ARR），这是一个强大而简单的指标，能够透明地展示治疗效果。您将学习区分 ARR 与更常见的相对风险降低（RRR）的基本原则，并探索它如何引出高度直观的需治数（NNT）。然后，我们将看到这些概念如何应用于不同领域，从指导医学和心理学中的个体化患者护理到塑造大规模的公共卫生策略，展示 ARR 如何构成现代循证实践的基石。

## 原理与机制

想象一下，你是一名医生，一种新药被研发出来了。制药公司告诉你，它“将心脏病发作的风险降低了50%！”这听起来非常了不起。但对于坐在你面前的病人来说，这到底*意味着*什么？是说他们心脏病发作的风险现在是以前的一半了吗？还是意味着别的什么？为了驾驭这个充满医学证据的世界，我们需要的不仅仅是华丽的百分比。我们需要一种清晰、定量地思考风险的方式，一种既真实又直观的方式。这就是绝对风险降低的故事。

### 风险降低的两个方面

让我们从最简单的概念开始：**风险**。在医学中，风险只是某个事件——如手术感染或心脏病发作——在特定时期内发生的概率。我们将没有任何新治疗时的风险称为**基线风险**，或 $p_{\text{control}}$。现在，我们引入一项干预措施——一种新药或新疗法——它将风险改变为一个新值 $p_{\text{intervention}}$。

我们如何衡量这项干预措施的效果？有两种根本不同的方式来看待这种变化，而它们之间的区别是现代医学中最重要的理念之一。

第一种方法是简单地用旧风险减去新风险。这就得到了**绝对风险降低（ARR）**。

$$ARR = p_{\text{control}} - p_{\text{intervention}}$$

这个数字非常简单和真实。它告诉你事件发生几率在现实世界中的实际降低量。例如，在一项研究结直肠手术后手术部位感染的试验中，接受标准护理组的风险为 $0.15$（即每100名患者中有15人），而接受新干预措施组的风险为 $0.08$（每100名患者中有8人）[@problem_id:5191626]。ARR 就是简单的 $0.15 - 0.08 = 0.07$。这意味着每100名接受新疗法的患者中，我们预计会减少7例感染。这没有任何模糊之处。它是对治疗在人群中影响的直接衡量。

第二种看待变化的方式是相对尺度。我们可以问：“原始风险降低了多少比例？”这就得到了**相对风险降低（RRR）**。

$$RRR = \frac{p_{\text{control}} - p_{\text{intervention}}}{p_{\text{control}}} = \frac{ARR}{p_{\text{control}}}$$

在我们的手术示例中，RRR 为 $\frac{0.07}{0.15} \approx 0.47$，即降低了 $47\%$。这就是“风险降低近一半！”这类新闻标题的来源。这听起来令人印象深刻，而且确实是一个显著的相对效应。然而，当只呈现相对指标而不提供其绝对对应值时，就会出现一个重大的伦理问题 [@problem_id:4868946]。将百万分之一的风险降低50%与将四分之一的风险降低50%是截然不同的。RRR 的本质决定了它会隐藏基线风险，并可能使一个绝对获益非常小的治疗看起来效果巨大。为了让患者做出真正知情的决定，他们必须了解获益的绝对大小，而为此，ARR 是不可或缺的。

### 从抽象到行动：需治数

ARR 是一个强大的概念，但我们可以让它变得更直观。如果一种治疗每治疗100人就能预防7个事件，那么我们需要治疗多少人才能预防*一个*事件呢？稍作算术就能得出答案：如果在100人中预防了7个事件，那么预防1个事件就需要 $\frac{100}{7} \approx 14.3$ 人。这就引出了**需治数（NNT）**。

NNT 就是 ARR 的倒数：

$$NNT = \frac{1}{ARR}$$

对于我们的手术示例，$NNT = \frac{1}{0.07} \approx 14.3$。按照惯例，我们将其向上取整到下一个整数，所以 NNT 是 $15$。这意味着，平均而言，我们需要用新的干预措施治疗15名患者，才能预防一例本会发生的手术部位感染 [@problem_id:5191626]。NNT 将一个抽象的概率转化为了一个对临床决策非常有用的具体数字。

NNT 在可能性边界上的表现也非常符合逻辑 [@problem_id:4615175]。想象一种完美的治疗方法，可以治愈一种否则总是致命的疾病。在这里，$p_{\text{control}} = 1$，$p_{\text{intervention}} = 0$。$ARR = 1 - 0 = 1$。$NNT = \frac{1}{1} = 1$。你治疗一个人，就挽救一条生命。这是完美的。现在，想象一种获益极小的治疗，其 $ARR$ 趋近于零。NNT 作为 $1/ARR$，将趋近于无穷大。你将不得不治疗近乎无穷多的人才能预防一个事件。如果治疗完全没有效果，$ARR=0$ 呢？除以零是未定义的，NNT 也是如此。如果治疗实际上预防了零个事件，那么计算需要治疗多少人来预防一个事件是不可能的！数学完美地反映了现实 [@problem_id:4621604]。

### 基线的“专制”：为何背景决定一切

故事到这里变得更加有趣了。想象一种药物，已知其具有一致的*相对*效应——它总是将事件风险降低，比如说，30%。这种药物对每个人都同样有用吗？答案是响亮的“不”，理解其中原因是个体化医疗的关键。

绝对风险降低（ARR）以及因此产生的需治数（NNT），并不仅仅是药物本身的固定属性；它是药物*和*患者基线风险共同的属性。让我们用一个例子来探讨这一点，其中一种治疗的相对风险（RR）恒为 $0.7$（意味着30%的RRR）[@problem_id:4828674]。

考虑一位高风险患者，其事件的基线风险 $p_0$ 为 $0.20$。
治疗后的风险为 $p_1 = RR \cdot p_0 = 0.7 \cdot 0.20 = 0.14$。
$ARR = p_0 - p_1 = 0.20 - 0.14 = 0.06$。
$NNT = \frac{1}{0.06} \approx 17$。

现在考虑一位低风险患者，其基线风险 $p_0$ 为 $0.05$。
治疗后的风险为 $p_1 = 0.7 \cdot 0.05 = 0.035$。
$ARR = 0.05 - 0.035 = 0.015$。
$NNT = \frac{1}{0.015} \approx 67$。

这是一个深刻的结果。同一种药物，用于高风险患者时，“效率”是用于低风险患者时的四倍（NNT为17对67）。具有恒定相对效应的干预措施，在基线风险较高的人群中总会有更大的绝对效应。这一原则在两项主要的心力衰竭试验中得到了优雅的证明 [@problem_id:4916995]。两种不同的β-受体阻滞剂药物显示出几乎相同的相对获益（风险比分别为 $0.65$ 和 $0.66$）。然而，一项试验招募了病情更重的患者，其基线死亡率为 $24\%$，得出的 NNT 约为 $13$。另一项试验招募了更健康的患者，基线死亡率为 $12\%$，结果 NNT 约为 $26$。几乎相同的相对效应掩盖了绝对获益上的两倍差异，这一差异完全由被治疗患者的基线风险解释。

### 平均值的陷阱：如何正确思考人群

当我们治疗一个包含高风险和低风险个体的混合人群时会发生什么？似乎很自然地会去计算每个亚组 NNT 的加权平均值。但这是一个严重的错误。

NNT 是一个倒数尺度（$1/ARR$），它是非线性的。你不能对非线性量求平均并期望得到一个有意义的结果。真正线性并且可以平均的量是 ARR 本身。

让我们想象一个由 $25\%$ 高风险和 $75\%$ 低风险个体组成的人群 [@problem_id:4985575]。高风险组的 $ARR_{hi} = 0.05$（得出 $NNT_{hi} = 20$），低风险组的 $ARR_{lo} = 0.0125$（得出 $NNT_{lo} = 80$）。

错误的方法是平均NNT：$(0.25 \times 20) + (0.75 \times 80) = 5 + 60 = 65$。

*正确*的方法是首先计算整个人群的平均 ARR：
$ARR_{pop} = (0.25 \times ARR_{hi}) + (0.75 \times ARR_{lo}) = (0.25 \times 0.05) + (0.75 \times 0.0125) = 0.021875$。

现在，我们可以计算出真正的人群 NNT：
$NNT_{pop} = \frac{1}{ARR_{pop}} = \frac{1}{0.021875} \approx 46$。

该人群的真实 NNT 是46，而不是65！错误的平均方法给出了一个误导性的高数字，因为它没有正确地考虑到治疗在高风险亚组中更大的绝对影响。唯一能正确思考的方法是总是回到基本的、线性的量：绝对风险降低。

### 硬币的另一面：需伤数

到目前为止，我们只讨论了治疗的益处。但是，每一次干预，从最简单的药片到最复杂的手术，都带有伤害的风险。我们逻辑的美妙对称性在于它同样完美地适用于不良事件。

如果一种治疗增加了不良事件的风险，我们可以计算**绝对风险增加（ARI）**。
$ARI = p_{\text{harm, intervention}} - p_{\text{harm, control}}$。

由此，我们可以定义**需伤数（NNH）**，它告诉我们平均需要治疗多少人，才会额外发生一例不良事件。

$$NNH = \frac{1}{ARI}$$

例如，如果一种心血管疗法在五年内将大出血事件的风险从 $0.02$ 增加到 $0.03$，则 $ARI = 0.03 - 0.02 = 0.01$。$NNH = \frac{1}{0.01} = 100$ [@problem_id:4985575]。

现在我们有了全貌。医生和患者可以权衡NNT和NNH。“对于这种治疗，我们预计治疗大约20个像您这样的病人可以预防一次心脏病发作，同时我们预计治疗大约100人会有一人经历一次大出血。”这个框架将关于“利弊”的模糊讨论转变为透明的、定量的比较。这是现代、循证和伦理的医疗实践的语言——一种完全建立在绝对风险这个简单、强大而真实理念之上的语言。
