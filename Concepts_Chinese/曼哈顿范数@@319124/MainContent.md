## 引言
当我们想到距离时，我们本能地会想象一条直线——两点之间最短的路径。这个概念被形式化为[欧几里得范数](@article_id:640410)，它主导着我们的物理直觉，并支撑了几个世纪的几何学。但是，如果世界并非总是能走直线呢？如果移动被限制在一个网格上，就像出租车在城市街道上穿行一样，情况又会如何？这个简单的限制催生了一种不同但同样强大的测量距离的方式：[曼哈顿范数](@article_id:313638)。本文将深入探讨这一迷人的度量标准，揭示一个拥有自身独特规则和惊人力量的几何世界。我们将弥合我们对距离的直观理解与现代科学的实际需求之间的知识鸿沟，在现代科学中，基于网格的逻辑和对简洁性的追求至关重要。

旅程始于第一章**“原理与机制”**，我们将解构[曼哈顿范数](@article_id:313638)，探索其不寻常的几何特性，如方形的圆，并理解为什么像毕达哥拉斯定理这样的概念不再成立。随后，**“应用与跨学科联系”**一章将展示这个看似抽象的概念如何在从[量子计算](@article_id:303150)和生物学到机器学习的高维挑战等广泛学科中提供深刻的见解和实用的解决方案。

## 原理与机制

我们已经了解了这种奇特的测量距离的新方法——**[曼哈顿范数](@article_id:313638)**。但它究竟是什么？它如何改变我们对世界的看法？如果你一生都认为距离是尺子画出的直线——我们称之为**[欧几里得范数](@article_id:640410)**——那么踏入[曼哈顿范数](@article_id:313638)的世界就像访问另一个星球上的城市，一个有着自己奇特但又严格一致的几何规则的城市。让我们在这个城市里漫步，发现它的规则。

### 一种不同的尺子：出租车世界

想象一下，你身处一个像曼哈顿一样布局在完美网格上的城市。你想从 A 点到 B 点。你不能像乌鸦一样直线飞越建筑物。你必须沿着街道，一个街区一个街区地移动，要么向南向北，要么向东向西。你走过的总距离是水平距离和垂直距离的总和。

这正是[曼哈顿距离](@article_id:340687)背后的思想，数学家称之为 $L_1$ 距离。对于两个点 $P_1 = (x_1, y_1)$ 和 $P_2 = (x_2, y_2)$，距离不再是我们熟悉的 $\sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}$。而是：

$$
d_1(P_1, P_2) = |x_1 - x_2| + |y_1 - y_2|
$$

它是它们坐标的绝对差之和。这不仅仅是一个古怪的思想实验。想象一下，一条装配线上的机械臂沿着一组固定的垂直轨道移动。要从位于 $A = (2.5, -4.0, 8.1)$ 的物料箱移动到位于 $B = (10.0, 1.5, 3.6)$ 的工位，它必须首先沿 x 轴移动，然后沿 y 轴，再沿 z 轴。其总行程距离就是[曼哈顿距离](@article_id:340687)：$|10.0 - 2.5| + |1.5 - (-4.0)| + |3.6 - 8.1| = 7.5 + 5.5 + 4.5 = 17.5$ 厘米 [@problem_id:2225312]。在任何移动受限于网格的系统中，[曼哈顿范数](@article_id:313638)不仅是一种替代方案；它是最自然、最直接的测量行程的方式。

### “邻近”的形状：方形的圆

我们的直觉从这里开始扭曲。在这个出租车世界里，“圆”是什么样子的？圆被定义为与一个中心点等距的所有点的集合。让我们画一个“[单位圆](@article_id:311954)”——所有与原点 $(0,0)$ 距离恰好为 1 的点。

用我们的欧几里得尺子，我们得到 $\|v\|_2 = \sqrt{x^2 + y^2} = 1$，这就是我们都熟悉并喜爱的圆形。

但用我们的出租车尺子，方程是 $\|v\|_1 = |x| + |y| = 1$。这个方程描述了什么形状？在第一象限，x 和 y 都是正数，方程是 $x+y=1$，这是一条连接 $(1,0)$ 和 $(0,1)$ 的直线。如果你为所有四个[象限](@article_id:352519)都画出这条线，你会得到一个旋转了 45 度的正方形，其顶点在 $(1,0)$、$(0,1)$、$(-1,0)$ 和 $(0,-1)$。

这是一个深刻的发现！在曼哈顿世界里，圆是正方形。

现在，让我们比较一下“单位球”——[单位圆](@article_id:311954)内所有点的集合。设 $B_2$ 是我们熟悉的圆形圆盘（$\|v\|_2 < 1$），$B_1$ 是出租车“圆盘”（$\|v\|_1 < 1$），也就是我们那个倾斜的正方形。如果你把它们叠在一起画，你会看到菱形的 $B_1$ 完全包含在圆形的 $B_2$ 内 [@problem_id:1873293]。

这不仅仅是一个视觉上的戏法；它是一个基本不等式的结果：对于任何向量 $v$，$\|v\|_2 \le \|v\|_1$。[欧几里得距离](@article_id:304420)永远不会超过出租车距离。想想看：两点之间的[最短路径](@article_id:317973)是直线。出租车被迫走一条更长的、曲折的路径。

但是否有在欧几里得圆内却在出租车圆外的点呢？当然有。考虑点 $(0.7, 0.7)$。它的[欧几里得距离](@article_id:304420)是 $\sqrt{0.7^2 + 0.7^2} = \sqrt{0.98}$，小于 1。所以它在圆形圆盘 $B_2$ 内。但它的出租车距离是 $|0.7| + |0.7| = 1.4$，大于 1，这使它位于菱形圆盘 $B_1$ 之外 [@problem_id:1896468]。这些点位于两个“圆”边界之间的四个新月形区域中。

### 尺子何时达成一致？一个方向问题

既然这两种范数对大多数向量给出不同的值，一个有趣的问题就出现了：它们何时一致？“乌鸦飞行”的距离何时与出租车的路线相同？让我们找出所有非零向量 $x = (x_1, x_2)$，使得 $\|x\|_2 = \|x\|_1$。

$$
\sqrt{x_1^2 + x_2^2} = |x_1| + |x_2|
$$

两边平方给我们一个美丽的惊喜：

$$
x_1^2 + x_2^2 = (|x_1| + |x_2|)^2 = |x_1|^2 + 2|x_1||x_2| + |x_2|^2 = x_1^2 + 2|x_1||x_2| + x_2^2
$$

从两边减去 $x_1^2 + x_2^2$，我们剩下 $2|x_1||x_2| = 0$。这个简单的方程告诉我们一切。要使其为真，要么 $x_1=0$，要么 $x_2=0$。

这意味着这两种范数仅在向量完全位于坐标轴上时才相等！[@problem_id:2308545]。一旦你沿对角线方向移动，[欧几里得距离](@article_id:304420)就变得严格短于[曼哈顿距离](@article_id:340687)。这揭示了一个深刻的真理：[曼哈顿范数](@article_id:313638)有内在的“偏好方向”。它偏爱沿着网格本身的移动。而[欧几里得范数](@article_id:640410)是各向同性的——它平等地对待所有方向。

### 一个没有毕达哥拉斯的世界

坐标轴的特殊地位暗示着一个更深层次的结构差异。在欧几里得几何中，[毕达哥拉斯定理](@article_id:351446)是神圣的。对于任何两个正交（垂直）向量 $u$ 和 $v$，我们有 $\|u\|_2^2 + \|v\|_2^2 = \|u+v\|_2^2$。这是[点积](@article_id:309438)的几何灵魂，也是我们对“角度”概念的定义。

这在出租车世界中成立吗？我们来检验一下。取最简单的[正交向量](@article_id:302666)：[标准基向量](@article_id:312830) $u = (1,0)$ 和 $v = (0,1)$。

- $\|u\|_1 = |1| + |0| = 1$
- $\|v\|_1 = |0| + |1| = 1$
- $u+v = (1,1)$，所以 $\|u+v\|_1 = |1| + |1| = 2$

现在我们来检验[毕达哥拉斯定理](@article_id:351446)：$\|u\|_1^2 + \|v\|_1^2 = \|u+v\|_1^2$ 是否成立？
$1^2 + 1^2 = 2$。
而在另一边，我们有 $2^2 = 4$。
$2 \ne 4$。该定理不成立！[@problem_id:1897291] [@problem_id:1898393]。

这不仅仅是一个小小的奇闻。它表明[曼哈顿范数](@article_id:313638)并非来自**内积**（如[点积](@article_id:309438)）。[毕达哥拉斯定理](@article_id:351446)的一个更普遍的版本是**平行四边形法则**：对于任何两个向量 $u$ 和 $v$，$\|u+v\|^2 + \|u-v\|^2 = 2(\|u\|^2 + \|v\|^2)$。当且仅当范数是由内积导出时，此法则才成立。对于我们的[出租车范数](@article_id:303471)和 $u=(1,0)$、$v=(0,1)$，左边是 $\|(1,1)\|_1^2 + \|(1,-1)\|_1^2 = 2^2 + 2^2 = 8$，而右边是 $2(1^2+1^2) = 4$。该法则不成立，证实了出租车世界的几何学缺乏我们从内积中得到的丰富的角度和投影结构。

### 网格的束缚：没有旋转自由

对坐标轴的偏好还有另一个惊人的后果。在我们的欧几里得世界中，距离在旋转下是不变的。如果你取两点，测量距离，然后旋转整个平面，变换后两点之间的距离是相同的。旋转是一种“[刚性运动](@article_id:349714)”或**等距变换**。

在出租车世界中，旋转是等距变换吗？让我们取两点，$P = (\sqrt{2}, 0)$ 和 $Q = (0, \sqrt{2})$。出租车距离是 $d_1(P,Q) = |\sqrt{2}-0| + |0-\sqrt{2}| = 2\sqrt{2}$。

现在，让我们将整个城市逆时针旋转 45 度。点 $P$ 移动到 $T(P) = (1,1)$，点 $Q$ 移动到 $T(Q) = (-1,1)$。新的出租车距离是多少？是 $d_1(T(P), T(Q)) = |1 - (-1)| + |1 - 1| = 2$。距离从 $2\sqrt{2}$ 变成了 $2$！[@problem_id:1662741]。

旋转网格改变了距离的根本结构。一条原本高效的路径可能变得低效，反之亦然。网格不仅仅是一个[坐标系](@article_id:316753)；它是一个决定几何的绝对结构。

### 相同的邻域，不同的视角

有了所有这些奇怪的差异，人们可能会认为欧几里得世界和曼哈顿世界是完全陌生的。但它们之间有一个微妙而强大的联系。我们之前看到的不等式，对于任意两点 $p_1, p_2$ 可以写成 $d_E(p_1, p_2) \le d_T(p_1, p_2) \le \sqrt{2} d_E(p_1, p_2)$，它告诉我们这两种距离虽然不相同，但总是[相差](@article_id:318112)一个常数因子。

这意味着它们在**拓扑上是等价的**。通俗地说，它们在“邻近”的概念上是一致的。一个在欧几里得意义下收敛到某个极限的点序列，在出租车意义下也会收敛到同一个极限 [@problem_id:1594348]。如果你放大任何一点，一个小的圆形邻域总会包含一个小的菱形邻域，反之亦然。它们描述了相同的“拓扑”，即空间相同的基本连通性，尽管它们测量其几何的方式不同。

一个能很好地说明这种[张力](@article_id:357470)的例子是，用一种尺子定义的形状，再用另一种尺子来测量。考虑标准的、圆形的欧几里得单位圆盘 $C = \{ (x, y) \mid x^2 + y^2 \le 1 \}$。如果我们用[出租车度量](@article_id:301568)，它的直径——其内部任意两点之间的最大可能距离——是多少？答案不是 2（欧几里得直径），而是 $2\sqrt{2}$ [@problem_id:39302]。这个最大的出租车距离是在点 $(\frac{\sqrt{2}}{2}, \frac{\sqrt{2}}{2})$ 和 $(-\frac{\sqrt{2}}{2}, -\frac{\sqrt{2}}{2})$ 之间实现的，这两点位于欧几里得圆的边界上。这个单一的数字 $2\sqrt{2}$，优雅地捕捉了两个世界之间的几何扭曲。

### “尖角”的力量：[稀疏性](@article_id:297245)与诺贝尔奖级的思想

我们为什么要使用这种奇怪的、有尖角的、锁定在网格上的几何学呢？事实证明，[曼哈顿范数](@article_id:313638)的“缺陷”在现代[数据科学](@article_id:300658)和机器学习的世界里却是其最大的优点。

这些领域的许多问题都涉及寻找一个简单的模型来解释复杂的数据。“简单”通常意味着模型中非零参数尽可能少——这一特性称为**[稀疏性](@article_id:297245)**。例如，在预测房价时，我们可能从一百个潜在特征开始，但一个[稀疏模型](@article_id:353316)会发现只有房屋面积、卧室数量和位置是真正重要的，并将所有其他特征的系数设为零。

这就是 $L_1$ [单位球](@article_id:302998)的尖角形状成为英雄的地方。想象一下，试图在一个单位球上找到离某个外部数据点最近的点。如果这个球是完美的圆形欧几里得球，解可能在其光滑表面的任何地方。但如果这个球是尖角的 $L_1$ 菱形，解通常会正好落在它的一个角上！角在哪里？它们在坐标轴上，那里有一个坐标是零。通过在一个 $L_1$ 约束下最小化一个函数，我们鼓励我们的解在许多分量上为零。这个思想，被称为 $L_1$ [正则化](@article_id:300216)或“[Lasso](@article_id:305447)”，是一项突破，为 2021 年诺贝尔经济学奖做出了贡献。

这背后的机制非常迷人。$L_1$ 范数在某个分量为零的点是不可微的。当我们使用像**[次梯度法](@article_id:344132)**这样的[优化算法](@article_id:308254)时，我们可以选择移动的“方向”。在一个像 $(v_1, 0, v_3)$ 这样的点上，[算法](@article_id:331821)可以选择一个将第二个分量推离零的次梯度，或者关键地，选择一个在改进其他分量的同时*保持第二个分量为零*的次梯度 [@problem_id:2207137]。这种“粘”在坐标轴上的能力是稀疏性的引擎。

因此，源于一个简单的城市网格模型的[曼哈顿范数](@article_id:313638)，给了我们一个没有毕达哥拉斯定理或[旋转对称](@article_id:297528)性的几何学。但它的“尖角性”，在经典几何学中是一个缺陷，却成了一个具有巨大威力的特性，让我们能够穿透高维数据的噪音，找到隐藏在其中的简单、稀疏的真理。
