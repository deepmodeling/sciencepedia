## 引言
如果宇宙惊人的复杂性源于少数几条异常简单的规则，那会怎样？这个问题是[细胞自动机](@article_id:328414)研究的核心——这些微型的数字宇宙根据局部互动逐步演化。虽然它们可能看起来像简单的游戏，但某些规则隐藏着深刻的内涵，对我们理解计算和复杂性提出了挑战。本文将聚焦于其中最引人注目的一个例子：规则 110。我们将揭示一个仅由八条微不足道的指令支配的系统，如何能生成不可预测的复杂模式，甚至达到[通用计算](@article_id:339540)的能力。首先，在“原理与机制”部分，我们将深入规则 110 的宇宙，探索其基本定律、类似粒子的“滑翔机”的涌现，以及其[图灵完备](@article_id:335210)性的惊人发现。接着，“应用与跨学科联系”部分将把这个抽象世界与我们自己的世界联系起来，揭示规则 110 的逻辑如何在计算机硬件、物理学、DNA 纳米技术和人工智能等迥然不同的领域中产生回响。

## 原理与机制

想象我们是物理学家，试图揭示一个新发现的微型宇宙的基本定律。这个宇宙极其简单：它只有一行细胞，就像坐标纸带上的方格。每个细胞可以处于两种状态之一——我们称之为“开”（1）和“关”（0），或者黑色和白色。这个宇宙中的时间不是连续流逝的，而是以离散的步长滴答作响。最大的问题是：是什么让一个细胞开启或关闭？

在我们的宇宙中，定律是局部的。一个细胞的命运不是由某个遥远的星球决定的，而仅仅由它的直系亲属决定：它自身以及它的左、右邻居。这三个细胞组成一个“邻域”。由于这三个细胞中的每一个都可以是开或关，因此总共有 $2^3 = 8$ 种可能的邻域模式。一条“规则”就是一份完整的说明书，它为这八种可能性中的每一种，都指定了中心细胞在下一个时间步的状态。

### 创造的密码

我们如何写下这样一条规则？我们可以列一个清单，就像生物学家对[基因相互作用](@article_id:339419)进行分类一样 [@problem_id:1421566]。例如：“如果一个细胞及其邻居都处于‘开’状态（`111`），它将在下一步变为‘关’。”在为所有八个邻域指定结果后，我们就得到了完整的物理定律。

为了高效地传达这些定律，数学家 Stephen Wolfram 设计了一种巧妙的命名方案。首先，我们按标准顺序列出八个邻域，从 `111` 到 `000`。然后，对于每个邻域，我们写下其结果——中心细胞的新状态，即 `1` 或 `0`。这样我们就得到了一个8位的二进制字符串。例如，如果邻域 `111, 110, 101, 100, 011, 010, 001, 000` 的结果分别是 `0, 1, 1, 0, 1, 0, 0, 0`，我们就会得到二进制字符串 `01101000`。这个二进制数的十进制值就是该规则的名称。在这种情况下，$0 \cdot 128 + 1 \cdot 64 + 1 \cdot 32 + 0 \cdot 16 + 1 \cdot 8 + 0 \cdot 4 + 0 \cdot 2 + 0 \cdot 1 = 104$。所以我们称之为“规则 104”。

我们的焦点是一条特别有趣的定律：**规则 110**。它的“源代码”是什么样的？数字 110 的8位二[进制表示](@article_id:641038)是 `01101110`。这一个字符串包含了规则 110 宇宙的全部物理学。让我们把它展开成一个表格，你可以把它看作我们一维世界的基本定律 [@problem_id:1974945]：

| 邻域模式 | 下一状态 |
| :---: | :---: |
| `111` | 0 |
| `110` | 1 |
| `101` | 1 |
| `100` | 0 |
| `011` | 1 |
| `010` | 1 |
| `001` | 1 |
| `000` | 0 |

就是这样。八条简单、确定性的指令。没有随机性，没有歧义。如果你知道现在这行细胞的状态，原则上，你可以计算出它在未来任何时刻的状态。但“原则上”这个词在这里大有文章，我们马上就会看到。

### 黑暗中的火花：复杂性的开端

让我们来做一个实验。我们将从一个完全“关闭”的宇宙开始，即一行寂静的零，只在中间有一个细胞被设为“开”。当我们一遍又一遍地应用规则 110 时，会发生什么？

`t=0: ...000000010000000...`

在第一个时间步，我们检查每个细胞的邻域。那个单独的‘1’被‘0’包围，形成一个 `010` 邻域。规则表告诉我们 `010` $\to$ `1`。它的右邻居是 `100` 邻域的中心，它将变为 `0`。它的左邻居是 `001` 邻域的中心，它将变为 `1`。所以，在下一步，模式变成了：

`t=1: ...000000110000000...`

一个光点变成了两个。让我们继续。再过几个时间步，一件惊人的事情发生了。从那个单一、不起眼的火花中，一个错综复杂的结构绽放开来 [@problem_id:2385572] [@problem_id:1666398]。一个美丽、混沌但又具有模式的 `1` 的三角形在网格上扩展，其清晰的边界向左移动。

这是我们的第一个重要教训：**简单、确定性、局部的规则可以产生极其复杂的全局行为**。规则表中并没有构建一个巨大的、不对称三角形的“总体规划”。这种复杂性是**涌现**的——它是由简单部分的重复相互作用自发产生的。

此外，这种复杂性有一种固执的特性。如果你想知道 1000 步之后图案会是什么样子，你不能只是把 $t=1000$ 代入一个简单的公式。不存在这样的公式。你必须去做这项工作。你必须模拟这个宇宙，一步一步地，就像宇宙自身所做的那样。这个属性被称为**[计算不可约性](@article_id:334547)** [@problem_id:1421579]。系统的演化本身就是计算其未来的最有效方式。在非常真实的意义上，知道答案的唯一方法就是观看故事的展开。

### 新的动物学：滑翔机与万物物理学

当我们观察这个宇宙从更复杂的初始条件演化时，我们注意到一些更了不起的事情。闪烁细胞组成的混沌海洋并不仅仅是[随机噪声](@article_id:382845)。在其中，某些模式似乎有了自己的生命。这些是稳定的、局域化的结构，它们以恒定的速度穿过网格，就像在原始汤中游泳的动物。这些就是规则 110 中著名的**滑翔机**。

就像物理学家不再思考单个的夸克和胶子，而是开始谈论质子和中子一样，我们可以转变我们的视角。我们不再跟踪数十亿个单独的 `0` 和 `1`，而是可以识别这些滑翔机并研究它们的行为 [@problem_id:1666355]。我们可以为规则 110 世界的居民建立一个新的“动物学”。存在一个稳定的、重复的背景模式，它充当真空或**[以太](@article_id:338926)** [@problem_id:870624]。而穿行于这个以太之中的是不同形状和大小的滑翔机，每个都有自己的速度和方向。

这不仅仅是一个类比。我们可以像对待粒子一样对待这些滑翔机。它们有属性。它们会相互作用。例如，如果一个‘E’型滑翔机与一个‘B’型滑翔机碰撞，它们会相互湮灭，并产生三个新的滑翔机：‘A’、‘C’和‘D’ [@problem_id:1666355]。这是一个粒子反应，就像在高能物理实验中一样！我们甚至可以计算系统的“[质心](@article_id:298800)”及其速度等量，并且我们发现这些计算与台球的计算结果完全相同。一种新的、更高层次的物理学从底层规则的简单位翻转中涌现出来。

### 机器中的幽灵：[通用计算](@article_id:339540)

故事在这里迎来了最深刻的转折。这些滑翔机和它们的碰撞不仅仅是一种奇观，它们是一个计算机的组件。

通过精心安排宇宙的初始状态，人们可以设置滑翔机在特定轨迹上运行以执行计算。例如，可以构建一个固定的结构作为“发射器”，周期性地发射出信号滑翔机。如果我们再向这个发射器发射其他滑翔机——我们的“输入”——我们就可以控制它的行为。事实证明，通过精确安排两个输入滑翔机在恰当的时间到达，我们可以暂时“卡住”发射器，阻止它发射输出。当只有一个或没有输入滑翔机时，输出会正常产生 [@problem_id:1666353]。

这种行为——(`1`,`1`) $\to$ `0`，以及 (`1`,`0`) $\to$ `1`，(`0`,`1`) $\to$ `1`，(`0`,`0`) $\to$ `1`——是**[与非门](@article_id:311924)**的逻辑。而这里的神奇之处在于：只要有足够多的与非门，你就可以构建任何可以想象的逻辑电路。你可以构建一个处理器。你可以构建一台计算机。

这意味着，这一行由八条微不足道的规则支配的细胞，能够进行**[通用计算](@article_id:339540)**。它可以计算你的笔记本电脑能计算的任何东西，或者地球上最强大的超级计算机能计算的任何东西。具有这种能力的系统被称为**[图灵完备](@article_id:335210)**的。

Matthew Cook 完成的规则 110 [图灵完备](@article_id:335210)性的发现，在科学界引起了轩然大波。为什么？因为它为计算机科学的基本思想——**[丘奇-图灵论题](@article_id:298662)**——提供了强有力的证据。该论题指出，任何可以通过任何直观的[算法](@article_id:331821)过程“计算”的东西，都可以由一个称为图灵机的[标准模型](@article_id:297875)来计算。规则 110——一个与图灵机（单个读头顺序读取磁带）具有截然不同架构（所有位置同时进行局部、并行更新）的系统——却拥有完全相同的计算能力，这一事实表明，这种能力是现实世界的一种基本的、普遍的属性，而不是我们制造机器方式的偶然产物 [@problem_id:1450192]。

这种普遍性带来了一个黑暗而迷人的后果。因为规则 110 可以模拟任何计算机，它也继承了计算领域所有的怪异之处，包括著名的[停机问题](@article_id:328947)。这意味着关于规则 110 的某些简单问题是**不可判定**的。例如，“给定这个起始模式，网格最终会变成全零吗？”这个问题没有通用的答案。不存在可以为所有可能的起始模式解决这个问题的[算法](@article_id:331821) [@problem_id:1666355]。这个宇宙的未来不仅难以预测；在某些情况下，它是可证明的不可知。

最后，规则 110 还拥有一个近乎神秘的属性。如果我们问：“规则 110 能创造出什么样的[空间模式](@article_id:360081)？”，答案是惊人的。系统的**[拓扑熵](@article_id:326867)**，一个衡量其可能构型复杂性的指标，恰好是 $\ln(2)$ [@problem_id:436689]。用通俗的话说，这意味着你能想象到的任何有限的 `0` 和 `1` 的序列，都可以通过规则 110 的演化产生。《哈姆雷特》的文本、你家猫的数码照片、星舰的蓝图——只要用二进制编码，所有这些都是规则 110 宇宙中有效的模式。从一个近乎痛苦般简单的定律中，诞生了一个具有无限创造潜力的宇宙。
