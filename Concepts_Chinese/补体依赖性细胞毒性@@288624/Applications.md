在经历了补体依赖的细胞毒性作用 (CDC) 复杂的分子编排之后，我们现在退后一步，看看更大的图景。这个强大的机制在我们的生活中扮演着什么角色？如同自然界中许多 fundamental (基本) 的力量一样，我们发现 CDC是一把双刃剑。它是一个警惕的守护者，一种对抗入侵者的武器，一个在自身免疫性疾病中造成自我伤害的原因，并且最近，一个由现代生物工程师进行精细调整的工具。它的原理回响在各个学科中，从手术室的高风险戏剧，到研究实验室的安静 precision ( precision )，再到药物设计的前沿。

### 门前哨兵：CDC 在诊断与移植中的应用

但故事更加 nuanced ( nuanced )。CDC 交叉配型不仅仅是一个简单的“是/否”测试；它是一场 interrogation ( interrogation )。免疫学家可以用像二硫苏糖醇 (DTT) 这样的化学物质处理受者的血清，以 specific (特异性) 破坏五聚体 IgM 抗体。如果测试仍然是阳性，那就 implicates ( implicates ) 更危险的、能固定补体的 IgG 抗体。这种区分对于风险评估至关重要。此外，CDC 检测 fundamentally ( fundamentally ) 是一种*功能性*检测——它测量*杀伤能力*。这使它与更现代、更敏感的测试如流式细胞术交叉配型 (FCXM) 区别开来，后者主要测量*结合能力*。阳性的 FCXM 告诉我们存在抗体，但阳性的 CDC 交叉配型告诉我们它们 armed and dangerous ( armed and dangerous )。移植团队必须权衡两者的结果：FCXM 的高灵敏度以检测任何结合抗体，以及 CDC 交叉配型的 potent ( potent ) 功能性警告。[@problem_id:5232982]
