## 引言
“手性”的概念是生命分子的一个基本属性，从为我们细胞提供能量的糖到编码我们存在的 DNA 都是如此。分子的特定三维形状，包括其手性，决定了其生物学功能。这就提出了一个关键问题：我们如何能感知这些分子的结构，并区分“左手”版本与其“右手”孪生体？答案在于一种被称为[圆二色谱](@entry_id:166583) (CD) 的强大[光谱](@entry_id:185632)技术，它使用手性形式的光作为探针。这种方法已成为分子科学中不可或缺的工具，为我们提供了一个窥探[生物分子结构](@entry_id:169093)和动态的窗口。

本文对[圆二色谱](@entry_id:166583)[光谱学](@entry_id:141940)进行了全面概述。我们将首先探讨支配 CD 工作原理的核心**原理与机制**，从信号的量子力学起源到解读[光谱](@entry_id:185632)以确定蛋白质结构（如 [α-螺旋](@entry_id:139282)和 β-折叠）。随后，我们将遍览其多样的**应用与跨学科联系**，展示 CD 如何在[分子生物学](@entry_id:140331)、生物物理学和化学中用于阐明[蛋白质功能](@entry_id:172023)、量化[分子稳定性](@entry_id:137744)、理解疾病机制以及设计新型催化剂。

## 原理与机制

想象一下，你正试图在黑暗中戴上一副手套。即使它们表面上看起来很相似，你也能感觉到一只是左手用的，另一只是右手用的。你的手和手套都拥有一种我们称之为**手性**的属性，这个词源于希腊语中的“手”(*cheir*)。它们互为镜像，但无法完全重叠。这种“手性”并不仅仅是日常物品的一种奇特现象；它是宇宙构造中一个基本的属性，从亚原子粒子的自旋到星系的宏大螺旋都存在手性。对我们来说最重要的是，它处于生命分子的核心。糖、氨基酸、DNA——它们都是手性的。

自然界似乎对这种手性极为敏感。一个分子的生物学功能可能完全依赖于其特定的手性形式。要从分子水平理解生命，我们必须能够“看到”这种手性。但是，我们如何才能窥探分子世界并区分左手分子与其右手孪生体呢？答案在于使用一种本身就是手性的探针：光。

### 光与手性的舞蹈

我们通常认为光波在传播时是上下或左右[振荡](@entry_id:267781)的。这被称为**线偏振光**。但光也可以具有手性。想象一下，光的[电场](@entry_id:194326)矢量尖端在前进时描绘出一条螺旋或螺旋钻的路径。如果当螺旋向你靠近时是顺时针旋转的，我们称之为**[右旋圆偏振](@entry_id:267955)光 (RCP)**。如果它逆时针旋转，则为**左旋圆偏振光 (LCP)**。这两种形式的光是彼此不可重叠的镜像。它们是手性的。

那么，当这种手性光遇到手性分子时会发生什么呢？就像右手手套比左手手套更适合右手一样，[右旋圆偏振](@entry_id:267955)光与“右手”分子的相互作用也不同于左旋圆偏振光。这种差异性相互作用是关键。

这种差异主要通过两种方式表现出来。首先，这两种光在溶液中传播的速度可能略有不同。这导致一种称为**[旋光](@entry_id:201162) (OR)** 的现象，即[线偏振光](@entry_id:165445)（可视为 LCP 和 RCP 光的组合）的偏振面发生旋转。其次，对我们来说更核心的概念是，分子可能对一种圆偏振光的*吸收*比另一种更强。这种现象被称为**[圆二色谱](@entry_id:166583) (CD)** [@problem_id:2550719]。

名称中的“dichroism”（二色性）部分仅表示差异吸收（“di”表示二，“chroism”表示颜色或吸收）。因此，[圆二色谱](@entry_id:166583)是左旋和[右旋圆偏振](@entry_id:267955)[光吸收](@entry_id:136597)的差异。我们可以非常简单地写出：

$$
\Delta A(\lambda) = A_L(\lambda) - A_R(\lambda)
$$

在这里，$\Delta A$ 是在给定光波长（颜色）$\lambda$ 下的[圆二色谱](@entry_id:166583)值。CD 光谱仪是一种极其灵敏的设备，它在扫描一系列波长时测量这个微小的差异 $\Delta A$，从而产生一个 CD [光谱](@entry_id:185632)。这个[光谱](@entry_id:185632)是[分子手性](@entry_id:164907)的独特指纹。重要的是不要将 CD 与**线二色谱 (LD)** 混淆，后者是*取向*样品对线偏振光的差异吸收，告诉我们关于[排列](@entry_id:136432)的信息，而不一定是手性 [@problem_id:2550719]。相比之下，CD 是各向同性（无取向）溶液中手性的标志。

### 信号的起源：量子力学的扭转

为什么手性分子会差异吸收 LCP 和 RCP 光呢？答案在于光[吸收的量子力学](@entry_id:169728)性质。当一个分子吸收一个[光子](@entry_id:145192)时，一个电子会从较低的能级（[基态](@entry_id:150928)）跃迁到较高的能级（[激发态](@entry_id:261453)）。这个过程可以看作是光波的[振荡](@entry_id:267781)[电场](@entry_id:194326)推动分子内的电子[电荷](@entry_id:275494)。

对于一个简单的非手性分子，[光子](@entry_id:145192)的[电场](@entry_id:194326)可能会引起[电荷](@entry_id:275494)的线性往复运动——即[电偶极跃迁](@entry_id:149662)。但在手性分子中，电子所看到的[势能](@entry_id:748988)景观是扭曲的。它不是一个简单的对称谷地，而是一条螺旋路径。因此，电子的[电荷](@entry_id:275494)在被激发时不仅仅是[直线运动](@entry_id:165142)，而是被迫沿着螺旋路径移动。

这时，光的手性就变得至关重要。如果（比如说）LCP 光的螺旋运动与电子跃迁的自然螺旋路径相匹配，光和分子就会发生共振，光被强烈吸收。而具有相反螺旋运动的 RCP 光则匹配不佳，吸收较弱。结果就是非零的 CD 信号。

更正式地说，只有当吸收一个[光子](@entry_id:145192)的行为同时引起[电荷](@entry_id:275494)的线性位移（**[电偶极跃迁](@entry_id:149662)矩**，$\vec{\mu}_e$）和[电荷](@entry_id:275494)的圆周位移（**[磁偶极跃迁](@entry_id:154694)矩**，$\vec{\mu}_m$）时，CD 信号才能存在。为了使 CD 信号不为零，这两个跃迁矩不能相互垂直。它们的[点积](@entry_id:149019)，定义了跃迁的**旋转强度** $R$，必须不为零。这只在手性环境中才会发生 [@problem_id:3701477]。CD [光谱](@entry_id:185632)本质上是旋转强度与波长的关系图。

### 当[发色团](@entry_id:182442)对话时：[激子耦合](@entry_id:169937)

当一个分子含有多个吸光基团（**[发色团](@entry_id:182442)**）时，情况变得更加有趣。如果两个或多个[发色团](@entry_id:182442)以固定的手性[排列](@entry_id:136432)彼此靠近，它们就不会独立行动。就像一对音叉一样，它们通过空间相互“交谈”。一个发色团的电子激发会影响另一个。

这种通过空间的相互作用，即**[激子耦合](@entry_id:169937)**，会产生显著效应。假设两个相同的[发色团](@entry_id:182442)，每个都在某个波长吸收光，当它们被放在一起时，它们的相互作用会将原始的[激发态](@entry_id:261453)分裂成两个新的能态，一个能量稍低，一个能量稍高。这意味着系统现在有两个新的跃迁，而不是一个。

在手性[排列](@entry_id:136432)中，美妙的事情发生了。原始跃迁的旋转强度被分配到这两个新跃迁之间，但符号相反。一个变为正，另一个变为负，且大小几乎相等。这导致了一种特征性的 CD 信号，称为**耦合信号**或**双符号曲线**：一个正峰紧挨着一个负峰 [@problem_id:3701477]。

这种被称为**[激子耦合](@entry_id:169937)[圆二色谱](@entry_id:166583) (ECCD)** 的现象的力量不容小觑。耦合信号的符号（正谱带的波长是高于还是低于负谱带）直接取决于发色团空间[排列](@entry_id:136432)的手性。这提供了一种直接的、非经验性的方法来确定分子的[绝对构型](@entry_id:192422) [@problem-id:2180246]。值得注意的是，单个发色团本身甚至可以是非手性的！正是它们的 chiral [排列](@entry_id:136432)产生了强大的 CD 信号，就像由相同的矩形台阶组成的螺旋楼梯是手性的一样。

### 解码生命结构

也许 CD 最广泛的用途是在[结构生物学](@entry_id:151045)中，用于破解蛋白质的结构。蛋白质是由氨基酸组成的长链，其骨架是重复的[肽键](@entry_id:144731)序列。每个肽键都是一个在远紫外 (far-UV) 区域（约 190-250 nm）吸收光的发色团。这条链折叠成特定形状——即蛋白质的**二级结构**——的方式，创造了这些[肽键](@entry_id:144731)发色团的固定[排列](@entry_id:136432)，从而产生独特的 CD 信号。

*   **α-螺旋：** 在这种常见结构中，骨架盘绕成一个完美的右手螺旋。[肽键](@entry_id:144731)以规则的螺旋阵列堆叠，导致强烈的[激子耦合](@entry_id:169937)。其结果是一个明确无误的 CD [光谱](@entry_id:185632)：在约 222 nm 和 208 nm 处有两个深的负谷，以及在约 193 nm 处有一个强的正峰。看到这种“双谷”特征是存在 α-螺旋的明确标志 [@problem_id:2074861]。

*   **[β-折叠](@entry_id:176165)：** 在这里，蛋白质链伸展并并排[排列](@entry_id:136432)，形成一种[褶皱](@entry_id:199664)的片状结构。[肽键](@entry_id:144731)的几何[排列](@entry_id:136432)与 α-螺旋完全不同，[激子耦合](@entry_id:169937)也不同。其特征性的 CD [光谱](@entry_id:185632)是在约 217 nm 处有一个宽的负谱带，在 198 nm 附近有一个正峰 [@problem_id:2147649]。

*   **无规卷曲：** 如果蛋白质没有稳定的结构会怎样？对于**本质无序蛋白 (IDPs)**，其链条高度灵活和动态。肽键相对于彼此随机取向，因此[激子耦合](@entry_id:169937)效应平均后几乎为零。剩下的是单个[肽键](@entry_id:144731)的内在 CD 信号，导致在 200 nm 附近有一个单一、强的负谱带 [@problem_id:2115477] [@problem_id:2949947]。

一个真实蛋白质的 CD [光谱](@entry_id:185632)是其所有部分信号的总体加权平均值。通过将蛋白质的实验[光谱](@entry_id:185632)与这些参考[光谱](@entry_id:185632)进行比较，我们可以很好地估计其二级结构组成——例如，它是 40% 的 α-螺旋、20% 的 [β-折叠](@entry_id:176165)和 40% 的无规卷曲。这就是为什么 CD 被称为“低分辨率”技术。与 X 射线[晶体学](@entry_id:140656)或核[磁共振](@entry_id:143712)不同，它不告诉你每个原子的位置。它给你的是蛋白质结构的全局、平均配方，而不是详细的蓝图 [@problem_id:2104055] [@problem_id:2104079]。

### 窥探蛋白质折叠的窗口

CD 提供的不只是一张静态的图片。[蛋白质结构](@entry_id:140548)不仅关乎骨架的[二级结构](@entry_id:138950)，还关乎这些螺旋和折叠如何堆积在一起形成独特的三维形状，即**[三级结构](@entry_id:138239)**。这种堆积为[芳香族氨基酸](@entry_id:194794)侧链（色氨酸、[酪氨酸](@entry_id:199366)和苯丙氨酸）创造了特定的、不对称的环境。

这些芳香族侧链是吸收**近紫外 (near-UV)** 区域（250-350 nm）光的[发色团](@entry_id:182442)。当被蛋白质折叠锁定在固定位置时，它们的手性环境会产生近紫外 CD 信号。这个信号是正确折叠的[三级结构](@entry_id:138239)的灵敏且独特的指纹 [@problem_id:2130636]。

这为我们提供了一个强大的工具来观察蛋白质的动态行为。通过在我们改变温度或添加变性化学物质时监测 CD 信号，我们可以观察蛋白质的展开过程。我们可以看到随着[三级结构](@entry_id:138239)的融化，近紫外信号消失；我们也可以看到随着[二级结构](@entry_id:138950)的丧失，远紫外信号从折叠蛋白的信号变为无规卷曲的信号。因此，[圆二色谱](@entry_id:166583)打开了一扇窗，让我们能够观察[蛋白质折叠](@entry_id:136349)和稳定性的复杂动态过程，让我们得以一窥生命的内在机制。即使对于无序蛋白，其 CD [光谱](@entry_id:185632)的细微变化也能揭示瞬时结构的短暂形成，而这往往是其生物学功能的关键 [@problem_id:2949947]。
