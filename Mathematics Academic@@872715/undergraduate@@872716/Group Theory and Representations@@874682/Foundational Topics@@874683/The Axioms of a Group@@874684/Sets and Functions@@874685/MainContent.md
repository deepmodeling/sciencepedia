## Introduction
In the study of modern mathematics, particularly abstract algebra, the concepts of sets and functions are the fundamental building blocks upon which all complex structures are built. While often introduced informally, a deep and rigorous understanding of their properties is essential for grasping more advanced topics like group theory. This article bridges the gap between the intuitive notion of a function as a "rule" and the formal, powerful framework used in higher mathematics. It demonstrates how these elementary concepts give rise to profound structural ideas like partitions, [equivalence relations](@entry_id:138275), and [quotient spaces](@entry_id:274314).

## Principles and Mechanisms

The study of abstract algebra, and group theory in particular, is fundamentally about structure. We study sets endowed with operations and investigate the properties of these structures and the maps between them that preserve this structure. At the most elementary level, before we can speak of operations and groups, we must have a firm grasp of the language used to describe the relationships between elements within sets and between different sets. This language is that of sets and functions. This chapter lays the rigorous foundation for these concepts, moving from their formal definitions to their crucial role in partitioning sets and describing the symmetries inherent in [group actions](@entry_id:268812).

### The Formal Definition of a Function

While we intuitively understand a function as a "rule" that assigns to each input a unique output, a more formal and powerful definition is required for rigorous mathematics. This definition is based on the concepts of sets and the Cartesian product.

A **function** $f$ from a set $X$, called the **domain**, to a set $Y$, called the **codomain**, is formally a subset of the Cartesian product $X \times Y$. This subset, denoted as the **graph of the function** $G_f$, is not just any collection of [ordered pairs](@entry_id:269702). It must satisfy a specific condition: for every element $x \in X$, there exists exactly one element $y \in Y$ such that the pair $(x, y)$ is in $G_f$.

This condition has two parts:
1.  **Existence**: For every $x \in X$, there is at least one pair $(x, y) \in G_f$. This ensures the function is defined for all elements of the domain.
2.  **Uniqueness**: For every $x \in X$, there is at most one pair $(x, y) \in G_f$. If $(x, y_1) \in G_f$ and $(x, y_2) \in G_f$, then it must be that $y_1 = y_2$. This is the familiar "vertical line test" from introductory calculus, generalized to abstract sets. It ensures that each input has only one output.

As a concrete illustration, let's consider the sets $X = \{a, b, c\}$ and $Y = \{1, 2, 3\}$. The Cartesian product $X \times Y$ consists of nine [ordered pairs](@entry_id:269702), from $(a, 1)$ to $(c, 3)$. Let's examine a few subsets of $X \times Y$ to see which ones can be the [graph of a function](@entry_id:159270) $f: X \to Y$ [@problem_id:1673292].
-   The set $G_A = \{(a, 1), (b, 2), (c, 1)\}$ represents a valid function. For each element $a, b, c$ in the domain $X$, there is precisely one [ordered pair](@entry_id:148349) in $G_A$. It does not matter that both $a$ and $c$ map to the same value, $1$.
-   The set $G_B = \{(a, 3), (b, 3), (c, 3)\}$ also represents a valid function—a constant function where every element of the domain maps to the value $3$.
-   The set $G_C = \{(a, 1), (b, 2)\}$ does not represent a function from $X$ to $Y$ because the element $c \in X$ does not appear as the first component of any pair, violating the existence condition.
-   The set $G_D = \{(a, 1), (a, 2), (b, 3), (c, 1)\}$ does not represent a function because the element $a \in X$ is associated with two different output values, $1$ and $2$, violating the uniqueness condition.

This definition of a function as a set allows us to treat functions themselves as elements of other sets. This abstract viewpoint is particularly powerful. For example, the Cartesian product of an indexed family of sets $\{A_i\}_{i \in I}$ can be defined in this language. The **generalized Cartesian product** $\prod_{i \in I} A_i$ is the set of all functions $f: I \to \bigcup_{i \in I} A_i$ such that for every index $i \in I$, the condition $f(i) \in A_i$ holds [@problem_id:1673263]. In this view, an element of the product is a "choice function" that selects one element from each set $A_i$. For instance, if $I = \{1, 2\}$ and $A_1 = \{a,b\}, A_2 = \{c,d\}$, then an element of $A_1 \times A_2$ is a function $f: \{1,2\} \to \{a,b,c,d\}$ with $f(1) \in A_1$ and $f(2) \in A_2$. The function defined by $f(1)=a, f(2)=c$ corresponds to the familiar [ordered pair](@entry_id:148349) $(a,c)$.

A particularly important type of function is a **bijection**, which is a function that is both **injective** (one-to-one) and **surjective** (onto). An [injective function](@entry_id:141653) maps distinct elements of the domain to distinct elements of the codomain. A [surjective function](@entry_id:147405) is one whose **image**—the set of all actual output values—is equal to the entire codomain. A [bijection](@entry_id:138092) establishes a perfect one-to-one correspondence between the elements of two sets.

A classic and beautiful example of a [bijection](@entry_id:138092) arises when relating the subsets of a set to a particular class of functions. Let $X$ be any set. Its **[power set](@entry_id:137423)**, denoted $P(X)$, is the set of all subsets of $X$. Consider the set $F$ of all functions from $X$ to the two-element set $\{0, 1\}$. There exists a natural [bijection](@entry_id:138092) $\Phi: P(X) \to F$. For any subset $A \subseteq X$, we associate it with its **characteristic function**, $f_A: X \to \{0, 1\}$, defined by $f_A(x) = 1$ if $x \in A$ and $f_A(x) = 0$ if $x \notin A$ [@problem_id:1673295]. This mapping $\Phi$ is a [bijection](@entry_id:138092):
-   It is injective because if two subsets $A$ and $B$ are different, there must be an element $x$ in one but not the other, meaning their characteristic functions $f_A$ and $f_B$ will differ at $x$.
-   It is surjective because any given function $g: X \to \{0, 1\}$ is the [characteristic function](@entry_id:141714) of the subset $A = \{x \in X \mid g(x) = 1\}$.
This correspondence is fundamental in computer science and [combinatorics](@entry_id:144343) and highlights the deep connection between set-theoretic structures and functions.

### Functions, Partitions, and Equivalence Relations

Functions do more than just map elements from one set to another; they naturally structure their domain by grouping elements that map to the same output. This structuring is formalized through the concepts of fibers, partitions, and [equivalence relations](@entry_id:138275).

For a function $f: X \to Y$, the **fiber** over an element $y \in Y$ is the set of all elements in the domain that map to $y$. This set is also called the **[preimage](@entry_id:150899)** of $y$, denoted $f^{-1}(y) = \{x \in X \mid f(x) = y\}$. Note that if $y$ is not in the image of $f$, its fiber is the [empty set](@entry_id:261946).

The collection of all non-empty fibers provides a fundamental structuring of the domain $X$. Specifically, the set of fibers corresponding to elements in the image of $f$, $\mathcal{P} = \{ f^{-1}(y) \mid y \in \mathrm{Im}(f) \}$, forms a **partition** of the domain $X$ [@problem_id:1673242]. A [partition of a set](@entry_id:147307) is a collection of non-empty, pairwise disjoint subsets whose union is the entire set.
1.  **Union is $X$**: Every element $x \in X$ must map to some value $y = f(x)$ in the image. Thus, $x$ belongs to the fiber $f^{-1}(y)$, so the union of all such fibers covers all of $X$.
2.  **Pairwise Disjoint**: Consider two distinct fibers, $f^{-1}(y_1)$ and $f^{-1}(y_2)$, where $y_1 \neq y_2$. If they had an element $x$ in common, then by definition $f(x) = y_1$ and $f(x) = y_2$. But this would imply $y_1 = y_2$, a contradiction. Thus, distinct non-empty fibers are disjoint.

This partitioning property gives rise to a natural **[equivalence relation](@entry_id:144135)** on the domain $X$. An [equivalence relation](@entry_id:144135) $\sim$ on a set is a relation that is reflexive ($x \sim x$), symmetric ($x \sim y \implies y \sim x$), and transitive ($x \sim y \text{ and } y \sim z \implies x \sim z$). Any function $f: X \to Y$ induces such a relation on its domain, defined by:
$$ x_1 \sim x_2 \iff f(x_1) = f(x_2) $$
The **equivalence classes** of this relation are precisely the non-empty fibers of the function $f$ [@problem_id:1673248]. For instance, if $X$ is the set of [binary strings](@entry_id:262113) of length 5 and $f(s)$ is the number of '1's in string $s$, then two strings are equivalent if they have the same number of '1's. The equivalence class of the string '11010' is the set of all length-5 strings with exactly three '1's, as $f('11010') = 3$.

The set of all equivalence classes under a relation $\sim$ on $X$ is called the **[quotient set](@entry_id:137935)**, denoted $X/\sim$. There is a **canonical projection map** $\pi: X \to X/\sim$ that sends each element $x \in X$ to its equivalence class $[x]$. This leads to a crucial question in algebra: if we have a function $f: X \to Y$, can we define a related function on the [quotient set](@entry_id:137935), $\bar{f}: X/\sim \to Y$? For such a function $\bar{f}$ to exist and satisfy $f = \bar{f} \circ \pi$, it must be **well-defined**. This means its output for an equivalence class $[x]$ cannot depend on which representative element $x$ from that class we choose. That is, if $x_1 \sim x_2$, we must have $\bar{f}([x_1]) = \bar{f}([x_2])$. Since $\bar{f}([x])$ is meant to be $f(x)$, this condition is equivalent to requiring that $f(x_1) = f(x_2)$. This leads to the fundamental property for a function to **descend to the quotient**:

A function $f: X \to Y$ descends to a [well-defined function](@entry_id:146846) $\bar{f}: X/\sim \to Y$ if and only if $f$ is constant on the [equivalence classes](@entry_id:156032) of $\sim$. That is, for all $x_1, x_2 \in X$, if $x_1 \sim x_2$, then $f(x_1) = f(x_2)$.

Consider the plane $\mathbb{R}^2$ with the [equivalence relation](@entry_id:144135) $(x_1, y_1) \sim (x_2, y_2)$ if $x_1^2 + y_1^2 = x_2^2 + y_2^2$. The equivalence classes are circles centered at the origin. A function $f: \mathbb{R}^2 \to \mathbb{R}$ will descend to the [quotient set](@entry_id:137935) of these circles if and only if its value depends only on the squared radius $x^2 + y^2$ [@problem_id:1673261].
-   $f(x,y) = 3(x^2 + y^2) + 5$ descends, because if two points are on the same circle, their value for $x^2+y^2$ is the same.
-   $f(x,y) = x+y$ does not descend. For instance, $(1,0)$ and $(0,-1)$ are on the same circle of radius 1, but $f(1,0) = 1$ while $f(0,-1) = -1$. Since the function values differ, no well-defined value can be assigned to this circle.
-   $f(x,y) = (x^2+y^2)^2$ also descends, as it can be written as a function of $u=x^2+y^2$.

### Group Actions as Functions

The concepts of functions and partitions find their most potent application in group theory through the study of [group actions](@entry_id:268812). A (left) **group action** of a group $G$ on a set $X$ is a map from $G \times X$ to $X$, written as $(g,x) \mapsto g \cdot x$, that satisfies two axioms:
1.  $e \cdot x = x$ for all $x \in X$ (where $e$ is the [identity element](@entry_id:139321) of $G$).
2.  $(g_1 g_2) \cdot x = g_1 \cdot (g_2 \cdot x)$ for all $g_1, g_2 \in G$ and $x \in X$.

For each fixed element $g \in G$, the map $\phi_g: X \to X$ defined by $\phi_g(x) = g \cdot x$ is a bijection on $X$, i.e., a permutation of $X$. The inverse of this map is $\phi_{g^{-1}}$. The set of all [permutations](@entry_id:147130) of $X$ forms a group under composition, called the **[symmetric group](@entry_id:142255)** on $X$, denoted $S_X$. The second axiom of [group actions](@entry_id:268812) ensures that the mapping $\phi: G \to S_X$ given by $g \mapsto \phi_g$ is a **[group homomorphism](@entry_id:140603)** [@problem_id:1673247]. In fact, defining a [group action](@entry_id:143336) of $G$ on $X$ is entirely equivalent to providing such a homomorphism.

This perspective is powerful. For example, consider the [dihedral group](@entry_id:143875) $D_4$ (symmetries of a square) acting on the set of its four axes of symmetry. To find the permutation corresponding to the group element $sr$ (a rotation $r$ followed by a reflection $s$), we can find the individual permutations $\phi(s)$ and $\phi(r)$ and compose them: $\phi(sr) = \phi(s) \circ \phi(r)$ [@problem_id:1673247].

Two of the most important concepts related to [group actions](@entry_id:268812) are [orbits and stabilizers](@entry_id:137467).
-   The **orbit** of an element $x \in X$ under the action of $G$ is the set of all points that $x$ can be moved to by elements of $G$: $Gx = \{g \cdot x \mid g \in G\}$.
-   The **stabilizer** of an element $x \in X$ is the set of all group elements that leave $x$ fixed: $G_x = \{g \in G \mid g \cdot x = x\}$. The stabilizer $G_x$ is always a subgroup of $G$.

Let us revisit our function-theoretic framework. For a fixed $x \in X$, we can define the **orbit map** $\pi_x: G \to X$ by $\pi_x(g) = g \cdot x$. The image of this map is precisely the orbit $Gx$. The fibers of this map are subsets of the group $G$. What is the fiber over $x$ itself?
$$ \pi_x^{-1}(x) = \{g \in G \mid \pi_x(g) = x\} = \{g \in G \mid g \cdot x = x\} = G_x $$
The fiber over $x$ is exactly the [stabilizer subgroup](@entry_id:137216) $G_x$. More generally, the fibers of the orbit map are the left **[cosets](@entry_id:147145)** of the stabilizer. This insight forms the basis of the renowned Orbit-Stabilizer Theorem, which states that the size of an orbit is the index of the [stabilizer subgroup](@entry_id:137216): $|Gx| = [G : G_x]$. Calculating the size of a stabilizer is a common task. For instance, in the action of the [matrix group](@entry_id:156202) $GL(2, \mathbb{Z}_3)$ on the vector space $(\mathbb{Z}_3)^2$ by [matrix multiplication](@entry_id:156035), the stabilizer of a vector like $x_0 = \begin{pmatrix} 1 \\ 2 \end{pmatrix}$ can be found by solving the [matrix equation](@entry_id:204751) $A x_0 = x_0$ for matrices $A \in GL(2, \mathbb{Z}_3)$ [@problem_id:1673258].

### Maps That Respect Group Structure

Just as we study structure-preserving maps (homomorphisms) between groups, we can study functions between sets that are equipped with a $G$-action (called **$G$-sets**). A function $f: X \to Y$ between two $G$-sets is called a **$G$-[equivariant map](@entry_id:143787)** (or a morphism of $G$-sets) if it respects the group action. This means that for all $g \in G$ and all $x \in X$, the following condition holds:
$$ f(g \cdot x) = g \cdot f(x) $$
This condition states that it doesn't matter whether we first apply the [group action](@entry_id:143336) in $X$ and then map to $Y$, or first map to $Y$ and then apply the group action there; the result is the same.

The number and nature of such maps depend intimately on the orbit and stabilizer structure of the actions. If the action of $G$ on $X$ is **transitive** (meaning there is only one orbit, $X$ itself), then any $G$-[equivariant map](@entry_id:143787) $f: X \to Y$ is completely determined by its value on a single element $x_0 \in X$. The value $f(x_0)$ cannot be just any element of $Y$; it must be an element that is fixed by the [stabilizer subgroup](@entry_id:137216) $G_{x_0}$. The set of all such $G$-equivariant maps from $X$ to $Y$ is in bijection with the set of elements in $Y$ fixed by $G_{x_0}$ [@problem_id:1641044].

The study of orbits becomes even more interesting when multiple groups act on the same set. If a group $G$ and a group $H$ both act on a set $X$, and their actions commute (i.e., $g \cdot (h \cdot x) = h \cdot (g \cdot x)$ for all $g \in G, h \in H, x \in X$), this gives rise to a well-defined action of the [direct product](@entry_id:143046) group $G \times H$. The orbits under this combined action can be analyzed in stages. One can first find the orbits of $X$ under the action of $G$, let's call this set of orbits $X/G$. Then, the action of $H$ induces an action on the set $X/G$. The orbits of the [product group](@entry_id:276017) $G \times H$ on $X$ are then in [one-to-one correspondence](@entry_id:143935) with the orbits of this induced action of $H$ on $X/G$ [@problem_id:1641040].

Finally, it is worth noting that the core idea of partitioning a set via an action-like structure is not limited to groups. A **semigroup** is a set with an associative [binary operation](@entry_id:143782) (but not necessarily inverses or an identity). A semigroup can also act on a set. An equivalence relation can be constructed where two elements are related if one can be reached from the other by a finite sequence of action steps, forwards or backwards. For example, the semigroup of non-negative integers $(\mathbb{N}_0, +)$ acts on the positive integers $\mathbb{Z}^+$ via $n \cdot x = x \cdot 2^n$. The resulting equivalence relation partitions $\mathbb{Z}^+$ into classes where all numbers in a given class share the same unique odd factor [@problem_id:1673297]. This provides another compelling example of how functions and actions impose a deep and meaningful structure upon a set.
