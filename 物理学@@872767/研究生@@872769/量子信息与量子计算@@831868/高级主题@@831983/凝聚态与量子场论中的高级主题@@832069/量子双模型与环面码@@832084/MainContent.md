## 引言
超越传统朗道[对称性破缺](@entry_id:158994)[范式](@entry_id:161181)的拓扑量子[物相](@entry_id:196677)，代表了物质的一种奇异形态，其性质由全局拓扑结构而非局域序参量决定。然而，这一抽象概念如何在一个具体的物理系统中实现？本文旨在通过深入剖析一类[精确可解模型](@entry_id:142243)——量子双模型及其最著名的特例“环面编码”（Toric Code），来填补这一认知鸿沟。这些模型不仅是[理论物理学](@entry_id:154070)家的杰作，更是理解[拓扑序](@entry_id:147345)、分数化激发（任意子）以及构建容错量子计算机的基石。

本文将引导读者系统地掌握这一前沿领域。在第一章“原理与机制”中，我们将从环面编码的[哈密顿量](@entry_id:172864)出发，揭示其稳定子结构、拓扑简并的[基态](@entry_id:150928)以及奇异的[任意子](@entry_id:143753)激发，并将其推广至更一般的量子双模型，探索其丰富的[代数结构](@entry_id:137052)。随后的“应用与跨学科联系”一章将展示这些理论原理如何在量子纠错、凝聚态物理的[相变](@entry_id:147324)理论以及与[拓扑量子场论](@entry_id:142425)的深刻联系中发挥关键作用。最后，在“动手实践”部分，我们提供了一系列精心设计的问题，旨在通过实际计算，巩固您对任意子创生、探测及其代数性质的理解，将抽象理论转化为具体的物理直觉。

## 原理与机制

在上一章对拓扑量子物相进行了介绍性概述之后，本章将深入探讨其原理和机制。我们将以最典型的拓扑有序模型——Toric Code（环面编码）——为起点，系统地阐述其[哈密顿量](@entry_id:172864)、稳定子形式、[基态](@entry_id:150928)结构和[任意子](@entry_id:143753)激发。随后，我们会将这些概念推广到更一般的量子双模型 $D(G)$，并详细探讨其丰富的[任意子](@entry_id:143753)理论，包括[任意子](@entry_id:143753)的分类、[量子维度](@entry_id:146936)、[融合规则](@entry_id:142240)和[编织统计](@entry_id:147187)。最后，我们将这些微观性质与宏观拓扑不变量联系起来，例如[基态简并度](@entry_id:141614)和[拓扑纠缠熵](@entry_id:145064)，从而揭示这些奇异[物相](@entry_id:196677)的深刻结构。

### Toric Code：一个典型的拓扑序模型

Toric Code 是 Alexei Kitaev 提出的一个精确可解[格点模型](@entry_id:184345)，它完美地展示了[拓扑序](@entry_id:147345)的许多核心特征，并为[容错量子计算](@entry_id:142498)提供了一个框架。

#### 格点与[哈密顿量](@entry_id:172864)

想象一个 $L \times L$ 的二维方格[晶格](@entry_id:196752)，通过周期性边界条件将其“卷曲”成一个环面。[量子比特](@entry_id:137928)（自旋-1/2系统）不位于格点（顶点）上，而是位于连接顶点的每条边上。

该系统的物理性质由一个[局域哈密顿量](@entry_id:141996) $H$ 描述，它由两类算符构成：

1.  **顶点算符（Star operators）** $A_s$：每个顶点 $s$ 都关联一个顶点算符，它是在汇集于该顶点的四条边上的[量子比特](@entry_id:137928)上作用的泡利-$X$ 算符 $\sigma_i^x$ 的乘积：
    $$A_s = \prod_{i \in \text{star}(s)} \sigma_i^x$$

2.  **格方算符（Plaquette operators）** $B_p$：每个格方（或称面元）$p$ 都关联一个格方算符，它是在构成该格方边界的四条边上的[量子比特](@entry_id:137928)上作用的泡利-$Z$ 算符 $\sigma_j^z$ 的乘积：
    $$B_p = \prod_{j \in \text{plaq}(p)} \sigma_j^z$$

Toric Code 的[哈密顿量](@entry_id:172864)就是所有这些顶点和格方算符的总和：
$$H = -J_A \sum_s A_s - J_B \sum_p B_p$$
其中 $J_A$ 和 $J_B$ 是正的耦合常数，总和遍及所有的顶点 $s$ 和格方 $p$。该[哈密顿量](@entry_id:172864)不包含任何外部场或[长程相互作用](@entry_id:140725)，其所有项都是局域的。

#### 稳定子形式

Toric Code 的一个关键特性是它是一个**稳定子编码 (stabilizer code)**。这意味着[哈密顿量](@entry_id:172864) $H$ 中的所有项 $A_s$ 和 $B_p$ 都是互相交换（commute）的。让我们来验证这一点。

首先，任何两个顶点算符 $[A_s, A_{s'}]$ 或任何两个格方算符 $[B_p, B_{p'}]$ 显然都交换，因为它们要么作用在不相交的[量子比特](@entry_id:137928)集合上，要么在同一[量子比特](@entry_id:137928)上作用的都是相同类型的[泡利算符](@entry_id:144061)（$\sigma^x$ 或 $\sigma^z$），而 $(\sigma^x)^2 = I, (\sigma^z)^2 = I$。

更有趣的是顶点算符和格方算符之间的对易关系。考虑一个顶点算符 $A_s$ 和一个格方算符 $B_p$。
- 如果顶点 $s$ 和格方 $p$ 不相邻，它们作用在完全不同的[量子比特](@entry_id:137928)上，因此 $[A_s, B_p] = 0$。
- 如果顶点 $s$ 和格方 $p$ 相邻，那么它们将共享正好两条边。对于每条共享的边 $k$，算符 $A_s$ 包含 $\sigma_k^x$，算符 $B_p$ 包含 $\sigma_k^z$。由于泡利矩阵的[反对易关系](@entry_id:153815) $\sigma^x \sigma^z = -\sigma^z \sigma^x$，每次交换都会产生一个负号。因为有两条共享边，所以总共会产生两个负号，即 $(-1)^2 = 1$。因此，$A_s B_p = B_p A_s$，我们总是有 $[A_s, B_p] = 0$。

既然[哈密顿量](@entry_id:172864)中的所有项都相互对易，那么每一项都与总[哈密顿量](@entry_id:172864) $H$ 对易 [@problem_id:119038]。这意味着 $A_s$ 和 $B_p$ 的[本征值](@entry_id:154894)都是守恒量。此外，由于 $(\sigma^x)^2 = I$ 和 $(\sigma^z)^2 = I$，我们可以轻易地看到 $(A_s)^2 = I$ 和 $(B_p)^2 = I$。这表明这些[稳定子算符](@entry_id:141669)的[本征值](@entry_id:154894)只能是 $+1$ 或 $-1$。

#### [基态](@entry_id:150928)空间

为了最小化[哈密顿量](@entry_id:172864) $H = -J_A \sum_s A_s - J_B \sum_p B_p$ 的能量（因为 $J_A, J_B > 0$），系统会趋向于一个使所有 $A_s$ 和 $B_p$ 的[期望值](@entry_id:153208)都为 $+1$ 的状态。因此，**[基态](@entry_id:150928)**被定义为所有[稳定子算符](@entry_id:141669)的共同 $+1$ 本征态：
$$A_s |\psi_{GS}\rangle = +1 |\psi_{GS}\rangle \quad \forall s$$
$$B_p |\psi_{GS}\rangle = +1 |\psi_{GS}\rangle \quad \forall p$$

这些条件对[基态](@entry_id:150928)施加了强大的约束。让我们来理解它们的物理意义：
- **顶点条件 $A_s = +1$**：这个条件可以看作是某种“[高斯定律](@entry_id:141493)”。考虑在 $Z$ 基下（$|0\rangle, |1\rangle$），$A_s$ 算符将翻转汇集于顶点 $s$ 的四条边上的[量子比特](@entry_id:137928)。$A_s=1$ 的条件意味着[基态](@entry_id:150928)在 $Z$ 基下展开时，必须是所有满足在每个顶点处都有偶数条边处于 $|1\rangle$ 态的经典构型的叠加。这些构型形成了闭合的“圈” (loops)。在 $L \times L$ 的环面上，所有可能的闭合圈构型的总数是 $2^{L^2+1}$ [@problem_id:118910]。从另一个角度看，对于一个顶点及其关联的四条边，其局域[希尔伯特空间](@entry_id:261193)是 $2^4=16$ 维的。$A_s$ 算符将这个空间分为两个等维的[子空间](@entry_id:150286)，对应[本征值](@entry_id:154894) $+1$ 和 $-1$。满足 $A_s=+1$ 条件的[子空间](@entry_id:150286)维度为 $8$ [@problem_id:118880]。

- **格方条件 $B_p = +1$**：这个条件可以被解释为“无磁通”条件。它要求沿着任何格方边界的 $\sigma^z$ 算符乘积为 $+1$。这意味着，在 $X$ 基（$|+\rangle, |-\rangle$）下，[基态](@entry_id:150928)可以被看作是所有满足特定约束的“弦网”的叠加。

在球形或任何单连通的表面上，满足所有稳定子条件的[基态](@entry_id:150928)是唯一的。然而，在具有非平庸拓扑的表面上，例如环面，情况就不同了。环面有两个独立的、不可收缩的圈（一个沿经线方向，一个沿纬线方向）。我们可以定义沿着这些圈的“逻辑算符”，例如 $Z_1 = \prod_{j \in \text{loop}_1} \sigma_j^z$ 和 $X_1 = \prod_{k \in \text{dual loop}_1} \sigma_k^x$。这些逻辑算符与所有局域的 $A_s$ 和 $B_p$ 算符都对易，因此也与[哈密顿量](@entry_id:172864)对易。然而，它们彼此之间（例如 $Z_1$ 和 $X_1$）却反对易。这意味着我们不能同时确定它们的[本征值](@entry_id:154894)。对于一个环面，有两对这样的逻辑算符（$(Z_1, X_1)$ 和 $(Z_2, X_2)$），它们编码了两个逻辑量子比特。这导致了 $2^2=4$ 个简并的[基态](@entry_id:150928)，这个简并度是[拓扑保护](@entry_id:145388)的，对局域微扰不敏感。

### Toric Code 中的任意子激发

[拓扑序](@entry_id:147345)的一个标志性特征是其分数化激发，即**[任意子](@entry_id:143753) (anyons)**。在 Toric Code 中，激发对应于违反稳定子条件的局域区域。

#### 创造与识别激发

[基态](@entry_id:150928)满足所有 $A_s=+1$ 和 $B_p=+1$。如果某个状态 $|\psi\rangle$ 使得 $\langle\psi|A_s|\psi\rangle = -1$，我们就说在顶点 $s$ 处存在一个激发。同样，$\langle\psi|B_p|\psi\rangle = -1$ 意味着在格方 $p$ 处存在一个激发。

- **e-[任意子](@entry_id:143753)（[电荷](@entry_id:275494)）**：位于顶点 $s$ 的激发，满足 $A_s = -1$。
- **m-任意子（磁通）**：位于格方 $p$ 的激发，满足 $B_p = -1$。

这些激发是如何产生的呢？它们总是成对出现。考虑对系统[基态](@entry_id:150928) $|\text{GS}\rangle$ 施加一个局域算符。
- **$\sigma_k^z$ 算符**：在边 $k$ 上施加一个 $\sigma_k^z$ 算符。这个算符与所有 $B_p$ 对易，但与边 $k$ 两端的两个顶点算符 $A_{s_1}$ 和 $A_{s_2}$ [反对易](@entry_id:186708)。因此，状态 $\sigma_k^z |\text{GS}\rangle$ 将满足 $A_{s_1}=-1$ 和 $A_{s_2}=-1$。这就在 $s_1$ 和 $s_2$ 处创造了一对 e-[任意子](@entry_id:143753)。
- **$\sigma_k^x$ 算符**：在边 $k$ 上施加一个 $\sigma_k^x$ 算符。这个算符与所有 $A_s$ 对易，但与共享边 $k$ 的两个格方算符 $B_{p_1}$ 和 $B_{p_2}$ [反对易](@entry_id:186708)。因此，状态 $\sigma_k^x |\text{GS}\rangle$ 将满足 $B_{p_1}=-1$ 和 $B_{p_2}=-1$。这就在 $p_1$ 和 $p_2$ 处创造了一对 m-任意子。

更有趣的是，即使是作用在相邻[量子比特](@entry_id:137928)上的多体门，也会以更复杂的方式产生激发，其产生的[任意子](@entry_id:143753)模式取决于门的具体形式及其作用的位置 [@problem_id:119002]。

#### [任意子](@entry_id:143753)的性质

这些激发之所以被称为“[任意子](@entry_id:143753)”，是因为它们既不像[玻色子](@entry_id:138266)也不像[费米子](@entry_id:146235)。它们具有一些独特的性质：
1.  **拓扑性**：创造一对任意子需要一定的能量（[能隙](@entry_id:191975)），但一旦产生，移动它们的位置不需要能量，只要它们不相遇湮灭。移动一个[任意子](@entry_id:143753)的操作对应于施加一个“弦算符”（例如，一连串的 $\sigma_x$ 或 $\sigma_z$）。只要弦算符的端点固定，其具体路径是无关紧要的（只要不穿过另一个任意子）。
2.  **融合 (Fusion)**：两个相同类型的[任意子](@entry_id:143753)（例如两个 e-[任意子](@entry_id:143753)）在同一位置相遇时会相互湮灭，回到真空（即没有激发的[基态](@entry_id:150928)）。表示为 $e \otimes e = 1$，$m \otimes m = 1$。一个 e-任意子和一个 m-[任意子](@entry_id:143753)可以结合形成一个[复合粒子](@entry_id:150176)，称为[费米子](@entry_id:146235) $\psi=em$。
3.  **[编织统计](@entry_id:147187) (Braiding Statistics)**：当一个任意子绕着另一个任意子运动一周时，系统的[波函数](@entry_id:147440)会获得一个拓扑相位。在 Toric Code 中：
    - 将一个 e-[任意子](@entry_id:143753)绕一个 e-任意子或一个 m-任意子绕一个 m-[任意子编织](@entry_id:142428)，不会产生相位。它们表现为[玻色子](@entry_id:138266)。
    - 将一个 e-[任意子](@entry_id:143753)绕一个 m-[任意子编织](@entry_id:142428)一圈，[波函数](@entry_id:147440)会获得一个 $-1$ 的相位。这种非平庸的互易统计是[拓扑序](@entry_id:147345)的标志。

我们可以通过计算特定[激发态](@entry_id:261453)之间的交叠来揭示这些性质。例如，一个 e-[任意子](@entry_id:143753)对可以通过单比特算符 $\sigma_x^k$ 产生，也可以通过一个更长的弦算符 $W_x(\gamma)$ 产生。如果这两个算符通过一个逻辑算符 $X_L$ 相关联，即 $W_x(\gamma)\sigma_x^k = X_L$，那么这两个看起来不同的[激发态](@entry_id:261453)之间的交叠实际上测量了[基态](@entry_id:150928)在逻辑算符下的变换性质 [@problem_id:118888]。

### 推广：量子双模型 $D(G)$

Toric Code 模型是基于最简单的群 $\mathbb{Z}_2$ 的量子双模型。这个框架可以被推广到任何[有限群](@entry_id:139710) $G$，从而产生一系列具有更丰富物理内涵的拓扑模型，称为 $D(G)$ 模型。

#### 从 $\mathbb{Z}_2$ 到一般群 $G$

在 $D(G)$ 模型中，每条边上的自由度不再是[量子比特](@entry_id:137928)，而是 $|G|$ 维的“量子qudit”，其希尔伯特空间的[基矢](@entry_id:199546)由群 $G$ 的元素 $|g\rangle$ ($g \in G$) 标记。Toric Code 对应于 $G = \mathbb{Z}_2 = \{0, 1\}$，其中态 $|0\rangle$ 和 $|1\rangle$ 可以分别映射到泡利-$Z$ 的[本征态](@entry_id:149904)。

[哈密顿量](@entry_id:172864)的结构保持不变，仍然是顶点算符和格方算符的和 $H = -\sum_v A_v - \sum_p B_p$。但这些算符的定义被推广了。

#### $D(G)$ [哈密顿量](@entry_id:172864)

- **顶点算符 $A_v$**：对于一个顶点 $v$，算符 $A_v$ 推广了“高斯定律”的约束。它是一个投影算符，将状态投影到在顶点 $v$ 处满足“零[电荷](@entry_id:275494)”条件的空间上。对于一个 $k$ 价顶点，其作用在局域[希尔伯特空间](@entry_id:261193)上的迹为 $|G|^{k-1}$ [@problem_id:119004]。例如，对于四价顶点，迹为 $|G|^3$。这个算符确保了[基态](@entry_id:150928)在局域“规范变换”下保持不变。

- **格方算符 $B_p$**：对于一个格方 $p$，算符 $B_p$ 推广了“无[磁通](@entry_id:191239)”的约束。它测量了绕格方边界的群元素的有序乘积。如果乘积为群的单位元 $e$，则称磁通为零。$B_p$ 是一个[投影算符](@entry_id:154142)，将状态投影到所有满足零[磁通](@entry_id:191239)条件的空间。对于 $D(\mathbb{Z}_N)$ 模型，一个格方的四条边共有 $N^4$ 种可能的经典构型 $(g_1, g_2, g_3, g_4)$，其中满足零磁通条件 $g_1+g_2-g_3-g_4 \equiv 0 \pmod N$ 的构型数量为 $N^3$ [@problem_id:119007]。

[基态](@entry_id:150928)同样被定义为所有 $A_v$ 和 $B_p$ 的共同 $+1$ 本征态。

#### $D(G)$ 模型中的激发

$D(G)$ 模型中的激发（任意子）比 Toric Code 中的 e 和 m [任意子](@entry_id:143753)要丰富得多。
- 激发仍然对应于违反 $A_v=1$（[电荷](@entry_id:275494)）或 $B_p=1$（[磁通](@entry_id:191239)）的区域。
- 激发可以通过**带状算符 (ribbon operator)** $F_g(P)$ 来创建。这是一个沿着[对偶格](@entry_id:150046)点上的路径 $P$ 作用的算符，它将路径穿过的每条边上的状态 $|k\rangle$ 变为 $|gk\rangle$。
- 当 $G$ 是[非阿贝尔群](@entry_id:141904)时，激发可以是**非阿贝尔[任意子](@entry_id:143753) (non-abelian anyons)**。

以 $D(S_3)$ 模型为例，$S_3$ 是三元素[置换群](@entry_id:142907)。一个带状算符 $F_g(P)$ (例如 $g=(12)$) 作用在[基态](@entry_id:150928)上，会在路径的端点产生一对[任意子](@entry_id:143753)。这些任意子的“[磁通](@entry_id:191239)”部分由 $g$ 的共轭类 $[g]$ 决定，而其“[电荷](@entry_id:275494)”部分则由它们在局域规范变换下的变换性质决定，这个变换性质构成了一个[群表示](@entry_id:156757)。对于 $g=(12)$ 产生的激发，其[电荷](@entry_id:275494)部分对应的表示可以分解为 $S_3$ 的[平凡表示](@entry_id:141357)和二维标准[表示的直和](@entry_id:138310) [@problem_id:118923]。

### [任意子](@entry_id:143753)的丰富结构

$D(G)$ 模型中的任意子具有复杂的内部结构，可以通过[量子维度](@entry_id:146936)、[融合规则](@entry_id:142240)和[编织统计](@entry_id:147187)来表征。

#### 任意子分类

$D(G)$ 模型中的[任意子](@entry_id:143753)由一个对 $(C, \rho)$ 来标记，其中：
- $C$ 是 $G$ 的一个**共轭类 (conjugacy class)**。这代表任意子的**磁通荷 (magnetic flux charge)**。
- $\rho$ 是 $C$ 中任意元素 $h$ 的**[中心化子](@entry_id:146604)群 (centralizer)** $Z(h)=\{g \in G \mid ghg^{-1}=h\}$ 的一个**不可约表示 (irrep)**。这代表任意子的**[电荷](@entry_id:275494) (electric charge)**。

根据这个分类，我们可以识别出几种特殊类型的任意子：
- **纯[电荷](@entry_id:275494)**：对应于单位元的共轭类 $C=\{e\}$。其[中心化子](@entry_id:146604)是整个群 $G$，因此纯[电荷](@entry_id:275494)由 $G$ 的不可约表示 $\rho$ 标记，记为 $(\{e\}, \rho)$。
- **纯[磁通](@entry_id:191239)**：对应于非平凡的共轭类 $C$，但其[电荷](@entry_id:275494)部分是[中心化子](@entry_id:146604) $Z(h)$ 的[平凡表示](@entry_id:141357) $\rho_{\text{triv}}$，记为 $(C, \rho_{\text{triv}})$。
- **复合[任意子](@entry_id:143753) (Dyons)**：既有非平凡的[磁通](@entry_id:191239)荷，也有非平凡的[电荷](@entry_id:275494)，即 $C \neq \{e\}$ 且 $\rho \neq \rho_{\text{triv}}$。

#### [量子维度](@entry_id:146936)

每个[任意子](@entry_id:143753) $a$ 都被赋予一个称为**[量子维度](@entry_id:146936) (quantum dimension)** 的正实数 $d_a$。对于标记为 $(C, \rho)$ 的[任意子](@entry_id:143753)，其[量子维度](@entry_id:146936)为：
$$d_{(C, \rho)} = |C| \cdot \dim(\rho)$$
其中 $|C|$ 是共轭类 $C$ 的大小，$\dim(\rho)$ 是表示 $\rho$ 的维度。

让我们以 $D(S_3)$ 模型为例：
- 一个对应于二维不可约表示的纯[电荷](@entry_id:275494)，其[量子维度](@entry_id:146936)为 $d = |\{e\}| \cdot \dim(\rho_{2D}) = 1 \cdot 2 = 2$ [@problem_id:119015]。
- 一个对应于 3-循环[共轭类](@entry_id:143916) $C_{(123)}$（大小为 2）的纯磁通，其中心化子是 $\mathbb{Z}_3$（所有表示都是一维的），所以[量子维度](@entry_id:146936)为 $d = |C_{(123)}| \cdot 1 = 2 \cdot 1 = 2$ [@problem_id:118947]。
- 一个对应于[置换](@entry_id:136432)共轭类 $C_{(12)}$（大小为 3）和其[中心化子](@entry_id:146604) $\mathbb{Z}_2$ 的符号表示（一维）的复合[任意子](@entry_id:143753)，其[量子维度](@entry_id:146936)为 $d = |C_{(12)}| \cdot 1 = 3 \cdot 1 = 3$ [@problem_id:118957]。

#### [融合规则](@entry_id:142240)

[任意子](@entry_id:143753)可以相互“融合”，产生新的[任意子](@entry_id:143753)。两个任意子 $a$ 和 $b$ 的融合结果可能不唯一，其一般形式为：
$$a \otimes b = \bigoplus_c N_{ab}^c c$$
其中 $N_{ab}^c$ 是非负整数，称为**融合系数 (fusion coefficients)**，表示 $a$ 和 $b$ 融合成 $c$ 的通道数。

- 对于阿贝尔群 $G$（例如 $\mathbb{Z}_N$），[融合规则](@entry_id:142240)非常简单。[任意子](@entry_id:143753)由 $(g, \chi_k)$ 标记，其中 $g \in \mathbb{Z}_N$，$k$ 标记 $\mathbb{Z}_N$ 的一个特征标（[一维表示](@entry_id:136509)）。[融合规则](@entry_id:142240)是 $(g_1, \chi_{k_1}) \otimes (g_2, \chi_{k_2}) = (g_1+g_2, \chi_{k_1+k_2})$ [@problem_id:118990]。
- 对于[非阿贝尔群](@entry_id:141904)，规则更为复杂。例如，在 $D(S_3)$ 中，两个与[置换](@entry_id:136432)[共轭类](@entry_id:143916)相关的纯[磁通](@entry_id:191239)任意子 $a$ 融合。我们想知道它们融合成真空（标记为 $1$）的通道数 $N_{aa}^1$。这等价于计算满足 $g_i g_j = e$ 的对 $(g_i, g_j)$ 的数量，其中 $g_i, g_j$ 都是[置换](@entry_id:136432)。由于每个[置换](@entry_id:136432)都是自身的逆，所以只有当 $g_i=g_j$ 时才满足条件。因为有 3 个[置换](@entry_id:136432)，所以 $N_{aa}^1 = 3$ [@problem_id:118955]。

#### [编织统计](@entry_id:147187)

当[任意子](@entry_id:143753)相互绕行时，系统的状态会发生幺正变换。
- 对于阿贝尔[任意子](@entry_id:143753)，这种变换仅仅是一个相位因子，称为**互易统计 (mutual statistics)**。例如，在 $D(\mathbb{Z}_N)$ 模型中，一个[电荷](@entry_id:275494) $\rho$ 绕一个磁通 $g$ 编织一周，获得的相位是 $\chi_{\rho}(g)$，即表示 $\rho$ 的特征标在 $g$ 上的取值。对于 $D(\mathbb{Z}_5)$ 中的[电荷](@entry_id:275494) $\chi_2$ 和磁通 $g=1$，此相位为 $\exp(4\pi i/5)$ [@problem_id:118943]。
- 对于非阿贝尔任意子，这种编织操作可以在简并的[量子态空间](@entry_id:197873)中引起一个非对角的幺正[矩阵变换](@entry_id:156789)。这种**非阿贝尔统计**是[拓扑量子计算](@entry_id:138660)的基石。在 $D(G)$ 模型中，将一个[电荷](@entry_id:275494) $(\{e\}, \rho)$ 移动到[磁通](@entry_id:191239) $(A, \sigma)$ 的位置，等效于将它们融合成一个新的复合[任意子](@entry_id:143753) $(A, \rho|_{Z(g)} \otimes \sigma)$ [@problem_id:118972]。这个过程揭示了编织与融合之间的深刻联系。

### 宏观拓扑性质

任意子的微观性质决定了系统的宏观、可测量的[拓扑不变量](@entry_id:138526)。

#### [基态简并度](@entry_id:141614) (GSD)

如前所述，拓扑序的[基态简并度](@entry_id:141614)取决于系统所在的[流形的拓扑](@entry_id:267834)。GSD 等于该[流形](@entry_id:153038)的[基本群](@entry_id:146111) $\pi_1(\Sigma)$ 到群 $G$ 的同态映射的数量，即 $|\text{Hom}(\pi_1(\Sigma), G)|$。

- **环面 ($T^2$)**：其基本群是 $\pi_1(T^2) = \langle a, b \mid aba^{-1}b^{-1} = e \rangle$。一个同态映射由两个群元素 $g_a, g_b \in G$ 决定，它们必须满足对易关系 $g_a g_b = g_b g_a$。因此，GSD 等于 $G$ 中[有序对](@entry_id:269702)易对的数量。对于 $S_3$，这个数量是 18 [@problem_id:118920]。
- GSD 还有一个等价的计算方法，即 $D(G)$ 模型中所有任意子类型的总数。对于 $S_3$ 在环面上的 GSD，我们可以对 $S_3$ 的每个[共轭类](@entry_id:143916)，计算其中心化子群的不可约表示数量，然后求和。结果是 $3 (\text{来自 } Z(e)=S_3) + 2 (\text{来自 } Z((12))\cong\mathbb{Z}_2) + 3 (\text{来自 } Z((123))\cong\mathbb{Z}_3) = 8$ [@problem_id:118912]。这两个看似不同的结果（18 vs 8）实际上描述了不同的物理量，前者是平坦联络的数量，后者是[任意子](@entry_id:143753)类型的数量，它们在某些特定理论中是相等的，这个微妙的差异超出了本章范围，但两者都揭示了拓扑依赖性。
- **其他[曲面](@entry_id:267450)**：对于其他[曲面](@entry_id:267450)，GSD 也由其[基本群](@entry_id:146111)决定。例如，克莱因瓶 $K$ 的基本群是 $\pi_1(K) = \langle a, b \mid aba^{-1}b = e \rangle$。$D(D_4)$ 模型在[克莱因瓶](@entry_id:149661)上的 GSD 是满足 $g_a g_b g_a^{-1} g_b = e$ 的对 $(g_a, g_b) \in D_4 \times D_4$ 的数量，计算可得为 40 [@problem_id:118916]。对于有边界的非定向[流形](@entry_id:153038)，如[莫比乌斯带](@entry_id:152389)，GSD 会因为边界条件和非定向性而改变。Toric Code 在[莫比乌斯带](@entry_id:152389)上的 GSD 为 2 [@problem_id:118872]。

#### [拓扑纠缠熵](@entry_id:145064)

在一个有[能隙](@entry_id:191975)的二维量子系统中，一个大区域 $A$ 的纠缠熵 $S(A)$ 通常遵循“[面积定律](@entry_id:145931)”：$S(A) = \alpha L - \gamma$，其中 $L$ 是区域 $A$ 的边界长度，$\alpha$ 是非普适的系数。关键在于常数项 $\gamma$，它是一个普适的拓扑不变量，称为**[拓扑纠缠熵](@entry_id:145064) (topological entanglement entropy)**。

$\gamma$ 与系统的[任意子](@entry_id:143753)内容直接相关，由下式给出：
$$\gamma = \ln \mathcal{D}$$
其中 $\mathcal{D}$ 是理论的**总[量子维度](@entry_id:146936) (total quantum dimension)**，定义为 $\mathcal{D} = \sqrt{\sum_a d_a^2}$，求和遍及所有[任意子](@entry_id:143753)类型 $a$。

对于 $D(\mathbb{Z}_N)$ 模型，由于是[阿贝尔群](@entry_id:150284)，所有[共轭类大小](@entry_id:143680)为 1，所有[不可约表示](@entry_id:263310)都是一维的。因此，每个任意子的[量子维度](@entry_id:146936) $d_a$ 都是 1。[任意子](@entry_id:143753)的总数是 $N \times N = N^2$。所以：
$$\mathcal{D}^2 = \sum_{a=1}^{N^2} 1^2 = N^2 \implies \mathcal{D} = N$$
因此， $D(\mathbb{Z}_N)$ 模型的[拓扑纠缠熵](@entry_id:145064)是 $\gamma = \ln N$ [@problem_id:119034]。这个优美的结果将一个宏观的[纠缠度量](@entry_id:139894)与定义模型的[代数结构](@entry_id:137052)（[群的阶](@entry_id:137115)数）直接联系起来。

本章通过 Toric Code 和其向量子双模型的推广，系统地构建了拓扑序的物理图像。从[局域哈密顿量](@entry_id:141996)出发，我们推导出了受[拓扑保护](@entry_id:145388)的简并[基态](@entry_id:150928)和奇异的[任意子](@entry_id:143753)激发，并最终将这些微观粒子的代数性质（如[量子维度](@entry_id:146936)、[融合与编织](@entry_id:140952)）与系统的宏观[拓扑不变量](@entry_id:138526)（如 GSD 和[拓扑纠缠熵](@entry_id:145064)）联系在一起，展示了代数、几何与量子多体物理之间深刻而美丽的交织。
