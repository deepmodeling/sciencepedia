## 引言
[X射线衍射](@entry_id:147790)是揭示物质微观结构的核心工具，它如同一双能够洞察原子[排列](@entry_id:136432)的“眼睛”，为固态物理、[材料科学](@entry_id:152226)及众多相关领域奠定了基石。从基础的[晶格参数](@entry_id:191810)测定到复杂的[蛋白质结构](@entry_id:140548)解析，衍射现象提供了连接原子尺度世界与宏观[材料性质](@entry_id:146723)的关键桥梁。然而，面对[劳厄法](@entry_id:185410)、转晶法、[粉末衍射](@entry_id:157495)等多种多样的实验技术，初学者往往难以建立起它们之间内在的逻辑联系，以及它们如何统一在布拉格定律和[结构因子](@entry_id:158623)等共同的物理原理之下。理解这些方法各自的优势、局限以及它们如何从不同角度探测晶体信息，是有效运用衍射技术解决实际问题的关键。

本文旨在系统性地梳理[晶体衍射](@entry_id:139605)的核心实验方法。在第一章“原理与机制”中，我们将深入探讨衍射发生的基本条件、决定衍射强度的[结构因子](@entry_id:158623)，以及各种实验方法的几何构型。随后的第二章“应用与跨学科联系”将通过丰富的实例展示这些方法如何用于[物相鉴定](@entry_id:159361)、缺陷分析、[相变](@entry_id:147324)研究，并触及其在结构生物学等前沿领域的应用。最后，第三章“动手实践”部分提供了具体的计算练习，旨在巩固理论知识并培养解决实际衍射问题的能力。通过这一结构化的学习路径，读者将能够构建一个关于[晶体衍射](@entry_id:139605)实验的完整知识框架，从基本原理出发，掌握核心应用，并最终具备解决实际问题的能力。让我们首先进入第一章，探索支撑所有衍射技术的普适原理与机制。

## 原理与机制

晶体对[X射线](@entry_id:187649)、中子或电子等波的衍射现象，是揭示其内部原子尺度结构的基石。虽然衍射实验技术多种多样，但它们都遵循共同的物理原理。本章旨在深入阐述这些核心原理与机制，从基本的[衍射条件](@entry_id:160781)到衍射强度的定量分析，为理解和应用各种衍射方法，如[劳厄法](@entry_id:185410)、转晶法和粉末法，奠定坚实的理论基础。

### 衍射的基本条件：劳厄方程与[布拉格定律](@entry_id:140129)

[晶体衍射](@entry_id:139605)的本质是波与周期性[排列](@entry_id:136432)的散射中心（原子）发生的[相干散射](@entry_id:267724)。当散射波在特定方向上满足相长干涉条件时，便会形成可观测的衍射束。描述这一现象存在两种等效但视角不同的物理图像：基于倒易空间的劳厄方程和基于真[实空间](@entry_id:754128)晶面的[布拉格定律](@entry_id:140129)。

从[波动光学](@entry_id:271428)的角度看，一个更基本、更普适的[衍射条件](@entry_id:160781)来自于对[晶格](@entry_id:196752)周期性的傅里叶分析。晶体的周期性势场可以用一系列[倒易晶格矢量](@entry_id:263351) $\mathbf{G}$ 来描述。当一束平面波（入射[波矢](@entry_id:178620)为 $\mathbf{k}$）被晶体散射后（散射波矢为 $\mathbf{k}'$），发生相长干涉的条件是[散射矢量](@entry_id:262662) $\Delta\mathbf{k} = \mathbf{k}' - \mathbf{k}$ 必须等于一个[倒易晶格矢量](@entry_id:263351) $\mathbf{G}$。这便是**劳厄方程**：

$$
\mathbf{k}' - \mathbf{k} = \mathbf{G}
$$

同时，对于[弹性散射](@entry_id:152152)（即波的能量或频率不变），波矢的大小保持不变，即 $|\mathbf{k}| = |\mathbf{k}'| = 2\pi/\lambda$，其中 $\lambda$ 是波长。劳厄方程可以被理解为波在与周期性[晶格](@entry_id:196752)相互作用过程中的“晶体动量”守恒，其中[晶格](@entry_id:196752)整体吸收或提供了一个大小为 $\hbar\mathbf{G}$ 的动量。这个描述是[衍射理论](@entry_id:167098)的基石，因为它直接源于晶体的[平移对称性](@entry_id:171614)，并适用于所有[晶系](@entry_id:137271) [@problem_id:2492860]。

虽然劳厄方程具有根本性，但在许多实际应用中，一个更直观的几何图像——**布拉格定律**——则更为便捷。布拉格将衍射视为[X射线](@entry_id:187649)在晶体中一系列相互平行的原子平面（[晶面](@entry_id:166481)）上的“镜面反射”。设[晶面间距](@entry_id:138338)为 $d$，入射束与晶面间的夹角为 $\theta$（布拉格角）。当相邻晶面反射的两束光的路[程差](@entry_id:201533)等于波长的整数倍时，就会发生[相长干涉](@entry_id:276464)。这个路程差为 $2d\sin\theta$。因此，[布拉格定律](@entry_id:140129)可以写作：

$$
n\lambda = 2d\sin\theta
$$

其中 $n$ 是一个正整数，称为[衍射级](@entry_id:174263)数。

[布拉格定律](@entry_id:140129)实际上是劳厄方程在特定几何下的一个推论。我们可以证明这一点：从劳厄方程 $\mathbf{k}' = \mathbf{k} + \mathbf{G}$ 出发，由于 $|\mathbf{k}'|^2 = |\mathbf{k}|^2$，两边平方得到 $|\mathbf{k}|^2 + |\mathbf{G}|^2 + 2\mathbf{k}\cdot\mathbf{G} = |\mathbf{k}|^2$，化简得 $|\mathbf{G}|^2 = -2\mathbf{k}\cdot\mathbf{G}$。根据[倒易晶格](@entry_id:136718)的定义，矢量 $\mathbf{G}$ 的方向垂直于其对应的[晶面](@entry_id:166481)族 $(hkl)$，其大小为 $|G_{hkl}| = 2\pi n/d_{hkl}$（这里将[衍射级](@entry_id:174263)数 $n$ 吸收到 $\mathbf{G}$ 的定义中）。几何关系表明，$\mathbf{k}$ 与 $\mathbf{G}$ 之间的夹角为 $\pi/2 + \theta$。因此，$\mathbf{k}\cdot\mathbf{G} = |\mathbf{k}||\mathbf{G}|\cos(\pi/2+\theta) = -|\mathbf{k}||\mathbf{G}|\sin\theta$。代入化简后的方程，我们得到 $|\mathbf{G}| = 2|\mathbf{k}|\sin\theta$。最后，将 $|\mathbf{G}| = 2\pi/d$ 和 $|\mathbf{k}|=2\pi/\lambda$ 代入（此处取 $n=1$），便可重新得到布拉格定律 $\lambda = 2d\sin\theta$。

这两种描述各有其用。劳厄方程在理论分析和[单晶衍射](@entry_id:198678)的倒易空间构建中至关重要。而[布拉格定律](@entry_id:140129)则极其适用于[粉末衍射](@entry_id:157495)的分析，因为它直接将实验中可测量的衍射角 $2\theta$ 与晶体的一个核心结构参数——[晶面间距](@entry_id:138338) $d$ 联系起来，是进行[物相鉴定](@entry_id:159361)和晶胞参数计算的第一步 [@problem_id:2492860]。

### 结构因子：晶胞内部原子排布的指纹

劳厄方程或布拉格定律决定了衍射峰 *可能* 出现的位置，这些位置仅由[晶格](@entry_id:196752)的几何形状和尺寸（即单位[晶胞](@entry_id:143489)的形状和大小）决定。然而，每个衍射峰的 *强度* 则取决于单位晶胞内部原子的种类和具体排布。描述这种依赖关系的物理量是**结构因子**（Structure Factor），记为 $F_{hkl}$。

对于一个包含 $j$ 个原子的单位[晶胞](@entry_id:143489)，其位于分数坐标 $(x_j, y_j, z_j)$ 处，结构因子定义为：

$$
F_{hkl} = \sum_{j} f_j e^{2\pi i (hx_j + ky_j + lz_j)}
$$

这里，$(hkl)$ 是衍射[晶面](@entry_id:166481)的[密勒指数](@entry_id:138901)，$f_j$ 是第 $j$ 个原子的**[原子散射因子](@entry_id:197944)**（Atomic Scattering Factor），它描述单个原子对[波的散射](@entry_id:202024)能力，并依赖于[散射角](@entry_id:171822)和波长。指数项则代表了来自不同原子位置的散射波之间的相位关系。衍射强度 $I_{hkl}$ 正比于结构因子模的平方，即 $I_{hkl} \propto |F_{hkl}|^2$。

结构因子的一个重要推论是**系统性消光**（Systematic Absences）。在某些情况下，由于[晶胞](@entry_id:143489)内部的对称性，特定 $(hkl)$ 指数组合的[结构因子](@entry_id:158623)会恒等于零，导致相应的衍射峰完全消失。这些消光规律是判断[晶格类型](@entry_id:269667)的关键依据。例如：
- **体心立方（BCC）** [晶格](@entry_id:196752)：原子位于 $(0,0,0)$ 和 $(\frac{1}{2},\frac{1}{2},\frac{1}{2})$。其结构因子为 $F_{hkl} = f[1 + e^{\pi i(h+k+l)}]$。只有当 $h+k+l$ 为偶数时，$F_{hkl} = 2f$；若为奇数，则 $F_{hkl} = 0$。因此，[BCC晶格](@entry_id:146999)的衍射指标必须满足 $h+k+l = \text{偶数}$。
- **面心立方（FCC）** [晶格](@entry_id:196752)：原子位于 $(0,0,0)$, $(\frac{1}{2},\frac{1}{2},0)$, $(\frac{1}{2},0,\frac{1}{2})$, 和 $(0,\frac{1}{2},\frac{1}{2})$。计算表明，只有当[密勒指数](@entry_id:138901) $h, k, l$ 全为奇数或全为偶数时，$F_{hkl}$ 才不为零。这被称为FCC的消光规律。

在[粉末衍射](@entry_id:157495)实验中，可以通过分析一系列衍射峰对应的 $\sin^2\theta$ 值的比例来推断[晶格类型](@entry_id:269667)。对于立方晶系，$\sin^2\theta = \frac{\lambda^2}{4a^2}(h^2+k^2+l^2)$。不同[晶格类型](@entry_id:269667)其允许的 $(h^2+k^2+l^2)$ 值序列是特征性的。例如，简单立方（SC）的序列是1, 2, 3, 4, 5, 6...；BCC的序列是2, 4, 6, 8, 10...；FCC的序列是3, 4, 8, 11, 12...。通过比较实验测得的第一个和第二个衍射峰的 $\sin^2\theta$ 值的比值，就可以有力地鉴别出晶体的布拉维格子类型 [@problem_id:100505]。

当单位晶胞中包含多种类型的原子（即存在一个基元）时，[结构因子](@entry_id:158623)的计算变得更加丰富。例如，在[闪锌矿](@entry_id:159841)（ZnS）结构中，它是一个带双原子基元的FCC[晶格](@entry_id:196752)。我们可以将A类原子（如S）置于FCC格点上，位置为 $(0,0,0)$，B类原子（如Zn）则位于相对位移为 $a(\frac{1}{4},\frac{1}{4},\frac{1}{4})$ 的位置。此时[结构因子](@entry_id:158623)为 $F_{hkl} = f_A S_{hkl}^A + f_B S_{hkl}^B$，其中 $S^A$ 和 $S^B$ 分别是与A原子和B原子相关的相位和。具体而言，对于ZnS结构，$F_{hkl} = f_A(\text{FCC项}) + f_B e^{2\pi i (h/4+k/4+l/4)}(\text{FCC项})$。即使对于FCC允许的反射，其强度也会因为 $f_A$ 和 $f_B$ 的干涉而受到调制。例如，对于 $(220)$ 反射，$h+k+l=4$，指数项为 $e^{2\pi i} = 1$，于是 $F_{220}$ 正比于 $(f_A+f_B)$。而对于 $(111)$ 反射，$h+k+l=3$，指数项为 $e^{3\pi i/2}=-i$，于是 $F_{111}$ 正比于 $(f_A - i f_B)$。因此，这两个反射的强度比为 $|F_{220}|^2/|F_{111}|^2 = (f_A+f_B)^2 / (f_A^2+f_B^2)$，这直接反映了基元内部的原子构成 [@problem_id:100494]。

### 实验方法及其几何特征

不同的衍射实验技术通过控制入射波（单色或连续谱）和样品状态（单晶或粉末，静止或转动）来满足[衍射条件](@entry_id:160781)，从而获得不同形式的衍射图样，每种图样都蕴含着特定的[晶体结构](@entry_id:140373)信息。

#### [劳厄法](@entry_id:185410)

[劳厄法](@entry_id:185410)的特点是使用**连续谱（白色）[X射线](@entry_id:187649)**照射**静止的单晶**。在这种情况下，波长 $\lambda$ 是连续变化的。对于晶体中的每一组晶面 $(hkl)$，总能从连续谱中找到一个合适的波长 $\lambda$ 来满足布拉格定律。因此，[劳厄法](@entry_id:185410)可以同时记录下来自许多不同晶面的衍射斑点。其主要用途是**测定[晶体对称性](@entry_id:198772)和取向**。

劳厄衍射斑点的[几何分布](@entry_id:154371)具有高度的规律性。例如，在背反射劳厄实验中，[X射线](@entry_id:187649)束穿过探测器中心的孔洞入射到晶体上，衍射束向后散射到探测器上。通过测量衍射斑点在探测器上的坐标 $(X, Y)$ 以及样品到探测器的距离 $L$，就可以精确计算出产生该衍射的晶面[法线](@entry_id:167651) $\mathbf{n}$ 的空间取向。晶面[法线](@entry_id:167651)的取向可以由其相对于坐标轴的极角 $\phi_n$ 和[方位角](@entry_id:164011) $\psi_n$ 描述。可以推导出，法线的极角与实验可测量之间的关系为 $\tan(\phi_n) = \frac{\sqrt{X^2+Y^2}}{L+\sqrt{X^2+Y^2+L^2}}$ [@problem_id:100548]。

劳厄图样中一个显著的特征是，来自同一个**晶带**（zone）的衍射斑点会[排列](@entry_id:136432)在特定的曲线上。一个晶带是指所有平行于同一个方向（晶带轴 $\mathbf{u}$）的[晶面](@entry_id:166481)的集合。在倒易空间中，这些[晶面](@entry_id:166481)的倒易格点 $\mathbf{G}$ 都位于一个垂直于晶带轴 $\mathbf{u}$ 的平面上，即满足 $\mathbf{G} \cdot \mathbf{u} = 0$。可以证明，来自同一晶带的所有衍射束构成一个以晶体为顶点、以晶带轴为对称轴的圆锥。当这个衍射锥与一个平面探测器相交时，其轨迹在背反射几何下通常是一条**双曲线**。通过对这些双曲线进行分析，可以确定出晶体的主要[对称轴](@entry_id:177299) [@problem_id:100546]。

#### 转晶法

转晶法使用**单色[X射线](@entry_id:187649)**照射**绕某一主轴旋转的单晶**。旋转样品使得不同的晶面 $(hkl)$ 能够轮流满足[布拉格定律](@entry_id:140129)。这种方法的主要目的是**测定晶格常数**。

当晶体绕着一个[主轴](@entry_id:172691)（例如 $c$ 轴）旋转时，[衍射条件](@entry_id:160781)在垂直于[旋转轴](@entry_id:187094)的方向上受到一个特殊的约束。设旋转轴为 $z$ 轴，入射束沿 $x$ 轴。劳厄方程的 $z$ 分量变为 $k'_z - k_z = G_z$。由于入射束垂直于旋转轴，$k_z=0$。而倒易格矢的 $z$ 分量为 $G_z = 2\pi n/c$，其中 $c$ 是沿[旋转轴](@entry_id:187094)的晶格常数，$n$ 是一个整数。因此，[衍射条件](@entry_id:160781)要求 $k'_z = 2\pi n/c$。

这意味着，对于一个给定的整数 $n$，所有可能的衍射[波矢](@entry_id:178620) $\mathbf{k}'$ 的末端都必须位于倒易空间中一系列平行于 $xy$ 平面的圆上。相应地，在真实空间中，所有对应于同一个 $n$ 值的衍射束都形成一个以旋转轴为中心轴的圆锥。这些圆锥与垂直于[旋转轴](@entry_id:187094)放置的圆柱形胶片相交，形成一系列平行的直线，称为**层线**（layer lines）。如果使用平面探测器，例如置于 $x=D$ 处，这些衍射锥与平面的交线则是一系列**双曲线**。第 $n$ 条层线在探测器上的坐标 $(Y,Z)$ 满足方程 $Z^2 = \frac{n^2\lambda^2(D^2+Y^2)}{c^2-n^2\lambda^2}$。通过测量层线之间的间距，就可以直接、精确地计算出沿旋转轴的晶格常数 $c$ [@problem_id:100576]。

#### [粉末衍射](@entry_id:157495)法

[粉末衍射](@entry_id:157495)法使用**单色[X射线](@entry_id:187649)**照射由大量随机取向的微小晶粒组成的**多晶或粉末样品**。由于样品中包含了所有可能的晶体取向，对于任何一组[晶面间距](@entry_id:138338)为 $d_{hkl}$ 的晶面，总有部分晶粒恰好处于满足[布拉格定律](@entry_id:140129)的取向。因此，来自这些晶粒的衍射束会形成一个以入射束为轴、[半顶角](@entry_id:177010)为 $2\theta$ 的圆锥，称为**德拜-谢乐锥**（Debye-Scherrer cone）。探测器记录下这些衍射锥与探测器表面相交的位置，最终得到一张衍射强度随[散射角](@entry_id:171822) $2\theta$ 变化的图谱，即[粉末衍射](@entry_id:157495)图。

[粉末衍射](@entry_id:157495)法是[材料科学](@entry_id:152226)中最常用的结构分析技术，主要用于**[物相鉴定](@entry_id:159361)**和**晶胞参数的精确测定**。通过测量一系列衍射峰的位置 $2\theta_i$，利用布拉格定律计算出相应的 $d_i$ 值。然后，通过一个称为**指标化**（indexing）的过程，为每个 $d_i$ 值赋予一组[密勒指数](@entry_id:138901) $(hkl)$，并确定一个能够解释所有衍射峰的晶胞参数。如前所述，对于立方[晶系](@entry_id:137271)，可以通过分析 $\sin^2\theta$ 值的比值来快速确定[晶格类型](@entry_id:269667)并进行指标化 [@problem_id:100505]。

### 从衍射强度到[晶体结构](@entry_id:140373)：必要的修正因子

为了从实验测量的原始[衍射积分](@entry_id:182089)强度 $I_{obs}$ 中提取出与[晶体结构](@entry_id:140373)直接相关的物理量 $|F_{hkl}|^2$，必须对一系列与实验几何、散射物理以及样品本身性质相关的因素进行校正。一个完整的衍射强度公式可以近似写为：

$$
I_{obs}(hkl) \propto |F_{hkl}|^2 \cdot m_{hkl} \cdot P(\theta) \cdot L(\theta) \cdot A(\theta) \cdot e^{-2W}
$$

要获得与 $|F_{hkl}|^2$ 成正比的数值，就需要将测得的积分强度除以其他所有因子 [@problem_id:2492893]。

- **[多重性](@entry_id:136466)因子 $m_{hkl}$**：在[粉末衍射](@entry_id:157495)中，由于晶粒的随机取向，所有对称等价的[晶面](@entry_id:166481)族（例如立方[晶系](@entry_id:137271)中的 $\{100\}$ 族包含 $(100), (\bar{1}00), (010)$ 等6个[晶面](@entry_id:166481)）都会在同一个 $2\theta$ 角产生衍射。[多重性](@entry_id:136466)因子就是计算这些等价晶面族的数量，它取决于 $(hkl)$ [指数和](@entry_id:199860)晶体的[点群对称性](@entry_id:141230)。

- **偏振因子 $P(\theta)$**：[X射线](@entry_id:187649)是[电磁波](@entry_id:269629)。当非偏振的入射[X射线](@entry_id:187649)束发生散射时，其强度会因偏振效应而随[散射角](@entry_id:171822)变化。对于来自实验室[X光](@entry_id:187649)管的非偏振光束，在无分析晶的情况下，偏振因子为 $P(\theta) = \frac{1+\cos^2(2\theta)}{2}$。该因子使得高角度的衍射强度相对减弱。

- **洛伦兹因子 $L(\theta)$**：这是一个纯粹的几何因子，它说明了在衍射仪扫描过程中，一个倒易格点停留在[衍射条件](@entry_id:160781)（即与[厄瓦尔德球](@entry_id:146867)相交）上的“机会”或“时间”。其具体形式依赖于衍射几何。对于最常见的布拉格-布伦坦诺（Bragg-Brentano）对称聚焦几何的[粉末衍射](@entry_id:157495)仪，洛伦兹因子为 $L(\theta) \propto \frac{1}{\sin^2\theta\cos\theta}$。

- **吸收因子 $A(\theta)$**：[X射线](@entry_id:187649)在样品内部的穿透过程中会发生吸收，导致来自样品深处的衍射信号被衰减。在对称的布拉格-布伦坦诺几何中，[X射线](@entry_id:187649)在样品中走过的总路程与入射角 $\theta$ 有关。可以证明，这种几何下的吸收效应导致了一个角度依赖的**平均[穿透深度](@entry_id:136478)** $\langle z \rangle = \frac{\sin\theta}{2\mu}$，其中 $\mu$ 是材料的线性吸收系数。这意味着在低角度（小 $\theta$）时，衍射信号主要来自样品表面非常薄的一层，而在高角度时，[X射线](@entry_id:187649)可以穿透得更深。对于厚样品，这个吸收因子通常被认为是一个与角度无关的常数，并被并入总的标度因子中，但在处理薄膜或有特殊表面层的样品时，必须仔细考虑其角度依赖性 [@problem_id:100593]。

- **[德拜-瓦勒因子](@entry_id:142464)（[温度因子](@entry_id:190408)）$e^{-2W}$**：[晶格](@entry_id:196752)中的原子并非静止不动，而是在其平衡位置附近进行热[振动](@entry_id:267781)。这种无规的热位移破坏了[晶格](@entry_id:196752)的完美周期性，使得相长干涉的条件不能被完美满足，从而导致衍射峰的强度被削弱，尤其是在高角度（对应大的[散射矢量](@entry_id:262662) $G$）时。德拜-瓦勒指数 $W$ 正比于原子沿[散射矢量](@entry_id:262662)方向的[均方位移](@entry_id:159665) $\langle u^2 \rangle$。在[高温近似](@entry_id:154509)下，$W$ 与温度 $T$ 成正比，与原子质量 $M$ 成反比。这意味着，使用不同同位素构成的样品会因为原子质量的差异而表现出可测量的衍射强度差异，这为研究[晶格动力学](@entry_id:145448)提供了精细的手段 [@problem_id:100418]。

### [仪器展宽](@entry_id:203159)与峰形分析

在理想情况下，一个衍射峰应该是一个没有宽度的狄拉克 $\delta$ 函数。然而，在实际测量中，衍射峰总具有一定的宽度和特定的形状（峰形）。这部分由样品的微观结构（如[晶粒尺寸](@entry_id:161460)、微观应变）决定，另一部分则来自于仪器本身的不完美性，即**[仪器展宽](@entry_id:203159)**。

[仪器展宽](@entry_id:203159)的来源包括[X射线源](@entry_id:268482)的尺寸、狭缝的宽度、探测器的分辨率以及单色器的性质等。一个重要的发现是，[仪器展宽](@entry_id:203159)本身是[散射角](@entry_id:171822) $2\theta$ 的函数。描述这种角度依赖性的经典模型是**Caglioti方程**，它将峰宽的平方（通常用半高宽 FWHM 的平方来表示）描述为 $\tan\theta$ 的二次多项式：$H^2 = U\tan^2\theta + V\tan\theta + W$。

这个经验公式背后有其深刻的物理根源。以由晶体单色器和准直系统构成的光路为例，单色器本身就是一个衍射元件，它会引入光束的角度发散度和波长散度之间的耦合。可以证明，入射光束的角度发散[分布](@entry_id:182848)（[方差](@entry_id:200758)为 $\sigma_i^2$）和探测系统的角度接收函数（[方差](@entry_id:200758)为 $\sigma_f^2$）共同导致了衍射峰角度[方差](@entry_id:200758) $\sigma_{2\theta}^2$ 的角度依赖性。在[小角度近似](@entry_id:145423)下，可以推导出该[方差](@entry_id:200758)与布拉格角 $\theta$ 的关系，其形式恰好是 $\tan\theta$ 的一个二次多项式，从而为Caglioti方程提供了物理基础 [@problem_id:100607]。对[仪器展宽](@entry_id:203159)的精确表征是进行峰形分析（例如，使用[Rietveld精修](@entry_id:147438)法）以提取样品微观结构信息的先决条件。
