## 引言
整数量子霍尔效应（IQHE）是凝聚态物理学中的一个里程碑式发现，它揭示了一种在宏观尺度上精确展现的深刻量子行为。当二维电子系统处于强[磁场](@entry_id:153296)和低温条件下时，其[霍尔电阻](@entry_id:141556)并非如经典理论预测那样连续变化，而是呈现出一系列被精确量子化的平台，其精度之高，使其成为现代计量学的基础。这种出人意料的量子化平台现象，以及其对材料细节和无序不敏感的惊人鲁棒性，向经典电子输运理论提出了严峻挑战，并开启了利用拓扑学思想理解物质形态的新纪元。

本文旨在系统性地剖析整数[量子霍尔效应](@entry_id:136283)。在“原理与机制”一章中，我们将从单个电子的量子力学行为出发，建立[朗道能级](@entry_id:144244)的概念，并阐明无序和拓扑不变量在形成量子化平台中的关键作用。随后的“应用与跨学科关联”一章将展示该效应如何作为电阻基准，并如何在石墨烯等新材料中展现出新的物理，同时揭示其与理论物理中更广泛概念的深刻联系。最后，通过“动手实践”部分，读者将有机会通过具体的计算问题，加深对朗道能级、陈数和拓扑保护等核心概念的理解。

## 原理与机制

在本章中，我们将深入探讨整数[量子霍尔效应](@entry_id:136283)（IQHE）背后的基本物理原理和机制。我们将从单个电子在[磁场](@entry_id:153296)中的量子力学行为出发，建立朗道能级的概念。随后，我们将探讨在真实材料中无序所起的关键作用，正是无序导致了霍尔[电导](@entry_id:177131)平台的形成。最后，我们将揭示该效应深刻的拓扑本质，阐明为何霍尔[电导](@entry_id:177131)的量子化精度如此之高，并对扰动具有惊人的鲁棒性。

### 二维电子在[磁场](@entry_id:153296)中的量子力学

理解整数[量子霍尔效应](@entry_id:136283)的第一步是分析[二维电子气](@entry_id:146876)体（2DEG）中单个电子在垂直强[磁场](@entry_id:153296)下的行为。其经典对应物是电子在[洛伦兹力](@entry_id:145104)作用下进行的[回旋运动](@entry_id:204632)。在量子力学中，这种运动的能量是量子化的，形成了分立的能级，即**[朗道能级](@entry_id:144244)**。

#### 朗道能级与回旋频率

我们考虑一个质量为 $m$、[电荷](@entry_id:275494)为 $-e$（其中 $e>0$）的电子，被限制在 $xy$ 平面内。一个均匀的静[磁场](@entry_id:153296) $\mathbf{B} = B\hat{\mathbf{z}}$ 垂直于该平面。通过[最小耦合](@entry_id:148226)原理，[正则动量](@entry_id:155151) $\mathbf{p}$ 被替换为力学动量 $\boldsymbol{\Pi} = \mathbf{p} + e\mathbf{A}$，其中 $\mathbf{A}$ 是[磁场](@entry_id:153296)的矢量势（$\nabla \times \mathbf{A} = \mathbf{B}$）。系统的[哈密顿量](@entry_id:172864)为：

$$H = \frac{1}{2m}\boldsymbol{\Pi}^2 = \frac{1}{2m}(\mathbf{p} + e\mathbf{A})^2$$

为了求解其[能谱](@entry_id:181780)，我们需要选择一个特定的规范。**朗道规范**是一个便捷的选择，我们令 $\mathbf{A} = (0, Bx, 0)$。在此规范下，[哈密顿量](@entry_id:172864)变为：

$$H = \frac{1}{2m} [p_x^2 + (p_y + eBx)^2]$$

由于该[哈密顿量](@entry_id:172864)不显含坐标 $y$，它与 $y$ 方向的[动量算符](@entry_id:151743) $p_y$ 对易，即 $[H, p_y] = 0$。这意味着我们可以寻找 $H$ 和 $p_y$ 的共同本征态。这样的本征态可以写成[平面波](@entry_id:189798)与一个只依赖于 $x$ 的函数的乘积形式：$\psi(x,y) = e^{ik_y y} \phi(x)$，其中 $\hbar k_y$ 是 $p_y$ 的[本征值](@entry_id:154894)。

将此[波函数](@entry_id:147440)代入定态薛定谔方程 $H\psi = E\psi$，我们得到一个只关于 $\phi(x)$ 的一维方程：

$$\left[ \frac{p_x^2}{2m} + \frac{1}{2m}(\hbar k_y + eBx)^2 \right] \phi(x) = E \phi(x)$$

通过整理，上式可以写为：

$$\left[ \frac{p_x^2}{2m} + \frac{1}{2} m \left(\frac{eB}{m}\right)^2 \left( x + \frac{\hbar k_y}{eB} \right)^2 \right] \phi(x) = E \phi(x)$$

这正是一个一维量子谐振子的薛定谔方程。通过与标准形式 $H_{QHO} = \frac{P^2}{2M} + \frac{1}{2} M \omega^2 (X - X_0)^2$ 对比，我们可以确定该谐振子的角频率为**回旋频率** $\omega_c$：

$$\omega_c = \frac{eB}{m}$$

[量子谐振子](@entry_id:140678)的[能量本征值](@entry_id:144381)是众所周知知的，为 $E_n = (n + \frac{1}{2})\hbar\omega_c$，其中 $n=0, 1, 2, \dots$ 是一个非负整数，称为**[朗道能级](@entry_id:144244)指数**。因此，二维电子在[磁场中的能量](@entry_id:262427)谱是一系列分立的[朗道能级](@entry_id:144244) [@problem_id:2830156]：

$$E_n = \left(n + \frac{1}{2}\right) \hbar \omega_c = \left(n + \frac{1}{2}\right) \frac{\hbar e B}{m}$$

值得注意的是，[能量本征值](@entry_id:144381) $E_n$ 与连续变化的[量子数](@entry_id:145558) $k_y$ 无关。这意味着每个[朗道能级](@entry_id:144244)都是高度简并的。谐振子中心位置的平移 $x_0 = -\hbar k_y / (eB)$ 仅移动了[波函数](@entry_id:147440)在 $x$ 方向上的中心，而不影响能量。

在真实的晶体材料中，电子的[有效质量](@entry_id:142879)可能具有各向异性，例如，在 $x$ 和 $y$ 方向上具有不同的有效质量 $m_x$ 和 $m_y$。在这种情况下，可以通过对力学动量进行适当的缩放变换，将[哈密顿量](@entry_id:172864)重新化为一个具有等效各向同性质量 $m^* = \sqrt{m_x m_y}$ 的谐振子问题。这导致[回旋频率](@entry_id:156231)变为 $\omega_c = eB/\sqrt{m_x m_y}$ [@problem_id:2996106]。

#### 磁长度与[朗道能级简并](@entry_id:154452)度

力学动量算符的分量之间存在一个基本的[对易关系](@entry_id:136780)：

$$[\Pi_x, \Pi_y] = [p_x + eA_x, p_y + eA_y] = -i\hbar eB$$

这个关系独立于规范选择和电子质量，反映了[磁场](@entry_id:153296)施加在量子力学坐标上的内在几何结构。这个非[对易关系](@entry_id:136780)意味着我们无法同时精确测量电子的 $x$ 和 $y$ 方向的力学动量。它也引出了一个基本的长度尺度，即**磁长度** $\ell_B$，其定义为：

$$\ell_B = \sqrt{\frac{\hbar}{eB}}$$

磁长度代表了在强[磁场](@entry_id:153296)中电子[波函数](@entry_id:147440)所能被局域的最小尺度，可以看作是[基态](@entry_id:150928)朗道[轨道](@entry_id:137151)（$n=0$）的半径。

现在我们回到朗道能级的简并度问题。简并度是指在每个朗道能级上可以容纳多少个独立的[量子态](@entry_id:146142)。考虑一个面积为 $A = L_x L_y$ 的矩形样品，并施加[周期性边界条件](@entry_id:147809)。沿 $y$ 方向的[周期性边界条件](@entry_id:147809)要求[波矢](@entry_id:178620) $k_y$ 是量子化的：$k_y = \frac{2\pi j}{L_y}$，其中 $j$ 是整数。

如前所述，[波函数](@entry_id:147440)在 $x$ 方向的中心位置 $x_0 = -\hbar k_y / (eB)$。为了使电子的[波函数](@entry_id:147440)位于样品内部，其中心位置必须在 $0 \le x_0 \le L_x$ 的范围内。这反过来限制了 $k_y$ 的取值范围。$k_y$ 的允许总范围是 $\Delta k_y = eBL_x/\hbar$。由于 $k_y$ 的取值是分立的，步长为 $2\pi/L_y$，因此在一个朗道能级内，独立状态（即不同的 $k_y$ 值）的总数 $N_\phi$ 为：

$$N_\phi = \frac{\Delta k_y}{2\pi/L_y} = \frac{eBL_x/\hbar}{2\pi/L_y} = \frac{eB L_x L_y}{2\pi\hbar} = \frac{eB A}{h}$$

其中 $h=2\pi\hbar$ 是普朗克常数。这个数字 $N_\phi$ 也等于穿过样品面积 $A$ 的磁通量 $\Phi = BA$ 与**[磁通量子](@entry_id:136429)** $\Phi_0 = h/e$ 的比值。因此，每个朗道能级的单位面积简并度 $g_B$ 为 [@problem_id:2830109]：

$$g_B = \frac{N_\phi}{A} = \frac{eB}{h}$$

这个结果至关重要。它表明，每个[朗道能级](@entry_id:144244)可以容纳的电子数与[磁场强度](@entry_id:197932)成正比。基于此，我们可以定义**[填充因子](@entry_id:146022)** $\nu$，它表示被电子完全填满的[朗道能级](@entry_id:144244)的数量：

$$\nu = \frac{n}{g_B} = \frac{nh}{eB}$$

其中 $n$ 是二维电子气体的[面密度](@entry_id:161889)。在整数[量子霍尔效应](@entry_id:136283)中，这个[填充因子](@entry_id:146022) $\nu$ 将扮演核心角色，它正是出现在霍尔[电导量子化](@entry_id:138407)表达式中的那个整数。

#### 自旋与[塞曼效应](@entry_id:154144)

到目前为止，我们忽略了电子的内禀自旋。电子自旋与[磁场](@entry_id:153296)之间存在相互作用，即**[塞曼效应](@entry_id:154144)**，其[哈密顿量](@entry_id:172864)为 $H_{\text{spin}} = \frac{g\mu_B}{2}\boldsymbol{\sigma} \cdot \mathbf{B}$，其中 $g$ 是朗德因子，$\mu_B$ 是[玻尔磁子](@entry_id:151037)，$\boldsymbol{\sigma}$ 是[泡利矩阵](@entry_id:139493)矢量。由于[磁场](@entry_id:153296)沿 $z$ 方向，该项简化为 $\frac{g\mu_B B}{2}\sigma_z$。$\sigma_z$ 的[本征值](@entry_id:154894)为 $s = \pm 1$。

因此，每个朗道能级会因自旋分裂成两个子能级。包含[塞曼分裂](@entry_id:156350)的总能谱为 [@problem_id:2830205]：

$$E_{n,s} = \left(n + \frac{1}{2}\right) \frac{\hbar e B}{m} + s \frac{g\mu_B B}{2}$$

在强[磁场](@entry_id:153296)下，[塞曼分裂](@entry_id:156350)可能非常显著。[填充因子](@entry_id:146022) $\nu$ 此时代表被填满的自旋分裂后的[朗道能级](@entry_id:144244)的总数。

### 无序的角色：平台与局域化

在一个理想的、完全纯净的[二维电子气](@entry_id:146876)中，随着[磁场](@entry_id:153296) $B$ 的变化，[填充因子](@entry_id:146022) $\nu = nh/eB$ 会连续变化。根据经典霍尔效应理论，霍尔[电导率](@entry_id:137481) $\sigma_{xy} = ne/B$（或[电阻率](@entry_id:266481) $\rho_{xy} = B/ne$）也应该随 $B$ 连续变化。然而，实验观察到的却是 $\sigma_{xy}$ 在特定值上保持不变，形成宽阔的平台。这一现象的解释，恰恰在于真实材料中不可避免存在的**无序**（如杂质、缺陷等）。

#### 朗道能级的展宽与[迁移率边](@entry_id:145044)

无序势的存在会破坏朗道能级的完美简并，使其展宽成一系列能带。根据安德森局域化理论，在二维强[磁场](@entry_id:153296)下，这些能带中的电子态并非都具有相同的性质。

- **局域态（Localized States）**：位于能带尾部的态，其[波函数](@entry_id:147440)被束缚在空间的有限区域内，无法在整个样品中[传导电流](@entry_id:265343)。
- **[扩展态](@entry_id:138810)（Extended States）**：位于每个展宽能带中心附近的一个或多个离散能量上的态，其[波函数](@entry_id:147440)可以贯穿整个样品，从而能够[传导电流](@entry_id:265343)。

分隔局域态和[扩展态](@entry_id:138810)的能量被称为**[迁移率边](@entry_id:145044)（Mobility Edge）**。在[迁移率边](@entry_id:145044)上，[局域化长度](@entry_id:146276)发散，态的性质从局域转变为扩展 [@problem_id:2830124]。

#### 霍尔平台的形成机制

霍尔平台的出现正是局域态和[扩展态](@entry_id:138810)共同作用的结果。考虑在零温下，逐渐改变[磁场强度](@entry_id:197932) $B$ 或电子密度 $n$，从而改变费米能级 $\mu$ 在能[态密度](@entry_id:147894)中的位置。

1.  当[费米能级](@entry_id:143215) $\mu$ 位于一个**迁移率隙**中，即处于一个只包含局域态的能量区间时，费米面上的所有电子态都是局域的。这些局域态虽然可以容纳电子，但它们就像一个个孤岛，无法对宏观的直流输运做出贡献。因此，纵向[电导率](@entry_id:137481) $\sigma_{xx}$ 趋于零。

2.  更重要的是，当 $\mu$ 在这个局域态区域内移动时，虽然总电子数在变化（电子被局域态“捕获”或“释放”），但被完全填充的、能够承载电流的**[扩展态](@entry_id:138810)能带**的数量保持不变。

3.  霍尔[电导](@entry_id:177131)由这些处于费米能级以下的[扩展态](@entry_id:138810)决定。只要填充的[扩展态](@entry_id:138810)能带数目 $\nu$ 不变，霍尔[电导](@entry_id:177131) $\sigma_{xy}$ 就会被“钉扎”在一个量子化的值上，形成一个平台 [@problem_id:2996097]。

4.  只有当[费米能级](@entry_id:143215)扫过[扩展态](@entry_id:138810)所在的能量（即穿越[迁移率边](@entry_id:145044)）时，填充的[扩展态](@entry_id:138810)能带数目才会发生整数跳变（例如从 $\nu$ 变为 $\nu+1$）。此时，纵向[电导率](@entry_id:137481) $\sigma_{xx}$ 会出现一个峰值，而霍尔[电导](@entry_id:177131) $\sigma_{xy}$ 则从一个平台跃迁到下一个平台。

因此，无序的存在非但不是量子霍尔效应的阻碍，反而是形成稳定、宽阔的霍尔平台的**必要条件**。局域态提供了一个电子“蓄水池”，使得系统可以在一定范围内改变总电子数或[磁场](@entry_id:153296)，而霍尔[电导](@entry_id:177131)保持不变。从实验上测得的[霍尔电阻](@entry_id:141556) $R_{xy}$，我们可以直接推断出[填充因子](@entry_id:146022) $\nu = \frac{h}{e^2 R_{xy}}$，并进一步计算出载流子密度 $n(B) = \nu (eB/h) = B/(eR_{xy}(B))$ [@problem_id:2830181]。

### [拓扑不变性](@entry_id:181048)与量子化

霍尔[电导量子化](@entry_id:138407)值的惊人精度（可达十亿分之一）暗示其背后存在一个深刻的保护机制，使其对样品的具体细节（如几何形状、无序强度）不敏感。这个机制就是**拓扑学**。

#### 劳夫林[规范不变性](@entry_id:137857)论证

物理学家 Robert Laughlin 提出了一个基于[规范不变性](@entry_id:137857)的优雅思想实验，揭示了霍尔[电导量子化](@entry_id:138407)的拓扑根源。

想象将[二维电子气](@entry_id:146876)弯成一个圆柱体，沿 $y$ 方向是周长为 $L_y$ 的闭合环路。现在，我们从圆柱体中心缓慢地穿入一个[磁通量](@entry_id:268943) $\Phi(t)$ [@problem_id:973980]。根据法拉第电磁感应定律，这个变化的磁通量会在圆柱表面感生一个沿 $y$ 方向的环形[电场](@entry_id:194326) $E_y = -\frac{1}{L_y}\frac{d\Phi}{dt}$。这个[电场](@entry_id:194326)会驱动一个沿 $x$ 方向的[霍尔电流](@entry_id:196652) $I_x$。根据霍尔[电导](@entry_id:177131)的定义，我们有 $I_x = \sigma_{xy} E_y L_y = -\sigma_{xy} \frac{d\Phi}{dt}$。

当磁通量从 0 增加到一个磁通量子 $\Phi_0 = h/e$ 时，从量子力学的规范变换角度看，增加一个磁通量子的效果等价于一个“大”[规范变换](@entry_id:176521)。对于一个具有[能隙](@entry_id:191975)的系统（这里的[能隙](@entry_id:191975)即迁移率隙），在这样一个[绝热过程](@entry_id:138150)中，系统最终必须回到一个与其初始[状态等价](@entry_id:261329)的[量子态](@entry_id:146142)。Laughlin 论证，这个过程的唯一物理后果是从圆柱体的一端向另一端精确地输运了整数个电子。

如果费米能级以下有 $\nu$ 个被填满的朗道能级，那么每插入一个[磁通量子](@entry_id:136429)，就会有 $\nu$ 个电子被泵浦过去。因此，总的输运[电荷](@entry_id:275494)为 $\Delta Q = -\nu e$。将这个[电荷](@entry_id:275494)与通过[霍尔电流](@entry_id:196652)的积分得到的[电荷](@entry_id:275494) $\Delta Q = \int I_x dt = \int (-\sigma_{xy}) d\Phi = -\sigma_{xy} \Phi_0$ 相比较，我们得到：

$$-\nu e = -\sigma_{xy} \frac{h}{e}$$

由此可得霍尔[电导](@entry_id:177131)的量子化公式：

$$\sigma_{xy} = \nu \frac{e^2}{h}$$

这个论证的强大之处在于它完全不依赖于[哈密顿量](@entry_id:172864)的具体细节，只需要系统存在一个[能隙](@entry_id:191975)（或迁移率隙）以及基本的规范不变性。这解释了为何量子化如此精确且普适。在此过程中，局域态由于其[波函数](@entry_id:147440)被钉扎在特定位置，对穿过系统中心的磁通变化不敏感，因此不会对泵浦的[电荷](@entry_id:275494)做出贡献，从而保证了泵浦[电荷](@entry_id:275494)的精确量子化 [@problem_id:2830113]。

#### TKNN [不变量](@entry_id:148850)与陈数

Laughlin 的物理图像可以通过一个更严谨的数学框架来描述，这个框架将霍尔[电导](@entry_id:177131)与一个称为**陈数（Chern number）**的拓扑不变量联系起来。这个理论由 Thouless、Kohmoto、Nightingale 和 den Nijs (TKNN) 发展而来。

该理论从**[久保公式](@entry_id:144041)（Kubo formula）**出发，这是计算[输运系数](@entry_id:136790)的[线性响应理论](@entry_id:145737)的基石。TKNN 表明，对于一个在环面（等价于施加双向周期性边界条件的平面）上运动的、具有[能隙](@entry_id:191975)的电子系统，其零温零频的霍尔[电导率](@entry_id:137481) $\sigma_{xy}$ 可以表示为对所有被占据能带的**[贝里曲率](@entry_id:136846)（Berry curvature）**在整个布里渊区（或等效的参数空间）上的积分 [@problem_id:2830171]。这个积分的结果，乘以一个[基本常数](@entry_id:148774)，就是陈数 $C$：

$$\sigma_{xy} = C \frac{e^2}{h}$$

陈数是一个整数，它刻画了电子[波函数](@entry_id:147440)在[动量空间](@entry_id:148936)中的一种“扭曲”或“缠绕”的拓扑性质。只要系统的[能隙](@entry_id:191975)不关闭，任何微小的、连续的扰动（如改变无序势的强度）都无法改变这个整数值。这为霍尔平台的稳定性提供了坚实的数学基础。

对于整数量子霍尔效应，可以证明，每个被填满的[朗道能级](@entry_id:144244)的陈数都精确地等于 1 [@problem_id:2830202]。因此，当有 $\nu$ 个[朗道能级](@entry_id:144244)被填满时，总陈数 $C = \nu$。这再次导出了量子化的霍尔[电导](@entry_id:177131)。

我们甚至可以在不涉及[朗道能级](@entry_id:144244)的[晶格模型](@entry_id:184345)中看到陈数的作用。例如，在一个破坏了[时间反演对称性](@entry_id:138094)的二维[紧束缚模型](@entry_id:143446)中，如果其[能带结构](@entry_id:139379)是拓扑非平庸的（即陈数非零），那么当[费米能级](@entry_id:143215)位于[能隙](@entry_id:191975)中时，系统就会表现出整数[量子霍尔效应](@entry_id:136283) [@problem_id:2996089]。这表明 IQHE 是一种更广泛的拓扑[物态](@entry_id:139436)现象，不局限于强[磁场](@entry_id:153296)下的朗道能级。

### 高级观点与推论

#### 体-边对应

拓扑学的一个深刻推论是**体-边对应（Bulk-Edge Correspondence）**。该原理指出，一个具有非平庸[拓扑性质](@entry_id:141605)的“体”（bulk）材料，在其与拓扑平庸的材料（如真空）的边界上，必须存在无能隙的边界态。

在整数[量子霍尔效应](@entry_id:136283)的背景下，体态的[拓扑不变量](@entry_id:138526)是陈数 $C=\nu$。这意味着，在一个有限尺寸的样品边缘，必定存在 $\nu$ 个[无能](@entry_id:201612)隙的**[手性边缘态](@entry_id:138111)（chiral edge states）**。这些[边缘态](@entry_id:142513)的[能量色散关系](@entry_id:145014) $E(k)$ 会跨越体[能隙](@entry_id:191975)，并且其传播方向是单向的（手性的），由[磁场](@entry_id:153296)方向决定 [@problem_id:2830217]。例如，在样品的一个边缘，电子只能向右移动，而在相对的边缘，电子只能向左移动。

这些边缘态为量子霍尔效应提供了一个非常直观的输运图像。从这个观点看，当样品两端加上电压时，电流完全由这些无耗散的边缘通道承载，而体态则是绝缘的。**兰道尔-布蒂克（Landauer-Büttiker）**形式论可以用来精确描述这种基于[边缘态](@entry_id:142513)的输运。对于一个具有 $\nu$ 个手性边缘通道的理想霍尔棒，该理论直接导出其霍尔[电导](@entry_id:177131)为 $G_{xy} = \nu e^2/h$ [@problem_id:2830110]，与基于体态的拓扑理论完美契合。

#### 鲁棒性与[非对易](@entry_id:136599)几何

拓扑保护是 IQHE 鲁棒性的根源。只要扰动（无论是无序还是弱的[电子-电子相互作用](@entry_id:139900)）不足以关[闭系](@entry_id:139565)统的[能隙](@entry_id:191975)（即迁移率隙或多体[能隙](@entry_id:191975)），霍尔[电导](@entry_id:177131)的量子化整数值就不会改变 [@problem_id:2830228] [@problem_id:2830238]。这是一个极其强大的结论，解释了为什么 IQHE 可以在各种不同的材料和实验条件下被精确地观测到。

另一个深入探讨 IQHE 物理的视角是**[非对易](@entry_id:136599)几何**。当我们把物理体系投影到最低[朗道能级](@entry_id:144244)（LLL）这个[子空间](@entry_id:150286)时，会发现坐标算符的投影 $\hat{x} = P_0 x P_0$ 和 $\hat{y} = P_0 y P_0$ 之间不再对易。它们的[对易关系](@entry_id:136780)为 [@problem_id:1155395]：

$$[\hat{x}, \hat{y}] = i\ell_B^2$$

这意味着在最低[朗道能级](@entry_id:144244)内，位置坐标本身具有了[量子不确定性](@entry_id:156130)，无法同时被精确测量。这个非对易的“量子平面”是分数[量子霍尔效应](@entry_id:136283)等更复杂多体现象的理论出发点。

#### 斯特reda公式

最后，霍尔[电导](@entry_id:177131)还与系统的一个[热力学性质](@entry_id:146047)密切相关，这由**斯特reda公式（Streda formula）**所揭示 [@problem_id:2996097]：

$$\sigma_{xy} = e \left( \frac{\partial n}{\partial B} \right)_{\mu}$$

这个公式表明，霍尔[电导](@entry_id:177131)衡量的是在保持化学势 $\mu$ 恒定的情况下，电子密度 $n$ 随[磁场](@entry_id:153296) $B$ 变化的速率。当化学势 $\mu$ 位于迁移率隙中时，可以证明 $(\partial n / \partial B)_\mu$ 的值被量子化为 $\nu e/h$。代入 Streda 公式，我们再次得到 $\sigma_{xy} = \nu e^2/h$ [@problem_id:2830105]。这个公式巧妙地将[输运性质](@entry_id:203130)与系统的[热力学](@entry_id:141121)响应联系在一起，为理解量子霍尔效应提供了又一个有力的理论工具。
