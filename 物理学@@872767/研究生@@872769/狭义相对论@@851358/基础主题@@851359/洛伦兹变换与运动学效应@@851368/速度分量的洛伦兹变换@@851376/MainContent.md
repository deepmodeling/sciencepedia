## 引言
在日常经验中，速度的叠加似乎遵循着简单的向量加法，这是牛顿经典力学的基石。然而，当物体以接近光速的速度运动时，这一直观的伽利略变换法则便宣告失效。狭义相对论的诞生彻底改变了我们对时间、空间和运动的理解，其中，洛伦兹[速度变换](@entry_id:265594)构成了其动力学部分的核心。它解决了经典物理无法解释的矛盾，即如何在所有[惯性系](@entry_id:266190)中维持[光速不变](@entry_id:265351)这一基本原理。本文旨在系统性地剖析洛伦兹速度分量变换的完整理论及其深远影响。

为了构建一个全面的理解，本文将分步展开。在第一章“原理与机制”中，我们将从洛伦兹[坐标变换](@entry_id:172727)出发，严谨推导出[速度变换](@entry_id:265594)的各个分量方程，并探讨其基本推论，如爱因斯坦速度叠加法则。接下来，在“应用与跨学科联系”一章中，我们将展示这些方程如何在粒子物理、天体物理学和[相对论光学](@entry_id:193063)等前沿领域中解释复杂的观测现象。最后，“动手实践”部分将提供一系列精心设计的问题，帮助读者巩固理论知识并将其应用于解决实际的物理问题。通过这一结构化的学习路径，读者将掌握分析相对论性运动学的关键工具。

## 原理与机制

在狭义相对论的框架下，一个基础的结论是，时间和空间不再是孤立和绝对的，而是交织成一个统一的四维时空。正如洛伦兹变换描述了不同[惯性参考系](@entry_id:276742)下时空坐标的转换关系一样，物体的速度测量也同样依赖于观测者的运动状态。经典力学中简单的伽利略速度叠加法则（即向量直接相加）在相对论领域不再适用。取而代之的是一套新的、更为深刻的[速度变换](@entry_id:265594)法则，它不仅保证了[光速不变原理](@entry_id:201268)的普适性，也揭示了[时空结构](@entry_id:158931)的诸多奇特效应。本章将系统地阐述洛伦兹[速度变换](@entry_id:265594)的原理与机制，并探讨其在物理学中的重要应用。

### 洛伦兹[速度变换](@entry_id:265594)方程

我们从两个惯性参考系 $S$ 和 $S'$ 开始。假设 $S'$ 系相对于 $S$ 系以恒定速度 $v$ 沿着共同的 $x$ 和 $x'$ 轴正方向运动。一个物体在 $S'$ 系中的速度为 $\vec{u}'$，其分量为 $(u'_x, u'_y, u'_z)$。我们的目标是确定该物体在 $S$ 系中被观测到的速度 $\vec{u}$，其分量为 $(u_x, u_y, u_z)$。

速度的定义是位移除以时间，例如 $u_x = \frac{dx}{dt}$。因此，我们可以通过对洛伦兹坐标变换方程取[微分](@entry_id:158718)来推导[速度变换](@entry_id:265594)关系。洛伦兹变换如下：
$t = \gamma \left( t' + \frac{vx'}{c^2} \right)$
$x = \gamma (x' + vt')$
$y = y'$
$z = z'$
其中 $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$ 是[洛伦兹因子](@entry_id:159588)。

对以上方程取[微分](@entry_id:158718)，我们得到：
$dt = \gamma \left( dt' + \frac{v}{c^2} dx' \right)$
$dx = \gamma (dx' + v dt')$
$dy = dy'$
$dz = dz'$

现在我们可以计算 $S$ 系中的速度分量：
$u_x = \frac{dx}{dt} = \frac{\gamma (dx' + v dt')}{\gamma \left( dt' + \frac{v}{c^2} dx' \right)} = \frac{\frac{dx'}{dt'} + v}{1 + \frac{v}{c^2} \frac{dx'}{dt'}}$
由于 $u'_x = \frac{dx'}{dt'}$，上式变为：
$$u_x = \frac{u'_x + v}{1 + \frac{v u'_x}{c^2}}$$

对于垂直于运动方向的分量，例如 $y$ 分量：
$u_y = \frac{dy}{dt} = \frac{dy'}{\gamma \left( dt' + \frac{v}{c^2} dx' \right)} = \frac{\frac{dy'}{dt'}}{\gamma \left( 1 + \frac{v}{c^2} \frac{dx'}{dt'} \right)}$
由于 $u'_y = \frac{dy'}{dt'}$，我们得到：
$$u_y = \frac{u'_y}{\gamma \left( 1 + \frac{v u'_x}{c^2} \right)}$$
同理可得 $z$ 分量：
$$u_z = \frac{u'_z}{\gamma \left( 1 + \frac{v u'_x}{c^2} \right)}$$

这三条方程构成了从 $S'$ 系到 $S$ 系的**洛伦兹[速度变换](@entry_id:265594)**。要得到从 $S$ 系到 $S'$ 系的**[逆变](@entry_id:192290)换**，我们只需将带撇和不带撇的变量互换，并将 $v$ 替换为 $-v$ 即可：
$$u'_x = \frac{u_x - v}{1 - \frac{v u_x}{c^2}}$$
$$u'_y = \frac{u_y}{\gamma \left( 1 - \frac{v u_x}{c^2} \right)}$$
$$u'_z = \frac{u_z}{\gamma \left( 1 - \frac{v u_x}{c^2} \right)}$$

这些方程是狭义[相对论动力学](@entry_id:264218)的核心，它们取代了直观但错误的伽利略[速度变换](@entry_id:265594) ($u_x = u'_x + v$, $u_y = u'_y$, $u_z = u'_z$)。

### 基本应用与推论

洛伦兹[速度变换](@entry_id:265594)最直接也最重要的推论之一是爱因斯坦速度叠加法则，以及对[光速不变原理](@entry_id:201268)的自洽性检验。

#### 共线速度叠加

最简单的情况是所有运动都发生在同一条直线上（例如 $x$ 轴）。此时 $u'_y = u'_z = 0$，变换方程简化为：
$$u_x = \frac{u'_x + v}{1 + \frac{v u'_x}{c^2}}$$
这个公式被称为**爱因斯坦速度叠加公式**。它表明，即使两个速度都是近光速的，它们的[相对速度](@entry_id:178060)也永远不会超过光速 $c$。

考虑一个在[粒子加速器](@entry_id:148838)中发生的思想实验 [@problem_id:1833347]。一个在实验室参考系 $S$ 中静止的[不稳定粒子](@entry_id:148663)衰变为两个子粒子，粒子1和粒子2。在 $S$ 系中，粒子1以速度 $u_1 = \alpha c$ 沿正 $x$ 方向运动，粒子2以速度 $u_2 = -\beta c$ 沿负 $x$ 方向运动（其中 $\alpha$ 和 $\beta$ 为小于1的正数）。我们想知道在粒子2的[静止参考系](@entry_id:262703) $S'$ 中测得的粒子1的速度 $u'_1$ 是多少。

在这里，[参考系](@entry_id:169232) $S'$（粒子2的静止系）相对于实验室 $S$ 系的速度是 $v = u_2 = -\beta c$。我们需要变换的物体是粒子1，其在 $S$ 系中的速度为 $u = u_1 = \alpha c$。应用逆变换公式：
$$u'_1 = \frac{u_1 - v}{1 - \frac{u_1 v}{c^2}} = \frac{\alpha c - (-\beta c)}{1 - \frac{(\alpha c)(-\beta c)}{c^2}} = \frac{(\alpha + \beta)c}{1 + \alpha \beta}$$
这个结果展示了相对论速度叠加的本质。在经典情况下，[相对速度](@entry_id:178060)是 $(\alpha + \beta)c$。然而，由于分母项 $1 + \alpha\beta$ 的存在，其值总是大于1（因为 $\alpha, \beta > 0$），使得 $u'_1$ 总是小于 $(\alpha + \beta)c$。即使 $\alpha$ 和 $\beta$ 都无限接近1，即两个粒子都以近光速背向飞行，$u'_1$ 的极限也是 $\frac{(1+1)c}{1+1\cdot1} = c$，而不是经典物理预测的 $2c$。这从根本上保证了没有任何物体的[相对速度](@entry_id:178060)可以超过光速。

#### [光速不变性](@entry_id:201268)的自洽性

[洛伦兹变换](@entry_id:176827)本身就是从[光速不变原理](@entry_id:201268)出发构建的，因此[速度变换](@entry_id:265594)法则必须与该原理自洽。我们可以通过一个例子来检验这一点 [@problem_id:1857354]。

假设一个高速介子探测器（[参考系](@entry_id:169232) $S'$）以速度 $v = 0.6c$ 沿实验室参考系 $S$ 的 $x$ 轴正向飞行。探测器上的一个光源发出了一束闪光。在探测器自身的[参考系](@entry_id:169232) $S'$ 中，这束光纯粹沿着 $y'$ 轴正向传播，因此其速度分量为 $u'_x = 0$ 和 $u'_y = c$。现在，我们计算实验室观察者测得的光脉冲速度。

首先，计算[洛伦兹因子](@entry_id:159588) $\gamma$：
$$\gamma = \frac{1}{\sqrt{1 - (0.6)^2}} = \frac{1}{\sqrt{1 - 0.36}} = \frac{1}{\sqrt{0.64}} = \frac{1}{0.8} = 1.25$$

接着，应用[速度变换](@entry_id:265594)公式：
$$u_x = \frac{u'_x + v}{1 + \frac{v u'_x}{c^2}} = \frac{0 + 0.6c}{1 + \frac{(0.6c)(0)}{c^2}} = 0.6c$$
$$u_y = \frac{u'_y}{\gamma \left( 1 + \frac{v u'_x}{c^2} \right)} = \frac{c}{1.25 \left( 1 + \frac{(0.6c)(0)}{c^2} \right)} = \frac{c}{1.25} = 0.8c$$

在实验室参考系 $S$ 中，光脉冲的[速度矢量](@entry_id:269648)为 $\vec{u} = (0.6c, 0.8c)$。现在计算其速率（速度矢量的大小）：
$$|\vec{u}| = \sqrt{u_x^2 + u_y^2} = \sqrt{(0.6c)^2 + (0.8c)^2} = \sqrt{0.36c^2 + 0.64c^2} = \sqrt{1.00c^2} = c$$
结果不多不少，正好是 $c$。这个例子完美地展示了[速度变换](@entry_id:265594)法则如何巧妙地组合速度分量，使得最终的合成速率对于任何惯性观察者来说都是光速 $c$，从而维护了狭义相对论的第二条基本公设。

### 多维空间中的[速度变换](@entry_id:265594)

当物体的运动不局限于与[参考系](@entry_id:169232)相对运动的轴线时，我们需要使用完整的[速度变换](@entry_id:265594)[方程组](@entry_id:193238)。这会引出一些非经典、甚至违反直觉的现象。

#### 一般变换与横向速度效应

考虑一个天体物理学的场景 [@problem_id:1880143]。一个[活动星系核](@entry_id:158029)（AGN）正以 $v = 0.5c$ 的速度沿 $x$ 轴远离地球。我们将地球视为 $S$ 系，AGN的静止系视为 $S'$ 系。天文学家观测到，从AGN中喷出的一团等离子体云，在 $S'$ 系中的速度分量为 $u'_x = 0.5c$ 和 $u'_y = 0.5c$。为了确定这团云在地球系 $S$ 中的速度，我们需要应用变换公式。

首先计算 $\gamma$ 因子和公共分母项：
$$\gamma = \frac{1}{\sqrt{1 - (0.5)^2}} = \frac{1}{\sqrt{0.75}} = \frac{2}{\sqrt{3}}$$
$$1 + \frac{v u'_x}{c^2} = 1 + \frac{(0.5c)(0.5c)}{c^2} = 1 + 0.25 = 1.25 = \frac{5}{4}$$

现在计算 $S$ 系中的速度分量：
$$u_x = \frac{u'_x + v}{1 + \frac{v u'_x}{c^2}} = \frac{0.5c + 0.5c}{5/4} = \frac{c}{5/4} = \frac{4}{5}c$$
$$u_y = \frac{u'_y}{\gamma \left( 1 + \frac{v u'_x}{c^2} \right)} = \frac{0.5c}{\frac{2}{\sqrt{3}} \cdot \frac{5}{4}} = \frac{\frac{1}{2}c}{\frac{5}{2\sqrt{3}}} = \frac{\sqrt{3}}{5}c$$
因此，在地球[参考系](@entry_id:169232)中测得的速度分量为 $(\frac{4}{5}c, \frac{\sqrt{3}}{5}c)$。

另一个有趣的特例是物体在 $S'$ 系中只具有横向速度 [@problem_id:2087617]。假设一艘飞船（$S'$ 系）以速度 $v$ 沿 $x$ 轴飞行，并垂直于飞行方向发射了一个探测器，速度为 $u_0$。在 $S'$ 系中，探测器的速度分量是 $u'_x = 0$ 和 $u'_y = u_0$。在空间站（$S$ 系）看来：
$$u_x = \frac{0 + v}{1 + 0} = v$$
$$u_y = \frac{u_0}{\gamma(1 + 0)} = \frac{u_0}{\gamma} = u_0 \sqrt{1 - \frac{v^2}{c^2}}$$
在 $S$ 系中，探测器的速度是 $(v, u_0/\gamma)$。一个重要的结论是，横向速度分量在 $S$ 系中被测量到的值 $u_y$ *小于* 在 $S'$ 系中的值 $u'_y$。这个效应可以归因于[时间膨胀](@entry_id:157877)：$S$ 系的观察者看到 $S'$ 系的时钟变慢，因此在单位时间内观察到的 $y$ 方向位移也相应变小。

该探测器在 $S$ 系中的运动方向与 $x$ 轴的夹角 $\theta$ 的正切值为：
$$\tan\theta = \frac{u_y}{u_x} = \frac{u_0/\gamma}{v} = \frac{u_0}{v\gamma} = \frac{u_0}{v}\sqrt{1-\frac{v^2}{c^2}}$$
这个角度的偏离是下面要讨论的[相对论光行差](@entry_id:161160)现象的基础。

### [相对论光行差](@entry_id:161160)

**[光行差](@entry_id:186866)**是指运动中的观察者所测量的光的传播方向发生变化的现象。在经典物理中，这类似于在雨中奔跑时感觉雨滴是从前方斜向射来。[相对论光行差](@entry_id:161160)是这一现象的精确描述。

我们可以利用速度[逆变](@entry_id:192290)换公式来推导[光行差](@entry_id:186866)的数学表达式 [@problem_id:1845786]。假设在 $S$ 系中，一束[激光](@entry_id:194225)信号的传播方向与 $x$ 轴成 $\theta$ 角。其速度分量为 $u_x = c \cos\theta$ 和 $u_y = c \sin\theta$。在以速度 $v$ 沿 $x$ 轴运动的探测器（$S'$ 系）中，这束光的角度 $\theta'$ 是多少？

我们应用逆变换公式计算 $S'$ 系中的速度分量：
$$u'_x = \frac{u_x - v}{1 - \frac{v u_x}{c^2}} = \frac{c \cos\theta - v}{1 - \frac{v c \cos\theta}{c^2}} = c \frac{\cos\theta - v/c}{1 - (v/c)\cos\theta}$$
$$u'_y = \frac{u_y}{\gamma \left( 1 - \frac{v u_x}{c^2} \right)} = \frac{c \sin\theta}{\gamma \left( 1 - \frac{v c \cos\theta}{c^2} \right)} = \frac{c \sin\theta \sqrt{1-v^2/c^2}}{1 - (v/c)\cos\theta}$$

在 $S'$ 系中，角度 $\theta'$ 的正切值为 $\tan(\theta') = \frac{u'_y}{u'_x}$：
$$\tan(\theta') = \frac{\frac{c \sin\theta \sqrt{1-v^2/c^2}}{1 - (v/c)\cos\theta}}{c \frac{\cos\theta - v/c}{1 - (v/c)\cos\theta}} = \frac{\sin\theta \sqrt{1 - v^2/c^2}}{\cos\theta - v/c}$$
这就是**[相对论光行差](@entry_id:161160)公式**。它精确地描述了光线方向如何随着观察者的运动而改变。

[光行差](@entry_id:186866)现象并非只针对光，它适用于任何运动的物体 [@problem_id:395288]。例如，一个高能粒子在 $S$ 系中沿负 $y$ 轴向原点运动，其速度为 $\vec{u} = (0, -u, 0)$。在以速度 $v$ 沿 $x$ 轴运动的 $S'$ 系中，其速度分量变为：
$$u'_x = \frac{0 - v}{1 - 0} = -v$$
$$u'_y = \frac{-u}{\gamma(1 - 0)} = -\frac{u}{\gamma}$$
在 $S'$ 系中，粒子不仅有 $y'$ 方向的速度，还出现了一个 $x'$ 方向的速度分量。其运动轨迹将不再是沿着 $y'$ 轴，而是从一个与 $y'$ 轴成一定角度的方向射来，这个角度的正切值（相对于负 $y'$ 轴）为 $\frac{|u'_x|}{|-u'_y|} = \frac{v}{u/\gamma} = \frac{\gamma v}{u}$。

### 高阶主题：洛伦兹变换的合成

一个深刻且不那么直观的相对论效应是，两个非共线的洛伦兹变换（boosts）的合成并不等同于一个简单的、速度矢量相加的[洛伦兹变换](@entry_id:176827)。它们的合成等效于一个单一的变换加上一个空间旋转，这种旋转被称为**维格纳转动 (Wigner Rotation)**。

#### 变换的不可交换性

我们可以通过一个具体的例子来展示洛伦兹变换的不可交换性 [@problem_id:395242]。考虑两种操作顺序：

**场景 A:** 先沿 $x$ 轴进行速度为 $v$ 的变换，再在新的[参考系](@entry_id:169232) $S'$ 中沿其 $y'$ 轴进行速度为 $v$ 的变换。
一个在最终[参考系](@entry_id:169232) $S''$ 中静止的粒子，其在 $S'$ 中的速度为 $\vec{u}'=(0,v,0)$。利用[速度变换](@entry_id:265594)公式，它在初始 $S$ 系中的速度 $\vec{u}_A$ 为：
$u_{A,x} = \frac{0+v}{1+0} = v$
$u_{A,y} = \frac{v}{\gamma(1+0)} = \frac{v}{\gamma} = v\sqrt{1-v^2/c^2}$
所以，$\vec{u}_A = (v, v/\gamma, 0)$。

**场景 B:** 先沿 $y$ 轴进行速度为 $v$ 的变换，再在新的[参考系](@entry_id:169232) $\tilde{S}'$ 中沿其 $x'$ 轴进行速度为 $v$ 的变换。
通过[对称性分析](@entry_id:174795)，我们可以知道最终在 $S$ 系中的速度 $\vec{u}_B$ 的分量将会互换：
$\vec{u}_B = (v/\gamma, v, 0)$。

显然，$\vec{u}_A \neq \vec{u}_B$。这[直接证明](@entry_id:141172)了非共线[洛伦兹变换](@entry_id:176827)的顺序是重要的，它们是不可交换的。这两个最终速度矢量之间的夹角 $\theta$ 可以通过[点积](@entry_id:149019)计算：
$$\cos\theta = \frac{\vec{u}_A \cdot \vec{u}_B}{|\vec{u}_A| |\vec{u}_B|} = \frac{v(v/\gamma) + (v/\gamma)v}{v^2 + (v/\gamma)^2} = \frac{2v^2/\gamma}{v^2(1+1/\gamma^2)} = \frac{2/\gamma}{1+(1-v^2/c^2)} = \frac{2\sqrt{1-v^2/c^2}}{2-v^2/c^2}$$
这个非零的角度 $\theta = \arccos\left(\frac{2\sqrt{1-v^2/c^2}}{2-v^2/c^2}\right)$ 就是维格纳转动的量化体现。

一个相关的问题是计算两个垂直变换合成后的有效速度及其方向 [@problem_id:395263]。场景A中的最终速度 $\vec{V}_{eff} = \vec{u}_A = (v, v\sqrt{1-v^2/c^2}, 0)$。这个有效速度与 $x$ 轴的夹角 $\theta$ 满足：
$$\tan\theta = \frac{V_y}{V_x} = \frac{v\sqrt{1-v^2/c^2}}{v} = \sqrt{1-v^2/c^2} = \frac{1}{\gamma}$$
因此，$\theta = \arctan(\sqrt{1 - v^2/c^2})$。在非相对论极限下 ($v \ll c$, $\gamma \approx 1$), $\tan\theta \approx 1$, $\theta \approx 45^\circ$，这与伽利略速度的矢量相加结果一致。但在极端相对论情况下 ($v \to c$, $\gamma \to \infty$), $\tan\theta \to 0$, $\theta \to 0$。这意味着，连续进行两个近光速的垂直变换，最终的速度方向会极度偏向第一次变换的方向。

### 延伸探讨

#### 假想的[超光速运动](@entry_id:158217)

[速度变换](@entry_id:265594)公式的数学结构可以用来进行一些有趣的理论探索，例如研究假想中的[超光速粒子](@entry_id:159813)（快子，Tachyon）的行为。请注意，这纯粹是一个思想实验，因为快子在物理上尚未被证实存在。

考虑一个特殊情况 [@problem_id:395278]：在何种条件下，一个粒子的运动方向在一个变换前后保持不变？即 $\vec{u}'$ 与 $\vec{u}$ 平行。这意味着它们速度分量的比值相等：$\frac{u'_x}{u'_y} = \frac{u_x}{u_y}$。
将[速度变换](@entry_id:265594)公式代入：
$$\frac{\frac{u_x-v}{1-u_x v/c^2}}{\frac{u_y}{\gamma(1-u_x v/c^2)}} = \frac{u_x}{u_y}$$
化简后得到：
$$\gamma(u_x - v) = u_x \implies \frac{u_x-v}{\sqrt{1-v^2/c^2}} = u_x$$
解出 $u_x$：
$$u_x = \frac{v}{1 - \sqrt{1 - v^2/c^2}} = \frac{c^2}{v}\left(1 + \sqrt{1-\frac{v^2}{c^2}}\right)$$
由于 $\sqrt{1-v^2/c^2}  1$，我们可以看到 $u_x > \frac{c^2}{v}$。因为 $v  c$，所以 $u_x > c$。这个结果表明，只有当一个（假想的）粒子以[超光速运动](@entry_id:158217)时，其运动方向才有可能在某个特定的纵向[洛伦兹变换](@entry_id:176827)下保持不变。

#### 四维速度形式主义

处理[相对论动力学](@entry_id:264218)问题的一种更优雅、更强大的方法是使用[四维矢量](@entry_id:275085)形式主义。在[闵可夫斯基时空](@entry_id:156421)中，一个粒子的**[四维速度](@entry_id:269673)** $U$ 被定义为 $U = \gamma_w(c, \vec{w})$，其中 $\vec{w}$ 是粒子的三维速度，$\gamma_w$ 是对应的洛伦兹因子。四维速度是一个四维矢量，它在[洛伦兹变换](@entry_id:176827)下的变换性质非常简洁。

两个四维矢量 $A=(A^0, \vec{A})$ 和 $B=(B^0, \vec{B})$ 的[标量积](@entry_id:138996)（在使用 $(+,-,-,-)$ 度规时）定义为 $A \cdot B = A^0 B^0 - \vec{A} \cdot \vec{B}$。这个[标量积](@entry_id:138996)是一个**洛伦兹不变量**，意味着它在所有惯性参考系中的值都相同。

让我们计算一个具体例子中的[不变量](@entry_id:148850)[标量积](@entry_id:138996) [@problem_id:395205]。在 $S$ 系中，粒子A的速度为 $\vec{u}_A=(u,0,0)$。在 $S'$ 系中，粒子B的速度为 $\vec{u}'_B=(0,u',0)$。$S'$ 系以速度 $v$ 沿 $x$ 轴相对于 $S$ 系运动。

首先，我们在同一个[参考系](@entry_id:169232)（比如 $S$ 系）中写出两个粒子的四维速度。
粒子A在 $S$ 系中的[四维速度](@entry_id:269673)为：
$$U_A = (\gamma_u c, \gamma_u u, 0, 0), \quad \text{其中} \quad \gamma_u = (1-u^2/c^2)^{-1/2}$$

为了得到粒子B在 $S$ 系中的四维速度 $U_B$，我们首先需要将其三维速度 $\vec{u}'_B$ 变换到 $S$ 系。根据我们之前的推导，粒子B在 $S$ 系中的速度为 $\vec{u}_B = (v, u'/\gamma_v, 0)$。与此对应的洛伦兹因子 $\gamma_B$ 为：
$$\gamma_B = \left(1 - \frac{v^2+(u'/\gamma_v)^2}{c^2}\right)^{-1/2} = \left(1 - \frac{v^2}{c^2} - \frac{u'^2(1-v^2/c^2)}{c^2}\right)^{-1/2} = \left((1 - \frac{v^2}{c^2})(1-\frac{u'^2}{c^2})\right)^{-1/2} = \gamma_v \gamma_{u'}$$
因此，粒子B在 $S$ 系中的[四维速度](@entry_id:269673)为：
$$U_B = (\gamma_B c, \gamma_B \vec{u}_B) = (\gamma_v \gamma_{u'} c, \gamma_v \gamma_{u'} v, \gamma_v \gamma_{u'} (u'/\gamma_v), 0) = (\gamma_v \gamma_{u'} c, \gamma_v \gamma_{u'} v, \gamma_{u'} u', 0)$$

现在计算[标量积](@entry_id:138996) $U_A \cdot U_B = U_A^0 U_B^0 - U_{A,x}U_{B,x}$：
$$U_A \cdot U_B = (\gamma_u c)(\gamma_v \gamma_{u'} c) - (\gamma_u u)(\gamma_v \gamma_{u'} v) = \gamma_u \gamma_v \gamma_{u'} (c^2 - uv)$$
将所有 $\gamma$ 因子写开，得到最终的[不变量](@entry_id:148850)表达式：
$$U_A \cdot U_B = \frac{c^2 - uv}{\sqrt{\left(1-\frac{u^2}{c^2}\right)\left(1-\frac{v^2}{c^2}\right)\left(1-\frac{u'^2}{c^2}\right)}}$$
这个结果的值不依赖于我们选择哪个[惯性参考系](@entry_id:276742)进行计算，体现了四维形式主义在处理相对论问题时的简洁性和深刻的物理洞察力。

总之，洛伦兹[速度变换](@entry_id:265594)是[狭义相对论](@entry_id:275552)的一个核心组成部分。它不仅是理论[自洽性](@entry_id:160889)的保证，也为我们理解高速世界中的各种物理现象——从[恒星光行差](@entry_id:171045)到粒子物理中的运动学——提供了坚实的数学基础。
