## 引言
大脑是宇宙中最复杂的动力系统之一，其功能——从感知、行动到思考和记忆——都源于数十亿个神经元之间复杂的动态相互作用。为了超越纯粹的描述性理解，神经科学亟需一个强大的理论框架来揭示这些动态行为背后的计算原理。动力系统理论，以其对随时间变化的系统进行[数学建模](@entry_id:262517)和分析的核心优势，恰好为此提供了理想的语言和工具集。它使我们能够从基本构成要素出发，构建关于神经功能如何涌现的、具有预测性的理论。

本文旨在系统性地介绍动力系统理论在神经科学中的核心应用。我们首先将在**“原理与机制”**一章中，从最基本的[神经元电学特性](@entry_id:178034)出发，逐步构建能够产生[动作电位](@entry_id:138506)和复杂发放模式的数学模型，并探讨[突触可塑性](@entry_id:137631)和[网络同步](@entry_id:266377)的动力学基础。接着，在**“应用与交叉学科联系”**一章中，我们将展示这些理论如何被应用于解释记忆、[感觉适应](@entry_id:153446)、大脑节律乃至帕金森等疾病的神经机制，并探索其与控制理论等领域的深刻联系。最后，通过**“动手实践”**部分，读者将有机会亲手实现和分析神经动力学模型，从而巩固所学知识。

让我们从构建[神经计算](@entry_id:154058)的基石开始，深入探索单个神经元和简单网络背后的核心动力学原理。

## 原理与机制

在神经科学中，动力系统理论为我们提供了一套强大的数学语言和概念框架，用以描述和理解神经元及其网络从单个分子到整体行为的复杂动态。本章将深入探讨构成[神经计算](@entry_id:154058)基础的一些核心原理和机制，我们将从单个神经元最基本的电学特性出发，逐步构建更复杂的模型，以揭示兴奋性、[振荡](@entry_id:267781)、同步和学习等关键神经现象背后的动力学基础。

### 神经元的基本电学特性：被动膜

神经元最基础的功能是整合输入信号，而这一过程的物理基础是其[细胞膜](@entry_id:146704)的电学特性。在没有[电压门控离子通道](@entry_id:175526)激活的阈下[电位](@entry_id:267554)范围内，[神经元膜](@entry_id:182072)的行为可以被一个简单的**[漏积分器](@entry_id:261862) (leaky integrator)** 模型所近似。该模型将[细胞膜](@entry_id:146704)类比为一个由电阻 $R_m$ 和电容 $C_m$ 并联组成的[RC电路](@entry_id:275926)。电容代表了膜储存[电荷](@entry_id:275494)的能力，而电阻代表了允许离子“泄漏”穿过膜的被动[离子通道](@entry_id:144262)。

根据电路理论，流经电容的电流为 $I_C = C_m \frac{dV}{dt}$，流经电阻的电流为 $I_R = \frac{V}{R_m}$。根据[基尔霍夫电流定律](@entry_id:270632)，总的输入电流 $I_{in}(t)$ 等于这两部分之和，即 $I_{in}(t) = I_C + I_R$。整理后，我们得到描述膜电位 $V(t)$（相对于[静息电位](@entry_id:176014)）随时间变化的[一阶线性常微分方程](@entry_id:164502)：

$C_m \frac{dV}{dt} = -\frac{V}{R_m} + I_{in}(t)$

这个方程通常被写成更简洁的形式：

$\tau \frac{dV}{dt} = -V + R_m I_{in}(t)$

在这里，$\tau = R_m C_m$ 被称为**[膜时间常数](@entry_id:168069) (membrane time constant)**，它代表了膜电位对电流变化的响应速度。$R_m$ 是**[膜电阻](@entry_id:174729) (membrane resistance)**，它决定了在[稳态](@entry_id:182458)时，单位输入电流能引起多大的电压变化。

为了具体理解这些参数的意义，我们可以考察一个典型的实验情景：一个神经元初始处于静息状态（$V(0) = 0$），在 $t=0$ 时刻，一个恒定的刺激电流 $I_0$ 被注入细胞 [@problem_id:1661286]。此时，动力学方程变为：

$\tau \frac{dV}{dt} = -V + R_m I_0$

这是一个[一阶线性常微分方程](@entry_id:164502)，其解为：

$V(t) = R_m I_0 \left(1 - \exp\left(-\frac{t}{\tau}\right)\right)$

从这个解中我们可以看到，膜电位 $V(t)$ 呈指数形式增长，并逐渐趋近于一个**[稳态](@entry_id:182458)值 (steady-state value)** $V_{\infty} = \lim_{t\to\infty} V(t) = R_m I_0$。这个[稳态](@entry_id:182458)值直接由[欧姆定律](@entry_id:276027)决定。

[时间常数](@entry_id:267377) $\tau$ 在此过程中扮演了核心角色。它决定了膜电位上升的速度。具体来说，当 $t=\tau$ 时，$V(\tau) = V_{\infty} (1 - \exp(-1)) \approx 0.63 V_{\infty}$。也就是说，[时间常数](@entry_id:267377)是[膜电位](@entry_id:150996)达到其最终值的约63%所需的时间。一个更有启发性的度量是达到[稳态](@entry_id:182458)值一半所需的时间。通过求解 $V(t_{1/2}) = \frac{1}{2} V_{\infty}$，我们得到：

$R_m I_0 \left(1 - \exp\left(-\frac{t_{1/2}}{\tau}\right)\right) = \frac{1}{2} R_m I_0$

$\exp\left(-\frac{t_{1/2}}{\tau}\right) = \frac{1}{2}$

解得 $t_{1/2} = \tau \ln 2$ [@problem_id:1661286]。这个简洁的结果为[时间常数](@entry_id:267377)提供了一个非常直观的物理解释：它与[系统响应](@entry_id:264152)的“半衰期”成正比。一个$\tau$值较大的神经元对输入的响应更慢，但也能在更长的时间尺度上整合输入信号。

### [离子通道](@entry_id:144262)的门控动力学

被动膜模型虽然基础，但它无法解释神经元最重要的特性——[动作电位](@entry_id:138506)的产生。动作电位是由[电压门控离子通道](@entry_id:175526)的协同活动产生的，这些通道的导电性依赖于膜电位。每个通道可以被想象成一个在不同构象状态（例如，**关闭态 (Closed state, C)** 和 **开放态 (Open state, O)**）之间随机转换的分子机器。

我们可以使用[连续时间马尔可夫过程](@entry_id:272118)来为这些转换建立模型。考虑一个简化的双态模型，其中通道从关闭态到开放态的转换速率为 $\alpha$，从开放态到关闭态的[转换速率](@entry_id:272061)为 $\beta$ [@problem_id:1661273]。令 $P_C(t)$ 和 $P_O(t)$ 分别为单个通道在时刻 $t$ 处于关闭态和开放态的概率。由于只有这两种状态，我们有 $P_C(t) + P_O(t) = 1$。

$P_C$ 的变化率取决于从开放态流入的概率流（$\beta P_O$）和流向开放态的[概率流](@entry_id:150949)（$-\alpha P_C$）。因此，我们可以写出描述 $P_C(t)$ 演化的**主方程 (master equation)**：

$\frac{dP_C}{dt} = -\alpha P_C + \beta P_O$

利用 $P_O = 1 - P_C$，我们可以将方程改写为只含 $P_C$ 的[一阶线性微分方程](@entry_id:164869)：

$\frac{dP_C}{dt} = -(\alpha + \beta)P_C + \beta$

这个方程的通解形式为指数松弛。假设在 $t=0$ 时，所有通道都被实验手段置于关闭态，即 $P_C(0) = 1$。方程的解为：

$P_C(t) = \frac{\beta}{\alpha+\beta} + \left(1 - \frac{\beta}{\alpha+\beta}\right) \exp(-(\alpha+\beta)t) = \frac{\beta}{\alpha+\beta} + \frac{\alpha}{\alpha+\beta} \exp(-(\alpha+\beta)t)$

随着时间的推移，$t \to \infty$，指数项衰减至零，系统达到**平衡态 (equilibrium)**，此时处于关闭态的概率为 $P_{C, \infty} = \frac{\beta}{\alpha+\beta}$。系统向该[平衡态](@entry_id:168134)松弛的时间尺度由 $(\alpha+\beta)^{-1}$ 决定。在更真实的[Hodgkin-Huxley](@entry_id:273564)类型模型中，[转换速率](@entry_id:272061) $\alpha$ 和 $\beta$ 本身是电压的函数，这使得[离子通道](@entry_id:144262)的动力学与膜电位的[动力学耦合](@entry_id:150387)起来，从而产生了丰富的[非线性](@entry_id:637147)行为。

### 单[神经元整合](@entry_id:170464)与发放模型

将膜的被动特性与[离子通道](@entry_id:144262)的主动门控动力学相结合，便可以构建出能够产生动作电位的神经元模型。这些模型在维度和复杂性上各不相同，但它们都旨在捕捉神经元从整合输入到产生输出（发放脉冲）的核心计算过程。

#### 简化模型与[相平面分析](@entry_id:272304)

完整描述[离子通道](@entry_id:144262)动力学的[Hodgkin-Huxley模型](@entry_id:163105)维度较高（四维），分析起来较为复杂。为了抓住核心动力学机制，神经科学家们发展了许多[降维](@entry_id:142982)的简化模型，如FitzHugh-Nagumo模型。这些模型通常包含一个代表[膜电位](@entry_id:150996)的**快变量 (fast variable)** $V$ 和一个或多个代表恢复过程（如[钾离子通道](@entry_id:174108)激活或钠离子通道失活）的**慢变量 (slow variable)** $w$。

一个典型的二维简化模型系统具有如下形式：
$\frac{dV}{dt} = f(V, w) + I_{ext}$
$\frac{dw}{dt} = g(V, w)$

分析这类二维系统的有力工具是**[相平面分析](@entry_id:272304) (phase-plane analysis)**。[相平面](@entry_id:168387)是以状态变量（这里是$V$和$w$）为坐标轴构成的空间。系统在任意时刻的状态都对应于[相平面](@entry_id:168387)上的一个点，而系统的演化则对应于该点在[相平面](@entry_id:168387)上的一条轨迹。

轨迹的走向由向量场 $(\frac{dV}{dt}, \frac{dw}{dt})$ 决定。在[相平面](@entry_id:168387)上，两个特别重要的曲线集合是**[零斜线](@entry_id:261510) (nullclines)**，即导数为零的曲线。$V$-[零斜线](@entry_id:261510)是满足 $\frac{dV}{dt} = 0$ 的所有点的集合，而$w$-[零斜线](@entry_id:261510)是满足 $\frac{dw}{dt} = 0$ 的所有点的集合。在$V$-零斜线上，轨迹的运动方向是垂直的；在$w$-零斜线上，轨迹的运动方向是水平的。

系统的**[不动点](@entry_id:156394) (fixed point)** 或[平衡点](@entry_id:272705)是两个零斜线的交点，因为在这些点上 $\frac{dV}{dt} = 0$ 且 $\frac{dw}{dt} = 0$，系统状态不再随时间改变。例如，考虑一个描述阈下适应性的[线性模型](@entry_id:178302) [@problem_id:1661320]：

$\frac{dv}{dt} = -v - w + I$
$\tau \frac{dw}{dt} = v - w$

其[不动点](@entry_id:156394) $(v^*, w^*)$ 满足 $-v^* - w^* + I = 0$ 和 $v^* - w^* = 0$。解这个线性方程组得到唯一的[固定点](@entry_id:156394)为 $(v^*, w^*) = (\frac{I}{2}, \frac{I}{2})$。这个[不动点的稳定性](@entry_id:265683)决定了神经元在[持续电流](@entry_id:146997) $I$ 作用下的最终静息状态。

对于[非线性系统](@entry_id:168347)，如简化的Izhikevich模型 [@problem_id:1661269]，寻找[不动点](@entry_id:156394)的方法是相同的：

$\frac{dv}{dt} = 0.04v^2 + 5v + 140 - u + I$
$\frac{du}{dt} = a(bv - u)$

通过令导数为零并代入参数，我们可以解出系统唯一的[稳定不动点](@entry_id:262720)，它代表了神经元在给定输入下的[静息电位](@entry_id:176014)和恢复状态。

[相平面分析](@entry_id:272304)的威力在于它能直观地解释复杂的[非线性](@entry_id:637147)现象。一个经典的例子是**[阳极](@entry_id:140282)阻断兴奋 (anode break excitation)** [@problem_id:1661311]。这种现象指的是，在施加一个持续的超[极化电流](@entry_id:196744)（使[膜电位](@entry_id:150996)比[静息电位](@entry_id:176014)更负）并随后突然撤销该电流后，神经元会发放一个[动作电位](@entry_id:138506)。

在一个类似FitzHugh-Nagumo的模型中，这个过程可以在[相平面](@entry_id:168387)上清晰地展现。超[极化电流](@entry_id:196744) $I_{hyp}  0$ 会使系统的$V$-[零斜线](@entry_id:261510)（通常呈N形）向下移动，导致[不动点](@entry_id:156394)移动到一个更负的电压和更低的恢[复变量](@entry_id:175312) $w$ 值。如果这个超[极化电流](@entry_id:196744)足够强且持续时间足够长，系统会稳定在新的[不动点](@entry_id:156394) $(V_{hyp}, w_{hyp})$。当电流突然撤销时，$V$-[零斜线](@entry_id:261510)恢复到其原始位置，但系统的状态点 $(V_{hyp}, w_{hyp})$ 仍然停留在原来的位置。如果这个位置处于新[相平面](@entry_id:168387)中一个特定的区域（通常是在$V$-[零斜线](@entry_id:261510)中间分支的“阈值”区域之外），轨迹将无法直接回到新的静息点，而是会先沿着一条大的环路行进，这在时间上表现为一个完整的[动作电位](@entry_id:138506)，之后才返回静息点。这巧妙地解释了为何“关闭”一个抑制性输入反而能“激发”一个神经元。

#### 兴奋性的[分岔](@entry_id:273973)机制

神经元最显著的特征之一是它能够在静息态和发放态之间进行转换。这种行为上的质变在动力系统理论中被称为**[分岔](@entry_id:273973) (bifurcation)**。当一个系统参数，如外部输入电流 $I_{ext}$，缓慢变化并穿过一个临界值时，[不动点](@entry_id:156394)的数量或其稳定性会发生改变。

神经元的兴奋性通常被分为两种类型。I型兴奋性神经元可以在任意低的频率开始发放，其[发放频率](@entry_id:275859)随输入电流平滑增加。这通常对应于**[不变圆上的鞍结分岔](@entry_id:261045) (saddle-node on invariant circle, SNIC) **。

相比之下，[II型兴奋性](@entry_id:189863)神经元有一个明确的[发放频率](@entry_id:275859)阈值，一旦超过该阈值，它们就开始以一个非零的频率发放。这种类型的转换通常与**[亚临界霍普夫分岔](@entry_id:262144) (subcritical Andronov-Hopf bifurcation)** 相关，并常常表现出**滞后现象 (hysteresis)** [@problem_id:1661289]。

[滞后现象](@entry_id:268538)意味着系统的状态取决于其历史。当输入电流从低到高缓慢增加时，神经元会保持静息直到一个较高的阈值 $\mu_{up}$，然后突然跳变到高频发放状态。反之，当电流从高到低缓慢减小时，神经元会继续发放，直到一个更低的阈值 $\mu_{down}$，然后才突然回到静息状态。在 $\mu_{down}  \mu  \mu_{up}$ 的区间内，系统是**双稳态的 (bistable)**：静息态和发放态都是可能的稳定状态。

这种行为可以用一个描述[振荡](@entry_id:267781)幅度 $r$ 的简化方程（范德波尔[振子](@entry_id:271549)的正规型）来精确刻画：

$\frac{dr}{dt} = r (\mu + \alpha r^2 - \beta r^4)$

其中 $\mu$ 是与输入电流相关的[分岔参数](@entry_id:264730)，$\alpha, \beta$ 是正实数。$r=0$ 对应静息态，而 $r0$ 的[稳定不动点](@entry_id:262720)对应于持续的[振荡](@entry_id:267781)（发放）。通过分析该方程[不动点的稳定性](@entry_id:265683)和分岔点，可以发现静息态在 $\mu_{up} = 0$ 时失稳，而发放态在 $\mu_{down} = -\frac{\alpha^2}{4\beta}$ 处通过鞍结分岔消失。因此，滞后环的宽度为 $\Delta\mu = \mu_{up} - \mu_{down} = \frac{\alpha^2}{4\beta}$。

#### [动作电位](@entry_id:138506)的产生机制：[快慢系统](@entry_id:262083)分析

虽然简化模型很有用，但要理解动作电位本身定型的、刻板的波形，我们需要回到更具生物物理真实性的模型，如[Hodgkin-Huxley模型](@entry_id:163105)。该模型的一个关键特性是其内在变量演化的**[时间尺度分离](@entry_id:149780) (timescale separation)**。[膜电位](@entry_id:150996) $V$ 以及钠离子通道的激活变量 $m$ 和失活变量 $h$ 是快变量，而[钾离子通道](@entry_id:174108)的激活变量 $n$ 则是一个慢变量。

这使得我们可以使用**快慢动力系统 (fast-slow dynamical system)** 的方法进行分析 [@problem_id:1661275]。我们将慢变量 $n$ 暂时视为一个固定的参数，然后分析由 $(V, m, h)$ 构成的**快子系统 (fast subsystem)** 的行为。快子系统的[不动点](@entry_id:156394)（即快变量停止变化的点）依赖于 $n$ 的取值。将这些[不动点](@entry_id:156394)的电压值作为 $n$ 的函数绘制出来，通常会得到一条Z形或S形的曲线，称为**[临界流形](@entry_id:263391) (critical manifold)**。这条曲线的下支和上支对应于稳定的[不动点](@entry_id:156394)（静息态和兴奋态），而中间分支对应于不稳定的[不动点](@entry_id:156394)。

[动作电位](@entry_id:138506)的产生可以被看作是系统轨迹在相空间中的一次“旅行”：
1.  **静息态**：系统处于[临界流形](@entry_id:263391)的下支，对应于一个稳定的、低电压的[不动点](@entry_id:156394)。此时 $n$ 的值很小。
2.  **启动**：一个超阈值的刺激将系统状态点快速“踢”离下支，由于快动力学的支配，轨迹会迅速向上跳跃，被上支的[稳定不动点](@entry_id:262720)所吸引。
3.  **兴奋态**：系统到达上支，处于高电压的兴奋状态。此时，由于电压 $V$ 很高，慢变量 $n$ 开始根据其自身的[动力学方程](@entry_id:751029) $\frac{dn}{dt} = (n_{\infty}(V) - n) / \tau_n(V)$ 缓慢增加。这导致系统状态点沿着上支缓慢向右移动。
4.  **终止**：随着 $n$ 的增加，系统点到达了上支的“膝点”。在这个点，稳定的上支[不动点](@entry_id:156394)与不稳定的中支[不动点](@entry_id:156394)相遇并湮灭。这种分岔类型是**[鞍结分岔](@entry_id:263507) (saddle-node bifurcation)**。
5.  **[复极化](@entry_id:150957)**：一旦上支的[稳定不动点](@entry_id:262720)消失，系统在当前 $n$ 值下不再有高电压的稳定状态。快动力学再次主导，轨迹迅速向下“跌落”，被下支的[稳定不动点](@entry_id:262720)吸引。这对应于[动作电位](@entry_id:138506)的快速[复极化](@entry_id:150957)阶段。
6.  **恢复**：系统回到下支后，电压 $V$ 变低，慢变量 $n$ 开始缓慢减小，使系统点沿下支向左移动，最终回到最初的静息点，准备下一次兴奋。

这种快慢分析方法不仅完美地解释了动作电位刻板的形状，还将这一复杂的生理过程与基础的动力系统[分岔理论](@entry_id:143561)深刻地联系在了一起。

### 神经元间的相互作用

神经系统的计算能力源于神经元之间通过[突触形成](@entry_id:167681)的[复杂网络](@entry_id:261695)。动力系统理论同样为我们提供了分析神经元间相互作用的工具。

#### [突触可塑性](@entry_id:137631)：学习与记忆的动力学

学习和记忆被认为与神经元之间连接的强度，即**突触权重 (synaptic weight)** 的改变有关。一个著名的学习法则是赫布法则（Hebb's rule）：“一起发放的神经元，连接在一起”。我们可以构建一个简单的动力学模型来描述突触权重 $w(t)$ 的演化，该模型包含两个过程：一个是与前后神经元发放相关性成正比的增强过程（[赫布学习](@entry_id:156080)），另一个是权重自身的衰减过程（遗忘）[@problem_id:1661324]。

一个简单的模型可以写为：

$\frac{dw}{dt} = -\frac{w}{\tau} + \alpha C$

其中 $w$ 是突触权重，$\tau$ 是衰减的时间常数，$\alpha$ 是[学习率](@entry_id:140210)，而 $C$ 是一个代表前后神经元发放相关性的恒定正数。这个方程描述了增强和遗忘之间的竞争。通过令 $\frac{dw}{dt} = 0$，我们可以找到系统的**平衡权重 (equilibrium weight)**：

$w_{eq} = \alpha C \tau$

这个结果表明，突触权重会稳定在一个由[学习率](@entry_id:140210)、神经元相关性以及遗忘速率共同决定的值上。这为记忆的形成和维持提供了一个简单的动力学解释。

#### 电耦合与同步

除了[化学突触](@entry_id:147038)，神经元之间还可以通过**[间隙连接](@entry_id:143226) (gap junctions)** 进行直接的电耦合。这种耦合允许电流在细胞间直接流通，其强度与细胞间的电压差成正比。考虑两个相同的、相互电耦合的细胞，其膜电位为 $v_1$ 和 $v_2$ [@problem_id:1661307]：

$\frac{dv_1}{dt} = g(v_2 - v_1)$
$\frac{dv_2}{dt} = g(v_1 - v_2)$

其中 $g$ 是耦合[电导](@entry_id:177131)。这是一个[线性系统](@entry_id:147850)，我们可以通过引入新的变量来简化分析。定义总[电位](@entry_id:267554) $s(t) = v_1(t) + v_2(t)$ 和电位差 $d(t) = v_1(t) - v_2(t)$。对它们求导可得：

$\frac{ds}{dt} = \frac{dv_1}{dt} + \frac{dv_2}{dt} = g(v_2 - v_1) + g(v_1 - v_2) = 0$
$\frac{dd}{dt} = \frac{dv_1}{dt} - \frac{dv_2}{dt} = g(v_2 - v_1) - g(v_1 - v_2) = -2g d(t)$

第一个方程表明 $s(t)$ 是一个[守恒量](@entry_id:150267)，即 $v_1(t) + v_2(t) = v_1(0) + v_2(0)$ 始终成立。第二个方程的解是 $d(t) = d(0) \exp(-2gt)$。因为 $g0$，电位差会指数衰减至零。这意味着两个细胞的电压最终会达到相同的值。结合守恒律，我们可以求出最终的[稳态](@entry_id:182458)电压为 $\frac{v_1(0) + v_2(0)}{2}$。这个简单的模型揭示了电耦合的一个基本功能：促使相连的[神经元活动](@entry_id:174309)**同步化 (synchronization)**。

#### 相位耦合与[神经振荡](@entry_id:274786)

大脑中的许多活动，如脑电图（EEG）中的节律，都表现为大规模的[神经振荡](@entry_id:274786)。为了研究这种集[体节](@entry_id:187163)律现象，我们可以将每个神经元抽象为一个**[振子](@entry_id:271549) (oscillator)**，用其**相位 (phase)** $\theta(t)$ 来描述其在发放周期中的位置。

研究两个相同[振子](@entry_id:271549)之间耦合的最简化模型是[Kuramoto模型](@entry_id:273877) [@problem_id:1661292]。假设两个神经元的[相位动力学](@entry_id:274204)由以下方程描述：

$\dot{\theta}_1 = \omega + K \sin(\theta_2 - \theta_1)$
$\dot{\theta}_2 = \omega + K \sin(\theta_1 - \theta_2)$

这里 $\omega$ 是每个神经元固有的自然频率，$K0$ 是耦合强度。耦合项 $K \sin(\theta_j - \theta_i)$ 的形式使得当一个[振子](@entry_id:271549)领先时，它会减速，而当它落后时，它会加速，这是一种趋向于同步的机制。

为了分析同步行为，我们关注相位差 $\phi = \theta_2 - \theta_1$ 的动力学。对其求导：

$\dot{\phi} = \dot{\theta}_2 - \dot{\theta}_1 = (\omega + K \sin(\theta_1 - \theta_2)) - (\omega + K \sin(\theta_2 - \theta_1))$
$\dot{\phi} = K \sin(-\phi) - K \sin(\phi) = -2K \sin(\phi)$

现在我们得到了一个描述相位差演化的单一维度方程。其[不动点](@entry_id:156394)满足 $\dot{\phi}=0$，即 $\sin(\phi)=0$，在 $[0, 2\pi)$ 区间内解为 $\phi^*=0$ 和 $\phi^*=\pi$。
- $\phi=0$ 对应**同相完全同步 (in-phase synchronization)**。
- $\phi=\pi$ 对应**反相同步 (anti-phase synchronization)**。

为了判断哪个是稳定状态，我们进行[线性稳定性分析](@entry_id:154985)。令 $f(\phi) = -2K\sin(\phi)$，一个[不动点](@entry_id:156394) $\phi^*$ 是稳定的条件是 $f'(\phi^*)  0$。
- 在 $\phi=0$ 处，$f'(0) = -2K \cos(0) = -2K  0$（因为 $K0$），所以同相同步是稳定的。
- 在 $\phi=\pi$ 处，$f'(\pi) = -2K \cos(\pi) = 2K  0$，所以反相同步是不稳定的。

这个结果表明，对于这种吸引性耦合，两个神经元最终会演化到[完全同步](@entry_id:267706)的状态。这个简单的模型虽然高度抽象，但它揭示了神经元群体通过相互作用自发产生[宏观有序](@entry_id:155481)节律的基本原理。
