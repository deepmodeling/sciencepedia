## 引言
[电荷](@entry_id:275494)的运动是自然界中最普遍的现象之一，它驱动着从原子尺度的[化学反应](@entry_id:146973)到宇宙尺度的[等离子体动力学](@entry_id:185550)。理解和描述[电荷](@entry_id:275494)的流动是整个电磁学理论的基石。本文的核心主题——电流与[电流密度](@entry_id:190690)，正是量化这一动态过程的两个基本物理量。

对于初学者而言，电流通常被简化为电路理论中的一个标量“I”。然而，这种简化忽略了[电荷](@entry_id:275494)在三维空间中复杂的[分布](@entry_id:182848)和流动行为。为了建立一个更深刻、更普适的理解，我们必须从标量电流走向矢量[电流密度](@entry_id:190690)，从简单的电路走向连续介质中的[场论](@entry_id:155241)描述。本文旨在填补这一认知上的空白，揭示电流现象背后丰富的物理内涵。

本文将通过三个章节引导读者逐步深入。在“原理与机制”一章中，我们将建立电流和电流密度的严谨定义，探讨其微观起源，并推导[电荷守恒](@entry_id:264158)的[连续性方程](@entry_id:195013)和欧姆定律。随后，在“应用与交叉学科联系”一章中，我们将展示这些基本原理如何应用于[半导体](@entry_id:141536)、等离子体、电化学和[生物系统](@entry_id:272986)等前沿领域，彰显其强大的解释力和预测力。最后，在“动手实践”部分，通过精选的计算练习，读者将有机会巩固所学知识，并将其应用于解决实际问题。

## 原理与机制

在介绍章节之后，我们现在深入探讨[电荷](@entry_id:275494)运动的核心物理原理。本章的目标是建立对电流和[电流密度](@entry_id:190690)的严谨理解，从它们的基本定义到支配它们行为的深刻物理定律。我们将从宏观现象入手，逐步揭示其微观起源，并最终将这些概念与[电磁场](@entry_id:265881)理论的基本框架联系起来。

### 电流与[电流密度](@entry_id:190690)：宏观定义

[电荷](@entry_id:275494)的运动构成了 **电流** (electric current)。最直观的定义是，电流 $I$ 是指单位时间内通过某一给定[曲面](@entry_id:267450)的净[电荷](@entry_id:275494)量。如果在一个微小的时间间隔 $dt$ 内，有净[电荷](@entry_id:275494)量 $dQ$ 通过一个[曲面](@entry_id:267450)，那么瞬时电流定义为：

$$I(t) = \frac{dQ}{dt}$$

这个定义虽然简单，但却异常强大。它将一个动态过程（[电荷](@entry_id:275494)流动）与一个标量时间函数 $I(t)$ 联系起来。电流的单位是安培 (A)，即库仑/秒 (C/s)。

值得注意的是，电流的方向习惯上定义为正[电荷](@entry_id:275494)流动的方向。如果电流是由负[电荷](@entry_id:275494)（如金属中的电子）产生的，那么电流的方向与负[电荷](@entry_id:275494)的实际运动方向相反。此外，当我们考虑一个封闭体积内的[电荷](@entry_id:275494)变化时，流出该体积的电流与体积内[电荷](@entry_id:275494)量的变化率之间存在一个重要的符号关系。如果一个系统内的总[电荷](@entry_id:275494)量 $Q(t)$ 正在减少，这意味着有净正[电荷](@entry_id:275494)流出。因此，流出的电流 $I_{out}$ 为：

$$I_{out}(t) = -\frac{dQ(t)}{dt}$$

例如，在一个正在放电的[电容器](@entry_id:267364)中，其内部[电荷](@entry_id:275494) $Q(t)$ 随时间减少，导致 $\frac{dQ}{dt}  0$，从而产生一个正的向外电流。考虑一个实验场景，其中一个球形电极在 $t=0$ 后开始将内部离子释放到[电解](@entry_id:146038)液中，其内部总[电荷](@entry_id:275494)量 $Q(t)$ 由 $Q(t) = Q_{max} (\frac{t}{\tau}) \exp(1 - \frac{t}{\tau})$ 描述。通过对时间求导，我们可以得到流出电极的瞬时电流 $I_{out}(t) = -\frac{dQ}{dt} = \frac{Q_{max}}{\tau^{2}} \exp(1 - \frac{t}{\tau}) (t - \tau)$。这个例子清晰地展示了电流与[电荷](@entry_id:275494)量随时间变化之间的直接数学关系 [@problem_id:1576197]。

然而，电流 $I$ 是一个宏观量，它描述的是通过整个[曲面](@entry_id:267450)的总效应。为了更精细地描述[电荷](@entry_id:275494)在空间中每一点的流动情况，我们引入一个更为基本的矢量场——**电流密度** (current density) $\vec{J}$。电流密度 $\vec{J}$ 的方向代表了该点正[电荷](@entry_id:275494)的流动方向，其大小 $|\vec{J}|$ 代表了单位时间内垂直通过单位面积的[电荷](@entry_id:275494)量，单位是安培/平方米 ($A/m^2$)。

电流主要分为两类：
1.  **传导电流 (Conduction Current)**：这是最常见的一种电流，由[电荷](@entry_id:275494)载流子（如[金属中的电子](@entry_id:138687)、电解液中的离子）在[电场](@entry_id:194326)作用下于导体内部的漂移运动产生。
2.  **[对流](@entry_id:141806)电流 (Convection Current)**：这种电流并非由载流子在静止介质中运动产生，而是由带电的物体或流体自身的宏观运动所致。如果一个区域的[体电荷密度](@entry_id:264747)为 $\rho(\vec{r})$，并且该区域整体以速度 $\vec{v}$ 运动，那么它就构成了一个[对流](@entry_id:141806)电流，其[电流密度](@entry_id:190690)为：

    $$\vec{J}(\vec{r}) = \rho(\vec{r})\vec{v}$$

    例如，一个[体电荷密度](@entry_id:264747)为 $\rho(r) = \alpha r$ 的绝缘球体以恒定速度 $\vec{v}$ 运动，球体内部任意一点的[电流密度](@entry_id:190690)就是 $\vec{J}(r) = (\alpha r) \vec{v}$。这种电流是[电荷](@entry_id:275494)的宏观输运造成的 [@problem_id:1576188]。

### [传导电流](@entry_id:265343)的微观模型

为了理解传导电流的本质，我们需要深入到材料的微观层面。在导体中，存在着可以自由移动的[电荷](@entry_id:275494)载流子。在没有外[电场](@entry_id:194326)时，这些载流子（例如金属中的自由电子）进行着高速、随机的热运动，但它们的平均速度为零，因此不产生[宏观电流](@entry_id:203974)。

当施加一个[电场](@entry_id:194326) $\vec{E}$ 时，每个[电荷](@entry_id:275494)为 $q$ 的载流子会受到[电场](@entry_id:194326)力 $\vec{F} = q\vec{E}$ 的作用。在力的作用下，它们会加速，但很快会与[晶格](@entry_id:196752)中的原子或其他载流子发生碰撞，失去一部分动能。这个过程不断重复，使得载流子在随机热运动的基础上，叠加了一个沿着（或逆着，取决于 $q$ 的符号）[电场](@entry_id:194326)方向的缓慢的平均速度，这个速度被称为**[漂移速度](@entry_id:262489)** (drift velocity) $\vec{v}_d$。

考虑一个体积元，其中载流子的[数密度](@entry_id:268986)为 $n$（单位体积内的载流子数量）。在时间 $dt$ 内，所有在底面积为 $dA$、高为 $v_d dt$ 的斜柱体内的载流子都会穿过该面积。这个体积是 $v_d dt dA_{\perp}$，其中 $dA_{\perp}$ 是垂直于 $\vec{v}_d$ 的面积分量。穿过的总[电荷](@entry_id:275494)量为 $dQ = (n q) (v_d dt dA_{\perp})$。因此，电流密度的大小 $J = \frac{dQ}{dt dA_{\perp}} = n q v_d$。将其写成矢量形式，我们得到传导电流密度的基本微观表达式：

$$\vec{J} = nq\vec{v}_d$$

如果材料中存在多种类型的载流子，总的[电流密度](@entry_id:190690)是每种载流子贡献的矢量和：

$$\vec{J} = \sum_{i} n_i q_i \vec{v}_{d,i}$$

一个极佳的例子是熔融盐电解质。假设在电解质中，带[电荷](@entry_id:275494)为 $+2e$ 的阳离子以[数密度](@entry_id:268986) $n$、速度 $v$向东漂移，而带[电荷](@entry_id:275494)为 $-e$ 的阴离子以速度 $1.5v$ 向西漂移。首先，由于材料整体呈电中性，阴离子的数密度 $n_{-}$ 必须满足 $(+2e)n + (-e)n_{-} = 0$，即 $n_{-} = 2n$。然后，我们可以计算总[电流密度](@entry_id:190690)。将向东定义为正方向 $(\hat{x})$，阳离子的贡献是 $\vec{J}_c = (n)(+2e)(+v\hat{x}) = 2nev\hat{x}$。阴离子的贡献是 $\vec{J}_a = (2n)(-e)(-1.5v\hat{x}) = 3nev\hat{x}$。注意到，负[电荷](@entry_id:275494)向西运动（负速度）产生的电流与正[电荷](@entry_id:275494)向东运动（正速度）产生的电流方向相同。总[电流密度](@entry_id:190690)为 $\vec{J} = \vec{J}_c + \vec{J}_a = 5nev\hat{x}$。这个例子凸显了正确进行矢量求和的重要性 [@problem_id:1576201]。

### 电流与[电流密度](@entry_id:190690)的关系

电流密度 $\vec{J}$ 是一个局域量，而电流 $I$ 是一个通过有限大小[曲面](@entry_id:267450)的通量。它们之间的关系通过一个[曲面积分](@entry_id:144805)来建立。通过任意一个开放[曲面](@entry_id:267450) $S$ 的总电流 $I$ 等于[电流密度](@entry_id:190690)矢量 $\vec{J}$ 在该[曲面](@entry_id:267450)上的通量：

$$I = \iint_S \vec{J} \cdot d\vec{A}$$

其中 $d\vec{A}$ 是[曲面](@entry_id:267450)上的一个面元矢量，其方向垂直于[曲面](@entry_id:267450)。

在最简单的情况下，如果[电流密度](@entry_id:190690) $\vec{J}$ 是均匀的，并且垂直于一个面积为 $A$ 的平坦[曲面](@entry_id:267450)，那么积分就简化为 $I = J A$。

然而，在许多实际情况中，[电流密度](@entry_id:190690)或[截面](@entry_id:154995)积并非均匀。考虑一个[稳恒电流](@entry_id:271551) $I$ 流过一个[截面](@entry_id:154995)积变化的导体，例如一个圆台形导体，其半径随轴向位置 $z$ 变化， $r(z) = r_0 + \alpha z$。由于电流是稳恒的，通过任何一个垂直于 $z$ 轴的[截面](@entry_id:154995)的总电流 $I$ 都必须是相同的。如果我们假设在任何给定的 $z$ 处，电流密度在其圆形[截面](@entry_id:154995) $A(z) = \pi [r(z)]^2$ 上是均匀的，那么 $I = J(z)A(z)$。由此可得，[电流密度](@entry_id:190690)的大小必须随位置 $z$ 而变化：

$$J(z) = \frac{I}{A(z)} = \frac{I}{\pi(r_0 + \alpha z)^2}$$

这表明，在导体较细的地方，[电流密度](@entry_id:190690)较大；在较粗的地方，[电流密度](@entry_id:190690)较小 [@problem_id:1576220]。

另一个复杂情况是电流密度在[截面](@entry_id:154995)上本身就不是均匀的。例如，在一个方形[截面](@entry_id:154995) ($0 \le x \le a, 0 \le y \le a$) 的长直导体中，电流密度可能因制造工艺而不均匀，比如 $J(x) = C(1 + \frac{x}{a})$。要找到总电流 $I$，我们必须对整个[截面](@entry_id:154995)进行积分：

$$I = \int_0^a \int_0^a J(x) \,dx\,dy = \int_0^a \int_0^a C(1 + \frac{x}{a}) \,dx\,dy$$

通过执行这个积分，我们可以将总电流 $I$ 与常数 $C$ 和尺寸 $a$ 联系起来。在这种情况下，积分结果为 $I = C(\frac{3}{2}a^2)$，因此可以反解出 $C = \frac{2I}{3a^2}$。这个过程展示了如何从一个已知的非均匀电流密度[分布](@entry_id:182848)计算出宏观的总电流 [@problem_id:1576205]。

### 电荷守恒与连续性方程

物理学中最基本的定律之一是 **电荷守恒定律** (principle of charge conservation)：[电荷](@entry_id:275494)不能被凭空创造，也不能被凭空消灭。这个基本原理可以用一个优美的数学方程来表达，即**连续性方程** (continuity equation)。

考虑一个任意的封闭[曲面](@entry_id:267450) $S$，它包围着一个体积 $V$。流出这个封闭[曲面](@entry_id:267450)的净电流是 $\oint_S \vec{J} \cdot d\vec{A}$。根据电荷守恒，流出的净电流必须等于体积 $V$ 内部总[电荷](@entry_id:275494)量 $Q_{in}$ 的减少速率。因此，我们有[连续性方程的积分形式](@entry_id:187481)：

$$\oint_S \vec{J} \cdot d\vec{A} = -\frac{dQ_{in}}{dt} = -\frac{d}{dt}\iiint_V \rho \,dV$$

这里 $\rho$ 是体积[电荷密度](@entry_id:144672)。根据[高斯散度定理](@entry_id:188065)，任何矢量场 $\vec{J}$ 在闭合[曲面](@entry_id:267450)上的通量等于该矢量场的散度在所包围体积内的[体积分](@entry_id:171119)，即 $\oint_S \vec{J} \cdot d\vec{A} = \iiint_V (\nabla \cdot \vec{J}) \,dV$。将此代入上式，并假设体积 $V$ 不随时间变化，我们得到：

$$\iiint_V (\nabla \cdot \vec{J}) \,dV = -\iiint_V \frac{\partial \rho}{\partial t} \,dV$$

由于这个关系式对任何任意选择的体积 $V$ 都必须成立，所以两个被积函数必须处处相等。这就得到了[连续性方程的微分形式](@entry_id:265419)：

$$\nabla \cdot \vec{J} + \frac{\partial \rho}{\partial t} = 0$$

这个方程的物理意义非常深刻：在空间中的任何一点，[电流密度的散度](@entry_id:266331)（即单位体积的净流出率）必须恰好等于该点电荷密度的减少率。如果一个点的 $\nabla \cdot \vec{J} > 0$，说明[电荷](@entry_id:275494)正在从该点流散出去，那么该点的电荷密度 $\rho$ 必须随时间减小。对于[稳恒电流](@entry_id:271551)（steady current），[电荷密度](@entry_id:144672)在任何地方都不随时间变化，即 $\frac{\partial \rho}{\partial t} = 0$，此时[连续性方程](@entry_id:195013)简化为 $\nabla \cdot \vec{J} = 0$。

我们可以利用连续性方程来解决实际问题。例如，如果一个材料内部的电流密度场为 $\vec{J}(x,y,z) = (\alpha x^2)\hat{i} + (\beta y z)\hat{j} - (\gamma z^2)\hat{k}$，我们可以通过计算其散度 $\nabla \cdot \vec{J} = 2\alpha x + \beta z - 2\gamma z$ 来确定局部[电荷密度](@entry_id:144672)的变化率。要找到整个体积内总[电荷](@entry_id:275494)的变化率 $\frac{dQ}{dt}$，我们只需对 $-\nabla \cdot \vec{J}$ 在整个体积上积分即可 [@problem_id:1576209]。

反过来，如果已知电荷密度如何随时间变化，我们可以用[连续性方程](@entry_id:195013)来推断导致这种变化的[电流密度](@entry_id:190690)。假设一个球体内的[电荷密度](@entry_id:144672)随时间和半径变化，$\rho(r,t) = \rho_0 (1 - \frac{r}{R}) \exp(-\frac{t}{\tau})$。其时间导数为 $\frac{\partial \rho}{\partial t} = -\frac{1}{\tau}\rho(r,t)$。代入连续性方程 $\nabla \cdot \vec{J} = -\frac{\partial \rho}{\partial t}$，并考虑到[球对称性](@entry_id:272852)（$\vec{J} = J_r(r,t)\hat{r}$），我们可以对得到的方程进行积分，从而求解出径向[电流密度](@entry_id:190690) $J_r(r,t)$。这个过程揭示了[电荷](@entry_id:275494)耗散必然伴随着内部电流的产生 [@problem_id:1576199]。

### 欧姆定律与材料中的传导

我们已经知道，[电场](@entry_id:194326)可以驱动导体中的电流。对于许多材料（称为**欧姆材料**），[电流密度](@entry_id:190690) $\vec{J}$ 与产生它的[电场](@entry_id:194326) $\vec{E}$ 成正比。这个经验关系被称为**欧姆定律** (Ohm's Law) 的微观形式：

$$\vec{J} = \sigma \vec{E}$$

比例常数 $\sigma$ 是材料的**[电导率](@entry_id:137481)** (conductivity)，单位是西门子/米 (S/m)。电导率的倒数 $\varrho = 1/\sigma$ 被称为**[电阻率](@entry_id:266481)** (resistivity)。

对于**各向同性** (isotropic) 材料，[电导率](@entry_id:137481) $\sigma$ 是一个标量，这意味着电流密度 $\vec{J}$ 的方向总是与[电场](@entry_id:194326) $\vec{E}$ 的方向相同。然而，在**各向异性** (anisotropic) 材料中，如某些晶体或[复合材料](@entry_id:139856)，沿不同方向的导电能力可能不同。在这种情况下，电导率必须用一个张量 $\boldsymbol{\sigma}$ 来描述，[欧姆定律](@entry_id:276027)写为 $\vec{J} = \boldsymbol{\sigma} \vec{E}$，或者用分量形式表示为 $J_i = \sum_j \sigma_{ij} E_j$。此时，电流密度 $\vec{J}$ 和[电场](@entry_id:194326) $\vec{E}$ 通常不平行。

例如，考虑一种二维各向异性材料，其本构关系为 $J_x = \sigma_x E_x$ 和 $J_y = \sigma_y E_y$。如果施加的[电场](@entry_id:194326) $\vec{E}$ 与x轴成 $\theta_E$ 角，即 $E_x = E_0\cos\theta_E, E_y = E_0\sin\theta_E$，那么产生的电流分量为 $J_x = \sigma_x E_0\cos\theta_E$ 和 $J_y = \sigma_y E_0\sin\theta_E$。电流密度 $\vec{J}$ 与x轴的夹角 $\theta_J$ 由 $\tan\theta_J = \frac{J_y}{J_x} = \frac{\sigma_y}{\sigma_x}\tan\theta_E$ 决定。只有当 $\sigma_x = \sigma_y$（各向同性）或[电场](@entry_id:194326)沿主轴方向时，$\theta_J$ 才等于 $\theta_E$ [@problem_id:1576211]。

在处理[时变场](@entry_id:180620)时，[传导电流](@entry_id:265343)只是故事的一部分。麦克斯韦修正[安培定律](@entry_id:140092)时引入了**位移电流密度** (displacement current density) 的概念，$\vec{J}_d = \frac{\partial \vec{D}}{\partial t} = \epsilon \frac{\partial \vec{E}}{\partial t}$（其中 $\vec{D}$ 是[电位移矢量](@entry_id:197092)，$\epsilon$ 是[介电常数](@entry_id:146714)）。它与变化的[电场](@entry_id:194326)相关联，即使在没有[电荷](@entry_id:275494)流动的真空中也可能存在。

在一个有微弱导电性的“漏电”[电容器](@entry_id:267364)中，[传导电流](@entry_id:265343)和位移电流同时存在。当这样一个充电后的[电容器](@entry_id:267364)被断开电路时，其极板上的[电荷](@entry_id:275494)会通过内部的导[电介质](@entry_id:147163)缓慢泄漏，形成一个[传导电流](@entry_id:265343) $\vec{J}_c = \sigma \vec{E}$。[电荷](@entry_id:275494)的减少导致极板间的[电场](@entry_id:194326) $\vec{E}$ 随时间减弱，从而产生一个[位移电流](@entry_id:190231) $\vec{J}_d = \epsilon \frac{d\vec{E}}{dt}$。由于[电容器](@entry_id:267364)是孤立的，通过任何一个[截面](@entry_id:154995)的总电流（[传导电流](@entry_id:265343)+位移电流）必须为零。因此，$\vec{J}_c + \vec{J}_d = 0$，这导致 $\sigma E(t) = -\epsilon \frac{dE(t)}{dt}$。这个关系式将材料的传导特性 $(\sigma)$ 和介电特性 $(\epsilon)$ 与[电场](@entry_id:194326)的动态行为联系在一起，是理解介质中电磁弛豫过程的关键 [@problem_id:1576202]。

### [电流密度](@entry_id:190690)的边界条件

当电流从一种导体流向另一种具有不同电导率的导体时，在它们的交界面上，[电流密度](@entry_id:190690)和[电场](@entry_id:194326)必须满足特定的**边界条件** (boundary conditions)。这些条件可以从电磁学的基本定律推导出来。

1.  **法向分量**：对于[稳恒电流](@entry_id:271551)，[连续性方程](@entry_id:195013)为 $\nabla \cdot \vec{J} = 0$。将此方程应用于跨越界面的一个微小“药盒”形[高斯面](@entry_id:272964)，可以推断出[电流密度](@entry_id:190690)的法向分量在界面上是连续的：
    $$J_{1n} = J_{2n}$$
    这意味着从一侧流入界面的单位面积[电荷](@entry_id:275494)率等于从另一侧流出的[电荷](@entry_id:275494)率，保证了界面上不会有[电荷](@entry_id:275494)的无限积累。

2.  **切向分量**：对于静电场或[稳恒电流](@entry_id:271551)产生的[电场](@entry_id:194326)，环路定律 $\nabla \times \vec{E} = 0$ 成立。将此定律应用于跨越界面的一个微小矩形环路，可以推断出[电场](@entry_id:194326)的切向分量是连续的：
    $$E_{1t} = E_{2t}$$

结合欧姆定律 $\vec{J}=\sigma\vec{E}$，我们可以从这些边界条件中得出更多结论。例如，$E_{1t} = E_{2t}$ 意味着 $\frac{J_{1t}}{\sigma_1} = \frac{J_{2t}}{\sigma_2}$。结合 $J_{1n} = J_{2n}$，我们可以得到电流线在界面处的“[折射定律](@entry_id:165991)”：$\frac{\tan\theta_1}{\tan\theta_2} = \frac{\sigma_1}{\sigma_2}$，其中 $\theta_1$ 和 $\theta_2$ 是电流线与界面法线在两种介质中的夹角。

一个更深刻的问题是：为了维持这些条件，界面上会发生什么？当[稳恒电流](@entry_id:271551)流过界面时，可能会在界面上形成一层静态的**[表面电荷密度](@entry_id:272693)** (surface charge density) $\rho_s$。这层[电荷](@entry_id:275494)的存在是为了调整[电场](@entry_id:194326)，使其既能满足[电场](@entry_id:194326)的边界条件，又能驱动满足电流边界条件的电流。

我们可以通过[电位移矢量](@entry_id:197092) $\vec{D}$ 的边界条件来确定这个表面电荷。[高斯定律](@entry_id:141493)指出，$\vec{D}$ 的法向分量的不连续性等于自由[表面电荷密度](@entry_id:272693)：$D_{1n} - D_{2n} = \rho_s$。利用 $\vec{D} = \epsilon \vec{E}$ 和 $\vec{J} = \sigma \vec{E}$，我们可以将这个方程与电流的边界条件联系起来。

从 $J_{1n} = J_{2n}$，我们有 $\sigma_1 E_{1n} = \sigma_2 E_{2n}$。代入 $\rho_s = \epsilon_1 E_{1n} - \epsilon_2 E_{2n}$，并用 $E_{1n}$ 来表示 $E_{2n}$，我们得到：
$$\rho_s = \epsilon_1 E_{1n} - \epsilon_2 \left(\frac{\sigma_1}{\sigma_2} E_{1n}\right) = E_{1n} \left(\epsilon_1 - \frac{\epsilon_2 \sigma_1}{\sigma_2}\right)$$
再用 $E_{1n} = J_{1n} / \sigma_1$ 代入，其中 $J_{1n} = J_1 \cos\theta_1$ 是入射电流密度的法向分量，最终得到：

$$\rho_s = J_1 \cos\theta_1 \left(\frac{\epsilon_1}{\sigma_1} - \frac{\epsilon_2}{\sigma_2}\right)$$

这个重要的结果表明，只有当两种介质的弛豫时间常数 $\tau = \epsilon/\sigma$ 相等时，界面上才没有表面电荷积累。在一般情况下，[稳恒电流](@entry_id:271551)的通过必然伴随着界面上静态[电荷](@entry_id:275494)的形成，这是一个连接静电学和[稳恒电流](@entry_id:271551)理论的深刻结论 [@problem_id:16055]。
