## 引言
在爱因斯坦的广义相对论中，时空的几何由[度规张量](@entry_id:160222) $g_{\mu\nu}$ 描述，然而，物理定律在普适的弯曲[坐标系](@entry_id:156346)下往往形式复杂。为了解决这一问题并忠实于等效原理——即在任意点总能找到一个物理定律回归狭义相对论形式的局部惯性系——我们需要一个强大的数学工具。[标架场](@entry_id:160577)（tetrad）或[协变](@entry_id:634097)[标架场](@entry_id:160577)（vierbein）形式论应运而生，它恰好为在弯曲时空的每一点上架设这样一个局部平直[参考系](@entry_id:169232)提供了严谨的框架。更根本的是，诸如描述电子的[狄拉克旋量](@entry_id:181944)等物质场是在[洛伦兹群](@entry_id:139964)下变换的，这与[广义坐标](@entry_id:156576)变换群格格不入。[标架场形式](@entry_id:157842)论通过引入一个作为“中间人”的局部洛伦兹标架，完美地解决了将这类自旋场与[引力](@entry_id:175476)理论相耦合的难题。

本文旨在系统性地介绍[标架场形式](@entry_id:157842)论。在“原理与机制”一章中，我们将深入探讨[标架场](@entry_id:160577)的定义、其与[度规张量](@entry_id:160222)的核心关系、选择的自由度，以及描述[标架场](@entry_id:160577)变化的关键概念——[自旋联络](@entry_id:161745)。随后的“应用与跨学科联系”一章将展示这一理论的强大威力，我们将看到如何利用[标架场](@entry_id:160577)计算真实观测者的物理测量值，如何将[费米子](@entry_id:146235)与[引力](@entry_id:175476)耦合，并探讨其在宇宙学和[黑洞](@entry_id:158571)物理中的具体应用。最后，通过“动手实践”中的精选习题，读者将有机会亲手应用这些概念，从而将理论知识转化为解决问题的实用技能。

## 原理与机制

在广义相对论中，我们用度规张量 $g_{\mu\nu}$ 来描述弯曲时空的几何结构。然而，在一个普适的[坐标系](@entry_id:156346)中，物理定律的形式可能变得相当复杂。爱因斯坦的[等效原理](@entry_id:157518)告诉我们，在时空中的任意一点，我们总能找到一个局部惯性参考系，在此[参考系](@entry_id:169232)中，物理定律恢复其在[狭义相对论](@entry_id:275552)中的简洁形式。这启发我们，应当有一种数学工具，能够在弯曲时空的每一点上建立这样一个局部平直（闵可夫斯基）[参考系](@entry_id:169232)。这个工具就是[标架场](@entry_id:160577)（tetrad）或[协变](@entry_id:634097)[标架场](@entry_id:160577)（vierbein）形式论。

更深层次地，引入[标架场](@entry_id:160577)的需求源于某些物理场的内在属性。例如，描述电子等[费米子](@entry_id:146235)的[狄拉克旋量](@entry_id:181944)场，根据其定义，是在[洛伦兹群](@entry_id:139964) $SO(1,3)$（或更准确地说是其双[覆盖群](@entry_id:161571) $Spin(1,3)$）的表示下进行变换的。然而，[广义坐标](@entry_id:156576)变换所诱导的切空间变换属于一般的线性群 $GL(4, \mathbb{R})$。由于 $Spin(1,3)$ 并非 $GL(4, \mathbb{R})$ 的[子群](@entry_id:146164)，我们无法直接将[旋量](@entry_id:158054)场与普适的[坐标变换](@entry_id:172727)联系起来。因此，我们必须在每个时空点上引入一个“中间人”——一个局部洛伦兹[参考系](@entry_id:169232)。[标架场](@entry_id:160577)正是实现这一构想的数学框架，它在每个点上都架设了一座桥梁，连接了弯曲的“世界”坐标和局部的“平直”洛伦兹坐标 [@problem_id:1881205]。

### [标架场](@entry_id:160577)的定义与核心关系

**[标架场](@entry_id:160577)**（tetrad），或更常被称为**[协变](@entry_id:634097)[标架场](@entry_id:160577)**（vierbein），是在[时空流形](@entry_id:262092)每一点上定义的一组四个相互正交的[基矢](@entry_id:199546)量 $\{e_a\}$。这里的拉丁字母索引 $a, b, c, \dots \in \{0, 1, 2, 3\}$ 标记了局部洛伦兹[参考系](@entry_id:169232)中的四个[基矢](@entry_id:199546)量，而希腊字母索引 $\mu, \nu, \lambda, \dots \in \{0, 1, 2, 3\}$ 则代表我们熟悉的时空[坐标基](@entry_id:270149)。矢量 $e_a$ 在[坐标基](@entry_id:270149) $\partial_\mu$ 下的分量记为 $e_a^\mu$。这四个矢量构成一个完备的基，其中 $e_0$ 是类时矢量，而 $e_1, e_2, e_3$ 是类空矢量。

[标架场](@entry_id:160577)的核心在于它联系了描述[弯曲时空](@entry_id:159822)的度规张量 $g_{\mu\nu}(x)$ 和描述局部惯性系的[闵可夫斯基度规](@entry_id:154660) $\eta_{ab} = \text{diag}(-1, 1, 1, 1)$。这种联系通过**正交[归一化条件](@entry_id:156486)**来表达：

$g_{\mu\nu}(x) e_a^\mu(x) e_b^\nu(x) = \eta_{ab}$

这个方程是[标架场形式](@entry_id:157842)论的基石。它表明，在任意一点，当我们用弯曲时空的度规 $g_{\mu\nu}$ 去测量[局部基](@entry_id:151573)矢量 $e_a$ 和 $e_b$ 之间的“[内积](@entry_id:158127)”时，得到的结果总是[闵可夫斯基度规](@entry_id:154660)的常数分量。这意味着 $\{e_a\}$ 确实构成了一个洛伦兹[参考系](@entry_id:169232)。

为了更好地理解这个条件，我们来看一个具体的例子。考虑一个空间平直的弗里德曼-罗伯逊-沃尔克（FRW）宇宙，其线元为 $ds^2 = -dt^2 + a(t)^2 (dx^2 + dy^2 + dz^2)$。这里的 $a(t)$ 是宇宙尺度因子。对于一个随宇宙膨胀而运动的“共动”观测者，一个自然的选择是让其四速度与[坐标时](@entry_id:263720)间方向对齐。因此，我们可以选择局部[参考系](@entry_id:169232)中的类时[基矢](@entry_id:199546)量 $e_0$ 为 $e_0^\mu = (1, 0, 0, 0)$。现在，我们来构造一个与 $e_0$ 正交的类空[基矢](@entry_id:199546)量，例如 $e_1$。根据正交[归一化条件](@entry_id:156486)，我们要求：

1.  $g_{\mu\nu} e_0^\mu e_1^\nu = \eta_{01} = 0$ (正交性)
2.  $g_{\mu\nu} e_1^\mu e_1^\nu = \eta_{11} = 1$ (归一化)

[FRW度规](@entry_id:158831)的非零分量为 $g_{00} = -1$ 和 $g_{11} = g_{22} = g_{33} = a(t)^2$。假设 $e_1^\mu$ 的形式为 $(A, B, 0, 0)$。第一个条件给出 $g_{00} e_0^0 e_1^0 = (-1)(1)(A) = 0$，这意味着 $A=0$。因此 $e_1^\mu$ 必须是纯空间性的，形如 $(0, B, 0, 0)$。接着，第二个条件给出 $g_{11} e_1^1 e_1^1 = a(t)^2 B^2 = 1$。求解 $B$ 得到 $B = \pm 1/a(t)$。选择正解，我们得到 $e_1^1 = 1/a(t)$ [@problem_id:1853747]。通过类似的方法，我们可以构建一个完整的对角[标架场](@entry_id:160577)：

$e_0^\mu = (1, 0, 0, 0), \quad e_1^\mu = (0, 1/a(t), 0, 0), \quad e_2^\mu = (0, 0, 1/a(t), 0), \quad e_3^\mu = (0, 0, 0, 1/a(t))$

这个例子清晰地展示了如何利用正交[归一化条件](@entry_id:156486)，从一个给定的度规出发，构建出与之适配的[标架场](@entry_id:160577)。

反过来，如果我们已知[标架场](@entry_id:160577) $e_a^\mu$，我们也可以重构出时空度规。与正交[归一化条件](@entry_id:156486)等价的关系式是：

$g^{\mu\nu}(x) = \eta^{ab} e_a^\mu(x) e_b^\nu(x)$

$g_{\mu\nu}(x) = \eta_{ab} e^a_\mu(x) e^b_\nu(x)$

在这里，我们引入了**对偶[标架场](@entry_id:160577)**（cotetrad 或 co-vierbein）$e^a_\mu$，它是一组[1-形式](@entry_id:270392)基底，与矢量基 $e_a^\mu$ 互为[逆矩阵](@entry_id:140380)，满足 $e_a^\mu e^a_\nu = \delta^\mu_\nu$ 和 $e^\mu_a e^b_\mu = \delta^b_a$。

考虑一个[德西特宇宙](@entry_id:159695)的例子，其中一个给定的[标架场](@entry_id:160577)（以矩阵形式 $e_a^\mu$ 表示，a为行，$\mu$为列）为 $e_a^\mu = \text{diag}(1, \exp(-Ht), \exp(-Ht), \exp(-Ht))$ [@problem_id:1853751]。我们可以利用上述公式计算[逆变](@entry_id:192290)度规张量 $g^{\mu\nu}$。例如，其 $(0,0)$ 分量为：

$g^{00} = \eta^{ab} e_a^0 e_b^0 = \eta^{00} e_0^0 e_0^0 = (-1)(1)(1) = -1$

其 $(i,j)$ 空间分量（其中 $i,j \in \{1,2,3\}$）为：

$g^{ij} = \eta^{ab} e_a^i e_b^j = \eta^{ij} e_i^i e_j^j = \delta^{ij} \exp(-Ht) \exp(-Ht) = \delta^{ij} \exp(-2Ht)$

最终得到的度规矩阵为 $g^{\mu\nu} = \text{diag}(-1, \exp(-2Ht), \exp(-2Ht), \exp(-2Ht))$。这正是描述一个具有指数膨胀的平直宇宙的度规的逆。

### 物理诠释与选择自由度

[标架场](@entry_id:160577)的强大之处在于其清晰的物理图像。它代表了一个真实观测者所携带的局部测量装置。特别地，**类时[基矢](@entry_id:199546)量 $e_0^\mu$ 被诠释为在该局部[参考系](@entry_id:169232)中静止的观测者的四速度**。如果一个观测者的四速度为 $U^\mu$，那么他在自己的[参考系](@entry_id:169232)（由 $e_a^\mu$ 定义）中测量到的四速度分量 $U^a$ 是通过投影得到的：$U^a = e^a_\mu U^\mu$。

如果这个观测者相对于自己的[参考系](@entry_id:169232)是静止的，那么他的局部[空间速度](@entry_id:190294)分量应该为零，即 $U^i = 0$ for $i=1,2,3$。根据四速度的[归一化条件](@entry_id:156486) $\eta_{ab} U^a U^b = -c^2$（这里我们暂时恢复光速 $c$），我们有 $\eta_{00} (U^0)^2 = -(U^0)^2 = -c^2$，所以 $U^0 = c$（选择未来指向解）。因此，一个在标架[参考系](@entry_id:169232)中静止的观测者，其局部四速度分量为 [@problem_id:1853726]：

$U^a = \begin{pmatrix} c \\ 0 \\ 0 \\ 0 \end{pmatrix}$

这为[标架场](@entry_id:160577)赋予了操作性的意义：$e_0^\mu$ 就是该[参考系](@entry_id:169232)观测者的[世界线](@entry_id:199036)切矢量。

然而，对于给定的时空几何，[标架场](@entry_id:160577)的选择并不是唯一的。在时空的每一点，我们可以对[局部基](@entry_id:151573)矢量进行任意的洛伦兹变换（旋转或加速），得到的新[基矢](@entry_id:199546)量组仍然是正交归一的，因此也是一个有效的[标架场](@entry_id:160577)。这种在每一点独立进行洛伦兹变换的自由度被称为**局部洛伦兹对称性**，这是一种**[规范自由度](@entry_id:160491)**。

如果旧的对偶[标架场](@entry_id:160577)是 $e^a_\mu$，新的对偶[标架场](@entry_id:160577)是 $e'^a_\mu$，它们之间的关系由一个依赖于时空位置的[洛伦兹变换](@entry_id:176827)矩阵 $\Lambda^a_{\ b}(x)$ 决定：

$e'^a_\mu(x) = \Lambda^a_{\ b}(x) e^b_\mu(x)$

例如，在描述史瓦西黑洞外部时空的对角[标架场](@entry_id:160577)基础上，我们可以对局部空间坐标进行一次绕径向（$\hat{r}$ 方向）的旋转，角度为 $\psi$ [@problem_id:1853717]。新的对偶[标架场](@entry_id:160577)分量，如 $e'^{\hat{\theta}}_{\phi}$，将由旧分量混合而成：

$e'^{\hat{\theta}}_{\phi} = \Lambda^{\hat{\theta}}{}_{\hat{b}} e^{\hat{b}}_{\phi} = \Lambda^{\hat{\theta}}{}_{\hat{\theta}} e^{\hat{\theta}}_{\phi} + \Lambda^{\hat{\theta}}{}_{\hat{\phi}} e^{\hat{\phi}}_{\phi}$

由于初始标架是对角的 ($e^{\hat{\theta}}_{\phi} = 0$)，该表达式简化为 $e'^{\hat{\theta}}_{\phi} = \Lambda^{\hat{\theta}}{}_{\hat{\phi}} e^{\hat{\phi}}_{\phi} = (\sin\psi)(r \sin\theta)$。这表明新的[标架场](@entry_id:160577)可以是“非对角”的，反映了观测者的[局部坐标](@entry_id:181200)轴相对于“自然”的[球坐标](@entry_id:146054)方向发生了旋转。

同样，我们也可以施加一个局部的[洛伦兹助推](@entry_id:201972)（boost）。在描述[匀加速](@entry_id:268628)观测者的林德勒时空中，对一个静止标架施加一个沿 $e_1$ 方向、[快度](@entry_id:265131)为 $\phi$ 的助推，会改变标架矢量的分量 $e_a^\mu$ [@problem_id:1853727]。例如，新的类时矢量 $e'_0$ 将是旧的类时和类空矢量的线性组合：$e'_0 = (\cosh\phi) e_0 - (\sinh\phi) e_1$。这对应于一个相对于[林德勒坐标](@entry_id:183624)系运动的新的加速观测者。

### 导数与[自旋联络](@entry_id:161745)

当我们在[弯曲时空](@entry_id:159822)中移动时，不仅坐标会变化，我们为定义物理定律而建立的局部洛伦兹[参考系](@entry_id:169232)本身也会从一点到另一点发生“转动”。这意味着，对于一个带有局部洛伦兹指标的场（如旋量场 $\psi$ 或[标架场](@entry_id:160577) $e^a_\mu$ 本身），普通的偏导数 $\partial_\mu$ 不再是协变的。我们需要一种新的导数来计入这种局部[参考系](@entry_id:169232)的逐点变化。

这个新工具就是**[自旋联络](@entry_id:161745)**（spin connection），记为 $\omega_\mu{}^a{}_b$。它是一个类似于规范场论中[规范势](@entry_id:188985)的量，其作用是“补偿”局部[洛伦兹变换](@entry_id:176827)在[微分](@entry_id:158718)过程中产生的不[协变](@entry_id:634097)项。[自旋联络](@entry_id:161745)精确地描述了当沿着 $\mu$ 方向移动时，[局部基](@entry_id:151573)矢量 $e_b$ 相对于 $e_a$ 的无穷小转动。

决定[自旋联络](@entry_id:161745)的基本原则是**标架约定**（tetrad postulate），它要求标架矢量在同时考虑[广义坐标](@entry_id:156576)变换和局部洛伦兹变换的“完全”协变导数下是守恒的（即平行输运时保持不变）。这最终给出了[自旋联络](@entry_id:161745)的表达式：

$\omega_{\mu b}{}^a = e^a_\nu \left( \partial_\mu e^\nu_b + \Gamma^\nu_{\mu\lambda} e^\lambda_b \right)$

这里，$\Gamma^\nu_{\mu\lambda}$ 是通常的克里斯托费尔符号（[Levi-Civita联络](@entry_id:161107)），它描述了[坐标基](@entry_id:270149)矢量的变化。这个公式表明，[自旋联络](@entry_id:161745)由两部分贡献：一部分来自[标架场](@entry_id:160577)分量自身的显式变化（$\partial_\mu e^\nu_b$），另一部分来自背景时空的曲率（通过 $\Gamma^\nu_{\mu\lambda}$）。

-   **最简单情况**：在平直的[闵可夫斯基空间](@entry_id:160715)中，使用笛卡尔坐标系，我们可以选择最简单的“笛卡尔标架” $e^\mu_a = \delta^\mu_a$。此时，度规是常数，因此所有克里斯托费尔符号 $\Gamma^\nu_{\mu\lambda}$ 均为零。[标架场](@entry_id:160577)分量也是常数，所以它们的导数 $\partial_\mu e^\nu_b$ 也为零。因此，[自旋联络](@entry_id:161745)的所有分量都为零：$\omega_{\mu b}{}^a = 0$ [@problem_id:1853739]。这符合我们的直觉：在一个全局[惯性系](@entry_id:266190)中，局部[参考系](@entry_id:169232)无需任何转动就可以处处保持一致。

-   **[平直空间](@entry_id:204618)中的非平凡情况**：考虑用极坐标 $(r, \theta)$ 描述的二维平面。尽管空间是平直的，但[坐标系](@entry_id:156346)本身是“弯曲”的。一个自然的对偶标架选择是 $e^1 = dr, e^2 = r d\theta$。这里的标架分量 $e^2_\theta = r$ 依赖于坐标 $r$。这导致了非零的[克里斯托费尔符号](@entry_id:159831)（如 $\Gamma^r_{\theta\theta}=-r$）和非零的[自旋联络](@entry_id:161745)，即使[黎曼曲率张量](@entry_id:160189)为零。

计算[自旋联络](@entry_id:161745)有一种更优雅的方法，即使用**[嘉当结构方程](@entry_id:162100)**。对于无挠率时空，第一[结构方程](@entry_id:274644)为：

$de^a + \omega^a{}_b \wedge e^b = 0$

其中 $d$ 是外微分算子，$\wedge$ 是[楔积](@entry_id:147029)。$e^a$ 和 $\omega^a{}_b$ 都是微分形式。对于上述二维平面的例子，我们有 $de^1 = 0$ 和 $de^2 = d(r d\theta) = dr \wedge d\theta$。将 $dr = e^1$ 和 $d\theta = e^2/r$ 代入，得到 $de^2 = \frac{1}{r} e^1 \wedge e^2$。[结构方程](@entry_id:274644) $de^2 + \omega^2{}_1 \wedge e^1 = 0$（利用 $\omega^2{}_2=0$）给出 $\omega^2{}_1 \wedge e^1 = -de^2 = -\frac{1}{r} e^1 \wedge e^2$。由此可以解出 $\omega^2{}_1 = \frac{1}{r} e^2 = d\theta$ [@problem_id:1876076]。这个结果 $\omega^2{}_1 = d\theta$ 简洁地表明，当你绕着原点移动时，局部笛卡尔基底必须以与角坐标变化相同的速率转动，以始终将一个轴指向径向。

-   **内禀弯曲空间**：在一个半径为 $R$ 的二维球面上，时空本身是弯曲的。即使我们选择一个自然的对角标架 $e^1_\theta = R, e^2_\phi = R \sin\theta$，由于球面的内禀曲率，[平行输运](@entry_id:160671)一个矢量也会导致其方向改变。这反映在非零的[自旋联络](@entry_id:161745)上。通过计算，可以得到例如 $\omega_{\phi~2}^{~1} = -\cos\theta$ [@problem_id:1876112]，这反映了在球面上沿恒定纬度线移动时，[局部基](@entry_id:151573)底的转动情况。

### [自旋联络](@entry_id:161745)的规范场性质

[自旋联络](@entry_id:161745)作为局部洛伦兹对称性的规范场，其变换性质是其核心特征。在一次[局域洛伦兹变换](@entry_id:158544) $e'^a_\mu = \Lambda^a_{\ b}(x) e^b_\mu$ 下，[自旋联络](@entry_id:161745)的变换规律（以矩阵形式表示）为：

$\omega'_\mu = \Lambda \omega_\mu \Lambda^{-1} + \Lambda (\partial_\mu \Lambda^{-1})$

这个变换由两部分组成：一个“齐次”部分 $\Lambda \omega_\mu \Lambda^{-1}$，这与张量的变换类似；以及一个“非齐次”部分 $\Lambda (\partial_\mu \Lambda^{-1})$，它依赖于[变换矩阵](@entry_id:151616)自身的梯度。这正是杨-米尔斯[规范场](@entry_id:159627)的标志性变换规律。

我们可以通过一个例子来验证这一点。在二维平面上，对初始[自旋联络](@entry_id:161745) $\omega_{\theta 12} = -1$ 施加一个位置相关的旋转 $\Lambda(\theta)$ [@problem_id:1539741]。计算表明，新的[自旋联络](@entry_id:161745) $\omega'_{\theta 12}$ 的齐次部分变为 $\Lambda_1^{\ c}\Lambda_2^{\ d} \omega_{\theta cd} = -1$，而非齐次部分 $(\Lambda \partial_\theta \Lambda^{-1})_{12}$ 则贡献了一个额外的项。如果旋转角度为 $\alpha(\theta) = k\theta$，这个额外项就是 $-k$。因此，变换后的[自旋联络](@entry_id:161745)为 $\omega'_{\theta 12} = -1 - k$。这个非齐次项的存在是至关重要的，它保证了物理定律在局部[洛伦兹变换](@entry_id:176827)下的协变性。

### 应用：将旋量场耦合到[引力](@entry_id:175476)

现在我们拥有了所有必要的工具，可以回到最初的问题：如何构建一个与[引力](@entry_id:175476)相互作用的[狄拉克旋量](@entry_id:181944)场的理论。

1.  **定义[旋量协变导数](@entry_id:185871)**：为了抵消在求导时由局部洛伦兹变换引起的非协变项，我们定义[旋量](@entry_id:158054)的[协变导数](@entry_id:152476) $D_\mu$ 如下：

    $D_\mu \psi = \left( \partial_\mu + \frac{1}{4} \omega_{\mu ab} \gamma^{ab} \right) \psi$

    其中 $\gamma^a$ 是通常的常数[狄拉克伽马矩阵](@entry_id:196236)，而 $\gamma^{ab} = \frac{1}{2}[\gamma^a, \gamma^b]$ 是[旋量表示](@entry_id:141362)下洛伦兹[群的生成元](@entry_id:137215)。正是[自旋联络](@entry_id:161745) $\omega_{\mu ab}$ 的[非齐次变换](@entry_id:185246)部分，精确地抵消了对[旋量变换](@entry_id:161968)矩阵 $S(\Lambda(x))$ 求导时产生的附加项，从而确保了 $D_\mu \psi$ 在局部[洛伦兹变换](@entry_id:176827)下能够正确地协变。

2.  **构建广义协变的狄拉克作用量**：利用这些工具，我们可以写出描述一个质量为 $m$ 的[狄拉克场](@entry_id:156753)在[弯曲时空](@entry_id:159822)中运动的作用量：

    $S = \int d^4x \, e \, \bar{\psi} (i\gamma^\mu D_\mu - m) \psi$

    在这个作用量中：
    -   $e = \det(e^a_\mu) = \sqrt{-g}$ 是[体积元](@entry_id:267802)因子，确保积分在[广义坐标](@entry_id:156576)变换下是[不变量](@entry_id:148850)。
    -   $\gamma^\mu(x) = e^\mu_a(x) \gamma^a$ 是**依赖于位置的伽马矩阵**。它们通过[标架场](@entry_id:160577) $e^\mu_a$ 将平直空间的伽马矩阵 $\gamma^a$ “提升”到[弯曲时空](@entry_id:159822)中，并满足弯曲空间的[克利福德代数](@entry_id:137625) $\{\gamma^\mu, \gamma^\nu\} = 2g^{\mu\nu}$。
    -   $D_\mu$ 是我们上面定义的[旋量协变导数](@entry_id:185871)，它确保了[拉格朗日量](@entry_id:174593)在局部[洛伦兹变换](@entry_id:176827)下的不变性。

综上所述，[标架场](@entry_id:160577)和[自旋联络](@entry_id:161745)形式论不仅是实现等效原理的优雅工具，更是将具有自旋的物质场（如构成宇宙中大部分物质的[费米子](@entry_id:146235)）纳入广义相对论框架的根本要求。它通过引入局部惯性系的“[规范自由度](@entry_id:160491)”，并将[自旋联络](@entry_id:161745)作为相应的规范场，成功地将[引力](@entry_id:175476)的几何描述与物质场的量子力学描述统一在了一个自洽的理论框架之内。
