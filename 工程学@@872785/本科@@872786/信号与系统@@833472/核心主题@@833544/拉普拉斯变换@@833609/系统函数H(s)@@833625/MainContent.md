## 引言
在[信号与系统](@entry_id:274453)领域，理解和分析线性时不变（LTI）系统是核心任务之一。虽然时域中的卷积运算为我们提供了精确的数学描述，但其复杂性往往给[系统分析](@entry_id:263805)与设计带来挑战。为了克服这一障碍，工程师和科学家们转向变换域，引入了一个更为强大和直观的工具——[系统函数](@entry_id:267697)H。它不仅将繁琐的卷积运算简化为简单的代[数乘](@entry_id:155971)法，更深刻地揭示了系统内在的动态特性。

本文旨在系统性地介绍[系统函数](@entry_id:267697)的核心知识。我们将首先在“原理与机制”一章中，深入探讨[系统函数](@entry_id:267697)的定义、推导方法，并揭示其[极点和零点](@entry_id:262457)如何决定系统的稳定性、因果性与频率响应。接着，在“应用与跨学科联系”一章中，我们将展示[系统函数](@entry_id:267697)如何在电气工程、数字信号处理和控制系统等领域中发挥关键作用，将理论与实际应用紧密联系。最后，通过“动手实践”部分，您将有机会通过解决具体问题来巩固所学知识，真正掌握这一分析动态系统的有力武器。

## 原理与机制

在对线性时不变（LTI）系统的研究中，我们从时域的卷积运算过渡到变换域的代数运算，极大地简化了分析过程。这一转变的核心是**[系统函数](@entry_id:267697)（System Function）**，通常用 $H(s)$ 表示[连续时间系统](@entry_id:276553)，用 $H(z)$ 表示离散时间系统。本章将深入探讨[系统函数](@entry_id:267697)的定义、推导方法，并揭示其如何通过[极点和零点](@entry_id:262457)的[分布](@entry_id:182848)，揭示系统的内在特性，如稳定性、因果性和频率选择性。

### [系统函数](@entry_id:267697)：一种变换域视角

对于一个[LTI系统](@entry_id:271946)，其输入与输出的关系在时域中由卷积描述。然而，变换域提供了一种更简洁的表示。

[系统函数](@entry_id:267697)最基础的定义是系统**冲激响应（impulse response）**的变换。对于连续时间（CT）系统，[系统函数](@entry_id:267697) $H(s)$ 是其冲激响应 $h(t)$ 的拉普拉斯变换：

$$H(s) = \mathcal{L}\{h(t)\} = \int_{-\infty}^{\infty} h(t) \exp(-st) dt$$

对于离散时间（DT）系统，[系统函数](@entry_id:267697) $H(z)$ 是其冲激响应 $h[n]$ 的[Z变换](@entry_id:157804)：

$$H(z) = \mathcal{Z}\{h[n]\} = \sum_{n=-\infty}^{\infty} h[n] z^{-n}$$

这一定义揭示了[系统函数](@entry_id:267697)的本质：它是系统固有动态特性的[频域](@entry_id:160070)或复[频域](@entry_id:160070)表示。然而，在实践中，我们更常使用一个等效的操作性定义。根据[拉普拉斯变换](@entry_id:159339)和Z变换的[卷积定理](@entry_id:264711)，输出信号的变换等于输入信号的变换与[系统函数](@entry_id:267697)的乘积。即 $Y(s) = X(s)H(s)$ 以及 $Y(z) = X(z)H(z)$。由此，我们可以将[系统函数](@entry_id:267697)定义为输出变换与输入变换的比值：

$$H(s) = \frac{Y(s)}{X(s)}$$
$$H(z) = \frac{Y(z)}{X(z)}$$

这里必须强调一个至关重要的条件：这一定义仅在**系统初始状态为零（at rest）**时成立。也就是说，输出 $y(t)$ 或 $y[n]$ 完全是由输入 $x(t)$ 或 $x[n]$ 引起的，不包含任何由非零初始条件产生的响应。这种响应被称为**[零状态响应](@entry_id:273280)（zero-state response）**。

一个系统的总响应是[零状态响应](@entry_id:273280)与**[零输入响应](@entry_id:274925)（zero-input response）**之和。[零输入响应](@entry_id:274925)是仅由系统初始条件（如电容电压、电感电流或之前时刻的输出值）产生的输出，此时输入为零。[系统函数](@entry_id:267697) $H(s)$ 是系统的一个内在属性，它只描述系统如何对外部输入做出反应，因此它与初始条件无关。[@problem_id:1766330] 这意味着，无论系统是否从静止状态启动，其[系统函数](@entry_id:267697) $H(s)$ 或 $H(z)$ 都是固定不变的。

### [系统函数](@entry_id:267697)的推导

根据不同的系统描述方式，我们可以采用多种方法来确定其[系统函数](@entry_id:267697)。

#### 从冲激响应出发

如果一个[LTI系统](@entry_id:271946)的冲激响应 $h(t)$ 或 $h[n]$ 已知，我们可以直接通过计算其拉普拉斯变换或Z变换来得到[系统函数](@entry_id:267697)。

例如，考虑一个[连续时间系统](@entry_id:276553)，其冲激响应为一个衰减指数函数 $h(t) = A \exp(-at) u(t)$，其中 $u(t)$ 为[单位阶跃函数](@entry_id:268807)。其[系统函数](@entry_id:267697) $H(s)$ 就是该函数的拉普拉斯变换：
$$H(s) = \mathcal{L}\{A \exp(-at) u(t)\} = \frac{A}{s+a}$$
这是一个典型的一阶低通滤波器的[系统函数](@entry_id:267697)。[@problem_id:1766339]

类似地，对于一个[离散时间系统](@entry_id:263935)，如果其冲激响应是一个几何序列 $h[n] = a^n u[n]$，其中 $u[n]$ 为单位阶跃序列，其[系统函数](@entry_id:267697) $H(z)$ 是这个序列的Z变换。这是一个标准的几何级数求和：
$$H(z) = \sum_{n=0}^{\infty} a^n z^{-n} = \sum_{n=0}^{\infty} (az^{-1})^n = \frac{1}{1 - az^{-1}}$$
为了方便分析，我们通常将其表示为 $z$ 的正幂次有理函数，通过分子分母同乘 $z$ 实现：
$$H(z) = \frac{z}{z - a}$$
这在数字信号处理中，例如设计一个简单的混响效果时非常常见。[@problem_id:1766317]

#### 从线性[常系数](@entry_id:269842)方程出发

许多物理系统和数字滤波器是通过[线性常系数微分方程](@entry_id:276881)（LCCDE）或[差分方程](@entry_id:262177)来描述的。这是推导[系统函数](@entry_id:267697)最常见和强大的方法。其核心思想是对整个方程应用[拉普拉斯变换](@entry_id:159339)或[Z变换](@entry_id:157804)，并利用变换的[微分](@entry_id:158718)或[时移性质](@entry_id:275667)。

对于一个由[微分方程](@entry_id:264184)描述的[连续时间系统](@entry_id:276553)，例如一个[磁悬浮](@entry_id:275771)车辆的垂直悬挂系统模型，其运动方程为：
$$m\frac{d^2y(t)}{dt^2} + b\frac{dy(t)}{dt} + ky(t) = x(t)$$
其中 $x(t)$ 是输入的控制力，$y(t)$ 是输出的位移。[@problem_id:1766349] 为了求[系统函数](@entry_id:267697) $H(s) = Y(s)/X(s)$，我们对方程两边取拉普拉斯变换，并应用[微分性质](@entry_id:275298) $\mathcal{L}\{\frac{d^n y(t)}{dt^n}\} = s^n Y(s)$（假设初始条件为零）：
$$m s^2 Y(s) + b s Y(s) + k Y(s) = X(s)$$
对方程进行[因式分解](@entry_id:150389)：
$$(m s^2 + b s + k) Y(s) = X(s)$$
整理得到[系统函数](@entry_id:267697)：
$$H(s) = \frac{Y(s)}{X(s)} = \frac{1}{m s^2 + b s + k}$$
这种方法将一个复杂的[微分方程](@entry_id:264184)[问题转换](@entry_id:274273)为了一个简单的代数问题。

同样的方法也适用于由差分方程描述的[离散时间系统](@entry_id:263935)。考虑一个由以下方程描述的因果数字滤波器：
$$y[n] - a y[n-1] = x[n] + b x[n-2]$$
我们对方程两边取Z变换，并应用[时移性质](@entry_id:275667) $\mathcal{Z}\{f[n-k]\} = z^{-k} F(z)$（假设系统初始为静止）：
$$Y(z) - a z^{-1} Y(z) = X(z) + b z^{-2} X(z)$$
进行[因式分解](@entry_id:150389)：
$$(1 - a z^{-1}) Y(z) = (1 + b z^{-2}) X(z)$$
整理得到[系统函数](@entry_id:267697)：
$$H(z) = \frac{Y(z)}{X(z)} = \frac{1 + b z^{-2}}{1 - a z^{-1}}$$
为了便于分析，我们通常通过乘以 $z$ 的适当幂次来消除负幂次。例如，如果给定 $a=0.7$ 和 $b=-0.4$，我们乘以 $z^2/z^2$：
$$H(z) = \frac{1 - 0.4 z^{-2}}{1 - 0.7 z^{-1}} \cdot \frac{z^2}{z^2} = \frac{z^2 - 0.4}{z^2 - 0.7 z}$$
这个有理函数形式对于后续的[极零点分析](@entry_id:192470)至关重要。[@problem_id:1766522]

### [LTI系统](@entry_id:271946)的互连

[系统函数](@entry_id:267697)的一个巨大优势在于它能极其简便地分析由多个子系统构成的复杂系统。

**级联（Cascade Connection）**：当两个[LTI系统](@entry_id:271946) $H_1$ 和 $H_2$ 级联时，第一个系统的输出是第二个系统的输入。在时域中，总的冲激响应是 $h(t) = h_1(t) * h_2(t)$。根据卷积定理，在变换域中，这对应于[系统函数](@entry_id:267697)的乘积：
$$H_{total}(s) = H_1(s) H_2(s)$$
例如，如果一个系统由一个冲激响应为 $h_1(t) = \exp(-2t)u(t)$ 的滤波器和一个冲激响应为 $h_2(t) = 5\delta(t) - \exp(-3t)u(t)$ 的整形单元级联而成 [@problem_id:1766339]，它们的[系统函数](@entry_id:267697)分别为 $H_1(s) = \frac{1}{s+2}$ 和 $H_2(s) = 5 - \frac{1}{s+3} = \frac{5(s+3)-1}{s+3} = \frac{5s+14}{s+3}$。总的[系统函数](@entry_id:267697)就是：
$$H(s) = H_1(s) H_2(s) = \frac{1}{s+2} \cdot \frac{5s+14}{s+3} = \frac{5s+14}{s^2+5s+6}$$

**并联（Parallel Connection）**：当两个[LTI系统](@entry_id:271946) $H_1$ 和 $H_2$ 并联时，它们接收相同的输入，而总输出是它们各自输出的和。在时域中 $y(t) = y_1(t) + y_2(t)$。由于变换的线性性质，总的[系统函数](@entry_id:267697)是各个子[系统函数](@entry_id:267697)的和：
$$H_{total}(s) = H_1(s) + H_2(s)$$
考虑一个由增益为 $K$ 的比例环节 $H_A(s) = K$ 与一个由[微分器](@entry_id:272992) $H_{B1}(s)=s$ 和低通滤波器 $H_{B2}(s) = \frac{1}{\tau s + 1}$ 级联构成的支路并联的系统 [@problem_id:1766334]。B支路的[系统函数](@entry_id:267697)是 $H_B(s) = H_{B1}(s)H_{B2}(s) = \frac{s}{\tau s + 1}$。整个并联系统的总[系统函数](@entry_id:267697)为：
$$H(s) = H_A(s) + H_B(s) = K + \frac{s}{\tau s + 1} = \frac{K(\tau s + 1) + s}{\tau s + 1} = \frac{(K\tau+1)s+K}{\tau s+1}$$

### 极点、零点与系统行为

对于由线性[常系数](@entry_id:269842)方程描述的系统，其[系统函数](@entry_id:267697)总是一个有理函数，即两个多项式的比：
$$H(s) = \frac{N(s)}{D(s)} \quad \text{or} \quad H(z) = \frac{N(z)}{D(z)}$$

#### [极零点图](@entry_id:271787)

- **零点（Zeros）** 是使分子多项式为零的复数值，即 $N(s)=0$ 或 $N(z)=0$ 的根。在这些频率点上，系统的响应为零。
- **极点（Poles）** 是使分母多项式为零的复数值，即 $D(s)=0$ 或 $D(z)=0$ 的根。在这些频率点上，[系统函数](@entry_id:267697)的幅度趋于无穷大。极点决定了系统的自然响应模式（natural modes）。

将这些[极点和零点](@entry_id:262457)绘制在复平面（[s平面](@entry_id:271584)或z平面）上，就构成了**[极零点图](@entry_id:271787)**，这是理解和设计[LTI系统](@entry_id:271946)的强大可视化工具。例如，一个由[差分方程](@entry_id:262177) $y[n] - 0.9 y[n-2] = x[n] + x[n-1]$ 描述的系统 [@problem_id:1766551]，其[系统函数](@entry_id:267697)为：
$$H(z) = \frac{1+z^{-1}}{1-0.9z^{-2}} = \frac{z(z+1)}{z^2 - 0.9}$$
其零点是分子 $z(z+1)=0$ 的根，即 $z=0$ 和 $z=-1$。其极点是分母 $z^2 - 0.9 = 0$ 的根，即 $z = \pm\sqrt{0.9} \approx \pm 0.949$。这些点的位置蕴含了关于系统行为的丰富信息。

#### 极点、[因果性与稳定性](@entry_id:260582)

**有界输入有界输出（BIBO）稳定性**是[系统设计](@entry_id:755777)中的一个核心要求。一个稳定的系统对于任何有界的输入，其输出也必须是有界的。这一性质与系统[函数的[极](@entry_id:189069)点位置](@entry_id:271565)和收敛域（Region of Convergence, ROC）紧密相关。

对于**[连续时间系统](@entry_id:276553)**，稳定性的条件是其冲激响应 $h(t)$ 是绝对可积的，即 $\int_{-\infty}^{\infty} |h(t)| dt \lt \infty$。在[s域](@entry_id:260604)中，这等价于**系统[函数的收敛](@entry_id:152305)域（ROC）必须包含虚轴 $j\omega$**。
- **因果性**要求系统的ROC是一个位于最右边极点右侧的[右半平面](@entry_id:277010)。
- **稳定性**要求ROC包含 $j\omega$ 轴。

这两个条件有时会发生冲突。考虑一个系统，其极点位于 $s = -2$ 和 $s = +1$ [@problem_id:1766352]。
1.  如果系统是**因果**的，其ROC必须是 $\operatorname{Re}\{s\} > 1$。这个区域不包含 $j\omega$ 轴，因此系统是**不稳定**的（因为极点 $+1$ 对应于一个[指数增长](@entry_id:141869)项 $\exp(t)$）。
2.  如果想让系统是**稳定**的，ROC必须包含 $j\omega$ 轴。对于这两个极点，唯一包含 $j\omega$ 轴的ROC是带状区域 $-2 \lt \operatorname{Re}\{s\} \lt 1$。然而，这个ROC既不是最右边极点的[右半平面](@entry_id:277010)，也不是最左边极点的左半平面，因此对应的系统是**非因果**的。

结论是：如果一个[LTI系统](@entry_id:271946)在[s平面](@entry_id:271584)的[右半平面](@entry_id:277010)（$\operatorname{Re}\{s\} > 0$）中存在任何极点，那么该系统**不可能同时是因果和稳定**的。对于一个**因果[LTI系统](@entry_id:271946)**，其稳定的充要条件是**所有极点都必须位于s平面的左半平面（LHP）**。

对于**[离散时间系统](@entry_id:263935)**，稳定性的条件是其冲激响应 $h[n]$ 是绝对可和的，即 $\sum_{n=-\infty}^{\infty} |h[n]| \lt \infty$。在z域中，这等价于**[系统函数](@entry_id:267697)的ROC必须包含单位圆 $|z|=1$**。
- **因果性**要求系统的ROC是位于最外层极点外部的区域。
- **稳定性**要求ROC包含[单位圆](@entry_id:267290)。

因此，对于一个**因果[离散时间LTI系统](@entry_id:188941)**，其稳定的充要条件是**所有极点都必须位于z平面的单位圆内部**，即所有极点 $p_k$ 的模都必须满足 $|p_k| \lt 1$。

让我们通过一个例子来理解这一点 [@problem_id:1766338]。一个因果系统的[极点位置](@entry_id:271565)由参数 $k$ 控制，位于 $p = (1-k)\exp(\pm j\omega_0)$。为了使系统稳定，极点的模必须小于1：
$$|p| = |(1-k)\exp(\pm j\omega_0)| = |1-k| \cdot |\exp(\pm j\omega_0)| = |1-k|$$
稳定性条件是 $|1-k| \lt 1$。解这个不等式：
$$-1 \lt 1-k \lt 1$$
$$-2 \lt -k \lt 0$$
$$0 \lt k \lt 2$$
这表明，只有当参数 $k$ 在 $(0, 2)$ 区间内取值时，极点才会落在[单位圆](@entry_id:267290)内，从而保证[因果系统](@entry_id:264914)的稳定性。如果 $k=0$ 或 $k=2$，极点正好在[单位圆](@entry_id:267290)上，系统是临界稳定的；如果 $k$ 在此范围之外，系统将变得不稳定。

#### [频率响应](@entry_id:183149)的几何评估

[极零点图](@entry_id:271787)不仅能判断稳定性，还能直观地评估系统的**频率响应** $H(j\omega)$ 或 $H(e^{j\omega})$。[频率响应](@entry_id:183149)的**幅度**可以通过几何方法来估计。

对于一个离散时间系统 $H(z) = G \frac{\prod_k (z-z_k)}{\prod_i (z-p_i)}$，其在[单位圆](@entry_id:267290)上某一点 $z = e^{j\omega}$ 的频率响应幅度为：
$$|H(e^{j\omega})| = |G| \frac{\prod_k |e^{j\omega} - z_k|}{\prod_i |e^{j\omega} - p_i|}$$
这里，$|e^{j\omega} - z_k|$ 是从[单位圆](@entry_id:267290)上的点 $e^{jω}$ 到零点 $z_k$ 的距离，而 $|e^{j\omega} - p_i|$ 是到极点 $p_i$ 的距离。因此，[频率响应](@entry_id:183149)的幅度是**从频率点到所有零点的距离乘积，除以到所有极点的距离乘积**。

- 如果单位圆上的某个频率点靠近一个**零点**，则该频率的响应幅度会很小（被衰减）。
- 如果单位圆上的某个频率点靠近一个**极点**，则该频率的响应幅度会很大（被放大）。

这为滤波器设计提供了直观的思路。例如，要设计一个**低通滤波器**，我们可以在高频区域（例如 $z=-1$ 附近）放置零点，在低频区域（例如 $z=1$ 附近）放置极点。

考虑一个在 $z_0=1$ 处有零点，在 $p_0=-0.8$ 处有极点的系统 [@problem_id:1766308]。
- 零点在 $z=1$ 处，对应于直流（DC）频率（$\omega=0$）。这意味着 $|H(e^{j0})| = |H(1)| = 0$，系统完全阻断[直流分量](@entry_id:272384)。
- 极点在 $z=-0.8$ 处，非常靠近最高频率点 $z=-1$（对应 $\omega=\pi$）。这预示着系统会在高频处有显著的增益。

我们可以定量地验证这一点。我们计算最高频率（$\omega=\pi$）和中频（$\omega=\pi/2$）的增益比值 $\frac{|H(e^{j\pi})|}{|H(e^{j\pi/2})|}$。
- 在最高频率 $\omega=\pi$，$z=e^{j\pi}=-1$。到零点 $z_0=1$ 的距离是 $|-1 - 1| = 2$。到极点 $p_0=-0.8$ 的距离是 $|-1 - (-0.8)| = 0.2$。幅度（忽略增益常数）与 $\frac{2}{0.2} = 10$ 成正比。
- 在中频 $\omega=\pi/2$，$z=e^{j\pi/2}=j$。到零点的距离是 $|j - 1| = \sqrt{2}$。到极点的距离是 $|j - (-0.8)| = |0.8+j| = \sqrt{0.8^2+1^2} = \sqrt{1.64}$。幅度与 $\frac{\sqrt{2}}{\sqrt{1.64}}$ 成正比。

因此，比值为：
$$\frac{|H(e^{j\pi})|}{|H(e^{j\pi/2})|} = \frac{2 / 0.2}{\sqrt{2} / \sqrt{1.64}} = 10 \frac{\sqrt{1.64}}{\sqrt{2}} = 10\sqrt{0.82} \approx 9.06$$
这个巨大的比值证实了我们的几何直觉：该系统显著放大了高频信号，同时抑制了低频信号，表现出高通滤波器的特性。

综上所述，[系统函数](@entry_id:267697)不仅是一种数学上的便利工具，它更是连接系统时域行为与[频域](@entry_id:160070)特性的桥梁。通过分析其有理函数形式，特别是极点和零点的位置，我们可以深刻理解和预测一个[LTI系统](@entry_id:271946)的稳定性、因果性以及其作为滤波器的频率选择能力。
