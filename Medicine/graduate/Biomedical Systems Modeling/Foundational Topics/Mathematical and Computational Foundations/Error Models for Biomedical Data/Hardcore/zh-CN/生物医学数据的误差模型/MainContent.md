## 引言
在生物医学研究的广阔领域中，从单次生化检测到大规模[基因组测序](@entry_id:916422)，我们收集的几乎所有数据都不可避免地伴随着误差。这些误差可能源于仪器本身的局限性、实验条件的微[小波](@entry_id:636492)动，或是[数据采集](@entry_id:273490)过程中的系统性偏差。长期以来，研究者们常常在分析中忽略或简单处理这些测量误差，但这可能导致严重的后果，如错误地评估风险因素的效应大小，甚至得出与事实相悖的科学结论。本文旨在系统性地解决这一知识鸿沟，为生物医学研究者和数据科学家提供一个关于误差模型的全面框架。

本文将通过三个核心章节，引导读者深入理解测量误差的世界。首先，在“原理与机制”一章中，我们将建立描述测量误差的数学语言，区分[系统误差与随机误差](@entry_id:912653)，并详细剖析两种最基本的范式——经典误差模型和Berkson误差模型，揭示它们如何从根本上影响[统计推断](@entry_id:172747)。接着，在“应用与跨学科交叉”一章中，我们将理论付诸实践，展示误差模型如何在仪器校准、[高通量数据](@entry_id:275748)（如[基因组学](@entry_id:138123)和[蛋白质组学](@entry_id:155660)）的[批次效应校正](@entry_id:269846)、以及复杂的动态[系统建模](@entry_id:197208)等前沿领域中发挥关键作用。最后，通过“动手实践”环节，读者将有机会通过解决具体问题，将所学知识内化，真正掌握处理带误差数据的核心技能。通过本次学习，你将不仅能认识到误差的存在，更能精确地对其进行建模、评估其影响，并为后续的校正方法打下坚实的基础。

## 原理与机制

在生物医学研究中，我们测量的几乎所有量都受到误差的影响。无论是实验室分析中的仪器噪声，还是临床试验中的报告偏差，理解这些误差的结构和影响对于从数据中得出有效和可靠的结论至关重要。本章深入探讨了[测量误差模型](@entry_id:751821)的基本原理和机制。我们将建立一个形式化的框架来描述不同类型的误差，并研究它们如何影响[统计推断](@entry_id:172747)，特别是在[回归分析](@entry_id:165476)等常见应用中。我们的目标是超越简单地承认误差的存在，而是要精确地描述它，并理解其对我们分析能力的后果。

### 基本加性误差模型：[系统误差与随机误差](@entry_id:912653)

对测量误差进行建模的最基本方法是**加性误差模型**。假设我们感兴趣的某个潜在的真实生物学量为 $X$（例如，真实的血浆[皮质醇](@entry_id:152208)浓度），而我们通过仪器或实验观察到的值为 $W$。一个简单而强大的模型将观测值 $W$ 表示为真实值 $X$ 与一个误差项 $U$ 的和：

$W = X + U$

在这个模型中，误差项 $U$ 捕捉了观测值与真实值之间的所有偏差。然而，为了使这个模型有用，我们需要更精确地定义 $U$ 的性质。一个至关重要的区别是**系统误差**和**[随机误差](@entry_id:144890)**。

系统误差，或称**偏倚 (bias)**，是一种在[重复测量](@entry_id:896842)中持续存在的、非随机的偏差。例如，一个未校准的仪器可能会持续地将所有读数高估一个固定的量。我们可以通过在模型中引入一个常数项 $b$ 来表示这种偏倚：

$W = X + U + b$

在这里，$U$ 代表纯粹的**随机误差**，其特点是在多次测量中平均为零，即其[期望值](@entry_id:150961) $\mathbb{E}[U] = 0$。而 $b$ 则代表固定的系统偏倚。这种区分对于理解我们能从数据中学到什么至关重要。假设我们从一个群体中收集了大量的观测值 $\{W_i\}$，并希望估计该群体中真实生物学量 $X$ 的平均值 $\mu_X = \mathbb{E}[X]$ 和方差 $\sigma_X^2 = \operatorname{Var}(X)$。

让我们来分析一下可观测数据的均值和方差：

$\mathbb{E}[W] = \mathbb{E}[X + U + b] = \mathbb{E}[X] + \mathbb{E}[U] + \mathbb{E}[b] = \mu_X + 0 + b = \mu_X + b$

$\operatorname{Var}(W) = \operatorname{Var}(X + U + b) = \operatorname{Var}(X + U)$

如果我们假设随机误差 $U$ 与真实值 $X$ 是独立的，那么 $X$ 和 $U$ 的协方差为零，和的方差等于方差的和：

$\operatorname{Var}(W) = \operatorname{Var}(X) + \operatorname{Var}(U) = \sigma_X^2 + \sigma_U^2$

这些结果揭示了一个关于**可识别性 (identifiability)** 的核心问题 。从观测数据中，我们可以估计 $\mathbb{E}[W]$ 和 $\operatorname{Var}(W)$。通过上述方程，我们发现我们能确定的只是复合参数 $\mu_X + b$，而不是 $\mu_X$ 或 $b$ 本身。没有外部信息（如使用“金标准”仪器进行校准以确定 $b$），就不可能将真实群体的平均值与仪器的系统偏倚分离开来。即使我们对每个个体进行[重复测量](@entry_id:896842)，也无法解决这个问题，因为系统偏倚 $b$ 会影响每一次测量。

然而，情况并非完全悲观。对于方差，由于偏倚 $b$ 只是一个常数，它不影响数据的离散程度，因此从方差的计算中消失了。如果我们能够通过仪器特性表征（例如，通过对已知[标准品](@entry_id:754189)进行[重复测量](@entry_id:896842)）来了解[随机误差](@entry_id:144890)的方差 $\sigma_U^2$，我们就可以通过以下方式识别出真实信号的方差：

$\sigma_X^2 = \operatorname{Var}(W) - \sigma_U^2$

这个简单的例子说明了一个深刻的原理：[测量误差模型](@entry_id:751821)不仅描述了误差的来源，还界定了我们从受污染的数据中恢复真实信号的能力边界。

### 经典误差模型与Berkson误差模型：两种基本范式

在对随机误差 $U$ 的结构进行更深入的建模时，统计学文献中出现了两种主要的范式：经典误差模型和Berkson误差模型。它们之间的区别在于对误差项与模型中其他变量之间独立性的假设不同。

#### 经典误差模型

**[经典测量误差](@entry_id:1122426)模型**是最常用和最直观的模型。其核心假设是，误差 $U$ 与真实值 $X$ 是统计独立的（记为 $U \perp X$），并且误差的均值为零（$\mathbb{E}[U] = 0$）。这种模型适用于许多情况，例如，当测量仪器产生的噪声独立于被测量的实际大小时。

形式上，一个无偏的[经典测量误差](@entry_id:1122426)模型由以下条件定义 ：
1.  **加性结构**: $W = X + U$
2.  **零均值误差**: $\mathbb{E}[U] = 0$
3.  **独立性**: $U$ 与 $X$ 统计独立 ($U \perp X$)

这些假设带来了一些重要的推论。首先，测量过程是**无偏的**，意味着在给定真实值 $X$ 的条件下，观测值的期望等于真实值：
$\mathbb{E}[W \mid X] = \mathbb{E}[X + U \mid X] = \mathbb{E}[X \mid X] + \mathbb{E}[U \mid X] = X + \mathbb{E}[U] = X + 0 = X$
这里，从 $U \perp X$ 可以推导出 $\mathbb{E}[U \mid X] = \mathbb{E}[U]$。

其次，给定真实值 $X$ 时，观测值的方差仅由误差的方差决定，并且这个方差是恒定的，这种性质称为**[同方差性](@entry_id:634679) (homoscedasticity)**：
$\operatorname{Var}(W \mid X) = \operatorname{Var}(X + U \mid X) = \operatorname{Var}(U \mid X) = \operatorname{Var}(U) = \sigma_U^2$
经典模型捕捉了这样一种思想：观测值 $W$ 是围绕真实值 $X$ 的一个随机散布，其散布的程度与 $X$ 本身的大小无关。

#### Berkson误差模型

与经典模型形成对比的是**Berkson误差模型**。在某些实验设置中，我们控制或设定的值是 $W$，而真实值 $X$ 是围绕 $W$ 随机波动的。例如，在环境暴露研究中，我们可能根据地理位置为一组受试者分配一个估计的平均空气污染物暴露水平 $W$。然而，由于微环境和个体行为的差异，每个受试者的真实暴露水平 $X$ 会与这个指定的平均值 $W$ 有所不同。

Berkson误差模型的形式化表示如下 ：
1.  **加性结构**: $X = W + U$
2.  **零均值误差**: $\mathbb{E}[U] = 0$
3.  **独立性**: 误差 $U$ 与受控变量 $W$ 统计独立 ($U \perp W$)

这里的关键区别在于，误差项现在被假定为与观测值（或设定值）$W$ 独立，而不是与未知的真实值 $X$ 独立。正如我们将在下一节中看到的，这种看似微小的差异对[统计推断](@entry_id:172747)的后果有着巨大的影响。

### 误差结构对[统计推断](@entry_id:172747)的影响：以线性回归为例

理解不同误差模型最有效的方法之一是考察它们对一个标准统计任务——线性回归——的影响。假设我们感兴趣的是一个真实的生物学关系，由一个线性结构模型描述：

$Y = \beta_0 + \beta_1 X + \epsilon$

其中，$Y$ 是一个结果变量（如临床指标），$X$ 是一个暴露变量（如[生物标志物](@entry_id:914280)浓度），$\beta_1$ 是我们希望估计的效应大小，$\epsilon$ 是独立于 $X$ 的附加生物学或随机变异。问题在于，我们无法直接观测到 $X$，只能观测到其代理 $W$。一个“朴素”的分析师可能会忽略测量误差，直接用[普通最小二乘法](@entry_id:137121) (OLS) 将 $Y$ 对 $W$ 进行回归。这种朴素回归得到的斜率估计量在样本量很大时会收敛到总体斜率 $\frac{\operatorname{Cov}(W,Y)}{\operatorname{Var}(W)}$。让我们来分析在不同误差模型下，这个朴素斜率与真实斜率 $\beta_1$ 之间的关系。

#### 经典误差导致衰减

在经典误差模型 ($W=X+U, U \perp X$)下，我们来计算协方差和方差项：

$\operatorname{Cov}(W, Y) = \operatorname{Cov}(X+U, \beta_0 + \beta_1 X + \epsilon) = \beta_1 \operatorname{Var}(X) = \beta_1 \sigma_X^2$
（因为 $U$ 与 $X$ 和 $\epsilon$ 都独立，$\epsilon$ 与 $X$ 独立）

$\operatorname{Var}(W) = \operatorname{Var}(X+U) = \operatorname{Var}(X) + \operatorname{Var}(U) = \sigma_X^2 + \sigma_U^2$
（因为 $U$ 与 $X$ 独立）

因此，朴素回归的总体斜率是：

$\beta_{\text{naive}} = \frac{\operatorname{Cov}(W,Y)}{\operatorname{Var}(W)} = \frac{\beta_1 \sigma_X^2}{\sigma_X^2 + \sigma_U^2} = \beta_1 \left( \frac{\sigma_X^2}{\sigma_X^2 + \sigma_U^2} \right)$

这个结果揭示了一个称为**[回归稀释](@entry_id:925147) (regression dilution)** 或**衰减偏倚 (attenuation bias)** 的基本现象  。括号中的因子 $\lambda = \frac{\sigma_X^2}{\sigma_X^2 + \sigma_U^2}$ 被称为**[衰减因子](@entry_id:1121239)**。由于方差总是非负的，这个因子总是在 $0$ 和 $1$ 之间。这意味着由朴素回归估计出的效应大小将系统性地被低估，其大小偏向于零。测量误差越大（即 $\sigma_U^2$ 越大），衰减就越严重。此外，值得注意的是，这种对斜率的偏倚也会导致对截距的估计产生偏倚，除非真实暴露 $X$ 的均值为零 。

#### Berkson误差保留斜率

现在，考虑Berkson误差模型 ($X=W+U, U \perp W$)。我们再次进行相同的计算。首先，将 $X$ 代入 $Y$ 的模型中：
$Y = \beta_1 (W+U) + \epsilon = \beta_1 W + \beta_1 U + \epsilon$

现在计算协方差和方差：

$\operatorname{Cov}(W, Y) = \operatorname{Cov}(W, \beta_1 W + \beta_1 U + \epsilon) = \beta_1 \operatorname{Var}(W)$
（因为 $W$ 与 $U$ 和 $\epsilon$ 都独立）

$\operatorname{Var}(W)$ 是观测预测变量的方差，我们不需要分解它。

朴素回归的总体斜率是：

$\beta_{\text{naive}} = \frac{\operatorname{Cov}(W,Y)}{\operatorname{Var}(W)} = \frac{\beta_1 \operatorname{Var}(W)}{\operatorname{Var}(W)} = \beta_1$

这是一个引人注目的结果：在Berkson误差模型下，对观测数据进行朴素的[线性回归](@entry_id:142318)，其斜率估计是**无偏的** 。这可以通过考察[条件期望](@entry_id:159140)来更直观地理解。回归的目标是估计 $E[Y|W]$。在Berkson模型下：
$E[Y \mid W] = E[\beta_1 X + \epsilon \mid W] = \beta_1 E[X \mid W] + E[\epsilon \mid W]$
由于 $E[X \mid W] = E[W+U \mid W] = W+E[U|W] = W$ 且 $E[\epsilon \mid W] = E[\epsilon] = 0$，我们得到：
$E[Y \mid W] = \beta_1 W$
这表明 $Y$ 对 $W$ 的真实关系确实是线性的，并且斜率恰好是 $\beta_1$ 。

#### Berkson误差扩大预测方差

尽管Berkson误差不偏倚斜率估计，但它并非没有后果。它会影响我们对新观测进行预测的精度。预测的精度取决于给定 $W$ 时 $Y$ 的[条件方差](@entry_id:183803)，即残差的方差。让我们计算这个量：

$\operatorname{Var}(Y \mid W) = \operatorname{Var}(\beta_1 X + \epsilon \mid W) = \operatorname{Var}(\beta_1(W+U) + \epsilon \mid W)$
$\operatorname{Var}(Y \mid W) = \operatorname{Var}(\beta_1 W + \beta_1 U + \epsilon \mid W)$

在给定 $W$ 的条件下，$\beta_1 W$ 是一个常数，所以它不贡献方差。由于 $U$ 和 $\epsilon$ 都与 $W$ 独立，我们有：

$\operatorname{Var}(Y \mid W) = \operatorname{Var}(\beta_1 U + \epsilon) = \beta_1^2 \operatorname{Var}(U) + \operatorname{Var}(\epsilon) = \beta_1^2 \sigma_U^2 + \sigma_\epsilon^2$

在一个没有测量误差的理想世界里（$\sigma_U^2=0$），残差方差将是 $\sigma_\epsilon^2$。Berkson误差引入了一个额外的方差项 $\beta_1^2 \sigma_U^2$ 。这意味着即使我们知道了真实的斜率 $\beta_1$，使用代理变量 $W$ 进行预测时，我们的预测区间也会比使用真实变量 $X$ 时更宽。方差的增加反映了这样一个事实：对于一个给定的 $W$，真实的 $X$ 仍然是不确定的，这种不确定性传播到了对 $Y$ 的预测中。

### 生物医学数据中的高级误差结构

虽然经典模型和Berkson模型是重要的基础，但生物医学数据中的误差结构往往更为复杂。

#### [乘性](@entry_id:187940)误差和对数正态误差

在许多生物学测量中，特别是涉及浓度的测量，误差的大小与被测量的大小成正比。这种**乘性误差**模型可以表示为：

$W = X \cdot V$

其中 $V$ 是一个均值为 $1$ 的[随机误差](@entry_id:144890)因子。一个常见的特定形式是使用指数函数，$W = X \cdot \exp(U)$，其中 $U$ 是一个均值为零的[随机变量](@entry_id:195330)。这种模型确保了观测值 $W$ 保持正值（如果 $X$ 是正的），这对于浓度等物理量是必需的。

对这种模型进行分析的一个强大技巧是取对数。令 $Y = \ln(W)$，则模型变为：

$Y = \ln(X \cdot \exp(U)) = \ln(X) + \ln(\exp(U)) = \ln(X) + U$

通过对数变换，[乘性误差模型](@entry_id:907207)在对数尺度上转化为了一个加性误差模型 。这在实践中非常有用，因为它允许我们在对数尺度上应用为加性误差开发的许多工具。然而，这种变换也带来了微妙之处。例如，如果在对数尺度上的误差 $U$ 服从均值为 $0$、方差为 $\sigma_U^2$ 的正态分布，即 $U \sim N(0, \sigma_U^2)$，那么原始尺度上的观测值的期望是多少？

$E[W \mid X] = E[X \cdot \exp(U) \mid X] = X \cdot E[\exp(U)]$

对于一个正态分布的[随机变量](@entry_id:195330) $U$，其[矩生成函数](@entry_id:154347)在 $t=1$ 处的值为 $E[\exp(U)] = \exp(\mu + \sigma^2/2)$。在我们的例子中，$\mu=0$ 且 $\sigma^2 = \sigma_U^2$，所以：

$E[W \mid X] = X \cdot \exp\left(\frac{\sigma_U^2}{2}\right)$

这个结果表明，即使误差在对数尺度上是无偏的（均值为零），在原始尺度上的测量也是有偏的，会系统性地高估真实值。这个因子 $\exp(\sigma_U^2/2)$ 总是大于 $1$（对于 $\sigma_U^2 > 0$）。这是凸函数（[指数函数](@entry_id:161417)）的**琴生不等式 (Jensen's inequality)** 的一个直接体现。

#### 相关多元误差

现代生物医学技术，如多重[生物标志物](@entry_id:914280)分析（multiplex assays）或[基因表达微阵列](@entry_id:911531)，可以同时测量成百上千个变量。在这种情况下，测量误差本身可能在不同通道之间是相关的。例如，一个共同的校准步骤或试剂批次的偏差可能会同时影响多个[分析物](@entry_id:199209)的读数。

我们可以将加性误差模型推广到多变量情况。令 $W$ 和 $X$ 为 $p$ 维向量，模型为：

$W = X + U$

其中 $U$ 是一个 $p$ 维误差向量，我们假设它服从均值为 $0$、[协方差矩阵](@entry_id:139155)为 $\Sigma_U$ 的[多元正态分布](@entry_id:175229)，即 $U \sim N(0, \Sigma_U)$。[协方差矩阵](@entry_id:139155) $\Sigma_U$ 的对角线元素 $(\Sigma_U)_{ii} = \operatorname{Var}(U_i)$ 代表每个通道的误差方差。而其**非对角[线元](@entry_id:196833)素** $(\Sigma_U)_{ij} = \operatorname{Cov}(U_i, U_j)$ (for $i \neq j$) 则捕捉了不同通道 $i$ 和 $j$ 之间测量误差的相关性 。

如果误差向量 $U$ 与真实信号向量 $X$ 独立，那么观测向量 $W$ 的[协方差矩阵](@entry_id:139155)是：

$\operatorname{Cov}(W) = \operatorname{Cov}(X+U) = \operatorname{Cov}(X) + \operatorname{Cov}(U) = \Sigma_X + \Sigma_U$

其中 $\Sigma_X$ 是真实信号的协方差矩阵。这个简单的公式表明，观测到的变量间的协方差是真实生物学协方差和误差协方差的总和。如果真实生物信号在不同通道间是不相关的（即 $\Sigma_X$ 是[对角矩阵](@entry_id:637782)），那么任何观测到的跨通道相关性都完全是由测量误差的相关性引起的。

#### 重尾误差与稳健性

正态分布（高斯分布）是描述[随机误差](@entry_id:144890)的经典选择，但它可能无法充分捕捉某些现实世界现象，例如由于运动伪影或瞬态生理事件导致的偶发性、大幅度的测量偏差或**离群点 (outliers)**。这些事件产生的误差分布具有比正态分布更“重”的尾部。

一个灵活的重尾分布是**[学生t分布](@entry_id:267063) ([Student's t-distribution](@entry_id:142096))**。我们可以将误差模型设为 $Y_i = \theta + U_i$，其中 $U_i$ 服从自由度为 $\nu$、尺度参数为 $\sigma$ 的[学生t分布](@entry_id:267063)。自由度 $\nu$ 控制了分布尾部的“重量”：$\nu$ 越小，尾部越重，出现极端值的概率越高。当 $\nu \to \infty$ 时，[t分布](@entry_id:267063)收敛于正态分布。

使用这种重尾误差模型如何影响我们对中心参数 $\theta$ 的估计？如果我们使用样本均值 $\hat{\theta} = \frac{1}{n} \sum Y_i$ 作为估计量，其（渐近）方差与单次观测的方差成正比，即 $\operatorname{Var}(Y_i) = \operatorname{Var}(U_i)$。对于自由度为 $\nu > 2$ 的[t分布](@entry_id:267063)，其方差为 $\sigma^2 \frac{\nu}{\nu-2}$。而对于具有相同[尺度参数](@entry_id:268705) $\sigma$ 的高斯误差，方差为 $\sigma^2$。

因此，我们可以定义一个“稳健性影响因子”，即两种误差模型下样本均值方差的比率 ：

$I(\nu) = \frac{\text{t分布误差下的方差}}{\text{高斯误差下的方差}} = \frac{\sigma^2 \frac{\nu}{\nu - 2}}{\sigma^2} = \frac{\nu}{\nu - 2}$

这个因子揭示了样本均值在面对[重尾](@entry_id:274276)数据时的脆弱性。当 $\nu$ 接近 $2$ 时，该因子趋于无穷大，表明样本均值的方差会急剧膨胀，使其成为一个非常不可靠的估计量。这促使了**稳健统计 (robust statistics)** 的发展，该领域旨在开发在非正态误差（特别是[重尾](@entry_id:274276)误差）存在时表现良好的估计方法（如使用中位数或[M估计量](@entry_id:169257)代替样本均值）。

### 复杂相互作用：[批次效应](@entry_id:265859)与缺失数据

在真实的[生物医学数据分析](@entry_id:899234)中，测量误差很少孤立存在。它常常与其他[数据质量](@entry_id:185007)问题相互作用，产生复杂的挑战。本节探讨两个这样的相互作用：[批次效应](@entry_id:265859)和缺失数据。

#### 批次效应的建模与混淆

在高通量实验（如基因组学、[蛋白质组学](@entry_id:155660)）中，样本通常分批处理。由于试剂、仪器设置或操作人员的微小差异，不同批次之间常常会出现系统性的测量偏差，这被称为**[批次效应](@entry_id:265859) (batch effects)**。一个描述批次效应的有用模型是 ：

$W_{ib} = a_b + b_b X_i + U_{ib}$

其中 $W_{ib}$ 是在批次 $b$ 中处理的个体 $i$ 的观测值，$X_i$ 是其真实生物信号。$a_b$ 是批次 $b$ 特有的**加性效应**（偏移），$b_b$ 是批次 $b$ 特有的**[乘性](@entry_id:187940)效应**（缩放），$U_{ib}$ 是均值为零的随机误差。

当批次分配与我们感兴趣的生物学条件（例如，病例组 vs. [对照组](@entry_id:747837)）发生**混淆 (confounding)** 时，[批次效应](@entry_id:265859)会变得尤其有害。例如，如果所有病例样本都在一个批次中处理，而所有对照样本都在另一个批次中处理，那么批次效应和真实的生物学效应就变得无法区分。在这种完全混淆的情况下，任何试图同时估计[批次效应](@entry_id:265859)和条件效应的[线性模型](@entry_id:178302)都会失败，因为[设计矩阵](@entry_id:165826)是**[秩亏](@entry_id:754065)的 (rank-deficient)**，不存在唯一的解 。即使混淆不是完全的，不平衡的批次分配也会在朴素的组间比较中引入严重偏倚，这个偏倚既包含加性效应的差异，也包含[乘性](@entry_id:187940)效应与真实信号之间更复杂的相互作用。

#### 测量误差与[缺失数据机制](@entry_id:173251)

另一个复杂的相互作用发生在测量误差和**[缺失数据](@entry_id:271026)**之间。假设数据的记录与否取决于一个缺失[指示变量](@entry_id:266428) $R$（$R=1$ 表示记录， $R=0$ 表示缺失）。缺失机制通常分为三类：

1.  **[完全随机缺失](@entry_id:170286) (MCAR)**: 缺失的概率与任何变量（观测到的或未观测到的）都无关。形式化为 $R \perp (X, W)$。
2.  **[随机缺失](@entry_id:164190) (MAR)**: 在给定观测数据 $W$ 的条件下，缺失的概率与未观测的真实数据 $X$ 无关。形式化为 $R \perp X \mid W$。
3.  **[非随机缺失](@entry_id:899134) ([MNAR](@entry_id:899134))**: 即使在给定观测数据 $W$ 的条件下，缺失的概率仍然依赖于未观测的真实数据 $X$。

现在，考虑这样一种情景：数据的缺失概率取决于真实的生物学值 $X$。例如，一个设备可能在[生物标志物](@entry_id:914280)水平 $X$ 过高或过低时更容易发生故障并导致数据缺失。这种机制，如果 $X$ 可以被直接观察，将被归类为 MAR（因为它仅依赖于“可观测”的 $X$）。

然而，在我们的框架中，我们只能观测到带有误差的 $W=X+U$。缺失机制由 $P(R=1 \mid X)$ 决定。我们可以证明，在这种情况下，$P(R=1 \mid X, W) = P(R=1 \mid X)$。MAR 条件要求 $P(R=1 \mid X, W) = P(R=1 \mid W)$。除非在非常特殊的情况下，否则 $P(R=1 \mid X)$ 通常不等于 $P(R=1 \mid W)$。因此，一个在真实值层面可能是 MAR（甚至MCAR）的机制，在存在测量误差时，相对于观测数据 $W$ 而言，变成了一个**[MNAR](@entry_id:899134)机制** 。

这一转变具有严重的实际后果。许多标准用于处理缺失数据的方法，如基于观测数据进行**[逆概率加权](@entry_id:1126661) (inverse probability weighting, IPW)**，都依赖于 MAR 假设。在上述 [MNAR](@entry_id:899134) 情景下应用这些方法会导致有偏的估计。更糟糕的是，从观测数据 $(W, R)$ 中，我们无法唯一地确定真实信号的分布 $p(X)$、误差的分布 $p_U(U)$ 和选择模型 $P(R=1 \mid X)$。这是一个严重的**非可识别性**问题，通常需要额外的假设或来自验证子研究的外部信息才能解决 。

总之，本章概述了描述生物医学数据中测量误差的数学框架。从简单的加性模型到复杂的[批次效应](@entry_id:265859)和与缺失数据的相互作用，我们看到，对误差结构的精确理解是进行有效[统计建模](@entry_id:272466)和推断的先决条件。忽视误差或错误地指定其结构可能导致从衰减偏倚到完全错误的科学结论等一系列问题。在后续章节中，我们将探讨用于修正这些误差影响的各种方法。
