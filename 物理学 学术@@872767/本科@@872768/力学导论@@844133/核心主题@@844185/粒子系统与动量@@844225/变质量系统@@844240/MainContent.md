## 引言
在经典力学的学习中，我们通常处理质量恒定的物体，这一简化假设使得问题分析变得直接。然而，从升空的火箭到行驶中撒沙的卡车，再到宇宙中吸积物质的星体，现实世界充满了质量随时间变化的系统。如何精确描述这些**[变质量系统](@entry_id:177386) (systems with changing mass)** 的运动，是标准力学框架需要扩展来解决的一个重要问题。传统的牛顿第二定律形式 $F=ma$ 在此不再直接适用，我们需要回归其更本源的动量形式。

本文旨在系统性地构建分析[变质量系统](@entry_id:177386)的理论框架。我们将首先在“**原理与机制**”一章中，从[牛顿第二定律](@entry_id:274217)的动量表述出发，推导出[变质量系统](@entry_id:177386)动力学的普遍方程，并探讨其在质量抛射（如火箭）和[质量吸积](@entry_id:163137)（如传送带）两种典型情景下的具体应用，包括著名的[齐奥尔科夫斯基火箭方程](@entry_id:178340)。随后，在“**应用与跨学科联系**”一章中，我们将展示这些原理如何应用于解决航空航天、土木工程、天体物理学等多个领域的实际问题，揭示其广泛的适用性。最后，“**动手实践**”部分将提供一系列精心挑选的练习题，帮助您巩固所学知识并将其付诸实践。通过本次学习，您将掌握分析开放力学系统的强大工具，并深刻理解动量守恒在更广阔物理情境下的普适性。

## 原理与机制

在经典力学中，我们分析的大多数系统都具有恒定的质量。然而，在自然界和工程技术中，许多重要系统的质量是随时间变化的。例如，火箭通过喷射燃料来加速，其总质量不断减小；传送带上不断有货物落下，其总质量不断增加。处理这类**[变质量系统](@entry_id:177386) (systems with changing mass)** 的动力学问题，要求我们回归到牛顿第二定律最基本、最普适的动量形式。本章将系统地阐述分析[变质量系统](@entry_id:177386)的基本原理，并探讨其在不同物理情境下的具体机制和应用。

### [变质量系统](@entry_id:177386)动力学的普遍方程

牛顿第二定律的原始表述是，施加在系统上的净外力等于系统总动量的变化率，即 $\vec{F}_{\text{ext}} = \frac{d\vec{P}_{\text{total}}}{dt}$。对于一个质量恒定的物体，这个定律可以简化为我们熟知的形式 $\vec{F}_{\text{ext}} = m\vec{a}$。然而，当系统质量 $m$ 发生变化时，我们必须仔细考虑进出系统的质量所携带的动量。

让我们考虑一个在时刻 $t$ 质量为 $m$、速度为 $\vec{v}$ 的主体。在一个微小的时间间隔 $dt$ 内，它的质量变为 $m+dm$，速度变为 $\vec{v}+d\vec{v}$。我们定义 $dm$ 的正负表示质量的增加或减少。在此期间，离开或进入主体的微小质量元（我们统一称之为“交换质量”）在[惯性参考系](@entry_id:276742)中的速度为 $\vec{u}$。

我们将主体和即将发生相互作用的交换质量作为一个孤立的系统。在时刻 $t$，系统的总动量是主体的动量 $\vec{P}(t) = m\vec{v}$。在时刻 $t+dt$，主体的动量变为 $(m+dm)(\vec{v}+d\vec{v})$，而质量为 $-dm$ 的部分（如果 $dm \lt 0$，则 $-dm$ 为正，表示喷射出的质量）的动量为 $(-dm)\vec{u}$。根据动量-冲量定理，外力冲量等于系统[总动量](@entry_id:173071)的增量：

$\vec{F}_{\text{ext}} dt = d\vec{P}_{\text{total}} = \left[ (m+dm)(\vec{v}+d\vec{v}) + (-dm)\vec{u} \right] - m\vec{v}$

展开并忽略二阶无穷小量 $(dm)(d\vec{v})$，我们得到：

$\vec{F}_{\text{ext}} dt = m\,d\vec{v} + \vec{v}\,dm - \vec{u}\,dm$

两边同除以 $dt$：

$\vec{F}_{\text{ext}} = m\frac{d\vec{v}}{dt} + (\vec{v} - \vec{u})\frac{dm}{dt}$

这个方程是分析[变质量系统](@entry_id:177386)的基础。通常，将交换质量的速度相对于主体的速度 $\vec{v}_{\text{rel}}$ 表示更为方便，其中 $\vec{v}_{\text{rel}} = \vec{u} - \vec{v}$。代入上式，我们得到一个更具洞察力的形式：

$m\frac{d\vec{v}}{dt} = \vec{F}_{\text{ext}} + \vec{v}_{\text{rel}}\frac{dm}{dt}$

这个方程被称为**[变质量系统](@entry_id:177386)动力学普遍方程**。方程左边是熟悉的“质量乘以加速度”项。右边第一项是作用在系统上的所有**外力**之和。第二项 $\vec{v}_{\text{rel}}\frac{dm}{dt}$ 是一个有效力，通常称为**推力 (thrust)**，它源于质量交换过程本身。这个推力的方向取决于相对速度 $\vec{v}_{\text{rel}}$ 的方向以及质量是增加 ($\frac{dm}{dt} \gt 0$) 还是减少 ($\frac{dm}{dt} \lt 0$)。

### 质量抛射系统：推进原理

当系统通过向外抛射物质来改变自身运动状态时，我们称之为质量抛射系统。这是火箭、喷气式飞机等推进装置的基本原理。在这种情况下，系统质量减少，即 $\frac{dm}{dt} \lt 0$。

#### 理想火箭与[齐奥尔科夫斯基火箭方程](@entry_id:178340)

最纯粹的质量抛射系统是工作在无外力、无阻力太空中的理想火箭。设火箭沿[直线运动](@entry_id:165142)，$\vec{F}_{\text{ext}} = 0$。燃料以相对于火箭为 $u_{\text{ex}}$ 的恒定速度向后喷出。因此，废气相对于火箭的速度矢量 $\vec{v}_{\text{rel}}$ 的方向与火箭前进方向相反，大小为 $u_{\text{ex}}$。如果我们将前进方向定为正方向，则 $v_{\text{rel}} = -u_{\text{ex}}$。

[动力学方程](@entry_id:751029)简化为：

$m\frac{dv}{dt} = (-u_{\text{ex}})\frac{dm}{dt}$

这是一个可分离变量的[微分方程](@entry_id:264184)：$dv = -u_{\text{ex}}\frac{dm}{m}$。

假设火箭从初始质量 $m_i$、初始速度 $v_i$ 开始，到燃料耗尽时最终质量为 $m_f$、最终速度为 $v_f$。对上式进行积分：

$\int_{v_i}^{v_f} dv = -u_{\text{ex}} \int_{m_i}^{m_f} \frac{dm}{m}$

$v_f - v_i = -u_{\text{ex}} (\ln(m_f) - \ln(m_i)) = u_{\text{ex}} \ln\left(\frac{m_i}{m_f}\right)$

这个著名的方程就是**[齐奥尔科夫斯基火箭方程](@entry_id:178340)**。它表明，火箭的[速度增量](@entry_id:176263) $\Delta v$ 直接取决于废气的有效喷射速度 $u_{\text{ex}}$ 和初始质量与最终质量之比，即**[质量比](@entry_id:167674) (mass ratio)** $R = m_i/m_f$。这个方程揭示了[火箭推进](@entry_id:265657)的核心：要获得更高的[速度增量](@entry_id:176263)，要么提高喷气速度，要么携带更多燃料以增大质量比。

一个类似的例子是，一个静止在无摩擦冰面上的雪橇，人通过不断向后扔雪球来使雪橇前进 [@problem_id:2216549]。每一次扔雪球的过程，都像一次微型的火箭点火，累积效应使得雪橇不断加速。通过对[火箭方程](@entry_id:274435)得到的[瞬时速度](@entry_id:167797) $v(t)$ 进行积分，我们甚至可以计算出在扔完所有雪球后雪橇滑行的总距离 [@problem_id:2216518]。

#### 考虑外力的火箭运动

在实际情况中，火箭通常需要克服重力等外力。例如，从地面垂直发射的火箭，其外力为重力 $F_{\text{ext}} = -mg$（取向上为正）。[动力学方程](@entry_id:751029)变为：

$m\frac{dv}{dt} = -u_{\text{ex}}\frac{dm}{dt} - mg$

由于质量 $m(t)$ 是随时间变化的（通常假设燃料消耗率 $R = -\frac{dm}{dt}$ 为常数，则 $m(t) = m_i - Rt$），这是一个更复杂的[微分方程](@entry_id:264184)。但我们仍然可以求解它来得到火箭在燃料耗尽（burnout）时的速度 $v_b$ 和高度 $z_b$。

火箭的飞行通常分为两个阶段：动力飞行阶段和惯性飞行阶段。在燃料耗尽后，火箭作为一个质量恒定的抛体，在重力作用下继续上升，直到达到最大高度 $H_{\text{max}}$。通过分别计算这两个阶段的高度增量，我们可以得到火箭所能达到的最大高度的完整表达式 [@problem_id:2216532]。这个分析对于[航天任务设计](@entry_id:177598)至关重要。

#### 流体介质中的[喷气推进](@entry_id:273907)

另一种常见的推进形式是喷气艇，它从周围的静止水中吸水，然后向后高速喷出以获得推力 [@problem_id:2216534]。与火箭不同，喷气艇的总质量（船体+发动机）是恒定的。我们可以通过分析水流的动量变化来计算推力。

设水的吸入和喷出速率均为 $\lambda$ (kg/s)。水被吸入时在湖[参考系](@entry_id:169232)中的速度为 0。水被以相对于船 $u_{\text{rel}}$ 的速度向后喷出，此时船的速度为 $v$，因此喷出的水在湖[参考系](@entry_id:169232)中的速度为 $v - u_{\text{rel}}$。

单位时间内，水流的动量变化率为：

$\frac{dp_{\text{water}}}{dt} = (\text{流出动量率}) - (\text{流入动量率}) = \lambda(v - u_{\text{rel}}) - \lambda(0) = \lambda(v - u_{\text{rel}})$

根据[牛顿第三定律](@entry_id:166652)，水对船的推力 $F_{\text{thrust}}$ 与船对水的作用力大小相等、方向相反：

$F_{\text{thrust}} = -\frac{dp_{\text{water}}}{dt} = \lambda(u_{\text{rel}} - v)$

有趣的是，推力的大小取决于船的当前速度 $v$。船速越快，推力越小。如果船还受到与速度成正比的阻力 $F_{\text{drag}} = -kv$，其[运动方程](@entry_id:170720)为：

$M_{\text{boat}}\frac{dv}{dt} = F_{\text{thrust}} + F_{\text{drag}} = \lambda(u_{\text{rel}} - v) - kv$

当加速度为零时，船达到**[终端速度](@entry_id:147799) (terminal velocity)** $v_T$。此时推力与阻力平衡：$\lambda(u_{\text{rel}} - v_T) = kv_T$。解得 $v_T = \frac{\lambda u_{\text{rel}}}{\lambda + k}$。这表明[终端速度](@entry_id:147799)由推进系统的参数（$\lambda, u_{\text{rel}}$）和流体的阻力特性（$k$）共同决定。

### [质量吸积](@entry_id:163137)系统

与质量抛射相反的是**[质量吸积](@entry_id:163137) (mass accretion)**，即系统在运动过程中不断吸纳外界物质，导致总质量增加，此时 $\frac{dm}{dt} \gt 0$。

#### 吸积动力学

对于[质量吸积](@entry_id:163137)，我们的普遍方程 $m\frac{d\vec{v}}{dt} = \vec{F}_{\text{ext}} + \vec{v}_{\text{rel}}\frac{dm}{dt}$ 依然适用。其中 $\vec{v}_{\text{rel}} = \vec{u}_{\text{in}} - \vec{v}$，$\vec{u}_{\text{in}}$ 是被吸积物质的初始速度。代入后可得：

$m\frac{d\vec{v}}{dt} = \vec{F}_{\text{ext}} + (\vec{u}_{\text{in}}-\vec{v})\frac{dm}{dt}$

整理后可以得到一个对吸积问题更直观的形式：

$\frac{d(m\vec{v})}{dt} = \vec{F}_{\text{ext}} + \vec{u}_{\text{in}}\frac{dm}{dt}$

这个形式的物理意义是：系统总动量的变化率等于外力与单位时间吸积的物质所带入的动量之和。

#### 吸积过程中的动量守恒

考虑一个在无摩擦[轨道](@entry_id:137151)上运动的小车，它不断地从[轨道](@entry_id:137151)上铲起静止的沙子 [@problem_id:2216530]。在这个情境下，外力 $\vec{F}_{\text{ext}}=0$，沙子初始速度 $\vec{u}_{\text{in}}=0$。因此，动力学方程简化为：

$\frac{d(m\vec{v})}{dt} = 0$

这意味着系统的[总动量](@entry_id:173071) $m\vec{v}$ 是守恒的。设小车初始质量为 $M_0$，初始速度为 $v_0$。当它行进了距离 $x$ 后，如果沙子的[线密度](@entry_id:158735)为 $\lambda$，则其总质量变为 $m(x) = M_0 + \lambda x$。根据[动量守恒](@entry_id:149964)：

$m(x)v(x) = M_0 v_0 \implies (M_0 + \lambda x)v(x) = M_0 v_0$

解得小车的速度为 $v(x) = \frac{M_0 v_0}{M_0 + \lambda x}$。随着质量的增加，小车的速度不断减小。

一个看似不同但本质相似的问题是一块在无摩擦平面上滑行的冰块，它在滑行过程中融化，融化的水停留在原地 [@problem_id:2216547]。虽然这是质量损失问题，但被抛射的物质（水）在[惯性系](@entry_id:266190)中的最终速度为零，即 $\vec{u}=0$。让我们用普遍方程分析：$\vec{F}_{\text{ext}}=0$，$\vec{v}_{\text{rel}} = \vec{u} - \vec{v} = 0 - \vec{v} = -\vec{v}$。[动力学方程](@entry_id:751029)为 $m\frac{d\vec{v}}{dt} = \vec{v}_{\text{rel}}\frac{dm}{dt} = -\vec{v}\frac{dm}{dt}$。这可以写成 $\frac{d\vec{v}}{\vec{v}} = -\frac{dm}{m}$。积分后得到 $v = v_0 \frac{m_0}{m}$。这与吸积静止沙子的小车在数学形式上完全相同！这揭示了一个深刻的对称性：吸积静止的物质与抛射出最终静止的物质，对主体动量的影响是等效的。

#### 有外力作用的吸积与[能量耗散](@entry_id:147406)

当存在外力时，情况会变得更加复杂。一个典型的例子是从静止的漏斗向水平传送带上持续落沙，同时电机用恒定的力 $F$ 驱动传送带 [@problem_id:2216570]。沙子初始水平速度为零（$u_{\text{in}}=0$），传送带质量 $M(t)=M_0+\lambda t$。[运动方程](@entry_id:170720)为：

$\frac{d(Mv)}{dt} = F \implies (M_0+\lambda t)\frac{dv}{dt} + \lambda v = F$

方程中的 $\lambda v$ 项可以看作是一种“阻力”，它源于需要不断加速新落到传送带上的沙子。当加速度为零时，传送带达到[终端速度](@entry_id:147799) $v_t = F/\lambda$。

更有趣的是能量分析。电机提供的功率为 $P = Fv$。系统（传送带+其上的沙子）总动能的变化率为：

$\frac{dK}{dt} = \frac{d}{dt}\left(\frac{1}{2}Mv^2\right) = \frac{1}{2}\frac{dM}{dt}v^2 + Mv\frac{dv}{dt} = \frac{1}{2}\lambda v^2 + v(F-\lambda v) = Fv - \frac{1}{2}\lambda v^2$

动能增加率与输入功率的比值为：

$\frac{dK/dt}{P} = \frac{Fv - \frac{1}{2}\lambda v^2}{Fv} = 1 - \frac{\lambda v}{2F} = 1 - \frac{v}{2v_t}$

这个比值总是小于1。这意味着电机提供的能量只有一部分转化为了系统的宏观动能，另一部分则在沙子与传送带的[非弹性碰撞](@entry_id:137360)中转化为了内能（热量）。当传送带达到终端速度 $v_t$ 时，这个比值为 $1/2$，即一半的功率用于增加新落下沙子的动能，另一半则完全耗散掉了。

另一个类似的例子是从地面匀速提起一根绳子 [@problem_id:562177]。提力 $F$ 不仅要克服已提起部分绳子的重力 $W(y)$，还需要提供一个额外的力来使绳子的微元从静止加速到速度 $v$。这个“[惯性力](@entry_id:169104)”的大小等于被提起质量的变化率 $\dot{M}$ 乘以速度 $v$，即 $F_{\text{inertial}} = \dot{M}v = (\lambda(y)v)v = \lambda(y)v^2$，其中 $\lambda(y)$ 是在高度 $y$ 处绳子的[线密度](@entry_id:158735)。总提力就是这两部分之和：$F(y) = W(y) + F_{\text{inertial}}$。

### 扩展到[转动动力学](@entry_id:267911)

[变质量系统](@entry_id:177386)的分析方法同样适用于转动问题。其基本原理是角动量形式的牛顿第二定律：$\vec{\tau}_{\text{ext}} = \frac{d\vec{L}}{dt}$。

考虑一个在无摩擦轴承上以角速度 $\omega_0$ 自由旋转的水平转盘。从 $t=0$ 开始，有细沙持续、均匀地落在转盘上并附着 [@problem_id:2216560]。由于沙子在落下前没有绕中心轴的角动量，且整个系统（转盘+沙子）没有受到外力矩作用（$\vec{\tau}_{\text{ext}}=0$），因此系统的总[角动量守恒](@entry_id:156798)。

初始角动量为 $L_i = I_0 \omega_0$，其中 $I_0 = \frac{1}{2}M_0 R^2$ 是转盘的初始转动惯量。
在时刻 $t$，总质量增加到 $M(t) = M_0 + \alpha t$（其中 $\alpha$ 是质量增加率）。由于沙子[均匀分布](@entry_id:194597)，系统的转动惯量变为 $I(t) = \frac{1}{2}M(t)R^2 = \frac{1}{2}(M_0 + \alpha t)R^2$。
根据[角动量守恒](@entry_id:156798)，$I(t)\omega(t) = I_0 \omega_0$，我们可以解出任意时刻的[角速度](@entry_id:192539)：

$\omega(t) = \omega_0 \frac{I_0}{I(t)} = \omega_0 \frac{M_0}{M_0 + \alpha t}$

随着质量的不断增加，转盘的[角速度](@entry_id:192539)会逐渐减小，这与滑冰运动员张开双臂时转速变慢是类似的道理。

### 高等视角：[相对论火箭](@entry_id:272473)

当火箭的速度接近光速 $c$ 时，经典力学不再适用，我们必须使用狭义相对论进行分析。经典[火箭方程](@entry_id:274435)是对真实情况的低速近似。通过对[相对论动量](@entry_id:159500)和[能量守恒](@entry_id:140514)定律的精细分析，可以推导出**[相对论火箭方程](@entry_id:165132)**。其微分形式为：

$-u \frac{dm}{m} = \frac{dv}{1-v^2/c^2}$

其中 $u$ 是喷射[物相](@entry_id:196677)对于火箭的（可能也是相对论性的）速度，$m$ 是火箭的瞬时[静止质量](@entry_id:264101)。积分后，最终速度 $V$ 与质量比 $M_i/M_f$ 的关系为：

$\frac{M_i}{M_f} = \left[ \frac{1+V/c}{1-V/c} \right]^{c/(2u)}$

这个方程揭示了星际航行的巨大挑战。假设一个探测器要达到 $V=0.8c$ 的速度 [@problem_id:2216541]。如果它使用喷射速度为 $u_A=0.95c$ 的聚变驱动，其所需质量比 $R_A = 3^{20/19}$。如果使用理论上极致的[光子](@entry_id:145192)火箭（喷射[光子](@entry_id:145192)， $u_B=c$），其[质量比](@entry_id:167674) $R_B=3$。两者之比 $R_A/R_B = 3^{1/19} \approx 1.059$。这表明，即使喷射速度从 $0.95c$ 提升到 $c$，所带来的质量比优势也并非极其巨大。[相对论火箭方程](@entry_id:165132)为未来深空探测的推进技术评估提供了坚实的理论基础。
