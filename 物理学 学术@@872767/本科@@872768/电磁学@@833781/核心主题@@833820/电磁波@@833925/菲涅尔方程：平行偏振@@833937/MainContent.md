## 引言
[电磁波](@entry_id:269629)与物质的相互作用是光学和电磁学领域的核心课题。当光从一种介质传播到另一种介质时，例如从空气进入水中，其部分能量会被反射，部分则会透射。精确描述这一现象的物理规律对于从基础科学研究到尖端技术应用的各个层面都至关重要。然而，光的行为与其偏振状态——即其[电场](@entry_id:194326)矢量的[振荡](@entry_id:267781)方向——密切相关。本文旨在解决一个具体而深刻的问题：当光的[电场](@entry_id:194326)平行于入射面[振荡](@entry_id:267781)（即p偏振光）时，其在介质界面处的反射和透射遵循怎样的定量规律？这与我们熟知的垂直[偏振光](@entry_id:273160)（[s偏振](@entry_id:262966)光）有何本质区别？

为了系统地解答这些问题，本文将引导读者进行一次深入的探索。在**第一章“原理与机制”**中，我们将从[电磁场](@entry_id:265881)的边界条件出发，严谨推导p偏振光的菲涅尔方程，并着重剖析其最独特的特性——布儒斯特角，一个能实现完美透射的“魔法”角度。接着，在**第二章“应用与跨学科联系”**中，我们将展示这些看似抽象的方程如何在现实世界中大放异彩，从我们日常使用的[偏光太阳镜](@entry_id:273077)，到[激光](@entry_id:194225)器中的[布儒斯特窗](@entry_id:272241)，再到前沿的[生物传感](@entry_id:274809)技术，揭示理论与实践的紧密联系。最后，**第三章“动手实践”**将提供一系列计算练习，帮助读者将理论知识转化为解决实际问题的能力。通过这一结构化的学习路径，读者将全面掌握p[偏振光](@entry_id:273160)菲涅尔方程的精髓及其广泛影响。

## 原理与机制

在本章中，我们将深入探讨[电磁波](@entry_id:269629)在两种不同介质界面处的行为，重点关注[平行偏振](@entry_id:266013)光（p[偏振光](@entry_id:273160)）的[反射与折射](@entry_id:184887)。与前一章讨论的垂直[偏振光](@entry_id:273160)（[s偏振](@entry_id:262966)光）相比，p偏振光展现出一些独特的、具有重要物理意义和应用价值的现象。我们将系统地推导和解释描述这些现象的菲涅尔方程，并揭示其背后的物理机制。

### p偏振光的菲涅尔方程

当一束平面[电磁波](@entry_id:269629)入射到两种介质的光滑界面时，其行为由入射波、反射波和透射（或[折射](@entry_id:163428)）波的[电场和磁场](@entry_id:261347)振幅之间的关系决定。我们定义**入射面**为由入射光波矢量和界面法线共同构成的平面。如果光的[电场](@entry_id:194326)矢量平行于入射面[振荡](@entry_id:267781)，我们就称之为**p[偏振光](@entry_id:273160)**（p-polarized light），其中“p”代表“平行”(parallel)。

通过在界面两侧应用[电磁场](@entry_id:265881)的边界条件（[电场和磁场](@entry_id:261347)的切向分量连续），我们可以推导出p[偏振光](@entry_id:273160)的菲涅尔[振幅反射系数](@entry_id:171753) $r_p$ 和[振幅透射系数](@entry_id:165894) $t_p$。这些系数是复数，其模表示振幅的比值，其相位表示相应的相位移动。

对于从[折射率](@entry_id:168910)为 $n_i$ 的介质入射到[折射率](@entry_id:168910)为 $n_t$ 的介质，p[偏振光](@entry_id:273160)的[振幅反射系数](@entry_id:171753) $r_p$ 有两种等价的表达形式：

1.  基于入射角 $\theta_i$ 和[折射](@entry_id:163428)角 $\theta_t$ 的正切形式：
    $$r_p = \frac{\tan(\theta_i - \theta_t)}{\tan(\theta_i + \theta_t)}$$
    这种形式在几何上非常直观，并且能非常方便地揭示p偏振光的一个核心特性，我们稍后将详细讨论。

2.  基于[折射率](@entry_id:168910)和角度余弦的形式：
    $$r_p = \frac{n_t \cos(\theta_i) - n_i \cos(\theta_t)}{n_t \cos(\theta_i) + n_i \cos(\theta_t)}$$
    这种形式在处理全内反射等 $\theta_t$ 变为复数的情形时更为通用和方便。入射角 $\theta_i$ 和折射角 $\theta_t$ 依然通过[斯涅尔定律](@entry_id:162003)（Snell's Law）相关联：$n_i \sin(\theta_i) = n_t \sin(\theta_t)$。

相应地，[振幅透射系数](@entry_id:165894) $t_p$ 为：
$$t_p = \frac{2n_i \cos(\theta_i)}{n_t \cos(\theta_i) + n_i \cos(\theta_t)}$$
这两个系数描述了在界面处[电场](@entry_id:194326)振幅的变化。然而，在光学实验中，我们通常更关心能量的分配，这由**反射率**（Reflectance）和**[透射率](@entry_id:168546)**（Transmittance）来描述。

**[反射率](@entry_id:155393)** $R_p$ 定义为反射光功率与入射[光功率](@entry_id:170412)之比，对于非磁性介质，它等于[振幅反射系数](@entry_id:171753)模的平方：
$$R_p = |r_p|^2$$

**透射率** $T_p$ 的定义则更为精细，它是在界面法线方向上，单位面积上透射的功率通量与入射的功率通量之比。[能量流](@entry_id:142770)由[时间平均](@entry_id:267915)的[坡印廷矢量](@entry_id:269386) $\langle \vec{S} \rangle$ 描述，其法向分量为 $\langle S_{\perp} \rangle = \langle \vec{S} \rangle \cdot \hat{z}$ (假设界面为 $xy$ 平面)。可以证明，对于入射波和透射波，其法向功率通量分别为 $\langle S_{i,\perp} \rangle \propto n_i |E_i|^2 \cos(\theta_i)$ 和 $\langle S_{t,\perp} \rangle \propto n_t |E_t|^2 \cos(\theta_t)$。因此，[透射率](@entry_id:168546) $T_p$ 为 [@problem_id:1799737]：
$$T_p = \frac{\langle S_{t,\perp} \rangle}{\langle S_{i,\perp} \rangle} = \frac{n_t \cos(\theta_t)}{n_i \cos(\theta_i)} \left| \frac{E_t}{E_i} \right|^2 = \frac{n_t \cos(\theta_t)}{n_i \cos(\theta_i)} |t_p|^2$$
将 $t_p$ 的表达式代入，我们得到：
$$T_p = \frac{4 n_i n_t \cos\theta_i \cos\theta_t}{(n_t \cos\theta_i + n_i \cos\theta_t)^2}$$
对于无吸收的介质，[能量守恒](@entry_id:140514)定律要求所有入射能量要么被反射，要么被透射，即：
$$R_p + T_p = 1$$

作为一个具体的计算实例，考虑p偏振光从空气（$n_i=1$）入射到[折射率](@entry_id:168910)为 $n_t=2$ 的介质中，[入射角](@entry_id:192705)为 $\theta_i=30^\circ$。通过[斯涅尔定律](@entry_id:162003)计算出 $\sin(\theta_t) = (1/2) \sin(30^\circ) = 1/4$，进而得到 $\cos(\theta_t) = \sqrt{15}/4$。将这些值代入 $r_p$ 的第二种形式并计算其模的平方，可以求得[反射率](@entry_id:155393) $R_p$ [@problem_id:1582590]。

### 极限情况分析

通过考察两个极限[入射角](@entry_id:192705)——法向入射和掠射入射——我们可以加深对p偏振光行为的理解。

#### 法向入射

当光线垂直于界面入射时，$\theta_i \to 0$。根据斯涅尔定律，$\theta_t \to 0$。此时，$r_p$ 的正切形式呈现出 $0/0$ 的[不定型](@entry_id:150990)。我们可以使用[洛必达法则](@entry_id:147503)或者[小角度近似](@entry_id:145423)（$\tan x \approx x$ 和 $\sin x \approx x$）来求解此极限。从[斯涅尔定律](@entry_id:162003)的[小角度近似](@entry_id:145423) $n_i \theta_i \approx n_t \theta_t$ 可得 $\theta_t \approx (n_i/n_t)\theta_i$。代入正切形式的 $r_p$ 并取极限 [@problem_id:1799748]：
$$r_p(\theta_i \to 0) = \lim_{\theta_i \to 0} \frac{\theta_i - \theta_t}{\theta_i + \theta_t} = \frac{1 - n_i/n_t}{1 + n_i/n_t} = \frac{n_t - n_i}{n_t + n_i}$$
这个结果与[s偏振](@entry_id:262966)光在法向入射时的[反射系数](@entry_id:194350)完全相同。这符合物理直觉：在法向入射时，入射面没有明确定义，因此平行于入射面和垂直于入射面的区别消失了。

#### 掠射入射

当[入射角](@entry_id:192705)趋近于 $90^\circ$（$\pi/2$ 弧度）时，即**掠射入射**（grazing incidence），我们有 $\theta_i \to \pi/2$ 且 $\cos(\theta_i) \to 0$。此时，根据斯涅尔定律，$\sin(\theta_t) \to n_i/n_t$。只要不是全内反射的情况，$\theta_t$ 是一个小于 $\pi/2$ 的确定角度，因此 $\cos(\theta_t)$ 是一个有限的正值。将这些极限值代入 $r_p$ 的第二种形式 [@problem_id:1582561]：
$$r_p(\theta_i \to \pi/2) = \frac{n_t \cdot 0 - n_i \cos(\theta_t)}{n_t \cdot 0 + n_i \cos(\theta_t)} = -1$$
这意味着在掠射入射时，[振幅反射系数](@entry_id:171753)为 $-1$，反射率 $R_p = |-1|^2 = 1$。所有入射的p偏振光都被反射回来，并且相对于入射波，反射波的[电场](@entry_id:194326)经历了 $\pi$ [弧度](@entry_id:171693)的相[位反转](@entry_id:143600)。这是一个普遍现象，适用于任何光滑介质界面。

### 布儒斯特角：零反射的奇迹

p偏振光最引人注目的特性是存在一个特定的入射角，使得[反射率](@entry_id:155393)为零。这个特殊的角度被称为**[布儒斯特角](@entry_id:267927)**（Brewster's angle），记为 $\theta_B$。

#### 数学推导

要找到这个零反射角，我们令 $r_p=0$。使用 $r_p$ 的正切形式最为便捷：
$$r_p = \frac{\tan(\theta_i - \theta_t)}{\tan(\theta_i + \theta_t)} = 0$$
为了使该分式为零，除非分子为零（$\tan(\theta_i - \theta_t) = 0 \Rightarrow \theta_i = \theta_t$，这只在 $n_i=n_t$ 的平庸情况下成立），否则必须是分母趋于无穷大。这意味着：
$$\tan(\theta_i + \theta_t) \to \infty$$
由此我们得到[布儒斯特角](@entry_id:267927)的一个基本几何条件 [@problem_id:1582602]：
$$\theta_B + \theta_t = \frac{\pi}{2}$$
其中 $\theta_B$ 是布儒斯特[入射角](@entry_id:192705)，$\theta_t$ 是相应的[折射](@entry_id:163428)角。这个关系表明，在布儒斯特角入射时，反射光线和折射光线相互垂直。

结合[斯涅尔定律](@entry_id:162003) $n_i \sin(\theta_B) = n_t \sin(\theta_t)$ 和 $\sin(\theta_t) = \sin(\pi/2 - \theta_B) = \cos(\theta_B)$，我们得到：
$$n_i \sin(\theta_B) = n_t \cos(\theta_B)$$
整理后即得到计算布儒斯特角的著名公式：
$$\tan(\theta_B) = \frac{n_t}{n_i}$$

#### 物理机制

布儒斯特角的存在有着深刻的物理根源，可以通过考虑介质中微观偶极子的辐射来解释 [@problem_id:1582613]。当[电磁波](@entry_id:269629)进入介质（如玻璃或水）时，其[电场](@entry_id:194326)驱动介质中的电子做[受迫振荡](@entry_id:169842)。这些[振荡](@entry_id:267781)的电子就像微小的**[电偶极子](@entry_id:186870)**，它们自身会向四面八方辐射[电磁波](@entry_id:269629)。这些次级波在“前方”相干叠加形成[折射](@entry_id:163428)波，在“后方”[相干叠加](@entry_id:170209)则形成反射波。

电偶极子辐射的一个基本特性是：**它不会沿着自身的[振荡](@entry_id:267781)轴方向辐射能量**。对于p偏振光，其[电场](@entry_id:194326)矢量（以及由此驱动的偶极子[振荡](@entry_id:267781)方向）位于入射面内，并且始终垂直于其自身的传播方向。因此，透射波中的[电偶极子](@entry_id:186870)[振荡](@entry_id:267781)方向垂直于折射光线的传播方向。

在[布儒斯特角](@entry_id:267927)入射时，我们已经知道反射光线的方向与[折射](@entry_id:163428)光线的方向垂直。这意味着，透射介质中电偶极子的[振荡](@entry_id:267781)轴方向（垂直于折射光线）恰好指向了反射光线应该传播的方向。由于偶极子不沿其轴向辐射，因此在那个方向上没有辐射产生，即没有反射光。这个优雅的物理图像完美地解释了为何p[偏振光](@entry_id:273160)在[布儒斯特角](@entry_id:267927)会发生零反射现象 [@problem_id:1582613] [@problem_id:1799716]。

### 反射中的相移

反射[波的相位](@entry_id:171303)是否相对于入射波发生变化，取决于[振幅反射系数](@entry_id:171753) $r_p$ 的符号。一个负的实数 $r_p$ 意味着一个大小为 $|r_p|$ 的振幅比和一个 $\pi$ 弧度（$180^\circ$）的相移。

我们来分析 $r_p = \frac{n_t \cos(\theta_i) - n_i \cos(\theta_t)}{n_t \cos(\theta_i) + n_i \cos(\theta_t)}$ 的符号。由于分母总是正的，符号由分子 $N = n_t \cos(\theta_i) - n_i \cos(\theta_t)$ 决定。

#### 外反射 ($n_i  n_t$)

这种情况对应于光从光疏介质（如空气）射向光密介质（如水或玻璃）。
- 在法向入射时（$\theta_i=0$），$r_p = (n_t - n_i) / (n_t + n_i)  0$。没有相移。
- 随着 $\theta_i$ 从0增加，$\cos(\theta_i)$ 减小，而 $\cos(\theta_t)$ 也减小但速度较慢。分子 $N$ 的值随之减小。
- 当 $\theta_i = \theta_B$ 时，$N=0$，$r_p=0$。
- 当 $\theta_i  \theta_B$ 时，分子 $N$ 变为负值，因此 $r_p  0$。这意味着反射波发生了一个 $\pi$ 的相移 [@problem_id:1582593]。

总结来说，对于外反射，p偏振光在 $0 \le \theta_i  \theta_B$ 范围内反射时没有相移，而在 $\theta_B  \theta_i \le \pi/2$ 范围内反射时有 $\pi$ 的相移。例如，对于从空气（$n_i=1.00$）到水（$n_t=1.33$）的界面，布儒斯特角为 $\theta_B = \arctan(1.33) \approx 53.1^\circ$。因此，当入射角在 $[53.1^\circ, 90.0^\circ]$ 区间内时，反射的p偏振光会经历 $180^\circ$ 的相移 [@problem_id:1799738]。

### p[偏振光](@entry_id:273160)的全内反射

当光从光密介质射向光疏介质（$n_i > n_t$）时，如果[入射角](@entry_id:192705)大于一个特定的**[临界角](@entry_id:169189)** $\theta_c = \arcsin(n_t/n_i)$，就会发生**全内反射**（Total Internal Reflection, TIR）。

当 $\theta_i > \theta_c$ 时，斯涅尔定律 $n_i \sin(\theta_i) = n_t \sin(\theta_t)$ 导致 $\sin(\theta_t) = (n_i/n_t)\sin(\theta_i)  1$。这是一个无实数解的方程，意味着 $\theta_t$ 必须是一个复数。因此，$\cos(\theta_t)$ 也变为一个纯虚数：
$$\cos(\theta_t) = \sqrt{1 - \sin^2(\theta_t)} = i \sqrt{\sin^2(\theta_t) - 1} = i \alpha$$
其中 $\alpha = \sqrt{((n_i/n_t)\sin(\theta_i))^2 - 1}$ 是一个正实数。

将这个纯虚的 $\cos(\theta_t)$ 代入 $r_p$ 的表达式中 [@problem_id:1799725]：
$$r_p = \frac{n_t \cos(\theta_i) - n_i (i \alpha)}{n_t \cos(\theta_i) + n_i (i \alpha)}$$
这个表达式的形式是 $\frac{a - ib}{a + ib}$，其中 $a=n_t\cos(\theta_i)$ 和 $b=n_i\alpha$ 都是实数。这是一个复数，其分子是分母的共轭。因此，它的模长恒为1：
$$|r_p| = \left| \frac{a - ib}{a + ib} \right| = \frac{\sqrt{a^2 + (-b)^2}}{\sqrt{a^2 + b^2}} = 1$$
由于反射率 $R_p = |r_p|^2$，我们得到 $R_p = 1$。这从数学上证明了在[全内反射](@entry_id:179014)条件下，所有入射的p偏振光都被完[全反射](@entry_id:179014)，没有能量进入第二介质的[远场](@entry_id:269288)。尽管能量被完[全反射](@entry_id:179014)，但反射波会经历一个非0或 $\pi$ 的特定相移 $\delta_p$，该相移随[入射角](@entry_id:192705)的变化而变化。同时，在第二介质中会存在一个沿界面传播并迅速衰减的倏逝波（evanescent wave）。
