## 引言
在量子力学的奇妙世界里，我们对“相同”的认知被彻底颠覆。与经典世界中可以被轻易区分的宏观物体不同，微观领域中的全同粒子，如电子或[光子](@entry_id:145192)，是真正不可区分的。这一深刻的“不可区分性”原理并非测量技术的局限，而是自然界的基本法则，它直接导致了物质被划分为两大基本家族：[玻色子](@entry_id:138266)和[费米子](@entry_id:146235)。理解这两类粒子行为的根本差异，是解开从原子稳定性、化学元素周期性到恒星演化终局等一系列物理之谜的关键。本文旨在填补经典直觉与量子现实之间的鸿沟，系统地阐明支配[全同粒子](@entry_id:142755)系统的量子统计规则。

在接下来的内容中，我们将踏上一段从基本原理到前沿应用的探索之旅。在“原理与机制”一章中，我们将深入探讨[交换对称性](@entry_id:151892)的概念，揭示它如何催生了[费米子](@entry_id:146235)所遵循的[泡利不相容原理](@entry_id:141850)，以及[玻色子](@entry_id:138266)独特的聚集行为。随后，在“应用与跨学科联系”一章，我们将看到这些抽象规则如何在原子物理、[凝聚态物质](@entry_id:747660)、天体物理乃至粒子物理中展现其塑造世界的力量，解释超导、白矮星稳定性和[夸克模型](@entry_id:147763)等多种现象。最后，通过“动手实践”部分，你将有机会运用所学知识解决具体问题，加深对这些核心概念的理解。

## 原理与机制

在[经典物理学](@entry_id:150394)中，我们可以通过连续追踪其轨迹来区分两个相同的粒子，例如两个钢珠。然而，在量子领域，粒子由弥散的[波函数](@entry_id:147440)描述，它们的身份因海森堡不确定性原理而变得模糊不清。当两个相同粒子的[波函数](@entry_id:147440)重叠时，我们无法确定哪个是哪个。这一事实并非我们测量能力的局限，而是自然界的一个基本属性。全同粒子是真正不可区分的。这一深刻的见解是理解多粒子量子系统的基石，它将我们引向一套全新的规则，这些规则支配着从[原子结构](@entry_id:137190)到恒星核心的万物。本章将深入探讨[全同粒子](@entry_id:142755)的基本原理及其深远的物理后果。

### 不可区分性与[交换对称性](@entry_id:151892)

考虑一个由 $N$ 个[全同粒子](@entry_id:142755)组成的系统。系统的状态由一个总[波函数](@entry_id:147440) $\Psi(1, 2, ..., N)$ 描述，其中数字 $1, 2, ..., N$ 代表每个粒子的全套坐标（例如，空间位置 $\vec{r}$ 和自旋分量 $s_z$）。

由于粒子是不可区分的，任何可观测的物理量——例如[概率密度](@entry_id:175496) $|\Psi|^2$——在交换任意两个粒子（比如粒子 $i$ 和粒子 $j$）的坐标后，必须保持不变。这意味着：
$|\Psi(..., i, ..., j, ...)|^2 = |\Psi(..., j, ..., i, ...)|^2$

这暗示交换后的[波函数](@entry_id:147440)最多只能与原[波函数](@entry_id:147440)相差一个相位因子，即 $\Psi(..., j, ..., i, ...) = e^{i\alpha} \Psi(..., i, ..., j, ...)$。

为了更形式化地描述这一过程，我们引入**[交换算符](@entry_id:156554)**（**particle exchange operator**）$\hat{P}_{ij}$。该算符的作用是交换粒子 $i$ 和粒子 $j$ 的所有坐标：
$\hat{P}_{ij} \Psi(..., i, ..., j, ...) = \Psi(..., j, ..., i, ...)$

再次应用该算符会使系统恢复到原始状态：
$\hat{P}_{ij}^2 \Psi(..., i, ..., j, ...) = \hat{P}_{ij} \Psi(..., j, ..., i, ...) = \Psi(..., i, ..., j, ...)$
因此，$\hat{P}_{ij}^2 = \mathbb{I}$（其中 $\mathbb{I}$ 是单位算符）。这意味着[交换算符](@entry_id:156554)的[本征值](@entry_id:154894)只能是 $+1$ 或 $-1$。

对于[全同粒子](@entry_id:142755)系统，物理上允许的[波函数](@entry_id:147440)必须是[交换算符](@entry_id:156554)的本征态。这是因为系统的[哈密顿量](@entry_id:172864) $\hat{H}$ 对于[粒子交换](@entry_id:154910)必然是不变的（因为粒子是相同的），所以 $[\hat{H}, \hat{P}_{ij}] = 0$。这意味着[哈密顿量](@entry_id:172864)的[本征态](@entry_id:149904)（即系统的定态）可以（也必须）同时是[交换算符](@entry_id:156554)的本征态。

一个简单的乘积态，例如 $|\Psi(1,2)\rangle = |\phi_A(1)\rangle |\phi_B(2)\rangle$，其中粒子1处于状态 $\phi_A$，粒子2处于状态 $\phi_B$，通常不满足这个要求 [@problem_id:2082537]。对这个态应用[交换算符](@entry_id:156554)得到 $\hat{P}_{12} |\Psi(1,2)\rangle = |\phi_A(2)\rangle |\phi_B(1)\rangle = |\phi_B(1)\rangle |\phi_A(2)\rangle$。除非 $\phi_A$ 和 $\phi_B$ 是同一个态，否则交换后的态既不等于原始态，也不等于原始态的负值。因此，这种简单的乘积[波函数](@entry_id:147440)对于描述全同粒子是物理上不可接受的，因为它没有确定的**[交换对称性](@entry_id:151892)**（**exchange symmetry**）[@problem_id:2082558]。

为了构建具有确定对称性的态，我们必须采用线性组合。对于两个粒子分别占据不同单粒子态 $|\phi_A\rangle$ 和 $|\phi_B\rangle$ 的系统，我们可以构造两种具有确定对称性的组合：
对称组合：$|\Psi_S(1,2)\rangle = \frac{1}{\sqrt{2}} (|\phi_A(1)\rangle|\phi_B(2)\rangle + |\phi_B(1)\rangle|\phi_A(2)\rangle)$
反对称组合：$|\Psi_A(1,2)\rangle = \frac{1}{\sqrt{2}} (|\phi_A(1)\rangle|\phi_B(2)\rangle - |\phi_B(1)\rangle|\phi_A(2)\rangle)$

对这两个态应用[交换算符](@entry_id:156554)，我们发现：
$\hat{P}_{12} |\Psi_S(1,2)\rangle = \frac{1}{\sqrt{2}} (|\phi_A(2)\rangle|\phi_B(1)\rangle + |\phi_B(2)\rangle|\phi_A(1)\rangle) = + |\Psi_S(1,2)\rangle$
$\hat{P}_{12} |\Psi_A(1,2)\rangle = \frac{1}{\sqrt{2}} (|\phi_A(2)\rangle|\phi_B(1)\rangle - |\phi_B(2)\rangle|\phi_A(1)\rangle) = - |\Psi_A(1,2)\rangle$

这两种组合都是[交换算符](@entry_id:156554)的[本征态](@entry_id:149904)，[本征值](@entry_id:154894)分别为 $+1$（对称）和 $-1$（反对称），因此它们是描述[全同粒子](@entry_id:142755)系统的有效候选态 [@problem_id:2082558]。

### [玻色子与费米子](@entry_id:147957)：[对称化公设](@entry_id:148962)

自然界中的所有基本粒子都优雅地分为两大家族，这取决于它们的内禀自旋和它们所遵循的[交换对称性](@entry_id:151892)规则。这一深刻的联系被称为**[自旋统计定理](@entry_id:147864)**（Spin-Statistics Theorem），或者更操作性地称为**[对称化公设](@entry_id:148962)**（Symmetrization Postulate）。

1.  **[玻色子](@entry_id:138266) (Bosons)**：具有整数自旋（$s = 0, 1, 2, ...$）的粒子。由全同[玻色子](@entry_id:138266)组成的系统的总[波函数](@entry_id:147440)在交换任意两个粒子时必须是**对称的**。
    $\hat{P}_{ij} \Psi_{\text{Boson}} = +\Psi_{\text{Boson}}$
    典型的[玻色子](@entry_id:138266)包括[光子](@entry_id:145192)（$s=1$）、胶子（$s=1$）以及赋予[粒子质量](@entry_id:156313)的[希格斯玻色子](@entry_id:155560)（$s=0$）。

2.  **[费米子](@entry_id:146235) (Fermions)**：具有[半整数自旋](@entry_id:148826)（$s = 1/2, 3/2, ...$）的粒子。由全同[费米子](@entry_id:146235)组成的系统的总[波函数](@entry_id:147440)在交换任意两个粒子时必须是**反对称的**。
    $\hat{P}_{ij} \Psi_{\text{Fermion}} = -\Psi_{\text{Fermion}}$
    构成我们日常物质世界的基本粒子，如电子、质子、中子以及夸克，都是[费米子](@entry_id:146235)（均为 $s=1/2$）。

这个规则也适用于复合粒子。一个由多个基本[费米子](@entry_id:146235)组成的复合粒子（如[原子核](@entry_id:167902)或整个原子），如果其包含的[费米子](@entry_id:146235)总数为偶数，则该[复合粒子](@entry_id:150176)表现为[玻色子](@entry_id:138266)；如果总数为奇数，则表现为[费米子](@entry_id:146235) [@problem_id:1983927]。例如，一个中性的氦-4（$^4$He）原子包含2个质子、2个中子和2个电子，总共有6个[费米子](@entry_id:146235)（偶数），因此 $^4$He 原子是[玻色子](@entry_id:138266)。相比之下，[氦-3](@entry_id:195175)（$^3$He）原子包含2个质子、1个中子和2个电子，总共有5个[费米子](@entry_id:146235)（奇数），因此 $^3$He 原子是[费米子](@entry_id:146235)。这种差异在低温下会导致截然不同的宏观行为，例如液氦的超流性。

### 构建多粒子[波函数](@entry_id:147440)

多粒子[波函数](@entry_id:147440)的对称性要求是对**总[波函数](@entry_id:147440)**（包含空间和自旋部分）的要求。总[波函数](@entry_id:147440)通常可以近似写成空间部分与自旋部分的乘积：$\Psi_{total} = \psi_{spatial} \chi_{spin}$。总[波函数](@entry_id:147440)的对称性由其各部分的对称性组合决定。

-   对称(S) $\times$ 对称(S) = 对称(S)
-   反对称(A) $\times$ 反对称(A) = 对称(S)
-   对称(S) $\times$ 反对称(A) = 反对称(A)
-   反对称(A) $\times$ 对称(S) = 反对称(A)

对于[费米子](@entry_id:146235)系统，总[波函数](@entry_id:147440)必须是反对称的。这意味着如果空间部分是对称的，自旋部分必须是反对称的；反之亦然。对于[玻色子](@entry_id:138266)系统，总[波函数](@entry_id:147440)必须是对称的，所以空间和自旋部分必须具有相同的对称性。

让我们考虑一个由两个自旋1/2的[费米子](@entry_id:146235)组成的系统。自旋部分可以组合成一个反对称的**自旋单态**（spin singlet, 总自旋$S=0$）或三个对称的**自旋[三重态](@entry_id:156705)**（spin triplet, 总自旋$S=1$）。

-   自旋单态 (A): $\chi_A = \frac{1}{\sqrt{2}}(|\uparrow_1\downarrow_2\rangle - |\downarrow_1\uparrow_2\rangle)$
-   自旋三重态 (S): $$ \begin{cases} |\uparrow_1\uparrow_2\rangle \\ \frac{1}{\sqrt{2}}(|\uparrow_1\downarrow_2\rangle + |\downarrow_1\uparrow_2\rangle) \\ |\downarrow_1\downarrow_2\rangle \end{cases} $$

考虑一个[波函数](@entry_id:147440) [@problem_id:1966129]：
$\Psi(1, 2) = \frac{1}{2} \left[ \psi_A(\vec{r}_1)\psi_B(\vec{r}_2) + \psi_A(\vec{r}_2)\psi_B(\vec{r}_1) \right] \left[ |\uparrow_1\downarrow_2\rangle - |\downarrow_1\uparrow_2\rangle \right]$
其空间部分 $\psi_{spatial} = \frac{1}{\sqrt{2}}[\psi_A(\vec{r}_1)\psi_B(\vec{r}_2) + \psi_A(\vec{r}_2)\psi_B(\vec{r}_1)]$ 是对称的。其自旋部分 $\chi_{spin} = \frac{1}{\sqrt{2}}[|\uparrow_1\downarrow_2\rangle - |\downarrow_1\uparrow_2\rangle]$ 是反对称的（自旋单态）。由于总[波函数](@entry_id:147440)是 $\Psi_{total} = \psi_S \chi_A$，它是反对称的。因此，这个[波函数](@entry_id:147440)描述的是一对处于自旋单态的[费米子](@entry_id:146235)。

#### [泡利不相容原理](@entry_id:141850)

[费米子](@entry_id:146235)[波函数](@entry_id:147440)的反对称性要求带来了一个极为重要的推论：**[泡利不相容原理](@entry_id:141850)**（**Pauli exclusion principle**）。该原理指出：**两个全同的[费米子](@entry_id:146235)不能占据完全相同的单粒子[量子态](@entry_id:146142)。**

我们可以从数学上轻松验证这一点。假设两个[费米子](@entry_id:146235)试图占据同一个单粒子态 $|\phi_k\rangle$（包括空间和自旋）。根据[反对称性](@entry_id:261893)要求，我们需要构建一个反对称的总[波函数](@entry_id:147440)：
$\Psi(1, 2) = \frac{1}{\sqrt{2}} (|\phi_k(1)\rangle|\phi_k(2)\rangle - |\phi_k(2)\rangle|\phi_k(1)\rangle) = 0$
[波函数](@entry_id:147440)恒为零，意味着这种状态在物理上是不存在的。

[泡利不相容原理](@entry_id:141850)是理解原子结构和元素周期表的关键。例如，考虑[氦原子](@entry_id:150244)（Helium）的[基态](@entry_id:150928) [@problem_id:1983911]。它有两个电子。为了获得最低能量，两个电子都想占据能量最低的 $1s$ 空间[轨道](@entry_id:137151)。因此，它们的空间[波函数](@entry_id:147440)是 $\psi_{spatial}(\mathbf{r}_1, \mathbf{r}_2) = \phi_{1s}(\mathbf{r}_1)\phi_{1s}(\mathbf{r}_2)$，这是一个[对称函数](@entry_id:177113)。为了使总[波函数](@entry_id:147440)对电子交换是反对称的，自旋部分必须是反对称的。这意味着两个电子的自旋必须配对成自旋单态。所以，氦原子的[基态](@entry_id:150928)[波函数](@entry_id:147440)是：
$\Psi(1, 2) = \phi_{1s}(\mathbf{r}_1)\phi_{1s}(\mathbf{r}_2) \frac{1}{\sqrt{2}} (|\alpha(1)\beta(2)\rangle - |\beta(1)\alpha(2)\rangle)$
这里 $|\alpha\rangle$ 和 $|\beta\rangle$ 分别代表自旋向上和自旋向下的态。

#### [斯莱特行列式](@entry_id:139034)

对于一个包含 $N$ 个[费米子](@entry_id:146235)的系统，系统地构造一个满足[反对称性](@entry_id:261893)要求的总[波函数](@entry_id:147440)的方法是使用**[斯莱特行列式](@entry_id:139034)**（**Slater determinant**）。如果 $N$ 个[费米子](@entry_id:146235)占据的单[粒子自旋](@entry_id:142910)-[轨道](@entry_id:137151)态是 $\{\psi_1, \psi_2, ..., \psi_N\}$，那么归一化后的总[波函数](@entry_id:147440)为：
$\Psi(1, 2, ..., N) = \frac{1}{\sqrt{N!}} \begin{vmatrix} \psi_1(1)  \psi_2(1)  \cdots  \psi_N(1) \\ \psi_1(2)  \psi_2(2)  \cdots  \psi_N(2) \\ \vdots  \vdots  \ddots  \vdots \\ \psi_1(N)  \psi_2(N)  \cdots  \psi_N(N) \end{vmatrix}$
[行列式](@entry_id:142978)的两个基本性质自动保证了费米统计的要求：
1.  交换任意两行（对应交换两个粒子）会使[行列式](@entry_id:142978)变号，这满足了[反对称性](@entry_id:261893)。
2.  如果任意两列相同（即两个粒子占据同一个单粒子态），[行列式](@entry_id:142978)的值为零，这体现了[泡利不相容原理](@entry_id:141850)。

例如，锂原子（Lithium）有三个电子。其[基态](@entry_id:150928)电子组态是 $1s^2 2s^1$。这意味着三个电子分别占据了三个不同的自旋-[轨道](@entry_id:137151)态：$1s$ 空间[轨道](@entry_id:137151)配自旋向上（我们记为 $\psi_a$），$1s$ 空间[轨道](@entry_id:137151)配自旋向下（$\psi_b$），以及 $2s$ 空间[轨道](@entry_id:137151)配自旋向上（$\psi_c$）。该系统的[基态](@entry_id:150928)[波函数](@entry_id:147440)可以用一个 $3 \times 3$ 的[斯莱特行列式](@entry_id:139034)来表示 [@problem_id:1983881]：
$\Psi(1, 2, 3) = \frac{1}{\sqrt{6}} \begin{vmatrix} \psi_a(1)  \psi_b(1)  \psi_c(1) \\ \psi_a(2)  \psi_b(2)  \psi_c(2) \\ \psi_a(3)  \psi_b(3)  \psi_c(3) \end{vmatrix}$

### [交换对称性](@entry_id:151892)的物理后果

[交换对称性](@entry_id:151892)不仅仅是一个抽象的数学约束，它导致了可观测的、深刻的物理现象，从根本上区分了[玻色子](@entry_id:138266)和[费米子](@entry_id:146235)的世界。

#### [粒子统计](@entry_id:145640)与态占据

最直接的后果体现在粒子如何占据可用的[量子态](@entry_id:146142)。
-   对于**[费米子](@entry_id:146235)**，[泡利不相容原理](@entry_id:141850)限制每个单粒子态的占据数 $n_k$ 只能是 $0$ 或 $1$。
-   对于**[玻色子](@entry_id:138266)**，没有这样的限制，任意数量的全同[玻色子](@entry_id:138266)都可以占据同一个单粒子态 ($n_k = 0, 1, 2, ...$)。

这种差异极大地影响了系统的可能状态数。假设有3个粒子要[分布](@entry_id:182848)在5个不同的单粒子态上 [@problem_id:1966095]。
-   如果它们是[费米子](@entry_id:146235)，我们必须从5个态中选择3个不同的态来放置粒子。总的状态数是组合数：
    $\Omega_F = \binom{5}{3} = \frac{5!}{3!2!} = 10$
-   如果它们是[玻色子](@entry_id:138266)，粒子是不可区分的，但它们可以占据相同的态。这相当于一个“[隔板法](@entry_id:152143)”问题，状态数为：
    $\Omega_B = \binom{3+5-1}{3} = \binom{7}{3} = \frac{7!}{3!4!} = 35$
显然，在相同的条件下，[玻色子](@entry_id:138266)系统可实现的微观状态数远多于[费米子](@entry_id:146235)系统。在低温下，[玻色子](@entry_id:138266)倾向于聚集在能量最低的态上，最终导致**玻色-爱因斯坦凝聚**（Bose-Einstein Condensation）。

#### 空间关联：[聚束与反聚束](@entry_id:194025)

即使粒子之间没有经典的相互作用（如[库仑力](@entry_id:174598)），[交换对称性](@entry_id:151892)也会在它们之间引入一种纯粹量子力学性质的有效“相互作用”，称为**[交换力](@entry_id:149395)**（exchange force）。这种效应影响了在空间中同时找到两个粒子的概率。

考虑两个粒子，它们的空间[波函数](@entry_id:147440)是对称的（$\Psi_S$）或反对称的（$\Psi_A$）。找到粒子1在 $x_1$、粒子2在 $x_2$ 的[联合概率](@entry_id:266356)密度分别是 $P_S(x_1,x_2) = |\Psi_S(x_1,x_2)|^2$ 和 $P_A(x_1,x_2) = |\Psi_A(x_1,x_2)|^2$。

让我们考察在同一点找到两个粒子的概率，即令 $x_1 = x_2 = x$。
-   对于反对称空间态（例如，自旋平行的两个[费米子](@entry_id:146235)）：
    $\Psi_A(x, x) = \frac{1}{\sqrt{2}} (\psi_a(x)\psi_b(x) - \psi_b(x)\psi_a(x)) = 0$
    这意味着两个处于相同自旋态的[费米子](@entry_id:146235)（或者任何空间[波函数](@entry_id:147440)是反对称的[费米子](@entry_id:146235)对）在空间同一点出现的概率为零。它们倾向于互相“排斥”，这种现象被称为**量子[反聚束](@entry_id:194774)**（quantum anti-bunching）或**交换洞**（exchange hole）。
-   对于[对称空间](@entry_id:181790)态（适用于[玻色子](@entry_id:138266)，或自旋反平行的[费米子](@entry_id:146235)）：
    $|\Psi_S(x, x)|^2 = |\frac{1}{\sqrt{2}} (\psi_a(x)\psi_b(x) + \psi_b(x)\psi_a(x))|^2 = 2|\psi_a(x)\psi_b(x)|^2$
    这个概率是如果粒子可区[分时](@entry_id:274419)的两倍。这意味着[玻色子](@entry_id:138266)（或处于[对称空间](@entry_id:181790)态的[费米子](@entry_id:146235)）在空间中同一点出现的概率被增强了。它们倾向于“聚集”在一起，这种现象被称为**量子聚束**（quantum bunching）。

这种聚束和[反聚束](@entry_id:194774)效应是真实可测的。例如，考虑一个[一维无限深势阱](@entry_id:271157)中的两个粒子，一个在 $n=1$ 态，一个在 $n=2$ 态。通过计算在特定位置（如 $x_1 = L/6, x_2 = L/3$）找到它们的[概率密度](@entry_id:175496)之比 $R = P_S/P_A$，可以发现该比值显著偏离1，这定量地展示了[交换对称性](@entry_id:151892)如何深刻地扭曲了粒子的[空间分布](@entry_id:188271) [@problem_id:2082496]。

#### 简并压力

[泡利不相容原理](@entry_id:141850)最宏观、最壮观的后果之一是**简并压力**（**degeneracy pressure**）。在一个[费米子](@entry_id:146235)气体中，即使在绝对[零度](@entry_id:156285)（$T=0$），粒子也不能全部挤在最低能量态。它们必须从低到高依次填充所有可用的能级，直到最高占据能级，即**费米能**（Fermi energy）。

因此，即使在 $T=0$ 时，[费米子](@entry_id:146235)气体也具有巨大的总动能。这个总能量依赖于系统的体积 $V$。根据[热力学](@entry_id:141121)关系，系统会抵抗压缩，从而产生一种压力 $P = -(\partial E_{tot}/\partial V)$。这种纯粹由[泡利不相容原理](@entry_id:141850)引起的压力就是[简并压力](@entry_id:141985)。

考虑一个简单的模型：将 $N$ 个电子（[费米子](@entry_id:146235)）限制在一个一维[量子线](@entry_id:142481)中 [@problem_id:2082544]。根据泡利原理，每个能级最多容纳两个自旋相反的电子。例如，4个电子将填充 $n=1$ 和 $n=2$ 能级。系统的总能量为 $E_{tot} \propto 1/L^2$。因此，[电子气](@entry_id:140692)体会对限制壁施加一个力 $F = -dE_{tot}/dL \propto 1/L^3$。这个力即使在 $T=0$ 也存在。

这个原理在天体物理学中至关重要。[白矮星](@entry_id:159122)的巨大[引力](@entry_id:175476)试图使其坍缩，但这种坍缩被其内部[电子气](@entry_id:140692)的[简并压力](@entry_id:141985)所阻止 [@problem_id:2082521]。对于一个非相对论性的费米气体，可以推导出其简并压力 $P \propto (N/V)^{5/3}$。正是这种量子压力支撑着像太阳这样的恒星在耗尽核燃料后留下的致密残骸。如果[恒星质量](@entry_id:157648)更大，[引力能](@entry_id:193726)克服[电子简并压力](@entry_id:143329)，恒星会进一步坍缩成[中子星](@entry_id:147259)，此时由中子（也是[费米子](@entry_id:146235)）的简并压力来抵抗[引力](@entry_id:175476)。

与此形成鲜明对比的是，一个由非相互作用的[玻色子](@entry_id:138266)组成的气体，在 $T=0$ 时，所有粒子都可以占据[基态](@entry_id:150928)。[基态](@entry_id:150928)动能很小，因此在 $T=0$ 时它们不产生简并压力。这再次凸显了[费米子与玻色子](@entry_id:138279)在行为上的根本差异，这种差异源于它们截然不同的[交换对称性](@entry_id:151892)规则。
