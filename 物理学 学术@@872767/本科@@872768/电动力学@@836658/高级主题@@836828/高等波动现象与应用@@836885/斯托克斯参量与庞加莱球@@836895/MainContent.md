## 引言
[光的偏振](@entry_id:262080)是理解[光与物质相互作用](@entry_id:142166)的关键，从我们日常使用的[液晶](@entry_id:147648)显示屏到天文学家探测遥远星系[磁场](@entry_id:153296)的工具，其应用无处不在。虽然琼斯微积分提供了一种优雅的方式来描述完全[偏振光](@entry_id:273160)，但它在面对现实世界中更普遍存在的[部分偏振光](@entry_id:267467)或非偏振光时却显得力不从心。如何建立一个能够统一描述所有偏振态的普适框架，是光学领域的一个基本问题。

本文旨在系统地介绍解决这一问题的强大工具：斯托克斯参数和[庞加莱球](@entry_id:265729)。通过学习本文，您将能够超越琼斯微积分的局限，掌握一个与物理测量直接关联的理论体系。

在接下来的章节中，我们将首先在“原理与机制”中深入探讨斯托克斯参数的定义、物理意义，并构建[庞加莱球](@entry_id:265729)这一优美的几何模型。随后，在“应用与交叉学科联系”中，我们将展示这一理论如何在[光学工程](@entry_id:272219)、天文学乃至基础物理学中发挥作用，连接起看似无关的现象。最后，通过“动手实践”部分的练习，您将有机会巩固所学知识，将其应用于具体问题的分析与计算中。让我们开始探索这个描述光偏振的完整世界。

## 原理与机制

在光学中，虽然琼斯微积分 ([Jones calculus](@entry_id:182044)) 为描述完全偏振光的相干叠加提供了一个强大的数学框架，但它无法描述[部分偏振](@entry_id:187644)或非偏振光。为了克服这一限制，我们需要一个更普适的体系，能够刻画任何[偏振态](@entry_id:175130)，无论是完全偏振、[部分偏振](@entry_id:187644)还是完全非偏振。这个体系便是基于斯托克斯参数 (Stokes parameters) 的描述方法，它直接与可测量的光强相关联。本章将系统地阐述斯托克斯参数的定义、物理意义及其在[庞加莱球](@entry_id:265729) (Poincaré sphere) 上的几何表示，从而为理解和分析[光的偏振](@entry_id:262080)态提供一个完整而直观的工具。

### 斯托克斯参数的定义

描述光束[偏振态](@entry_id:175130)的斯托克斯参数是一个由四个实数组成的矢量 $(S_0, S_1, S_2, S_3)$，它由 George Gabriel Stokes 于1852年引入。这四个参数都具有光强的量纲，并且可以直接通过实验测量得到。

#### 从物理测量出发的定义

斯托克斯参数最直观的理解方式是将其与一系列强度测量联系起来。想象我们有一束待测光，并使用一系列[偏振片](@entry_id:269119)和[光电探测器](@entry_id:264291)来测量通过它们的光强。

*   **$S_0$ (总光强):** 第一个参数 $S_0$ 代表光束的总光强。它可以简单地通过在光路中不放置任何[偏振片](@entry_id:269119)，直接测量探测器接收到的光强来获得。

*   **$S_1$ (水平/垂直线偏振优势):** 第二个参数 $S_1$ 描述了光束中水平线偏振分量相对于[垂直线](@entry_id:174147)偏振分量的优势。具体而言，如果我们分别测量光束通过水平（$0^\circ$）[偏振片](@entry_id:269119)和垂直（$90^\circ$）[偏振片](@entry_id:269119)后的光强，记为 $I_H$ 和 $I_V$，那么 $S_1$ 定义为它们的差值：
    $S_1 = I_H - I_V$
    一个正的 $S_1$ 值表示光束偏向于水平偏振，负值表示偏向于垂直偏振，而零值表示两者没有优势差别。值得注意的是，总光强 $S_0$ 也等于这两个光强之和：$S_0 = I_H + I_V$ [@problem_id:1606739]。

*   **$S_2$ (+45°/-45°线偏振优势):** 第三个参数 $S_2$ 描述了光束中 $+45^\circ$ 线偏振分量相对于 $-45^\circ$（或等效的 $135^\circ$）线偏振分量的优势。类似地，通过测量光束通过这两个方向的[线偏振](@entry_id:273116)片后的光强 $I_{45}$ 和 $I_{135}$，我们定义：
    $S_2 = I_{45} - I_{135}$
    这个参数量化了光在对角线方向上的偏振倾向 [@problem_id:1606740]。

*   **$S_3$ (右旋/左旋圆偏振优势):** 第四个参数 $S_3$ 则量化了[圆偏振](@entry_id:261702)的特性。它定义为光束中右旋圆偏振 (Right-Circularly Polarized, RHC) 分量的光强 $I_R$ 与左旋圆偏振 (Left-Circularly Polarized, LHC) 分量的光强 $I_L$ 之差：
    $S_3 = I_R - I_L$
    一个正的 $S_3$ 值表示光束具有[右旋圆偏振](@entry_id:267955)的倾向，负值表示具有左旋[圆偏振](@entry_id:261702)的倾向。

总的来说，斯托克斯参数 $(S_1, S_2, S_3)$ 分别回答了三个关于偏振优势的基本问题：水平 vs. 垂直？+45° vs. -45°？右旋 vs. 左旋？这种基于强度差的定义使得斯托克斯参数成为一种极其有用的实验工具。

#### 从[电场](@entry_id:194326)分量出发的定义

斯托克斯参数也可以从更底层的[电场](@entry_id:194326)矢量来定义，从而与描述相干光的琼斯微积分建立联系。考虑一束沿 z 轴传播的单色平面[电磁波](@entry_id:269629)，其[电场](@entry_id:194326)分量可以写为：
$E_x(t) = E_{0x} \cos(\omega t + \delta_x)$
$E_y(t) = E_{0y} \cos(\omega t + \delta_y)$
或者使用更方便的复数表示法，[琼斯矢量](@entry_id:176164)为 $\vec{J} = \begin{pmatrix} E_x \\ E_y \end{pmatrix} = \begin{pmatrix} E_{0x} e^{i\delta_x} \\ E_{0y} e^{i\delta_y} \end{pmatrix}$。斯托克斯参数由这些[复振幅](@entry_id:164138)的[时间平均](@entry_id:267915)值定义（在光学频率下，时间平均值正比于振幅的模的平方）：

$S_0 = |E_x|^2 + |E_y|^2$
$S_1 = |E_x|^2 - |E_y|^2$
$S_2 = 2 \text{Re}(E_x^* E_y)$
$S_3 = 2 \text{Im}(E_x^* E_y)$

这里，$^*$ 表示复共轭，$\text{Re}(\cdot)$ 和 $\text{Im}(\cdot)$ 分别表示复数的实部和虚部。请注意，关于 $S_3$ 的定义存在不同的符号约定；我们在此采用的定义在物理学中较为常见。

让我们通过一个具体的例子来理解这些定义。假设一个[琼斯矢量](@entry_id:176164)为 $\vec{J} = \begin{pmatrix} A \\ A e^{i\pi/4} \end{pmatrix}$，其中 $A$ 是一个实常数 [@problem_id:1606694]。我们有 $E_x = A$ 和 $E_y = A e^{i\pi/4}$。
$S_0 = |A|^2 + |A e^{i\pi/4}|^2 = A^2 + A^2 = 2A^2$
$S_1 = |A|^2 - |A e^{i\pi/4}|^2 = A^2 - A^2 = 0$
为了计算 $S_2$ 和 $S_3$，我们先计算 $E_x^* E_y$：
$E_x^* E_y = A^* (A e^{i\pi/4}) = A^2 e^{i\pi/4} = A^2 (\cos(\pi/4) + i \sin(\pi/4)) = A^2 (\frac{\sqrt{2}}{2} + i \frac{\sqrt{2}}{2})$
因此：
$S_2 = 2 \text{Re}(A^2 (\frac{\sqrt{2}}{2} + i \frac{\sqrt{2}}{2})) = 2 \cdot A^2 \frac{\sqrt{2}}{2} = \sqrt{2} A^2$
$S_3 = 2 \text{Im}(A^2 (\frac{\sqrt{2}}{2} + i \frac{\sqrt{2}}{2})) = 2 \cdot A^2 \frac{\sqrt{2}}{2} = \sqrt{2} A^2$
所以，该偏振态的[斯托克斯矢量](@entry_id:168374)为 $(2A^2, 0, \sqrt{2}A^2, \sqrt{2}A^2)$。

### [偏振态](@entry_id:175130)与[斯托克斯矢量](@entry_id:168374)

利用[斯托克斯矢量](@entry_id:168374)，我们可以唯一地表征任何光的偏振态。

#### 完全[偏振光](@entry_id:273160)

对于完全[偏振光](@entry_id:273160)（如[激光](@entry_id:194225)器直接发出的光），[电场](@entry_id:194326)分量的振幅和相位是确定且恒定的。在这种情况下，斯托克斯参数之间存在一个重要的关系。从[电场](@entry_id:194326)定义出发，可以证明：
$S_1^2 + S_2^2 + S_3^2 = (|E_x|^2 - |E_y|^2)^2 + (2 \text{Re}(E_x^* E_y))^2 + (2 \text{Im}(E_x^* E_y))^2$
$= (|E_x|^2 - |E_y|^2)^2 + 4|E_x^* E_y|^2$
$= (|E_x|^4 - 2|E_x|^2|E_y|^2 + |E_y|^4) + 4|E_x|^2|E_y|^2$
$= |E_x|^4 + 2|E_x|^2|E_y|^2 + |E_y|^4$
$= (|E_x|^2 + |E_y|^2)^2 = S_0^2$
因此，对于**完全[偏振光](@entry_id:273160)**，我们有恒等式：
$S_0^2 = S_1^2 + S_2^2 + S_3^2$
这个关系是区分完全[偏振光](@entry_id:273160)与[部分偏振光](@entry_id:267467)的关键 [@problem_id:1606747]。

一些基本[偏振态](@entry_id:175130)的[斯托克斯矢量](@entry_id:168374)（总光强归一化为 $I$）如下：
*   水平线偏振光 ($E_y=0$): $(I, I, 0, 0)$
*   垂直[线偏振光](@entry_id:165445) ($E_x=0$): $(I, -I, 0, 0)$
*   +45° [线偏振光](@entry_id:165445) ($E_x=E_y$, 实数): $(I, 0, I, 0)$
*   右旋圆偏振光 ($E_y = iE_x$): $(I, 0, 0, I)$  (根据我们的约定 $S_3 = 2 \text{Im}(E_x^* i E_x) = 2|E_x|^2$)
*   左旋[圆偏振光](@entry_id:198374) ($E_y = -iE_x$): $(I, 0, 0, -I)$

#### 非偏振光与[部分偏振光](@entry_id:267467)

**[非偏振光](@entry_id:176162)** (unpolarized light)，如来自太阳或白炽灯泡的光，其[电场](@entry_id:194326)矢量的方向在时间上是完全随机和快速变化的。因此，在任何给定的方向上都没有偏振优势。这意味着在测量上，$I_H = I_V$, $I_{45} = I_{135}$, $I_R = I_L$。因此，[非偏振光](@entry_id:176162)的斯托克斯参数为：
$S_1 = S_2 = S_3 = 0$
其[斯托克斯矢量](@entry_id:168374)为 $(S_0, 0, 0, 0)$。我们可以通过一个思想实验来理解这一点：如果[非偏振光](@entry_id:176162)是大量随机取向的[线偏振光](@entry_id:165445)子的时间平均效应，那么对于一个线偏振角为 $\theta$ 的光，其归一化[斯托克斯矢量](@entry_id:168374)为 $(1, \cos(2\theta), \sin(2\theta), 0)$。对所有方向 $\theta \in [0, 2\pi)$ 进行均匀平均，$\cos(2\theta)$ 和 $\sin(2\theta)$ 的平均值都为零，因此最终得到的平均[斯托克斯矢量](@entry_id:168374)就是 $(1, 0, 0, 0)$ [@problem_id:1606732]。

**[部分偏振光](@entry_id:267467)** (partially polarized light) 是最常见的情况，可以看作是完全[偏振光](@entry_id:273160)和非偏振光的非[相干叠加](@entry_id:170209)。对于这样的光，斯托克斯参数满足不等式：
$S_0^2 > S_1^2 + S_2^2 + S_3^2$

任何一束光都可以唯一地分解为一个完全非偏振[部分和](@entry_id:162077)一个完全偏振部分。设非偏振部分的光强为 $I_{\text{unpol}}$，完全偏振部分的光强为 $I_{\text{pol}}$。由于是非相干叠加，总光强为两者之和：
$S_0 = I_{\text{pol}} + I_{\text{unpol}}$
而非零的 $S_1, S_2, S_3$ 参数完全来自于偏振部分。因此，偏振部分的光强满足：
$I_{\text{pol}} = \sqrt{S_1^2 + S_2^2 + S_3^2}$
由此，我们可以从测得的斯托克斯参数中分离出非偏振光的强度 [@problem_id:1606711]:
$I_{\text{unpol}} = S_0 - I_{\text{pol}} = S_0 - \sqrt{S_1^2 + S_2^2 + S_3^2}$
例如，对于一个测得的[斯托克斯矢量](@entry_id:168374)为 $S = (10, 3, 4, 5\sqrt{2})$ 的光束，其[偏振光](@entry_id:273160)强度为 $I_{\text{pol}} = \sqrt{3^2 + 4^2 + (5\sqrt{2})^2} = \sqrt{9+16+50} = \sqrt{75} = 5\sqrt{3}$。而非偏振光强度为 $I_{\text{unpol}} = 10 - 5\sqrt{3}$。这两者强度的比值为 $\frac{I_{\text{pol}}}{I_{\text{unpol}}} = \frac{5\sqrt{3}}{10 - 5\sqrt{3}} = 3+2\sqrt{3}$ [@problem_id:1606731]。

为了量化光的偏振程度，我们定义**[偏振度](@entry_id:276690) (Degree of Polarization, DoP)**, $P$：
$P = \frac{I_{\text{pol}}}{S_0} = \frac{\sqrt{S_1^2 + S_2^2 + S_3^2}}{S_0}$
$P$ 的取值范围为 0 到 1。对于完全[偏振光](@entry_id:273160)，$P=1$；对于完全非偏振光，$P=0$；对于[部分偏振光](@entry_id:267467)，$0  P  1$。例如，如果一束强度为 $I_c$ 的左旋圆偏振光和一束强度为 $I_u$ 的非偏振光非相干地混合，得到的总[斯托克斯矢量](@entry_id:168374)为 $(I_u+I_c, 0, 0, -I_c)$。其[偏振度](@entry_id:276690)为 $P = \frac{\sqrt{0^2+0^2+(-I_c)^2}}{I_u+I_c} = \frac{I_c}{I_u+I_c}$ [@problem_id:1606747]。

### [庞加莱球](@entry_id:265729)：偏振的几何表示

斯托克斯参数的强大之处不仅在于其代数完备性，还在于它引出了一种优美的几何表示方法——**[庞加莱球](@entry_id:265729)**。

#### 构造与坐标

为了将偏振态可视化，我们首先定义**归一化的斯托克斯参数**:
$s_1 = S_1/S_0, \quad s_2 = S_2/S_0, \quad s_3 = S_3/S_0$
然后，我们将这三个参数 $(s_1, s_2, s_3)$ 视为三维笛卡尔空间中的一个点的坐标。

对于完全[偏振光](@entry_id:273160)，我们有 $S_0^2 = S_1^2 + S_2^2 + S_3^2$，两边同除以 $S_0^2$ 得到：
$s_1^2 + s_2^2 + s_3^2 = 1$
这正是一个单位球面的方程。这个[单位球](@entry_id:142558)面就是**[庞加莱球](@entry_id:265729)**。因此，**所有完全偏振态都唯一地对应于[庞加莱球](@entry_id:265729)球面上的一个点**。

对于[部分偏振光](@entry_id:267467)，其[偏振度](@entry_id:276690) $P = \sqrt{s_1^2 + s_2^2 + s_3^2} \le 1$。这意味着**所有[部分偏振](@entry_id:187644)态都对应于[庞加莱球](@entry_id:265729)内部的一个点**，该点到原点的距离等于其[偏振度](@entry_id:276690) $P$。球心（原点）代表[偏振度](@entry_id:276690)为零的**完全[非偏振光](@entry_id:176162)**。

#### [偏振态](@entry_id:175130)的映射

[庞加莱球](@entry_id:265729)上的特定点和区域与特定的偏振类型相对应，提供了一种直观的“偏振地图” [@problem_id:1606740]。

*   **两极 (Poles):**
    *   **北极** $(0, 0, 1)$ 对应于 $s_3=1$，即 $S_3=S_0$。这意味着 $I_R=S_0$ 且 $I_L=0$，代表**纯[右旋圆偏振](@entry_id:267955)光 (RHC)**。
    *   **南极** $(0, 0, -1)$ 对应于 $s_3=-1$，即 $S_3=-S_0$。这意味着 $I_L=S_0$ 且 $I_R=0$，代表**纯左旋[圆偏振光](@entry_id:198374) (LHC)**。

*   **赤道 (Equator):**
    *   赤道上的所有点都满足 $s_3=0$，这意味着它们代表**纯[线偏振光](@entry_id:165445)**。
    *   点 $(1, 0, 0)$ 对应于 $s_1=1$，代表**水平[线偏振光](@entry_id:165445)**。
    *   点 $(-1, 0, 0)$ 对应于 $s_1=-1$，代表**垂直线偏振光**。
    *   点 $(0, 1, 0)$ 对应于 $s_2=1$，代表**+45° [线偏振光](@entry_id:165445)**。
    *   点 $(0, -1, 0)$ 对应于 $s_2=-1$，代表**-45° 线偏振光**。

更一般地，一个与水平方向成 $\alpha$ 角的线偏振光，其归一化斯托克斯参数为 $(\cos(2\alpha), \sin(2\alpha), 0)$。这对应于[庞加莱球](@entry_id:265729)赤道上一个[方位角](@entry_id:164011)（经度）为 $2\psi = 2\alpha$ 的点 [@problem_id:1606743]。这个角度加倍的效应是[庞加莱球](@entry_id:265729)表示法的一个关键特征：物理空间中 $180^\circ$ 的旋转（例如从水平到垂直）对应于[庞加莱球](@entry_id:265729)上 $360^\circ$ 的经度变化，即绕球一周。

球的其他区域（非两极、非赤道）代表各种**[椭圆偏振光](@entry_id:195140)**。点的纬度 $2\chi = \arcsin(s_3)$ 描述了椭圆的[椭率](@entry_id:199972)，而经度 $2\psi = \arctan(s_2/s_1)$ 描述了椭圆长轴的取向。

#### 几何解释与运算

[庞加莱球](@entry_id:265729)不仅是一个分类工具，它还揭示了不同[偏振态](@entry_id:175130)之间深刻的几何关系。

*   **正交态 (Orthogonal States):** 在琼斯微积分中，两个[偏振态](@entry_id:175130) $\vec{E}_A$ 和 $\vec{E}_B$ 如果满足 $\vec{E}_A^\dagger \vec{E}_B = 0$，则称它们是正交的。在[庞加莱球](@entry_id:265729)上，这个代数关系有非常简洁的几何对应：**两个相互正交的偏振态位于[庞加莱球](@entry_id:265729)上相互对跖(antipodal)的两个点** [@problem_id:1606682]。也就是说，如果状态 A 的归一化[斯托克斯矢量](@entry_id:168374)为 $\vec{s}_A$，那么其正交态 B 的矢量为 $\vec{s}_B = -\vec{s}_A$。例如，水平偏振 $(1,0,0)$ 与垂直偏振 $(-1,0,0)$ 正交；[右旋圆偏振](@entry_id:267955) $(0,0,1)$ 与左旋[圆偏振](@entry_id:261702) $(0,0,-1)$ 正交。

*   **非相干叠加 (Incoherent Superposition):** 当两束光非相干地混合时，其总的[斯托克斯矢量](@entry_id:168374)是各分量[斯托克斯矢量](@entry_id:168374)的和。设光束 A 和 B 的强度分别为 $I_A$ 和 $I_B$，其在庞加莱空间中的点分别为 $P_A$ (矢量 $\vec{s}_A$) 和 $P_B$ (矢量 $\vec{s}_B$) 。混合后光束的点 $P_{mix}$ 的位置矢量为：
    $\vec{s}_{mix} = \frac{I_A \vec{s}_A + I_B \vec{s}_B}{I_A + I_B}$
    这是一个加权平均。从几何上看，**[混合态](@entry_id:141568) $P_{mix}$ 位于连接 $P_A$ 和 $P_B$ 的线段上** [@problem_id:1606684]。$P_{mix}$ 在该线段上的具体位置由强度比决定，满足杠杆原理：$P_{mix}$ 到 $P_A$ 的距离与到 $P_B$ 的距离之比等于强度之反比，即 $||\vec{s}_{mix} - \vec{s}_A|| / ||\vec{s}_{mix} - \vec{s}_B|| = I_B / I_A$。这个原理为分析和预测非[相干光](@entry_id:170661)学过程（如散射和[热辐射](@entry_id:145102)）提供了极大的便利。

综上所述，斯托克斯参数和[庞加莱球](@entry_id:265729)共同构成了一个描述和分析光[偏振态](@entry_id:175130)的强大框架。它不仅将抽象的偏振概念与可测量的物理量联系起来，还通过几何直观性揭示了不同偏振态之间的深刻联系，在天文学、[材料科学](@entry_id:152226)、[光通信](@entry_id:200237)等领域发挥着至关重要的作用。
