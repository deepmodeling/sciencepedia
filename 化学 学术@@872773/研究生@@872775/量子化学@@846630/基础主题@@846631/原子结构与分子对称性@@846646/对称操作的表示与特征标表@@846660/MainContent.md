## 引言
分子对称性是现代化学中一个基础而深刻的概念，它不仅决定了分子的美学形态，更深刻地支配着其物理和化学性质。然而，仅仅直观地识别分子的[对称元素](@entry_id:136566)（如[旋转轴](@entry_id:187094)和镜面）是不够的；为了在量子力学的框架内充分利用对称性，我们必须解决一个核心问题：如何将这些几何概念转化为一套严谨的代数工具，从而简化薛定谔方程的求解并揭示其解的内在结构。[群的表示](@entry_id:140711)理论正是弥合这一差距的强大数学桥梁。

本文旨在系统地阐述分子对称性群的表示理论及其在[量子化学](@entry_id:140193)中的应用。通过学习本文，读者将能够掌握如何将抽象的[对称操作](@entry_id:143398)转化为具体的矩阵表示和特征标，并利用这些工具来解决实际的化学问题。文章分为三个核心部分：

第一章，“原理与机制”，将从第一性原理出发，建立[对称操作](@entry_id:143398)的[矩阵表示](@entry_id:146025)，严格定义可约与不可约表示、特征标等核心概念。我们将重点介绍特征标大[正交定理](@entry_id:141650)，并展示如何利用它来系统地构建和解读特征标表——这是化学家应用群论的“密码本”。本章还将理论框架扩展至包含电子自旋的[双值群](@entry_id:187359)。

第二章，“应用与[交叉](@entry_id:147634)学科联系”，将理论付诸实践。我们将探讨群论如何在[分子光谱学](@entry_id:148164)中预测红外、拉曼和[电子跃迁的选择定则](@entry_id:192423)，以及如何通过构建对称匹配线性组合（SALCs）来系统地构造[分子轨道能级](@entry_id:197804)图，从而深化对化学键本质的理解。

第三章，“动手实践”，通过一系列精心设计的问题，引导读者亲手应用所学知识，从推导矢量变换到预测[光谱](@entry_id:185632)活性，巩固对理论的掌握并增[强解](@entry_id:198344)决问题的能力。

现在，让我们从[对称操作](@entry_id:143398)的数学描述开始，步入[群表示](@entry_id:156757)理论的严谨世界。

## 原理与机制

在上一章引言的基础上，本章深入探讨[分子对称性](@entry_id:202199)[群表示](@entry_id:156757)理论的核心原理与机制。我们将从对称操作的具体数学描述出发，建立[群表示](@entry_id:156757)的严谨框架，并阐述如何利用表示的特征（即“特征标”）来揭示分子的内在对称性。最终，我们将展示如何系统地构建和运用[特征标表](@entry_id:146676)这一强大工具，并将其推广至包含电子自旋的复杂情形。

### [对称操作](@entry_id:143398)的[线性变换](@entry_id:149133)表示

分子[对称操作](@entry_id:143398)，如旋转和反映，本质上是三维欧几里得空间中的几何变换。这些变换保持分子中所有原子间的距离不变（即等距变换），并使分子在变换后看起来与变换前无法区分。为了在[量子化学](@entry_id:140193)中运用这些对称性，我们需要一种方法将这些几何概念转化为代数语言。这通过将[对称操作](@entry_id:143398)表示为作用于某个[向量空间](@entry_id:151108)上的线性算符（或矩阵）来实现。

最直观的[向量空间](@entry_id:151108)是由描述分子位置的笛卡尔坐标 $(x, y, z)$ 构成的。任何一个对称操作 $g$ 都会将空间中的任意点 $\mathbf{r}=(x,y,z)^{\mathsf{T}}$ 映射到一个新的点 $\mathbf{r}'=(x',y',z')^{\mathsf{T}}$。由于[对称操作](@entry_id:143398)是[线性变换](@entry_id:149133)，这种映射关系可以通过一个 $3 \times 3$ 的矩阵 $\mathbf{D}(g)$ 来表示：

$$
\mathbf{r}' = \mathbf{D}(g)\mathbf{r}
$$

矩阵 $\mathbf{D}(g)$ 构成了对对称操作 $g$ 的一个**三维[矩阵表示](@entry_id:146025) (three-dimensional matrix representation)**。一个完整的对称性群 $G = \{g_1, g_2, \dots, h\}$ 对应着一个矩阵集合 $\{\mathbf{D}(g_1), \mathbf{D}(g_2), \dots, \mathbf{D}(g_h)\}$，该集合同样构成一个群，其[乘法规则](@entry_id:197368)与原[对称操作](@entry_id:143398)群的[乘法规则](@entry_id:197368)一一对应，即 $\mathbf{D}(g_i)\mathbf{D}(g_j) = \mathbf{D}(g_i g_j)$。

让我们通过构建一些基本对称操作的矩阵表示来具体理解这一概念。[@problem_id:2920953]

- **恒等操作 ($E$)**: 该操作使所有点保持不变，$\mathbf{r}'=\mathbf{r}$。其[矩阵表示](@entry_id:146025)是单位矩阵 $\mathbf{I}$。
- **[绕轴旋转](@entry_id:185161) ($C_n$)**: 绕通过原点的轴 $\hat{\mathbf{u}}$ 旋转角度 $\theta = 2\pi/n$。只有位于旋转轴上的点在操作下保持不变，该旋转轴就是该操作的**[对称元素](@entry_id:136566) (symmetry element)**。
- **[镜面](@entry_id:148117)反映 ($\sigma$)**: 通过一个包含原点的[镜面](@entry_id:148117)进行反映。所有位于[镜面](@entry_id:148117)上的点都保持不变，该[镜面](@entry_id:148117)即为[对称元素](@entry_id:136566)。
- **反演 ($i$)**: 通过原点进行反演，$\mathbf{r}'=-\mathbf{r}$。只有原点 $\mathbf{0}$ 在此操作下保持不变，原点是反演中心。
- **瑕转 ($S_n$)**: 先绕轴 $\hat{\mathbf{u}}$ 旋转 $2\pi/n$，再通过一个垂直于该轴的平面进行反映。对于 $n \ge 2$ 的瑕转操作，只有原点保持不变。

**示例：$C_{2v}$ [点群](@entry_id:142456)的[矩阵表示](@entry_id:146025)**

以水分子所属的 $C_{2v}$ 点群为例，它包含四个[对称操作](@entry_id:143398)：$E$, $C_2$, $\sigma_v(xz)$, 和 $\sigma_v'(yz)$。我们约定[主轴](@entry_id:172691) $C_2$ 为 $z$ 轴。我们可以通过观察这些操作如何变换笛卡尔坐标 $(x,y,z)$ 来构建它们的 $3 \times 3$ 矩阵表示。[@problem_id:2920989]

- $E: (x, y, z) \mapsto (x, y, z)$, 对应的矩阵为 $\mathbf{D}(E) = \begin{pmatrix} 1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix}$。
- $C_2(z)$: 绕 $z$ 轴旋转 $180^\circ$, $(x, y, z) \mapsto (-x, -y, z)$。对应的矩阵为 $\mathbf{D}(C_2) = \begin{pmatrix} -1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix}$。
- $\sigma_v(xz)$: 通过 $xz$ 平面反映, $(x, y, z) \mapsto (x, -y, z)$。对应的矩阵为 $\mathbf{D}(\sigma_v(xz)) = \begin{pmatrix} 1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix}$。
- $\sigma_v'(yz)$: 通过 $yz$ 平面反映, $(x, y, z) \mapsto (-x, y, z)$。对应的矩阵为 $\mathbf{D}(\sigma_v'(yz)) = \begin{pmatrix} -1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix}$。

这些矩阵不仅描述了单个操作，还保持了群的乘法结构。例如，在 $C_{2v}$ 群中，$C_2 \circ \sigma_v(xz) = \sigma_v'(yz)$ (操作顺序为先右后左)。对应的矩阵乘法也应满足此关系：
$$
\mathbf{D}(C_2)\mathbf{D}(\sigma_v(xz)) = \begin{pmatrix} -1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix} \begin{pmatrix} 1  0  0 \\ 0  -1  0 \\ 0  0  1 \end{pmatrix} = \begin{pmatrix} -1  0  0 \\ 0  1  0 \\ 0  0  1 \end{pmatrix} = \mathbf{D}(\sigma_v'(yz))
$$
这验证了我们构建的矩阵集合确实是该群的一个有效表示。

### [群表示](@entry_id:156757)与特征标的严格定义

上述例子引出了[群表示](@entry_id:156757)的普适定义。一个群 $G$ 在[向量空间](@entry_id:151108) $V$ 上的一个**表示 (representation)** $\Gamma$ 是一个[群同态](@entry_id:140603)映射 $\Gamma: G \to \mathrm{GL}(V)$，其中 $\mathrm{GL}(V)$ 是 $V$ 上所有可逆线性算符构成的群。如果 $V$ 是一个 $n$ 维[复向量空间](@entry_id:264355)，那么 $\mathrm{GL}(V)$ 同构于 $n \times n$ 可逆[复矩阵](@entry_id:190650)群 $\mathrm{GL}(n, \mathbb{C})$。这意味着每个群元 $g \in G$ 都对应一个矩阵 $\mathbf{D}(g)$，并且矩阵乘法遵循群的运算规则。[@problem_id:2920994]

在实际应用中，处理完整的[矩阵表示](@entry_id:146025)可能非常繁琐。幸运的是，我们可以通过一个更简单的量来捕捉表示的关键信息，这个量就是**特征标 (character)**。表示 $\Gamma$ 中元素 $g$ 的特征标 $\chi^{\Gamma}(g)$ 定义为其表示矩阵 $\mathbf{D}(g)$ 的迹 (trace)：

$$
\chi^{\Gamma}(g) = \mathrm{Tr}[\mathbf{D}(g)] = \sum_i D_{ii}(g)
$$

特征标是一个标量，但它具有两个至关重要的性质：
1.  **相似变换下的不变性**：如果两个表示 $\Gamma_1$ 和 $\Gamma_2$ 是**等价的 (equivalent)**，即它们可以通过一个固定的相似变换矩阵 $S$ 相互转换 ($\mathbf{D}_2(g) = S \mathbf{D}_1(g) S^{-1}$ 对所有 $g \in G$ 成立)，那么它们的特征标完全相同。这是因为迹具有循环不变性：$\mathrm{Tr}(S \mathbf{D}_1 S^{-1}) = \mathrm{Tr}(\mathbf{D}_1 S^{-1} S) = \mathrm{Tr}(\mathbf{D}_1)$。[@problem_id:2920994]
2.  **类函数 (Class function)**：同一**共轭类 (conjugacy class)** 中的所有元素在任何表示中都具有相同的特征标。一个[共轭类](@entry_id:143916)是群中形如 $\{hgh^{-1} | h \in G\}$ 的元素集合。这使得我们只需为每个共轭类而不是每个单独的群元计算特征标，极大地简化了问题。[@problem_id:2920967]

以 $C_{3v}$ 群为例，我们可以构建其在 $(x,y,z)$ 笛卡尔基上的三维表示矩阵，并计算其特征标。该群有三个[共轭类](@entry_id:143916)：$\{E\}$，$\{C_3, C_3^2\}$，和 $\{\sigma_v^{(1)}, \sigma_v^{(2)}, \sigma_v^{(3)}\}$。计算表明，$\chi(E)=3$, $\chi(C_3)=\chi(C_3^2)=0$, $\chi(\sigma_v)=1$。这组特征标 $(3, 0, 1)$ 简洁地概括了这个表示。[@problem_id:2920972]

除了基于[坐标变换](@entry_id:172727)的表示，还有其他类型的表示。例如，考虑一个由分子中某些等价原子（如甲烷中的四个氢原子）上的定域函数（如[原子轨道](@entry_id:140819)）构成的[基组](@entry_id:160309)。[对称操作](@entry_id:143398)会使这些原子相互交换位置，从而导致[基函数](@entry_id:170178)的[置换](@entry_id:136432)。这种表示被称为**[置换表示](@entry_id:142960) (permutation representation)**。对于[置换表示](@entry_id:142960)，计算特征标有一个简便的法则：操作 $g$ 的特征标等于在该操作下保持位置不变的[基函数](@entry_id:170178)的数目。

**示例：甲烷 ($T_d$) 中的[置换表示](@entry_id:142960)**

甲烷分子中的四个氢原子构成一个正四面体，其对称性属于 $T_d$ 点群。以这四个氢原子为中心的[基函数](@entry_id:170178) $\{\chi_1, \chi_2, \chi_3, \chi_4\}$ 构成一个四维[置换表示](@entry_id:142960) $\Gamma_{\mathrm{perm}}$。我们可以通过计算每个对称操作下保持不变的氢原子数来得到该[表示的特征标](@entry_id:198072)。[@problem_id:2920943]

- $E$: 所有 4 个氢原子都不动。$\chi(E) = 4$。
- $C_3$: [旋转轴](@entry_id:187094)穿过一个 C-H 键，只有该轴上的 1 个氢原子不动。$\chi(C_3) = 1$。
- $C_2$: [旋转轴](@entry_id:187094)穿过对边中点，不经过任何氢原子。$\chi(C_2) = 0$。
- $S_4$: 瑕[转轴](@entry_id:187094)与 $C_2$ 轴重合，同样不经过任何氢原子。$\chi(S_4) = 0$。
- $\sigma_d$: 反映面包含 C 原子和 2 个氢原子，这 2 个氢原子不动。$\chi(\sigma_d) = 2$。

因此，$\Gamma_{\mathrm{perm}}$ 的特征标向量为 $(4, 1, 0, 0, 2)$。

### [可约表示与不可约表示](@entry_id:197263)

一个表示被称为**可约的 (reducible)**，如果存在一个非平凡的[子空间](@entry_id:150286) $W$（即非 $\{0\}$ 且非整个[向量空间](@entry_id:151108) $V$），它在群中所有对称操作下都是**不变的 (invariant)**，即对于所有 $g \in G$ 和所有向量 $w \in W$，都有 $\mathbf{D}(g)w \in W$。反之，如果不存在这样的不变子空间，则该表示是**不可约的 (irreducible)**，简称**irrep**。[@problem_id:2920968]

从物理上看，一个[可约表示](@entry_id:137110)意味着其[基函数](@entry_id:170178)可以被划分为若干个更小的、相互独立的集合，这些集合在任何对称操作下都只在内部混合，而不会相互转化。这意味着我们可以找到一个合适的[基变换](@entry_id:189626)（即相似变换），使得表示中的所有矩阵都同时呈现出相同的[块对角结构](@entry_id:746869)。

$$
S^{-1}\mathbf{D}(g)S = \begin{pmatrix} \mathbf{D}^{(1)}(g)  0 \\ 0  \mathbf{D}^{(2)}(g) \end{pmatrix}
$$

**[马施克定理](@entry_id:142097) (Maschke's Theorem)** 保证，对于有限群，任何[可约表示](@entry_id:137110)都可以被**完全约化 (completely reduced)** 为一系列[不可约表示](@entry_id:263310)的直和：

$$
\Gamma = \bigoplus_i a_i \Gamma^{(i)} = a_1 \Gamma^{(1)} \oplus a_2 \Gamma^{(2)} \oplus \dots
$$

其中 $\Gamma^{(i)}$ 是第 $i$ 个[不可约表示](@entry_id:263310)，整数 $a_i$ 是它在 $\Gamma$ 中出现的次数或**多重性 (multiplicity)**。[@problem_id:2920994]

[不可约表示](@entry_id:263310)是[群表示](@entry_id:156757)理论的“原子”，任何表示都可以由它们搭建而成。因此，识别一个表示是否可约以及如何将其分解为不可约表示是核心任务。以下是几个等价的判据：

1.  **[舒尔引理](@entry_id:136779) (Schur's Lemma)**：一个表示 $\Gamma$ 是不可约的，当且仅当唯一与所有表示矩阵 $\mathbf{D}(g)$ 对易的矩阵是单位矩阵的标量倍（即形如 $\lambda I$ 的矩阵）。如果存在任何非标量的对易矩阵，则表示是可约的。[@problem_id:2920968]
2.  **特征标判据**：利用[特征标的正交性](@entry_id:140971)（详见下节），可以导出一个极其有用的判据。一个表示 $\Gamma$ 是不可约的，当且仅当其特征标的“归一化”平方和为 1：
    $$
    \frac{1}{|G|} \sum_{g \in G} |\chi^{\Gamma}(g)|^2 = 1
    $$
    其中 $|G|$ 是群的阶（群中元素的总数）。如果这个和大于 1，则表示是可约的。这个和的值实际上等于 $\sum_i a_i^2$。[@problem_id:2920968]

### 特征标[正交定理](@entry_id:141650)与[特征标表](@entry_id:146676)

[群表示](@entry_id:156757)理论中最强大的工具之一是**特征标大[正交定理](@entry_id:141650) (Great Orthogonality Theorem)**。这个定理本身相当复杂，但它有几个直接应用于特征标的重要推论，这些推论是构建和使用特征标表的基础。

对于一个给定的群 $G$：
1.  不可约表示的数目等于群中[共轭类](@entry_id:143916)的数目。
2.  所有不可约表示的维数 $l_i$ 的平方和等于群的阶 $|G|$：$\sum_i l_i^2 = |G|$。
3.  **行正交性**：将不同不可约[表示的特征标](@entry_id:198072)向量视为[正交向量](@entry_id:142226)。权重为相应共轭类的大小 $n_k$：
    $$
    \sum_k n_k \chi^{(i)}(C_k)^* \chi^{(j)}(C_k) = |G| \delta_{ij}
    $$
    其中 $\chi^{(i)}(C_k)$ 是第 $i$ 个 irrep 在第 $k$ 个类 $C_k$ 上的特征标。
4.  **列正交性**：对特征标表中的任意两列（即两个共轭类）进行求和也满足[正交关系](@entry_id:145540)：
    $$
    \sum_i \chi^{(i)}(C_k)^* \chi^{(i)}(C_j) = \frac{|G|}{n_k} \delta_{kj}
    $$

利用这些规则，我们可以从零开始系统地推导出任何有限群的特征标表。[@problem_id:2920928]

**特征标表 (Character Table)** 是一个总结了群的所有[不可约表示](@entry_id:263310)特征标的表格，是[量子化学](@entry_id:140193)家不可或缺的工具。其标准布局如下：[@problem_id:2920967]

-   **表头**：最左上角是群的名称（例如 $C_{3v}$）。顶行是群的共轭类，通常用一个代表元素表示（如 $C_3$），并附上该类的阶数（如 $2C_3$）。恒等操作 $E$ 通常是第一列。
-   **第一列**：列出了所有[不可约表示](@entry_id:263310)的**穆利肯符号 (Mulliken symbols)**，如 $A_1, B_{2g}, E', T_{2u}$ 等。这些符号本身编码了表示的对称性信息：
    -   $A, B$ 表示[一维表示](@entry_id:136509)；$E$ 表示二维；$T$ (或 $F$) 表示三维。
    -   下标 $g$ (gerade) 和 $u$ (ungerade) 分别表示对反演操作的对称（特征标为+1）和反对称（特征标为-1）。
    -   上标 `'` 和 `''` 分别表示对水平反映面 $\sigma_h$ 的对称和反对称。
    -   下标 $1, 2$ 表示对次要对称操作（如垂直于主轴的 $C_2$ 轴或 $\sigma_v$ 面）的对称或反对称行为。
-   **表格主体**：表格的数值部分是各个不可约表示在各个[共轭类](@entry_id:143916)上的特征标。第一列（$E$ 列下）的特征标等于该不可约表示的维数。
-   **右侧区域**：最后几列显示了[笛卡尔坐标](@entry_id:167698) $(x, y, z)$、[绕轴转动](@entry_id:185161) $(R_x, R_y, R_z)$ 以及它们的二次组合（如 $x^2+y^2, z^2, xy$ 等）如何变换。如果一个函数（或一组函数）在某个[不可约表示](@entry_id:263310)的行中列出，意味着它（们）按该不可约表示进行变换。例如，这可以帮助我们确定[分子振动](@entry_id:140827)模式或分子[轨道](@entry_id:137151)的对称性。

**应用：约化[可约表示](@entry_id:137110)**

特征标的行正交性提供了一个直接的“投影”方法来分解[可约表示](@entry_id:137110)，这被称为**约化公式 (reduction formula)**。[可约表示](@entry_id:137110) $\Gamma$ 中不可约表示 $\Gamma^{(i)}$ 的[多重性](@entry_id:136466) $a_i$ 由下式给出：

$$
a_i = \frac{1}{|G|} \sum_k n_k \chi^{(i)}(C_k)^* \chi^{\Gamma}(C_k)
$$

这本质上是计算特征标向量 $\chi^{\Gamma}$ 在[正交基](@entry_id:264024)向量 $\chi^{(i)}$ 上的分量。

**示例：$D_{3h}$ 群的表示约化**

考虑一个 $D_{3h}$ 点群的分子，其某个七维[可约表示](@entry_id:137110) $\Gamma$ 的特征标为 $\chi^{\Gamma} = (7, 1, -1, -1, -1, 3)$。利用 $D_{3h}$ 的标准[特征标表](@entry_id:146676)和约化公式，我们可以计算出每个 irrep 的[多重性](@entry_id:136466) $a_i$。例如，对于全对称表示 $A_1'$（其特征标为 $(1, 1, 1, 1, 1, 1)$），其[多重性](@entry_id:136466)为：

$$
a_{A_1'} = \frac{1}{12}[1(7)(1) + 2(1)(1) + 3(-1)(1) + 1(-1)(1) + 2(-1)(1) + 3(3)(1)] = \frac{12}{12} = 1
$$

对所有 irreps 重复此过程，我们可以得到该表示的完整分解，例如 $\Gamma = 1A_{1}' \oplus 0A_{2}' \oplus 1E' \oplus 0A_{1}'' \oplus 2A_{2}'' \oplus 1E''$。[@problem_id:2920997]

### [双值群](@entry_id:187359)与[旋量表示](@entry_id:141362)

上述理论框架完美地适用于描述空间函数（如分子[轨道](@entry_id:137151)和[振动](@entry_id:267781)模式）的对称性，这些函数是空间坐标的单值函数。然而，当考虑电子的**自旋 (spin)** 时，情况变得复杂。自旋是半整数量子数 ($s=1/2$)，描述自旋的[波函数](@entry_id:147440)——**旋量 (spinor)**——具有独特的变换性质。

一个关键事实是，当一个自旋为 $1/2$ 的系统绕任意轴旋转 $2\pi$ ($360^\circ$) 时，其[波函数](@entry_id:147440)不会变回自身，而是乘以一个 $-1$ 的相位因子。旋转 $4\pi$ ($720^\circ$) 才能使其恢复原状。这可以通过[自旋角动量](@entry_id:149719)算符的指数形式来证明。自旋为 $1/2$ 的[旋转算符](@entry_id:136702)可以写为 $\hat{U}(\hat{\mathbf{n}}, \theta) = \exp(-\frac{i\theta}{2} \hat{\mathbf{n}} \cdot \boldsymbol{\sigma})$，其中 $\boldsymbol{\sigma}$ 是泡利矩阵。当 $\theta = 2\pi$ 时，该算符变为 $-I$（负[单位矩阵](@entry_id:156724)）。[@problem_id:2920955]

这种行为意味着[旋量](@entry_id:158054)构成了[旋转群](@entry_id:204412) $SO(3)$ 的**双值表示 (double-valued representation)**，因为空间中的一个操作（旋转 $2\pi$ 等同于不动）对应于表示中的两个不同算符 ($+I$ 和 $-I$)。为了处理这种情况，我们需要扩展点群的[代数结构](@entry_id:137052)。

这个扩展是通过引入一个与旋转 $2\pi$ 相关的新元素，通常记为 $\bar{E}$ 或 $R$，来构建**[双值群](@entry_id:187359) (double group)** $G^D$。该群的阶是原[点群](@entry_id:142456) $G$ 的两倍，即 $|G^D| = 2|G|$。$\bar{E}$ 与所有其他操作对易，并满足 $\bar{E}^2 = E$。[@problem_id:2920984]

[双值群](@entry_id:187359)的不可约表示分为两类：
1.  **单值表示 (Single-valued representations)**：这些是原[点群](@entry_id:142456) $G$ 的普通[不可约表示](@entry_id:263310)。在这些表示中，新元素 $\bar{E}$ 被表示为[单位矩阵](@entry_id:156724)，即 $\chi(\bar{E}) = \chi(E)$。
2.  **双值表示 (Double-valued representations)**：这些是新的、专属于[双值群](@entry_id:187359)的[不可约表示](@entry_id:263310)，也称为**[旋量表示](@entry_id:141362) (spinor representations)**。在这些表示中，$\bar{E}$ 被表示为负[单位矩阵](@entry_id:156724)，因此其特征标为维数的负值：$\chi(\bar{E}) = -\chi(E) = -l_i$。

例如，对于八面体群 $O_h$（阶为48），其[双值群](@entry_id:187359) $O_h^D$ 的阶为96。除了 $O_h$ 原有的10个单值 irreps (维数为1, 2, 3) 之外，$O_h^D$ 还引入了6个新的双值 irreps (维数为2, 4)，它们专门用于描述[半整数自旋](@entry_id:148826)系统的对称性。[@problem_id:2920984] 在处理包含重原子（其中[自旋-轨道耦合](@entry_id:143520)效应显著）的分子时，使用[双值群](@entry_id:187359)及其特征标表是必不可少的。
