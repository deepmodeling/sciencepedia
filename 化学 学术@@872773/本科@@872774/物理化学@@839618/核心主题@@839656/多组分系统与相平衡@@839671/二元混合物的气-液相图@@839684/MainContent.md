## 引言
在化学与工程领域，[纯净物](@entry_id:140474)质极为罕见，我们面对的绝大多数是混合物。因此，理解混合物的[物理化学](@entry_id:145220)行为，尤其是如何将其分离成纯组分，是科学研究和工业生产中的一项核心任务。其中，液体混合物的[气液平衡](@entry_id:147234)（Vapor-Liquid Equilibrium, VLE）是蒸馏、蒸发等关键分离技术背后的根本原理。然而，混合物在气液两相间的分配规律远比纯物质复杂，常常展现出非直观的行为。

本文旨在系统性地解决这一问题，为读者提供一个关于[二元混合物](@entry_id:168452)气液相图的完整知识框架。我们将通过三个章节的递进式学习，引领您从基础理论走向前沿应用。首先，在“原理与机制”一章中，我们将奠定[热力学](@entry_id:141121)基础，学习[吉布斯相律](@entry_id:145932)、[拉乌尔定律](@entry_id:139880)，并掌握如何构建和解读理想与非[理想混合物](@entry_id:180997)的[相图](@entry_id:144015)，理解共沸等复杂现象。接着，在“应用与跨学科联系”一章，我们将展示这些理论如何在[化学工程](@entry_id:143883)的[蒸馏](@entry_id:140660)设计、[环境科学](@entry_id:187998)的[海水淡化](@entry_id:268140)以及[生物物理化学](@entry_id:150393)的[超速离心](@entry_id:167138)等领域发挥关键作用。最后，通过“动手实践”部分，您将有机会通过解决具体问题来巩固和检验所学知识。学完本文，您将能够自信地分析和预测[二元混合物](@entry_id:168452)的[相行为](@entry_id:199883)，并理解其在现代科技中的重要地位。

## 原理与机制

本章在前一章介绍气液相平衡基本概念的基础上，深入探讨描述二元液体混合物[气液平衡](@entry_id:147234)行为的根本[热力学原理](@entry_id:142232)与核心机制。我们将从[约束系统](@entry_id:164587)状态的[吉布斯相律](@entry_id:145932)出发，构建[理想混合物](@entry_id:180997)的物理模型，并学习如何利用[相图](@entry_id:144015)这一强大工具来直观地表示和分析平衡状态。最后，我们将扩展到真实世界的非[理想混合物](@entry_id:180997)，并阐释活度、[拉乌尔定律偏差](@entry_id:165326)以及共沸现象等关键概念。

### [吉布斯相律](@entry_id:145932)：自由度的[热力学约束](@entry_id:755911)

在研究多相、多组分系统的平衡状态时，我们首先需要了解系统有多少个变量可以独立改变，而系统仍能维持其平衡状态。这个问题的答案由 **[吉布斯相律](@entry_id:145932) (Gibbs phase rule)** 给出，它为复杂系统的[热力学分析](@entry_id:142723)提供了坚实的理论框架。其数学表达式为：

$F = C - P + 2$

在此方程中，$F$ 是系统的 **自由度 (degrees of freedom)** 或称 **[方差](@entry_id:200758) (variance)**，代表在不改变相数的情况下，可以独立变化的 **强度变量 (intensive variables)**（如温度、压力、组成）的数量。$C$ 是系统的独立 **组分 (components)** 数量，即确定系统中所有相的组成所需的最少化学物质种类数。$P$ 则是系统在平衡时共存的 **相 (phases)** 的数量。常数“2”代表两个典型的强度变量——温度和压力。

让我们应用相律来分析一个由两种可完全互溶的挥发性组分A和B构成的二元体系 [@problem_id:1990577]。

1.  **二元[气液平衡](@entry_id:147234)系统：** 当液相与气[相共存](@entry_id:147284)时，系统有两个组分 ($C=2$) 和两个相 ($P=2$)。根据相律，自由度为：
    $F_1 = 2 - 2 + 2 = 2$
    这意味着，为了完全确定一个二元[气液平衡](@entry_id:147234)系统的状态，我们需要指定两个独立的强度变量。例如，如果我们固定了温度和压力，那么平衡时液相和气相的组成就唯一确定了。同样，如果固定温度和液[相组成](@entry_id:197559)，则平衡时的压力和气[相组成](@entry_id:197559)也随之确定。

2.  **二元共沸系统：** **[共沸物](@entry_id:146150) (azeotrope)** 是一种特殊状态，此时气相和液相的组成完全相同 ($x_A = y_A$)。这个额外的约束条件减少了一个自由度。因此，对于一个处于共沸点的二元体系 ($C=2, P=2$)，自由度为：
    $F_2 = C - P + 2 - (\text{约束数}) = 2 - 2 + 2 - 1 = 1$
    这表明，对于一个给定的共沸物，我们只需指定一个变量（例如压力），其共沸温度和共沸组成就会被唯一确定。

3.  **三[相平衡](@entry_id:136822)点：** 作为对比，考虑一个更受限的系统，例如一个[二元混合物](@entry_id:168452)在某个特定条件下形成了两个不互溶的液相（一个富A相，一个富[B相](@entry_id:200534)）并与一个气相平衡。此时，$C=2, P=3$，自由度为：
    $F_4 = 2 - 3 + 2 = 1$
    若考虑纯物质（$C=1$）的三相点（固、液、气三[相共存](@entry_id:147284)），则 $P=3$，自由度为：
    $F_3 = 1 - 3 + 2 = 0$
    自由度为零意味着纯物质的[三相点](@entry_id:142815)是一个不变点，其温度和压力是该物质固有的、唯一确定的[物理常数](@entry_id:274598)，我们无法改变其中任何一个而不破坏三相平衡。

[吉布斯相律](@entry_id:145932)的这些应用揭示了相图结构背后的基本约束，并为我们后续理解相图的各个区域和边界提供了理论依据。

### 理想[二元混合物](@entry_id:168452)的[气液平衡](@entry_id:147234)

为了构建描述[气液平衡](@entry_id:147234)（VLE）的基本模型，我们首先从最简单的情况入手：**[理想溶液](@entry_id:148303) (ideal solution)**。一个理想溶液的形成过程在[热力学](@entry_id:141121)上没有能量变化（[混合焓](@entry_id:158999) $\Delta H_{mix} = 0$）和体积变化（[混合体积](@entry_id:183492) $\Delta V_{mix} = 0$）。从微观角度看，这意味着混合物中不同分子间的相互作用力（A-B）与纯组分中同类分子间的相互作用力（A-A 和 B-B）的平均强度是相同的。

#### [拉乌尔定律](@entry_id:139880)

对于[理想溶液](@entry_id:148303)，法国科学家 François-Marie Raoult 在19世纪80年代提出了一个简洁而强大的定律，即 **[拉乌尔定律](@entry_id:139880) (Raoult's Law)**。该定律指出，在一定温度下，[理想溶液](@entry_id:148303)上方蒸气中某组分 $i$ 的 **分压 (partial pressure)** $p_i$ 等于该组分的 **纯组分饱和蒸气压 (pure component vapor pressure)** $P_i^*$ 与其在液相中的 **摩尔分数 (mole fraction)** $x_i$ 的乘积：

$p_i = x_i P_i^*$

纯组分的饱和[蒸气压](@entry_id:136384) $P_i^*$ 是物质固有属性，它反映了该物质分子从液相逃逸到气相的倾向，是衡量物质 **挥发性 (volatility)** 的一个关键指标。$P_i^*$ 越高的物质，其挥发性越强。

根据[道尔顿分压定律](@entry_id:140483)，气相[总压](@entry_id:265293) $P$ 是各组分分压之和。对于[二元混合物](@entry_id:168452)，总压为：

$P = p_A + p_B = x_A P_A^* + x_B P_B^*$

由于 $x_A + x_B = 1$，我们可以将总压表示为单一组成变量的函数：

$P = x_A P_A^* + (1 - x_A) P_B^* = P_B^* + x_A (P_A^* - P_B^*)$

这个[线性关系](@entry_id:267880)是理想溶液在恒定温度下[总压](@entry_id:265293)随液[相组成](@entry_id:197559)变化的标志。例如，如果我们知道正己烷和正庚烷在 60.0 °C 时的纯组分蒸气压分别为 $P_{hex}^* = 405$ torr 和 $P_{hept}^* = 140$ torr，我们就可以计算出在何种液体组成下，该混合物会在 300 torr 的外部压力下开始沸腾。沸腾的起始点意味着混合物的总蒸气压等于外部压力，即 $P = 300$ torr。代入上述方程求解 $x_{hex}$ [@problem_id:1990613]：

$300 = x_{hex} (405) + (1 - x_{hex}) (140)$
$160 = x_{hex} (405 - 140) = 265 x_{hex}$
$x_{hex} = \frac{160}{265} \approx 0.604$

这表明，当液相中正己烷的[摩尔分数](@entry_id:145460)为 0.604 时，该[理想混合物](@entry_id:180997)在 60.0 °C 和 300 torr 压力下达到[气液平衡](@entry_id:147234)的[临界点](@entry_id:144653)。

#### 气[相组成](@entry_id:197559)与[相对挥发度](@entry_id:141834)

[气液平衡](@entry_id:147234)不仅涉及总压，还涉及平衡时气相的组成。气相中组分 $i$ 的[摩尔分数](@entry_id:145460) $y_i$ 可由其[分压](@entry_id:168927)与总压的比值确定：

$y_i = \frac{p_i}{P} = \frac{x_i P_i^*}{P}$

一个重要的结论是：对于理想溶液，气相总是富含更易挥发的组分。也就是说，如果组分A比组分B更易挥发（即 $P_A^* > P_B^*$），那么在任何组成下，气相中A的摩尔分数将总是大于其在液相中的摩尔分数（$y_A > x_A$）。

我们可以通过一个实例来量化这一现象。考虑一个由环戊烷（C）和环己烷（X）组成的[理想溶液](@entry_id:148303)，在 298.15 K 时，$P_C^* = 41.5$ kPa，$P_X^* = 13.0$ kPa。如果液相中环戊烷的[摩尔分数](@entry_id:145460) $x_C = 0.450$，则平衡气相中环戊烷的[摩尔分数](@entry_id:145460) $y_C$ 可以计算如下 [@problem_id:1990597]：

首先计算[总压](@entry_id:265293) $P$：
$P = x_C P_C^* + (1 - x_C) P_X^* = (0.450)(41.5 \text{ kPa}) + (0.550)(13.0 \text{ kPa}) = 18.675 + 7.15 = 25.825 \text{ kPa}$

然后计算 $y_C$：
$y_C = \frac{x_C P_C^*}{P} = \frac{18.675 \text{ kPa}}{25.825 \text{ kPa}} \approx 0.723$

正如预期，由于环戊烷 ($P_C^*=41.5$ kPa) 比环己烷 ($P_X^*=13.0$ kPa) 更易挥发，平衡气相中环戊烷的浓度 (0.723) 远高于其在液相中的浓度 (0.450)。这种组成上的差异是[蒸馏](@entry_id:140660)分离技术的基础。

为了量化分离的难易程度，我们引入 **[相对挥发度](@entry_id:141834) (relative volatility)** $\alpha_{AB}$ 的概念，其定义为：

$\alpha_{AB} = \frac{y_A / x_A}{y_B / x_B}$

对于理想溶液，代入 $y_i = x_i P_i^* / P$ 可得：

$\alpha_{AB} = \frac{(x_A P_A^* / P) / x_A}{(x_B P_B^* / P) / x_B} = \frac{P_A^*}{P_B^*}$

$\alpha_{AB}$ 的值越大，表明组分A相对于组分B越容易挥发，两组分通过[蒸馏](@entry_id:140660)分离就越容易。当 $\alpha_{AB}=1$ 时，气液两[相组成](@entry_id:197559)相同，无法通过常规[蒸馏](@entry_id:140660)进行分离。

### 相图的构建与解读

[相图](@entry_id:144015)是将上述[热力学](@entry_id:141121)关系可视化的重要工具。对于二元气液系统，最常用的[相图](@entry_id:144015)是恒温下的压力-组成图（P-xy图）和恒压下的[温度-组成图](@entry_id:180991)（[T-xy图](@entry_id:170932)）。

#### 恒温 P-xy 图

在恒定温度下，P-xy图描绘了系统压力如何随组成变化。图中包含两条重要的边界线：

- **泡点线 (bubble-point curve):** 这条线表示在给定组成 $x_A$ 下，液体开始沸腾（出现第一个气泡）的压力。对于[理想溶液](@entry_id:148303)，这条线是连接 $(x_A=0, P=P_B^*)$ 和 $(x_A=1, P=P_A^*)$ 的直线，其方程为 $P = P_B^* + x_A (P_A^* - P_B^*)$。

- **[露点](@entry_id:153435)线 (dew-point curve):** 这条线表示在给定组成 $y_A$ 下，蒸气开始冷凝（出现第一滴液体）的压力。其方程可以通过联立 $y_A = x_A P_A^*/P$ 和 $P = x_A P_A^* + (1-x_A)P_B^*$ 消去 $x_A$ 得到：
  $P = \frac{1}{\frac{y_A}{P_A^*} + \frac{1-y_A}{P_B^*}}$

这两条曲线在纯组分处重合，但在混合物区域则将[相图](@entry_id:144015)分成了三个区域 [@problem_id:1990625]：
1.  **液相区 (Liquid Phase):** 位于泡点线之上。在此高压区域，系统为单一的亚冷液体。
2.  **气相区 (Vapor Phase):** 位于[露点](@entry_id:153435)线之下。在此低压区域，系统为单一的[过热](@entry_id:147261)蒸气。
3.  **气液共存区 (Two-Phase Region):** 位于泡点线和[露点](@entry_id:153435)线之间。在此区域内，液相和气相[稳定共存](@entry_id:170174)。

对于任意混合物组成，其泡点压力总是高于其[露点](@entry_id:153435)压力。这是因为从液相开始升华（泡点）与从气相开始冷凝（[露点](@entry_id:153435)），始末状态的组成不同所致。例如，在苯和甲苯的[理想混合物](@entry_id:180997)中，对于一个组成为0.400的体系，其泡点[压力计](@entry_id:138596)算值（1.148 atm）显著高于其[露点](@entry_id:153435)[压力计](@entry_id:138596)算值（约0.963 atm），其比值约为1.19 [@problem_id:1990625]。

#### 恒压 T-xy 图

在[化学工程](@entry_id:143883)实践中，如常压[蒸馏](@entry_id:140660)，恒压下的 T-xy 图更为常用。此时，压力 $P$ 是一个固定参数。由于纯组分的饱和[蒸气压](@entry_id:136384) $P_i^*$ 是温度的强函数（通常由安托万方程描述），T-xy 图中的曲线不再是简单的直[线或](@entry_id:170208)双曲线。

在 T-xy 图中，横轴仍然是[摩尔分数](@entry_id:145460)，纵轴是温度。与 P-xy 图相反，**泡点曲线 (bubble-point curve)** 位于下方，而 **[露点](@entry_id:153435)曲线 (dew-point curve)** 位于上方。这是因为在固定压力下，需要更高的温度才能将所有液体蒸发成气体。

- **泡点曲线** 显示了不同组成的液体在给定压力下的[沸点](@entry_id:139893)。
- **[露点](@entry_id:153435)曲线** 显示了不同组成的蒸气在给定压力下的冷凝点。

这两条曲线同样将相[图划分](@entry_id:152532)为三个区域 [@problem_id:1990620]：
1.  **液相区 (Liquid Phase):** 位于泡点曲线之下。在此低温区域，系统为单一的亚冷液体。
2.  **气相区 (Vapor Phase):** 位于[露点](@entry_id:153435)曲线之上。在此高温区域，系统为单一的[过热](@entry_id:147261)蒸气。
3.  **气液共存区 (Two-Phase Region):** 位于泡点曲线和[露点](@entry_id:153435)曲线之间，呈纺锤形。

要判断一个特定状态下系统的相态，我们只需将该点在相图上定位。假设一个总组成为 $z_A$ 的[二元混合物](@entry_id:168452)被置于温度 $T$ 和恒定压力 $P$ 下。我们可以在 T-xy 图上找到对应温度 $T$ 的泡点组成 $x_A(T)$ 和[露点](@entry_id:153435)组成 $y_A(T)$。
- 如果 $z_A  x_A(T)$，系统处于液相区。
- 如果 $z_A > y_A(T)$，系统处于气相区。
- 如果 $x_A(T)  z_A  y_A(T)$，系统处于气液共存区，会分离成液相和气相。

例如，对于一个在1 atm下，总组成为 $z_A=0.60$ 的A-B混合物，如果在360 K时，通过实验或模型计算得出该温度下的平衡液[相组成](@entry_id:197559)为 $x_A(360) \approx 0.553$，平衡气[相组成](@entry_id:197559)为 $y_A(360) = 0.80$。由于 $0.553  0.60  0.80$，该系统的状态点位于气液共存区内，因此它将以气液两[相混合](@entry_id:199798)物的形式存在 [@problem_id:1990620]。

#### 联结线与杠杆定律

在气液共存区内，任何一个状态点都代表一个整体组成。系统会自发分离成一个液相和一个气相，它们的组成由该状态点所在水平线（在[T-xy图](@entry_id:170932)中）或垂直线（在P-xy图中）与泡点线和[露点](@entry_id:153435)线的交点确定。这条连接平衡的气相点和液相点的线段被称为 **联结线 (tie line)**。

那么，这两相的相对量是多少呢？这可以通过 **杠杆定律 (lever rule)** 来计算。该定律源于简单的物料守恒。设系统总摩尔数为 $n_T$，总组成为 $z_A$。在平衡时，系统分为摩尔数为 $n_L$、组成为 $x_A$ 的液相，和摩尔数为 $n_V$、组成为 $y_A$ 的气相。

总物料守恒： $n_L + n_V = n_T$
组分A守恒： $n_L x_A + n_V y_A = n_T z_A$

联立求解可得液相摩尔数 $n_L$ 和气相摩尔数 $n_V$ 的表达式：

$\frac{n_L}{n_T} = \frac{y_A - z_A}{y_A - x_A}$
$\frac{n_V}{n_T} = \frac{z_A - x_A}{y_A - x_A}$

这个关系在[相图](@entry_id:144015)上有一个直观的几何解释：液相的相对量与联结线上“气相点到总组成点的距离”成正比，而气相的相对量与“液相点到总组成点的距离”成正比，就像一个以总组成点为支点的杠杆。

例如，一个总摩尔数为 5.00 mol、A的摩尔分数为 $z_A=0.450$ 的密闭体系，在加热到平衡后，测得液[相组成](@entry_id:197559)为 $x_A=0.300$，气[相组成](@entry_id:197559)为 $y_A=0.680$。我们可以利用杠杆定律计算平衡时液相的总质量 [@problem_id:1990579]。首先计算液相的摩尔数 $n_L$：

$n_L = n_T \frac{y_A - z_A}{y_A - x_A} = 5.00 \text{ mol} \times \frac{0.680 - 0.450}{0.680 - 0.300} = 5.00 \times \frac{0.230}{0.380} \approx 3.026 \text{ mol}$

如果已知组分A和B的[摩尔质量](@entry_id:146110)（例如 $M_A = 78.11$ g/mol, $M_B = 92.14$ g/mol），则可以计算出液相的平均[摩尔质量](@entry_id:146110) $\bar{M}_L = x_A M_A + (1-x_A) M_B$，进而得到液相的总质量 $m_L = n_L \bar{M}_L \approx 266$ g。

### 非[理想混合物](@entry_id:180997)与共沸现象

现实世界中的大多数混合物都不是理想的。[分子间作用力](@entry_id:203760)的差异导致了与[拉乌尔定律](@entry_id:139880)的偏差，并可能产生共沸等复杂现象。

#### [拉乌尔定律](@entry_id:139880)的偏差

非理想行为的根源在于混合物中不同分子间的相互作用力（A-B）与纯组分中同类分子间的相互作用力（A-A, B-B）不相等。

- **正偏差 (Positive Deviation):** 当异种分子间的吸[引力](@entry_id:175476)（A-B）弱于同种分子间的平均吸[引力](@entry_id:175476)时，混合物中的分子更容易“逃逸”到气相。这通常伴随着吸热的混合过程 ($\Delta H_{mix}  0$)。结果是，混合物的实际蒸气压高于[拉乌尔定律](@entry_id:139880)的预测值 ($P_{actual}  P_{Raoult}$)。典型的例子是乙醇和水的混合物，[氢键网络](@entry_id:750458)被破坏。

- **负偏差 (Negative Deviation):** 当异种分子间形成更强的吸[引力](@entry_id:175476)（如[氢键](@entry_id:142832)或偶极-偶极作用）时，混合过程通常是放热的 ($\Delta H_{mix}  0$)。分子在液相中被更紧密地束缚，导致“逃逸”倾向降低。因此，实际[蒸气压](@entry_id:136384)低于[拉乌尔定律](@entry_id:139880)的预测值 ($P_{actual}  P_{Raoult}$) [@problem_id:1990574]。一个经典的例子是丙酮和氯仿的混合物，它们之间可以形成[氢键](@entry_id:142832)。

#### [活度系数](@entry_id:148405)

为了定量描述这种非理想行为，我们引入 **活度系数 (activity coefficient)** $\gamma_i$ 来修正[拉乌尔定律](@entry_id:139880)。修正后的表达式称为 **修正的[拉乌尔定律](@entry_id:139880) (modified Raoult's law)**：

$p_i = x_i \gamma_i P_i^*$

[活度系数](@entry_id:148405) $\gamma_i$ 是一个无量纲的修正因子，它依赖于温度、压力和组成。
- 对于正偏差体系，$\gamma_i > 1$。
- 对于负偏差体系，$\gamma_i  1$。
- 对于[理想溶液](@entry_id:148303)，$\gamma_i = 1$。

[活度系数](@entry_id:148405)可以通过[热力学](@entry_id:141121)模型（如[Margules方程](@entry_id:143054)、van Laar方程、Wilson方程等）与实验数据拟合得到。例如，一个简单的 **单参数Margules模型** 将[超额吉布斯能](@entry_id:172602) $G^E$ 表示为 $G^E = \alpha x_A x_B$，由此可以推导出活度系数的表达式：

$\ln\gamma_A = \frac{\alpha}{RT} x_B^2$
$\ln\gamma_B = \frac{\alpha}{RT} x_A^2$

其中 $\alpha$ 是一个经验参数。若 $\alpha  0$，则 $\gamma_i  1$，系统表现为正偏差。若 $\alpha  0$（有时记为 $\beta$），则 $\gamma_i  1$，系统表现为负偏差。

我们可以利用这类模型来量化偏差的程度。例如，一个由Chloretone(A)和Methyform(B)组成的混合物，其行为可以用 $\beta=-1.50$ 的Margules模型描述。对于 $x_A=0.400$ 的混合物，我们可以计算其活度系数并求出实际[总压](@entry_id:265293) $P_{actual}$。然后与理想情况下的 $P_{Raoult}$ 比较，可以得到相对偏差 [@problem_id:1990618]。计算表明，该混合物的实际[压力比](@entry_id:137698)理想预测值低约31.7%，这是一个显著的负偏差。

#### [共沸物](@entry_id:146150)

当非理想偏差足够大时，T-xy或P-xy图的纺锤形区域会发生扭曲，出现极值点，这就是 **共沸点**。在共[沸点](@entry_id:139893)，气相和液相的组成完全相同 ($x_i = y_i$)，因此混合物像纯物质一样在恒定温度下沸腾，其组分无法通过简单的蒸馏分离。

- **最[低沸点共沸物](@entry_id:143101) (Minimum-boiling azeotrope):** 由强烈的正偏差引起。在[T-xy图](@entry_id:170932)上表现为一个温度最低点，在P-xy图上则对应一个压力最高点（**最高压力共沸物**）。这意味着在共沸组成下，混合物的沸点比任一纯组分都低。

- **[最高沸点共沸物](@entry_id:138386) (Maximum-boiling azeotrope):** 由强烈的负偏差引起。在[T-xy图](@entry_id:170932)上表现为一个温度最高点，在P-xy图上则对应一个压力最低点（**最低压力共沸物**）。在这种情况下，共沸物的[沸点](@entry_id:139893)高于任一纯组分的沸点 [@problem_id:1990574]。

共沸现象的发生条件可以用[相对挥发度](@entry_id:141834)来精确描述。在共沸点，由于 $x_i=y_i$，我们有 $y_A/x_A = 1$ 和 $y_B/x_B = 1$。因此，[相对挥发度](@entry_id:141834)：

$\alpha_{AB} = \frac{y_A/x_A}{y_B/x_B} = 1$

将修正的[拉乌尔定律](@entry_id:139880)代入[相对挥发度](@entry_id:141834)的表达式 $\alpha_{AB} = (\gamma_A P_A^*)/(\gamma_B P_B^*)$，我们得到共沸的条件：

$\frac{\gamma_A P_A^*}{\gamma_B P_B^*} = 1 \quad \implies \quad \gamma_A P_A^* = \gamma_B P_B^*$

这个简单的关系式是寻找共沸组成的理论基础。例如，对于丙酮(A)和甲醇(M)的混合物，即使不知道具体的共沸组成，我们也可以在共[沸点](@entry_id:139893)处确定活度系数的比值 [@problem_id:1990622]：

$\frac{\gamma_A}{\gamma_M} = \frac{P_M^*}{P_A^*}$

利用给定的纯组分[蒸气压](@entry_id:136384) $P_A^* = 625$ torr 和 $P_M^* = 415$ torr，可以立即求出在共沸点 $\gamma_A / \gamma_M = 415/625 = 0.664$。

更有甚者，结合[活度系数](@entry_id:148405)模型，我们可以预测体系是否会形成共沸。以单参数Margules模型为例，将活度系数表达式代入共沸条件，可以解出潜在的共沸组成 $x_A^{az}$。只有当解出的 $x_A^{az}$ 位于 $(0, 1)$ 区间内时，共沸物才实际存在。这通常转化为一个关于模型参数 $\alpha$ 和纯组分[蒸气压](@entry_id:136384)比值的判据。例如，对于一个正偏差系统，当 $\alpha/(RT)  |\ln(P_A^*/P_B^*)|$ 时，系统就会形成一个最[低沸点共沸物](@entry_id:143101) [@problem_id:1990598]。这个判据为化学工程师在设计分离过程时，提供了预测复杂[相行为](@entry_id:199883)的有力工具。
