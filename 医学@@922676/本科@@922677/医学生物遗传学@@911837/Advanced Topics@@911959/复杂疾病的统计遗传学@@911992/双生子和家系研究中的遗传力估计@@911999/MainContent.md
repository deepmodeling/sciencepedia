## 引言
人类的性状，从身高、体重到患上精神分裂症的风险，无一不是先天遗传（nature）与后天环境（nurture）复杂互动的产物。几个世纪以来，“先天与后天”之争一直是生物学和心理学领域的核心议题。然而，如何科学且定量地剖析这两股力量的相对贡献，始终是一个巨大的挑战。双生子与家庭研究为解决这一问题提供了强有力的经典方法，它通过巧妙地利用不同亲缘关系个体间遗传和环境共享度的差异，使我们能够估算“遗传度”——即遗传差异在多大程度上解释了人群中的表型差异。

本文旨在系统性地介绍从双生子与家庭研究中估计遗传度的核心原理、应用与实践。我们将超越简单的概念介绍，深入探讨其背后的[统计模型](@entry_id:755400)、关键假设以及在解释结果时必须考虑的复杂因素。通过阅读本文，你将不仅理解遗传度是什么，更将学会如何批判性地评估和应用这一重要概念。

文章分为三个核心部分。在第一章“原则与机制”中，我们将深入探讨分解[表型方差](@entry_id:274482)的生物统计学模型，特别是经典的ACE模型，并阐明其数学基础和关键假设。接着，在第二章“应用与跨学科交叉”中，我们将展示这些理论如何在医学遗传学、心理学等领域得到应用，以揭示从精神障碍到健康行为等多种复杂性状的遗传结构，并讨论其与现代基因组学的联系。最后，在第三章“动手实践”中，你将有机会通过具体案例，亲自运用这些方法来解决实际的[遗传分析](@entry_id:167901)问题，从而巩固所学知识。

## 原则与机制

本章在前一章介绍性背景的基础上，深入探讨了从双生子和家庭研究中估计遗传度的核心原则和生物统计学机制。我们将从分解[表型方差](@entry_id:274482)的基本模型出发，逐步构建用于估计遗传和[环境影响](@entry_id:161306)的数学框架。最后，我们将审视这些模型的关键假设及其局限性，确保读者对遗传度这一概念及其在医学遗传学中的应用有一个严谨和批判性的理解。

### 生物统计学模型：分解[表型方差](@entry_id:274482)

任何[数量性状](@entry_id:144946)（如身高、血压或某种生物标志物的水平）的个体差异都可以被概念化为遗传（**genotype**, $G$）和环境（**environment**, $E$）因素共同作用的结果。一个简单的模型可以表示为：

$P = G + E$

其中 $P$ 代表个体的表型值。然而，为了进行更精细的分析，我们需要进一步分解这些组成部分。在[数量遗传学](@entry_id:154685)中，总的[表型方差](@entry_id:274482)（**phenotypic variance**, $V_P$）被分解为各个独立成分的方差之和。

遗传方差（**genetic variance**, $V_G$）可以被分解为三个主要部分：
- **加性遗传方差 ($V_A$)**：由等位基因的加性效应引起，这些效应可以简单地相加以预测后代的表型。这是亲代能够可靠地传递给子代的遗传变异部分。
- **[显性遗传方差](@entry_id:197376) ($V_D$)**：源于特定基因座上等位基因之间的相互作用（显性或隐性）。这种效应依赖于基因型组合，因此不会简单地从亲代传递给子代。
- **[上位性](@entry_id:136574)遗传方差 ($V_I$)**：来自不同基因座上等位基因之间的相互作用。与显性一样，这些是特定的基因组合效应，不会稳定遗传。

同样，环境方差（**environmental variance**, $V_E$）也被分为两个部分：
- **共同环境方差 ($V_C$)**：由使得同一家庭成员（如一同抚养的双生子）相似的环境因素引起，例如家庭社会经济地位、共享的饮食习惯或父母教养方式。
- **独特环境方差 ($V_E$)**：由使得同一家庭成员不同的环境因素以及测量误差引起。例如，不同的朋友、意外事故或特定于个体的经历。

假设所有这些成分彼此不相关，总[表型方差](@entry_id:274482)可以表示为：

$V_P = V_A + V_D + V_I + V_C + V_E$

基于此，我们定义两种类型的遗传度[@problem_id:5045658]：

- **广义遗传度 ($H^2$)**：指总遗传方差占总[表型方差](@entry_id:274482)的比例，$H^2 = \frac{V_G}{V_P} = \frac{V_A + V_D + V_I}{V_P}$。它衡量了所有遗传因素（加性、显性和上位性）对[表型变异](@entry_id:163153)的总体贡献。

- **狭义遗传度 ($h^2$)**：指加性遗传方差占总[表型方差](@entry_id:274482)的比例，$h^2 = \frac{V_A}{V_P}$。狭义遗传度在[医学遗传学](@entry_id:262833)和进化生物学中尤为重要，因为它反映了性状能够从亲代传递给子代的程度，并决定了对人工或自然选择的响应潜力。

从定义可知，广义遗传度包含了所有类型的遗传效应，而狭义遗传度仅包含加性效应。因此，当且仅当存在非加性遗传效应（即$V_D > 0$或$V_I > 0$）时，才会出现$H^2 > h^2$。如果一个性状的遗传基础完全是加性的（$V_D = 0$且$V_I = 0$），则$H^2 = h^2$ [@problem_id:5045662]。

### 双生子与家庭研究的逻辑

估计上述方差成分的核心逻辑在于，不同亲缘关系的个体在遗传和环境上的相似度是可预测的。通过比较不同亲属对（如双生子、兄弟姐妹、父母与子女）的表型相似性，我们可以反向推断出$V_A, V_D, V_C$等潜在方差的大小。

#### 经典双生子设计

[双生子研究](@entry_id:263760)是这一领域最有力的工具，因为它提供了一个自然的实验来分离遗传和环境的影响。该设计比较了两种类型的双生子：
- **同卵双生子 (Monozygotic, MZ)**：由单个[受精](@entry_id:274949)卵分裂而成，他们在遗传上是完全相同的。因此，他们共享$100\%$的加性基因（$A$）、显性基因组合（$D$）和上位性基因组合（$I$）。如果他们被共同抚养，他们也共享共同环境（$C$）。
- **异卵双生子 (Dizygotic, DZ)**：由两个独立的[受精](@entry_id:274949)卵发育而来，他们的遗传关系与普通兄弟姐妹相同，平均共享$50\%$的加性基因（$A$）和$25\%$的显性基因组合（$D$）。同样，共同抚养的DZ双生子也共享共同环境（$C$）。

亲属对之间的表型相关性（**correlation**, $r$）等于他们的表型协方差（**covariance**, $Cov$）除以总[表型方差](@entry_id:274482)（$V_P$）。协方差则由他们共享的方差成分构成。因此，我们可以推导出预期的双生子相关性[@problem_id:5045658]：

$r_{MZ} = \frac{V_A + V_D + V_I + V_C}{V_P}$

$r_{DZ} = \frac{\frac{1}{2}V_A + \frac{1}{4}V_D + k_I V_I + V_C}{V_P}$

其中$k_I$是[上位性方差](@entry_id:263723)的共享系数，对于加性-加性[上位性](@entry_id:136574)，该值为$(\frac{1}{2})^2 = \frac{1}{4}$。

### AC[E模](@entry_id:160271)型：一个实用的简化框架

完整地分解所有五个方差成分（$V_A, V_D, V_I, V_C, V_E$）仅凭MZ和DZ双生子数据是困难的，因为我们只有两个观测值（$r_{MZ}$和$r_{DZ}$）却要估计多个未知数。这是一个典型的模型不可识别（**underdetermined**）问题[@problem_id:5045662]。因此，研究者通常采用简化的模型。最常用的是**ACE模型**，它假设所有遗传效应都是加性的，即不存在显性（$V_D=0$）和上位性（$V_I=0$）效应。

#### ACE路径模型

ACE模型可以通过[路径图](@entry_id:274599)进行直观和严谨的表示[@problem_id:5045685]。在一个双生子对中，每个双生子（$P_1, P_2$）的表型被建模为三个潜在（**latent**）因素的[线性组合](@entry_id:155091)：加性遗传因素（$A$）、共同环境因素（$C$）和独特环境因素（$E$）。每个因素都被标准化为方差为1，并通过路径系数（**path coefficients**）$a, c, e$与表型相连：

$P_i = a A_i + c C_i + e E_i$

根据方差和协方差的基本性质，我们可以推导出：
- **总[表型方差](@entry_id:274482)**：由于$A_i, C_i, E_i$在个体内部不相关，总方差为：
$V_P = \operatorname{Var}(P_i) = a^2 \operatorname{Var}(A_i) + c^2 \operatorname{Var}(C_i) + e^2 \operatorname{Var}(E_i) = a^2 + c^2 + e^2$
- **双生子协方差**：
  - 对于MZ双生子，$A_1$与$A_2$的相关性为1，$C_1$与$C_2$的相关性为1，$E_1$与$E_2$的相关性为0。因此，协方差为：
  $\operatorname{Cov}(P_1, P_2)_{MZ} = a^2 \operatorname{Corr}(A_1, A_2) + c^2 \operatorname{Corr}(C_1, C_2) = a^2 + c^2$
  - 对于DZ双生子，$A_1$与$A_2$的相关性为$0.5$，$C_1$与$C_2$的相关性为1，$E_1$与$E_2$的相关性为0。因此，协方差为：
  $\operatorname{Cov}(P_1, P_2)_{DZ} = a^2 \operatorname{Corr}(A_1, A_2) + c^2 \operatorname{Corr}(C_1, C_2) = \frac{1}{2}a^2 + c^2$

#### Falconer公式：估计方差成分

将协方差转换为相关性（通过除以$V_P$），我们得到AC[E模](@entry_id:160271)型下的预期双生子相关性：

$r_{MZ} = \frac{a^2 + c^2}{a^2 + c^2 + e^2}$
$r_{DZ} = \frac{\frac{1}{2}a^2 + c^2}{a^2 + c^2 + e^2}$

通过简单的代数运算，我们可以求解这些方程来估计代表狭义遗传度（$h^2$）、共同环境效应和独特环境效应比例的路径系数的平方（$a^2, c^2, e^2$）。这组求解公式通常被称为**Falconer公式**[@problem_id:5045662]：

$a^2 = 2(r_{MZ} - r_{DZ})$
$c^2 = r_{MZ} - a^2 = 2r_{DZ} - r_{MZ}$
$e^2 = 1 - r_{MZ}$

为了阐明这一点，我们来看一个具体的例子。假设一项研究发现某性状的MZ双生子相关性为$r_{MZ} = 0.70$，DZ双生子相关性为$r_{DZ} = 0.40$ [@problem_id:5045658]。首先，一个重要的经验法则是，如果$r_{DZ} > \frac{1}{2} r_{MZ}$，这通常表明存在共同环境（$C$）的影响，因为纯粹的加性-显性（ADE）模型预测$r_{DZ} \le \frac{1}{2} r_{MZ}$。在本例中，$0.40 > \frac{1}{2}(0.70) = 0.35$，符合存在$C$的模式。应用Falconer公式，我们得到：

- 加性遗传贡献：$a^2 = 2(0.70 - 0.40) = 0.60$
- 共同环境贡献：$c^2 = 0.70 - 0.60 = 0.10$
- 独特环境贡献：$e^2 = 1 - 0.70 = 0.30$

因此，我们估计该性状的狭义遗传度约为$60\%$，共同环境解释了$10\%$的变异，独特环境和测量误差解释了剩余的$30\%$。

在计算这些相关性时，值得注意的是，双生子数据具有“可交换性”——在双生子对中，“双生子1”和“双生子2”的标签是任意的。因此，最合适的相似性度量是**组内相关系数 (Intraclass Correlation Coefficient, ICC)**，它正是为衡量这种可交换数据组内的相似性而设计的。相比之下，标准的皮尔逊相关系数（Pearson correlation）将两个变量（如“双生子1”的集合和“双生子2”的集合）区别对待，这与双生子模型的基本假设不符 [@problem_id:5045683]。

### 遗传度：一个依赖于群体的统计量

一个普遍存在的误解是，遗传度是某个性状固有不变的生物学常数。事实并非如此。遗传度是一个特定于**群体**和**环境**的统计量。它的定义，$h^2 = \frac{V_A}{V_P}$，明确表明它是一个比率，其值不仅取决于分子（加性遗传方差$V_A$），也取决于分母（总[表型方差](@entry_id:274482)$V_P$）。

为了说明这一点，请考虑两个群体，群体A和群体B。假设这两个群体具有完全相同的遗传背景（即相同的等位基因频率和效应大小），因此具有相同的[加性遗传方差](@entry_id:154158)$V_A$。然而，群体B生活在一个环境更多样化（例如，营养和教育水平差异更大）的社会中。这种环境异质性的增加会增大环境方差（$V_C$或$V_E$），从而增大总[表型方差](@entry_id:274482)$V_P$。即使$V_A$保持不变，由于分母$V_P$变大，群体B的遗传度$h^2$也会降低 [@problem_id:5045708] [@problem_id:5045655]。

例如，假设一项研究在两个群体中获得了以下数据 [@problem_id:5045708]：
- **群体A**（环境同质）：$V_P^{(A)} \approx 60$, $r_{MZ}^{(A)} \approx 0.83$, $r_{DZ}^{(A)} \approx 0.50$
- **群体B**（环境异质）：$V_P^{(B)} \approx 100$, $r_{MZ}^{(B)} \approx 0.50$, $r_{DZ}^{(B)} \approx 0.30$

在群体A中，遗传度估计为$h^2_{(A)} = 2(0.83 - 0.50) = 0.66$。绝对[加性遗传方差](@entry_id:154158)为$V_A^{(A)} \approx 0.66 \times 60 \approx 39.6$。
在群体B中，遗传度估计为$h^2_{(B)} = 2(0.50 - 0.30) = 0.40$。绝对[加性遗传方差](@entry_id:154158)为$V_A^{(B)} \approx 0.40 \times 100 = 40$。

正如预期的那样，尽管两个群体的绝对遗传方差（$V_A$）几乎相同（约为40个单位），但由于群体B的环境更加多变，导致其总[表型方差](@entry_id:274482)更大，其性状的遗传度显著降低。这突出表明，遗传度衡量的是在特定时间和特定群体现有变异范围内的相对贡献。

这一原则引出了几个重要的推论 [@problem_id:5045655]：
- **遗传度不适用于个体**：遗传度是一个描述群体变异来源的比例，不能用于解释单个个体的性状成因。一个人的身高是其基因和一生环境经历不可分割的产物。
- **高遗传度不等于基因[决定论](@entry_id:158578)**：一个性状具有高遗传度，仅意味着在该群体当前的環境範圍內，个体间的差异主要由遗传差异解释。这并不意味着该性状不可改变。即使是$100\%$遗传的疾病（如[苯丙酮尿症](@entry_id:202323)），也可以通过环境干预（特定饮食）得到有效控制。
- **MZ双生子的表型差异是独特环境的证据**：一对MZ双生子在性状上的任何差异都直接证明了独特环境（$E$）的影响，即$V_E > 0$。这并不与高遗传度相矛盾。

### 经典双生[子模](@entry_id:148922)型的假设与局限性

尽管[双生子研究](@entry_id:263760)功能强大，但其结论的有效性依赖于几个关键假设。理解这些假设以及当它们被违反时会发生什么，对于批判性地评估遗传度估计至关重要。

#### 相等环境假设 (Equal Environments Assumption, EEA)

**EEA**是双生子方法中最受争议的假设。它假定与性状相关的共同环境因素对MZ双生子和DZ双生子的影响程度是相同的[@problem_id:5045700]。在AC[E模](@entry_id:160271)型中，这体现为对MZ和DZ双生子使用同一个$c^2$参数。

然而，如果MZ双生子因为外貌更相似而被父母、老师或朋友更相似地对待，或者他们更倾向于选择相似的活动，那么他们的共同环境相似性可能会高于DZ双生子。这种情况下，EEA被违反，MZ双生子额外的环境相似性会被错误地归因于他们额外的遗传相似性，从而**高估遗传度$a^2$**。

检验EEA的一种方法是直接测量可能相关的环境因素。例如，在一项研究中，研究人员可以创建一个“环境相似性得分”（$S$）。如果未经调整的双生子相关性差异很大（如$r_{MZ}=0.60, r_{DZ}=0.35$），但在根据$S$进行匹配或[统计控制](@entry_id:636808)后，相关性变得非常接近（如$r_{MZ|S}=0.45, r_{DZ|S}=0.43$），这就强烈表明最初的MZ“超额”相似性大部分是由环境而非遗传驱动的，这意味着EEA可能不成立 [@problem_id:5045700]。

#### 随机婚配假设 (Random Mating Assumption)

经典双生子模型假设婚配是随机的，即配偶的选择与所研究的性状无关。在此假设下，DZ双生子的[遗传相关](@entry_id:176283)性为$0.5$。然而，对于许多性状，如身高、智商或某些精神特质，存在**选择性婚配 (assortative mating)**，即人们倾向于选择与自己表型相似的配偶 [@problem_id:5045681]。

**正向选择性婚配**（配偶表型正相关）会增加DZ双生子的遗传相似性，使其高于$0.5$。这是因为表型相似的父母在遗传上也更可能相似。当研究者忽略这一点并使用标准的AC[E模](@entry_id:160271)型时，DZ双生子相关性被人为抬高，导致$r_{MZ} - r_{DZ}$的差值变小。结果是，[加性遗传方差](@entry_id:154158)$A$被**低估**，而共同环境方差$C$被**高估**。

例如，如果一个性状的真实方差成分是$A=0.60, C=0.20$，在正向选择性婚配下，DZ相关性可能会从预期的$0.5(0.60)+0.20 = 0.50$上升到$0.55$。如果研究者使用观测到的$r_{MZ}=0.80$和$r_{DZ}=0.55$进行分析，他们会错误地计算出$A = 2(0.80-0.55) = 0.50$和$C = 0.80 - 0.50 = 0.30$，这与真实值相比，低估了$A$并高估了$C$ [@problem_id:5045681] [@problem_id:5045662]。反之，**负向选择性婚配**（配偶表型不相似）则会产生相反的偏误，即高估$A$和低估$C$ [@problem_id:5045681]。

#### 基因-环境相关性 (Gene-Environment Correlation, rGE)

AC[E模](@entry_id:160271)型假设遗传和环境因素是独立的。然而，在现实中，它们常常是相关的，这种相关性被称为**基因-环境相关性 (rGE)**。rGE主要有三种类型，它们在双生[子模](@entry_id:148922)型中会以不同方式歪曲结果 [@problem_id:5045677]：

- **被动型rGE (Passive rGE)**：当儿童被动地接受与其遗传倾向相关的环境时发生。例如，有音乐天赋的父母不仅将音乐相关的基因传给孩子，还可能在家中创造一个充满音乐的环境。这个由父母创造的环境对MZ和DZ双生子来说都是共同的，因此，其效应会被错误地归入**共同环境方差$C$**。

- **诱发型rGE (Evocative rGE)**：当个体的遗传倾向引发他人特定的环境反应时发生。例如，一个天生外向的孩子可能会从老师和同伴那里获得更多的社交互动。由于MZ双生子在遗传上完全相同，他们会比DZ双生子引发更相似的环境反应。这种与[遗传相关](@entry_id:176283)的环境相似性模式（MZ相似度约为DZ的两倍）与加性遗传的模式无法区分，因此会被错误地归入**[加性遗传方差](@entry_id:154158)$A$**，从而高估遗传度。

- **主动型rGE (Active rGE)**：当个体根据其遗传倾向主动选择或创造环境时发生（也称为“ niche-picking”，生态位选择）。例如，有运动天赋的孩子可能会主动加入运动队。同样，由于MZ双生子在遗传上更相似，他们会比DZ双生子选择更相似的环境。这种效应也会被错误地归入**加性遗传方差$A$**，导致遗传度被高估。

#### 测量误差 (Measurement Error)

任何测量都不可避免地存在误差。在经典[测量理论](@entry_id:153616)中，一个观察值（$X$）被看作是真实值（$T$）和[随机误差](@entry_id:144890)（$\varepsilon$）之和。随机测量误差在双生子对中是不相关的，因此它构成了**独特环境方差$E$**的一部分。

这意味着，测量的不精确性会人为地增加$V_E$，从而增加总[表型方差](@entry_id:274482)$V_P$。由于$h^2 = \frac{V_A}{V_P}$，测量误差的存在会**系统性地低估遗传度**。

幸运的是，如果我们可以通过信度研究（如重测信度）估计测量的信度（$r_{xx}$），我们就可以校正这种偏误。信度$r_{xx}$代表真实方差占观察方差的比例。我们可以通过以下公式获得校正后的遗传度[@problem_id:5045695]：

$h^2_{corrected} = \frac{h^2_{naive}}{r_{xx}}$

例如，如果从观察数据中计算出的“朴素”遗传度为$h^2_{naive} = 0.52$，而测量的信度为$r_{xx}=0.80$，那么校正测量误差影响后，真实性状的遗传度估计为$h^2_{corrected} = \frac{0.52}{0.80} = 0.65$。

### 超越经典双生子设计：领养研究

为了克服双生[子模](@entry_id:148922)型的一些局限性，研究者可以利用其他家庭设计。**领养研究**是其中特别有价值的一种，因为它提供了一种独特的方式来分离遗传和抚养环境[@problem_id:5045638]。

领养研究的基本逻辑是比较[遗传相关](@entry_id:176283)但环境不相关（如被分开抚养的同卵双生子）以及环境相关但遗传不相关（如被共同抚养的无血缘关系的领养兄弟姐妹）的个体。

考虑一个简单的领养设计，比较一同抚养的亲生兄弟姐妹和领养兄弟姐妹：
- **领养兄弟姐妹**：他们在遗传上不相关（$A=0$），但共享一个共同的家庭环境（$C$）。因此，他们的表型相关性直接估计了共同环境的贡献：$r_{adopt} \approx c^2$。
- **亲生兄弟姐妹**：他们平均共享$50\%$的加性基因和共同的家庭环境。因此，他们的相关性为：$r_{bio} \approx \frac{1}{2}a^2 + c^2$。

通过比较这两种相关性，我们可以清晰地分离$A$和$C$：
$c^2 \approx r_{adopt}$
$a^2 \approx 2(r_{bio} - r_{adopt})$

例如，如果领养兄弟姐妹的相关性为$0.10$，而亲生兄弟姐妹的相关性为$0.35$，我们可以估计$c^2 \approx 0.10$，而$a^2 \approx 2(0.35 - 0.10) = 0.50$。

当然，领养研究也有其自身的局限性，如**选择性安置**（领养机构可能将孩子安置在与亲生父母相似的家庭中）、领养家庭环境范围可能受限、以及产前环境的潜在影响等 [@problem_id:5045638]。这再次提醒我们，没有任何一种研究设计是完美的。对[复杂性状](@entry_id:265688)的遗传结构最稳健的理解来自于整合多种设计（双生子、领养、大家系等）的证据。
