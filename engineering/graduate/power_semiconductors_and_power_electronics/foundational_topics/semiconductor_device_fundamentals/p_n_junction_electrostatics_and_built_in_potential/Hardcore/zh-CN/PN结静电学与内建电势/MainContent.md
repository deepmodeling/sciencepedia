## 引言
p-n结是现代半导体电子学的基石，从最简单的二[极管](@entry_id:909477)到复杂的[集成电路](@entry_id:265543)，其核心功能都源于两种不同掺杂类型半导体接触界面处的独特物理性质。理解其内部的静电学行为——即[空间电荷](@entry_id:199907)如何分布、电场如何形成、以及一个内建的电势垒为何会存在——是掌握[半导体器件物理](@entry_id:191639)的关键。然而，对于初学者而言，[内建电势](@entry_id:137446)的起源、其不可测量的特性以及理想模型与现实器件之间的差异往往是知识上的难点。

本文旨在系统性地阐明p-n结的[静电学](@entry_id:140489)。在第一章“原理与机制”中，我们将从[热力学平衡](@entry_id:141660)出发，推导内建电势的表达式，并介绍强大的耗尽近似模型，同时探讨影响实际器件的非理想效应。随后，在“应用与交叉学科联系”一章中，我们将展示这些静电学原理如何决定器件的偏压特性、[击穿电压](@entry_id:265833)，以及其在功率电子、[光电子学](@entry_id:144180)和材料科学中的广泛应用。最后，“动手实践”部分将通过具体计算问题，巩固您对关键参数（如载流子浓度和耗尽宽度）的理解和应用能力。通过这些章节，读者将建立一个从第一性原理到实际应用的完整知识框架。

## 原理与机制

本章旨在从第一性原理出发，深入探讨 p-n 结的核心[静电学](@entry_id:140489)特性。我们将首先从热力学平衡的视角建立[内建电势](@entry_id:137446)的物理图像，然后阐述其产生机制，并介绍用于描述其静电特性的数学模型。最后，我们将讨论一些在实际功率半导体器件中必须考虑的非理想效应，例如掺杂补偿、[不完全电离](@entry_id:1126446)和[带隙](@entry_id:138445)变窄。

### 平衡的[热力学](@entry_id:172368)基础：恒定的[费米能](@entry_id:143977)级

在热力学平衡状态下，一个系统的任何部分之间都没有净的粒子流、能量流或电荷流。对于[半导体器件](@entry_id:192345)而言，这一宏观条件等价于一个更为深刻的微观准则：在整个连通的系统中，**[电化学势](@entry_id:141179)**（electrochemical potential）必须是一个空间常数。在半导体物理中，电子的[电化学势](@entry_id:141179)通常被称为**[费米能](@entry_id:143977)级**（Fermi level），记作 $E_F$。

因此，一个处于[热力学平衡](@entry_id:141660)状态的 p-n 结，其最基本的特征是具有一个贯穿整个器件、空间上恒定不变的[费米能](@entry_id:143977)级。即 $\nabla E_F = \mathbf{0}$。这一准则是一切[平衡态](@entry_id:270364)[静电学](@entry_id:140489)分析的基石。

从不[可逆热](@entry_id:1130995)力学的角度看，载流子输运过程中的[熵产生](@entry_id:141771)率与电化学势的梯度成正比。因此，$\nabla E_F = \mathbf{0}$ 的条件等价于[载流子输运](@entry_id:196072)所导致的熵产生为零。这正是电子子系统处于平衡状态的[热力学](@entry_id:172368)表述。任何偏离平衡的状态，例如施加电压或光照，都会导致[费米能](@entry_id:143977)级（或更广义的[准费米能级](@entry_id:1130433)）出现梯度，从而驱动电流并产生熵。

### [内建电势](@entry_id:137446)的起源与特性

当一块 p 型半导体和一块 n 型半导体接触形成 p-n 结时，两种材料中[载流子浓度](@entry_id:143028)存在巨大的差异。p 区的多数载流子是空穴，其浓度远高于 n 区；n 区的多数载流子是电子，其浓度远高于 p 区。这种巨大的浓度梯度会驱动一个[扩散过程](@entry_id:268015)：空穴从 p 区向 n 区扩散，电子从 n 区向 p 区扩散。

随着多数载流子的扩散，它们会离开原先所在的区域，留下不能移动的、被电离的掺杂原子。在 p 区一侧，扩散走的空穴留下了带负电的受主离子 ($N_A^-$)；在 n 区一侧，扩散走的电子留下了带正电的施主离子 ($N_D^+$)。这些固定的空间电荷在[冶金](@entry_id:158855)结（metallurgical junction）附近形成了一个区域，该区域几乎耗尽了可移动的载流子，因此被称为**[空间电荷区](@entry_id:136997)**（space-charge region, SCR），或更通俗地称为**[耗尽区](@entry_id:136997)**（depletion region）。

空间电荷区的存在，即 n 区一侧的净正电荷和 p 区一侧的净负电荷，会产生一个从 n 区指向 p 区的内建电场 $E(x)$。这个电场会对载流子施加一个漂移力，其方向与扩散趋势相反。例如，对于从 n 区向 p 区扩散的电子，内建电场会把它向 n 区拉回。

平衡最终在[漂移与扩散](@entry_id:148816)两种作用达到动态平衡时建立。在平衡状态下，对于电子和空穴这两种载流子，由电场驱动的**[漂移电流](@entry_id:192129)**（drift current）必须在每一点都精确地被由浓度梯度驱动的**[扩散电流](@entry_id:262070)**（diffusion current）所抵消。因此，在没有外部激励的情况下，电子和空穴的净电流密度各自为零：

$J_n(x) = J_{n, \text{drift}}(x) + J_{n, \text{diff}}(x) = 0$
$J_p(x) = J_{p, \text{drift}}(x) + J_{p, \text{diff}}(x) = 0$

这个“细致平衡”原则比总电流为零 ($J = J_n + J_p = 0$) 的要求更为严格，它是热力学平衡的标志。 

为了维持这个平衡，必须在空间电荷区两端建立一个[静电势](@entry_id:188370)差。这个[电势差](@entry_id:275724)被称为**[内建电势](@entry_id:137446)**（built-in potential），记作 $V_{bi}$。它是一个电势垒，阻止了多数载流子的进一步大规模扩散。

我们可以从两个等价的角度推导 $V_{bi}$ 的表达式。

**1. [热力学](@entry_id:172368)推导**

在平衡状态下，[费米能](@entry_id:143977)级 $E_F$ 恒定。在远离结区的[电中性](@entry_id:138647)区，[载流子浓度](@entry_id:143028)与[费米能](@entry_id:143977)级和本征能级 $E_i$ 相关。假设玻尔兹曼统计适用，我们有：
在 p 区[电中性](@entry_id:138647)区：$p_p \approx N_A = n_i \exp\left(\frac{E_{i,p} - E_F}{kT}\right)$
在 n 区[电中性](@entry_id:138647)区：$n_n \approx N_D = n_i \exp\left(\frac{E_F - E_{i,n}}{kT}\right)$

其中 $n_i$ 是本征载流子浓度，$k$ 是[玻尔兹曼常数](@entry_id:142384)，$T$ 是[绝对温度](@entry_id:144687)。本征能级 $E_i$ 与静电势 $\psi$ 的关系是 $E_i(x) = E_{i,ref} - q\psi(x)$，其中 $q$ 是元电荷。

通过重新整理上述两个关于 $E_F$ 的表达式，并令它们相等，可以得到 p 区和 n 区[电中性](@entry_id:138647)区之间的[静电势](@entry_id:188370)差。按照惯例，内建电势定义为 n 区相对于 p 区的电势抬高量，即 $V_{bi} \equiv \psi_n - \psi_p$。经过推导可得：

$qV_{bi} = E_{i,p} - E_{i,n} = kT \ln\left(\frac{N_A N_D}{n_i^2}\right)$

因此，内建电势为：
$$V_{bi} = \frac{kT}{q} \ln\left(\frac{N_A N_D}{n_i^2}\right)$$

这个公式表明，$V_{bi}$ 是由半导体材料（通过 $n_i$）、[掺杂浓度](@entry_id:272646)（$N_A$ 和 $N_D$）以及温度（$T$）共同决定的一个内禀属性。在一个一维模型中，它与结的面积或几何形状无关。这是因为 $V_{bi}$ 是一个由局部[载流子密度](@entry_id:143028)决定的“强度”性质，而非依赖于总电荷量的“广延”性质。 

**2. 动力学推导**

我们也可以从漂移-[扩散平衡](@entry_id:150874)的条件出发。以空穴为例，其一维电流密度方程为：
$J_p(x) = q\mu_p p(x) E(x) - qD_p \frac{dp(x)}{dx}$

在平衡时 $J_p(x) = 0$，可得电场表达式：
$E(x) = \frac{D_p}{\mu_p} \frac{1}{p(x)} \frac{dp(x)}{dx}$

根据爱因斯坦关系式，$\frac{D_p}{\mu_p} = \frac{kT}{q}$，代入上式得：
$E(x) = \frac{kT}{q} \frac{1}{p(x)} \frac{dp(x)}{dx}$

将电场 $E(x) = -\frac{d\psi(x)}{dx}$ 在整个[耗尽区](@entry_id:136997)从 p 侧边界（$x_p$）积分到 n 侧边界（$x_n$），即可得到内建电势 $V_{bi} = \psi(x_n) - \psi(x_p)$：
$V_{bi} = - \int_{x_p}^{x_n} E(x) dx = - \int_{p_p}^{p_n} \frac{kT}{q} \frac{dp}{p} = \frac{kT}{q} \ln\left(\frac{p_p}{p_n}\right)$

在[电中性](@entry_id:138647)区，我们有 $p_p \approx N_A$，并且根据[质量作用定律](@entry_id:916274) $p_n = n_i^2 / n_n \approx n_i^2 / N_D$。代入后，我们得到了与[热力学](@entry_id:172368)推导完全相同的结果：
$$V_{bi} = \frac{kT}{q} \ln\left(\frac{N_A N_D}{n_i^2}\right)$$
例如，对于一个在 $300\,\mathrm{K}$ 的硅 p-n 结，若 $N_A = 2 \times 10^{17}\,\mathrm{cm}^{-3}$，$N_D = 5 \times 10^{16}\,\mathrm{cm}^{-3}$，且 $n_i = 1.0 \times 10^{10}\,\mathrm{cm}^{-3}$，其内建电势约为 $0.83\,\mathrm{V}$。

### [静电学](@entry_id:140489)建模：耗尽近似

为了求解 p-n 结内部的电场和电势分布，我们需要求解泊松方程：
$$\frac{d^2\psi(x)}{dx^2} = -\frac{\rho(x)}{\varepsilon_s}$$
其中 $\rho(x)$ 是净[空间电荷](@entry_id:199907)密度，$\varepsilon_s$ 是半导体的介[电常数](@entry_id:272823)。$\rho(x)$ 的精确表达式是 $\rho(x) = q(N_D^+ - N_A^- + p(x) - n(x))$，这使得泊松方程成为一个难以解析求解的[非线性微分方程](@entry_id:175929)。

为了简化问题，我们引入一个非常有效且广泛使用的模型——**耗尽近似**（depletion approximation）。其核心假设是 ：
1.  在[空间电荷区](@entry_id:136997)（$-x_p \lt x \lt x_n$）内，可动载流子（电子和空穴）被完全耗尽，因此[电荷密度](@entry_id:144672)完全由固定的、被电离的施主和受主原子决定。
    $$\rho(x) = \begin{cases} -qN_A,  \text{当 } -x_p  x  0 \\ +qN_D,  \text{当 } 0  x  x_n \end{cases}$$
2.  在[空间电荷区](@entry_id:136997)之外，即**[准中性](@entry_id:197419)区**（quasi-neutral regions），半导体是完全[电中性](@entry_id:138647)的，即 $\rho(x) = 0$。

在耗尽近似下，泊松方程变得分段可解。为了唯一地确定电场 $E(x) = -d\psi/dx$ 和电势 $\psi(x)$ 的分布，以及耗尽宽度 $x_p$ 和 $x_n$，我们需要一组完整且自洽的边界条件 ：
*   **[耗尽区](@entry_id:136997)边缘的电场**：由于[准中性](@entry_id:197419)区是[电中性](@entry_id:138647)的，其内部电场为零。因此，在耗尽区的边界上，电场必须为零：$E(-x_p) = 0$ 和 $E(x_n) = 0$。
*   **冶金结处的连续性**：在 $x=0$ 的冶金结处，由于不存在无限薄的电荷片，电场 $E(x)$ 必须是连续的。同样，电势 $\psi(x)$ 也必须是连续的，否则将对应无穷大的电场。
*   **总电势降**：整个耗尽区上的总电势降必须等于内建电势（在有外加偏压 $V_a$ 时，则为 $V_{bi} - V_a$）：$\psi(x_n) - \psi(-x_p) = V_{bi}$。

一个特别重要的推论来自于电场在 $x=0$ 处的连续性条件。通过对泊松方程积分得到电场，并令 $E(0^-)=E(0^+)$，可以导出**电荷中性条件**：
$$N_A x_p = N_D x_n$$
这个条件表明，p 侧[耗尽区](@entry_id:136997)中的总负电荷（每单位面积）必须等于 n 侧[耗尽区](@entry_id:136997)中的总正电荷（每单位面积）。这个全局性的[电荷平衡](@entry_id:1122292)关系是求解耗尽宽度 $x_p$ 和 $x_n$ 的关键方程之一。

### 内建电势的可测量性之谜

既然 p-n 结内部存在一个不可忽略的[电势差](@entry_id:275724) $V_{bi}$（通常为零点几伏特），一个自然的问题是：为什么我们不能用一个普通的直流电压表直接测出这个电压？

答案在于，任何测量都必须形成一个闭合回路。当我们将一个高阻抗电压表的两个金属探针分别接触到 p-n 结的 p 区和 n 区时，我们构成的完整系统是：金属探针1 → p区 → n区 → 金属探针2 → 电压表内部电路 → 金属探针1。

在热力学平衡下，这个闭合回路的[费米能](@entry_id:143977)级 $E_F$ 处处相等。电压表的设计原理决定了它测量的是其两个输入端之间的电化学势（即[费米能](@entry_id:143977)级）之差。由于整个回路的 $E_F$ 是恒定的，电压表两端的 $E_F$ 也相同，因此其读数必然为零。

从[静电势](@entry_id:188370)的角度看，这个零读数是由于回路中各个电势降的精确抵消所致。当金属探针与半导体接触时，为了对齐[费米能](@entry_id:143977)级，会在接触面形成一个**接触电势**（contact potential），其大小等于[金属与半导体](@entry_id:269023)功函数之差除以电荷 $q$。在一个由相同金属探针构成的测量回路中，p-n 结内部的[内建电势](@entry_id:137446) $V_{bi}$，会与两个[金属-半导体接触](@entry_id:144862)面上形成的接触电势之和精确地相互抵消。根据[基尔霍夫电压定律](@entry_id:276614)，沿任何闭合路径的[静电势](@entry_id:188370)差之和必须为零。因此，电压表测得的[电势差](@entry_id:275724)为零。

这个现象深刻地揭示了 $V_{bi}$ 是一个内建的、用于维持平衡的电势，它无法通过简单的外部宏观测量来提取，因为它总是被接触电势所补偿。

### 高级课题与非理想效应

在实际的[功率半导体](@entry_id:1130060)器件中，尤其是在高掺杂或宽禁带材料中，上述理想模型需要修正。

#### 掺杂补偿

真实的[半导体制备](@entry_id:187383)过程中，除了有意引入的施主或[受主杂质](@entry_id:157874)外，通常还存在背景（非故意）的相反类型的杂质。这种同一区域内同时存在施主和受主的现象称为**掺杂补偿**（dopant compensation）。

在补偿型半导体中，[电中性](@entry_id:138647)区的多数[载流子浓度](@entry_id:143028)由**[净掺杂浓度](@entry_id:1128552)**决定。假设杂质完全电离：
*   在 n 型区，净施主浓度为 $N_{D,\text{net}} = N_D - N_A^{\text{bg}}$，其中 $N_D$ 是施主浓度，$N_A^{\text{bg}}$ 是背景受主浓度。
*   在 p 型区，净受主浓度为 $N_{A,\text{net}} = N_A - N_D^{\text{bg}}$，其中 $N_A$ 是受主浓度，$N_D^{\text{bg}}$ 是背景施主浓度。

相应地，[耗尽区](@entry_id:136997)内的空间电荷密度也由[净掺杂浓度](@entry_id:1128552)决定：n 侧为 $\rho = +q N_{D,\text{net}}$，p 侧为 $\rho = -q N_{A,\text{net}}$。计算内建电势时，必须使用这些[净掺杂浓度](@entry_id:1128552)。例如，一个硅结的 n 区掺杂为 $N_D = 5.0 \times 10^{16}\,\mathrm{cm^{-3}}$ 和 $N_A^{\text{bg}} = 8.0 \times 10^{15}\,\mathrm{cm^{-3}}$，p 区掺杂为 $N_A = 2.0 \times 10^{16}\,\mathrm{cm^{-3}}$ 和 $N_D^{\text{bg}} = 1.0 \times 10^{16}\,\mathrm{cm^{-3}}$。其[净掺杂浓度](@entry_id:1128552)分别为 $N_{D,\text{net}} = 4.2 \times 10^{16}\,\mathrm{cm^{-3}}$ 和 $N_{A,\text{net}} = 1.0 \times 10^{16}\,\mathrm{cm^{-3}}$。在 $300\,\mathrm{K}$ 计算出的内建电势约为 $0.752\,\mathrm{V}$。

#### [不完全电离](@entry_id:1126446)

**完全电离**（complete ionization）的假设——即所有掺杂原子都贡献了一个载流子（$N_D^+ \approx N_D, N_A^- \approx N_A$）——并非总是成立。掺杂原子的电离程度取决于其**激活能**（activation energy，$E_C - E_D$ 或 $E_A - E_V$）与热能 $kT$ 的相对大小。

*   对于硅（Si）中的常见浅能级掺杂（如磷、硼），激活能约为 $45\,\mathrm{meV}$。在室温（$T=300\,\mathrm{K}$，$kT \approx 26\,\mathrm{meV}$），热能足以使绝大多数掺杂原子电离，因此完全电离是很好的近似。例如，对于 $N_D = 10^{16}\,\mathrm{cm^{-3}}$ 的硅，电离率超过 $99\%$。
*   然而，对于宽禁带半导体，如碳化硅（SiC）和氮化镓（GaN），许多常用掺杂剂具有更深的能级（更大的激活能）。例如，4H-SiC 中铝（Al）受主的激活能约为 $0.20\,\mathrm{eV}$。在室温下，热能 $kT$ 远小于此激活能，导致**[不完全电离](@entry_id:1126446)**（incomplete ionization）。对于 $N_A = 10^{16}\,\mathrm{cm^{-3}}$ 的 Al 掺杂 4H-SiC，在 $300\,\mathrm{K}$ 时只有约 $40\%$ 的受主被电离。这意味着完全电离的假设会严重高估自由空穴的浓度。

升高温度可以显著提高电离率。在上述 SiC 例子中，当温度升高到 $600\,\mathrm{K}$ 时，Al 受主的电离率可接近 $97\%$，此时完全电离成为一个较好的近似。

忽略[不完全电离](@entry_id:1126446)会对内建电势的计算产生显著影响。由于 $V_{bi}$ 的表达式对数项中包含多数载流子浓度（$n_n p_p$），而实际的多数载流子浓度因[不完全电离](@entry_id:1126446)而低于[掺杂浓度](@entry_id:272646)，因此，若错误地采用完全电离假设，将会**高估**内建电势的真实值。

#### [带隙](@entry_id:138445)变窄

当[半导体掺杂](@entry_id:157714)浓度非常高时（通常大于 $10^{18}\,\mathrm{cm^{-3}}$），掺杂原子间的相互作用、自由载流子的屏蔽效应以及[晶格](@entry_id:148274)的无序性等多种物理机制会导致导带底 $E_c$ 向下移动，价带顶 $E_v$ 向上移动。这种现象称为**[带隙](@entry_id:138445)变窄**（Bandgap Narrowing, BGN）。有效[带隙](@entry_id:138445) $E_g^{\text{eff}}$ 小于其本征值 $E_g$，即 $E_g^{\text{eff}} = E_g - \Delta E_g$。

[带隙](@entry_id:138445)变窄直接影响了[本征载流子浓度](@entry_id:144530)。由于 $n_i^2 \propto \exp(-E_g/kT)$，有效[带隙](@entry_id:138445)的减小会使[有效本征载流子浓度](@entry_id:1124181) $n_{ie}$ 指数级增加：
$$n_{ie} = n_{i0} \exp\left(\frac{\Delta E_g}{2kT}\right)$$
其中 $n_{i0}$ 是未考虑 BGN 时的[本征载流子浓度](@entry_id:144530)。

在计算[内建电势](@entry_id:137446)时，必须使用这个增大了的 $n_{ie}$。将上式代入 $V_{bi}$ 的标准公式中，可以得到：
$$V_{bi} = \frac{kT}{q} \ln\left(\frac{N_A N_D}{n_{ie}^2}\right) = \frac{kT}{q} \ln\left(\frac{N_A N_D}{n_{i0}^2}\right) - \frac{kT}{q} \ln\left(\exp\left(\frac{\Delta E_g}{kT}\right)\right)$$
$$V_{bi} = V_{bi,0} - \frac{\Delta E_g}{q}$$
其中 $V_{bi,0}$ 是忽略 BGN 时的[内建电势](@entry_id:137446)。这个结果表明，[带隙](@entry_id:138445)变窄会使内建电势减小，减小量恰好等于[带隙](@entry_id:138445)变窄的能量值（以伏特为单位）。例如，对于一个在 $300\,\mathrm{K}$ 的对称掺杂（$N_d = N_a = 10^{18}\,\mathrm{cm^{-3}}$）硅结，如果不考虑 BGN，其 $V_{bi}$ 约为 $0.95\,\mathrm{V}$。若考虑到 $\Delta E_g \approx 0.12\,\mathrm{eV}$ 的[带隙](@entry_id:138445)变窄，其 $n_i$ 会增加约一个数量级，而 $V_{bi}$ 则会降低 $0.12\,\mathrm{V}$，降至约 $0.83\,\mathrm{V}$。

总之，对 p-n 结静电学的深刻理解，不仅需要掌握理想模型，更需要认识到这些二阶效应在现代高性能功率器件设计与分析中的关键作用。
