## 引言
在[代数拓扑学](@entry_id:138192)中，同调群为我们提供了一种强大的代数工具来研究和区分拓扑空间的形状。一个核心问题是，当我们用不交并这样的基本操作来组合空间时，其同调[不变量](@entry_id:148850)会发生什么变化？本文旨在解决这一问题，揭示一个简洁而深刻的原理：不交并空间的同调是其各个组分空间同调的直和。通过本文的学习，读者将首先在【原理与机制】一章中，从链复合物的层面严格理解这一可加性原理的推导过程及其对贝蒂数和[诱导同态](@entry_id:266478)的影响。接着，在【应用与交叉学科联系】一章中，我们将探索该原理如何作为一种“拓扑指纹”来区分空间，并了解其在[流形理论](@entry_id:263722)、[几何分析](@entry_id:157700)等高等主题中的应用。最后，【动手实践】部分将提供具体练习，帮助读者将理论知识转化为解决问题的能力。

## 原理与机制

在代数拓扑学中，同调论为我们提供了一套强大的代数[不变量](@entry_id:148850)，用以区分不同的[拓扑空间](@entry_id:155056)。一个核心问题是，当我们通过基本操作（如不交并、乘积、[商空间](@entry_id:274314)）来构造新空间时，其同调群会如何变化。本章将深入探讨最基本的一种构造——不交并（disjoint union）——及其在同调论中的表现。我们将揭示一个简洁而深刻的原理：一个不交并空间的同调群，是其各个组分空间[同调群的直和](@entry_id:263088)。这一原理不仅具有理论上的优雅性，也为实际计算提供了极大的便利。

### 基本可加性原理

不交并是拓扑学中最简单的空间组合方式之一。给定一族[拓扑空间](@entry_id:155056) $\{X_i\}_{i \in I}$，它们的不交并 $X = \bigsqcup_{i \in I} X_i$ 是集合的并集，其拓扑结构规定一个[子集](@entry_id:261956)是开集，当且仅当它与每个 $X_i$ 的交集在 $X_i$ 的原始拓扑中都是开集。在这种拓扑下，每个 $X_i$ 都成为 $X$ 的一个既开又闭的[子集](@entry_id:261956)，彼此分离。

同调论中的一个基本结论是，同调群在这种[空间分解](@entry_id:755142)下表现出完美的**可加性**。也就是说，对于任意维数 $n \geq 0$，不交并空间 $X$ 的 $n$ 阶同调群（以整数 $\mathbb{Z}$ 为系数）同构于其所有组分空间 $X_i$ 的 $n$ 阶[同调群的直和](@entry_id:263088)（direct sum）。

$$
H_n(X) = H_n\left(\bigsqcup_{i \in I} X_i\right) \cong \bigoplus_{i \in I} H_n(X_i)
$$

这个同构关系是自然的。该原理构成了本章的基石。在接下来的小节中，我们将首先从奇异链复合物的层面严格推导这一结论，然后探讨其在计算贝蒂数（Betti numbers）、理解[诱导同态](@entry_id:266478)以及处理简约同调和[相对同调](@entry_id:159348)等变体时的深刻应用。

### 从链复合物推导

为了理解同调群的可加性，我们必须回到其定义的基础——奇异链复合物。回忆一下，$n$ 阶奇异链群 $C_n(Z)$ 是由所有从标准 $n$-单纯形 $\Delta^n$ 到空间 $Z$ 的连续映射（即奇异 $n$-单纯形）生成的自由[阿贝尔群](@entry_id:150284)。

现在，考虑一个由两个空间 $X$ 和 $Y$ 构成的不交并 $X \sqcup Y$。一个奇异 $n$-单纯形是一个[连续映射](@entry_id:153855) $\sigma: \Delta^n \to X \sqcup Y$。由于标准单纯形 $\Delta^n$ 是一个[连通空间](@entry_id:156017)，而 $X$ 和 $Y$ 是 $X \sqcup Y$ 中两个不相交的开[闭子集](@entry_id:155133)，因此 $\sigma$ 的像必定完全落在 $X$ 中，或者完全落在 $Y$ 中。换言之，任何一个 $X \sqcup Y$ 中的[奇异单纯形](@entry_id:265569)，要么是一个 $X$ 中的[奇异单纯形](@entry_id:265569)，要么是一个 $Y$ 中的[奇异单纯形](@entry_id:265569)。

这意味着，$C_n(X \sqcup Y)$ 的生成元集合恰好是 $C_n(X)$ 的生成元集合与 $C_n(Y)$ 的生成元集合的不交并。因此，在[阿贝尔群](@entry_id:150284)的层面上，我们得到一个自然的同构：

$$
C_n(X \sqcup Y) \cong C_n(X) \oplus C_n(Y)
$$

接下来我们考察[边界算子](@entry_id:160216) $\partial_n: C_n(Z) \to C_{n-1}(Z)$。如果一个[奇异单纯形](@entry_id:265569) $\sigma$ 的像在 $X$ 中，那么它的所有面（faces）的像也必然在 $X$ 中。因此，[边界算子](@entry_id:160216) $\partial_n^X$ 将 $C_n(X)$ 映射到 $C_{n-1}(X)$。同理，$\partial_n^Y$ 将 $C_n(Y)$ 映射到 $C_{n-1}(Y)$。综合来看，$X \sqcup Y$ 上的[边界算子](@entry_id:160216) $\partial_n^{X \sqcup Y}$ 恰好是两个[边界算子](@entry_id:160216)的直和：

$$
\partial_n^{X \sqcup Y} = \partial_n^X \oplus \partial_n^Y
$$

这意味着 $X \sqcup Y$ 的奇异链复合物 $C_*(X \sqcup Y)$ 同构于 $X$ 和 $Y$ 的链复合物的直和 $C_*(X) \oplus C_*(Y)$。

代数上有一个一般性结论：链复合物[直和](@entry_id:156782)的同调群是各自[同调群的直和](@entry_id:263088)。具体来说，核（cycles）和像（boundaries）都相应地分解：
$Z_n(X \sqcup Y) = \ker(\partial_n^{X \sqcup Y}) = \ker(\partial_n^X) \oplus \ker(\partial_n^Y) = Z_n(X) \oplus Z_n(Y)$
$B_n(X \sqcup Y) = \text{im}(\partial_{n+1}^{X \sqcup Y}) = \text{im}(\partial_{n+1}^X) \oplus \text{im}(\partial_{n+1}^Y) = B_n(X) \oplus B_n(Y)$

因此，商群（即同调群）也相应地分解：
$$
H_n(X \sqcup Y) = \frac{Z_n(X \sqcup Y)}{B_n(X \sqcup Y)} \cong \frac{Z_n(X) \oplus Z_n(Y)}{B_n(X) \oplus B_n(Y)} \cong \frac{Z_n(X)}{B_n(X)} \oplus \frac{Z_n(Y)}{B_n(Y)} = H_n(X) \oplus H_n(Y)
$$
这就严格证明了同调的可加性原理。

### 特例：[零阶同调群](@entry_id:261808) $H_0$

对于 $n=0$ 的情况，同调的可加性原理有一个特别直观的拓扑解释。一个基本结论是，任意拓扑空间 $Z$ 的[零阶同调群](@entry_id:261808) $H_0(Z; \mathbb{Z})$ 是一个自由[阿贝尔群](@entry_id:150284)，其秩（rank）等于 $Z$ 的[路径连通分支](@entry_id:275432)（path-connected components）的数量。如果 $Z$ 有 $k$ 个[路径连通分支](@entry_id:275432)，则 $H_0(Z) \cong \mathbb{Z}^k$。

现在，考虑一个由 $k$ 个非空且[路径连通](@entry_id:148704)的空间 $X_1, X_2, \dots, X_k$ 构成的不交并 $X = \bigsqcup_{i=1}^k X_i$。由于任意两个不同的 $X_i$ 和 $X_j$ 之间没有路径相连，因此 $X$ 的[路径连通分支](@entry_id:275432)恰好就是这 $k$ 个 $X_i$。所以，$X$ 恰有 $k$ 个[路径连通分支](@entry_id:275432)。

根据上述结论，$H_0(X)$ 的秩就是 $k$。这与我们的可加性原理完全吻合：由于每个 $X_i$ 都是[路径连通](@entry_id:148704)的，所以 $H_0(X_i) \cong \mathbb{Z}$。因此，
$$
H_0(X) \cong \bigoplus_{i=1}^k H_0(X_i) \cong \bigoplus_{i=1}^k \mathbb{Z} \cong \mathbb{Z}^k
$$
例如，如果空间 $X$ 是三个圆的不交并，而空间 $Y$ 是两条闭线段的不交并，那么 $X$ 有3个[路径连通分支](@entry_id:275432)，$Y$ 有2个。它们的不交并 $X \sqcup Y$ 就有 $3+2=5$ 个[路径连通分支](@entry_id:275432)。因此，它的[零阶同调群](@entry_id:261808) $H_0(X \sqcup Y) \cong \mathbb{Z}^5$。

### 计算应用：贝蒂数

可加性原理最直接的应用之一是计算复杂空间的贝蒂数。空间 $Z$ 的 $k$ 阶贝蒂数 $b_k(Z)$ 定义为其 $k$ 阶同调群 $H_k(Z; \mathbb{Z})$ 的秩，即其自由部分的秩。由于[阿贝尔群](@entry_id:150284)直和的秩等于各分量秩的和，我们立即得到一个强大的计算法则：

$$
b_k\left(\bigsqcup_{i \in I} X_i\right) = \sum_{i \in I} b_k(X_i)
$$

这意味着，只要我们知道构成一个不交并空间的各个组分空间的[贝蒂数](@entry_id:153109)，我们就可以通过简单的加法求出整个空间的贝蒂数。

让我们通过一个具体的例子来展示其威力。假设一个空间 $M$ 是由以下部分构成的不交并：
- 2 个 2-球面 ($S^2$)
- 3 个实射影平面 ($\mathbb{R}P^2$)
- 1 个[克莱因瓶](@entry_id:149661) ($K$)
- 4 个圆 ($S^1$)
- 1 个 3-维环面 ($T^3$)

首先，我们需要知道各个组分空间的[贝蒂数](@entry_id:153109)（对于 $k=0, 1, 2, 3$）：
- $S^2$: $(b_0, b_1, b_2, b_3) = (1, 0, 1, 0)$
- $\mathbb{R}P^2$: $(b_0, b_1, b_2, b_3) = (1, 0, 0, 0)$ (注意 $H_1(\mathbb{R}P^2; \mathbb{Z}) \cong \mathbb{Z}_2$ 是[挠群](@entry_id:144787)，秩为0)
- $K$: $(b_0, b_1, b_2, b_3) = (1, 1, 0, 0)$ (注意 $H_1(K; \mathbb{Z}) \cong \mathbb{Z} \oplus \mathbb{Z}_2$，秩为1)
- $S^1$: $(b_0, b_1, b_2, b_3) = (1, 1, 0, 0)$
- $T^3$: $(b_0, b_1, b_2, b_3) = (\binom{3}{0}, \binom{3}{1}, \binom{3}{2}, \binom{3}{3}) = (1, 3, 3, 1)$

现在，我们可以通过对每阶贝蒂数求和来计算 $b_k(M)$：
- $b_0(M) = 2 \cdot b_0(S^2) + 3 \cdot b_0(\mathbb{R}P^2) + 1 \cdot b_0(K) + 4 \cdot b_0(S^1) + 1 \cdot b_0(T^3) = 2(1) + 3(1) + 1(1) + 4(1) + 1(1) = 11$。这正如预期，等于[路径连通分支](@entry_id:275432)的总数。
- $b_1(M) = 2(0) + 3(0) + 1(1) + 4(1) + 1(3) = 0 + 0 + 1 + 4 + 3 = 8$。
- $b_2(M) = 2(1) + 3(0) + 1(0) + 4(0) + 1(3) = 2 + 0 + 0 + 0 + 3 = 5$。
- $b_3(M) = 2(0) + 3(0) + 1(0) + 4(0) + 1(1) = 0 + 0 + 0 + 0 + 1 = 1$。

因此，$M$ 的前四阶贝蒂数序列为 $(11, 8, 5, 1)$。这个例子清楚地表明，可加性原理能将一个看似复杂的空间的同调计算问题，分解为对若干个更简单、更标准空间的分析。

### [函子性](@entry_id:150069)：[诱导同态](@entry_id:266478)

[同调的函子性](@entry_id:269512)（functoriality）意味着任何连续映射 $f: Z_1 \to Z_2$ 都会诱导一个[群同态](@entry_id:140603) $f_*: H_n(Z_1) \to H_n(Z_2)$。当涉及不交并时，理解[诱导同态](@entry_id:266478)如何与[直和](@entry_id:156782)结构相互作用至关重要。

**从组分空间到不交并的映射**

考虑一个自然的包含映射 $i_A: A \to A \sqcup B$。这个映射诱导的同态 $i_{A*}: H_n(A) \to H_n(A \sqcup B)$ 是什么样子的？利用同构 $H_n(A \sqcup B) \cong H_n(A) \oplus H_n(B)$，我们可以精确地刻画 $i_{A*}$ 的行为。在链的层面上，$i_A$ 将 $A$ 中的链 $c$ 映射为 $A \sqcup B$ 中的链 $(c, 0)$。这个结构在传递到同调群时保持不变。因此，对于一个同调类 $[z] \in H_n(A)$，其像为：
$$
i_{A*}([z]) = ([z], 0) \in H_n(A) \oplus H_n(B)
$$
这[实质](@entry_id:149406)上是到直和的第一个分量的自然包含同态。从这个描述中我们可以得出两个重要结论：
1.  **$i_{A*}$ 总是单射（injective）。** 它的核是平凡的，因为 $([z], 0)$ 为零元当且仅当 $[z]$ 为零元。我们也可以通过构造一个[左逆](@entry_id:153819)来证明这一点：定义一个收缩映射 $r: A \sqcup B \to A$，它在 $A$ 上是恒等映射，并将整个 $B$ 映射到 $A$ 中的某个点。那么 $r \circ i_A = \text{id}_A$，由[函子性](@entry_id:150069)可知 $r_* \circ i_{A*} = \text{id}_{H_n(A)}$，这证明了 $i_{A*}$ 是单射。
2.  **$i_{A*}$ 通常不是满射（surjective）。** 它的像仅为 $H_n(A) \oplus \{0\}$。只要 $H_n(B)$ 不是平凡群，这个像就是 $H_n(A) \oplus H_n(B)$ 的一个[真子群](@entry_id:141915)。

**从不交并到其他空间的映射**

现在考虑从不交并出发的映射。根据不交并的泛性质，一个映射 $f: A \sqcup B \to Z$ 完全由它在 $A$ 和 $B$ 上的限制 $f|_A: A \to Z$ 和 $f|_B: B \to Z$ 唯一确定。诱导的同态 $f_*: H_n(A \sqcup B) \to H_n(Z)$ 同样遵循这一分解。在同构 $H_n(A \sqcup B) \cong H_n(A) \oplus H_n(B)$ 下，对于一个元素 $([a], [b])$，其像为：
$$
f_*([a], [b]) = (f|_A)_*([a]) + (f|_B)_*([b])
$$
让我们通过一个例子来具体说明。令 $X=S^1$，$Y=T^2=S^1 \times S^1$，不交并为 $W = X \sqcup Y$。我们有 $H_1(W) \cong H_1(X) \oplus H_1(Y) \cong \mathbb{Z} \oplus (\mathbb{Z} \oplus \mathbb{Z}) \cong \mathbb{Z}^3$。一个元素可以记为 $(k, l, m)$。考虑两个到 $X=S^1$ 的映射：
1.  $p: W \to X$，定义为 $p|_X = \text{id}_X$，$p|_Y$ 为将整个 $T^2$ 映射到 $X$ 中某一点的常映射。常映射在正维数同调上诱导零同态，所以 $(p|_Y)_* = 0$。因此 $p_*(k, l, m) = (\text{id}_X)_*(k) + (p|_Y)_*(l, m) = k+0 = k$。
2.  $q: W \to X$，定义为 $q|_X = \text{id}_X$，$q|_Y = \text{proj}_1: T^2 \to S^1$ (到第一个 $S^1$ 因子的投影)。$H_1(T^2)$ 的生成元对应于两个 $S^1$ 因子。[投影映射](@entry_id:153398) $(\text{proj}_1)_*$ 将第一个生成元映为 $H_1(S^1)$ 的生成元，将第二个生成元映为零。因此 $(\text{proj}_1)_*(l, m) = l$。所以 $q_*(k, l, m) = (\text{id}_X)_*(k) + (\text{proj}_1)_*(l, m) = k+l$。

这个例子清晰地展示了[诱导同态](@entry_id:266478)如何分量式地作用于[直和分解](@entry_id:263004)，将拓扑映射的几何行为转化为代数上的线性操作。

### 扩展与变体

同调的可加性原理不仅适用于标准的[奇异同调](@entry_id:158380)，也适用于其相关的变体，如简约同调和[相对同调](@entry_id:159348)。

**简约同调**
简约同调群 $\tilde{H}_n(Z)$ 与普通同调群的关系是：对于 $n \geq 1$，$\tilde{H}_n(Z) = H_n(Z)$；而对于 $n=0$，$H_0(Z) \cong \tilde{H}_0(Z) \oplus \mathbb{Z}$ (假设 $Z$ 非空)。

对于不交并 $X \sqcup Y$，可加性原理对于 $n \geq 1$ 的简约同调群直接成立：
$$
\tilde{H}_n(X \sqcup Y) \cong \tilde{H}_n(X) \oplus \tilde{H}_n(Y) \quad \text{for } n \ge 1
$$
然而，对于 $\tilde{H}_0$ 需要特别注意。假设 $X$ 和 $Y$ 都是非空[路径连通空间](@entry_id:152443)，则 $X \sqcup Y$ 有2个[路径连通分支](@entry_id:275432)。我们知道 $H_0(X \sqcup Y) \cong \mathbb{Z} \oplus \mathbb{Z}$。利用关系式，我们有 $\mathbb{Z} \oplus \mathbb{Z} \cong \tilde{H}_0(X \sqcup Y) \oplus \mathbb{Z}$。通过比较自由[阿贝尔群](@entry_id:150284)的秩，我们得出 $\text{rank}(\tilde{H}_0(X \sqcup Y)) = 1$，即 $\tilde{H}_0(X \sqcup Y) \cong \mathbb{Z}$。更一般地，若空间 $Z$ 是 $k$ 个非空空间的不交并，则 $\tilde{H}_0(Z) \cong \mathbb{Z}^{k-1}$。

这个性质可以用来计算简约[欧拉示性数](@entry_id:152513) $\tilde{\chi}(Z) = \sum_{n=0}^\infty (-1)^n \tilde{\beta}_n(Z)$，其中 $\tilde{\beta}_n(Z)$ 是 $\tilde{H}_n(Z)$ 的秩。例如，对于 $X = S^2 \sqcup S^2 \sqcup T^2$，这是一个由3个组分构成的空间。
- $\tilde{\beta}_0(X) = 3 - 1 = 2$。
- $\tilde{\beta}_1(X) = \tilde{\beta}_1(S^2) + \tilde{\beta}_1(S^2) + \tilde{\beta}_1(T^2) = 0 + 0 + 2 = 2$。
- $\tilde{\beta}_2(X) = \tilde{\beta}_2(S^2) + \tilde{\beta}_2(S^2) + \tilde{\beta}_2(T^2) = 1 + 1 + 1 = 3$。
- 其他 $\tilde{\beta}_n(X)$ 均为0。
因此，$\tilde{\chi}(X) = \tilde{\beta}_0 - \tilde{\beta}_1 + \tilde{\beta}_2 = 2 - 2 + 3 = 3$。

**[相对同调](@entry_id:159348)**
可加性原理同样优雅地扩展到[相对同调](@entry_id:159348)。给定一对空间 $(X, A)$ 和另一对 $(Y, B)$，它们的不交并可以自然地看作一对空间 $(X \sqcup Y, A \sqcup B)$。与绝对同调的推导完全类似，相对链复合物 $C_*(X \sqcup Y, A \sqcup B) = C_*(X \sqcup Y) / C_*(A \sqcup B)$ 也表现出[直和分解](@entry_id:263004)的性质。最终我们得到：
$$
H_n(X \sqcup Y, A \sqcup B) \cong H_n(X, A) \oplus H_n(Y, B)
$$
这个性质允许我们将复杂的[相对同调](@entry_id:159348)计算分解为更简单的部分。例如，计算 $H_n(S^2 \sqcup T^2, \{p_1, p_2\} \sqcup S^1)$ 的问题，可以分解为分别计算 $H_n(S^2, \{p_1, p_2\})$ 和 $H_n(T^2, S^1)$，然后将结果（包括秩和挠扑部分）相加。通过各自的长正合序列分析，可以分别求得这两对空间的[相对同调群](@entry_id:159711)，最终组合得到原问题的答案。

**与[Mayer-Vietoris序列](@entry_id:161286)的联系**
最后，值得一提的是，同调的可加性原理与另一个强大的计算工具——[Mayer-Vietoris序列](@entry_id:161286)——之间的关系。[Mayer-Vietoris序列](@entry_id:161286)适用于任何[开覆盖](@entry_id:140020) $X=A \cup B$。如果我们特别地考虑 $A$ 和 $B$ 是不交开集的情况（即 $X = A \sqcup B$），那么它们的交集 $A \cap B = \emptyset$。
[空集](@entry_id:261946)的各阶同调群都是平凡群，$H_n(\emptyset) = 0$。将此代入Mayer-Vietoris长正合序列：
$$
\dots \to H_n(A \cap B) \to H_n(A) \oplus H_n(B) \to H_n(X) \to H_{n-1}(A \cap B) \to \dots
$$
序列变成了：
$$
\dots \to 0 \to H_n(A) \oplus H_n(B) \xrightarrow{f} H_n(X) \xrightarrow{g} 0 \to \dots
$$
根据[正合序列](@entry_id:151503)的定义，$\ker(g) = \text{im}(f)$。由于 $g$ 是到零群的映射，所以 $\ker(g) = H_n(X)$。由于f是从零群出发的映射的前一个映射，所以f是单射。结合起来，$f: H_n(A) \oplus H_n(B) \to H_n(X)$ 是一个同构。
因此，在这种“退化”的情况下，[Mayer-Vietoris序列](@entry_id:161286)恰好给出了我们已经建立的可加性原理 $H_n(X) \cong H_n(A) \oplus H_n(B)$。这不仅是对可加性原理的有趣验证，也加深了我们对这两个基本工具内在联系的理解。

综上所述，不交并的同调可加性是一个贯穿同调论多个方面的基本原理。从其在链复合物层面的根源，到在计算[贝蒂数](@entry_id:153109)、分析[诱导同态](@entry_id:266478)以及推广到相对和简约同调中的应用，它都展现了代数拓扑如何将空间的几何分解转化为代数上的[直和分解](@entry_id:263004)，是连接拓扑与代数的关键桥梁之一。
