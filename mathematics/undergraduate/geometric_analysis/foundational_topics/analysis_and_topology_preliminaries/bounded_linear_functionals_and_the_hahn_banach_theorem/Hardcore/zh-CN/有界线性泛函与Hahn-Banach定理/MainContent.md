## 引言
在[现代分析学](@entry_id:146248)，尤其是[泛函分析](@entry_id:146220)的宏伟殿堂中，[有界线性泛函](@entry_id:271069)与[Hahn-Banach定理](@entry_id:145406)扮演着基石般的角色。它们不仅是连接[代数结构](@entry_id:137052)与拓扑结构的桥梁，更是我们理解无限维空间几何形态的强大透镜。然而，初学者往往困惑于这些抽象概念的实际意义：我们为何关心泛函的“有界性”？一个定义在局部（[子空间](@entry_id:150286)）的函数，如何能保证其性质可以“无损”地延伸至全局（整个空间）？这正是本文旨在解决的核心知识鸿沟。

本文将带领读者踏上一段从基本原理到前沿应用的探索之旅，系统地揭示[有界线性泛函](@entry_id:271069)与[Hahn-Banach定理](@entry_id:145406)的奥秘。你将学到：

- **第一章：原理与机制** 将从[赋范空间](@entry_id:137032)和[线性泛函](@entry_id:276136)的定义出发，阐明连续性与有界性的[等价关系](@entry_id:138275)，并引入算子范数作为度量。随后，我们将深入剖析[Hahn-Banach定理](@entry_id:145406)的多种形式——从由[次线性泛函](@entry_id:143368)控制的解析扩张，到至关重要的[保范扩张](@entry_id:268703)，并探讨其深刻的几何解释，如[超平面分离定理](@entry_id:272999)。

- **第二章：应用与跨学科联系** 将展示[Hahn-Banach定理](@entry_id:145406)的巨大威力，看它如何保证对偶空间的“丰富性”，如何被用于刻画巴拿赫空间的结构（如[自反性](@entry_id:137262)），以及如何在测度论、最[优化理论](@entry_id:144639)乃至[偏微分方程](@entry_id:141332)等领域中，成为证明关键结果的利器。

- **第三章：动手实践** 将通过一系列精心设计的练习，将抽象理论付诸实践。你将亲手计算[泛函的范数](@entry_id:142833)，构造泛函的[保范扩张](@entry_id:268703)，并在具体的函数空间中应用这些工具，从而将理论知识内化为解决问题的能力。

通过这三个层次的递进学习，你将不仅掌握[Hahn-Banach定理](@entry_id:145406)的数学内容，更能深刻体会到它作为[泛函分析](@entry_id:146220)“第一定理”的强大力量和深远影响。

## 原理与机制

在介绍篇中，我们已经了解了[有界线性泛函](@entry_id:271069)和[Hahn-Banach定理](@entry_id:145406)在现代分析中的核心地位。本章将深入探讨其背后的基本原理和关键机制。我们将从[赋范空间](@entry_id:137032)上的线性泛函的基本定义开始，逐步构建起理解[Hahn-Banach定理](@entry_id:145406)所需的概念框架，并最终揭示其深刻的几何意义和广泛的应用。

### 基本概念：[线性泛函](@entry_id:276136)与[赋范空间](@entry_id:137032)

[泛函分析](@entry_id:146220)的研究对象是作用于函数空间上的算子，其中最基础的一类是**[线性泛函](@entry_id:276136)**（linear functional）。给定一个域 $\mathbb{K}$（在本文中，$\mathbb{K}$ 通常指[实数域](@entry_id:151347) $\mathbb{R}$ 或[复数域](@entry_id:153768) $\mathbb{C}$）上的[向量空间](@entry_id:151108) $X$，一个[线性泛函](@entry_id:276136)是从 $X$ 到其标量域 $\mathbb{K}$ 的一个[线性映射](@entry_id:185132) $f: X \to \mathbb{K}$。线性性质意味着对于所有向量 $x, y \in X$ 和标量 $\lambda \in \mathbb{K}$，都满足以下两个条件：
1.  可加性：$f(x+y) = f(x) + f(y)$
2.  齐次性：$f(\lambda x) = \lambda f(x)$

这是一个纯粹的代数定义，不涉及任何关于“距离”或“极限”的概念。然而，在分析学中，我们更关心的是那些与空间的拓扑结构相容的泛函，即**[连续线性泛函](@entry_id:262913)**。为了引入拓扑，我们需要为[向量空间](@entry_id:151108)配备一个**范数**（norm）。

一个**[赋范向量空间](@entry_id:274725)**（normed vector space）是一个对偶 $(X, \lVert \cdot \rVert)$，其中 $X$ 是一个[向量空间](@entry_id:151108)，而范数 $\lVert \cdot \rVert: X \to [0, \infty)$ 是一个函数，它为每个向量赋予一个非负的“长度”或“大小”，并满足以下三个公理：
1.  **[正定性](@entry_id:149643)**（Definiteness）：$\lVert x \rVert \ge 0$，且 $\lVert x \rVert = 0$ 当且仅当 $x=0$。
2.  **[绝对齐次性](@entry_id:274917)**（Absolute Homogeneity）：对所有 $\lambda \in \mathbb{K}$，有 $\lVert \lambda x \rVert = |\lambda| \lVert x \rVert$。
3.  **[三角不等式](@entry_id:143750)**（Triangle Inequality）：$\lVert x+y \rVert \le \lVert x \rVert + \lVert y \rVert$。

范数在空间 $X$ 上自然地诱导了一个度量 $d(x,y) = \lVert x-y \rVert$，从而赋予了 $X$ 一个度量空间结构和相应的拓扑。在此拓扑下，我们可以讨论[函数的连续性](@entry_id:193744)。一个线性泛函 $f: X \to \mathbb{K}$ 被称为**连续的**（continuous），如果它在 $X$ 的每一个点都连续。由于其线性性质，我们只需检验其在原点 $0$ 处的连续性即可。

对于[线性泛函](@entry_id:276136)而言，连续性有一个等价且更为实用的描述：**有界性**（boundedness）。一个线性泛函 $f$ 被称为**有界的**，如果存在一个常数 $M \ge 0$，使得对于所有 $x \in X$，不等式 $|f(x)| \le M \lVert x \rVert$ 恒成立。这个不等式直观地说明了泛函的值的增长被[向量的范数](@entry_id:154882)所控制。可以证明，一个线性泛函是连续的当且仅当它是有界的。

因此，我们通常将“[有界线性泛函](@entry_id:271069)”和“[连续线性泛函](@entry_id:262913)”视为同义词。所有从 $X$ 到 $\mathbb{K}$ 的[有界线性泛函](@entry_id:271069)构成的空间称为 $X$ 的**连续对偶空间**（continuous dual space），记为 $X^*$。

值得注意的是，并非所有[线性泛函](@entry_id:276136)都是有界的。在一个有限维[赋范空间](@entry_id:137032)中，所有线性泛函自动是有界的。然而，在无限维[赋范空间](@entry_id:137032)中，情况则大不相同。利用代数基（[Hamel基](@entry_id:155991)），我们可以构造出无界的线性泛函。这意味着，对于无限维空间，$X$ 的纯代数意义下的对偶空间（所有线性泛函的集合）严格地大于其连续[对偶空间](@entry_id:146945) $X^*$。[Hahn-Banach定理](@entry_id:145406)及其应用，主要关注的是 $X^*$ 中的元素。

还有一个深刻的等价刻画：一个非零线性泛函 $f$ 是连续的，当且仅当其**核**（kernel），即集合 $\ker f = \{x \in X : f(x) = 0\}$，是 $X$ 中的一个[闭子空间](@entry_id:267213)。这为我们从几何角度（[子空间](@entry_id:150286)的[拓扑性质](@entry_id:141605)）理解连续性提供了重要的视角。

### [算子范数](@entry_id:752960)：一种尺度度量

一旦我们确认一个[线性泛函](@entry_id:276136) $f$ 是有界的，下一个自然的问题就是：如何精确地度量其“大小”？或者说，在不等式 $|f(x)| \le M \lVert x \rVert$ 中，最小的那个常数 $M$ 是多少？这个最优的界就是 $f$ 的**算子范数**（operator norm），记为 $\lVert f \rVert$。

[算子范数](@entry_id:752960)有几种等价的定义形式，每一种都从不同角度揭示了其含义。

1.  **[最小上界](@entry_id:142911)定义**：
    $\lVert f \rVert = \inf \{ M \ge 0 : |f(x)| \le M\lVert x \rVert \text{ for all } x \in X \}$
    这个定义直接体现了 $\lVert f \rVert$ 作为最佳“[利普希茨常数](@entry_id:146583)”的角色。

2.  **[单位球](@entry_id:142558)上的上确界定义**：
    $\lVert f \rVert = \sup \{ |f(x)| : x \in X, \lVert x \rVert \le 1 \}$
    这是最常用的定义。它将问题简化为考察 $f$ 在空间中[单位球](@entry_id:142558)上的行为。由于 $f$ 的线性，其在整个空间的行为完全由其在单位球上的取值决定。

3.  **[单位球](@entry_id:142558)面上的[上确界](@entry_id:140512)定义**：
    对于非零空间，上述定义也等价于在单位球面上的[上确界](@entry_id:140512)：
    $\lVert f \rVert = \sup \{ |f(x)| : x \in X, \lVert x \rVert = 1 \}$
    这是因为任何非[零向量](@entry_id:156189) $x$ 都可以通过 $x/\lVert x \rVert$ 标准化到单位球面上，而 $|f(x/\lVert x \rVert)| = |f(x)|/\lVert x \rVert$。

4.  **比值的[上确界](@entry_id:140512)定义**：
    $\lVert f \rVert = \sup \{ \frac{|f(x)|}{\lVert x \rVert} : x \in X, x \neq 0 \}$
    这个形式强调了 $\lVert f \rVert$ 是泛函值相对于[向量长度](@entry_id:156432)的最大“拉伸因子”。

从这些定义出发，我们可以得到支配所有[有界线性泛函](@entry_id:271069)行为的基本不等式：
$$ |f(x)| \le \lVert f \rVert \lVert x \rVert \quad \text{for all } x \in X $$

一个自然的问题是：这个在[单位球](@entry_id:142558)上的[上确界](@entry_id:140512)（supremum）是否总能被某个向量“达到”（attained）？也就是说，是否存在一个向量 $x_0$ 使得 $\lVert x_0 \rVert = 1$ 且 $|f(x_0)| = \lVert f \rVert$？在[有限维空间](@entry_id:151571)中，答案是肯定的。这是因为[单位球](@entry_id:142558)面是紧集，而[连续函数](@entry_id:137361) $|f(x)|$ 在[紧集](@entry_id:147575)上必能取到其最大值。然而，在无限维空间中，单位球（和球面）通常不是紧的，因此范数不一定能被达到。一个泛函能否达到其范数，与空间的几何性质（如[自反性](@entry_id:137262)）密切相关。

### Hahn-Banach [扩张定理](@entry_id:139304)

[Hahn-Banach定理](@entry_id:145406)是泛函分析的基石之一。它本质上是一个关于**扩张**（extension）的定理：定义在一个[子空间](@entry_id:150286)上的“行为良好”的线性泛函，可以被扩张到整个空间，同时保持其“良好行为”。该定理有多种形式，我们将从最抽象的“控制”形式开始。

#### 控制与[次线性泛函](@entry_id:143368)

[Hahn-Banach定理](@entry_id:145406)最一般的形式是在一个纯代数框架下，通过一个所谓的**[次线性泛函](@entry_id:143368)**（sublinear functional）来“控制”[线性泛函](@entry_id:276136)的增长。

给定一个实[向量空间](@entry_id:151108) $V$，一个函数 $p: V \to \mathbb{R}$ 被称为**次线性**的，如果它满足：
1.  **[次可加性](@entry_id:137224)**（Subadditivity）：$p(x+y) \le p(x) + p(y)$ 对所有 $x, y \in V$ 成立。
2.  **[正齐次性](@entry_id:262235)**（Positive Homogeneity）：$p(\lambda x) = \lambda p(x)$ 对所有 $x \in V$ 和所有非负实数 $\lambda \ge 0$ 成立。

这个概念是范数概念的推广。任何范数都是一个[次线性泛函](@entry_id:143368)。另一个重要的例子是**[半范数](@entry_id:264573)**（seminorm），它满足范数的所有公理，除了 $\lVert x \rVert = 0$ 不再能推出 $x=0$。例如，在 $\mathbb{R}^2$ 上，函数 $p(x_1, x_2) = |x_1-x_2|$ 就是一个[半范数](@entry_id:264573)（因为它在直线 $x_1=x_2$ 上的所有点都取值为0），因此它也是一个[次线性泛函](@entry_id:143368)。相比之下，像 $p(x_1, x_2) = x_1^2 + x_2^2$ 这样的函数，虽然是凸的，但它不满足[正齐次性](@entry_id:262235)（而是二次齐次），因此不是次线性的。

[次线性泛函](@entry_id:143368)为我们提供了一个非常普适的工具来“衡量”向量的大小，并为[线性泛函的扩张](@entry_id:269901)设定一个上界。

#### 实 Hahn-Banach 定理：控制下的扩张

现在我们可以陈述[Hahn-Banach定理](@entry_id:145406)的第一个核心版本。

**定理（实Hahn-Banach控制[扩张定理](@entry_id:139304)）**：设 $X$ 是一个实[向量空间](@entry_id:151108)， $M$ 是 $X$ 的一个[线性子空间](@entry_id:151815)。设 $p: X \to \mathbb{R}$ 是一个[次线性泛函](@entry_id:143368)，而 $f_0: M \to \mathbb{R}$ 是一个[线性泛函](@entry_id:276136)，它在 $M$ 上被 $p$ 所**控制**（dominated），即对于所有 $x \in M$，都有 $f_0(x) \le p(x)$。那么，存在一个线性泛函 $F: X \to \mathbb{R}$，它是 $f_0$ 的一个**扩张**（extension），即 $F$ 在 $M$ 上的限制等于 $f_0$（记作 $F|_M = f_0$），并且 $F$ 在整个空间 $X$ 上仍然被 $p$ 所控制，即对于所有 $x \in X$，都有 $F(x) \le p(x)$。

该定理的证明是泛函分析中一个经典的非构造性论证，它依赖于**[佐恩引理](@entry_id:154284)**（Zorn's Lemma），而[佐恩引理](@entry_id:154284)等价于**选择公理**（Axiom of Choice）。这暗示了[Hahn-Banach定理](@entry_id:145406)的强大威力，同时也揭示了其结论的非构造性本质。

#### [保范扩张](@entry_id:268703)

[Hahn-Banach定理](@entry_id:145406)在[赋范空间](@entry_id:137032)中的一个直接且极为重要的推论是**[保范扩张](@entry_id:268703)定理**。

**定理（Hahn-Banach[保范扩张](@entry_id:268703)定理）**：设 $M$ 是[赋范空间](@entry_id:137032) $X$ 的一个（实或复）[线性子空间](@entry_id:151815)， $f_0: M \to \mathbb{K}$ 是 $M$ 上的一个[有界线性泛函](@entry_id:271069)。那么，存在一个[有界线性泛函](@entry_id:271069) $F: X \to \mathbb{K}$，它是 $f_0$ 的一个扩张，并且保持其范数不变，即 $\lVert F \rVert = \lVert f_0 \rVert$。

这个结果可以从上述的控制[扩张定理](@entry_id:139304)推导出来。对于[实数域](@entry_id:151347)的情形，我们只需巧妙地选择一个特定的[次线性泛函](@entry_id:143368)。给定 $f_0: M \to \mathbb{R}$，我们定义 $p(x) = \lVert f_0 \rVert \lVert x \rVert$。容易验证 $p$ 是 $X$ 上的一个[次线性泛函](@entry_id:143368)（实际上它是一个[半范数](@entry_id:264573)）。对于任何 $x \in M$，我们有 $f_0(x) \le |f_0(x)| \le \lVert f_0 \rVert \lVert x \rVert = p(x)$。因此，$f_0$ 被 $p$ 控制。根据实[Hahn-Banach定理](@entry_id:145406)，存在一个扩张 $F: X \to \mathbb{R}$ 使得 $F(x) \le p(x) = \lVert f_0 \rVert \lVert x \rVert$ 对所有 $x \in X$ 成立。通过考虑 $-x$，我们也能得到 $-F(x) \le \lVert f_0 \rVert \lVert x \rVert$，从而 $|F(x)| \le \lVert f_0 \rVert \lVert x \rVert$。这表明 $\lVert F \rVert \le \lVert f_0 \rVert$。另一方面，由于 $F$ 是 $f_0$ 的扩张，其范数不可能小于 $f_0$ 的范数（因为它作用的范围更广），即 $\lVert F \rVert \ge \lVert f_0 \rVert$。结合两者，我们得到 $\lVert F \rVert = \lVert f_0 \rVert$。

对于复数域的情形，证明过程则更为精妙。我们不能直接应用控制[扩张定理](@entry_id:139304)，因为 $f_0$ 的值是复数。标准做法是“化复为实”：
1.  将复[线性泛函](@entry_id:276136) $f_0$ 分解为其实部 $g_0 = \operatorname{Re}(f_0)$。可以证明 $g_0$ 是一个实[线性泛函](@entry_id:276136)，且其作为实[泛函的范数](@entry_id:142833)等于 $f_0$ 作为复[泛函的范数](@entry_id:142833)，即 $\lVert g_0 \rVert = \lVert f_0 \rVert$。
2.  对实[线性泛函](@entry_id:276136) $g_0$ 应用实数版本的[保范扩张](@entry_id:268703)定理，得到一个保范的实线性扩张 $G: X \to \mathbb{R}$，满足 $\lVert G \rVert = \lVert g_0 \rVert = \lVert f_0 \rVert$。
3.  利用复线性泛函实部和虚部之间的关系（即 $f(x) = \operatorname{Re}(f(x)) - i\operatorname{Re}(f(ix))$），通过 $G$ 来重构一个复[线性泛函](@entry_id:276136) $F(x) = G(x) - iG(ix)$。
4.  可以证明，这样构造出的 $F$ 不仅是复线性的，并且是 $f_0$ 的一个扩张，更重要的是，它的范数也被保持了下来，即 $\lVert F \rVert = \lVert f_0 \rVert$。

#### 扩张的族与唯一性

[Hahn-Banach定理](@entry_id:145406)保证了至少存在一个[保范扩张](@entry_id:268703)，但它是否唯一呢？通常情况下，**扩张不是唯一的**。

为了刻画所有可能的扩张，我们引入**[零化子](@entry_id:155446)**（annihilator）的概念。给定[子空间](@entry_id:150286) $Y \subset X$，其在[对偶空间](@entry_id:146945) $X^*$ 中的[零化子](@entry_id:155446) $Y^\perp$ 定义为所有在 $Y$ 上取值为零的 $X^*$ 中的泛函的集合：
$$ Y^\perp = \{ \Phi \in X^* : \Phi|_Y = 0 \} $$
$Y^\perp$ 本身是 $X^*$ 的一个闭[线性子空间](@entry_id:151815)。

假设 $F_0$ 是 $f \in Y^*$ 的一个扩张。那么，任何其他的扩张 $F$ 与 $F_0$ 之差 $g = F - F_0$ 必然在整个 $Y$ 上为零，因为 $F(y) - F_0(y) = f(y) - f(y) = 0$ 对所有 $y \in Y$ 成立。这意味着 $g \in Y^\perp$。反之，对于任何 $g \in Y^\perp$，$F_0+g$ 也是 $f$ 的一个扩张。因此， $f$ 的所有有界线性扩张的集合构成了一个**仿射[子空间](@entry_id:150286)**（affine subspace）：
$$ \{ F \in X^* : F|_Y = f \} = F_0 + Y^\perp $$
其中 $F_0$ 是 $f$ 的任意一个特定扩张。

从这个刻画可以清楚地看到，扩张是唯一的当且仅当 $Y^\perp = \{0\}$。而 $Y^\perp = \{0\}$ 的充分必要条件是[子空间](@entry_id:150286) $Y$ 在 $X$ 中是**稠密**的（dense），即 $\overline{Y} = X$。如果一个[子空间](@entry_id:150286)不是稠密的，那么我们总能找到一个非零的泛函在其上为零，从而导致扩张的不唯一性。

### 几何解释与推论

[Hahn-Banach定理](@entry_id:145406)的真正威力体現在其等价的几何形式中，这些形式通常被称为**[分离定理](@entry_id:268390)**（separation theorems）。它们揭示了[对偶空间](@entry_id:146945)与原空间几何结构之间的深刻联系。

#### [超平面](@entry_id:268044)分离

在一个[向量空间](@entry_id:151108)中，一个非零[线性泛函](@entry_id:276136) $f$ 的核 $\ker f$ 是一个余一维的[子空间](@entry_id:150286)。其平移 $\{x \in X : f(x) = c\}$ 称为一个**[超平面](@entry_id:268044)**（hyperplane）。[Hahn-Banach定理](@entry_id:145406)的一个重要几何推论是，不相交的[凸集](@entry_id:155617)在一定条件下可以被超平面分离开。

我们关注一个最基本的情形：分离一个点和一个[子空间](@entry_id:150286)。
**推论（分离点与[闭子空间](@entry_id:267213)）**：设 $Y$ 是[赋范空间](@entry_id:137032) $X$ 的一个闭[线性子空间](@entry_id:151815)，且 $x_0 \in X \setminus Y$。那么，存在一个[连续线性泛函](@entry_id:262913) $f \in X^*$，使得：
1.  $f$ 在 $Y$ 上为零，即 $f|_Y = 0$（或 $f \in Y^\perp$）。
2.  $f(x_0) \neq 0$。

这意味着超平面 $\ker f$ 包含了整个[子空间](@entry_id:150286) $Y$，但没有包含点 $x_0$。这个推论保证了对偶空间 $X^*$ 的“丰富性”：对于任何不在一个[闭子空间](@entry_id:267213)中的点，我们总能找到一个[连续线性泛函](@entry_id:262913)来“侦测”到它。

这个推论中，“闭”的条件至关重要。如果一个集合不是闭的，严格分离可能会失败。一个经典的例子是开[单位球](@entry_id:142558) $C = \{x: \lVert x \rVert  1\}$ 和一个边界点 $x_0$（$\lVert x_0 \rVert = 1$）。虽然 $x_0 \notin C$，但我们无法找到一个泛函 $f$ 使得 $f(x_0)  \sup_{x \in C} f(x)$。这是因为对于任何[有界线性泛函](@entry_id:271069) $f$，我们总有 $\sup_{x \in C} |f(x)| = \lVert f \rVert$（如果考虑实泛函，则是 $\sup_{x \in C} f(x) = \lVert f \rVert$），而 $|f(x_0)| \le \lVert f \rVert \lVert x_0 \rVert = \lVert f \rVert$。因此，我们最多只能做到非严格分离。[Hahn-Banach定理](@entry_id:145406)确实能保证存在一个泛函 $f$ 使得 $f(x_0) = \lVert f \rVert$，从而实现 $f(x_0) = \sup_{x \in C} f(x)$ 这种“接触式”的非严格分离。

#### 距离的[对偶表示](@entry_id:146263)

[Hahn-Banach定理](@entry_id:145406)的几何形式还能导出一个优美的公式，用[对偶空间](@entry_id:146945)中的泛函来表示一个点到一个[子空间](@entry_id:150286)的距离。

设 $Y$ 是一个闭[线性子空间](@entry_id:151815)， $x_0 \notin Y$。点 $x_0$ 到 $Y$ 的距离定义为 $\mathrm{dist}(x_0, Y) = \inf_{y \in Y} \lVert x_0 - y \rVert$。由于 $Y$ 是闭的且 $x_0 \notin Y$，这个距离是严格正的。

[Hahn-Banach定理](@entry_id:145406)的一个推论给出了这个距离的[对偶表示](@entry_id:146263)：
$$ \mathrm{dist}(x_0, Y) = \sup \{ |f(x_0)| : f \in X^*, \lVert f \rVert \le 1, f|_Y = 0 \} $$
这个公式的右边是在 $Y$ 的[零化子](@entry_id:155446) $Y^\perp$ 的单位球上，评估在 $x_0$ 处的取值的[上确界](@entry_id:140512)。

这个等式的证明分为两部分：
1.  “$\ge$”：对于任何满足条件的 $f$（即 $\lVert f \rVert \le 1$ 且 $f|_Y=0$），我们有 $|f(x_0)| = |f(x_0 - y)| \le \lVert f \rVert \lVert x_0 - y \rVert \le \lVert x_0 - y \rVert$ 对任意 $y \in Y$ 成立。对右边取关于 $y$ 的下确界，得到 $|f(x_0)| \le \mathrm{dist}(x_0, Y)$。这对所有这样的 $f$ 都成立，所以右边的上确界也满足这个不等式。
2.  “$\le$”：这一部分需要构造一个特殊的泛函来证明上确界可以达到 $\mathrm{dist}(x_0, Y)$。这正是[Hahn-Banach定理](@entry_id:145406)发挥作用的地方。我们可以构造一个定义在[子空间](@entry_id:150286) $M = \mathrm{span}\{x_0, Y\}$ 上的[线性泛函](@entry_id:276136) $f_0$，使其满足 $f_0|_Y = 0$ 且 $f_0(x_0) = \mathrm{dist}(x_0, Y)$。然后证明这个 $f_0$ 的范数恰好为 $1$。根据Hahn-Banach[保范扩张](@entry_id:268703)定理，我们可以将 $f_0$ 扩张到整个空间 $X$ 上得到一个泛函 $F$，其范数仍为 $1$，且 $F|_Y = 0, F(x_0) = \mathrm{dist}(x_0, Y)$。这个 $F$ 就在右边上确界考虑的集合中，它的值 $|F(x_0)|$ 等于 $\mathrm{dist}(x_0, Y)$。因此，上确界至少是这么大。

结合两部分，等式成立。这个结果是几何与分析之间对偶关系的一个深刻体现。

### 一个非构造性应用：Banach 极限

最后，我们通过一个更高等的应用——**[Banach极限](@entry_id:140080)**（Banach limit）的存在性——来展示[Hahn-Banach定理](@entry_id:145406)的威力及其非构造性的本质。

考虑所有有界实序列的空间 $\ell_\infty$，其上的范数为[上确界范数](@entry_id:145717) $\lVert x \rVert_\infty = \sup_n |x_n|$。我们自然地想将普通极限的概念推广到 $\ell_\infty$ 上的所有序列，即使它们不收敛。一个“理想”的推广，称为[Banach极限](@entry_id:140080) $\Lambda: \ell_\infty \to \mathbb{R}$，应该是一个[线性泛函](@entry_id:276136)，并满足一些良好性质，如：
1.  **正性**：如果 $x_n \ge 0$ 对所有 $n$ 成立，则 $\Lambda(x) \ge 0$。
2.  **[移位不变性](@entry_id:754776)**：$\Lambda(Sx) = \Lambda(x)$，其中 $S$ 是左移算子 $(Sx)_n = x_{n+1}$。
3.  **与普通极限相容**：如果序列 $x$ 收敛到 $L$，则 $\Lambda(x) = L$。
4.  **范数为1**：$\lVert \Lambda \rVert = 1$。

然而，我们无法写下一个具体的公式来计算任意[有界序列](@entry_id:161392)的[Banach极限](@entry_id:140080)。其存在性完全依赖于[Hahn-Banach定理](@entry_id:145406)。

构造的大致思路如下：
1.  定义一个[次线性泛函](@entry_id:143368) $p(x) = \limsup_{n \to \infty} \frac{1}{n} \sum_{k=1}^n x_k$，即序列 $x$ 的**切萨罗均值**（Cesàro means）的上极限。
2.  在一个合适的[子空间](@entry_id:150286)（包含所有[收敛序列](@entry_id:144123)和形如 $x - Sx$ 的序列）上定义一个线性泛函，使其等于普通极限。
3.  证明这个泛函被 $p$ 所控制。
4.  应用Hahn-Banach控制[扩张定理](@entry_id:139304)，将其扩张到整个 $\ell_\infty$ 空间。

最终得到的扩张 $\Lambda$ 就满足[Banach极限](@entry_id:140080)的所有性质。由于该构造的核心步骤依赖于[Hahn-Banach定理](@entry_id:145406)，而后者又依赖于选择公理，因此[Banach极限](@entry_id:140080)的存在性是一个纯粹的存在性结果，是**非构造性**的。这个例子完美地诠释了[Hahn-Banach定理](@entry_id:145406)如何保证了分析学中某些理想对象的存在，即使我们无法具体地“指认”它们。
