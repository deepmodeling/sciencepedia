## 引言
在[微分几何](@entry_id:145818)与拓扑学中，理解[光滑流形](@entry_id:160799)上函数的行为是一个核心问题。函数的[临界点](@entry_id:144653)——那些梯度为零的点——尤其关键，因为它们标志着[流形](@entry_id:153038)拓扑结构发生变化的位置。然而，在这些点附近，函数的局部形态可能极其复杂，构成了一道分析上的难题。[莫尔斯理论](@entry_id:151962)应运而生，它提供了一个优雅而强大的框架，通过关注一类被称为“莫尔斯函数”的“良好”函数，极大地简化了对[临界点](@entry_id:144653)的分析，从而揭示了函数与其所定义[流形](@entry_id:153038)之间的深刻联系。

本文将系统地引导读者深入[莫尔斯理论](@entry_id:151962)的世界。在第一章“原理与机制”中，我们将建立理论基础，详细定义[非退化临界点](@entry_id:271108)、莫尔斯指标，并阐明核心的[莫尔斯引理](@entry_id:182795)。随后的“应用与跨学科联系”章节将展示该理论的强大威力，探索它如何连接拓扑学、物理学和化学，用于计算拓扑不变量、分析系统稳定性以及寻找[化学反应](@entry_id:146973)路径。最后，通过“动手实践”环节，读者将有机会运用所学知识解决具体问题，巩固理解。让我们从[莫尔斯理论](@entry_id:151962)的基本原理开始，一探究竟。

## 原理与机制

在对光滑流形上的函数进行分析时，一个核心任务是理解其在[临界点](@entry_id:144653)附近的局部行为。[临界点](@entry_id:144653)，即函数的梯度为零的点，捕捉了函数[拓扑变化](@entry_id:136654)的“事件”。[莫尔斯理论](@entry_id:151962)为研究这些[临界点](@entry_id:144653)提供了一个强大而优雅的框架，其基本思想是，在“一般”情况下，函数在[临界点](@entry_id:144653)附近的结构可以被极大地简化。本章将系统地阐述[莫尔斯理论](@entry_id:151962)的核心原理与机制，从[临界点](@entry_id:144653)的分类出发，逐步深入到[莫尔斯引理](@entry_id:182795)及其在几何与动力学中的深刻应用。

### [临界点](@entry_id:144653)与非退化条件

我们从基本定义开始。对于一个在 $\mathbb{R}^n$ 的开[子集](@entry_id:261956)上定义的[光滑函数](@entry_id:267124) $f: U \to \mathbb{R}$，如果点 $p \in U$ 处的梯度为零，即 $\nabla f(p) = 0$，则称 $p$ 为 $f$ 的一个**[临界点](@entry_id:144653)**。在这些点上，函数的[一阶导数](@entry_id:749425)信息消失，其局部行为由更高阶的导数决定。

根据[泰勒展开](@entry_id:145057)，在[临界点](@entry_id:144653) $p$ 附近，函数 $f$ 可以近似表示为：
$f(x) \approx f(p) + \frac{1}{2} (x-p)^T H_f(p) (x-p)$
其中，$H_f(p)$ 是 $f$ 在点 $p$ 的**黑塞矩阵（Hessian matrix）**。这是一个由[二阶偏导数](@entry_id:635213)构成的对称矩阵：
$(H_f)_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}(p)$
黑塞矩阵描述了函数在[临界点](@entry_id:144653)附近的二次曲面近似。这个二次型的性质，完全由黑塞矩阵的性质决定，是区分不同类型[临界点](@entry_id:144653)的关键。

一个[临界点](@entry_id:144653) $p$ 被称为**非退化的（non-degenerate）**，如果其黑塞矩阵 $H_f(p)$ 是可逆的，即 $\det(H_f(p)) \neq 0$。反之，如果 $\det(H_f(p)) = 0$，则称该[临界点](@entry_id:144653)是**退化的（degenerate）**。[非退化临界点](@entry_id:271108)是“简单”的，它们的局部结构稳定且易于分类。退化[临界点](@entry_id:144653)则更为复杂，其附近可能发生更微妙的[拓扑变化](@entry_id:136654)，例如多个[临界点](@entry_id:144653)的合并或[分叉](@entry_id:270606)。

让我们通过一个例子来具体理解这一概念。考虑一个由函数 $f(x, y) = A \cos(x) + B \cos(y) + C xy$ 描述的二维地貌，其中 $A, B, C$ 为实常数。原点 $(0,0)$ 始终是该函数的一个[临界点](@entry_id:144653)，因为 $\frac{\partial f}{\partial x}|_{(0,0)} = -A \sin(0) + C(0) = 0$ 且 $\frac{\partial f}{\partial y}|_{(0,0)} = -B \sin(0) + C(0) = 0$。为了判断该[临界点](@entry_id:144653)何时是退化的，我们需要计算其黑塞矩阵。[二阶偏导数](@entry_id:635213)为：
$\frac{\partial^2 f}{\partial x^2} = -A \cos(x)$, $\frac{\partial^2 f}{\partial y^2} = -B \cos(y)$, $\frac{\partial^2 f}{\partial x \partial y} = C$
在原点 $(0,0)$ 处，黑塞矩阵为：
$H_f(0,0) = \begin{pmatrix} -A \cos(0)  C \\ C  -B \cos(0) \end{pmatrix} = \begin{pmatrix} -A  C \\ C  -B \end{pmatrix}$
该[临界点](@entry_id:144653)是退化的，当且仅当其[行列式](@entry_id:142978)为零：
$\det(H_f(0,0)) = (-A)(-B) - C^2 = AB - C^2 = 0$
例如，若给定参数 $A=3$ 和 $B=12$，则退化条件变为 $36 - C^2 = 0$，解得 $C = \pm 6$。这意味着，当 $C$ 取这两个特定值时，原点处的地貌结构比简单的二次曲面（如[抛物面](@entry_id:264713)或双曲抛物面）更为复杂，无法仅通过[二阶导数](@entry_id:144508)完全刻画 [@problem_id:1654077]。

### 莫尔斯函数

基于[非退化临界点](@entry_id:271108)的概念，我们引入[莫尔斯理论](@entry_id:151962)的核心对象。一个在[流形](@entry_id:153038) $M$ 上定义的光滑函数 $f: M \to \mathbb{R}$ 被称为**莫尔斯函数（Morse function）**，如果它的所有[临界点](@entry_id:144653)都是非退化的。

这个定义看似简单，但意义深远。它要求函数在整个定义域上表现出良好的性质。一个函数是否为莫尔斯函数，是一个全局性的属性。函数在某一点可能是非退化的，但在另一点可能是退化的，只要存在一个退化[临界点](@entry_id:144653)，该函数就不是莫尔斯函数。

考虑一个由参数 $k$ 决定的函数族 $f_k: \mathbb{R}^2 \to \mathbb{R}$，定义为 $f_k(x, y) = \frac{1}{3}x^3 + \frac{1}{2}(k - 5)x^2 + xy + \frac{1}{2}y^2$。我们可以探究对于哪个 $k$ 值，该函数会失去莫尔斯性质。
首先，我们求解 $\nabla f_k = 0$ 来寻找[临界点](@entry_id:144653)：
$\frac{\partial f_k}{\partial x} = x^2 + (k-5)x + y = 0$
$\frac{\partial f_k}{\partial y} = x + y = 0$
由第二式得 $y = -x$，代入第一式得 $x^2 + (k-6)x = 0$，解出 $x=0$ 或 $x=6-k$。因此，[临界点](@entry_id:144653)为 $(0,0)$ 和 $(6-k, k-6)$。
接下来，计算黑塞矩阵的行列式：
$H_f(x,y) = \begin{pmatrix} 2x + k - 5  1 \\ 1  1 \end{pmatrix}$
$\det(H_f(x,y)) = (2x + k - 5) - 1 = 2x + k - 6$
现在，我们在[临界点](@entry_id:144653)处评估此[行列式](@entry_id:142978)：
在 $(0,0)$ 处，$\det(H_f(0,0)) = k-6$。
在 $(6-k, k-6)$ 处，$\det(H_f(6-k, k-6)) = 2(6-k) + k-6 = 6-k$。
函数 $f_k$ 不是莫尔斯函数，当且仅当存在一个[临界点](@entry_id:144653)使其黑塞[行列式](@entry_id:142978)为零。这发生在 $k-6=0$ 或 $6-k=0$ 时，两者都给出 $k=6$。因此，当 $k=6$ 时，函数 $f_6$ 至少有一个退化[临界点](@entry_id:144653)，从而不再是莫尔斯函数 [@problem_id:1654078]。

一个关键的洞察是，莫尔斯函数在某种意义上是“普遍的”或“泛函的”。对于任意一个光滑函数，通常可以通过一个任意小的扰动将其变为一个莫尔斯函数。这一性质被称为莫尔斯函数的**稳定性（stability）**，我们将在后续章节中进一步探讨。

### 莫尔斯指标

对于一个[非退化临界点](@entry_id:271108)，其黑塞矩阵 $H_f(p)$ 是一个[实对称矩阵](@entry_id:192806)，因此它的所有[特征值](@entry_id:154894)都是实数。又因为 $H_f(p)$ 可逆，所以所有[特征值](@entry_id:154894)都非零。这使得我们可以对[非退化临界点](@entry_id:271108)进行进一步的分类。

一个[非退化临界点](@entry_id:271108) $p$ 的**莫尔斯指标（Morse index）**，记作 $\text{ind}(p)$ 或 $k$，定义为 $H_f(p)$ 的负[特征值](@entry_id:154894)的数量。莫尔斯指标是一个整数，其取值范围为 $0 \le k \le n$，其中 $n$ 是函数定义域的维数。这个指标是描述[临界点](@entry_id:144653)局部结构的核心[不变量](@entry_id:148850)。

例如，我们来计算函数 $f(x, y, z) = \frac{1}{4}x^4 - 2x^2 - \frac{1}{2}y^2 - \frac{1}{2}z^2$ 的所有[非退化临界点](@entry_id:271108)的莫尔斯指标之和。
首先找到[临界点](@entry_id:144653)，令 $\nabla f = 0$：
$\frac{\partial f}{\partial x} = x^3 - 4x = x(x^2-4) = 0 \implies x \in \{0, 2, -2\}$
$\frac{\partial f}{\partial y} = -y = 0 \implies y=0$
$\frac{\partial f}{\partial z} = -z = 0 \implies z=0$
[临界点](@entry_id:144653)为 $p_1=(0,0,0)$, $p_2=(2,0,0)$, $p_3=(-2,0,0)$。
黑塞矩阵为：
$H_f(x,y,z) = \begin{pmatrix} 3x^2-4  0  0 \\ 0  -1  0 \\ 0  0  -1 \end{pmatrix}$
现在计算每个[临界点](@entry_id:144653)的指标：
- 在 $p_1=(0,0,0)$ 处，$H_f(p_1) = \text{diag}(-4, -1, -1)$。所有三个[特征值](@entry_id:154894)均为负，因此指标为 $k=3$。
- 在 $p_2=(2,0,0)$ 处，$H_f(p_2) = \text{diag}(3(2^2)-4, -1, -1) = \text{diag}(8, -1, -1)$。有两个负[特征值](@entry_id:154894)，因此指标为 $k=2$。
- 在 $p_3=(-2,0,0)$ 处，$H_f(p_3) = \text{diag}(3(-2)^2-4, -1, -1) = \text{diag}(8, -1, -1)$。同样，指标为 $k=2$。
所有[临界点](@entry_id:144653)都是非退化的（因为没有零[特征值](@entry_id:154894)）。指标之和为 $3+2+2=7$ [@problem_id:1654031]。

莫尔斯指标直接关联到[临界点](@entry_id:144653)附近的几何形态：
- **指标 $k=0$**：所有[特征值](@entry_id:154894)均为正。$H_f(p)$ 是正定的。这意味着 $p$ 是一个**局部极小点**。函数图像在 $p$ 附近形如一个向上开口的碗。
- **指标 $k=n$**：所有[特征值](@entry_id:154894)均为负。$H_f(p)$ 是负定的。这意味着 $p$ 是一个**局部极大点**。函数图像在 $p$ 附近形如一个向下开口的穹顶。
- **指标 $0  k  n$**：[特征值](@entry_id:154894)有正有负。$H_f(p)$ 是不定的。这意味着 $p$ 是一个**[鞍点](@entry_id:142576)**。在某些方向上函数值增加，在另一些方向上函数值减少。

考虑一个二维函数 $f(x,y)$，其在[临界点](@entry_id:144653) $p_0$ 的[二阶偏导数](@entry_id:635213)为 $\frac{\partial^2 f}{\partial x^2}(p_0) = -1$, $\frac{\partial^2 f}{\partial y^2}(p_0) = -1$ 和 $\frac{\partial^2 f}{\partial x \partial y}(p_0) = 0$。其黑塞矩阵为 $H_f(p_0) = \begin{pmatrix} -1  0 \\ 0  -1 \end{pmatrix}$。[特征值](@entry_id:154894)为 $-1$ 和 $-1$，均为负数。因此，莫尔斯指标 $k=2$。在二维空间中（$n=2$），$k=n$ 的情况对应一个局部极大点。因此，函数图像在 $p_0$ 附近局部地像一个倒扣的穹顶 [@problem_id:1654081]。

在更高维度中，指标的计算也遵循同样原则。例如，对于 $\mathbb{R}^4$ 上的函数 $f(x_1, x_2, x_3, x_4) = -\frac{3}{4} x_1^2 - \frac{3}{4} x_2^2 + 2 x_3^2 + 2 x_4^2 + \frac{1}{2} x_1 x_2 + x_3 x_4$，原点是一个[临界点](@entry_id:144653)。其黑塞矩阵是一个常数矩阵：
$H = \begin{pmatrix} -3/2  1/2  0  0 \\ 1/2  -3/2  0  0 \\ 0  0  4  1 \\ 0  0  1  4 \end{pmatrix}$
这是一个分[块对角矩阵](@entry_id:145530)。左上角 $2 \times 2$ 块的[特征值](@entry_id:154894)为 $-1$ 和 $-2$（均为负）。右下角 $2 \times 2$ 块的[特征值](@entry_id:154894)为 $3$ 和 $5$（均为正）。因此，整个 $4 \times 4$ 矩阵有两个负[特征值](@entry_id:154894)和两个正[特征值](@entry_id:154894)。该[临界点](@entry_id:144653)的莫尔斯指标为 $k=2$ [@problem_id:1654095]。

### [莫尔斯引理](@entry_id:182795)：局部规范型

莫尔斯指标的真正威力体现在**[莫尔斯引理](@entry_id:182795)（Morse Lemma）**中，这是[莫尔斯理论](@entry_id:151962)的基石。该引理断言，在任何一个[非退化临界点](@entry_id:271108)附近，一个[光滑函数](@entry_id:267124)（无论其表达式多么复杂）都可以通过一个光滑的[局部坐标](@entry_id:181200)变换，简化为一个简单的二次型。

**[莫尔斯引理](@entry_id:182795)**：设 $p$ 是函数 $f: \mathbb{R}^n \to \mathbb{R}$ 的一个指标为 $k$ 的[非退化临界点](@entry_id:271108)。那么，在 $p$ 的一个邻域内，存在一个以 $p$ 为原点的[局部坐标系](@entry_id:751394) $(y_1, y_2, \ldots, y_n)$，使得函数 $f$ 在该[坐标系](@entry_id:156346)下具有如下的**规范型（normal form）**：
$f(y_1, \ldots, y_n) = f(p) - \sum_{i=1}^k y_i^2 + \sum_{i=k+1}^n y_i^2$

这个引理的深刻之处在于，它表明[非退化临界点](@entry_id:271108)的所有局部信息都被一个常数 $f(p)$ 和一个整数 $k$（莫尔斯指标）完全捕捉。函数的局部几何形态本质上只有一种，即[二次曲面](@entry_id:264390)，不同之处仅在于“下降”方向（负号项）和“上升”方向（正号项）的数量。表达式中的负号项数量恰好是莫尔斯指标 $k$ [@problem_id:1654058]。

让我们将此引理应用于一个具体函数 $f(x, y) = \cos(x) + xy + \frac{1}{2}y^2$。原点 $p=(0,0)$ 是一个[临界点](@entry_id:144653)。我们来确定它在[莫尔斯引理](@entry_id:182795)下的规范型。
首先，计算黑塞矩阵以确定其指标：
$H_f(0,0) = \begin{pmatrix} -\cos(0)  1 \\ 1  1 \end{pmatrix} = \begin{pmatrix} -1  1 \\ 1  1 \end{pmatrix}$
其[行列式](@entry_id:142978)为 $(-1)(1) - (1)(1) = -2 \neq 0$，因此[临界点](@entry_id:144653)是非退化的。[特征方程](@entry_id:265849)为 $\det(H - \lambda I) = (-1-\lambda)(1-\lambda) - 1 = \lambda^2 - 2 = 0$，解得[特征值](@entry_id:154894)为 $\lambda = \pm\sqrt{2}$。有一个负[特征值](@entry_id:154894)，所以莫尔斯指标 $k=1$。
根据[莫尔斯引理](@entry_id:182795)，在原点附近存在一个[局部坐标系](@entry_id:751394) $(u_1, u_2)$，使得函数（减去其在原点的值 $f(p)=f(0,0)=1$）可以表示为：
$f(u_1, u_2) - f(p) = -u_1^2 + u_2^2$
这对应于一个[鞍点](@entry_id:142576)的形状 [@problem_id:1654057]。

### 指标的几何与动力学诠释

莫尔斯指标不仅是一个代数计数，它还具有深刻的几何和动力学含义。这可以通过分析函数在[临界点](@entry_id:144653)附近的等值线和梯度流来揭示。

考虑一个定义在二维[曲面](@entry_id:267450) $S$ 上的光滑函数 $f$ 及其一个指标为 $k$ 的[非退化临界点](@entry_id:271108) $p$。

**1. 等值线的结构**
根据[莫尔斯引理](@entry_id:182795)，在 $p$ 附近的[局部坐标](@entry_id:181200) $(x,y)$ 中，$f(x,y) \approx f(p) + q(x,y)$，其中 $q(x,y)$ 是一个二次型。过[临界点](@entry_id:144653) $p$ 的等值线由方程 $f(x,y) = f(p)$ 给出，这近似于 $q(x,y)=0$。
- 如果指标 $k=0$（局部极小点），则 $q(x,y) = x^2+y^2$。方程 $x^2+y^2=0$ 的唯一实数解是 $(x,y)=(0,0)$。因此，等值线在局部仅为点 $p$ 本身。
- 如果指标 $k=2$（局部极大点），则 $q(x,y) = -x^2-y^2$。同样，等值线在局部也仅为点 $p$。
- 如果指标 $k=1$（[鞍点](@entry_id:142576)），则 $q(x,y) = -x^2+y^2$。方程 $-x^2+y^2=0$ 等价于 $y=\pm x$，这代表两条在原点横截相交的直线。

因此，若观察到过[临界点](@entry_id:144653) $p$ 的等值线由两条在此点横截相交的光滑曲线构成，这强烈地表明该[临界点](@entry_id:144653)的莫尔斯指标为1 [@problem_id:1654084]。

**2. [梯度流](@entry_id:635964)的动力学**
考虑向量场 $X = -\nabla f$，它指向函数值下降最快的方向。该向量场的[积分曲线](@entry_id:161858)（[流线](@entry_id:266815)）构成了**[梯度流](@entry_id:635964)**。[临界点](@entry_id:144653)是该动力系统的[平衡点](@entry_id:272705)。
在[临界点](@entry_id:144653) $p$ 附近，[流线](@entry_id:266815)由线性化系统 $\dot{\mathbf{y}} = -H_f(p)\mathbf{y}$ 主导。该系统的动力学行为由矩阵 $-H_f(p)$ 的[特征值](@entry_id:154894)决定。
- 如果 $\lambda$ 是 $H_f(p)$ 的一个[特征值](@entry_id:154894)，则 $-\lambda$ 是 $-H_f(p)$ 的一个[特征值](@entry_id:154894)。
- 流向[平衡点](@entry_id:272705) $p$ 的轨迹构成了**[稳定流形](@entry_id:266484)** $W^s(p)$。其维数等于 $-H_f(p)$ 的负[特征值](@entry_id:154894)的数量，也就是 $H_f(p)$ 的正[特征值](@entry_id:154894)的数量，即 $n-k$。
- 从[平衡点](@entry_id:272705) $p$ 流出的轨迹构成了**[不稳定流形](@entry_id:265383)** $W^u(p)$。其维数等于 $-H_f(p)$ 的正[特征值](@entry_id:154894)的数量，也就是 $H_f(p)$ 的负[特征值](@entry_id:154894)的数量，即莫尔斯指标 $k$。

因此，**莫尔斯指标 $k$ 直接等于[不稳定流形](@entry_id:265383)的维数**。不稳定流形是由所有当时间 $t \to -\infty$ 时收敛到 $p$ 的轨迹组成的。

回到二维[曲面](@entry_id:267450)的例子，如果观测到“恰好有两条[流线](@entry_id:266815)汇入点 $p$”，这意味着 $p$ 的[稳定流形](@entry_id:266484) $W^s(p)$ 是一维的（由这两条流线和点 $p$ 自身构成）。在二维空间中（$n=2$），$\dim(W^s(p)) = n-k = 2-k = 1$，这直接得出莫尔斯指标 $k=1$。这个结论与从等值线结构得到的结论完全一致，展示了莫尔斯指标在几何与动力学中的统一性 [@problem_id:1654084]。

### 高级主题与展望

[莫尔斯理论](@entry_id:151962)的原理和机制延伸到了更广阔的领域，下面我们简要介绍两个重要的方向。

#### 莫尔斯函数的稳定性

前面提到，莫尔斯函数在小的扰动下是稳定的。这意味着，如果 $f$ 是一个莫尔斯函数，那么任何与 $f$ “足够接近”的函数 $g$ (在 $C^2$ 范数意义下) 也是莫尔斯函数。非退化条件 $\det(H_f(p)) \neq 0$ 是一个开条件，在小的扰动下得以保持。

当一个函数族依赖于某个参数时，随着参数的变化，函数可能会在某个临界值上失去其莫尔斯性质。考虑定义在[2-环面](@entry_id:265991) $\mathbb{T}^2$ 上的函数族 $f_\epsilon(x, y) = \cos(x) + \cos(y) + \epsilon \cos(2x) \cos(y)$。当 $\epsilon=0$ 时，$f_0$ 是一个标准的莫尔斯函数。我们可以找到最小的正 $\epsilon$ 值，使得 $f_\epsilon$ 不再是莫尔斯函数。
通过细致的计算，可以找到 $f_\epsilon$ 的所有[临界点](@entry_id:144653)，并计算其黑塞[行列式](@entry_id:142978)。分析表明，对于 $0 \le \epsilon  1/4$，该函数的所有[临界点](@entry_id:144653)都是非退化的。在 $\epsilon = 1/4$ 这个临界值，位于 $(0, \pi)$ 和 $(\pi, 0)$ 的[临界点](@entry_id:144653)首次变为退化。因此，$\epsilon = 1/4$ 是函数族 $f_\epsilon$ 失去莫尔斯性质的第一个点 [@problem_id:1654046]。这个例子生动地展示了莫尔斯函数集合在函数空间中的开放性，以及当参数穿越某个“墙”时发生的[退化现象](@entry_id:183258)。

#### 莫尔斯-博特理论简介

经典[莫尔斯理论](@entry_id:151962)要求所有[临界点](@entry_id:144653)都是孤立的。然而，在许多重要的几何问题中，函数的[临界点](@entry_id:144653)集可能构成更高维的子流形。例如，一个旋转对称的函数，其[临界点](@entry_id:144653)可能是一整个圆周。为了处理这种情况，我们需要**莫尔斯-博特理论（Morse-Bott Theory）**。

一个函数 $f$ 被称为**莫尔斯-博特函数**，如果其[临界点](@entry_id:144653)集是若干不相交的闭子流形 $C_j$ 的并集，并且在每个临界[子流形](@entry_id:159439) $C_j$ 的[法丛](@entry_id:272447)上是**非退化**的。这具体是指，在任意点 $p \in C_j$，黑塞矩阵 $H_f(p)$ 的零空间（kernel）恰好是 $C_j$ 在该点的[切空间](@entry_id:199137) $T_p(C_j)$。

对于这样一个临界子流形 $C_j$，我们可以定义其**莫尔斯-博特指标**，即黑塞矩阵在法向空间上的负[特征值](@entry_id:154894)的数量。这个指标在整个 $C_j$ 上是恒定的。

考虑 $\mathbb{R}^4$ 上的函数 $f(x,y,z,w) = (x^2 + y^2 - R^2)^2 + z^2 - w^2$。其[临界点](@entry_id:144653)满足 $\nabla f = 0$，解得[临界点](@entry_id:144653)集由[孤立点](@entry_id:146695) $(0,0,0,0)$ 和一个圆 $C = \{(x,y,z,w) : x^2+y^2=R^2, z=0, w=0\}$ 构成。这个圆是一个一维的临界子流形，因此其维数 $d=1$。
为了计算其莫尔斯-博特指标，我们在 $C$ 上任意一点计算黑塞矩阵，并考察其在法向空间上的作用。 $C$ 的切方向由向量 $(-y, x, 0, 0)$ 张成。法空间则由径向向量 $(x, y, 0, 0)$、z轴[方向向量](@entry_id:169562) $(0,0,1,0)$ 和w轴方向向量 $(0,0,0,1)$ 张成。通过计算可以发现，黑塞矩阵在这些法向上的[特征值](@entry_id:154894)分别为 $8R^2$ (正)、$2$ (正) 和 $-2$ (负)。由于只有一个负[特征值](@entry_id:154894)，该临界子流形的莫尔斯-博特指标为 $i=1$ [@problem_id:1654056]。因此，这个一维临界子流形的莫尔斯-博特指标为1。莫尔斯-博特理论通过这种方式，将经典[莫尔斯理论](@entry_id:151962)的强大思想推广到了更广泛的函数类别，在[辛几何](@entry_id:160783)和规范场论等领域有着核心应用。
