## 引言
在[数学分析](@entry_id:139664)中，尤其是在处理一般实值函数时，我们常常需要一种方法来简化问题。将一个复杂的对象分解为更简单、更易于管理的组成部分，是一种强大而普遍的策略。对于任意一个可以取正值也可以取负值的函数，我们如何系统地研究它的性质，特别是如何定义它的积分呢？[函数分解](@entry_id:197881)为正部与负部，正是解决这一核心问题的关键工具。这一概念不仅是[勒贝格积分](@entry_id:140189)理论的支柱，也在泛函分析和概率论等多个领域中发挥着至关重要的作用。

本文旨在全面解析[函数的正部](@entry_id:203768)与负部这一基本概念。在“原理与机制”一章中，我们将深入探讨正部与负部的定义、基本代数性质及其与可测性的深刻联系。接下来，在“应用与跨学科联系”一章中，我们将展示这一分解如何在积分理论、[函数空间](@entry_id:143478)以及概率论中成为解决问题的有力工具。最后，“动手实践”部分将提供具体的练习，帮助您巩固所学知识。通过本文的学习，您将掌握现代分析中一个不可或缺的理论基石。

## 原理与机制

在分析学中，特别是在[勒贝格积分](@entry_id:140189)理论的构建过程中，我们经常需要处理任意实值函数。一个强大而基本的策略是将这些[函数分解](@entry_id:197881)为更简单、更易于处理的部分。具体来说，任何实值函数都可以表示为两个非负函数的差。这种分解方法是定义[可测函数的积](@entry_id:194504)分以及研究其性质的基石。本章将详细阐述[函数分解](@entry_id:197881)的核心原理，即[函数的正部](@entry_id:203768)与负部的概念、它们的根本性质以及在测度论与积分理论中的关键应用。

### 函数的规范分解：正部与负部

为了系统地处理任意实值函数，我们引入其**规范分解**（canonical decomposition）。对于一个在集合 $X$ 上定义的实值函数 $f: X \to \mathbb{R}$，我们逐点定义其**正部**（positive part）$f^+$ 和**负部**（negative part）$f^-$。

**定义**：函数 $f$ 的**正部** $f^+$ 定义为：
$$
f^+(x) = \max\{f(x), 0\}
$$
函数 $f$ 的**负部** $f^-$ 定义为：
$$
f^-(x) = \max\{-f(x), 0\}
$$

从定义中可以立刻观察到几个重要特性。首先，$f^+(x)$ 和 $f^-(x)$ 对于定义域中的所有 $x$ 都是非负的，即 $f^+(x) \ge 0$ 且 $f^-(x) \ge 0$。这一点至关重要，因为它允许我们将对任意函数的研究转化为对两个非负函数的研究，而非负函数在测度论中具有许多良好的性质。

其次，需要特别注意的是，尽管名称中带有“负”字，但函数的**负部 $f^-$ 始终是一个非负函数**。它实际上衡量的是原函数 $f(x)$ 在其取值为负的区域的“负的程度”的[绝对值](@entry_id:147688)。例如，如果 $f(x) = -5$，则 $f^+(x) = 0$，$f^-(x) = \max\{-(-5), 0\} = 5$。负部的另一种等价定义是 $f^-(x) = -\min\{f(x), 0\}$，这同样清晰地表明了其非负性。

最后，一个函数在任何一点上都不可能同时拥有非零的正部和非零的负部。这是因为：
- 如果 $f(x) \ge 0$，则 $f^+(x) = f(x)$ 且 $f^-(x) = 0$。
- 如果 $f(x)  0$，则 $f^+(x) = 0$ 且 $f^-(x) = -f(x)$。
因此，对于任意 $x \in X$，我们总有 $f^+(x) \cdot f^-(x) = 0$。这一性质表明，函数在一个点上的“正性”和“负性”是[互斥](@entry_id:752349)的。

### 基本代数恒等式

正部和负部的真正威力在于它们能够以一种简单而深刻的方式重构原函数及其[绝对值](@entry_id:147688)。

**定理**：对于任意实值函数 $f$，以下两个恒等式逐点成立：
1.  $f(x) = f^+(x) - f^-(x)$
2.  $|f(x)| = f^+(x) + f^-(x)$

**证明**：我们可以通过分情况讨论来验证这两个恒等式。
- 当 $f(x) \ge 0$ 时，我们有 $f^+(x) = f(x)$ 和 $f^-(x) = 0$。此时，$f^+(x) - f^-(x) = f(x) - 0 = f(x)$，$f^+(x) + f^-(x) = f(x) + 0 = f(x) = |f(x)|$。
- 当 $f(x)  0$ 时，我们有 $f^+(x) = 0$ 和 $f^-(x) = -f(x)$。此时，$f^+(x) - f^-(x) = 0 - (-f(x)) = f(x)$，$f^+(x) + f^-(x) = 0 + (-f(x)) = -f(x) = |f(x)|$。
两种情况都成立，因此定理得证。

这两个恒等式构成了[函数分解](@entry_id:197881)理论的代数基础。它们不仅展示了如何从正部和负部恢复原函数，还提供了一种反向求解 $f^+$ 和 $f^-$ 的方法。通过将上述两个方程视为关于 $f^+(x)$ 和 $f^-(x)$ 的线性方程组，我们可以解出它们关于 $f(x)$ 和 $|f(x)|$ 的表达式 [@problem_id:1435922]。

将两式相加得到 $2f^+(x) = f(x) + |f(x)|$。
将第二式减去第一式得到 $2f^-(x) = |f(x)| - f(x)$。
由此我们得到以下重要关系：

$$
f^+(x) = \frac{1}{2}(|f(x)| + f(x))
$$
$$
f^-(x) = \frac{1}{2}(|f(x)| - f(x))
$$

这些表达式在理论推导中极其有用，尤其是在证明可测性相关命题时。

### [可测性](@entry_id:199191)与分解

在测度论中，一个核心问题是：一个函数经过某种运算后，其[可测性](@entry_id:199191)是否得以保持？对于[函数的正部](@entry_id:203768)和负部分解，我们有以下重要的[可测性](@entry_id:199191)定理。

**定理**：若函数 $f$ 是 $(\mathcal{M}, \mathcal{B}(\mathbb{R}))$-可测的，则其正部 $f^+$、负部 $f^-$ 以及[绝对值](@entry_id:147688) $|f|$ 也都是可测的。

**证明**：证明这个定理的最优雅的方式就是利用我们刚刚推导出的代数恒等式。
1.  [绝对值函数](@entry_id:160606) $g(y) = |y|$ 是[连续函数](@entry_id:137361)，因此是波莱尔可测的。若 $f$ 可测，则复合函数 $|f| = g \circ f$ 也是可测的。
2.  根据恒等式 $f^+ = \frac{1}{2}(|f| + f)$ 和 $f^- = \frac{1}{2}(|f| - f)$，由于[可测函数](@entry_id:159040)的和、差以及与常数的乘积仍然是可测的，所以 $f^+$ 和 $f^-$ 都是可测的。

那么，这个定理的逆命题是否成立呢？如果 $f^+$ 和 $f^-$ 都是可测函数，那么 $f$ 是否一定可测？答案是肯定的，因为 $f = f^+ - f^-$，两个[可测函数](@entry_id:159040)的差仍然是可测的。

然而，一个更微妙的问题是：如果只知道 $f^+$ 是可测的，$f$ 是否一定可测？答案是否定的。这个结论揭示了[可测性](@entry_id:199191)性质的一个重要边界。我们可以构造一个反例来说明这一点 [@problem_id:1435916]。

考虑勒贝格测度空间 $(\mathbb{R}, \mathcal{L}, \lambda)$，并设 $A \subset [0, 1)$ 是一个勒贝格[不可测集](@entry_id:161390)（其存在性是测度论中的一个基本结果）。我们定义一个函数 $f: [0, 1) \to \mathbb{R}$ 如下：
$$
f(x) = \chi_A(x) - 1 = \begin{cases} 0  \text{if } x \in A \\ -1  \text{if } x \notin A \end{cases}
$$
其中 $\chi_A$ 是集合 $A$ 的特征函数。要判断 $f$ 是否可测，我们考察开集 $U = (-1/2, 1/2)$ 的[原像](@entry_id:150899)：$f^{-1}(U) = \{x \in [0, 1) \mid f(x) = 0\} = A$。由于 $A$ 是[不可测集](@entry_id:161390)，所以函数 $f$ 是不可测的。

然而，我们来计算它的正部 $f^+$：
$$
f^+(x) = \max\{f(x), 0\} = \max\{\chi_A(x) - 1, 0\} = 0
$$
因为 $f(x)$ 的取值只能是 $0$ 或 $-1$，所以 $\max\{f(x), 0\}$ 恒为 $0$。[常数函数](@entry_id:152060) $f^+(x) = 0$ 显然是可测的。因此，我们成功地构造了一个不[可测函数](@entry_id:159040)，其正部却是可测的。

### 分解的代数性质

理解[函数的正部](@entry_id:203768)和负部如何与加法、数乘等代数运算相互作用，对于在实际问题中应用它们至关重要。

#### 函数和的分解

对于两个函数 $f$ 和 $g$，它们的和的正部 $(f+g)^+$ 与各自正部的和 $f^+ + g^+$ 之间存在着重要的不等关系。

**定理**：对于任意函数 $f$ 和 $g$，以下不等式逐点成立：
1.  $(f+g)^+ \le f^+ + g^+$
2.  $(f+g)^- \le f^- + g^-$

**证明**：
1.  对于任意实数 $a$ 和 $b$，我们有 $a \le a^+$ 且 $b \le b^+$。因此 $a+b \le a^+ + b^+$。由于函数 $t \mapsto t^+ = \max\{t, 0\}$ 是单调不减的，所以 $(a+b)^+ \le (a^++b^+)^+$。又因为 $a^+ \ge 0$ 且 $b^+ \ge 0$，所以 $a^++b^+ \ge 0$，从而 $(a^++b^+)^+ = a^++b^+$。结合起来，我们得到 $(a+b)^+ \le a^+ + b^+$。将此结论应用于 $f(x)$ 和 $g(x)$ 即可证明第一条不等式 [@problem_id:1435908]。
2.  利用恒等式 $h^- = (-h)^+$，我们可以推导第二条不等式：$(f+g)^- = (-(f+g))^+ = (-f + (-g))^+$。对 $-f$ 和 $-g$ 应用第一条不等式，得到 $(-f + (-g))^+ \le (-f)^+ + (-g)^+ = f^- + g^-$。因此，$(f+g)^- \le f^- + g^-$ [@problem_id:1435908]。

需要强调的是，上述不等式中的等号通常不成立。例如，考虑函数 $f(x) = \sin(x)$ 和 $g(x) = \cos(x)$。在点 $x_0 = \frac{3\pi}{4}$ 处 [@problem_id:1435892]：
$f(x_0) = \sin(\frac{3\pi}{4}) = \frac{\sqrt{2}}{2}$，所以 $f^+(x_0) = \frac{\sqrt{2}}{2}$。
$g(x_0) = \cos(\frac{3\pi}{4}) = -\frac{\sqrt{2}}{2}$，所以 $g^+(x_0) = 0$。
因此，$f^+(x_0) + g^+(x_0) = \frac{\sqrt{2}}{2}$。
而 $f(x_0)+g(x_0) = 0$，所以 $(f+g)^+(x_0) = \max\{0, 0\} = 0$。
显然，在这种情况下，$(f+g)^+(x_0)  f^+(x_0) + g^+(x_0)$。

#### [标量乘法](@entry_id:155971)的分解

当函[数乘](@entry_id:155971)以一个常数 $c$ 时，其正部和负部的行为取决于 $c$ 的符号。
- 如果 $c \ge 0$，则 $(cf)^+ = c f^+$ 且 $(cf)^- = c f^-$。
- 如果 $c  0$，情况则会发生“翻转”。具体来说，我们有 $(cf)^+ = -c f^-$ 和 $(cf)^- = -c f^+$ [@problem_id:1435882]。

我们可以证明第一个等式：设 $c  0$，则 $c = -|c|$。
$(cf)^+(x) = \max\{cf(x), 0\} = \max\{-|c|f(x), 0\}$。
由于 $-|c|$ 是负数，当 $f(x) > 0$ 时，$-|c|f(x)  0$，$\max=0$；当 $f(x) \le 0$ 时，$-|c|f(x) \ge 0$，$\max = -|c|f(x) = c(-f(x))$。
这恰好等于 $-c \max\{-f(x), 0\} = -c f^-(x)$。第二个等式 $(cf)^- = -c f^+$ 的证明与此类似。

### 在勒贝格积分理论中的应用

[函数分解](@entry_id:197881)为正部和负部是勒贝格积分理论从非负函数推广到一般实值函数的核心步骤。
对于一个[可测函数](@entry_id:159040) $f$，其[勒贝格积分](@entry_id:140189)被定义为：
$$
\int_X f \,d\mu = \int_X f^+ \,d\mu - \int_X f^- \,d\mu
$$
这个定义的先决条件是右侧的两个积分至少有一个是有限的。如果两个积分都是有限的，则称函数 $f$ 是**[勒贝格可积](@entry_id:192052)的**（Lebesgue integrable）。

一个函数 $f$ 可积的充分必要条件是其[绝对值](@entry_id:147688) $|f|$ 的积分是有限的，即 $\int_X |f| \,d\mu  \infty$。这与正部和负部的可积性直接相关，因为 $|f| = f^+ + f^-$，所以
$$
\int_X |f| \,d\mu = \int_X (f^+ + f^-) \,d\mu = \int_X f^+ \,d\mu + \int_X f^- \,d\mu
$$
因此，$\int_X |f| \,d\mu$ 有限当且仅当 $\int_X f^+ \,d\mu$ 和 $\int_X f^- \,d\mu$ 都有限。

让我们通过一个例子来具体应用这些定义 [@problem_id:1435920]。考虑在区间 $(0, 1]$ 上的函数 $f$，它在每个子区间 $I_n = (\frac{1}{2^n}, \frac{1}{2^{n-1}}]$ 上取常数值 $c_n = (-1)^n \frac{2^n \alpha}{n^2}$，其中 $\alpha > 0$。我们要计算其负部的积分 $\int_{(0,1]} f^- d\lambda$。

根据定义，$f^-(x) = \max\{-f(x), 0\}$。在区间 $I_n$ 上，$f^-$ 的值是 $\max\{-c_n, 0\}$。
- 当 $n$ 是偶数时，$c_n > 0$，所以 $-c_n  0$，$f^-(x) = 0$。
- 当 $n$ 是奇数时，$c_n  0$，所以 $-c_n > 0$，$f^-(x) = -c_n = \frac{2^n \alpha}{n^2}$。

由于勒贝格积分对于非负函数具有[可数可加性](@entry_id:186580)，我们可以将总积分写成在每个子区间上积分的和：
$$
\int_{(0,1]} f^- \,d\lambda = \sum_{n=1}^{\infty} \int_{I_n} f^- \,d\lambda = \sum_{n=1}^{\infty} \max\{-c_n, 0\} \cdot \lambda(I_n)
$$
其中 $\lambda(I_n) = \frac{1}{2^{n-1}} - \frac{1}{2^n} = \frac{1}{2^n}$。代入数值：
$$
\int_{(0,1]} f^- \,d\lambda = \sum_{\substack{n=1 \\ n \text{ is odd}}}^{\infty} \left(\frac{2^n \alpha}{n^2}\right) \cdot \frac{1}{2^n} = \alpha \sum_{\substack{n=1 \\ n \text{ is odd}}}^{\infty} \frac{1}{n^2}
$$
利用已知的级数求和公式 $\sum_{k=1}^{\infty} \frac{1}{k^2} = \frac{\pi^2}{6}$，我们可以计算奇数项的平方倒数和：
$$
\sum_{\substack{n=1 \\ n \text{ is odd}}}^{\infty} \frac{1}{n^2} = \sum_{n=1}^{\infty} \frac{1}{n^2} - \sum_{m=1}^{\infty} \frac{1}{(2m)^2} = \frac{\pi^2}{6} - \frac{1}{4}\sum_{m=1}^{\infty} \frac{1}{m^2} = \frac{\pi^2}{6} - \frac{1}{4} \cdot \frac{\pi^2}{6} = \frac{3}{4} \cdot \frac{\pi^2}{6} = \frac{\pi^2}{8}
$$
因此，我们得到最终结果 $\int_{(0,1]} f^- \,d\lambda = \frac{\alpha \pi^2}{8}$。这个例子完美地展示了如何结合[函数分解](@entry_id:197881)的定义和[勒贝格积分](@entry_id:140189)的理论工具来解决具体问题。

### [若尔当分解](@entry_id:155802)的最小性

函数 $f$ 表示为其正部和负部之差 $f = f^+ - f^-$ 的形式，通常被称为函数的**[若尔当分解](@entry_id:155802)**（[Jordan decomposition](@entry_id:155802)）。这个分解在所有将 $f$ 表示为两个非负函数之差的分解中，具有一种深刻的**最小性**。

**定理**：设 $f = g - h$ 是函数 $f$ 的任意一个分解，其中 $g$ 和 $h$ 均为非负函数（即 $g(x) \ge 0, h(x) \ge 0$ 对所有 $x$ 成立）。那么，以下不等式逐点成立：
$$
g(x) \ge f^+(x) \quad \text{and} \quad h(x) \ge f^-(x)
$$

**证明**：我们来证明 $g(x) \ge f^+(x)$ [@problem_id:1435914]。由于 $f = g-h$，我们有 $g = f+h$。
- 如果 $f(x) \ge 0$，那么 $f^+(x) = f(x)$。因为 $h(x) \ge 0$，所以 $g(x) = f(x) + h(x) \ge f(x) = f^+(x)$。
- 如果 $f(x)  0$，那么 $f^+(x) = 0$。因为 $g$ 是非负函数，所以 $g(x) \ge 0 = f^+(x)$。
在任何情况下，不等式 $g(x) \ge f^+(x)$ 都成立。对 $h(x) \ge f^-(x)$ 的证明是类似的，只需考虑 $-f = h-g$ 并应用相同逻辑即可。

这个定理的直接推论是，在所有可能的分解 $f=g-h$ 中，[若尔当分解](@entry_id:155802) $f=f^+-f^-$ 的“正分量”和“负分量”是最小的。对不等式两边积分，我们得到：
$$
\int_X g \,d\mu \ge \int_X f^+ \,d\mu \quad \text{and} \quad \int_X h \,d\mu \ge \int_X f^- \,d\mu
$$
这意味着，[若尔当分解](@entry_id:155802)在积分意义下也是最小的。任何其他分解中的非负部分的积分都会大于或等于[若尔当分解](@entry_id:155802)中对应部分的积分。

例如，如果我们想找到一个分解 $f = g-h$ 使得积分 $\int_X g \,d\mu$ 最小化，那么最优选择就是 $g = f^+$ [@problem_id:1435914]。如果某个分解被构造成 $g(x) = f^+(x) + k(x)$ 和 $h(x) = f^-(x) + k(x)$，其中 $k(x)$ 是某个非负函数，那么这个分解是有效的，因为 $(f^+ + k) - (f^- + k) = f^+ - f^- = f$。然而，$\int g \,d\mu$ 将比最小可[能值](@entry_id:187992) $\int f^+ \,d\mu$ 多出 $\int k \,d\mu$ [@problem_id:1435917]。

总之，[函数的正部](@entry_id:203768)和负部分解不仅仅是一种技术上的便利，它深刻地揭示了实值函数的基本结构。从代数恒等式到[可测性](@entry_id:199191)，再到积分理论的构建和分解的最小性，这一概念贯穿了现代分析学的多个层面，为处理复杂函数提供了清晰而强大的理论框架。
