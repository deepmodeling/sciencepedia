## 引言
在从基础代数向高等分析过渡的过程中，[向量空间](@entry_id:151108)的研究需要引入能够度量“大小”和“距离”的工具，以便我们能够讨论收敛与极限等微积分核心概念。[赋范向量空间](@entry_id:274725)正是为此而生，它通过引入“范数”这一概念，为抽象的向量赋予了具体的几何长度。然而，仅仅拥有长度度量尚不足以构建一个稳固的分析框架，尤其是在处理[函数空间](@entry_id:143478)等无限维对象时，我们面临着极限过程可能“逃离”原空间的挑战。

本文旨在系统性地解决这一问题，引导读者从[赋范向量空间](@entry_id:274725)的基础概念，逐步深入到功能更为强大的巴拿赫空间。我们将阐明完备性——这一看似简单却极为深刻的性质——是如何成为现代[泛函分析](@entry_id:146220)的基石。通过学习本文，你将能够：

- 在第一章“原理与机制”中，掌握范数的基本公理，理解不同范数如何塑造空间的几何形态，并认识到完备性是确保分析可靠性的关键。
- 在第二章“应用与跨学科联系”中，见证这些抽象理论的巨大威力，了解它们如何统一并解释来自[偏微分方程](@entry_id:141332)、[量子化学](@entry_id:140193)和信号处理等领域的实际问题。
- 在第三章“动手实践”中，通过解决具体问题来巩固理论知识，将抽象的定义与可操作的计算联系起来，从而真正内化所学。

现在，让我们一同踏上这段旅程，从最基本的原理出发，逐步揭示[赋范向量空间](@entry_id:274725)与巴拿赫空间丰富而深刻的内在结构。

## 原理与机制

在本章中，我们将深入探讨[赋范向量空间](@entry_id:274725)的核心原理及其结构。我们将从赋予[向量空间](@entry_id:151108)长度概念的[范数公理](@entry_id:265195)出发，探索不同范数所揭示的丰富几何性质。随后，我们将引入完备性的关键概念，它将[赋范向量空间](@entry_id:274725)提升为强大的分析工具——巴拿赫空间。最后，我们将审视作用于这些空间上的映射，并阐明为何完备性是[泛函分析](@entry_id:146220)中一系列基本定理的基石。

### 范数空间的公理化基础

在[向量空间的基](@entry_id:191509)本[代数结构](@entry_id:137052)之上，分析学的研究需要引入拓扑和几何概念。实现这一目标的核心工具是**范数**（norm），它为[向量空间](@entry_id:151108)中的每个向量赋予了一个“长度”或“大小”的非负实数。一个定义在[向量空间](@entry_id:151108) $X$ 上的函数 $\| \cdot \|: X \to [0, \infty)$ 若要成为一个范数，必须对所有向量 $u, v \in X$ 和所有标量 $\alpha$ 满足以下三个公理：

1.  **正定性 (Definiteness)**：$\|v\| \ge 0$，且 $\|v\| = 0$ 当且仅当 $v=0$（零向量）。
2.  **[绝对齐次性](@entry_id:274917) (Absolute Homogeneity)**：$\|\alpha v\| = |\alpha| \|v\|$。
3.  **三角不等式 (Triangle Inequality)**：$\|u+v\| \le \|u\| + \|v\|$。

这三条公理共同捕捉了我们对长度的直观理解。三角不等式尤其重要，它推广了几何中“三角形两边之和大于第三边”的概念。

如果我们放宽第一个公理，只要求 $\|v\| \ge 0$ 而不要求 $\|v\|=0$ 必然推出 $v=0$，这样的函数被称为**[半范数](@entry_id:264573)** (seminorm)。[半范数](@entry_id:264573)允许非[零向量](@entry_id:156189)的“长度”为零。为了清晰地理解[范数与半范数](@entry_id:170322)的区别，我们可以构造一个具体的例子。考虑实[向量空间](@entry_id:151108) $\mathbb{R}^2$，我们可以定义一个函数 $p: \mathbb{R}^2 \to [0, \infty)$ 为 $p(x, y) = |x|$。这个函数只关注向量的第一个分量。它满足非负性、[绝对齐次性](@entry_id:274917)（$p(\alpha x, \alpha y) = |\alpha x| = |\alpha| |x| = |\alpha| p(x,y)$）和三角不等式（$p(x_1+x_2, y_1+y_2) = |x_1+x_2| \le |x_1|+|x_2| = p(x_1,y_1) + p(x_2,y_2)$），因此它是一个[半范数](@entry_id:264573)。然而，它不满足[正定性](@entry_id:149643)公理。任何形如 $(0, y)$ 且 $y \ne 0$ 的非[零向量](@entry_id:156189)，其[半范数](@entry_id:264573)值均为 $p(0, y) = |0| = 0$。这违反了范数的正定性要求。因此，$p(x, y) = |x|$ 是一个[半范数](@entry_id:264573)，但不是一个范数 [@problem_id:3057776]。这个例子揭示了，一个[半范数](@entry_id:264573)的**核** (kernel)，即所有使得[半范数](@entry_id:264573)为零的向量集合，可以是一个非平凡的[子空间](@entry_id:150286)（在此例中是 $y$ 轴），而范数的核必须仅仅是零向量组成的平凡[子空间](@entry_id:150286) $\{0\}$。

### 范数空间的几何学

范数不仅为向量提供了长度度量，还深刻地定义了空间的几何结构。一个范数所定义的**单位球** (unit ball) $B = \{x \in X : \|x\| \le 1\}$ 的形状，完全反映了该范数的性质。

在同一个[向量空间](@entry_id:151108)上，例如 $\mathbb{R}^n$，我们可以定义多种不同的范数。其中最著名的是 **$p$-范数**（或 $\ell^p$-范数）：
对于向量 $x = (x_1, \dots, x_n) \in \mathbb{R}^n$：
-   **$\ell^1$-范数** (或[曼哈顿范数](@entry_id:143036)): $\|x\|_1 = \sum_{i=1}^{n} |x_i|$
-   **$\ell^2$-范数** (或欧几里得范数): $\|x\|_2 = \left(\sum_{i=1}^{n} x_i^2\right)^{1/2}$
-   **$\ell^\infty$-范数** (或[上确界范数](@entry_id:145717)): $\|x\|_\infty = \max_{1 \le i \le n} |x_i|$

在 $\mathbb{R}^2$ 中，这些范数的单位球形状迥异：$\ell^2$ 范数的[单位球](@entry_id:142558)是我们熟悉的圆形；$\ell^1$ 范数的[单位球](@entry_id:142558)是一个旋转了45度的正方形（顶点在坐标轴上）；而 $\ell^\infty$ 范数的[单位球](@entry_id:142558)则是一个与坐标轴对齐的正方形。

[单位球](@entry_id:142558)的几何形状与一个重要的空间属性——**[严格凸性](@entry_id:193965)** (strict convexity) 密切相关。一个[赋范向量空间](@entry_id:274725)被称为严格凸的，如果其[单位球](@entry_id:142558)的边界上不包含任何线段。形式上，对于任意两个不同的单位向量 $x, y \in X$（即 $x \ne y$ 且 $\|x\| = \|y\| = 1$），它们的中点必须严格位于单位球内部，即 $\|\frac{x+y}{2}\|  1$。

[欧几里得空间](@entry_id:138052) $(\mathbb{R}^n, \|\cdot\|_2)$ 是严格凸的，但许多重要的空间并非如此。例如，$\ell^1$ 和 $\ell^\infty$ 空间就不是严格凸的。
-   在 $\ell^1$ 空间中，考虑两个不同的单位向量 $u^{(1)} = (1, 0, 0, \dots)$ 和 $v^{(1)} = (0, 1, 0, \dots)$。它们的 $\ell^1$-范数显然都是1。它们的中点是 $m^{(1)} = (\frac{1}{2}, \frac{1}{2}, 0, \dots)$，其范数为 $\|\frac{u^{(1)}+v^{(1)}}{2}\|_1 = |\frac{1}{2}| + |\frac{1}{2}| = 1$。中点仍然在[单位球](@entry_id:142558)的边界上，这违反了[严格凸性](@entry_id:193965)的定义。
-   在 $\ell^\infty$ 空间中，考虑 $u^{(\infty)} = (1, 0, 0, \dots)$ 和 $v^{(\infty)} = (1, 1, 0, \dots)$。它们都是单位向量，因为 $\|u^{(\infty)}\|_\infty = 1$ 且 $\|v^{(\infty)}\|_\infty = \sup\{|1|, |1|\} = 1$。它们的中点是 $m^{(\infty)} = (1, \frac{1}{2}, 0, \dots)$，其范数为 $\|\frac{u^{(\infty)}+v^{(\infty)}}{2}\|_\infty = \sup\{|1|, |\frac{1}{2}|\} = 1$。同样，中点位于单位球边界，证明了 $\ell^\infty$ 空间也不是严格凸的 [@problem_id:3057775]。

这些例子中的[单位球](@entry_id:142558)在某些地方是“平的”，允许两个边界点的连线段也位于边界上。

另一个深刻的几何问题是：一个范数是否由某个**[内积](@entry_id:158127)** (inner product) 诱导？由[内积](@entry_id:158127) $\langle \cdot, \cdot \rangle$ 诱导的范数定义为 $\|x\| = \sqrt{\langle x, x \rangle}$。一个重要的判别准则是**[平行四边形法则](@entry_id:154297)** (parallelogram law)：
$$ \|x+y\|^2 + \|x-y\|^2 = 2(\|x\|^2 + \|y\|^2) $$
一个范数是由[内积](@entry_id:158127)诱导的当且仅当它满足[平行四边形法则](@entry_id:154297)。我们可以用这个法则来检验 $\ell^1$-范数。在 $(\mathbb{R}^2, \|\cdot\|_1)$ 中，取 $x=(1,0)$ 和 $y=(0,1)$。我们计算可得：
$\|x\|_1 = 1$, $\|y\|_1 = 1$
$x+y = (1,1)$, $\|x+y\|_1 = |1|+|1|=2$
$x-y = (1,-1)$, $\|x-y\|_1 = |1|+|-1|=2$
代入[平行四边形法则](@entry_id:154297)的左边得到 $\|x+y\|_1^2 + \|x-y\|_1^2 = 2^2 + 2^2 = 8$。而右边是 $2(\|x\|_1^2 + \|y\|_1^2) = 2(1^2+1^2) = 4$。由于 $8 \ne 4$，$\ell^1$-范数不满足[平行四边形法则](@entry_id:154297)，因此它不能由任何[内积](@entry_id:158127)诱导 [@problem_id:3057761]。这从代数上证实了我们在几何直观上感受到的差异：只有当单位球是椭球体（在 $\mathbb{R}^n$ 中）时，范数才可能源于[内积](@entry_id:158127)。

### 完备性的概念：[巴拿赫空间](@entry_id:143833)

微积分的核心是极限过程。为了在抽象的[赋范向量空间](@entry_id:274725)中可靠地进行分析，我们需要确保极限过程的结果仍然留在空间之内。这引出了**完备性** (completeness) 的概念。

首先，我们定义**柯西序列** (Cauchy sequence)。一个序列 $\{x_n\}$ 被称为柯西序列，如果它的项最终变得任意地相互靠近。形式上，对于任意 $\varepsilon > 0$，存在一个整数 $N$，使得当 $m, n > N$ 时，$\|x_m - x_n\|  \varepsilon$。

一个[赋范向量空间](@entry_id:274725)如果其中的每一个柯西序列都收敛于该空间内的一个极限，那么这个空间就被称为**完备的** (complete)。一个完备的[赋范向量空间](@entry_id:274725)被称为**巴拿赫空间** (Banach space)。

所有有限维[赋范向量空间](@entry_id:274725)都是巴拿赫空间。然而，在无限维空间中，完备性并非理所当然。一个经典的非完备空间例子是 $C([0,1])$，即区间 $[0,1]$ 上的[连续函数空间](@entry_id:150395)，配以 $\ell^1$-范数 $\|f\|_1 = \int_0^1 |f(x)| dx$。考虑一个[函数序列](@entry_id:145607) $\{f_n\}$，其中 $f_n$ 在 $[0, 1/2 - 1/n]$ 上为0，在 $[1/2, 1]$ 上为1，并在中间的短区间线性过渡。这个序列在 $\ell^1$-范数下是柯西序列，因为它在 $L^1$ 意义下收敛于一个[阶跃函数](@entry_id:159192)（在 $x=1/2$ 处从0跳到1）。然而，这个极限函数是不连续的，因此它不属于 $C([0,1])$。由于这个柯西序列在空间内没有极限，所以 $(C([0,1]), \|\cdot\|_1)$ 不是一个巴拿赫空间 [@problem_id:2327333] [@problem_id:3057765]。

完备性有几个等价的刻画，它们为我们提供了更深刻的理解和更灵活的工具 [@problem_id:1861311]：
-   **等价刻画 I (子[序列收敛](@entry_id:143579))**：一个[赋范空间](@entry_id:137032)是[巴拿赫空间](@entry_id:143833)，当且仅当每个柯西序列都有一个收敛的子序列。这看起来比原始定义弱，但实际上是等价的，因为如果一个柯西序列有一个收敛的子序列，那么整个序列必然收敛到相同的极限。

-   **等价刻画 II ([绝对收敛级数](@entry_id:162098))**：一个[赋范空间](@entry_id:137032)是巴拿赫空间，当且仅当空间中每个**绝对收敛**的级数都是收敛的。一个级数 $\sum x_k$ 被称为绝对收敛，如果[实数级数](@entry_id:185930) $\sum \|x_k\|$ 收敛。这个判据在实践中非常有用。例如，要证明 $(C([0,1]), \|\cdot\|_\infty)$（赋以[上确界范数](@entry_id:145717)）是巴拿赫空间，我们可以证明任何在其中绝对收敛的[函数级数](@entry_id:139536)，其[部分和序列](@entry_id:161258)[一致收敛](@entry_id:146084)于一个[连续函数](@entry_id:137361)。

-   **等价刻画 III (康托尔交集定理)**：一个[赋范空间](@entry_id:137032)是[巴拿赫空间](@entry_id:143833)，当且仅当它满足康托尔交集性质。即对于任意一列非空、闭、有界的嵌套[子集](@entry_id:261956) $C_1 \supseteq C_2 \supseteq \dots$，如果它们的直径 $\text{diam}(C_n) = \sup\{\|x-y\| : x, y \in C_n\}$ 趋于0，那么它们的交集 $\bigcap_{n=1}^\infty C_n$ 非空（且恰好包含一个点）。这个定理将完备性与空间的拓扑性质联系起来。

值得注意的是，另一个看似相关的性质——单位球的紧致性——与完备性并不等价。**紧致性** (compactness) 是一个更强的条件。根据**里斯引理** (Riesz's Lemma)，一个[赋范空间](@entry_id:137032)的闭[单位球](@entry_id:142558)是紧的，当且仅当该空间是有限维的。因此，在任何无限维巴拿赫空间（如 $\ell^p$ 空间或 $C([0,1])$）中，闭[单位球](@entry_id:142558)都不是紧的。这是有限维和无限维空间之间最根本的区别之一。

### [赋范空间](@entry_id:137032)之间的映射

泛函分析的核心研究对象之一是[赋范空间](@entry_id:137032)之间的**线性算子** (linear operator)。对于线性算子 $T: X \to Y$，其**连续性** (continuity) 和**有界性** (boundedness) 是等价的。一个[线性算子](@entry_id:149003) $T$ 是有界的，如果存在一个常数 $M \ge 0$ 使得对于所有 $x \in X$，都有 $\|Tx\|_Y \le M \|x\|_X$。满足此条件的最小 $M$ 值被称为**[算子范数](@entry_id:752960)** (operator norm)，记为 $\|T\|$。

当一个[线性算子](@entry_id:149003)是[双射](@entry_id:138092)（既是[单射](@entry_id:183792)又是满射）时，我们可以讨论空间之间的同构关系。
-   **[拓扑同构](@entry_id:263643) (Topological Isomorphism)**：如果一个[双射](@entry_id:138092)[线性算子](@entry_id:149003) $T: X \to Y$ 及其逆算子 $T^{-1}: Y \to X$ 都是有界的，则称 $T$ 是一个[拓扑同构](@entry_id:263643)或**[巴拿赫空间](@entry_id:143833)同构**。这意味着 $X$ 和 $Y$ 在拓扑上是无法区分的（例如，开集[一一对应](@entry_id:143935)）。
-   **[等距同构](@entry_id:273188) (Isometric Isomorphism)**：如果一个[双射](@entry_id:138092)[线性算子](@entry_id:149003) $T$ 保持范数，即 $\|Tx\|_Y = \|x\|_X$ 对所有 $x$ 成立，则称 $T$ 是一个[等距同构](@entry_id:273188)。这是一个更强的条件，意味着 $X$ 和 $Y$ 在几何上是[全等](@entry_id:273198)的。

为了阐明这两种同构的区别，我们考察在 $\mathbb{R}^n$ ($n \ge 2$) 上的[恒等映射](@entry_id:634191) $I: (\mathbb{R}^n, \|\cdot\|_2) \to (\mathbb{R}^n, \|\cdot\|_1)$。由于在有限维空间中所有范数都是等价的（即存在常数 $c_1, c_2 > 0$ 使得 $c_1\|x\|_2 \le \|x\|_1 \le c_2\|x\|_2$），恒等映射 $I$ 和其逆 $I^{-1}$ 都是有界的。因此，$I$ 是一个[巴拿赫空间](@entry_id:143833)同构，这两个空间在拓扑上是相同的。然而，$I$ 并不是一个[等距同构](@entry_id:273188)，因为通常情况下 $\|x\|_1 \ne \|x\|_2$（例如，对于 $x=(1,1,0,\dots,0)$, $\|x\|_2=\sqrt{2}$ 而 $\|x\|_1=2$）。我们可以精确计算出算子范数：$\|I\| = \sup_{\|x\|_2=1} \|x\|_1 = \sqrt{n}$，而 $\|I^{-1}\| = \sup_{\|x\|_1=1} \|x\|_2 = 1$ [@problem_id:3057760]。这量化了在两个范数之间切换时[向量长度](@entry_id:156432)可能发生的最大“拉伸”。

### “三大定理”与完备性的作用

巴拿赫空间的理论基石是三个被称为泛函分析“三大定理”的深刻结果：[哈恩-巴拿赫定理](@entry_id:145406) (Hahn-Banach Theorem)、[一致有界性原理](@entry_id:144502) (Uniform Boundedness Principle) 和[开映射定理](@entry_id:137363) (Open Mapping Theorem)。这些定理揭示了巴拿赫空间的丰富结构，而它们的成立几乎都离不开**完备性**这一关键假设。下面我们将通过一系列反例来展示，如果缺少完备性，这些强大的定理将不再成立。

#### [开映射定理](@entry_id:137363) (Open Mapping Theorem)

该定理指出：若 $T: X \to Y$ 是巴拿赫空间 $X$ 到[巴拿赫空间](@entry_id:143833) $Y$ 的一个有界、满射的线性算子，则 $T$ 是一个**[开映射](@entry_id:155659)**（即它将 $X$ 中的开集映为 $Y$ 中的开集）。

考虑[恒等算子](@entry_id:204623) $T: (C([0,1]), \|\cdot\|_\infty) \to (C([0,1]), \|\cdot\|_1)$。我们知道，定义域 $(C([0,1]), \|\cdot\|_\infty)$ 是一个[巴拿赫空间](@entry_id:143833)。算子 $T$ 是线性的、有界的（因为 $\|f\|_1 \le \|f\|_\infty$），并且是满射的。然而，可以证明这个算子不是一个[开映射](@entry_id:155659)。根据[开映射定理](@entry_id:137363)，这意味着某个假设必须不成立。这里唯一不成立的假设就是陪域 $(C([0,1]), \|\cdot\|_1)$ 不是一个巴拿赫空间，正如我们之前所见 [@problem_id:2327333]。

#### [闭图像定理](@entry_id:136827) (Closed Graph Theorem)

作为[开映射定理](@entry_id:137363)的推论，[闭图像定理](@entry_id:136827)提供了一个判断算子连续性的便捷准则。它指出：若 $T: X \to Y$ 是[巴拿赫空间](@entry_id:143833) $X$ 到[巴拿赫空间](@entry_id:143833) $Y$ 的一个[线性算子](@entry_id:149003)，如果 $T$ 的**图像** (graph) $\text{Graph}(T) = \{(x, Tx) : x \in X\}$ 是乘[积空间](@entry_id:151693) $X \times Y$ 中的一个[闭集](@entry_id:136446)，则 $T$ 是连续的。

现在考虑[恒等算子](@entry_id:204623) $T: (C([0,1]), \|\cdot\|_1) \to (C([0,1]), \|\cdot\|_2)$。我们可以证明这个[算子的图像](@entry_id:261846)是闭的。然而，通过构造一个特定的函数序列，例如 $f_n(x)$ 是一个在 $[0, 2/n]$ 上支撑的“尖峰”函数，我们可以计算出比值 $\frac{\|T(f_n)\|_2}{\|f_n\|_1} = \sqrt{\frac{2n}{3}}$。当 $n \to \infty$ 时，这个比值无界，表明算子 $T$ 是无界的，因而不是连续的。[闭图像定理](@entry_id:136827)的结论在此失效，其根本原因在于[定义域和陪域](@entry_id:159300)都不是巴拿赫空间 [@problem_id:3057765]。

#### [一致有界性原理](@entry_id:144502) (Uniform Boundedness Principle, UBP)

该原理（也称[巴拿赫-斯坦豪斯定理](@entry_id:140127), Banach-Steinhaus Theorem）指出：设 $X$ 是一个巴拿赫空间， $Y$ 是一个[赋范空间](@entry_id:137032)。如果一族从 $X$到 $Y$ 的[连续线性算子](@entry_id:154042) $\{T_\alpha\}$ 是**[逐点有界](@entry_id:141887)**的（即对每个固定的 $x \in X$，集合 $\{\|T_\alpha(x)\|\}$ 是有界的），那么这族算子是**一致有界**的（即它们的[算子范数](@entry_id:752960)集合 $\{\|T_\alpha\|\}$ 是有界的）。

这个定理的威力在于它从关于单个点的“弱”信息（[逐点有界](@entry_id:141887)）推出了关于整个空间的“强”信息（一致有界）。完备性在这里至关重要。考虑空间 $c_{00}$，即所有有限支撑的实序列空间，赋以 $\ell^1$-范数。我们已知这个空间是不完备的。定义一族线性泛函 $T_n: c_{00} \to \mathbb{R}$ 为 $T_n(x) = n x_n$。对于任何一个 $x \in c_{00}$，由于它只有有限个非零项，所以序列 $\{|T_n(x)|\}_{n=1}^\infty$ 中只有有限个非零值，因此该序列是有界的。这意味着 $\{T_n\}$ 是[逐点有界](@entry_id:141887)的。然而，通过计算可知，算子 $T_n$ 的范数为 $\|T_n\| = n$。因此，这族算子的范数集合 $\{1, 2, 3, \dots\}$ 是无界的。[一致有界性原理](@entry_id:144502)的结论不成立，正是因为定义域 $c_{00}$ 不是一个巴拿赫空间 [@problem_id:3057759]。

### 对偶性与进一步的结构

对偶性是泛函分析中的一个核心概念，它通过研究作用于空间上的线性泛函来揭示空间自身的结构。

对于一个[赋范空间](@entry_id:137032) $X$，其**连续[对偶空间](@entry_id:146945)** (continuous dual space)，记为 $X^*$，是所有从 $X$到其标量域 $\mathbb{R}$（或$\mathbb{C}$）的[连续线性泛函](@entry_id:262913)组成的集合。$X^*$ 本身也是一个[赋范向量空间](@entry_id:274725)，其范数是[算子范数](@entry_id:752960)。一个非常重要的性质是：**[对偶空间](@entry_id:146945) $X^*$ 总是完备的**，即总是一个巴拿赫空间，无论 $X$ 本身是否完备。

对于[有限维空间](@entry_id:151571) $X = (\mathbb{R}^n, \|\cdot\|)$，其上的任何[线性泛函](@entry_id:276136) $f$ 都可以唯一地表示为 $f(x) = \sum_{i=1}^n x_i y_i$，其中 $y = (y_1, \dots, y_n)$是 $\mathbb{R}^n$ 中的一个固定向量。这建立了一个从 $\mathbb{R}^n$ 到 $X^*$ 的[线性同构](@entry_id:270529)。我们可以进一步定义 $\mathbb{R}^n$ 上的**[对偶范数](@entry_id:200340)** $\|\cdot\|_*$：
$$ \|y\|_* = \|f_y\| = \sup_{\|x\| \le 1} |f_y(x)| = \sup_{\|x\| \le 1} \left| \sum_{i=1}^n x_i y_i \right| $$
在这个[对偶范数](@entry_id:200340)下，空间 $(\mathbb{R}^n, \|\cdot\|_*)$ 与 $(X^*, \|\cdot\|)$ [等距同构](@entry_id:273188) [@problem_id:3057774]。例如，$\ell^p$-范数的[对偶范数](@entry_id:200340)是 $\ell^q$-范数，其中 $\frac{1}{p} + \frac{1}{q} = 1$。

最后，我们讨论[赋范空间](@entry_id:137032)的基。与代数中的[哈梅尔基](@entry_id:155991)（Hamel basis，允许任意有限线性组合）不同，分析中更常用的是**邵德基** (Schauder basis)。一个序列 $\{x_n\}$ 是巴拿赫空间 $X$ 的邵德基，如果每个 $x \in X$ 都可以唯一地表示为[范数收敛](@entry_id:261322)的级数 $x = \sum_{n=1}^\infty a_n x_n$。

拥有邵德基与空间的拓扑性质密切相关。一个重要的结果是：如果一个[巴拿赫空间](@entry_id:143833)有邵德基，那么它一定是**可分的** (separable)，即可分的意味着它包含一个可数的[稠密子集](@entry_id:264458)。例如，所有 $\ell^p$ 空间（对于 $1 \le p  \infty$）和 $c_0$（收敛到零的序列空间）都是可分的，并且它们都有标准的邵德基。

然而，并非所有[巴拿赫空间](@entry_id:143833)都是可分的。一个关键的反例是 $\ell^\infty$ 空间。考虑所有由0和1组成的序列构成的集合 $S \subset \ell^\infty$。这个集合是不可数的（通过康托尔对角线论证可知），并且其中任意两个不同的序列 $s, s'$ 之间的距离 $\|s-s'\|_\infty = 1$。一个空间如果包含这样一种不可数的、点与点之间距离保持恒定的集合，它就不可能是可分的。既然 $\ell^\infty$ 不是可分的，根据上述结论，它必然没有邵德基 [@problem_id:3057768]。这一事实突显了不同无限维巴拿赫空间之间深刻的结构性差异。
