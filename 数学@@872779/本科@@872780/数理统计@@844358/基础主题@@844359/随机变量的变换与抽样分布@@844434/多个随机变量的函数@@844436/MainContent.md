## 引言
在科学与工程的众多领域中，我们所关注的量往往不是直接测量的单一实体，而是由多个基础[随机变量](@entry_id:195330)通过函数关系组合而成的派生量。例如，一个复杂系统的总寿命是其各组件寿命的函数，一项投资组合的风险与回报由其包含的各项资产的随机表现共同决定，物理测量中的最终结果也常常依赖于多个具有不确定性的输入量。因此，能够系统地分析和预测这些派生[随机变量](@entry_id:195330)的统计特性——如它们的均值、[方差](@entry_id:200758)乃至完整的[概率分布](@entry_id:146404)——是进行定量分析、[风险评估](@entry_id:170894)和[科学推断](@entry_id:155119)的基石。

本文旨在填补从理解单个[随机变量](@entry_id:195330)到掌握它们复杂组合之间的知识鸿沟。我们将深入探讨处理多[随机变量](@entry_id:195330)函数的关键原理和实用技术。

在“原理与机制”一章中，我们将建立起一套强大的数学工具箱，包括处理[线性组合](@entry_id:154743)的矩、用于求和的卷积法、巧妙的[矩母函数](@entry_id:154347)法，以及处理一般变换的[雅可比方法](@entry_id:270947)。随后，在“应用与跨学科联系”一章中，我们将展示这些抽象的工具如何在工程、金融、经济学和数据科学等不同领域中解决具体问题，揭示其作为连接不同学科的桥梁作用。最后，通过“动手实践”部分，读者将有机会运用所学知识解决实际问题，从而巩固和深化理解。

现在，让我们从构建这些分析工具的核心原理与机制开始。

## 原理与机制

在概率论和统计学的许多应用中，我们感兴趣的量通常不是通过直接观测得到的，而是由一个或多个其他[随机变量](@entry_id:195330)通过函数关系派生出来的。例如，一个系统的总故障时间可能是其各个独立组件寿命之和；一项投资组合的总体回报是其包含的各项资产回报的加权平均；在信号处理中，信噪比是信号功率与噪声功率的比值。因此，理解和量化这些派生[随机变量](@entry_id:195330)的统计特性，例如它们的矩和[概率分布](@entry_id:146404)，是至关重要的。本章将系统地介绍分析多[随机变量](@entry_id:195330)函数的关键原理和核心机制。

### [线性组合](@entry_id:154743)的矩

在探究一个[随机变量](@entry_id:195330)函数的完整[分布](@entry_id:182848)之前，我们往往可以先从其关键的数字特征入手，例如期望和[方差](@entry_id:200758)。对于多个[随机变量的线性组合](@entry_id:275666)，其矩的计算具有简洁而优美的性质。

设 $X_1, X_2, \dots, X_n$ 为一组[随机变量](@entry_id:195330)（不一定独立），$a_1, a_2, \dots, a_n$ 为一组常数。我们定义一个新的[随机变量](@entry_id:195330) $Y$ 为它们的**[线性组合](@entry_id:154743)**：
$Y = a_1 X_1 + a_2 X_2 + \dots + a_n X_n = \sum_{i=1}^{n} a_i X_i$

**期望**最显著的性质是其**线性性**。无论这些[随机变量](@entry_id:195330)之间是否存在相关性，线性组合的期望总是等于[期望的线性](@entry_id:273513)组合：
$\mathbb{E}[Y] = \mathbb{E}\left[\sum_{i=1}^{n} a_i X_i\right] = \sum_{i=1}^{n} a_i \mathbb{E}[X_i]$

这个性质非常强大且应用广泛。例如，考虑一种由两种不同聚合物（A和B）混合制成的[复合材料](@entry_id:139856)。如果聚合物A的拉伸强度 $S_A$ 的均值为 $\mu_A$，聚合物B的拉伸强度 $S_B$ 的均值为 $\mu_B$，那么由两者等分混合而成的[复合材料](@entry_id:139856)的强度模型 $S_{comp} = \frac{S_A + S_B}{2}$ 的期望强度可以直接计算得出 [@problem_id:1919072]：
$\mathbb{E}[S_{comp}] = \mathbb{E}\left[\frac{1}{2}S_A + \frac{1}{2}S_B\right] = \frac{1}{2}\mathbb{E}[S_A] + \frac{1}{2}\mathbb{E}[S_B] = \frac{\mu_A + \mu_B}{2}$

**[方差](@entry_id:200758)**的计算则更为复杂，因为它涉及到变量之间的相互关系。对于两个[随机变量](@entry_id:195330) $X$ 和 $Y$ 的线性组合 $Z = aX + bY$，其[方差](@entry_id:200758)的一般公式为：
$\mathrm{Var}(Z) = \mathrm{Var}(aX + bY) = a^2\mathrm{Var}(X) + b^2\mathrm{Var}(Y) + 2ab\,\mathrm{Cov}(X, Y)$
其中 $\mathrm{Cov}(X, Y)$ 是 $X$ 和 $Y$ 的**协[方差](@entry_id:200758)**，定义为 $\mathrm{Cov}(X, Y) = \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])]$。协[方差](@entry_id:200758)度量了两个变量线性相关的程度和方向。

在许多实际问题中，变量之间并非[相互独立](@entry_id:273670)。例如，在制造过程中，基板的长度 $L$ 和宽度 $W$ 可能因为机器的共同影响而存在相关性，即 $\mathrm{Cov}(L, W) = \sigma_{LW} \ne 0$。如果一个质量控制指标被定义为 $S = \alpha L - \beta W$，那么它的[方差](@entry_id:200758)就必须考虑这种相关性 [@problem_id:1919098]：
$\mathrm{Var}(S) = \mathrm{Var}(\alpha L - \beta W) = \alpha^2\mathrm{Var}(L) + (-\beta)^2\mathrm{Var}(W) + 2(\alpha)(-\beta)\mathrm{Cov}(L, W) = \alpha^2\sigma_L^2 + \beta^2\sigma_W^2 - 2\alpha\beta\sigma_{LW}$
这个公式表明，如果长度和宽度正相关（$\sigma_{LW} > 0$），那么这种相关性会减小指标 $S$ 的[方差](@entry_id:200758)；反之，如果它们负相关，则会增大 $S$ 的[方差](@entry_id:200758)。

当[随机变量](@entry_id:195330)**[相互独立](@entry_id:273670)**时，情况得到极大简化。根据定义，[独立随机变量](@entry_id:273896)的协[方差](@entry_id:200758)为零。因此，对于相互独立的 $X_1, \dots, X_n$，[线性组合](@entry_id:154743)的[方差](@entry_id:200758)就是[方差](@entry_id:200758)的（加权）和：
$\mathrm{Var}\left(\sum_{i=1}^{n} a_i X_i\right) = \sum_{i=1}^{n} a_i^2 \mathrm{Var}(X_i)$

回到前述的[复合材料](@entry_id:139856)例子，如果聚合物A和B的强度 $S_A$ 和 $S_B$ 是统计独立的，且[方差](@entry_id:200758)分别为 $\sigma_A^2$ 和 $\sigma_B^2$，那么[复合材料](@entry_id:139856)强度的[方差](@entry_id:200758)为 [@problem_id:1919072]：
$\mathrm{Var}(S_{comp}) = \mathrm{Var}\left(\frac{1}{2}S_A + \frac{1}{2}S_B\right) = \left(\frac{1}{2}\right)^2\mathrm{Var}(S_A) + \left(\frac{1}{2}\right)^2\mathrm{Var}(S_B) = \frac{\sigma_A^2 + \sigma_B^2}{4}$
这个结果说明，通过平均两个独立的随机源，我们得到的[复合材料](@entry_id:139856)不仅期望强度是两者均值，其波动性（[方差](@entry_id:200758)）也得到了有效降低。

### 函数[分布](@entry_id:182848)的求解方法

虽然矩提供了关于[分布](@entry_id:182848)中心趋势和离散程度的重要信息，但它们并不能完全描述一个[随机变量](@entry_id:195330)。为了获得完整的图像，我们需要确定其[概率分布](@entry_id:146404)函数（PDF或PMF）。本节将介绍几种核心方法。

#### 直接法 (离散情形)

对于[离散随机变量的函数](@entry_id:184970)，其[概率质量函数](@entry_id:265484)（PMF）通常可以通过基本概率论的原理直接推导。其核心思想是：对于函数 $Z = g(X, Y)$ 的每一个可能取值 $z$，找出所有使得 $g(x, y) = z$ 的原始变量取值对 $(x, y)$，然后将这些取值对的概率相加。

考虑一个简单的例子：一个初创公司独立进行两场线上营销活动A和B。活动A成功的概率为 $p_A$，其结果由伯努利[随机变量](@entry_id:195330) $X$ 表示；活动B成功的概率为 $p_B$，由独立的伯努利[随机变量](@entry_id:195330) $Y$ 表示。总的成功活动数 $Z = X+Y$ 是一个新的[随机变量](@entry_id:195330)。为了求出 $Z$ 的PMF，我们可以枚举其所有可能的取值 $\{0, 1, 2\}$ [@problem_id:1919086]。

-   $Z=0$：这当且仅当 $X=0$ 且 $Y=0$ 时发生。由于独立性，$P(Z=0) = P(X=0, Y=0) = P(X=0)P(Y=0) = (1-p_A)(1-p_B)$。

-   $Z=2$：这当且仅当 $X=1$ 且 $Y=1$ 时发生。由于独立性，$P(Z=2) = P(X=1, Y=1) = P(X=1)P(Y=1) = p_A p_B$。

-   $Z=1$：这在两种[互斥](@entry_id:752349)的情况下发生：(A成功, B失败) 或 (A失败, B成功)，即 $(X=1, Y=0)$ 或 $(X=0, Y=1)$。因此，$P(Z=1) = P(X=1, Y=0) + P(X=0, Y=1) = p_A(1-p_B) + (1-p_A)p_B$。

通过这种枚举和加总的方式，我们便完整地确定了新变量 $Z$ 的[概率分布](@entry_id:146404)。

#### 卷积法

对于两个**独立[连续随机变量](@entry_id:166541)**的和 $Z = X+Y$，其[概率密度函数](@entry_id:140610)（PDF）可以通过一种称为**卷积**的数学运算得到。其背后的逻辑与离散情况类似：为了使 $Z$ 取到某个值 $z$，我们假设 $X$ 取了值 $x$，那么 $Y$ 就必须取到值 $z-x$。由于 $X$ 可以取任意值，我们需要对所有可能的 $x$ 进行积分。

若 $X$ 和 $Y$ 的PDF分别为 $f_X(x)$ 和 $f_Y(y)$，则 $Z=X+Y$ 的PDF $f_Z(z)$ 由以下**[卷积积分](@entry_id:155865)**给出：
$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) \,dx$

这个公式在理论和实践中都极为重要。考虑一个由三个独立阶段组成的自动化装配过程，每个阶段耗时 $T_1, T_2, T_3$ 均服从 $[0, 1]$ 上的[均匀分布](@entry_id:194597)。总耗时 $S = T_1+T_2+T_3$ 的[分布](@entry_id:182848)可以通过连续应用卷积来求解 [@problem_id:1919067]。

首先，我们计算 $U = T_1+T_2$ 的[分布](@entry_id:182848)。$T_1, T_2$ 的PDF为 $f_T(t)=1$ for $t \in [0,1]$。$U$ 的PDF为：
$f_U(u) = \int_0^1 f_T(x) f_T(u-x) \,dx$
这个积分的积分限取决于 $f_T(u-x)$ 非零的区域，即 $0 \le u-x \le 1$ 或 $u-1 \le x \le u$。结合 $x \in [0,1]$，最终得到一个在 $[0, 2]$ 区间上的三[角分布](@entry_id:193827)。

然后，我们再计算 $S = U+T_3$ 的[分布](@entry_id:182848)，即把刚刚得到的三角分布 $f_U(u)$ 与另一个[均匀分布](@entry_id:194597) $f_T(t)$ 进行卷积。这个过程涉及在分段定义的函数上进行积分，需要根据 $s$ 的不同取值范围仔细确定积分区间。最终得到一个由三段二次多项式构成的平滑[钟形曲线](@entry_id:150817)，即著名的Irwin-Hall[分布](@entry_id:182848)。这个例子充分展示了卷积方法的威力，也揭示了即使从最简单的[均匀分布](@entry_id:194597)出发，其和的[分布](@entry_id:182848)也会迅速变得复杂并趋向于中心集中的形态。

对于[离散随机变量](@entry_id:163471)，也存在类似的**[离散卷积](@entry_id:160939)**公式。对于独立的 $X, Y$，其和 $Z=X+Y$ 的PMF为：
$P(Z=k) = \sum_{i=-\infty}^{\infty} P(X=i) P(Y=k-i)$

一个经典例子是两个独立[泊松分布的和](@entry_id:263166)。设 $X \sim \text{Poisson}(\lambda_1)$ 和 $Y \sim \text{Poisson}(\lambda_2)$，则 $Z=X+Y$ 的PMF为 [@problem_id:1919070]：
$P(Z=k) = \sum_{x=0}^{k} P(X=x) P(Y=k-x) = \sum_{x=0}^{k} \frac{e^{-\lambda_1}\lambda_1^x}{x!} \frac{e^{-\lambda_2}\lambda_2^{k-x}}{(k-x)!}$
通过代数整理并利用[二项式定理](@entry_id:276665)，可以证明上式等于 $\frac{e^{-(\lambda_1+\lambda_2)}(\lambda_1+\lambda_2)^k}{k!}$，这表明 $Z$ 服从参数为 $\lambda_1+\lambda_2$ 的[泊松分布](@entry_id:147769)。这个性质被称为[泊松分布](@entry_id:147769)的**再生性**。

#### 矩生成函数法

虽然卷积在理论上很完美，但实际计算可能非常繁琐，正如我们在泊松求和的例子中看到的那样。对于**独立**[随机变量](@entry_id:195330)的和，**矩生成函数（MGF）**法提供了一条更为便捷的路径。

MGF的核心性质是：如果 $X$ 和 $Y$ 是独立的，那么它们的和 $Z=X+Y$ 的MGF就是它们各自MGF的乘积：
$M_Z(t) = \mathbb{E}[e^{t(X+Y)}] = \mathbb{E}[e^{tX}e^{tY}] = \mathbb{E}[e^{tX}]\mathbb{E}[e^{tY}] = M_X(t) M_Y(t)$

这个性质与MGF的**唯一性定理**（即一个[分布](@entry_id:182848)完全由其MGF唯一确定）相结合，构成了一种强大的[分布](@entry_id:182848)推导工具。其策略是：
1.  求出各个[独立变量](@entry_id:267118)的MGF。
2.  将它们相乘得到和的MGF。
3.  识别出这个新的MGF所对应的[分布](@entry_id:182848)。

让我们重新审视两个[独立泊松变量之和](@entry_id:186704)的问题 [@problem_id:1919070]。一个[泊松分布](@entry_id:147769) $\text{Poisson}(\lambda)$ 的MGF是 $M(t) = \exp(\lambda(e^t-1))$。因此，对于 $Z=X+Y$，其中 $X \sim \text{Poisson}(\lambda_1)$ 和 $Y \sim \text{Poisson}(\lambda_2)$ 独立，
$M_Z(t) = M_X(t)M_Y(t) = \exp(\lambda_1(e^t-1)) \exp(\lambda_2(e^t-1)) = \exp((\lambda_1+\lambda_2)(e^t-1))$
我们立刻就能认出，这是参数为 $\lambda_1+\lambda_2$ 的泊松分布的MGF。这一推导过程远比执行卷积求和要简单。

当变量服从伽玛[分布](@entry_id:182848)时，MGF法的优势更加明显。伽玛[分布](@entry_id:182848) $\text{Gamma}(\alpha, \theta)$（$\alpha$为形状参数，$\theta$为[尺度参数](@entry_id:268705)）的MGF为 $M(t) = (1-\theta t)^{-\alpha}$。如果 $X_1 \sim \text{Gamma}(\alpha_1, \theta)$ 和 $X_2 \sim \text{Gamma}(\alpha_2, \theta)$ 独立，那么它们的和 $Y = X_1+X_2$ 的MGF为 [@problem_id:1919091]：
$M_Y(t) = M_{X_1}(t) M_{X_2}(t) = (1-\theta t)^{-\alpha_1} (1-\theta t)^{-\alpha_2} = (1-\theta t)^{-(\alpha_1+\alpha_2)}$
这正是 $\text{Gamma}(\alpha_1+\alpha_2, \theta)$ [分布](@entry_id:182848)的MGF。这表明，具有相同[尺度参数](@entry_id:268705)的独立伽玛变量之和仍然是伽玛变量，其形状参数为原始形状参数之和。这个重要的结果若用卷积法证明将非常困难。

#### [累积分布函数法](@entry_id:268506)

**[累积分布函数](@entry_id:143135)（CDF）**法是一种通用性很强的方法，它不局限于求和运算，对求最大值、最小值等**[顺序统计量](@entry_id:266649)**尤为有效。其基本步骤是：
1.  写出新变量 $Z=g(X,Y)$ 的CDF定义：$F_Z(z) = P(Z \le z)$。
2.  将事件 $\{Z \le z\}$ 用原始变量 $X, Y$ 的事件来[等价表示](@entry_id:187047)。
3.  计算这个等价事件的概率。
4.  通过求导（连续情况，$f_Z(z) = F_Z'(z)$）或差分（离散情况，$P(Z=k) = F_Z(k) - F_Z(k-1)$）得到概率密度或[质量函数](@entry_id:158970)。

考虑投掷两个相同的、公平的 $n$ 面骰子，点数为 $1, \dots, n$。设两次结果为独立的[随机变量](@entry_id:195330) $X$ 和 $Y$。我们想求 $Z = \max(X, Y)$ 的[分布](@entry_id:182848) [@problem_id:1919101]。直接枚举所有情况会非常繁琐，但CDF法非常简洁。
事件 $\{Z \le k\}$ 等价于两个骰子的点数都必须不大于 $k$，即 $\{X \le k \text{ and } Y \le k\}$。由于 $X$ 和 $Y$ 独立：
$F_Z(k) = P(Z \le k) = P(X \le k, Y \le k) = P(X \le k)P(Y \le k)$
对于一个在 $\{1, \dots, n\}$ 上[均匀分布](@entry_id:194597)的[随机变量](@entry_id:195330)，其 $P(\cdot \le k) = k/n$。因此，
$F_Z(k) = \left(\frac{k}{n}\right)^2$
现在，我们可以通过差分得到PMF：
$P(Z=k) = P(Z \le k) - P(Z \le k-1) = \left(\frac{k}{n}\right)^2 - \left(\frac{k-1}{n}\right)^2 = \frac{k^2 - (k^2 - 2k + 1)}{n^2} = \frac{2k-1}{n^2}$
这个简洁的公式适用于所有 $k \in \{1, \dots, n\}$，展现了CDF方法的优雅和高效。

#### 变量变换法 ([雅可比方法](@entry_id:270947))

对于更一般的[函数变换](@entry_id:141095)，特别是当函数不是简单的和、积或商，或者当变量不独立时，**变量变换法**（常称为**[雅可比方法](@entry_id:270947)**）是最普适的工具。该方法适用于[连续随机变量](@entry_id:166541)，并允许我们直接从旧变量的联合PDF推导出新变量的联合PDF。

假设我们有一对一的变换，将[随机变量](@entry_id:195330) $(X, Y)$ 映射到 $(U, V)$，其中 $U=g_1(X, Y)$ 和 $V=g_2(X, Y)$。我们的目标通常是求 $U$ 的[边际分布](@entry_id:264862)。这个过程分为两步：
1.  找到 $(U, V)$ 的联合PDF $f_{U,V}(u,v)$。
2.  通过对 $v$ 积分，得到 $U$ 的边际PDF $f_U(u) = \int f_{U,V}(u,v) \,dv$。

关键的第一步利用以下公式：
$f_{U,V}(u,v) = f_{X,Y}(x(u,v), y(u,v)) |J|$
这里，$x(u,v)$ 和 $y(u,v)$ 是[逆变](@entry_id:192290)换，$|J|$ 是**[雅可比行列式](@entry_id:137120)**的[绝对值](@entry_id:147688)，它度量了从 $(x,y)$ 空间到 $(u,v)$ 空间微元面积的缩放比例。雅可比行列式 $J$ 定义为逆变换的偏导数组成的[行列式](@entry_id:142978)：
$J = \det \begin{pmatrix} \frac{\partial x}{\partial u}  \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u}  \frac{\partial y}{\partial v} \end{pmatrix}$

为了求单个变量 $U = g_1(X, Y)$ 的[分布](@entry_id:182848)，我们需要巧妙地引入一个**辅助变量** $V = g_2(X, Y)$，通常选择一个简单的形式，如 $V=X$ 或 $V=Y$，以简化计算。

例如，考虑两个独立的指数分布噪声信号 $X_1, X_2$，其参数均为 $\lambda$。我们想求它们的比值 $U = X_1/X_2$ 的[分布](@entry_id:182848) [@problem_id:1919105]。我们引入辅助变量 $V=X_2$。逆变换为 $X_1 = UV$ 和 $X_2 = V$。雅可比行列式为：
$J = \det \begin{pmatrix} v  u \\ 0  1 \end{pmatrix} = v$
[原始变量](@entry_id:753733)的联合PDF为 $f_{X_1,X_2}(x_1, x_2) = \lambda^2 e^{-\lambda(x_1+x_2)}$，其中 $x_1 > 0, x_2 > 0$。进行变换后，新的联合PDF为：
$f_{U,V}(u,v) = f_{X_1,X_2}(uv, v) |J| = (\lambda^2 e^{-\lambda(uv+v)}) \cdot v = \lambda^2 v e^{-\lambda v(1+u)}$
其支撑域为 $u>0, v>0$。最后，为了得到 $U$ 的边际PDF，我们对 $v$ 进行积分：
$f_U(u) = \int_0^\infty \lambda^2 v e^{-\lambda v(1+u)} \,dv = \frac{1}{(1+u)^2}$, for $u>0$
这个结果表明，两个[独立同分布](@entry_id:169067)的指数[随机变量](@entry_id:195330)之比服从一个参数为1的F(2,2)[分布](@entry_id:182848)。

[雅可比方法](@entry_id:270947)同样适用于**非独立**的[随机变量](@entry_id:195330)。考虑一个联合PDF $f_{X,Y}(x,y)=2$，定义在一个由 $(0,0), (1,0), (1,1)$ 构成的三角形区域上。我们想求 $Z = Y/X$ 的[分布](@entry_id:182848) [@problem_id:1919106]。我们引入辅助变量 $W=X$。逆变换为 $X=W, Y=ZW$。雅可比行列式为 $w$。
新的联合PDF $f_{Z,W}(z,w) = f_{X,Y}(w, zw) |J| = 2w$。
最关键的一步是确定新的支撑域。原始支撑域为 $0 \le y \le x$ 和 $0 \le x \le 1$。代入[逆变](@entry_id:192290)换，我们得到 $0 \le zw \le w$ 和 $0 \le w \le 1$。对于 $w>0$，这简化为 $0 \le z \le 1$ 和 $0 \le w \le 1$。
现在，我们对 $w$ 积分来求 $Z$ 的[边际分布](@entry_id:264862)：
$f_Z(z) = \int_0^1 f_{Z,W}(z,w) \,dw = \int_0^1 2w \,dw = [w^2]_0^1 = 1$, for $0 \le z \le 1$
出人意料的是，结果表明 $Z$ 在 $[0,1]$ 区间上服从[均匀分布](@entry_id:194597)。

### 派生的[条件分布](@entry_id:138367)

前面介绍的方法使我们能够找到[随机变量](@entry_id:195330)函数自身的[分布](@entry_id:182848)。一个更进一步且深刻的问题是：如果我们观测到了函数的值，那么关于[原始变量](@entry_id:753733)之一，我们能获得什么信息？这引出了**派生的条件分布**的概念。

假设我们观测到 $S=g(X,Y)$ 的值为 $s$。我们想知道在此条件下 $X$ 的[分布](@entry_id:182848)。其[条件PDF](@entry_id:164480)由以下公式给出：
$f_{X|S}(x|s) = \frac{f_{X,S}(x,s)}{f_S(s)}$
其中 $f_{X,S}(x,s)$ 是 $X$ 和 $S$ 的联合PDF，而 $f_S(s)$ 是 $S$ 的边际PDF。求解过程通常涉及：
1.  使用变量变换法（例如，从 $(X,Y)$ 变换到 $(X,S)$）求出联合PDF $f_{X,S}(x,s)$。
2.  对 $x$ 积分求出 $S$ 的边际PDF $f_S(s)$。
3.  两者相除得到[条件PDF](@entry_id:164480)。

一个在[随机过程](@entry_id:159502)理论中非常经典的例子可以阐释这个过程。假设 $T_1$ 和 $T_2$ 是两次放射性粒子到达的[独立同分布](@entry_id:169067)的[指数时间](@entry_id:265663)间隔，参数为 $\lambda$。第二次粒子到达的总时间为 $S = T_1 + T_2$。如果我们观测到 $S=s$，那么第一次粒子是在何时到达的呢？也即，求在 $S=s$ 条件下 $T_1$ 的[分布](@entry_id:182848) [@problem_id:1919104]。

1.  从 $(T_1, T_2)$ 到 $(T_1, S)$ 的变换是 $T_1=T_1, S=T_1+T_2$。其[雅可比行列式](@entry_id:137120)为1。联合PDF为：
    $f_{T_1, S}(t, s) = f_{T_1, T_2}(t, s-t) \cdot |1| = \lambda e^{-\lambda t} \lambda e^{-\lambda(s-t)} = \lambda^2 e^{-\lambda s}$
    其支撑域为 $0  t  s$（因为 $T_1>0$ 和 $T_2=s-t>0$）。

2.  $S$ 的边际PDF为：
    $f_S(s) = \int_0^s f_{T_1, S}(t, s) \,dt = \int_0^s \lambda^2 e^{-\lambda s} \,dt = \lambda^2 e^{-\lambda s} \int_0^s dt = \lambda^2 s e^{-\lambda s}$
    这表明 $S$ 服从 $\text{Gamma}(2, 1/\lambda)$ [分布](@entry_id:182848)，这与我们从MGF方法得到的结论一致。

3.  最后，计算[条件PDF](@entry_id:164480)：
    $f_{T_1|S}(t|s) = \frac{f_{T_1,S}(t,s)}{f_S(s)} = \frac{\lambda^2 e^{-\lambda s}}{\lambda^2 s e^{-\lambda s}} = \frac{1}{s}$, for $0  t  s$.

这个优美的结果表明，在已知两次事件总耗时为 $s$ 的条件下，第一次事件发生的时间在 $(0,s)$ 区间内是**[均匀分布](@entry_id:194597)**的。这个看似与指数分布的“无记忆性”相悖的结论，实际上深刻地揭示了泊松过程的内在结构。
