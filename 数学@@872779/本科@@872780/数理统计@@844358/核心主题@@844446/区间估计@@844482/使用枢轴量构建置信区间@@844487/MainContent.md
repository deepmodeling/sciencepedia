## 引言
在[统计推断](@entry_id:172747)领域，我们的目标是利用有限的样本数据来洞察更广泛的总体现状。[点估计](@entry_id:174544)，如样本均值，虽然为我们提供了对未知参数（如[总体均值](@entry_id:175446)）的最佳单点猜测，但它未能传达我们估计的不确定性程度。为了弥补这一不足，统计学引入了[置信区间](@entry_id:142297)的概念——一个我们有特定信心水平（例如95%）相信其包含真实参数值的[数值范围](@entry_id:752817)。然而，如何系统性地为各种不同的参数和[概率分布](@entry_id:146404)构建这些区间呢？

本文旨在解决这一核心问题，详细介绍一种功能强大且逻辑清晰的通用技术：**轴量法**（Pivotal Quantity Method）。这种方法是经典[统计推断](@entry_id:172747)的基石，为从简单到复杂的各种[参数估计](@entry_id:139349)问题提供了统一的解决思路。

通过本文的学习，您将能够掌握：
*   在**第一章：原理与机制**中，我们将深入探讨轴量的定义和构建[置信区间](@entry_id:142297)的“三步法”框架。您将学习如何为正态分布的均值和[方差](@entry_id:200758)，以及均匀、指数等其他重要[分布](@entry_id:182848)的参数找到合适的轴量并构建其置信区间。
*   在**第二章：应用与跨学科联系**中，我们将展示轴量法如何在质量控制、可靠性工程、[生物统计学](@entry_id:266136)和实验设计等多个领域解决实际问题。您将看到理论如何转化为比较过程变异性、进行未来观测值预测以及评估复杂模型严谨性的有力工具。
*   在**第三章：动手实践**部分，您将通过解决一系列精心设计的练习题，亲手应用所学知识，将抽象的理论转化为具体的计算和解释能力，从而真正内化这一核心统计技能。

现在，让我们从轴量法的基本原理出发，开启构建可信统计区间的旅程。

## 原理与机制

在[统计推断](@entry_id:172747)领域，我们的核心任务之一是基于样本数据来估计未知的总体参数。[点估计](@entry_id:174544)给出了参数的最佳单值猜测，但它并未提供与该估计相关的不确定性的度量。[置信区间](@entry_id:142297)通过提供一个我们有特定信心水平（例如95%）认为包含真实参数值的[数值范围](@entry_id:752817)，从而弥补了这一不足。本章将系统地介绍一种构建置信区间的强大而通用的方法——**轴量法**（Pivotal Quantity Method）。

### 轴量法：构建置信区间的通用框架

轴量法的精髓在于找到一个连接样本数据与未知参数的[特殊函数](@entry_id:143234)，该函数的[概率分布](@entry_id:146404)是已知的，并且不依赖于任何未知参数。这个[特殊函数](@entry_id:143234)被称为**轴量**或**[枢轴量](@entry_id:168397)**（pivotal quantity）。

**轴量的定义**

一个**轴量** $Q(\mathbf{X}, \theta)$ 是一个以样本数据 $\mathbf{X} = (X_1, \dots, X_n)$ 和未知参数 $\theta$ 为变量的函数，其关键特性是它的[抽样分布](@entry_id:269683)是完全已知的，并且不依赖于参数 $\theta$ 或任何其他未知参数。

一旦找到了这样一个轴量，我们就可以遵循一个标准的“三步法”来构建置信区间：

1.  **寻找轴量**：识别一个合适的轴量 $Q(\mathbf{X}, \theta)$。这是该方法中最具创造性的一步，通常依赖于对基础统计理论的理解。

2.  **确定临界值**：利用轴量 $Q$ 的已知[分布](@entry_id:182848)，找到两个常数 $c_1$ 和 $c_2$，使得 $P(c_1 \le Q \le c_2) = 1-\alpha$，其中 $1-\alpha$ 是我们期望的[置信水平](@entry_id:182309)。对于对称[分布](@entry_id:182848)，通常选择使两边尾部概率相等的临界值，即 $P(Q  c_1) = P(Q > c_2) = \alpha/2$。

3.  **反解不等式**：通过代数运算，将不等式 $c_1 \le Q(\mathbf{X}, \theta) \le c_2$ 变换为关于参数 $\theta$ 的形式 $L(\mathbf{X}) \le \theta \le U(\mathbf{X})$。这样，由样本数据决定的随机区间 $[L(\mathbf{X}), U(\mathbf{X})]$ 就是参数 $\theta$ 的一个 $100(1-\alpha)\%$ 置信区间。

我们通过一个简单的例子来阐明这个过程。假设一个科学仪器的一次测量值 $X$ 服从区间为 $[\theta - 1/2, \theta + 1/2]$ 的[均匀分布](@entry_id:194597)，其中 $\theta$ 是未知的物理常数 [@problem_id:1909625]。我们的目标是基于单次观测值 $x$ 为 $\theta$ 构建一个90%的置信区间。

首先，我们寻找一个轴量。考虑量 $Q = X - \theta$。由于 $X$ 在 $[\theta - 1/2, \theta + 1/2]$ 上[均匀分布](@entry_id:194597)，那么 $X - \theta$ 的值将均匀地[分布](@entry_id:182848)在 $[-1/2, 1/2]$ 上。即，$Q \sim U(-1/2, 1/2)$。这个[分布](@entry_id:182848)不依赖于 $\theta$，因此 $Q$ 是一个有效的轴量。

其次，对于90%的[置信水平](@entry_id:182309)，$\alpha = 0.1$。我们需要找到常数 $c_1$ 和 $c_2$ 使得 $P(c_1 \le Q \le c_2) = 0.9$。对于[均匀分布](@entry_id:194597) $U(-1/2, 1/2)$，其概率密度函数为常数1，因此概率就是区间的长度。我们可以选择一个对称的区间，即 $c_2 = -c_1$。因此，$c_2 - c_1 = 2c_2 = 0.9$，解得 $c_2 = 0.45$ 且 $c_1 = -0.45$。我们可以写出概率叙述：$P(-0.45 \le X - \theta \le 0.45) = 0.9$。

最后，我们反解不等式来隔离 $\theta$：
$$-0.45 \le X - \theta \le 0.45$$
$$-X - 0.45 \le -\theta \le -X + 0.45$$
$$X - 0.45 \le \theta \le X + 0.45$$
因此，基于观测值 $x$，$\theta$ 的一个90%[置信区间](@entry_id:142297)为 $[x - 0.45, x + 0.45]$，即 $[x - \frac{9}{20}, x + \frac{9}{20}]$。这个例子完美地展示了轴量法的三个步骤。

### 正态分布参数的轴量

轴量法在处理正态分布总体的[参数推断](@entry_id:753157)时尤为有效。[正态分布](@entry_id:154414)在统计学中的中心地位使得这部分内容成为经典[统计推断](@entry_id:172747)的基石。

#### 均值 $\mu$ 的推断（[方差](@entry_id:200758) $\sigma^2$ 已知）

这是最简单的情形。根据[中心极限定理](@entry_id:143108)，或当总体本身服从正态分布时，样本均值 $\bar{X}$ 的[抽样分布](@entry_id:269683)为 $N(\mu, \sigma^2/n)$。将其标准化，我们得到轴量：
$$Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0, 1)$$
$Z$ 服从[标准正态分布](@entry_id:184509)，其[分布](@entry_id:182848)不依赖于任何未知参数。对于一个 $100(1-\alpha)\%$ 的置信区间，我们找到临界值 $\pm z_{\alpha/2}$，使得 $P(-z_{\alpha/2} \le Z \le z_{\alpha/2}) = 1-\alpha$。反解不等式可得 $\mu$ 的置信区间：
$$\bar{X} - z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \le \mu \le \bar{X} + z_{\alpha/2} \frac{\sigma}{\sqrt{n}}$$

这一原理可以自然地推广到两个独立的、[方差](@entry_id:200758)已知的正态[总体均值](@entry_id:175446)差 $\mu_1 - \mu_2$ 的情况 [@problem_id:1909619]。例如，比较两条生产线电阻均值的差异，其中历史数据给出了各自的[方差](@entry_id:200758)。此时，样本均值差 $\bar{X}_1 - \bar{X}_2$ 服从 $N(\mu_1 - \mu_2, \frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2})$。相应的轴量为：
$$Z = \frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1} + \frac{\sigma_2^2}{n_2}}} \sim N(0, 1)$$
通过与单样本情况完全相同的逻辑，可以导出 $\mu_1 - \mu_2$ 的[置信区间](@entry_id:142297)。

#### 均值 $\mu$ 的推断（[方差](@entry_id:200758) $\sigma^2$ 未知）

在绝大多数实际应用中，总体[方差](@entry_id:200758) $\sigma^2$ 都是未知的。此时，我们不能再使用上述的 $Z$ 轴量，因为它包含了未知的 $\sigma$。一个自然的想法是用其[无偏估计量](@entry_id:756290)——样本[方差](@entry_id:200758) $S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$ 来替代 $\sigma^2$。然而，这样做会改变统计量的[分布](@entry_id:182848)。由此产生的统计量服从 **学生t分布**（[Student's t-distribution](@entry_id:142096)）：
$$T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t_{n-1}$$
其中 $\nu = n-1$ 是自由度。[t分布](@entry_id:267063)的形状与标准正态分布相似但尾部更厚，这反映了由于估计 $\sigma$ 而引入的额外不确定性。当样本量 $n$ 增大时，t分布收敛于标准正态分布。

因为 $T$ 的[分布](@entry_id:182848)不依赖于 $\mu$ 或 $\sigma^2$，所以它是一个有效的轴量。类似于Z[分布](@entry_id:182848)的情况，我们找到[t分布](@entry_id:267063)的临界值 $\pm t_{\alpha/2, n-1}$，并反解不等式，得到 $\mu$ 的[置信区间](@entry_id:142297)：
$$\bar{X} - t_{\alpha/2, n-1} \frac{S}{\sqrt{n}} \le \mu \le \bar{X} + t_{\alpha/2, n-1} \frac{S}{\sqrt{n}}$$

在某些情况下，我们可能只关心参数的一个界限。例如，在校准一种新型磁力计时，我们可能希望以95%的置信度确定真实[磁场强度](@entry_id:197932) $\mu$ 的一个下界 [@problem_id:1909621]。这对应于一个**单侧置信区间**。我们寻找一个临界值 $t_{\alpha, n-1}$ 使得 $P(T \le t_{\alpha, n-1}) = 1-\alpha$。反解该不等式 $\frac{\bar{X} - \mu}{S/\sqrt{n}} \le t_{\alpha, n-1}$，得到 $\mu$ 的 $100(1-\alpha)\%$ 的置信下界为 $L = \bar{X} - t_{\alpha, n-1} \frac{S}{\sqrt{n}}$。

#### [方差](@entry_id:200758) $\sigma^2$ 的推断

当兴趣的参数是总体[方差](@entry_id:200758) $\sigma^2$ 时，我们需要一个新的轴量。对于来自正态总体的样本，一个关键的统计学结果是，经过[标准化](@entry_id:637219)的样本[方差](@entry_id:200758)服从 **卡方分布**（Chi-squared distribution）：
$$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1}$$
这个统计量的[分布](@entry_id:182848)只依赖于自由度 $\nu = n-1$，因此它是一个有效的轴量。

由于[卡方分布](@entry_id:165213)是正偏的（不对称），其[置信区间](@entry_id:142297)的构建略有不同。我们寻找两个临界值 $\chi^2_{1-\alpha/2, n-1}$ 和 $\chi^2_{\alpha/2, n-1}$，它们分别截取了[分布](@entry_id:182848)的左尾和右尾各 $\alpha/2$ 的概率。因此，
$$P\left(\chi^2_{1-\alpha/2, n-1} \le \frac{(n-1)S^2}{\sigma^2} \le \chi^2_{\alpha/2, n-1}\right) = 1-\alpha$$
反解这个双重不等式以隔离 $\sigma^2$（注意，取倒数会反转不等号），我们得到 $\sigma^2$ 的[置信区间](@entry_id:142297)：
$$\left[ \frac{(n-1)S^2}{\chi^2_{\alpha/2, n-1}}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2, n-1}} \right]$$

同样，我们也可以构建单侧置信界。例如，在评估滚珠轴承制造过程的一致性时，我们主要关心[方差](@entry_id:200758) $\sigma^2$ 不要太大，因此需要一个[置信上界](@entry_id:178122) [@problem_id:1909578]。要构建一个95%的[置信上界](@entry_id:178122) $U$，我们找到下侧5%分位数 $\chi^2_{0.95, n-1}$，使得 $P(\frac{(n-1)S^2}{\sigma^2} \ge \chi^2_{0.95, n-1}) = 0.95$。反解后得到上界为 $U = \frac{(n-1)S^2}{\chi^2_{0.95, n-1}}$。

#### 两个[方差](@entry_id:200758)之比 $\sigma_1^2 / \sigma_2^2$ 的推断

在比较两个独立的制造过程的变异性时，我们常常对它们的[方差](@entry_id:200758)之比 $\sigma_1^2 / \sigma_2^2$ 感兴趣 [@problem_id:1909605]。为此，我们引入了 **[F分布](@entry_id:261265)**。[F分布](@entry_id:261265)定义为两个独立的、各自除以其自由度的卡方[随机变量](@entry_id:195330)之比。

基于我们已知的 $\frac{(n_i-1)S_i^2}{\sigma_i^2} \sim \chi^2_{n_i-1}$，可以构建如下轴量：
$$F = \frac{\frac{(n_1-1)S_1^2/\sigma_1^2}{n_1-1}}{\frac{(n_2-1)S_2^2/\sigma_2^2}{n_2-1}} = \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} = \frac{S_1^2}{S_2^2} \cdot \frac{\sigma_2^2}{\sigma_1^2}$$
该统计量服从[分子自由度](@entry_id:175192)为 $\nu_1 = n_1-1$、分母自由度为 $\nu_2 = n_2-1$ 的[F分布](@entry_id:261265)，记为 $F \sim F_{n_1-1, n_2-1}$。由于其[分布](@entry_id:182848)不依赖于任何未知参数，它是一个有效的轴量。

通过找到[F分布](@entry_id:261265)的临界值 $f_{1-\alpha/2, n_1-1, n_2-1}$ 和 $f_{\alpha/2, n_1-1, n_2-1}$，并反解不等式，我们可以得到[方差比](@entry_id:162608) $\sigma_1^2 / \sigma_2^2$ 的置信区间。利用[F分布](@entry_id:261265)的性质 $f_{1-p, \nu_1, \nu_2} = 1 / f_{p, \nu_2, \nu_1}$，区间可以表示为：
$$\left[ \frac{S_1^2}{S_2^2} \frac{1}{f_{\alpha/2, n_1-1, n_2-1}}, \frac{S_1^2}{S_2^2} f_{\alpha/2, n_2-1, n_1-1} \right]$$

### 非[正态分布](@entry_id:154414)情境下的轴量

轴量法的威力远不止于正态分布。只要能找到一个[分布](@entry_id:182848)与参数无关的函数，我们就可以为各种[分布](@entry_id:182848)的参数构建[置信区间](@entry_id:142297)。

#### [均匀分布](@entry_id:194597)的参数

我们已经见过为 $U(\theta - 1/2, \theta + 1/2)$ 的中心 $\theta$ 构建置信区间的例子。一个更有趣的情形是估计[均匀分布](@entry_id:194597) $U(0, \theta)$ 的上界参数 $\theta$ [@problem_id:1909599]。在这种情况下，一个有用的统计量是样本最大值 $X_{(n)} = \max(X_1, \dots, X_n)$。

为了构建轴量，我们首先需要 $X_{(n)}$ 的[分布](@entry_id:182848)。对于任意 $v \in [0, \theta]$，其累积分布函数（CDF）为：
$$F_{X_{(n)}}(v) = P(X_{(n)} \le v) = P(X_1 \le v, \dots, X_n \le v) = [P(X_i \le v)]^n = \left(\frac{v}{\theta}\right)^n$$
现在，我们考虑一个潜在的轴量 $Q = X_{(n)}/\theta$。它的取值范围是 $[0, 1]$。对于任意 $q \in [0, 1]$，其CDF为：
$$F_Q(q) = P(Q \le q) = P(X_{(n)}/\theta \le q) = P(X_{(n)} \le q\theta) = \left(\frac{q\theta}{\theta}\right)^n = q^n$$
由于 $Q$ 的[分布](@entry_id:182848)（其CDF为 $F_Q(q)=q^n$）不依赖于 $\theta$，所以 $Q$ 是一个有效的轴量。通过找到满足 $c_1^n = \alpha/2$ 和 $c_2^n = 1-\alpha/2$ 的临界值 $c_1$ 和 $c_2$，并反解不等式，就可以得到 $\theta$ 的置信区间。

#### 指数分布的参数

[指数分布](@entry_id:273894)常用于建模寿命或等待时间，例如评估新型LED的可靠性 [@problem_id:1909645]。其概率密度函数为 $f(x; \theta) = \frac{1}{\theta} \exp(-x/\theta)$，其中 $\theta$ 是平均寿命。

对于来自[指数[分](@entry_id:273894)布](@entry_id:182848)的随机样本，一个重要的理论结果是样本总和 $S = \sum_{i=1}^n X_i$ 与[卡方分布](@entry_id:165213)有关。具体来说，轴量为：
$$\frac{2S}{\theta} \sim \chi^2_{2n}$$
这个轴量的[分布](@entry_id:182848)只依赖于自由度 $2n$。因此，我们可以像处理[正态方差](@entry_id:167335)一样，使用[卡方分布](@entry_id:165213)的临界值来为[指数分布](@entry_id:273894)的均值 $\theta$ 构建置信区间：
$$\left[ \frac{2S}{\chi^2_{\alpha/2, 2n}}, \frac{2S}{\chi^2_{1-\alpha/2, 2n}} \right]$$

#### 参数函数的[置信区间](@entry_id:142297)

通常，我们感兴趣的可能不是[分布](@entry_id:182848)的直接参数，而是它的某个函数。例如，在[可靠性分析](@entry_id:192790)中，我们可能更关心某个产品寿命超过特定时间 $c$ 的概率 $p = P(X > c)$ [@problem_id:1909577]。对于[指数分布](@entry_id:273894)，这个概率是 $p = \exp(-c/\theta)$。

如果函数 $g(\theta)$ 是一个关于 $\theta$ 的[单调函数](@entry_id:145115)，那么我们可以利用所谓的**[不变性原理](@entry_id:199405)**。如果我们已经为 $\theta$ 构建了一个置信区间 $[L, U]$，那么 $g(\theta)$ 的置信区间就是 $[g(L), g(U)]$（如果 $g$ 是单调递增的）或 $[g(U), g(L)]$（如果 $g$ 是单调递减的）。

在指数寿命的例子中，$p(\theta) = \exp(-c/\theta)$ 是关于 $\theta$ 的单调递增函数。因此，一旦我们获得了 $\theta$ 的置信区间 $[\theta_L, \theta_U]$，我们就可以立即得到生存概率 $p$ 的置信区间：
$$[\exp(-c/\theta_U), \exp(-c/\theta_L)]$$
这是一个强大的技术，极大地扩展了[置信区间](@entry_id:142297)的应用范围。

### 近似[置信区间](@entry_id:142297)与[无分布方法](@entry_id:268310)

在许多情况下，找到一个精确的轴量是困难或不可能的。此时，我们可以求助于近似方法或不依赖于特定[分布](@entry_id:182848)假设的方法。

#### 大样本近似轴量

根据**[中心极限定理](@entry_id:143108)**（Central Limit Theorem），许多[统计估计量](@entry_id:170698)在样本量 $n$ 很大时，其[抽样分布](@entry_id:269683)近似于[正态分布](@entry_id:154414)。这为我们构建**近似置信区间**提供了理论基础。一个通用的近似轴量形式为：
$$\frac{\text{估计量} - \text{参数}}{\text{标准误}} \approx N(0, 1)$$
其中标准误是估计量[抽样分布](@entry_id:269683)的标准差，通常也需要用数据来估计。

一个经典的应用是比较两种处理（如新药与安慰剂）的成功概率 $p_1$ 和 $p_2$ 的差异 [@problem_id:1909608]。样本比例 $\hat{p}_1$ 和 $\hat{p}_2$ 是 $p_1$ 和 $p_2$ 的估计量。对于大样本，$\hat{p}_1 - \hat{p}_2$ 近似服从正态分布。其标准误的估计值为 $\text{SE} = \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}$。因此，近似轴量为：
$$Z = \frac{(\hat{p}_1 - \hat{p}_2) - (p_1 - p_2)}{\sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}}} \approx N(0, 1)$$
基于这个近似轴量，可以构建 $p_1 - p_2$ 的近似[置信区间](@entry_id:142297)，即著名的**[Wald区间](@entry_id:173132)**。

#### [无分布方法](@entry_id:268310)：[中位数的置信区间](@entry_id:636606)

有时我们甚至不愿意对总体的[分布](@entry_id:182848)形式做任何假定。**[无分布方法](@entry_id:268310)**（或称[非参数方法](@entry_id:138925)）为此提供了解决方案。例如，我们可以为任意连续分布的[中位数](@entry_id:264877) $\eta$ 构建一个[置信区间](@entry_id:142297) [@problem_id:1909647]。

其逻辑不依赖于[分布](@entry_id:182848)的具体形式，而只依赖于[中位数](@entry_id:264877)的定义：对于任意观测值 $X_i$，$P(X_i  \eta) = 0.5$。令 $B$ 为样本中小于[中位数](@entry_id:264877) $\eta$ 的观测值数量。那么 $B$ 服从二项分布 $B \sim \text{Binomial}(n, 0.5)$。

现在考虑由样本的**次序统计量** $X_{(1)} \le X_{(2)} \le \dots \le X_{(n)}$ 构成的区间 $(X_{(j)}, X_{(k)})$。真实中位数 $\eta$ 落在该区间内的事件 $(X_{(j)}  \eta  X_{(k)})$，等价于样本中至少有 $j$ 个观测值小于 $\eta$，且至少有 $n-k+1$ 个观测值大于 $\eta$。这又等价于小于 $\eta$ 的观测值数量 $B$ 满足 $j \le B \le k-1$。

因此，这个区间的[置信水平](@entry_id:182309)为：
$$P(X_{(j)}  \eta  X_{(k)}) = P(j \le B \le k-1) = \sum_{i=j}^{k-1} \binom{n}{i} (0.5)^n$$
这个概率完全可以通过二项分布计算出来，而无需知道原始数据的[分布](@entry_id:182848)形式，体现了[无分布方法](@entry_id:268310)的稳健性。

### 高级轴量方法：非中心[t分布](@entry_id:267063)

在某些更复杂的问题中，我们可能找到一个统计量，其[分布](@entry_id:182848)形式虽然已知，但[分布](@entry_id:182848)的参数本身就包含了我们感兴趣的未知参数。这可以看作是轴量概念的一种推广。

一个典型的例子是为[正态分布](@entry_id:154414) $N(\mu, \sigma^2)$ 的“质量因子”或[信噪比](@entry_id:185071) $\theta = \mu/\sigma$ 构建[置信区间](@entry_id:142297) [@problem_id:1909617]。可以证明，统计量
$$T = \frac{\sqrt{n}\bar{X}}{S}$$
服从一个**非中心t分布**（non-central t-distribution），记为 $t_{\nu}(\delta)$。其自由度为 $\nu=n-1$，而非中心参数为 $\delta = \frac{\sqrt{n}\mu}{\sigma} = \sqrt{n}\theta$。

非中心[t分布](@entry_id:267063)可以看作是 $(Z+\delta)/\sqrt{V/\nu}$ 的[分布](@entry_id:182848)，其中 $Z \sim N(0,1)$，$V \sim \chi^2_\nu$ 且两者独立。这里的关键是，虽然 $T$ 的[分布](@entry_id:182848)依赖于未知参数，但仅仅是通过非中心参数 $\delta$ 依赖于它们，而 $\delta$ 又直接与我们感兴趣的量 $\theta$ 相关。

构建[置信区间](@entry_id:142297)的方法是“反向”的：我们计算出观测到的统计量值 $t_{obs}$，然后寻找所有使得 $t_{obs}$ 成为该[分布](@entry_id:182848)下一个“合理”观测值的 $\delta$ 的集合。具体来说，我们寻找 $\delta_L$ 和 $\delta_U$，使得 $t_{obs}$ 分别是 $t_{n-1}(\delta_L)$ [分布](@entry_id:182848)的 $1-\alpha/2$ 分位数和 $t_{n-1}(\delta_U)$ [分布](@entry_id:182848)的 $\alpha/2$ 分位数。这个过程通常需要数值计算软件来求解。一旦找到 $\delta$ 的置信区间 $[\delta_L, \delta_U]$，就可以通过关系 $\theta = \delta/\sqrt{n}$ 得到 $\theta$ 的[置信区间](@entry_id:142297) $[\delta_L/\sqrt{n}, \delta_U/\sqrt{n}]$。这个高级例子揭示了轴量思想的深度和广度，并展示了其在解决非标准统计推断问题中的应用。
