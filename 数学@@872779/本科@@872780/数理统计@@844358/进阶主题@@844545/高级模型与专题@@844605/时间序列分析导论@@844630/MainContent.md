## 引言
在我们的世界里，从金融市场的股票价格波动，到地球的气候变化，再到工程系统中的传感器读数，许多数据都以时间为序呈现。这些按时间顺序[排列](@entry_id:136432)的数据点构成了**时间序列**，它们蕴含着关于系统动态、周期性行为和未来趋势的宝贵信息。然而，这些信息往往隐藏在看似随机的波动之下。因此，如何从这些复杂的[时间序列数据](@entry_id:262935)中提取有意义的模式、理解其底层机制并做出可靠的预测，便成为统计学和众多应用科学领域面临的核心挑战。本文正是为了解决这一知识鸿沟，为读者系统地介绍[时间序列分析](@entry_id:178930)的基础理论与实践方法。

为了构建一个清晰的学习路径，本文将通过三个核心章节引导读者逐步深入：

- 在 **第一章：原理与机制** 中，我们将奠定理论基础，深入探讨[时间序列分析](@entry_id:178930)的基石——[平稳性](@entry_id:143776)，并详细介绍构建现代时间序列模型的两个基本模块：自回归（AR）与[移动平均](@entry_id:203766)（MA）过程。读者将理解为何这些概念对于模型构建至关重要。

- **第二章：应用与跨学科联系** 将理论与现实世界连接起来。我们将通过来自经济学、生态学、工程学等多个领域的实例，展示如何运用[时间序列分析](@entry_id:178930)工具解决实际问题，从处理[非平稳数据](@entry_id:261489)到识别模型，再到进行预测和解释。

- **第三章：动手实践** 则提供了将知识转化为技能的机会。通过一系列精心设计的练习，读者可以亲手应用所学概念，检验模型的平稳性、推导自相关函数以及计算[预测误差](@entry_id:753692)，从而巩固和深化对核心原理的理解。

现在，让我们从[时间序列分析](@entry_id:178930)最根本的原理与机制开始，踏上探索数据背后动态世界的旅程。

## 原理与机制

在“导论”章节中，我们了解了时间序列是按时间顺序[排列](@entry_id:136432)的一系列数据点。为了从这些数据中提取有意义的模式、进行预测或理解其底层动态，我们需要建立一套形式化的理论框架。本章将深入探讨[时间序列分析](@entry_id:178930)的核心原理与机制，重点介绍**[平稳性](@entry_id:143776) (stationarity)** 的概念，并阐述构建现代时间序列模型的基本构件：**自回归 (Autoregressive, AR)** 与 **移动平均 (Moving Average, MA)** 过程。

### [时间序列分析](@entry_id:178930)的基石：平稳性

在分析一个时间序列之前，我们必须回答一个根本性问题：这个过程的统计特性是否随时间变化？如果一个过程的统计特性（如均值、[方差](@entry_id:200758)）不随时间推移而改变，我们称之为**平稳的 (stationary)**。这个概念至关重要，因为它意味着我们在一个时间段观察到的模式（在统计意义上）也可能出现在另一个时间段。这种稳定性使得建立一个在整个时间轴上都有效的、具有预测能力的模型成为可能。

#### 稳定性的定义：[弱平稳性](@entry_id:171204)

在实践中，我们通常关注一种更容易验证的[平稳性](@entry_id:143776)，即**[弱平稳性](@entry_id:171204) (weak stationarity)** 或**协[方差](@entry_id:200758)平稳性 (covariance stationarity)**。一个时间序列过程 $\{X_t\}$ 被定义为弱平稳的，如果它满足以下三个条件：

1.  **均值恒定**: [均值函数](@entry_id:264860) $E[X_t] = \mu$ 对于所有时间点 $t$ 都是一个常数。
2.  **[方差](@entry_id:200758)恒定**: [方差](@entry_id:200758) $\text{Var}(X_t) = \gamma(0) = \sigma^2$ 对于所有时间点 $t$ 都是一个常数。
3.  **[自协方差](@entry_id:270483)仅依赖于时间差**: 对于任意时间点 $t$ 和时间差 $h$，[自协方差](@entry_id:270483) $\text{Cov}(X_t, X_{t+h}) = \gamma(h)$ 仅是时间差（或称**滞后 (lag)**）$h$ 的函数，而与具体的时间 $t$ 无关。

违反这三个条件中的任何一个，过程就是**非平稳的 (non-stationary)**。

为了具体理解这一点，让我们设想一个场景：一个时间序列是通过拼接两个独立的、但具有不同均值的平稳[白噪声](@entry_id:145248)序列而形成的 [@problem_id:1925251]。具体来说，假设过程 $\{Y_t\}$ 的均值为 $\mu_Y$，过程 $\{Z_t\}$ 的均值为 $\mu_Z$，且 $\mu_Y \neq \mu_Z$。我们构造一个新的序列 $\{X_t\}$，其前半段来自 $\{Y_t\}$，后半段来自 $\{Z_t\}$。这个新序列 $\{X_t\}$ 的[期望值](@entry_id:153208)在前半部分是 $\mu_Y$，在后半部分是 $\mu_Z$。由于[均值函数](@entry_id:264860) $E[X_t]$ 随时间 $t$ 变化，它不满足[弱平稳性](@entry_id:171204)的第一个条件，因此整个过程 $\{X_t\}$ 是非平稳的。这个例子清晰地表明，即使一个序列由平稳的部分组成，结构性的突变（如均值的跃迁）也会破坏整体的平稳性。

#### 时间的重要性：[非平稳性](@entry_id:180513)的常见形式

[非平稳性](@entry_id:180513)在现实世界的数据中非常普遍，其形式也多种多样。

**确定性趋势 (Deterministic Trends)**: 过程的统计特性以一种可预测的方式随时间变化。例如，如果一个过程的均值包含一个线性时间项 $\mu(t) = a + bt$，或者其[方差](@entry_id:200758)随时间指数衰减。考虑一个[随机过程](@entry_id:159502) $Z_t = \Phi \cos(\omega t)$，其中 $\Phi$ 是一个[随机变量](@entry_id:195330) [@problem_id:1312108]。它的均值 $\mu_Z(t) = E[\Phi] \cos(\omega t)$ 依赖于时间 $t$，因此是非平稳的。同样，过程 $W_t = A \exp(-t) \sin(\omega t + \Phi)$ 的[方差](@entry_id:200758)为 $\text{Var}(W_t) = \frac{A^2}{2} \exp(-2t)$，它随着时间 $t$ 衰减，因此也是非平稳的。

**随机趋势 (Stochastic Trends)**: 这是另一种更微妙的[非平稳性](@entry_id:180513)，其最典型的例子是**[随机游走](@entry_id:142620) (random walk)**。假设一个资产价格 $P_t$ 遵循 $P_t = P_{t-1} + \Delta_t$，其中 $\Delta_t$ 是均值为零、[方差](@entry_id:200758)为 $\sigma^2$ 的[独立同分布](@entry_id:169067)随机扰动项（[白噪声](@entry_id:145248)）。我们可以将 $P_t$ 表示为从初始值 $P_0$ 开始的所有扰动的累加和：$P_t = P_0 + \sum_{i=1}^t \Delta_i$。其[方差](@entry_id:200758)为 $\text{Var}(P_t) = \sum_{i=1}^t \text{Var}(\Delta_i) = t\sigma^2$。由于[方差](@entry_id:200758)依赖于时间 $t$，[随机游走过程](@entry_id:171699)是非平稳的。此外，我们还可以考察任意长度为 $k$ 的时间区间的价格变化 $P_{t+k} - P_t$。这个差值的[方差](@entry_id:200758)为 $\text{Var}(P_{t+k} - P_t) = \text{Var}(\sum_{j=1}^k \Delta_{t+j}) = k\sigma^2$ [@problem_id:1312133]。这个结果表明，过程的波动性随着观测时间尺度的增加而[线性增长](@entry_id:157553)，这是随机趋势的一个标志性特征。

#### 创造[平稳性](@entry_id:143776)：随机性的作用

有趣的是，某些看似依赖于时间的确定性函数，在引入随机性后可以变为[平稳过程](@entry_id:196130)。考虑一个纯粹的[正弦波](@entry_id:274998) $Y_t = A_1 \sin(\omega t)$，其中 $A_1$ 是一个零均值、[方差](@entry_id:200758)为 $\sigma^2$ 的[随机变量](@entry_id:195330)。它的[方差](@entry_id:200758)为 $\text{Var}(Y_t) = \sin^2(\omega t) E[A_1^2] = \sigma^2 \sin^2(\omega t)$，这明显依赖于时间 $t$，因此该过程非平稳。

然而，如果我们引入一个随机的**相位 (phase)**，考虑过程 $X_t = A \sin(\omega t + \Phi)$，其中 $\Phi$ 是在 $[0, 2\pi]$ 上[均匀分布](@entry_id:194597)的[随机变量](@entry_id:195330) [@problem_id:1312108]。通过对所有可能的相位进行平均，我们可以计算出该过程的均值为 $E[X_t] = 0$。其[自协方差函数](@entry_id:262114)为 $\gamma_X(t_1, t_2) = \frac{A^2}{2} \cos(\omega(t_1 - t_2))$，它只依赖于时间差 $t_1-t_2$。因此，$X_t$ 是一个弱[平稳过程](@entry_id:196130)。这种通过引入随机相位来“抹平”时间依赖性的方法，是信号处理和时间序列建模中的一个经典思想。另一种实现类似效果的方法是构造 $V_t = A_1 \cos(\omega t) + A_2 \sin(\omega t)$，其中 $A_1$ 和 $A_2$ 是不相关的零均值[随机变量](@entry_id:195330)，该过程同样是弱平稳的 [@problem_id:1312108]。

### 基本模型构建模块：AR与MA过程

一旦我们理解了[平稳性](@entry_id:143776)，就可以开始介绍用于描述平稳时间序列的两个基本模型家族：[移动平均](@entry_id:203766)（MA）过程和自回归（AR）过程。它们是更复杂的ARMA和[ARIMA模型](@entry_id:146503)的基础。

#### 记忆冲击：[移动平均](@entry_id:203766) (MA) 过程

一个**[移动平均过程](@entry_id:178693) (Moving Average process)** 将当前观测值 $X_t$ 表示为当前和过去一系列**[白噪声](@entry_id:145248) (white noise)** 项的加权和。[白噪声过程](@entry_id:146877) $\{W_t\}$ 是一系列零均值、[方差](@entry_id:200758)恒定且相互不相关的[随机变量](@entry_id:195330)，它代表了无法预测的随机冲击。

一个 $q$ 阶的[移动平均过程](@entry_id:178693)，记作 **MA(q)**，其定义为：
$$X_t = \mu + W_t + \theta_1 W_{t-1} + \dots + \theta_q W_{t-q}$$
其中 $\mu$ 是过程的均值，$\theta_1, \dots, \theta_q$ 是模型参数。MA过程的一个美妙特性是，无论参数 $\theta_j$ 取何值，它**始终是弱平稳的**。

让我们以一个MA(2)模型为例，$X_t = Z_t + \theta_1 Z_{t-1} + \theta_2 Z_{t-2}$（此处假设均值为0，[白噪声](@entry_id:145248)为 $Z_t$） [@problem_id:1312119]。
-   **均值**: $E[X_t] = E[Z_t] + \theta_1 E[Z_{t-1}] + \theta_2 E[Z_{t-2}] = 0 + 0 + 0 = 0$，是一个常数。
-   **[方差](@entry_id:200758)**: 由于[白噪声](@entry_id:145248)项在不同时间点上是不相关的，[方差](@entry_id:200758)是各项[方差](@entry_id:200758)之和：$\text{Var}(X_t) = \text{Var}(Z_t) + \theta_1^2 \text{Var}(Z_{t-1}) + \theta_2^2 \text{Var}(Z_{t-2}) = \sigma_Z^2(1 + \theta_1^2 + \theta_2^2)$，也是一个常数。
-   **[自协方差](@entry_id:270483)**: 例如，对于滞后1，$\gamma_X(1) = \text{Cov}(X_t, X_{t-1}) = \text{Cov}(Z_t + \theta_1 Z_{t-1} + \theta_2 Z_{t-2}, Z_{t-1} + \theta_1 Z_{t-2} + \theta_2 Z_{t-3})$。由于只有相同时间的[白噪声](@entry_id:145248)项的协[方差](@entry_id:200758)非零，我们发现 $\gamma_X(1) = \theta_1 \sigma_Z^2 + \theta_1 \theta_2 \sigma_Z^2 = \sigma_Z^2 (\theta_1(1+\theta_2))$。这是一个不依赖于 $t$ 的常数。

一个关键特征是，[MA(q)过程](@entry_id:144037)的[自协方差函数](@entry_id:262114) $\gamma_X(h)$ 在滞后 $h>q$ 后会**截断 (cut off)** 为零，因为 $X_t$ 和 $X_{t-h}$ 之间将不再有共同的白噪声项。这为从数据中识别MA过程提供了重要线索。

#### [自我参照](@entry_id:170448)：自回归 (AR) 过程

与[MA模型](@entry_id:191881)不同，**[自回归过程](@entry_id:264527) (Autoregressive process)** 将当前值 $X_t$ 表示为其**自身过去值**的[线性组合](@entry_id:154743)，外加一个白噪声冲击。

一个 $p$ 阶的[自回归过程](@entry_id:264527)，记作 **AR(p)**，其定义为：
$$X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \dots + \phi_p X_{t-p} + W_t$$
其中 $c$ 是常数项，$\phi_1, \dots, \phi_p$ 是自[回归系数](@entry_id:634860)。

与MA过程不同，AR过程**并非总是平稳的**。平稳性取决于参数 $\phi_j$ 的取值。对于最简单的[一阶自回归过程](@entry_id:746502) **AR(1)**: $X_t = \phi X_{t-1} + W_t$，其平稳的充要条件是 $|\phi|  1$。当这个条件满足时，我们可以计算出过程的[方差](@entry_id:200758)。假设过程是平稳的，那么 $\text{Var}(X_t) = \text{Var}(X_{t-1})$。对方程两边取[方差](@entry_id:200758)，我们得到：
$$\text{Var}(X_t) = \text{Var}(\phi X_{t-1} + W_t) = \phi^2 \text{Var}(X_{t-1}) + \text{Var}(W_t)$$
因为 $X_{t-1}$ 只依赖于过去的[白噪声](@entry_id:145248)，与当前的 $W_t$ 不相关。记 $\text{Var}(X_t) = \gamma(0)$ 和 $\text{Var}(W_t) = \sigma_W^2$，我们有 $\gamma(0) = \phi^2 \gamma(0) + \sigma_W^2$。解这个方程，得到平稳[AR(1)过程](@entry_id:746502)的[方差](@entry_id:200758) [@problem_id:1312092]：
$$\text{Var}(X_t) = \gamma(0) = \frac{\sigma_W^2}{1 - \phi^2}$$
这个公式清晰地显示，当 $|\phi|$ 趋近于1时，过程的[方差](@entry_id:200758)会爆炸式增长，这正是平稳性被破坏的[临界点](@entry_id:144653)。当 $|\phi| \ge 1$ 时，过去的冲击对现在的影响不会衰减，过程会表现出类似[随机游走](@entry_id:142620)的行为，从而变得非平稳。

这些基本模型可以组合使用。例如，如果我们有两个独立的[平稳过程](@entry_id:196130)，如代表正面情绪的 $P_t$ 和代表负面情绪的 $N_t$，它们分别遵循独立的[AR(1)模型](@entry_id:265801)，那么它们的差值“净情绪指数” $S_t = P_t - N_t$ 的[方差](@entry_id:200758)就是它们各自[方差](@entry_id:200758)的和：$\text{Var}(S_t) = \text{Var}(P_t) + \text{Var}(N_t)$ [@problem_id:1312092]。

### 更深层次的结构：[因果性与可逆性](@entry_id:637032)

为了更深刻地理解AR和[MA模型](@entry_id:191881)，并确保它们在实践中的有效性，我们需要引入两个对偶的概念：**因果性 (causality)** 和 **可逆性 (invertibility)**。

#### 只回顾过去：[AR模型](@entry_id:189434)中的因果性

一个[AR模型](@entry_id:189434)是**因果的**，如果其当前值 $X_t$ 能够表示为当前和**过去**的[白噪声](@entry_id:145248)项 $W_t, W_{t-1}, W_{t-2}, \dots$ 的[线性组合](@entry_id:154743)。这个属性至关重要，因为它符合我们对物理世界的直观理解：现在的结果是由过去和现在的原因引起的，而不是由未来的事件决定的。一个非因果的模型在预测上是无用的。

让我们再次回到[AR(1)模型](@entry_id:265801)：$X_t = \phi X_{t-1} + W_t$。通过反复迭代替换 $X_{t-1}$，我们可以得到 [@problem_id:1925250]：
$$X_t = \phi(\phi X_{t-2} + W_{t-1}) + W_t = \phi^2 X_{t-2} + \phi W_{t-1} + W_t = \dots$$
如果这个过程无限地持续下去，我们最终会得到一个无穷[移动平均](@entry_id:203766) **MA($\infty$)** 的表示：
$$X_t = \sum_{j=0}^{\infty} \phi^j W_{t-j}$$
这个无穷级数要收敛（在均方意义上），必须满足 $\sum_{j=0}^{\infty} |\phi^j|  \infty$，这等价于[平稳性条件](@entry_id:191085) $|\phi|  1$。

对于更一般的[AR(p)模型](@entry_id:635980)，这个条件变得更加复杂。我们可以使用**[滞后算子](@entry_id:266398) (backshift operator)** $B$（定义为 $BX_t = X_{t-1}$）来简化表示。[AR(p)模型](@entry_id:635980)可以写为 $\Phi(B)X_t = W_t$，其中 $\Phi(z) = 1 - \phi_1 z - \phi_2 z^2 - \dots - \phi_p z^p$ 是模型的**[特征多项式](@entry_id:150909) (characteristic polynomial)**。因果性要求我们能将 $X_t$ 写成 $X_t = \Phi(B)^{-1}W_t$，其中 $\Phi(B)^{-1}$ 是一个关于 $B$ 的收敛幂级数。这引出了一个核心定理：**一个[AR(p)过程](@entry_id:142888)是因果的，当且仅当其[特征方程](@entry_id:265849) $\Phi(z) = 0$ 的所有根（复数根）都位于单位圆之外**，即所有根的模 $|z| > 1$ [@problem_id:1925237]。

#### 揭示冲击：[MA模型](@entry_id:191881)中的[可逆性](@entry_id:143146)

**[可逆性](@entry_id:143146) (invertibility)** 是[MA模型](@entry_id:191881)的一个对偶概念。一个[MA模型](@entry_id:191881)是可逆的，如果我们可以将当前的[白噪声](@entry_id:145248)冲击 $W_t$ 表示为当前和**过去**的观测值 $X_t, X_{t-1}, X_{t-2}, \dots$ 的收敛线性组合。这个属性非常实用，因为它允许我们从观测到的数据中“反推”出那些不可见的随机冲击，这对于[模型诊断](@entry_id:136895)和更新预测至关重要。

MA(q)模型可以写为 $X_t - \mu = \Theta(B)W_t$，其中 $\Theta(z) = 1 + \theta_1 z + \dots + \theta_q z^q$ 是MA的特征多项式。[可逆性](@entry_id:143146)意味着我们可以写出 $W_t = \Theta(B)^{-1}(X_t - \mu)$。与因果性类似，这要求 $\Theta(B)^{-1}$ 是一个收敛的[幂级数](@entry_id:146836)。因此，我们有另一个核心定理：**一个[MA(q)过程](@entry_id:144037)是可逆的，当且仅当其[特征方程](@entry_id:265849) $\Theta(z) = 0$ 的所有根都位于[单位圆](@entry_id:267290)之外**。

例如，对于一个MA(2)模型，其参数 $\theta_1$ 和 $\theta_2$ 必须满足一组条件，即 $|\theta_2|  1$, $\theta_1 + \theta_2 > -1$ 和 $\theta_1 - \theta_2  1$，才能保证可逆性 [@problem_id:1925241]。这些看似复杂的条件正是确保其[特征多项式](@entry_id:150909) $1 + \theta_1 z + \theta_2 z^2 = 0$ 的根都落在[单位圆](@entry_id:267290)外的具体体现。

#### 唯一性问题：为何可逆性如此重要

[可逆性](@entry_id:143146)不仅仅是一个技术上的要求，它还解决了一个关于[模型识别](@entry_id:139651)的根本性问题。考虑两个MA(1)过程 [@problem_id:1925218]：
-   模型一: $X_t = W_t + 4 W_{t-1}$
-   模型二: $Y_t = V_t + 0.25 V_{t-1}$

我们可以计算出任何MA(1)过程 $Z_t = \epsilon_t + \theta \epsilon_{t-1}$ 的[自相关函数 (ACF)](@entry_id:139144) 在滞后1时为 $\rho(1) = \frac{\theta}{1+\theta^2}$，在其他滞后（除0外）均为0。对于模型一，$\theta_X=4$，$\rho_X(1) = \frac{4}{1+16} = \frac{4}{17}$。对于模型二，$\theta_Y=0.25$，$\rho_Y(1) = \frac{0.25}{1+0.25^2} = \frac{0.25}{1.0625} = \frac{4}{17}$。

这两个截然不同的模型竟然拥有完全相同的自相关函数！这意味着仅凭ACF，我们无法区分它们。这个问题如何解决？答案就在可逆性。对于一个参数为 $\theta$ 的MA(1)模型，其特征方程是 $1+\theta z=0$，根为 $z=-1/\theta$。可逆性要求 $|-1/\theta| > 1$，即 $|\theta|  1$。在我们的例子中，模型一的参数是4，不可逆；模型二的参数是0.25，是可逆的。**按照惯例，当面临这种模糊性时，我们总是选择可逆的表示**。这不仅为每个ACF结构提供了唯一的[MA模型](@entry_id:191881)，而且保证了模型在实践中的良好表现。

### 从理论到实践：[尤尔-沃克方程](@entry_id:267787)

到目前为止，我们已经定义了[AR模型](@entry_id:189434)及其[平稳性条件](@entry_id:191085)。现在的问题是：如何将理论与实际数据联系起来？如果我们从数据中估计出了[自相关函数](@entry_id:138327)，能否反过来推断出[AR模型](@entry_id:189434)的参数呢？**[尤尔-沃克方程](@entry_id:267787) (Yule-Walker Equations)** 正是连接这两者的桥梁。

对于一个平稳的[AR(p)过程](@entry_id:142888) $X_t = \sum_{i=1}^p \phi_i X_{t-i} + W_t$，我们将方程两边同乘以 $X_{t-k}$（对于 $k > 0$），然后取期望：
$$E[X_t X_{t-k}] = \sum_{i=1}^p \phi_i E[X_{t-i} X_{t-k}] + E[W_t X_{t-k}]$$
根据定义， $E[X_s X_t] = \gamma(s-t)$。另外，由于 $X_{t-k}$ 仅依赖于 $t-k$ 时刻及之前的白噪声，它与未来的 $W_t$ 不相关，因此 $E[W_t X_{t-k}] = 0$。于是我们得到：
$$\gamma(k) = \sum_{i=1}^p \phi_i \gamma(k-i), \quad \text{for } k=1, 2, \dots, p$$
将上式两边同除以[方差](@entry_id:200758) $\gamma(0)$，并利用 $\rho(k) = \gamma(k)/\gamma(0)$ 和 $\rho(k-i)=\rho(i-k)$，我们得到以自相关函数表示的[尤尔-沃克方程](@entry_id:267787)：
$$\rho(k) = \sum_{i=1}^p \phi_i \rho(k-i), \quad \text{for } k=1, 2, \dots, p$$
这是一个关于未知参数 $\phi_1, \dots, \phi_p$ 的 $p$ 元[线性方程组](@entry_id:148943)。

设想一个场景：一个高精度传感器的随机噪声被建模为平稳的AR(2)过程，通过实验我们测得其自相关函数在滞后1和2时分别为 $\rho(1)=0.8$ 和 $\rho(2)=0.5$ [@problem_id:1312105]。我们可以利用[尤尔-沃克方程](@entry_id:267787)来确定模型参数 $\phi_1$ 和 $\phi_2$：
-   对于 $k=1$: $\rho(1) = \phi_1 \rho(0) + \phi_2 \rho(-1) = \phi_1 + \phi_2 \rho(1)$
-   对于 $k=2$: $\rho(2) = \phi_1 \rho(1) + \phi_2 \rho(0) = \phi_1 \rho(1) + \phi_2$

代入已知数值，我们得到[方程组](@entry_id:193238)：
$$0.8 = \phi_1 + 0.8 \phi_2$$
$$0.5 = 0.8 \phi_1 + \phi_2$$
解这个简单的[线性方程组](@entry_id:148943)，我们得到 $\phi_1 \approx 1.11$ 和 $\phi_2 \approx -0.389$。这个过程展示了如何利用样本的自相关性来估计模型参数，这种方法被称为**[矩估计法](@entry_id:270941) (method of moments)**，是连接时间序列理论与数据分析实践的第一步。

本章所介绍的[平稳性](@entry_id:143776)、AR/[MA模型](@entry_id:191881)、因果性、[可逆性](@entry_id:143146)以及[尤尔-沃克方程](@entry_id:267787)，共同构成了经典[时间序列分析](@entry_id:178930)的理论核心，为后续更高级的建模与预测技术奠定了坚实的基础。
